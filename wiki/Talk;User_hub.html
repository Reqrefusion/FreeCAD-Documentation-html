How to Talk

This page is for your questions and wishes on the FreeCAD user documentation. To start a new discussion thread use the +-button next to the edit button on the top of the page. To add a new comment to an existing discussion thread start it with one colon at the beginning of the each new line. The next comment uses two colons and so on. At the end of your entry add two dashes a blank and four tildes: 

-- ~~~~

This expands to your signature and clearly marks the end of your comment. And last but not least, before submitting your new entry, please use the preview button to check what you wrote. Thank you for remembering these rules on all talk pages.
