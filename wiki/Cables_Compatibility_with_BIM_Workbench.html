Cables Compatibility with BIM Workbench


Compatibility table of the Cables Workbench with the BIM Workbench

The table below contains a summary of areas in which the elements created in Cables Workbench match the rules and structure from BIM Workbench.

Elements are understood to be all objects with a shape type solid or compound of solids. There are currently six such objects in Cables Workbench: Cable, CableConduit, CableBox, CableConnector, CableLightPoint, ElectricalDevice
