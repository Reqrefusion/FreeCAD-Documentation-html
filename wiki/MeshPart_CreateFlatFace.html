Description

The MeshPart CreateFlatFace command creates a flat representation of a face of a shape object. The created outline is a Part Feature. At this time the command almost always creates a rectangular outline.

Usage

  1. Select a single face of a shape object in the 3D View.
  2. Select the Meshes → Unwrap Face option from the menu.

Properties

See: Part Feature.
