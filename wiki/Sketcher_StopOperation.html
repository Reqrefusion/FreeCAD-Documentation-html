Description

The Sketcher StopOperation tool stops any currently running Sketcher tool.

Usage

See also: Continue modes.

  1. Make sure a geometry or constraint creation tool is active.
  2. Select the Sketch → Stop Operation option from the menu.
  3. The running tool is deactivated and the cursor changes to a simple arrow.
