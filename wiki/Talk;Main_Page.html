Discussion about FreeCAD

For discusing topics general to FreeCAD please use:

  • the FreeCAD forum for general things, requesting help, discussing a topic, etc...
  • the Bug tracker to report Bugs or failures or to do feature requests
  • the FreeCAD IRC channel for general FreeCAD-related chat
  • the mailing list if you prefer to discuss by email

For discussion about the documentation use the discussion on the special pages.

Discussions sur FreeCAD

Pour discuter de sujets d'ordre général concernant FreeCAD, veuillez utiliser :

  • le forum FreeCAD pour les questions générales, les demandes d'aide, les discussions sur un sujet, etc.
  • le système de suivi des bugs pour signaler des bugs ou des dysfonctionnements ou pour demander de nouvelles fonctionnalités
  • le canal IRC de FreeCAD pour des discussions générales autour de FreeCAD
  • la liste de diffusion si vous préférez échanger par e-mail

Pour les discussions concernant la documentation, utilisez les espaces de discussion dédiés sur les pages correspondantes.
