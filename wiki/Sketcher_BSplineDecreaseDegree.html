Description

The Sketcher BSplineDecreaseDegree tool decreases the degree (order) of B-splines.

Usage

  1. Select one or more B-splines.
  2. There are several ways to invoke the tool:
    • Press the Decrease B-Spline Degree button.
    • Select the Sketch → B-Spline Tools → Decrease B-Spline Degree option from the menu.
    • introduced in 1.0: Right-click in the 3D View and select the Decrease B-Spline Degree option from the context menu.

See Sketcher BSplineIncreaseDegree.
