Description

Tar ut den gemensamma delen (skärningen) mellan valda Del objekt. Denna operation är helt parametrisk så de ingående Del objekten kan förändras och så kommer det gemensamma att förändras när det beräknas om.

Usage

  1. Select two shapes
  2. There are several ways to invoke the tool:
    • Press the Common button.
    • Select the Part → Boolean → Common option from the menu.

Supported inputs

Input objects must be OpenCASCADE shapes. Examples: stuff made with Part, PartDesign, Sketcher workbenches. Not meshes (unless those were converted to shapes) - for meshes, there are specific Boolean tools in MeshDesign workbench.
