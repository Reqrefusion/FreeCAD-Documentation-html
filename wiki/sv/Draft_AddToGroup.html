Description

The Draft AddToGroup command adds objects to a Std Group or a group-like BIM object. It can also ungroup objects.

Usage

  1. Select one or more objects.
  2. There are several ways to invoke the command:
    • Press the Add to Group button.
    • Select the Utilities → Add to Group option from the menu, or from the Tree View or 3D View context menu.
  3. A menu is displayed near the cursor. Do one of the following:
    • Select Ungroup to remove the objects from the group they are in.
    • Select the group you want to add the objects to.
    • Select Add to New Group to add the objects to a new group:
      1. The New Group dialog box opens. introduced in 1.1
      2. Enter a Group name.
      3. Press the OK button.

Notes

  • Objects can also be moved to a group by drag and dropping them on the group in the Tree View.
  • This command can be used to move objects to the Draft construction group, but, contrary to the Draft AddConstruction command, it does not apply the construction geometry color.
  • For more information about organizing your model see Document structure and Arch tutorial.
