簡介

這是 [https://freecad.org 

FreeCAD] 的wiki文件。使用這份文件有幾種方式：可以透過探索各個中心（hubs）、依照手冊學習，或是在 FreeCAD 內使用「說明」選單。這份文件仍在持續更新中，由 FreeCAD 的使用者與開發者社群共同撰寫。如果你發現有錯誤或遺漏的資訊，請 協助 FreeCAD！

中心（Hubs）

使用者中心：此頁面包含對一般 FreeCAD 使用者有用的文件：所有工作台列表、FreeCAD 應用程式的安裝與使用詳細說明、教學，以及開始使用所需的一切資訊。

高級使用者中心：此頁面收集了針對高級使用者及對撰寫 Python 腳本有興趣的人的文件。在這裡你也會找到巨集庫、安裝與使用說明，以及更多關於如何依照個人需求自訂 FreeCAD 的資訊。

開發者中心：此部分包含給開發者的資料：如何自行編譯 FreeCAD、FreeCAD 原始碼的結構、如何瀏覽原始碼、如何開發新工作台，以及如何將 FreeCAD 嵌入自己的應用程式。

手冊

FreeCAD 手冊 是另一種、更線性的方法來呈現此維基中的資訊。它設計成像書本般閱讀，並會循序引導你瀏覽上方各個中心的其他頁面。[https://www.gitbook.com/book/yorikvanhavre/a-freecad-manual/details 

電子書版本] 也可取得，還有 [https://www.freecad.org/manual/
幾種 PDF 格式的翻譯版本]。

參與貢獻

如何參與

如果你有興趣協助，我們的 FreeCAD 專案中有很多事情可以做。當然，對於 C++ 或 Python 程式設計師有程式相關的工作，但即使你不會程式，也有許多可以參與的方式，例如：

  • 撰寫文件與 編輯維基（英文）
  • 協助新手入門
  • 翻譯應用程式與文件
  • 協助打包最新版本的 FreeCAD，以支援你喜愛的作業系統
  • 幫助周遭的人發現並使用 FreeCAD

協助 FreeCAD 頁面提供了更詳細的說明。自 2016 年起，FreeCAD 也參與了 Google 夏季程式 計畫。

原始碼

FreeCAD 可以在所有平台上透過 [https://cmake.org/ 

CMake] 進行編譯。其原始碼採用 [https://en.wikipedia.org/wiki/GNU_Lesser_General_Public_License
LGPL] 授權，託管於 [https://github.com/FreeCAD/FreeCAD
GitHub]，並在 [https://gitlab.com/freecad/FreeCAD
GitLab]、[https://sourceforge.net/projects/free-cad/
SourceForge] 和 [https://codeberg.org/FreeCAD/FreeCAD
CodeBerg] 提供鏡像。

提供了 Windows、Linux 和 MacOS 的編譯指引。原始碼文件 [https://freecad.org/api/ 

託管於此]，由 Doxygen 生成，並在 維基上有文件說明。

關於開發

查看 開發路線圖 以了解計劃中的新功能，或到 [https://tracker.freecad.org/changelog_page.php 

更新日誌] 和 [https://tracker.freecad.org/roadmap_page.php
路線圖] 頁面（位於 [https://github.com/FreeCAD/FreeCAD/issues
FreeCAD 問題追蹤器]）查看下一版本的進展情況。若想獲取更多關於 FreeCAD 程式碼基礎的資訊，也可參考 [http://www.ohloh.net/p/freecad
專案統計]。

所有開發相關的交流都在 [https://forum.freecad.org 

論壇] 進行，如果你有興趣參與，務必前往參與討論。
