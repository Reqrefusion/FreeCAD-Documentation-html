Description

The FreeGrid BitCartridgeHolder tool creates a bit cartridge holder.

This tool is part of the FreeGrid Workbench, an external workbench that can be installed with the Addon Manager.

Usage

  1. There are several ways to invoke the tool:
    • Press the Bit cartridge holder button.
    • Select the FreeGrid → Bit cartridge holder option from the menu.
  2. A bit cartridge holder is created with properties based on the current preferences.
  3. Optionally select the object and change its properties in the Property editor.

Properties

A FreeGrid BitCartridgeHolder object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

Bit Cartridge Holder features

  • DataSide Length (Length): Default is 15.0 mm. Length of the longest side of the holder.

Box features

  • DataFloor Support (Bool): Default is true. Add integral floor support.

Magnet mount

  • DataMagnet Diameter (Length): Default is 6 mm. Diameter of the magnets.
  • DataMagnet Height (Length): Default is 2 mm. Height of the magnets.
  • DataMagnet Option (Enumeration): Default is allIntersections. Positions of the magnets. The other options are: cornersOnly and noMagnets.

Position on grid

  • DataPosition X (IntegerConstraint): Default is 0. Object position on the grid along the X axis. Starts at zero.
  • DataPosition Y (IntegerConstraint): Default is 0. Object position on the grid along the Y axis. Starts at zero.

Size

  • DataDepth (IntegerConstraint): Default is 1. Number of 50 mm units in the Y direction of the object.
  • DataHeight (Length): Default is 50 mm. Height of the object.
  • DataWidth (IntegerConstraint): Default is 1. Number of 50 mm units in the X direction of the object.
