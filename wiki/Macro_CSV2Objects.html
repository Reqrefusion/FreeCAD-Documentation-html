Description

CSV2Objects generates large batches of 3D objects with text driven by CSV files. It uses horizontal guide lines in a sketch to position text, creates ShapeStrings, extrudes them, and optionally fuses the result with a target solid. It supports batch export to STL, 3MF, and STEP.

Usage

  1. Open a document with a sketch containing horizontal guide lines, and optionally a target solid.
  2. Run the macro CSV2Objects.FCMacro from the Macro menu.
  3. In the task panel, choose a CSV file, encoding and delimiter.
  4. Scan guide lines and assign CSV columns to each active line.
  5. Choose a font, extrusion height, boolean mode and export format (STL/3MF/STEP).
  6. Use the live preview to adjust your settings.
  7. Press OK to generate and export the geometry for all CSV rows.
  8. Press Cancel to clear preview objects and close the dialog.
