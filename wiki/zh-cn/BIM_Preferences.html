Introduction

The preferences for the BIM Workbench can be found in the Preferences Editor. In the menu select Edit → Preferences... and then BIM. This group is only available if the BIM Workbench has been loaded in the current FreeCAD session.

There are three pages: General settings, Defaults and Native IFC.

Some advanced preferences can only be changed in the Parameter editor. See Fine-tuning.

This page has been updated for version 1.0.
