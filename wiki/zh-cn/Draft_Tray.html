Description

通过底图工具栏，您可以快速改变底图工作台对象的可视属性。

The Draft Tray

Available tools

  • Working Plane: selects the current Draft working plane.
  • Set Style: sets the default style for new objects.
  • Toggle Construction Mode: switches Draft construction mode on or off.
  • AutoGroup: changes the active Draft Layer or, optionally, the active Std Group or group-like BIM object.
