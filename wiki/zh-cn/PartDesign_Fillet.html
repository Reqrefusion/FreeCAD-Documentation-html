描述

零件设计 圆角 用于在选定对象的边缘创建圆角。它会在文档中添加一个 Fillet 对象，并在 树视图 中显示对应的节点。

所有边缘都已添加圆角的立方体

用法

添加一个圆角

  1. 选择 激活 的实体用于圆角。
  2. 有几种选择要倒圆角边的方式:
    • 单独选择实体的一条或多条边.
    • 选择实体的一个或多个面以选中它们所有的边.
    • 选择实体的一个特征（通常是最后一个特征）以选中其所有的边.
  3. 对于切线相连的边链，只需选择一条边，倒圆角将沿该链传播。
  4. 有几种调用该工具的方式:
    • 点击 圆角 按钮。
    • 从菜单选择 零件设计 → 修正特征 → 圆角 选项。
  5. 如果没有激活的实体，并且文档中有两个或两个以上的实体，则会打开 需要激活状态的实体 对话框，提示您选择并激活一个实体。如果只有一个实体，它将自动被激活。
  6. 圆角参数 任务面板 将被打开。查看 选项 获取更多信息。
  7. 点击 OK 按钮完成。

编辑一个圆角

  1. 执行以下操作之一:
    • 在 树视图 中双击 Fillet 对象。
    • 鼠标右键单击 树视图 Fillet 对象，在上下文菜单中选择 编辑圆角。
  2. 圆角参数 任务面板 将打开。更多信息请参见 选项。
  3. 点击 OK 按钮完成编辑。

选项

  • 要添加边，请执行以下操作之一:
    • 按下 添加 按钮，开始在 3D 视图中选择边和/或面。
    • 要选择所有剩余的边，请执行以下操作:
      1. 如有需要，按下 添加 按钮.
      2. 使用 Ctrl+Shift+A 键盘快捷键，或右键单击列表并从上下文菜单中选择 添加所有边。
  • 要移除边，请执行以下操作之一:
    • 按下 移除 按钮，开始在 3D 视图中取消选择边和/或面。选中的元素会以紫色高亮显示。
    • 在列表中选择一个或多个元素，然后按下 Del 键，或右键单击列表并从上下文菜单中选择 移除。
  • 设置圆角的 半径。
  • 勾选 使用所有边 复选框，以选择前一个特征的所有边。这将停用选择列表及相关按钮。

说明

  • 零件设计 圆角不应与 零件 圆角 混淆。除非您清楚自己在做什么，否则不应在零件设计实体上使用 零件 圆角。请参阅 零件与 零件设计。
  • 圆角半径不能超过相邻面的尺寸。

属性

另请参阅：属性视图。

零件设计 圆角对象派生自 零件 特征 对象，并继承其所有属性。它还具有以下附加属性：

数据

基础

  • 数据Base (LinkSub)：链接到父特征中选定的边和面。如果 数据Use All Edges 为 true，则可以是仅指向父特征的链接。
  • 数据Support Transform (Bool)：如果为 true，则当倒圆角对象包含在 阵列 中时，将使用加性/减性父特征的倒圆角形状；否则仅使用倒圆角本身的形状。默认值为 false。
  • 数据 (hidden)Add Sub Shape (PartShape)（隐藏）
  • 数据 (hidden)Base Feature (Link)（隐藏）：指向父特征的链接。
  • 数据 (hidden)_ Body (LinkHidden)（隐藏）：指向父实体的链接。

圆角

  • 数据Radius (QuantityConstraint)：圆角半径。默认值为 1 mm。
  • 数据Use All Edges (Bool)：如果为 true，则特征的所有边均被倒圆角，而由 数据Base 指定的边将被忽略。默认值为 false。

零件设计

  • 数据Refine (Bool)：如果为 true，则操作结果中的冗余边将被移除。默认值由 基于草图的操作后自动优化模型 偏好设置决定。请参阅 零件设计首选项。

已知问题

倒圆角、倒角及其他对实体进行操作的特征，依赖于 FreeCAD 所使用的底层 OpenCASCADE 技术(OCCT) 内核。OCCT 内核在处理两个面相交处的重合锐边时，有时会遇到困难。如果出现这种情况，FreeCAD 可能会在没有任何提示的情况下崩溃。

如果在终端中运行，FreeCAD 在崩溃后可能会输出类似如下的日志：

拓扑命名问题

边的编号并非完全稳定，因此建议您在完成实体主体的主要设计工作之后，再应用圆角和倒角等特征，否则边的名称可能会发生变化，导致已倒圆角的边失效。当 数据Use All Edges 属性为 true 时，可以在一定程度上避免此问题。因为在这种情况下，会使用基础对象的所有边，而不依赖于单个边的名称。

更多内容请阅读 拓扑命名问题。
