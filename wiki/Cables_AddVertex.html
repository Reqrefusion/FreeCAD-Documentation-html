Description

The Add Vertex adds a new vertex to the WireFlex object. It preserves proper assignment of existing attachments of wire vertexes. It should be used instead of "Add point" command from Draft Edit, see the WireFlex Description.

Usage

  1. Select one edge of an existing WireFlex object in the 3D view.
  2. Create a new vertex of WireFlex by one of these methods:
    • Press the Add Vertex button.
    • Select the Cable Wires → Add Vertex option from the menu.
    • Right-click in the Tree view or the 3D view and select the Cable Wires → Add Vertex option from the context menu.

The new vertex is added in the middle of selected edge. The vertex position can be edited later with Draft Edit command or attached to other objects with Attach Vertex.

Notes

  • See the WireFlex Description and WireFlex Usage for the details how to use WireFlex
