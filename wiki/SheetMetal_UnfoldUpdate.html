SheetMetal Unfold

Description

The Unfold Update command updates all unfold objects.

Usage

  1. There are several ways to invoke the command:
    • Press the Unfold Update button.
    • Select the Sheet Metal → Unfold Update option from the menu.
    • Right-click in the Tree View or the 3D View and select the Sheet Metal → Unfold Update option from the context menu.
  2. All Unfold objects are updated.
