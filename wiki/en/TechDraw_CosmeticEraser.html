Description

The TechDraw CosmeticEraser tool removes cosmetic objects. Cosmetic objects include cosmetic vertices, cosmetic lines and centerlines.

Usage

  1. Select one or more cosmetic objects on a page.
  2. There are several ways to invoke the tool:
    • Press the Remove Cosmetic Object button.
    • Select the TechDraw → Remove Cosmetic Object option from the menu.

Notes

  • You can also select cosmetic objects and press Delete. introduced in 1.0
