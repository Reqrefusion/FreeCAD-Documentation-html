Description

Defines a centrifugal force body load.

Fan subjected to centrifugal force load

Usage

  1. There are several ways to invoke the command:
    • Press the Centrifugal Load button.
    • Select the Model → Mechanical Boundary Conditions and Loads → Centrifugal Load option from the menu.
  2. Specify the rotation frequency in Hz.
  3. Click on Add in the first Geometry reference selector window and select a solid to which the load will be applied in the 3D View. introduced in 1.1: To remove solids from the selection, press the Remove button.
  4. Click on Add in the second Geometry reference selector window and select an edge about which the rotation will occur in the 3D View. introduced in 1.1: To remove edges from the selection, press the Remove button.

Notes

  • This feature uses the *DLOAD card in CalculiX.
