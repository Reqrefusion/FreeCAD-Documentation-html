Description

The Robot ExportKukaFull command exports a trajectory as a full KRL subroutine.

Usage

  1. Select a robot and a trajectory in the Tree View.
  2. Select the Robot → Export Trajectory → Kuka Full Subroutine option from the menu.
  3. A KRL file (*.src) is created.
