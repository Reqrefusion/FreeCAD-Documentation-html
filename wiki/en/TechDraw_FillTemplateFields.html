Description

The FillTemplateFields tool fills the editable text fields in the template of the current page with entries from document information.

Usage

  1. Before starting the tool, create a page with the selected template and add a view.
  2. There are several ways to invoke the tool:
    • Press the Update template fields button.
    • Select the TechDraw → Page → Update template fields option from the menu.
  3. The Fill Template Fields in Page window will appear.
  4. Deselect the Update checkboxes next to fields that you don't want to update. You can also use the Update All checkbox to enable and disable all fields. Optionally, edit the strings shown on the right if you want to modify the data taken from document information.
  5. Press Ok to update the fields or Cancel to discard the changes.

Notes

  • This tool relies on the FillTemplateFields.csv spreadsheet file in the FreeCAD installation directory under .../share/Mod/TechDraw/CSVdata.
  • introduced in 1.0: There is also the autofill tool.
