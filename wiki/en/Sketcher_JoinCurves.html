Description

The Sketcher JoinCurves tool creates a B-spline by joining two existing B-splines or other edges.

Usage

  1. Do one of the following:
    • Select an end point of two B-splines or other edges.
    • Select a coincident point connecting two B-splines or other edges.
  2. There are several ways to invoke the tool:
    • Press the Join curves button.
    • Select the Sketch → B-Spline Tools → Join Curves option from the menu.
    • introduced in 1.0: Right-click in the 3D View and select the Join Curves option from the context menu.
