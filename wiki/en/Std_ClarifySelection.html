Description

The Std ClarifySelection command displays a menu listing nearby geometrical entities of different types next to the cursor when hovering the model.

Usage

  1. Place the cursor over geometry in the 3D View.
  2. There are several ways to invoke the command:
    • Select the Clarify Selection option from the 3D View context menu.
    • Use the keyboard shortcut: G then G.
    • When using the OpenInventor, Gesture or OpenSCAD navigation style: hold Ctrl and press the left mouse button.
    • When using any other navigation style: hold the left mouse button for as long as defined by the Long press timeout preference (1 second by default).
  3. A menu listing object types appears next to the cursor. Depending on the objects near the cursor, it may include:
    • Object
    • Face
    • Edge
    • Vertex
    • Other
  4. Choose an object type.
  5. Move the cursor over the submenu that appears. Hovered submenu items are highlighted in the 3D View.
  6. Click an item to select it.
