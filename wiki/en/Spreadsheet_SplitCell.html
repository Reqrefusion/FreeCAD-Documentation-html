Description

The Spreadsheet SplitCell tool splits a previously merged cell.

Usage

  1. Make sure there is an active spreadsheet.
  2. Select a merged cell.
  3. There are several ways to invoke the tool:
    • Press the Split Cell button.
    • Select the Spreadsheet → Split Cell option from the menu.
    • Right-click the selected cell and select the Split Cell option from the context menu.
