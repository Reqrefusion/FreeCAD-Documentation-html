Description

Interpolates between two 2D curves. The interpolated curves can be resized in the bounds of some hullcurves.

Image of a simple FreeCAD project with two sketches using the CurvedSegment tool

Usage

  1. Select two sketches that shall be start and end of the interpolation.
  2. There are several ways to invoke the command:
    • Press the Curved Segment button
    • Select the Curved Shapes → Curved Segment option from the menu

Notes

  • If you need a solid of the interpolation enable the "make Solid" property.
