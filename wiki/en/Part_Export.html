Description

The Part Export command exports selected objects to a different file format. The supported file formats are: STEP (with or without colors), IGES (with or without colors), and BREP.

Usage

  1. Set up the Import Export Preferences in the Preferences Editor.
  2. Select one or more objects. See Std Export.
  3. Select the Part → Export CAD file... option from the menu.
  4. Select the correct file format in the dialog box.
  5. Enter a filename.
  6. Press the Save button.

Notes

  • The Std Export command supports many more file formats.
