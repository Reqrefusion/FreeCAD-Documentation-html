Macro Start MCP Bridge

Description
Start the MCP (Model Context Protocol) bridge server which is part of the FreeCAD Robust MCP server which helps facilitate AI assistant integration with FreeCAD. Enables AI tools like Claude Code to control FreeCAD programmatically.

Description

This Macro has been fully deprecated and has been replaced with a proper workbench.

- See: https://wiki.freecad.org/Robust_MCP_Bridge_Workbench

This macro was originally used to start the MCP (Model Context Protocol) bridge which is part of the FreeCAD Robust MCP server on GitHub. This enables AI assistants like Claude Code and other MCP-compatible clients to control FreeCAD programmatically.
