See also
None

Description

The SheetMetal NewSketch command creates a new sketch without switching workbenches.

It invokes either the Sketcher NewSketch tool, or the PartDesign NewSketch tool if a PartDesign Body is active.

Usage

  1. There are several ways to invoke the command:
    • Press the New Sketch button.
    • Select the Sheet Metal → New Sketch option from the menu.
    • Right-click in the Tree View or the 3D View and select the Sheet Metal → New Sketch option from the context menu.
  2. For further steps see Sketcher NewSketch or PartDesign NewSketch respectively.
