Description

Increases tolerance of edges/faces/vertex of selected object(s).

Usage

  1. Select object in the tree view
  2. Select the OpenSCAD → Increase Tolerance Feature option from the menu.
  3. Select Tolerance object in the Tree View.
  4. Select Data tab in Property View or Combo View.
  5. Change values for Edges/Faces/Vertex to your liking.
