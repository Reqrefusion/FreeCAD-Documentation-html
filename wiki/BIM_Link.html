Description

The BIM Link tool is a shortcut to Link, but starts a Draft Move operation immediately after, allowing a link to be created and placed in one go.

Usage

  1. Select an object.
  2. Start the BIM Link command.
  3. A link is created from the selected object.
  4. The Draft Move command starts automatically.
  5. Click in the 3D View to place the new link.

Notes

  • BIM Link is intended to provide a BIM-oriented workflow similar to BIM Clone, but using Link.
  • The command is useful when you want to create a linked copy and place it immediately.
