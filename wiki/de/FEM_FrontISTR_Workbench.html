Einleitung

Der FEM Arbeitsbereich FrontISTR ist ein FreeCAD-Addon, das FrontISTR ermöglicht, ein Open-Source-Programm für die nichtlineare Strukturanalyse mit groß angelegten parallelen FEM-Berechnungen.

Beabsichtigter Arbeitsablauf

  1. Ein Analysemodell mit dem FEM-Modul einrichten (auf die gleiche Weise wie in CalculiX).
  2. Zum Arbeitsbereich FrontISTR wechseln und ein FrontISTR-Löser-Objekt erstellen, indem man auf die Schaltfläche in der Symbolleiste klickt.
  3. Doppelklick auf das Löser-Objekt im Dokumentbaum um das Arbeitsverzeichnis festzulegen.
  4. Auf die Schaltfläche Eingabedatei schreiben klicken
  5. Auf die Schaltfläche FrontISTR ausführen klicken
  6. Die FISTR_Results für die Nachbearbeitung überprüfen.

Funktionen

  • static analysis, element check
  • geometrical linear|nonlinear analysis
  • elements: 1st/2nd order tetrahedron
  • loads: mechanical concentrated and distributed loads, gravity
  • boundary: points fixed or displacement
  • step control: auto time increment and cutback
  • linear equation solver
    • iterative
      • preconditioner: AMG, SSOR, Diagonal, ILU(k)(k=0,1,2)
      • method: CG, BiCGSTAB, GMRES, GPBiCG
    • direct: MUMPS
  • output file format: AVS, VTK (ParaView required)

Zukünftig unterstützte Funktionen

  • analysis: thermal transfer, dynamic, eigenfrequency
  • materials(mechanical): elastoplastic, hyperelastic, creep, viscoelastic
  • contact
  • MPC(TIE)
  • elements: prism, hexa, beam, shell, truss, etc.

Begrenzungen

  • FISTR_Results enthält nur Ergebnisse für Oberflächen. Wenn man die Ergebnisse für das Innere benötigt, ändert man das Ausgabe-Dateiformat in VTK und visualisiert die Ergebnisse mit ParaView.
  • Der FEM Arbeitsbereich FrontISTR unterstützt noch keine thermischen Analysen. FISTR selbst kann thermische Analysen durchführen, und die Unterstützung hierfür ist in naher Zukunft geplant.
  • Mechanische Analysen mit verschiedenen Materialien im Modell sind noch nicht möglich.

Installation

Addon-Manager

FEM_FrontISTR kann einfach über den FreeCAD- Addon-Manager aus dem Menü Werkzeuge → Addon-Manager installiert werden. FEM_FrontISTR befindet sich in aktiver Entwicklung und wird regelmäßig um neue Funktionen erweitert. Daher sollte es regelmäßig über das Menü aktualisiert werden. Werkzeuge → Addon-Manager. Der FEM_FrontISTR-Code wird gehostet und entwickelt auf GitHub.

Installation des Lösers FrontISTR

Die FrontISTR-Binärdateien werden beim ersten Start automatisch heruntergeladen und installiert. Wenn der Download nicht funktioniert, führen man die folgenden Schritte aus, um den Löser zu installieren.

Windows

  1. Download FrontISTR-latest.zip
  2. Create directory FEM_FrontISTR/bin
  3. Extract FrontISTR-latest.zip and put all files in FEM_FrontISTR/bin directory.
