Beschreibung

The Templater AuxView tool adds an auxiliary view to the active TechDraw drawing page based on a straight line in an existing base view. This line, together with the Z-axis of the base view, defines the orientation of the auxiliary view. The end vertices of the line can either be selected separately or derived from a selected edge.

Anwendung

  1. Zwei Knoten oder eine Kante in einer vorhanden Ansicht auswählen.
  2. Es gibt mehrere Möglichkeiten, das Werkzeug aufzurufen:
    • Die Schaltfläche Hilfsansicht drücken.
    • Den Menüeintrag Templater → Hilfsansicht auswählen.
  3. eine neue Hilfsansicht, ein AuxView-Objekt wird erstellt. Seine Sichtachse steht senkrecht auf der Linie zwischen den ausgewählten Knotenpunkten.
  4. Das Aufgaben-Fenster Einstellungen der Hilfsansicht wird geöffnet.
    • Wahlweise das Kennzeichen der Ansicht ändern.
    • Wahlweise die Richtung der Ansicht umkehren.
    • Wahlweise die Richtung entlang der Linie ausrichten.
  5. Ok anklicken, um das Aufgaben-Fenster zu schließen und das Werkzeug zu beenden.
  6. Die Ansicht und die Beschriftungselemente auf ihre endgültigen Positionen ziehen.

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

Ein AuxView-Objekt wird von einer TechDraw-Ansicht, formal ein TechDraw::DrawViewPart-Objekt abgeleitet und erbt alle seine Eigenschaften. Es besitzt auch die folgende zusätzliche Eigenschaft.
