Der Arbeitsbereich Start wird nach der Version 0.21. nicht mehr enthalten sein..
Er wurde durch die Start-Seite ersetzt, auf die über das Menü Hilfe zugegriffen werden kann.

Die Einstellungen für den Arbeitsbereich Start können im Einstellungseditor gefunden werden. Den Menüeintrag Bearbeiten → Einstellungen... auswählen und dann Start. Dieser Abschnitt ist nur verfügbar, nachdem der Arbeitsbereich verwendet wurde oder nach dem Laden des Arbeitsbereichs über den Reiter Verfügbare Arbeitsbereiche im Abschnitt Arbeitsbereiche.

Es gibt nur einen Reiter: Startseiten Einstellungen im Abschnitt Start. Auf diesem Reiter kann Folgendes angegeben werden:

Name Beschreibung
Startseiten-Vorlage Eine optionale HTML-Vorlage, die anstelle der Standardvorlage verwendet wird.
Notizblock anzeigen Wenn diese Option aktiviert ist, wird neben den Miniaturansichten der zuletzt verwendeten Dateien ein Notizblock angezeigt, in dem Sie über mehrere Sitzungen hinweg Notizen speichern können.
Beispiele für Ordnerinhalte anzeigen Wenn diese Option aktiviert ist, werden die Beispieldateien auf der Registerkarte Dokumente der Startseite angezeigt.
Zusätzlichen Ordner anzeigen Ein optionaler benutzerdefinierter Ordner, der unten auf der Registerkarte Documents der Startseite angezeigt wird.
Forum anzeigen Wenn diese Option aktiviert ist, werden die neuesten Beiträge aus dem FreeCAD-Forum auf der Registerkarte Activity der Startseite angezeigt.
Hinweise anzeigen Wenn diese Option aktiviert ist, werden Hilfethemen auf der Registerkarte Dokumente der Startseite angezeigt.
FreeCAD-Stylesheet verwenden Wenn diese Option aktiviert ist und in den allgemeinen Einstellungen ein Stylesheet angegeben ist, überschreibt das Stylesheet die Optionen in diesem Abschnitt.
Hintergrundfarbe Die Hintergrundfarbe hinter den Feldern. Standardmäßig ist sie blau.
Hintergrundbild Ein optionales Bild, das als Hintergrund angezeigt werden kann. Normalerweise ist dieses Feld leer.
Hintergrundtextfarbe Die Farbe des Versionstextes auf der Startseite. Standardmäßig ist diese weiß.
Seitenhintergrundfarbe Der Hintergrund des Hauptbereichs der Startseite. Standardmäßig ist er weiß.
Seitentextfarbe Die Farbe des Textes auf den Hauptseiten. Standardmäßig ist sie schwarz.
Rahmenhintergrundfarbe Die Hintergrundfarbe der Felder innerhalb der Seiten.
Verknüpfungsfarbe Die Farbe von URL-Verknüpfungen.
Schriftart Die für die Startseite zu verwendende Schriftartfamilie. Dies kann ein Schriftartname oder eine durch Kommas getrennte Reihe von Ersatzschriftarten sein. Der Standardwert ist Arial,Helvetica,sans, wodurch eine serifenlose Schriftart verwendet wird.

A spinbox allows setting the font size for all texts on the Start page. It defaults to 13px.

Use gradient for New File icon If checked, a gradient color will be used for the "New File" icon.
Switch workbench after loading Workbench to switch to after loading a file from the Start page, only used if Start is the start up workbench.
Close start page after loading If this preference and the Close and switch on opening file preference are both checked, and the application is started by opening a file, the Start page will be closed after FreeCAD finishes loading.
Close and switch on opening file See previous preference.
