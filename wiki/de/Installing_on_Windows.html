Standardinstallation

Der einfachste Weg, die aktuellste, stabile Version von FreeCAD zu installieren, ist die Verwendung des Installationsprogramms, siehe Seite Herunterladen.

Soll eine Entwicklungsversion, die möglicherweise instabil ist, herunterladen werden, siehe Weekly-Builds.

Nachdem das Installationsprogramm heruntergeladen wurde, startet der Installationsprozess nach einem Doppelklick darauf.

Nachfolgend finden sich weitere Informationen zu einigen technischen Optionen. Aber die meisten Nutzer benötigen nicht mehr als das Installationsprogramm. Nach Abschluss der Installation, geht man zu Erste Schritte.

Installation für alle Nutzer im Windowssystem

Voreingestellt wird FreeCAD für den Benutzer installiert, der das Installationsprogramm ausführt. Wenn dieser Benutzer nur Nutzerrechte hat, dann ist der Installationspfad: C:\Users\<username>\AppData\Local\Programs\FreeCAD X.YY

Wird das Installationsprogramm durch einen Administrator-Benutzer aktiviert, kann dieser wählen, ob FreeCAD für alle Benutzer im System oder nur für den aktuellen Benutzer installiert wird. Per Voreinstellung wird für alle Benutzer installiert.

Der voreingestellte Pfad für alle Benutzer ist: C:\Program Files\FreeCAD X.YY

Installation im Hintergrund

Um FreeCAD im Hintergrund zu installieren kann das Installationsprogramm über die Kommandozeile aktiviert werden: 

FreeCAD-~.exe /S

Für alle Optionen werden die Standardeinstellungen verwendet. Ein benutzerdefinierter Installationspfad kann wie folgt angegeben werden: 

FreeCAD-~.exe /S /D=A path to FreeCAD with spaces

Selbst bei einer Installation im Hintergrund wird per Vorgabe eine Meldung gezeigt, während der Installierer auf Korruption geprüft wird. Diese sogenannte zyklische Redundanzprüfung dauert meistens nur ein paar Sekunden. Um diese Prüfung auf Korruption zu unterbinden, muß folgendes eingegeben werden: 

FreeCAD-~.exe /S /NCRC

Beachte: diese /NCRC Angabe (flag) wird nicht empfohlen, da die Prüfung auf Korruption gewährleistet, daß das Installationsprogramm beispielsweise vollständig heruntergeladen wurde.

Chocolatey

Es wird jedoch dringend empfohlen, dass Du einen Paketmanager wie Chocolatey verwendest, um die Software auf dem neuesten Stand zu halten. Man kann Chocolatey wie folgt installieren diese Anweisungen und dann ein PowerShell-Terminal als Administrator öffnen und ausführen: 

choco install freecad

Von Zeit zu Zeit kann die Software aktualisiert werden, mit: 

choco upgrade freecad

Dies bekommt die neueste verfügbare Version aus dem Chocolatey-Repositorium. Wenn es irgendwelche Probleme mit dem Chocolateypaket gibt, kann man sich an die Betreuer unter auf dieser Seite wenden.

Deinstallation

Um FreeCAD zu de-installieren sollten vorzugsweise die Windowswerkzeuge verwendet werden. Andernfalls kann der Deinstallierer direkt verwendet werden. Dies ist die Datei: Uninstall-FreeCAD.exe Sie befindet sich im selben Ordner, in dem FreeCAD installiert ist.

Der Deinstallierer kann mit der folgenden Kommandozeile auch im Hintergrund ausgeführt werden: 

Uninstall-FreeCAD.exe /S

Beachte: die Deinstallation im Hintergrund verursacht einen Fehler, wenn eine Instanz von FreeCAD läuft, auch wenn die Instanz nicht die Version ist die deinstalliert werden soll.
