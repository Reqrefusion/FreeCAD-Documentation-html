Übersicht

Private Voreinstellungspakete können hinzugefügt werden, indem benutzerdefinierte GitHub-Repositorys zum Addon-Manager hinzugefügt werden. Dies ermöglicht Tests oder einfach die private Nutzung.

Siehe auch Voreinstellungspakete.

Benutzerdefiniertes Repository hinzufügen

Man beginnt mit dem Hinzufügen eines benutzerdefinierten Repositorys. Auf Bearbeiten → Einstellungen → Addon-Manager-Optionen gehen und auf die Schaltfläche + klicken, um das Dialogfeld Benutzerdefiniertes Repository zu öffnen.

Dialogfeld Benutzerdefiniertes Repository

Die URL des GitHub-Repositorys und den Branch eingeben.

In diesem Beispiel habe ich hinzugefügt:

https://github.com/macdroid53/ZolkoBase

Anschließend auf OK drücken.

Einstellungen mit hinzugefügtem benutzerdefinierten Repository

Anschließend auf OK drücken.

Benutzerdefiniertes Einstellungs-Paket im Addon-Manager

Das benutzerdefinierte Paket kann nun über Extras → Addon-Manager installiert werden.

Das Voreinstellungspaket anwenden

Nach der Installation geht man zu Bearbeiten → Einstellungen → Allgemein, sucht das Einstellungs-Paket und klickt auf Übernehmen. Alle angezeigten Anweisungen befolgen.
