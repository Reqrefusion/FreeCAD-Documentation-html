Template:PropertyView/de

Ansicht-Eigenschaft{{{1}}}

Diese Vorlge wird verwendet, um eine Ansicht-Eigenschaft darzustellen. Siehe auch Template:PropertyData/de, Template:TitleProperty/de.

Beispiel:

Das Schlüsselwort "Ansicht-Eigenschaft" wird ausgegeben. 

{{PropertyView/de|Line Width}}: Linienbreite der Kante.

Ansicht-EigenschaftLine Width: Linienbreite der Kante.

Beispiel 2:

Wird ein zweites Argument angegeben, wird damit die Art der Daten ausgegeben. 

{{PropertyView/de|Line Width|FloatConstraint}}: Linienbreite der Kante.

Ansicht-EigenschaftLine Width (FloatConstraint): Linienbreite der Kante.

Beispiel 3:

Wird ein drittes Argument angegeben, wird damit die Art der Eigenschaft ausgegeben. Eine oder mehrere dieser Arten lassen sich zuordnen: Hidden (ausgeblendet), ReadOnly (schreibgeschützt) and Output (Ausgabe). (alle drei Arten werden mit dem Schlüsselwort (ausgeblendet) gekennzeichnet) 

{{PropertyView/de|Proxy|PythonObject|hidden}}: Eine spezielle View-Provider-Klasse, die diesem Objekt zugeordnet ist.

Ansicht-Eigenschaft (ausgeblendet)Proxy (PythonObject): Eine spezielle View-Provider-Klasse, die diesem Objekt zugeordnet ist.
