Beschreibung

Das Werkzeug Curves ParametrischerFestkörper erstellt einen parametrischen Festkörper aus ausgewählten Flächen.

Eine aus 4 Flächen bestehende Tragflächenform vor (links) und ein einzelner Festkörper nach der Anwendung dieses Werkzeugs (rechts)

Anwendung

  1. Einige Flächen in der 3D-Ansicht auswählen oder einige Objekte in der Baumansicht.
  2. Es gibt mehrere Möglichkeiten, das Werkzeug aufzurufen:
    • Die Schaltfläche Parametrischer Körper drücken.
    • Den Menüeintrag Misc. → Parametrischer Körper auswählen.
  3. Ein Solid-Objekt wird erstellt, sein Status und das zugehörige Symbol beschreiben das Ergebnis:
    • Solid: Die ausgewählten Elemente ergeben eine geschlossene Hülle, die als Festkörperobjekt erkannt wird.
    • Shell: Die ausgewählten Elemente ergeben ein offenes Schalenobjekt.
    • Compound: ...
    • none: ...
  4. Wahlweise die Werte im Eigenschaften-Ansicht anpassen, um weitere Oberflächenelemente hinzuzufügen oder nur die offenen Kanten von Oberflächenformen anzuzeigen (siehe Eigenschaften).

Hinweise

  • This tool will try to build a solid from selected faces. If that is not possible, it falls back to a shell, then to a compound.
  • If the created shape is a solid it can be used as any other solid in other workbenches (i.e. Part or PartDesign workbenches). It has volume and weight.

Eigenschaften

Daten

Debug

  • DatenShowOpenEdges (Bool): If the output shape is not a solid, this will output the open edges.

Solid

  • DatenFaces (LinkSubList): List of faces to build the solid.
  • Daten (ReadOnly)ShapeStatus (String): Status of the created shape.
