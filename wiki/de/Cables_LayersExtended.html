Beschreibung

Die Erweiterten Ebenen sind eine vordefinierte Anzahl von Ebenen, von denen jede eine erweiterte Version einer Draft Ebene ist. Die erweiterte Ebene ermöglicht es zusätzlich zu den Standardeigenschaften einer Draft Ebene, die Farbe und Größe der Punkte der zu dieser Ebene gehörenden Elemente festzulegen. Jede erweiterte Ebene verfügt zudem über die Eigenschaft Auto Member Type, die dazu dient, Elemente der Ebene automatisch zu gruppieren.

Es wurden erweiterte Ebenen erstellt, um die Organisation von WireFlex lines, Befestigungslinie, Befestigungspunkte, Einrastlinien und Anschlüsse zu vereinfachen. Mit diesen Ebenen können nicht gruppierte Objekte eines bestimmten Typs schnell und einfach der entsprechenden Ebene zugewiesen werden, sie können anschließend ein- oder ausgeblendet werden sowie ihre Anzeigeeigenschaften in einem Schritt geändert werden.

Die ausgewählte Ebene einfach neu berechnen (z. B. durch einen Rechtsklick darauf und Auswahl von Objekt neu berechnen), und alle nicht gruppierten Elemente des ausgewählten Typs werden ihr zugewiesen.

Anwendung

  1. Create default set of extended layers by one of these methods:
    • Press the Layers Extended button.
    • Select the Cable Support → Layers Extended option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cable Support → Layers Extended option from the context menu.

Hinweise

  • Die Standardebene mit der Eigenschaft Auto Member Type wird auf None gesetzt und dient als Vorlage für die Erstellung anderer erweiterter Ebenen.
  • Es ist möglich, mehrere Ebenen mit derselben Eigenschaft Auto Member Type zu verwenden, indem man diese manuell ändert.
  • Wenn der Befehl LayersExtended“ erneut ausgeführt wird, werden Ebenen für alle möglichen Auto Member Type-Typen generiert, die noch nicht erstellt wurden, sowie eine zusätzliche Ebene mit dem Auto Member Type-Wert None.

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

Ein Layer Extended-Objekt ist von einem Draft Ebene-Objekt abgeleitet und erbt all seine Eigenschaften. Layer Erweitert hat die folgenden zusätzlichen Eigenschaften:

Daten

Layer

  • Daten-EigenschaftAuto Member Type (Enumeration): Legt den Typ des automatischen Elements dieser Ebene fest. Bei der Neuberechnung der Ebene werden automatisch alle Objekte des ausgewählten Typs einbezogen, die keiner anderen Ebene zugeordnet waren. Mögliche Werte: None, WireFlex, SupportLines, ExtSnapLines, CableTerminal. Wenn None ausgewählt ist, wird der automatische Elementtyp deaktiviert.

Ansicht

Layer

  • Daten-EigenschaftPoint Color (Color): Legt die Punktfarbe der Ebene fest.
  • Daten-EigenschaftPoint Size (Float): Legt die Punktgröße der Ebene fest.
