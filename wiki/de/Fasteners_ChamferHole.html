Beschreibung

Der Befehl Fasteners Lochsenkung erstellt eine Senkung an einem Loch für Senkschrauben.

Löcher mit Senkung für Senkschrauben

Anwendung

  1. Der Befehl kann Schraubendurchmesser automatisch erkennen. Damit dies funktioniert, müssen die Bohrungen den richtigen Durchmesser haben. Beispielsweise erfordert eine M5-Schraube eine Durchgangsbohrung von 5 mm oder eine Gewindebohrung von 4,2 mm. Der Befehl Fasteners Schraubenrechner kann zur Bestimmung der Durchmesser von Anbohrlöchern verwendet werden.
  2. Ein Teil mit kreisförmigen Bohrungen auswählen.
  3. Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
    • Die Schaltfläche Lochsenkung drücken.
    • Die Option Fasteners → Lochsenkung erstellen aus dem Menü wählen.
  4. Das Aufgabenfenster Fasenlöcher für Senkkopfschrauben wird geöffnet.
  5. Die Kanten zum Anfasen-Liste zeigt alle kreisförmigen Kanten des ausgewählten Teils an.
  6. Die Kanten, die man abschrägen möchte, auf eine der folgenden Arten auswählen:
    • In der 3D-Ansicht auf einzelne kreisförmige Kanten oder Flächen mit kreisförmigen Kanten klicken:
      • Die Strg-Taste muss nicht gedrückt gehalten werden.
      • Jede ausgewählte Kante wird in der Liste Kanten zum Abschrägen markiert.
      • Der Schraubendurchmesser für jede Kante wird automatisch erkannt.
      • Kanten können in der 3D-Ansicht nicht abgewählt werden.
    • Wählen Sie Kanten aus oder heben Sie die Auswahl auf, indem Sie sie in der Liste Edges to chamfer aktivieren oder deaktivieren:
      • In diesem Fall wird der Schraubendurchmesser nicht automatisch erkannt.
  7. Optional kann man den Schraubendurchmesser einzelner Kanten in der Liste Kanten zum Abschrägen ändern, indem man auf das Feld Durchmesser doppelklickt und einen neuen Wert aus der angezeigten Dropdown-Liste auswählt.
  8. Optional kann man den Schraubendurchmesser aller ausgewählten Kanten ändern, indem man einen neuen Wert aus der Dropdown-Liste Durchmesser unterhalb der Liste Kanten zum Abschrägen auswählt.
  9. Optional kann man einen Schraubentyp angeben.
  10. Auf die Schaltfläche OK drücken.
  11. Es wird ein Countersunk-Objekt (Senkkopf) mit einer abgeschrägten Aussparung für jede ausgewählte Kante erstellt.
  12. Optional kann man Schrauben anbringen. Siehe Arbeitsbereich Fasteners.

Hinweise

  • Versenkte Objekte sind parametrisch. Ein vorhandenes versenkte Objekt kann durch Doppelklicken in der Baumansicht bearbeitet werden. Das Aufgaben-Fenster Gefaste Löcher für Senkkopfschrauben wird geöffnet. Rundkanten können hinzugefügt oder entfernt werden, und die Parameter der Kanten können geändert werden.
