Beschreibung

Dieser Rechner ermittelt die Mindestgröße von Entlüftungsöffnungen für barometrische Höhenmesser auf der Grundlage des Raumvolumens und der Anzahl der Entlüftungsöffnungen.

Die Berechnung basiert auf der Faustregel, dass pro 100 Kubikzoll Volumen eine Entlüftungsöffnung von 1/4 Zoll erforderlich ist. Eine einzelne Entlüftungsöffnung mit der entsprechenden Größe ist anfällig für Geräusche, die durch Windböen und Windstöße verursacht werden. Diese Effekte werden durch die Verwendung mehrerer Entlüftungsöffnungen minimiert, die über das Volumen verteilt sind, wobei drei Öffnungen als praktisches Minimum angesehen werden. Die Gesamtfläche aller Entlüftungsöffnungen entspricht der einer einzelnen größeren Öffnung.

Anwendung

  1. There are several ways to invoke the command:
    • Press the Vent Hole Size Calculator button.
    • Select the Rocket → Calculators → Vent Hole Size Calculator option from the menu.
  2. Enter the parameters of your electronics bay.

Berechnung

Vent hole size is calculated using the following formula:

Dvent=0.004396DLN

where

Dvent= vent hole size
D= body tube diameter
L= body tube length
N= number of vent holes

Einheiten

Calculations are done using metric units, but will display in your preferred units. Values can also be entered using any supported units by including the units with the value (eg 5 oz, or 23.2 g)
