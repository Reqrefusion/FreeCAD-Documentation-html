(November 2018) Der Arbeitsbereich Reinforcement wird vom Arbeitsbereich Arch getrennt entwickelt. Sein Code und seine API werden unter FreeCAD Bewehrung, getrennt von FreeCADs Haupt-Repository, bereitgestellt.

Siehe auch die Arch API für die Hauptfunktionen des Arbeitsbereichs Arch.

Die Funktionen der API können in Makros und von der Python-Konsole aus verwendet werden, nachdem das entsprechende Modul importiert wurde. 

import StraightRebar
import UShapeRebar
import LShapeRebar
import BentShapeRebar
import Stirrup
import HelicalRebar
