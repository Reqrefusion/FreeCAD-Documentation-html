Beschreibung

Zusätzlicher Arbeitsbereich zum Erstellen verschiedener Arten von Zahnrädern, Evolventenzahnrädern, Evolventenzahnstangen, Zykloidenzahnrädern und Kegelrädern.

FCGear

Installieren

  • git clone https://github.com/looooo/freecad.gears.git
  • Die Datei freecad.gears in /.FreeCAD/Mod verknüpfen oder kopieren

Hinweis:

  • Wenn die ZIP-Datei heruntergeladen wurde, entpackt man sie und
  • kopiert einfach das gesamte oberste Verzeichnis FCGear in /freecad/Mod
    • Unter Windows müssen versteckte Dateien angezeigt werden, da Mod sonst versteckt ist
    • Unter Linux lautet der Mod-Pfad in der Regel usr/lib/freecad/Mod. Man muss .Mod als Administrator öffnen und Zugriffsrechte für alle Dateien von "gear" erteilen. Man kann auch in das lokale Mod kopieren: ~/.FreeCAD/Mod
  • Da FCGear "NUMPY" benötigt, das erst seit FreeCAD 0.15.4671 enthalten ist, läuft es nicht auf älteren Versionen von FreeCAD

Ein Zahnrad erstellen

  • Zum Arbeitsbereich Zahnrad gehen
  • Auf das Zahnrad-Symbol klicken
  • Die Parameter ändern
