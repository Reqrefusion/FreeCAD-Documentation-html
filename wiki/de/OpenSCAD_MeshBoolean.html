Beschreibung

Creates new Mesh object by boolean operation from shapes.

Anwendung

  1. Select shapes in Tree view.
  2. There are several ways to invoke the tool:
    • Press the Mesh Boolean button.
    • Select the OpenSCAD → Mesh Boolean option from the menu.
  3. Select Perform.
  4. A new mesh object is created.

Begrenzungen

OpenSCAD muss unter Bearbeiten → Einstellungen installiert und konfiguriert sein, um die Funktion nutzen zu können
