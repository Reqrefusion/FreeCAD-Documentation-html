Beschreibung

Der Arbeitsbereich CAM enthält eine Reihe versteckter Befehle. Diese sind standardmäßig ausgeblendet, da sie sich noch in der Testphase befinden. Ein Befehl kann aus einem der folgenden Gründe als experimentell betrachtet werden:

  • Er ist unvollständig.
  • Er ist fehlerhaft.
  • Er ist instabil.
  • Er erzeugt keine korrekten, stabilen und sicheren Pfade.
  • Es handelt sich nicht um einen Standardbefehl, der im traditionellen CAM-Arbeitsablauf regelmäßig verwendet wird.
  • Er ist ausgereift, wurde jedoch noch nicht in die Standard-Werkzeugliste aufgenommen.
  • ... andere Gründe.

Experimentelle Befehle aktivieren

Um auf die versteckten experimentellen Befehle des Arbeitsbereich CAM zugreifen zu können, muss der Benutzer diese im Parameter-Editor aktivieren.

  1. Open the Parameter Editor via Tools → Edit Parameters...
  2. Once in the editor the path is BaseApp → Preferences → Mod → CAM
  3. To enable the CAM Area and CAM Area Workplane commands:
    • Right-click in the parameter list area and select New → New Boolean item from the context menu.
    • Name the new parameter: EnableAdvancedOCLFeatures (case-sensitive).
    • Set it to: True.
  4. To enable the other experimental commands:
    • Again select New → New Boolean item from the context menu.
    • Name the new parameter: EnableExperimentalFeatures (case-sensitive).
    • Set it to: True.
  5. Save the settings.
  6. Restart FreeCAD.
