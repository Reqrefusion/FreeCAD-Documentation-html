Beschreibung

Der Befehl Fasteners Stückliste erstellt eine Tabelle mit einer Stückliste für die Verbindungselemente im Dokument.

Ein Modell mit einer Stücklisten-Tabelle für Verbindungselemente.

Anwendung

  1. Es gibt verschiedene Möglichkeiten das Werkzeug aufzurufen:
    • Die Schaltfläche Stückliste erstellen drücken.
    • Den Menüeintrag Fasteners → Stückliste erstellen auswählen.
  2. Eine Stücklisten-Tabelle für Verbindungselemente wird erstellt.

Hinweise

  • Eine Stücklisten-Tabelle für Verbindungselemente wird nicht aktualisiert, wenn Verbindungselemente ausgetauscht, hinzugefügt oder entfernt werden. in solchen Fällen muss der Befehl erneut angewendet werden, um eine neue Tabelle anzulegen.
  • Eine Stücklisten-Tabelle für Verbindungselemente kann mit dem Befehl Spreadsheet Exportieren exportiert werden.
