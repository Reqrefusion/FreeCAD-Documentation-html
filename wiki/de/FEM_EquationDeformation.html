Beschreibung

Diese Gleichung beschreibt die nichtlineare elastische Verformung von Festkörpern.

Informationen zur Mathematik der Gleichung findet man im Elmer-Modellhandbuch im Abschnitt Endliche Elastizität.

Anwendung

  1. Einen Löser Elmer hinzufügen.
  2. Diesen in der Baumansicht auswählen.
  3. Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
    • Die Schaltfläche Verformungsgleichung drücken.
    • Aus dem Menü die Option Lösen → Verformungsgleichung auswählen.
  4. Die Gleichungslöser-Einstellungen oder die allgemeinen Löser-Einstellungen bei Bedarf ändern.

Löser-Einstellungen

Die allgemeinen Löser-Einstellungen findet man unter Elmer-Löser-Einstellungen.

Die Verformungsgleichung bietet diese speziellen Einstellungen:

  • Daten-EigenschaftCalculate Pangle: Wenn die Hauptwinkel berechnet werden sollen.
  • Daten-EigenschaftCalculate Principal: Wenn alle Spannungen berechnet werden sollten.
  • Daten-EigenschaftCalculate Strains: Wenn Dehnungen berechnet werden sollen. Dadurch werden auch die Spannungen berechnet, selbst wenn Daten-EigenschaftCalculate Principal oder Daten-EigenschaftCalculate Stresses auf false gesetzt ist.
  • Daten-EigenschaftCalculate Stresses: Wenn Spannungen berechnet werden sollen. Im Vergleich zu Daten-EigenschaftCalculate Principal werden das Tresca-Fließkriterium und die Hauptspannung nicht berechnet.
  • Daten-EigenschaftInitialize State Variables: Weitere Informationen findet man im Elmer-Handbuch.
  • Daten-EigenschaftMixed Formulation: Weitere Informationen findet man im Elmer-Handbuch.
  • Daten-EigenschaftNeo Hookean Model: Verwendet das neo-Hooke'sche Materialmodell.
  • Daten-EigenschaftVariable: Die Variable für die Verformungsgleichung. Dort die 3 in 2 ändern, wenn man eine 2D-Geometrie hat. In dem Sonderfall, dass man Daten-EigenschaftMixed Formulation und Daten-EigenschaftNeo Hookean Model auf true gesetzt hat, muss die Variablenanzahl den Geometrieabmessungen + 1 entsprechen, sodass für eine 3D-Geometrie die 3 in 4 geändert werden muss.

Gleichung:

  • Daten-EigenschaftPlane Stress: Berechnet die Lösung entsprechend der ebenen Spannungslage. Gilt nur für 2D-Geometrien.

Informationen zum Analyse-Merkmal

Die Verformungsgleichung berücksichtigt die folgenden Analysefunktionen, sofern sie eingestellt sind:

  • Feste Randbedingung
  • Verdrängungsrandbedingung
  • Kraftbelastung
  • Anfängliche Temperaturbedingung
  • Druckbelastung
  • Schwerkraftbelastung
  • Feder

Hinweis

  • Mit Ausnahme von Berechnungen in 2D ist es für alle oben genannten Analysefunktionen wichtig, dass sie auf eine Fläche wirken. Funktionen in 3D, die auf Linien oder Eckpunkte eingestellt sind, werden vom Elmer-Löser nicht erkannt.

Ergebnisse

Die verfügbaren Ergebnisse hängen von der Löser-Einstellung ab. Wenn keine der Daten-EigenschaftCalculate * Einstellungen auf wahr gesetzt wurde, wird nur die Verdrängung berechnet. Andernfalls stehen auch die entsprechenden Ergebnisse zur Verfügung.
