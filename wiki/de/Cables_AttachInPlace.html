Beschreibung

Der Befehl Anbringen erstellt eine Befestigung eines Objekts, ohne dessen globale Position zu verändern. Die Map Mode-Eigenschaft eines angehängten Objekts wird auf Objekt XY gesetzt, die Eigenschaft Attacher Engine wird auf Engine 3D gesetzt. Die Eigenschaft Befestigungs-Versatz wird automatisch berechnet.

Dies ist beispielsweise nützlich, wenn zwei Objekte bereits einen festen Abstand und eine feste Drehung zueinander haben und eines davon an einen neuen Ort verschoben werden muss. Mit diesem Werkzeug kann eine Verbindung erstellt werden, ohne den Abstand und die Drehung zwischen diesen Objekten zu verändern. Sobald die Verbindung hergestellt ist, ändert sich die relative Position des verbundenen Objekts nicht mehr, wenn das Basisobjekt verschoben wird.

In der Praxis kann es z. B. zum schnellen Befestigen von Elektrischen Geräten an Installationsdosen, Installationsdosen zu der Arch Wand, Leitungsverbindung zu einem beliebigen eigenen PCB-Objekt oder zu einem anderen Anschluss genutzt werden.

Es kann auch verwendet werden, um „Haken“ herzustellen, indem man Befestigungspunkte oder Befestigungslinien zu der Arch Wand hinzufügt und Leitungsverläufe auf diesen erstellt und dann Leitungen von ihnen. Dadurch würde schnell ein Kabel entstehen, das an bestimmten Stellen an der Wand befestigt ist.

Anwendung

  1. Select two or more objects in the 3D View or Tree View holding the CTRL key. The last selected object will be the Attachment Support for other selected objects.
  2. Create an Attach In Place attachment by one of these methods:
    • Press the Attach In Place button.
    • Select the Cable Support → Attach In Place option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cable Support → Attach In Place option from the context menu.
    • Use the keyboard shortcut: X then A.

Hinweise

  • Auswahl von Elementen in der 3D-Ansicht wird als Auswahl ganzer Objekte behandelt. Die Befestigung an ein Objekt wird als Befestigung an dessen Placement.Base verstanden.
  • Aufgrund der in manchen Fällen langen Neuberechnungszeit (z. B. wenn viele komplexe Objekte ausgewählt wurden) wird das Dokument nach der Neuberechnung nicht automatisch nach dem Befehl Anbringen neu berechnet. Das Dokument kann manuell neu berechnet werden z. B. mithilfe der Tastenkombination: F5 (oder Strg+R).
