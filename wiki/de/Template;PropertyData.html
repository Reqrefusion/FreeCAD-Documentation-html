Template:PropertyData/de

Daten-Eigenschaft{{{1}}}

Diese Vorlge wird verwendet, um auf eine Daten-Eigenschaft hinzuweisen. Siehe auch Template:PropertyView/de, Template:Properties_Title/de.

Beispiel:

Das Schlüsselwort "Daten-Eigenschaft" wird ausgegeben. 

{{PropertyData/de|Make Face}}: Erstellt eine Fläche.

Daten-EigenschaftMake Face: Erstellt eine Fläche.

Beispiel 2:

Wird ein zweites Argument angegeben, wird damit die Art der Daten ausgegeben. 

{{PropertyData/de|Make Face|Bool}}: Erstellt eine Fläche.

Daten-EigenschaftMake Face (Bool): Erstellt eine Fläche.

Beispiel 3:

Wird ein drittes Argument angegeben, wird damit die Art der Eigenschaft ausgegeben. Eine oder mehrere dieser Arten lassen sich zuordnen: Hidden (ausgeblendet), ReadOnly (schreibgeschützt) and Output (Ausgabe). (alle drei Arten werden mit dem Schlüsselwort (ausgeblendet) gekennzeichnet) 

{{PropertyData/de|Proxy|PythonObject|hidden}}: Eine spezielle Klasse, die diesem Objekt zugeordnet ist.

Daten-Eigenschaft (ausgeblendet)Proxy (PythonObject): Eine spezielle Klasse, die diesem Objekt zugeordnet ist.
