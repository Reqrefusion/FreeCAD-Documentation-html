Diese Modul ist im FreeCAD-Module integriert und enthält Methoden, um Text auf der FreeCAD-Ausgabe-Konsole und Statuszeile auszugeben. Die Ausgaben haben eine unterschiedliche Farbe, abhängig von Meldung, Warnung oder Fehler.

Beispiel: 

import FreeCAD
FreeCAD.Console.PrintMessage("Hello World!\n")


GetStatus("Log" or "Msg" or "Wrn" or "Err")

Description: Ruft den Status für Log, Msg, Wrn oder Error für einen Beobachter ab.

Returns: eine Status-Zeichenkette

PrintError(string)

Description: Gibt eine Fehlermeldung in der Ausgabe aus

Returns: nichts

PrintLog(string)

Description: Gibt eine Protokollmeldung in der Ausgabe aus.

Returns: nichts

PrintMessage(string)

Description: Gibt eine Meldung auf der Ausgabe aus.

Returns: nichts

PrintWarning(string)

Description: Gibt eine Warnung in der Ausgabe aus.

Returns: nichts

SetStatus(string)

Description: Leget die Statistiken für Log, Msg, Wrn oder Error für einen Beobachter fest.

Returns:
