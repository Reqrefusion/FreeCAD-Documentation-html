Beschreibung

Der Befehl Cable Leitungsprofil erstellt eine Skizze, die den grundlegenden Querschnitt (Profil) einer Leitung darstellt. Der Leitungsquerschnitt ist vollständig parametrisch und wird durch Auswahl unterschiedlicher Leitungseigenschaften in einem Dialogfenster erstellt (siehe Anwendung). Die Profilskizze enthält alle Informationen zum Leitungsquerschnitt, die zur Erstellung einer Leitung (Cable-Objekt) benötigt werden.

Es gibt ein paar vordefinierte Arten von Leitungsquerschnitten, z. B. YDYp (Flachleitung) und YDY (Rundleitung). Zusätzlich können eigene Querschnittarten definiert werden (siehe Benutzerdefinierte Leitungsprofile hinzufügen).

Es ist ebenso möglich, den gesamten Leitungsquerschnitt manuell mit dem Sketcher zu zeichnen. Allerdings müssen einige Regeln beachtet werden, damit er erfolgreich als geeigneter Leitungsquerschnitt verwendet werden kann (siehe Leitungsquerschnitt manuell erstellen).

Anwendung

  1. Einen Leitungsquerschnitt (cable profile) mit einer der folgenden Methoden erstellen:
    • Die Schaltfläche Leitungsquerschnitt drücken.
    • Den Menüeintrag Cables → Leitungsquerschnitt auswählen.
    • Ein Rechtsklick in die Baumansicht oder die 3D-Ansicht und die Menüoption Cables → Leitungsquerschnitt im Kontextmenü auswählen.
  2. Das Aufgaben-Fenster Einen neuen Leitungsquerschnitt hinzufügen wird geöffnet.
  3. Wahlweise die Leitungsart auswählen.
  4. Die Anzahl der Adern eingeben.
  5. Den Standard-Leiterquerschnitt in mm² auswählen oder auf custom belassen.
  6. Wahlweise einen Wert unter Individueller Leiterquerschnitt in mm² eingeben, wenn in der vorherigen Option custom ausgewählt wurde.
  7. Die Schaltfläche OK drücken.

Sobald die Querschnittskizze erstellt wurde, befindet sie sich im 3D-Raum an der Position [0,0,0]. Es wird empfohlen, sie nicht zu drehen oder zu verschieben. Sie kann später dazu verwendet werden, eine Leitung zu erstellen, ohne die Position des Leitungsquerschnitts zu verändern. Bei dem Erstellen einer Leitung, wird ein Paar Klone erstellt und beide Klone werden, dem Leitungsverlauf entsprechend, richtig positioniert und ausgerichtet.

Eine Profilskizze kann für viele Leitungen benutzt werden. Es braucht keine weitere Profilskizze definiert zu werden, wenn eine weitere Leitung den gleichen Leitungsquerschnitt hat.

Hinweise

Für einige Profiltypen muss die Anzahl der Leiter > 1 (bspw. YdYp) sein. Bei falscher Auswahl wird ein Fehler angezeigt und das Leitungsprofil wird nicht erstellt.

Benutzerdefinierte Leitungsquerschnitte hinzufügen

Eine zusätzliche CSV-Datei kann durch den Benutzer erstellt werden, die Vorlagen für selbsterstellte Arten von Leitungsquerschnitten enthält. Sie muss profiles.csv heißen und hier gespeichert sein: 

$FREECAD_USER_DIR/Cables/

Das Verzeichnis $FREECAD_USER_DIR kann über die Python-Konsole abgefragt werden: 

FreeCAD.getUserAppDataDir()

Der Inhalt der benutzerdefinierten Datei profiles.csv muss nach den gleichen Regeln wie die Datei profiles.csv im Quellcode des Arbeitsbereichs Cables erstellt werden. Diese Datei kann bearbeitet werden, ohne dass FreeCAD erneut gestartet werden muss. Jede darin gemachte Änderung ist sofort in der Liste sichtbar, wenn ein neues Profil in der GUI erstellt wird.

Um eine eigene Profilart zu erstellen, wird einfach eine neue Linie zur Datei profiles.csv hinzugefügt, die dem im folgenden Beispiel gezeigten Format entspricht:

MyName,F,MyClass,0.5,0.3,0.1

Dabei ist:

  • MyName der Name des selbsterstellten Profils.
  • F die Formart eines Profils (Shape type). Derzeit werden zwei Formen unterstützt: F = Flachkabel und R = Rundkabel.
  • MyClass - Die Spannungsklasse des selbsterstellten Profils; kann ein beliebiger Name sein, der zusätzliche Kabelklassen anzeigt.
  • 0.5 die Manteldicke (Insulation jacket thickness), siehe folgende Abbildung.
  • 0.3 die Dicke der Aderisolierung (Single insulation thickness), siehe folgende Abbildung.
  • 0.1 der Abstand zwischen Aderisolierungen (Insulation distance), siehe folgende Abbildung.

Abmaße, die als Basis für neue Profilarten verwendet werden. 1 - Manteldicke, 2 - (Ader-) Isolierungsdicke, 3 - Isolierungsabstand

Leitungsquerschnitt manuell erstellen

Leitungsquerschnitte können manuell im Arbeitsbereich Sketcher erstellt werden. Die richtige Abfolge beim Zeichnen der geschlossenen Umrisse muss eingehalten werden, damit das Werkzeug Leitung korrekte Leitungen erstellt.

  1. Der zuerst gezeichnete Umriss entspricht der cable jacket (Ummantelung) (w0 im nachfolgenden Beispiel).
  2. Die nachfolgende Reihe von geschlossenen Umrissen muss innerhalb des ersten Umrisses gezeichnet werden, dabei dürfen sich die Umrisse untereinander weder berühren noch überschneiden. Diese entsprechen den insulations (Isolationen) der einzelnen Adern (w1, w2, w3 im nachfolgenden Beispiel).
  3. Die dann folgende Reihe von Umrissen sollte innerhalb der Isolationen gezeichnet werden, wobei jeder Isolationsumriss einen inneren Umriss erhalten muss. Diese entsprechen den conductor wires (Aderumrissen) (w4, w5, w6 im nachfolgenden Beispiel).
  4. Abschließend sollte eine Reihe von Punkten in der Mitte der Aderumrisse positioniert werden (p1, p2, p3 im nachfolgenden Beispiel).
