Beschreibung

Dieses Makro erstellt Ausschnitte für Gehäuse auf sehr praktische Weise

Hier eine Demo mit einem Arduino Uno-Gehäuse

Anwendung

Wähle eine Fläche des Objekts aus, die als Grundlage für den Ausschnitt dient, und das Makro erstellt einen Klon der Außenkontur mit einem Rand von 0,5 mm (im Makro festgelegt).

    FreeCAD.Console.PrintMessage(msg)

    FreeCAD.Console.PrintWarning(msg)

def getNormal(self):
    return self.Object.Shape.Faces[0].normalAt(0,0)

        if len(selF[0].SubElementNames) == 1:
