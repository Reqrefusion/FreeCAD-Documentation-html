Beschreibung

Dieses kleine Makro ist für Benutzer von Windows 10 gedacht. Die Explorer-Schriftarten für die Verwendung von ShapeString sind leer, und dieses kleine Makro kann helfen, die zu verwendende Schriftart leicht zu erkennen.

Makro Win10-Schriften PYMP

Anwendungen

Das Makro starten, die Schriftart auswählen und sie anwenden.

Makro Win10-Schriften PYMP

  • Text edit : Textfenster
  • Line edit : zur Visualisierung des Pfades und der verwendeten Datei
  • Combobox : Die Schriftart wählen
  • Color : Die Textfarbe wählen
  • 10.00  : Größe des Textes in der 3D-Ansicht
  • Text : Einen Text mit der Textform erstellen
  • 20.00  : Größe des Texts im Makro-Fenster
  • Exit : Fonts Win10 PYMP beenden
  • Ok : Text überprüfen
  • New folder : andere Auswahlpfad-Dateien Schriftart
  • Origin : Wiederherstellung der ursprünglichen Verzeichnisschriftart (System)
  • Help : Zeige die Wiki-Seite im FreeCAD-Browser an

Konfiguration

Nach dem ersten Start siehe:

Menu → Werkzeuge → Benutzer-Parameter : BaseApp/Preferences/Macros/FCMmacros/Fonts Win10 PYMP

  • switchModeTextList
    • false Der normale (und schwarze) Textmodus schaltet die switchFontComBox aus
    • true Erlaubt die switchFontComBox
  • switchFontComBox
    • false (und switchModeTextList= true) schnellerer Textmodus (in Farbe) in der ComboBox-Liste
    • true (und switchModeTextList= true) Die Liste in der ComboBox wird in fontFamily angezeigt, langsamer, aber schöner!
  • setSystemFonts
    • false matplotlib.font_manager.findSystemFonts("C:/", "ttf")
      • Alle Schriftarten (in allen Ordnern und Unterordnern des DD) bearbeiten!
    • true fontman.findSystemFonts(self.pathFont)
      • alle Schriftarten im Verzeichnis (und in allen Unterordnern)
  • seTtextAlignement
    • 0 = AlignLeft (Standard)
    • 1 = AlignCenter
    • 2 = AlignRight
  • setFontByDefault
    • Die zuletzt verwendete Schriftart wird in diesem Feld gespeichert
  • setLastFontPath
    • Der zuletzt verwendete Pfad wird in diesem Feld gespeichert

def createSpace(texte):    # detecte majuscule et ajoute un espace devant la lettre
    # return createSpace(TexTe) = Tex Te , if createSpace(TEXTE) = TEXTE
    if texte.isupper():
        stringSpace = texte
    else:
        try:
            stringSpace = texte[0]
            for i in texte[1:]:
                if re.search(r"[A-Z]", i): i = " " +  i
                stringSpace += i
        except Exception:
            stringSpace = texte
    return stringSpace

def family(chaine):
    # return family(chaine)[1] = Family , family(chaine)[2] = typeCar (form [a, b, c ...]
    famille = typeCar = ""
    try:
        if chaine.find('-') != -1:
            famille = chaine[:chaine.find('-')]
            typeCar = chaine[chaine.find('-')+1:]
        else:
            famille = chaine
            typeCar = ""
    except Exception:
        famille = chaine
        typeCar = ""
    typeCar = str(createSpace(typeCar)).split()
    return [createSpace(famille), typeCar]
