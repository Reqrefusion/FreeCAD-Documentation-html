Einleitung

Der Arbeitsbereich MOOC ist ein Externer Arbeitsbereich mit dem man interaktive Tutorials absolvieren und die Arbeit direkt in der FreeCAD-Oberfläche evaluieren kann. Der Arbeitsbereich MOOC bietet zwei Werkzeuge: interaktive Tutorials und Evaluierungen.

  • Derzeit nur auf Französisch (und fest codiert).
  • Nur kompatibel mit FreeCAD Py3 und Qt5 (PySide2)
  • LGPLv2 (oder ähnlicher) Code, finanziert von Europa über IMT und EESAB.
  • Modular: Dieser Arbeitsbereich wurde mit der Absicht erstellt, dass das Hinzufügen von Tutorials und Evaluierungen modular erfolgen kann. Mit anderen Worten: Man muss ein Tutorial im Ordner lessons oder eine Evaluation im Ordner exercises hinzufügen, damit es im jeweiligen Werkzeug angezeigt wird.

Interaktiv

Interaktive Tutorials (AKA -Abspieler) sind Schritt-für-Schritt-Übungen mit objektiven Kontrollen. Man startet direkt in FreeCAD und ermöglichen es, Schritt für Schritt bei der Modellierung eines Objekts voranzukommen. Der Benutzer verfügt über einen Text, ein Video und vor allem eine Kontrolle, ob die Ziele erreicht wurden.

Evaluations

Evaluationen

Evaluationen (AKA Benotung) bestehen aus einem kleinen Programm, das bestimmte Kriterien eines FreeCAD-Dokuments überprüft, beispielsweise das Vorhandensein eines Part Körpers, einer Skizze oder des endgültigen Volumens.

Installation

Dieser Arbeitsbereich kann über den Addon-Manager installiert werden. Für die manuelle Installation siehe Weitere Arbeitsbereiche installieren.

Einschränkungen

Momentan ist dieser Arbeitsbereich nur auf Französisch verfügbar.

Technische Details

Aus technischer Sicht besteht der Arbeitsbereich aus:

  • einer "API", die den Code zur Analyse des Dokuments enthält (MoocChecker.py)
  • dem Code, der die Tutorials im Ordner "lessons" ausführt (MoocPlayer.py)
  • dem Code, der die Evaluationen im Ordner "exercises" ausführt (MoocGrader.py)
