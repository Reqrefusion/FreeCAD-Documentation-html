Einleitung

Der Arbeitsbereich Quetzal ist ein externer Arbeitsbereich für die Planung von Rohrleitungssystemen. Er ist eine Abspaltung der Arbeitsbereiche Dodo und Flamingo die ursprünglich für den Hochbau und die Rohrleitungsplanung entwickelt wurden.

Werkzeuge

Die Werkzeuge sind in drei Werkzeugleisten und Menüs organisiert:

Rohr-Werkzeuge

Rohr-Werkzeug-Symbolleiste

  • Rohr einfügen: Ruft das Formular Rohr einfügen auf, um einen Rohrabschnitt mit ausgewählter Nennweite, Nennweite und Länge einzufügen.
  • Bogen einfügen: Lädt das Formular Bogen einfügen, um einen Bogen mit der ausgewählten Nennweite und dem ausgewählten Nenn-Durchmesser einzufügen. Über optionale Eingabefelder können der Standard-Biegewinkel und der Standard-Biegeradius für stumpfgeschweißte Formstücke überschrieben werden. Mit einem Drehregler lässt sich das Formstück nach dem Einfügen drehen.
  • T-Stück einfügen: Lädt das Formular T-Stück einfügen, um ein T-Stück mit ausgewählter Nennweite/Nennleistung, Nennweite der Hauptleitung und Nennweite des Abzweigs einzufügen. Über Optionsfelder kann das T-Stück entweder in die Hauptleitung (gerader Abschnitt) oder in den Abzweig eingefügt werden.
  • Verschlusskappe einfügen: Lädt das Formular Verschlusskappe einfügen, um eine Verschlusskappe mit dem ausgewählten Profil/der ausgewählten Nennweite und dem ausgewählten Nenndurchmesser einzufügen.
  • Kupplung/Verschraubung einfügen: Ruft das Formular Kupplung/Verschraubung einfügen auf, um eine Kupplung oder Verschraubung mit Stumpfschweiß- oder Gewindeanschluss in der ausgewählten Nennleistung einzufügen. Kupplungen können an jedem Ende unterschiedliche Nennweiten aufweisen.
  • Ventil einfügen: Ruft das Formular Ventil einfügen auf, um ein Ventil einzufügen. Derzeit werden drei Typen unterstützt: Flanschventil (wird als Kugelhahn mit Drehzapfen dargestellt und kann mit einem Griff oder einem Getriebe dargestellt werden), Gewinde-/Steckschweißventil (wird als typisches Sechskantventil mit Griff dargestellt) und „Generisch“, das mit einer für Prozessflussdiagramme typischen Doppelkegelform dargestellt wird.
  • Flansch einfügen: Lädt das Formular Flansch einfügen, um einen Flansch in der ausgewählten Größe einzufügen. Bei Schweißhalsflanschen kann auch die Bohrungsgröße ausgewählt werden. Über Optionsfelder kann das Ventil entweder an seinem Schweißende oder an seiner Flanschfläche (zur Befestigung an einem anderen Flansch) eingefügt werden.
  • Dichtungen/Flanschschrauben einfügen: Ruft das Formular Dichtungen einfügen auf, um Dichtungen und die entsprechenden Flanschschrauben einer ausgewählten Druckklasse und Größe einzufügen.
  • Abzweig einfügen: Fügt ein Abzweigstück mit Gewinde-/Muffenschweißanschluss oder Stumpfschweißanschluss in einer ausgewählten Größe und Nennweite ein. Die Stutzen können in einem Winkel von 90 Grad oder 45 Grad zum Trägerrohr oder zum Stutzen eingefügt werden und lassen sich um die eigene Achse drehen sowie entlang der axialen Länge des einzufügenden Rohrs verschieben, um an einer beliebigen Stelle des ausgewählten Rohrs positioniert zu werden.
  • U-Bügel einsetzen: Zum Einsetzen eines U-Bügel, um ein Rohr an einer Rohrhalterung zu befestigen. Der Nenndurchmesser des Rohrs kann ausgewählt werden.
  • PypeLine Manager: Loads the PypeLine Manager form that allows the insertion of pipe segments following a draft wire path. Bends are inserted as short radius (1D bend radius) butt weld elbows.
  • Abzweig einfügen: Ruft das Formular Abzweig einfügen auf, mit dem Rohrabzweige eingefügt werden (dargestellt als an einem Hauptrohr befestigte Rohrstücke mit Fischmaul-Schnitt). Der Verlauf des Abzweigs folgt einer Entwurfslinie.
  • Tank einfügen: Fügt einen Tank ein, der als rechteckiger Kasten mit den angegebenen Abmessungen dargestellt wird.
  • Insert a Terminal Adapter: inserts a terminal adapter (such as a PVC thread-socket adapter) of selected size and rating.
  • Insert a Pipe Route: (need to fix)
  • Break the Pipe: Break a pipe component into two segments with an optional gap between segments.
  • Mate pipe edges: Click two component edges to align centers. The first object clicked is stationary and the second object moves.
  • Fit One Elbow: Select two intersecting pipes and one elbow to fit the elbow between the two pipes. Only works with pype-objects.
  • Extend the pipe: By selecting two pipes, this command extend them both to their intersection point, if exists.
  • Extend the pipes: By selecting two pipes, this command extend the first to the intersection with the other, if exists.
  • Connect to Header
  • Lay Down the Pipe: With one pipe and one beam object selected, moves the selected pipe to match the face of a beam object.
  • Raise Up the Support: Similar to the Lay Down the Pipe tool, except moves the beam with a stationary pipe object.
  • Draw Tube Point by Point: Combines drafting a pipe path with the Pypeline Manager interface in one step.
  • Insert Any Shape: This is a tool to create a "pype" object from a .STEP or .IGES or .BREP file. It loads the imported file into the Shape property of a FeaturePython. Shape files should be added to the "shapez" folder in the Quetzal workbench directory.
