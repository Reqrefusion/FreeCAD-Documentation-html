Beschreibung

Der Befehl Leitungsverlauf von Kabelanschluss lösen löst vorher befestigte Knoten von Leitungsverläufen von bestehenden Kabelanschlüssen. Befestigungen von Knoten werden aus den besonderen Eigenschaften von Leitungsverlauf entfernt, für Details siehe Leitungsverlauf Eigenschaften.

Durch das Lösen werden die bestehenden Anschluss- oder Leitungsverlaufs-Positionen nicht verändert. Die elektrische Verbindung zwischen Kabelanschluss und Leitungsverlauf besteht nicht mehr. Die Kabelanschluss-Eigenschaften aus der Netz-Gruppe werden entsprechend aktualisiert.

Wenn das Kabelanschluss anschließend verschoben wird, bleiben die abgetrennten Leitungsverlaufs-Knotenpunkte an ihrer Position. Wenn der Leitungsverlauf mit Cables Bearbeiten bearbeitet wird, können die abgetrennten Knotenpunkte frei verschoben werden.


Anwendung

  1. Select WireFlex object in the 3D View or in the Tree View.
  2. Select CableTerminal in the 3D View or in the Tree View holding the CTRL.
  3. Run an Detach Wire From Terminal command by one of these methods:
    • Press the Detach Wire From Terminal button.
    • Select the Cable Wires → Detach Wire From Terminal option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cable Wires → Detach Wire From Terminal option from the context menu.
    • Use the keyboard shortcut: T then R.

Hinweise

  • Nach dem Abklemmen hat der Kabelanschluss freien Platz, um ein weitere Leiter anzuschließen.
  • Siehe auch Leitungsverlauf Beschreibung und Leitungsverlauf Anwendung für die Details zur Verwendung von Leitungsverläufen.
