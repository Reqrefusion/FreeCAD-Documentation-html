Beschreibung

Das Werkzeug Curves KurveTeilen teilt ausgewählte Kanten.

Anwendung

  1. Zum Arbeitsbereich Curves wechseln (muss mit dem Addon-Manager installiert werden, wenn noch nicht geschehen)
  2. Es gibt mehrere Möglichkeiten den Befehl aufzurufen:
    • Die Schaltfläche drücken.
    • Den Menüeintrag Curves → Split curve auswählen

Hinweise

  • This tool also allows the selection of an object containing a wire in the Tree View as an input edge. If it contains more than one edge, all but the first one are ignored.
  • Splitting methods that independently add splitting locations to the final result:
    1. By Number: Set the DatenNumber property to a value greater than 1 to receive segments of equal lengths.
    2. Single distance: Set the DatenDistance property to a value other than 0.0 mm.
    3. Several distances: Add distance values to the DatenValues property, either absolute (mm) or relative (%).
    4. Cutting objects: Add cutting objects to the DatenCutting Objects property,
      • Intersection points: Set the DatenCut At Vertexes property to false to split at intersection points.
      • Shortest distance: As before, to split at the shortest distance when a cutting object is neither intersecting nor touching.
      • Projected points: Set the DatenCut At Vertexes property to true to split where the vertices of the cutting object are projected onto the SplitCurve object.
  • The values of the DatenValues property can also be changed by dragging the vertices in the 3D view when the SplitCurve object is in edit mode (double-click on the SplitCurve object in the Tree view to toggle).

Eigenschaften

Siehe auch: Eigenschafteneditor.

Ein SplitCurve-Objekt wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem besitzt es die folgenden zusätzlichen Eigenschaften:

Daten

Base

  • Daten-EigenschaftSource (LinkSub): Zu teilende Kante.

Output

  • Daten (Hidden)Normalized Parameters (FloatList): Normalized parameters list.

Split

  • DatenCut At Vertexes (Bool): Create a split point at nearest of each vertex of the cutting objects.
  • DatenCutting Objects (LinkList): List of objects that cut the curve.
  • DatenDistance (Distance): Expression-ready distance value.
  • DatenKeep Solid (Bool): Rebuild and output the complete shape.
  • DatenNumber (Integer): Number of equal segments.
  • DatenValues (StringList): List of splitting locations.
    % and units are allowed.
    Negative values are computed from edge end.
