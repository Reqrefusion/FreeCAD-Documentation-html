Beschreibung

Der Befehl Fasteners 3dDruckeinstellungen kann verwendet werden, um die 3D-Druckeinstellungen zu ändern. Diese Einstellungen dienen dazu, Schrumpfung und andere Probleme auszugleichen, die sich auf 3D-gedruckte Befestigungselemente auswirken können.

Die Einstellungen findet man auch in den Voreinstellungen: im Menü Bearbeiten → Voreinstellungen und dann Fasteners.

Anwendung

  1. There are several ways to invoke the command:
    • Press the 3D Print Settings button.
    • Select the Fasteners → 3D Print Settings option from the menu.
  2. The 3D print settings task panel open.
  3. Optionally change the Thread generation to 3D printer compatible.
  4. Optionally change the Screw Scaling.
  5. Optionally change the Nut Scaling.
  6. Optionally press the Save and Update button to save the settings and update the threaded fasteners in the current document.
  7. Press the OK button to save the settings and close the task panel.
