Beschreibung

Makro MeshResetOrigin ist ein Makro, das Mesh-Transformationen konsolidiert, indem es den Ursprung jedes ausgewählten Meshes auf den Dokument- (Welt-)Ursprung zurücksetzt, ohne die Geometrie zu verschieben. Diese Funktion ist für die Verwendung nach einem Ausrichtungsvorgang vorgesehen (z. B. „Bearbeiten → Ausrichten auf…“ bei Mesh-Objekten), um die resultierende Ausrichtung in die Mesh-Koordinaten einzubacken, anstatt sie als Platzierungs-Versatz beizubehalten oder auf einen übergeordneten Container zurückzugreifen.

Jedes ausgewählte Mesh::Feature wird wie folgt verarbeitet:

  1. Read its current Placement (the transform that positions it in the world).
  2. Apply that placement matrix to a copy of the mesh, so every vertex moves into the position it currently occupies in world space.
  3. Reassign the transformed mesh and reset the Placement property to identity (0,0,0 / 0°).

Das Netz bewegt sich auf dem Bildschirm nicht; lediglich sein Ursprung stimmt nun mit dem Weltursprung überein, da die Transformation nun in der Netz-Geometrie statt in der Eigenschaft Placement gespeichert ist. Der gesamte Vorgang ist in einer einzigen Rückgängig-Transaktion zusammengefasst, sodass ein einziger Klick auf Undo (Rückgängig) alles rückgängig macht.

Objekte, die keine Netze sind, sowie Netze, die sich in einem übergeordneten Container befinden (z. B. App::Part), werden übersprungen, wobei eine Warnung im Ausgabefenster ausgegeben wird. Netze, die sich bereits am Ursprung befinden, bleiben unberührt.

Ort

Arbeitsbereich Mesh (extern/optionales Makro).

Anwendung

  1. Select one or more Mesh objects in the tree view or the 3D view.
  2. Run the macro.
  3. Each mesh's origin is reset to the world origin; the geometry does not move.
  4. Check the Report view for a per-mesh summary (reset / already at origin / skipped).

Befindet sich ein Mesh innerhalb eines App::Part-Objekts, zieht man es zunächst aus dem Container heraus und führt dann das Makro aus.

Beispiel

Ein typischer Arbeitsablauf besteht darin, ein 3D-Scan-Netz an den Weltkoordinatenachsen auszurichten, um die Bearbeitung zu vereinfachen:

  1. Import a 3D scan mesh.
  2. Create (or reuse) a reference geometry to align the mesh on, for example a plane.
  3. Select both the reference object and the mesh, then go to Edit → Align To....
  4. Pick at least 3 points on the face of the mesh to flatten, then at least 3 points on the reference face to align with, then click Align.
  5. The mesh is now aligned, but the Placement/transform is no longer trivial to work with.
  6. Run MeshResetOrigin on the mesh: it does not move, but its origin is now aligned with the world.
  7. Iterate as needed to refine the alignment — particularly useful when working with 3D-scan meshes or 3D-printer STL files.

Eine illustrierte Schritt-für-Schritt-Anleitung ist im GitHub-Repository zu finden.

Hinweise

  • Benötigt FreeCAD 1.1.1 oder neuer (nicht mit früheren Versionen getestet).
  • Netze mit einem übergeordneten App::Part-Container werden übersprungen; diese zuerst aus dem Container entfernen.
