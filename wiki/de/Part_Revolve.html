Beschreibung

Der Befehl Part Drehen dreht das ausgewählte Objekt um eine gegebene Achse. Die folgenden Formtypen sind erlaubt und führen zu den angegebenen Ausgabeformen:

Eingabeform Ausgabeform
Knoten Kante
Kante Fläche
Draht Hülle
Fläche Volumenkörper
Hülle Volumenkörperverbund (Compsolid)

Eine Skizze kann auch verwendet werden. Volumenkörper oder Verbundkörper sind nicht als Eingabeformen zulässig. Normale Verbünde sind derzeit ebenfalls nicht zulässig.

Beispiel einer gedrehten Form

Anwendung

  1. Wahlweise eine oder mehrere Formen in der 3D-Ansicht oder in der Baumansicht auswählen.
  2. Es gibt mehrere Möglichkeiten den Befehl aufzurufen:
    • Die Schaltfläche Drehen drücken.
    • Den Menüeintrag Part → Drehen auswählen.
  3. Das Aufgaben-Fenster Drehen wird geöffnet.
    • Wahlweise ein Element in der Form-Liste anklicken, um eine (andere) Form auszuwählen.
    • Wahlweise die Umschalttaste (Shift) gedrückt halten und ein Element in der Form-Liste anklicken, um die Form zur Auswahl hinzuzufügen oder sie aus der Auswahl zu entfernen..
    • Drehachse und Drehwinkel einstellen und wahlweise weitere Parameter (siehe Abschnitt Eigenschaften für weitere Einzelheiten).
  4. OK drücken, um das Aufgaben-Fenster zu schließen.
  5. Für jede ausgewählte Form wird ein Revolve-Objekt erstellt.

Jede Eingabeform wird unterhalb ihres Revolve-Objekts abgelegt.

Aufgaben-Fenster

  • Die Schaltfläche OK erzeugt die gedrehte Form und schließt das Fenster.
  • Die Schaltfläche Cancel schließt das Fenster, ohne etwas zu tun.
  • Form-Liste: Hier wird ausgewählt, welche Formen gedreht werden sollen. Wenn mehrere Objekte ausgewählt sind, werden mehrere Drehobjekte erstellt.
  • Abschnitt Drehachse: Legt die Drehachse fest.
    • Mittelpunkt X/Y/Z-Felder: Legt den Basispunkt der Drehachse fest.
    • X-/Y-/Z-Richtung-Felder: Legt die Richtung der Drehachse fest. Auf eine Schaltfläche klicken, um die Richtung für die entsprechende Achse festzulegen.
    • Schaltfläche Referenz auswählen: Darauf klicken und dann eine Kante in der 3D-Ansicht auswählen. Diese Kante erscheint im Textfeld unterhalb der Schaltfläche im Format „Objektname:KanteN“. Die Verknüpfung kann auch manuell eingeben oder gelöscht werden. Die Werte für die X-/Y-/Z-Richtung werden entsprechend der Kantenrichtung ausgefüllt.
  • Winkel-Feld: Legt den Winkel der Drehung in Grad fest.
  • Symmetrischer Winkel: Wenn diese Option aktiviert ist, wird die Drehung um die Hälfte des angegebenen Winkels nach vorne und nach hinten erweitert.
  • Kontrollkästchen Festkörper erstellen: Wenn diese Option aktiviert ist, wird durch das Drehen eines geschlossenen Linienzugs oder einer geschlossenen Kante ein Volumenkörper erzeugt. Diese Option ist standardmäßig aktiviert, wenn vor dem Aufrufen von „Part Drehen“ ein geschlossener Linienzug ausgewählt wurde.

Hinweise

  • Der Parameter Angle (Winkel) gibt an, wie weit das Objekt gedreht wird. Die Koordinaten bewegen den Ursprung der Drehachse relativ zum Ursprung des Koordinatensystems.
  • Wird eine benutzerdefinierte Achse ausgewählt, legen die Zahlen die Richtung der Drehachse mit Bezug zum Koordinatensystem fest: Ist die Z-Koordinate 0 und die Y- und X-Koordinaten sind nicht Null, wird die Z-Achse in der XY-Ebene liegen. Der Tangens ihres Winkels ist das Verhältnis der gegebene X- und Y-Koordinaten.
  • App Link-Objekte, die mit geeigneten Objektarten verknüpft sind, können auch als Formen und zum Festlegen der Achse verwendet werden.
  • Wenn das zu drehende Objekt die Drehachse schneidet, wird die Ausführung in den meisten Fällen scheitern.

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

A Part Revolve object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Daten

Extrude

  • DatenSource (Link): The shape to revolve.
  • DatenBase (Vector): The base point of the revolution axis.
  • DatenAxis (Vector): The direction of the revolution axis.
  • DatenAxis Link (LinkSub): Link to the edge to use as the revolution axis. Overrides DatenBase and DatenAxis if set.
  • DatenAngle (Angle): The angle span of the revolution. If zero and an arc is used for the DatenAxis Lik, the arc's span will be used.
  • DatenSymmetric (Bool): If true, the revolution is extended symmetrically from the profile.
  • DatenSolid (Bool): If true, revolving a closed edge or a closed wire will yield a solid. If False, a shell will result.
  • DatenFace Maker Class (String): If DatenSolid is true, the facemaker class to use when converting wires to faces, otherwise ignored.
