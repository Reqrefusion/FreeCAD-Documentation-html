Einleitung

Std Basis ist nicht wirklich ein Arbeitsbereich, sondern eher eine Kategorie von 'Standard'-Befehlen und Werkzeugen, die in allen Arbeitsbereichen verwendet werden können.

Werkzeuge

Die meisten Std Basis-Werkzeuge können aus dem Standardmenü erreicht werden. Diejenigen, die nur über eine Symbolleiste oder ein Kontextmenü zur Verfügung stehen, werden unten unter Symbolleiste Struktur und Zusätzliche Werkzeuge aufgelistet.

Standardmenü

Das Standardmenü setzt sich aus 7 Untermenüs zusammen. Jedes Untermenü hat eine eigene Seite. Einfach auf einen der untenstehenden Namen klicken.

Symbolleiste Struktur

  • Neues Part-Objekt: Erstellt ein neues Part-Objekt und macht es aktiv.
  • Bezugselemente erstellen: eingeführt in 1.1
  • Bezugssystem erstellen: Erstellt ein Koordinatensystem (coordinate system object), das an andere Objekte befestigt werden kann. eingeführt in 1.1
  • Bezugsebene erstellen: Erstellt eine Bezugsebene (datum plane object), die an andere Objekte befestigt werden kann. eingeführt in 1.1
  • Bezugselinie erstellen: Erstellt eine Bezugselinie (datum line object), die an andere Objekte befestigt werden kann. eingeführt in 1.1
  • Bezugspunkt erstellen: Erstellt einen Bezugspunkt (datum point object), der an andere Objekte befestigt werden kann. eingeführt in 1.1
  • Neue Gruppe: Erstellt eine neue Gruppe.
  • Verknüpfungswerkzeuge:
  • Verknüpfung erstellen: Erzeugt einen Link.
  • Unterverknüpfung erstellen: Erstellt eine Verknüpfung von Unterobjekten oder Unterelementen.
  • Durch Verknüpfung ersetzen: Ersetzt das/die Objekt(e) durch die neue(n) Verknüpfung(en).
  • Verknüpfung auflösen: Ersetzt den/die Link(s) durch das/die verknüpfte(n) Objekt(e).
  • Verknüpfungen importieren: Importiert den/die ausgewählten externen Link(s).
  • Alle Verknüpfungen importieren: Importiert alle externen Link(s).
  • Variablensatz: Erstellt einen Satz von Eigenschaften, die als Variablen in Ausdrücken verwendet werden können. eingeführt in 1.0

Zusätzliche Werkzeuge

  • Verknüpfungsgruppe erstellen: Erzeugt eine Gruppe von Links.
  • Ausdruck-Aktionen:
  • Ausgewählte kopieren: Kopiert die Ausdrucksdaten der ausgewählten Objekte in die Zwischenablage.
  • Aktives Dokument kopieren: Kopiert die Ausdrucksdaten des aktiven Dokuments in die Zwischenablage.
  • Alle Dokumente kopieren: Kopiert die Ausdrucksdaten aus allen Dokumenten in die Zwischenablage.
  • Einfügen: Fügt Ausdrucksdaten aus der Zwischenablage ein.
  • Auswahlfilter: eingeführt in 1.0
  • Knotenauswahl: Erlaubt nur die Auswahl von Eckpunkten.
  • Kantenauswahl: Erlaubt nur die Auswahl von Kanten.
  • Flächenauswahl: Erlaubt nur die Auswahl von Flächen.
  • Alle Auswahlfilter gelöscht: Erlaubt die Auswahl aller Unterelemente.
  • Alle Instanzen auswählen: Wählt alle Instanzen eines Objekts in der Baumansicht aus.
  • Einfrieren umschalten: Schaltet den Einfrierzustand von Objekten um. eingeführt in 1.0

Veraltete Werkzeuge

  • Std FreeCADFAQ: Öffnet die FreeCAD-FAQ-Wiki-Seite. Nicht verfügbar in 1.1 und neuer.
  • Std DokumentationSkripterstellungMitPython: Öffent die FreeCAD-Wiki-Seite mit Dokumentation für Power-User. Nicht verfügbar in 1.1 und neuer.
  • Std OnlineHilfe: Öffnet die Hilfe-Dokumentation. Nicht verfügbar in 1.1 and above.
  • Entfernung messen: Erzeugt ein Objekt zum Messen und Anzeigen einer Entfernung. Nicht verfügbar in 1.0 und neuer. Stattdessen kann Std Messen verwendet werden.
  • Messung löschen: Löscht Part-Messungen. Nicht verfügbar in 1.0 und neuer. Stattdessen kann Std Messen verwendet werden.
  • Messung umschalten: Schaltet die Anzeige der Part-Messungen um. Nicht verfügbar in 1.0 und neuer. Stattdessen kann Std Messen verwendet werden.
