SheetMetal
Standardtastenkürzel

Beschreibung

Der Befehl Kante ansetzen erzeugt Kanten (Falze) an ausgewählten Flächenrändern. Durch das Ändern der Daten-Eigenschaftangle kann aus einer Kante ein Falz werden.

Eine Kante (auch stehender Falz / Stehfalz genannt) besteht üblicherweise aus einem zylindrischen 90° Bogen und einem ebenen Streifen.

Zwei gewählte Kanten (edges) → zwei Kanten (flanges)

Dies wird als Falz' bezeichnet, wenn der Biegewinkel ungefähr 180° beträgt.

Zwei gewählte Kanten (edges) → zwei Falze mit Gehrung (hems)

Anwendung

  1. Eine oder mehrere Randflächen einer Festkörperform auswählen.
  2. Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
    • Die Schaltfläche Kante ansetzen drücken.
    • Den Menüeintrag Sheet Metal → Kante ansetzen auswählen.
    • Ein Rechtsklick in die Baumansicht oder die 3D-Ansicht und die Menüoption Sheet Metal → Kante ansetzen im Kontextmenü auswählen.
    • Das Tastaturkürzel: W.
  3. Ein Bend-Objekt wird erstellt, das eine neue Kante (Streifen) an jeder ausgewählten Kante (Flächenrand) enthält, und das Aufgaben-Fenster Kanteneigenschaften wird geöffnet.
    • Mit ausgewähltem Reiter Allgemein
      • Wahlweise die Schaltfläche Auswählen drücken, um weitere Kanten hinzuzufügen.
        • Wahlweise die Schaltfläche Auswahl leeren drücken, um die Liste der ausgewählten Kanten zu löschen (eingeführt in 0.7.11 ).
        • Die Schaltfläche Vorschau drücken, um die Auswahl abzuschließen und die Äderungen anzuzeigen.
      • Wahlweise die Parameter der Biegung in diesem Reiter anpassen. Siehe Abschnitt Allgemein der Beschreibung des Ausgaben-Fensters weiter unten.
    • Wahlweise zu einem anderen Reiter wechseln, um weitere Parameter einzustellen. Siehe die Abschnitte Abstände, Gehrung oder Perforation der Beschreibung des Ausgaben-Fensters weiter unten.
  4. Wahlweise einen anderen Reiter auswählen und weitere Parameter im Aufgaben-Fenster anpassen.
  5. Die Schaltfläche OK drücken, um den Befehl abzuschließen und das Aufgaben-Fenster zu schließen.
  6. Wahlweise die Parameter in der Eigenschaften-Ansicht anpassen.

Aufgaben-Fenster

Ein Aufgaben-Fenster mit vier Registerkarten wurde in Version 0.6.00 eingeführt.

Ein vorhandenes Bend-Objekt in der Baumansicht doppelt anklicken, um das Aufgaben-Fenster erneut zu öffnen und die Parameter zu bearbeiten.

Allgemein

  • Auswählen: Ändert die Anzahl der Kanten in der Eigenschaft base Object.
  • Länge: Eingabe für die Eigenschaft length.
  • Kante umkehren: Schaltet die Eigenschaft invert um.
  • Längenangabe: Eingabe für die Eigenschaft Length Spec.
  • Biegeradius: Eingabe für die Eigenschaft radius.
  • Biegewinkel: Eingabe für die Eigenschaft angle.
  • Flächenreferenz: Schaltet die Option Bend angle auf schreibgeschützt um und zeigt zwei weitere Optionen an:
    • Referenz: Erwartet die Auswahl einer Fläche in der 3D-Ansicht und verlinkt diese in die Eigenschaft Angle Face Reference.
    • Relativer Winkel: Eingabe für die Eigenschaft Relative Angle To Ref.
  • Kantenposition: Eingabe für die Eigenschaft Bend Type.
    Ist Abstand eingestellt, stehen zwei weitere Optionen zur Verfügung:
    • Abstand: Eingabe für die Eigenschaft offset.
    • Flächenreferenz: Schaltet die Option Abstand auf schreibgeschützt um und zeigt zwei weitere Optionen an:
      • Referenz: Erwartet die Auswahl einer Fläche in der 3D-Ansicht und verlinkt diese in die Eigenschaft Offset Face Reference.
      • Versetzte Position: Eingabe für die Eigenschaft Offset Type.
        Ist Abstand eingestellt, steht eine weitere Option zur Verfügung:If set to Offset another option is available:
        • Abstand von der Referenz: Eingabe für die Eigenschaft Offset Type Offset.
  • Unfold: Schaltet die Eigenschaft unfold um.

Versätze

  • Gap A: Setzt die gap1-Eigenschaft.
  • Gap B: Setzt die gap2-Eigenschaft.
  • Extend A: Setzt die extend1-Eigenschaft.
  • Extend B: Schaltet die extend2-Eigenschaft um.
  • Rectangle und Round Optionsfelder: relief Type-Eigenschaft umschalten.
  • Width: Setzt die reliefw-Eigenschaft.
  • Depth: Schaltet die reliefd-Eigenschaft um.

Gehrung

  • Automatische Gehrung: Schaltet die Eigenschaft Auto Miter um.
    Wenn aktiviert:
    • Mindestspalt: Eingabe für die Eigenschaft minGap.
    • Maximale Verlängerung: Eingabe für die Eigenschaft max Extend Dist.
    Wenn deaktiviert:
    • Winkel 1: Eingabe für die Eigenschaft miterangle1.
    • Winkel 2: Eingabe für die Eigenschaft miterangle2.

Perforation

  • Perforieren: Schaltet die Eigenschaft Perforate um.
    Wenn aktiviert:
    • Winkel: Eingabe für die Eigenschaft Perforation Angle property.
    • Länge des ersten Abschnitts: Eingabe für die Eigenschaft Perforation Initial Length.
    • Maximale Länge eines Abschnitts: Eingabe für die Eigenschaft Perforation Max Length.
    • Maximale Länge der Stege: Eingabe für die Eigenschaft Nonperforation Max Length.

Hinweise

  • Siehe Ein Blechobjekt erstellen für einen prinzipiellen Arbeitsablauf.

Beispiel

Eine einfache Schale

Vorbereitung

Diese Schale besteht aus einer rechteckigen Platine mit an ihren Umrisskanten (-Linien) angefügten Kanten. Also muss eine Skizze für die Platine vorbereitet werden.

Nur eine rechteckige Kontur

Arbeitsablauf

  1. Eine Platine erstellen
    1. Die Konturskizze auswählen
    2. Schaltfläche Basisprofil erstellen drücken
      oder Tastenkürzel C dann F verwenden

      (Die Platine wird in Z-Richtung aufgedickt)

  2. Kanten an die Umrisskanten anfügen
    1. Auswahl der Umrisskanten der Platine
    2. Schaltfläche Kante ansetzen drücken
      oder Tastenkürzel W verwenden
    3. Wenn die Kanten mit 90° nach unten zeigen, setzt man den Wert der Eigenschaft invert auf true, um die Richtung umzukehren (und den von length auf einen niedrigeren Wert für kürzere Kanten)


  3. Noch mehr Kanten hinzufügen
    1. Die oberen Umrisskanten auswählen
    2. Schaltfläche Kante ansetzen drücken
      oder Tastenkürzel W verwenden

    3. Die Kanten sind jetzt ein wenig zu lang (aber schön beschnitten), also muss die Eigenschaft length auf einen niedrigeren Wert gesetzt werden
    4. Sollen die Kanten nach außen aufgestellt werden, muss der Wert invert auf true gesetzt werden

Fertig!
