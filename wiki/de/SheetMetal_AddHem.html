Beschreibung

Der Befehl Falz ansetzen erstellt Falze an ausgewählten Flächenrändern. Durch das Ändern der Daten-Eigenschaftangle kann aus einer Kante ein Falz werden.

Ein Falz ein Falz besteht üblicherweise aus einem zylindrischen Bogen von ungefähr 180° und einem ebenen Streifen (Wand).

Zwei gewählte Kanten (edges) → zwei Falze mit Gehrung (hems)

Anwendung

  1. Eine oder mehrere Kanten einer Festkörperform auswählen.
  2. Es gibt mehrere Möhglichkeiten, den Befehl aufzurufen:
    • Die Schaltfläche Falz ansetzen drücken.
    • Den Menüeintrag Sheet Metal → Falz ansetzen auswählen
    • EinRechtsklick in die Baumansiht oder die 3D-Ansicht und die Menüoption Sheet Metal → Falz ansetzen im Kontextmenü auswählen.
    • Das Tastaturkürzel Z.
  3. Ein Hem-Objekt wird erstellt, das einen neuen Falz an jeder ausgewählten Kante (Flächenrand) enthält, und das Aufgaben-Fenster Falzeigenschaften wird geöffnet:
    • Mit ausgewähltem Reiter Allgemein:
      • Wahlweise die Schaltfläche Select drücken, um weitere Kanten hinzuzufügen.
        • Wahlweise die Schaltfläche Clear selection drücken, um die Liste der ausgewählten Kanten zu löschen
        • Die Schaltfläche Preview drücken, um die Auswahl abzuschließen und die Änderungen anzuzeigen.
      • Wahlweise die Parameter der Biegung in diesem Reiter anpassen. Siehe Abschnitt Allgemein der Beschreibung des Aufgaben-Fensters weiter unten.
    • Wahlweise zu einem anderen Reiter wechseln, um weitere Parameter einzustellen. Siehe die Abschnitte Abstände oder Gehrung der Beschreibung des Aufgaben-Fensters weiter unten.
  4. Die Schaltfläche OK drücken, um den Befehl abzuschließen und das Aufgaben-Fenster zu schließen.
  5. Wahlweise die Parameter in der Eigenschaften-Ansicht anpassen.

Aufgaben-Fenster

Ein vorhandenes Hem-Objekt in der Baumansicht doppelt anklicken, um das Aufgaben-Fenster erneut zu öffnen und die Parameter zu bearbeiten.

Allgemein


  • Select: Changes the amount of edges in the base Object property.
  • Type: Sets the HemType property.
    • Flat activates these sub-options:
      • Width: Sets the width property.
      • Reverse the wall: Toggles the invert property.
      • Width includes bend: Toggles the Include Bend property.
    • Open: activates these sub-options:
      • Width: Sets the width property.
      • Reverse the wall: Toggles the invert property.
      • Width includes bend: Toggles the Include Bend property.
      • Opening: Sets the opening property.
    • Tear Drop: activates these sub-options:
      • Width: Sets the width property.
      • Reverse the wall: Toggles the invert property.
      • Width includes bend: Toggles the Include Bend property.
      • Bend radius: Sets the radius property.
      • Open: Toggles the opened property.
        • Opening: Sets the opening property.
    • Rolled: activates these sub-options:
      • Bend radius: Sets the radius property.
      • Open: Toggles the opened property.
        • Roll angle: Sets the Roll Angle property.
  • Position: Sets the Bend Type property.
  • Unfold: Toggles the unfold property.


Offsets

  • Gap A: Sets the gap1 property.
  • Gap B: Sets the gap2 property.
  • Rectangle and Round radio buttons: Toggle relief Type property.
  • Width: Sets the reliefw property.
  • Depth: Toggles the reliefd property.


Gehrung

(Not available for rolled hems)

  • Auto Miter: Toggles the Auto Miter property.
    If checked:
    • Minimum Gap: Sets the minGap property.
    • Max Extend Distance: Sets the max Extend Dist property.
    If unchecked:
    • Angle 1: sets the miterangle1 property.
    • Angle 2: sets the miterangle2 property.


Hinweise

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

A SheetMetal Hem object is derived from a Part Feature object or, if it is inside a PartDesign Body, from a PartDesign Feature object, and inherits all its properties. It also has the following additional properties:

Daten

Parameters

  • DatenBend Type (Enumeration): "Bend Type". Material Outside (default), Material Inside, Thickness Outside.
  • DatenHem Type (Enumeration): "Type of Hem". Flat (default), Open, Teardrop, Rolled.
  • DatenInclude Bend (Bool): "Width includes bend geometry". Default: false.
  • DatenRoll Angle (Angle): "Roll Angle". Default angle: 90,00°.
  • Datenbase Object (LinkSub): "Base Object". Link to the planar face to receive a hem.
  • Datengap1 (Distance): "Gap from Left side". Default: 0,00 mm.
  • Datengap2 (Distance): "Gap from Right side". Default: 0,00 mm.
  • Dateninvert (Bool): "Invert Bend Direction". Default: false.
  • Datenopened (Bool): "Opened Hem". Default: false.
  • Datenopening (Length): "Openening. of Hem". Default: 1,00 mm
  • Datenradius (Length): "Bend Radius", the default value depends on the radius property of the parent feature:
    • That property is not existent: This property is set to 1,00 mm.
    • That property contains a numeric value: An expression linking that property is inserted into this property.
    • That property contains an expression: The expression is copied into this property.
  • Datenwidth (Length): "Width of Wall". Default: 10,00 mm.

Parameters Ex

  • DatenAuto Miter (Bool): "Enable Auto Miter". Default: true.
  • Datenkfactor (FloatConstraint): "Location of Neutral Line. Caution: Using ANSI standards, not DIN.".
    Default: 0,50. K factor (also known as neutral factor) for the bend. Used to calculate bend allowance when unfolding.
  • Datenmax Extend Dist (Length): "Auto Miter maximum Extend Distance". Default: 5,00 mm.
  • Datenmin Gap (Length): "Auto Miter Minimum Gap". Default: 0,20 mm.
  • Datenmin Relief Gap (Length): "Minimum Gap to Relief Cut". Default: 1,00 mm.
  • Datenunfold (Bool): "Shows Unfold View of Current Bend". Default: false
    true unfolds the bend.

Parameters Miterangle

  • Datenmiterangle1 (Angle): "Bend Miter Angle from Left Side". Default angle: 0,00°.
  • Datenmiterangle2 (Angle): "Bend Miter Angle from Right Side". Default angle: 0,00°.

Parameters Relief

  • DatenRelief Factor (Float): "Relief Factor". Default: 0,70.
  • DatenUse Relief Factor (Bool): "Use Relief Factor". Default: false.
  • Datenrelief Type (Enumeration): "Relief Type". Rectangle (default), Round. Enabled only when a gap value is set.
  • Datenreliefd (Length): "Relief Depth". Default: 1,00 mm. Enabled only when a gap value is set.
  • Datenreliefw (Length): "Relief Width". Default: 0,80 mm. Enabled only when a gap value is set.
