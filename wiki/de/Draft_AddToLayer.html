Beschreibung

Der Befehl Draft ZurEbeneHinzufügen fügt Objekte zu einer Ebene hinzu oder entfernt sie aus einer beliebigen Ebene. eingeführt in 1.1

Anwendung

  1. Select one or more objects.
  2. There are several ways to invoke the command:
    • Press the Add to Layer button.
    • Select the Utilities → Add to Layer option from the menu, or from the Tree View or 3D View context menu.
  3. A menu is displayed near the cursor. Do one of the following:
    • Select Remove From Layer to remove the objects from the layer they are in.
    • Select the layer you want to add the objects to.
    • Select Add to New Layer to add the objects to a new layer:
      1. The New Layer dialog box opens.
      2. Enter a Layer name.
      3. Press the OK button.
