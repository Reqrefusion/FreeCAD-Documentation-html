Description

The Std DockOverlayTransparentAll command toggles a special "transparent active" mode for all panels currently in the overlay system.

Normally, when overlay mode is active, panels become fully opaque when the mouse cursor hovers over them. The "transparent active" mode changes this behavior, causing the panels to remain transparent even when the mouse is over them. The controls inside the panel (buttons, tree items, etc.) are still fully interactive. This allows a user to work within a panel while still seeing the 3D model underneath, which can be useful on smaller screens.

This command acts as a global switch, enabling or disabling this special mode for all overlay panels at once.

Usage

  1. The overlay mode must be active.
  2. There are several ways to invoke the command:
    • Select the View → Dock panel overlay → Toggle transparent for all option from the menu.
    • Use the keyboard shortcut: SHIFT+F4.

Preferences

The appearance of the panels in this "transparent active" state is controlled by the same stylesheet as the standard overlay mode.
