Beschreibung

Das GL Transmission Format (glTF™) ist eine lizenzgebührenfreie Spezifikation für die effiziente Übertragung und das Laden von 3D-Szenen und -Modellen durch Anwendungen. Sie minimiert sowohl die Größe von 3D-Inhalten als auch die zum Entpacken und Verwendung dieser Inhalte erforderliche Laufzeitverarbeitung.

Importieren

Der Befehl Std Import kann in das glTF-Format importiert werden. Sowohl binär (.glb) als auch Textformate (.gltf) werden unterstützt. eingeführt in 1.0

Alternativer Import-Arbeitsablauf

The import operation is done via OCCT, FreeCAD's CAD kernel. As it is a resource-intensive task, it can take a considerable amount of time. An alternative approach is to use an external tool to reduce the amount of processing that is being done at the same time. Blender for instance has good support for .glTF and can very quickly do the first step of reading the data stream and creating a mesh object that can be further processed in FreeCAD. Or it can be directly used as a mesh without further processing.

Schritt 1: Umwandlung von STL mittels Blender

  1. Open Blender and import your source mesh file: File → Import → glTF 2.0.
  2. Select the imported object.
  3. Verify its dimensions are correct.
  4. Export the object via: File → Export → Stl (.stl).
    • In the export options, enable Selection Only and set Scale to 1.0.
    • Save the output STL file.

Schritt 2: Erstellen eines Volumenkörpers aus STL in FreeCAD

  1. Import the .stl file into FreeCAD. This will create a mesh object and will be a quick operation.
  2. If the imported mesh is scaled incorrectly (e.g., a meters-to-millimeters error), switch to the Mesh Design Workbench, select the mesh, and use Mesh Scale to apply the necessary correction factor (e.g., 1000).
  3. Switch to the Part Workbench.
  4. Select the scaled mesh object and convert it to a shape using Part → Create shape from mesh....
  5. Select the new shape object and create a solid using Part → Convert to solid.

Das Ergebnis ist ein natives FreeCAD-Volumenkörperobjekt, das für die weitere Modellierung bereit ist.

Exportieren

Der Befehl Std Export kann in das glTF-Format exportieren.

Wenn beim Speichern einer Datei die vom Benutzer angegebene Erweiterung .gltf lautet, wird sie als Kombination aus einem zweiteiligen Text filename.gltf und einer Binärdatei filename.bin gespeichert. Wenn die vom Benutzer angegebene Erweiterung .glb lautet, wird sie als eine Binärdatei .glb gespeichert.
