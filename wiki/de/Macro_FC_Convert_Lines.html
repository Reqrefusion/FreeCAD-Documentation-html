Konvertierte Linien Strich, Strichpunkt, Strich Punkt Punkt

Anwendung

Das Makro in den Makro-Ordner kopieren und die grafische Benutzeroberfläche FC Linien konvertieren starten.

Erster Abschnitt:

Linie schneiden (Standardmäßig aktiviert)

  • Eine oder mehrere Linien in der 3D-Ansicht auswählen
  • SpinBox : für die Eingabe der Anzahl der Schnitte oder Schnitte auf Länge ... (Standardmäßig wird das Suffix für die Anzahl der Schnitte "x Cut" angezeigt.)
  • Erzeuge L. : Wenn dieses Kontrollkästchen aktiviert ist, wird die Zeile erstellt
  • Dimension : Wenn dieses Kontrollkästchen aktiviert ist, wird die Anzahl der Stellen, um die die Definition geändert wird, durch drei Dezimalstellen angegeben, und das angezeigte Suffix lautet "x.000 Dim"
  • Zweifarbig : Wenn dieses Kontrollkästchen aktiviert ist, werden die erstellten Linien abwechselnd rot und weiß dargestellt. Wenn es nicht aktiviert ist, wird die mit der Schaltfläche Farbe festgelegte Farbe verwendet
  • Punkte : Wenn dieses Kontrollkästchen aktiviert ist, wird ein Punkt erstellt und die Farbe ist die definierte Farbe. Zweifarbig oder * Die Optionen "Optionslinien" sind verfügbar

Farbe

Zweiter Abschnitt:

Linientyp : Strich

  • Der Bildschirm am unteren Rand der Makro-Änderung in der Zeile Strich
  • Eine oder mehrere Zeilen in der 3D-Ansicht auswählen
  • Linie A : Abmessung des Strichs (siehe A auf dem Bildschirm)
  • Platz B : Abmessung des Leerraums (siehe B auf dem Bildschirm)
  • Die Linien wurden mit den in den Optionszeilen festgelegten Konfigurationsspezifikationen erstellt

Dritter Abschnitt:

Type line : DashDot

  • The screen at the bottom of the macro change in line DashDot
  • Select one or more lines in the 3D View
  • Line A : dimension of the dash (see A in the screen)
  • Space B : dimension of the space (see B in the screen) (May be different D)
  • Line 2 C : dimension of the dash (see C in the screen)
  • Space 2 D : dimension of the space (see D in the screen) (May be different B)

Vierter Abschnitt:

Type line : DashDotDot

  • The screen at the bottom of the macro change in line DashDotDot
  • Select one or more lines in the 3D View
  • Line A : dimension of the dash (see A in the screen)
  • Space B : dimension of the space (see B in the screen)
  • Line 2 C : dimension of the dash (see C in the screen)
  • Space 2 D : dimension of the space (see D in the screen)

Fünfter Abschnitt:

Type line : ZigZag

  • The screen at the bottom of the macro change in line ZigZag
  • This line are created
  • Number heads : number of heads for the total line
  • _____140.0_____ : The length total of the line are calculate in real time and displayed in this window
  • Begin A : dimension of begin line until first head, this section line hare same in the end line (see A in the screen)
  • Dimension B : dimension between the end and begin head (see B in the screen)
  • Gap C : dimension of the head (see C in the screen)
  • Height D : height dimension of the the head (see D in the screen)
  • Number E : number of heads contiguous (see E in the screen)
  • The options Plane and Options lines are available

Sechster Abschnitt:

Type line : Hand

  • This line are created
  • Length A : The length total of the line (see A in the screen)
  • Height B : Height of the line (see B in the screen)
  • Wave : compression or uncompress of the wave (see Wave in the screen)
  • The options Plane and Options lines are available

Siebter Abschnitt:

Optionen Linie :

Diese Optionen sind für alle Menüs verfügbar

  • __2,00 Width__ : Maß der Breite der Linie(n)
  • __2,00 Point S__ : Abstand der Punkte der Linie(n)
  • Color :Farbe der Linie(n) Diese Schaltfläche ist in der ausgewählten Farbe dargestellt (wenn das Kontrollkästchen „Zweifarbig“ im Menü „Schnittlinie“ aktiviert ist, wird diese Option für dieses Menü nicht verwendet)

Achter Abschnitt:

Options-Ebene

Diese Option ist nur verfügbar für "ZickZack" und "Freihand"

  • XY : Ebene XY
  • YZ : Ebene YZ
  • XZ : Ebene XZ

Neunter Abschnitt:

Button

  • Save type : the line configured is saved in one file (one file for one type line). While saving a file header is predetermined depending on the chosen menu (example : Dash is saved, in the save window is displayed "Dash_.FCConvertL" you can modify to "Dash_my_config_10.FCConvertL" or the name you want... this method allow sharpens clear)
  • Load type : load one file configured
  • Name type line________________ : name of the type configured this name is saved in the file
  • Reset : reset the data's in the original configuration
  • Create Comp : this button create on compound with the lines created (ten lines selected = one compound)
  • Create : create the lines separately (ten lines selected = ten lines created)
  • Quit : quit the macro

Die Dateien, die mit dem Makro in das Verzeichnis kopiert werden müssen (10 Bilddateien)
