Spickzettel

Initialisierung

Alle Module laden: 

pload ALL

Eine Draw/tcl-Datei (zum Beispiel test.tcl) laden: 

source test.tcl

Dateien auf dem System suchen: 

pwd
cd ..
ls

Eine BREP-Datei wiederherstellen

Datei model.test in die Variable model zurückspeichern: 

restore model.test model

Ein Modell anzeigen

Modell anzeigen model: 

vdisplay model
vfit

Eine Kante oder Fläche im Modell finden

Zuerst das Modell anzeigen.

Für Kanten den Auswahlmodus 2, für Flächen den Auswahlmodus 4 wählen: 

vselmod 2 on

Eine Kante oder Fläche im Fenster auswählen und die Nummer der Kante oder Fläche finden: 

vstate

Darin wird in etwa Folgendes stehen: 

Locally selected sub-shapes within model:
       Edge #4

Modell model mit e für Kanten oder f für Flächen zerlegen: 

explode model e

You can access edge #4 with model_4, for example displaying it: 

vdisplay model_4

Get a subshape of a model model directly, face 3 and display it: 

subshape model f 3
vdisplay model_3

Formen überprüfen

Verschiedene Befehle zur Überprüfung der Form shape: 

checkshape shape
bopcheck shape
whatis shape
nbshapes shape

Toleranzen

Die Formtoleranzen einer Form shape überprüfen: 

toler shape
dump shape

Lege die Toleranz einer Form shape fest: 

dset tolerance 0.0001
settolerance shape tolerance

Verrundungen und Fasen

Create a fillet with radius 0.5 or a chamfer with distance 0.5 on the first edge of model and store it in shape result. Notice that the position of the radius and distance is reversed for the two different calls, for fillet: 

blend result model 0.5 model_1

Für Fasen: 

chamf result model model_1 0.5

Einen Quader erstellen

Erstelle ein Quader b 

box b 0 0 0 1 2 3

Hintergrundinformationen

Installation

Das Programm Draw wird in der Regel zusammen mit der OpenCASCADE-Distribution installiert. Unter Linux findet man das Programm normalerweise im Pfad unter DRAWEXE.

If you followed Compile with OCCT, you can find the Draw tool of the custom version in your distribution directory, in the bin directory of your distribution.

Um sicherzustellen, dass man eine OCCT-Draw-Datei aus einer bestimmten Distribution lädt, hat man die folgende Funktion in seiner $HOME/.bashrc: 

function set-occt() {
    if [ -z "$1" ]; then
	echo "Usage: set-occt <dir>"
	return 1
    fi
    if [ ! -d "$1" ]; then
	echo "Error: Directory $1 does not exist"
	return 1
    fi
    pushd "$1" > /dev/null
    export CASROOT=
    export CSF_OCCTResourcePath=$PWD/share/opencascade/resources
    export LD_LIBRARY_PATH=$PWD/lib:$LD_LIBRARY_PATH
    export PATH=$PWD/bin:$PATH
    source ./bin/env.sh
    popd > /dev/null
}

Angenommen, man möchte die OCCT-Version verwenden, die sich im folgenden Installationsverzeichnis /scratch/dist-occt-main befindet, dann führt man in meiner Bash-Shell Folgendes aus: 

set-occt /scratch/dist-occt-main

Ich überprüfe, ob sich das richtige Draw-Programm tatsächlich in meinem Pfad befindet: 

which DRAWEXE

Das sollte Folgendes ausgeben: 

/scratch/dist-occt-main/bin/DRAWEXE

Die Versionsnummer kann wie folgt überprüft werden: 

DRAWEXE --help
