Beschreibung

Der Befehl Part Schnittansicht arbeitet nur mit Part- und PartDesign-Objekten sowie mit Baugruppen aus diesen zusammen. Er erstellt ein dauerhaft beschnittenes Objekt aus (vorhandenen) Objekten und Baugruppen. Da das ausgeschnittene Ergebnis ein normales Part Differenz-Objekt (Cut-Objekt) ist, kann es weiterbearbeitet werden oder z.B. 3D-gedruckt werden. Siehe unten für mögliche Anwendungen.

Eine Baugruppe mit Ausschnitt. Einige der Schnittflächen sind von Hand eingefärbt. Das gelbe Teil wurde nicht ausgeschnitten, da es absichtlich um ein Tausendstel (mm) in ein anderes Teil verschoben wurde.

Anwendung

Der Dialog Section Cut wird mit dem Menüeintrag Part → Dauerhafte Schnittdarstellung geöffnet. Einmal aktiviert ist er unabhängig vom aktuellen Arbeitsbereich und vom aktuell geöffneten Dokument. Er kann durch drücken der Schaltfläche in der linken oberen Ecke des Dialogs von seiner Ursprungsposition gelöst werden.

Die Funktion Schnittansicht berücksichtigt alle derzeit sichtbaren Teilobjekte im aktiven Dokument. Daher kann man steuern, was geschnitten wird, indem man ein Teil sichtbar macht oder nicht. Durch Aktivieren einer der Optionen unter Schneiden im Dialogfeld wird die Funktion aktiviert. Man kann dann entweder eine Position (in Koordinaten des Dokuments) eingeben oder die Schieberegler verwenden, um die Schnittposition festzulegen. Es ist auch möglich, Schnitte zu kombinieren, beispielsweise um in X- und Z-Richtung zu schneiden. Mit den Schaltflächen Umdrehen kann man die Seite, die geschnitten wird, umdrehen.

Sobald die Option Schneiden im Dialogfeld aktiviert ist, erhält man ein Schnitt-Objekt in der Baumansicht. Sein Name lautet z. B. SectionCutY, wenn es sich um einen Schnitt in Y-Richtung handelt.

Die Dialogoption Beim Schließen nur Schnitte sichtbar lassen blendet alles in der Baumansicht aus, außer dem Schnitt-Objekt, wenn die Schaltfläche Schließen zum Schließen des Dialogs angeklickt wird.

Um das ausgeschnittene Objekt zu entfernen, alle Optionen unter Ausschneiden deaktivieren.

Wenn man alle Optionen unter Schneiden deaktiviert, wird die Schaltfläche Ansicht aktualisieren aktiv. Wenn man darauf klickt, wird eine Art Bildschirmaufnahme der aktuell sichtbaren Teilobjekte erstellt. Dieser wird verwendet, wenn man das nächste Mal eine Option unter Schneiden aktiviert. Die Aktualisierung ist erforderlich, wenn man das Dokument gewechselt hat. Sie ist außerdem nützlich für Baugruppen, bei denen man möglicherweise einige Teile ausblendet oder später zum Schnitt hinzufügen möchte. In diesem Fall berechnet die Aktualisierung die Min-/Max-Werte der Schieberegler und Schnittpositionen entsprechend den aktuell sichtbaren Objektabmessungen neu.

Wenn die Option Auto im Abschnitt Schnittfläche aktiviert ist, werden Farbe und Transparenz der Schnittelemente für die Schnittfläche übernommen. Dies funktioniert nur, wenn alle Schnittelemente dieselbe Farbe oder Transparenz aufweisen.

Die Option Sich überschneidende Objekte schneiden ermöglicht es, auch Objekte zu schneiden, die sich gegenseitig überschneiden. In Baugruppen kommt es aufgrund numerischer Genauigkeitsprobleme manchmal zu Überschneidungen bei Objekten, die eigentlich nur einander berühren sollen. Der Nachteil dieser Option ist, dass alle sichtbaren Objekte dieselbe Farbe erhalten. Diese Farbe kann wie im Abschnitt Geschnittene Fläche des Dialogfelds festgelegt werden. Wenn beispielsweise den Schnitt für ein ansprechendes Bild mit mehreren Flächenfarben benötigt, kann man die Flächenfarben mit dem Werkzeug Farbe pro Fläche ändern.

Hinweis: Bei Baugruppen sind die Schieberegler im Dialogfeld deaktiviert (mit Ausnahme des Schiebereglers für die Transparenz). Der Grund dafür ist, dass eine Bewegung des Schiebereglers in kurzer Zeit zu vielen Schnittvorgängen führt. Bei Baugruppen wird dadurch schnell die gesamte CPU-Leistung beansprucht, und eine schwergängige Bewegung des Schiebereglers ist nicht hilfreich.

Wenn ein Schnittobjekt in der Baumansicht ausgewählt wird und dann das Dialogfeld Schnitt geöffnet wird, werden die Schnittpositionen in das Dialogfeld eingelesen.

Anwendungen

  • An important use case is that Section Cut creates filled cuts, not hollow ones like the Clip Plane feature.
  • Section Cut is useful for assemblies to visualize for example the working principle of a device. You thereby might want to color certain cut faces using the Color per face tool. To use the tool, switch to the Part or PartDesign workbench, right-click on the cut object in the tree view and select in the context menu Set colors.
  • Without the option Cut intersecting objects only parts that don't intersect others will be cut. This can be used as a collision test.
  • The Section Cut feature can be used for technical drawings to highlight certain areas or to be able to draw in dimensions. The image below shows an example where the TechDraw features ActiveView and View are used.

Einschränkungen

  • Important: The Section Cut feature works poorly with OpenCASCADE 7.4 and older due to bugs. It is therefore recommended to use OpenCASCADE 7.5 or newer (all builds of FreeCAD 0.20 and above assure this).
  • 1.0 and above: The option Cut intersecting objects will color all visible parts the same. This can technically not be avoided. However, if one needs the persistent cut for e.g. a presentation, see the method described above how to reset the color manually.
  • 0.20 and below: In assemblies parts that intersect each other cannot be cut. Normally intersecting objects will not be cut while the others will. However, sometimes the cutting can produce strange results which is a bug in the OpenCASCADE libraries.
    To get a cut view also for intersecting objects, you can use the the macro Cross Section.
  • 0.20 and below: Especially when using the A2plus workbench, some the assembled parts can overlap each other by just a micron due to internal rounding errors. To fix this, add a micron as space in the constraint settings.
  • There can be color artifacts in the cut result. If and how depends on the OpenCASCADE library and also on the view position. In many cases the color artifacts disappear when the 3D view is slightly rotated.
  • When having cut objects with different colors, it is not possible to apply automatically their color to the corresponding cut faces. All cut faces will get the same color selected in the dialog.
  • When using the A2plus workbench, it is not possible to apply automatically the color of the assembled parts to the corresponding cut faces. All cut faces will get the same color selected in the dialog. The reason is that A2plus does not input the parts as link but loads them as file.

Hintergrundinformationen

Schnittansicht ist von dem Makro Querschnitt inspiriert und funktioniert technisch auf diese Weise:

Alle sichtbaren Objekte werden in einen Part Verbund-Container gelegt. Für die Option Schneidende Objekte schneiden wird stattdessen ein BoolescheBestandteile-Container verwendet. Die Verbindung wird mit einem Part Quader-Objekt geschnitten. Der Quader muss so groß sein, dass er das gesamte Volumen aller sichtbaren Objekte abdeckt. Dazu wird der Begrenzungsrahmen der Objekte erfasst. Wenn die Ansicht durch Hinzufügen/Entfernen von Objekten oder Ändern des Dokuments geändert wird, muss die Begrenzungsbox aktualisiert werden. Dies geschieht, wenn die Schaltfläche Ansicht aktualisieren angeklickt wird.
