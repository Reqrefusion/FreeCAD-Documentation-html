Beschreibung

Der Befehl Part Toleranzangaben erstellt eine parametrische Kopie ausgewählter Objekte, wobei alle enthaltenen Toleranzen auf mindestens einen bestimmten Mindestwert gesetzt werden. Das erstellte Objekt verhält sich wie eine Verbindung, wenn mehrere Objekte ausgewählt/zum Feature hinzugefügt werden, oder wie eine einzelne Form, wenn nur ein Objekt ausgewählt wird.

Toleranzen wirken sich auf nachfolgende OCCT-Funktionen wie boolesche Operationen aus. Toleranzen von Teilformen können mit Hilfe von Teilformbindern und dem Befehl GeometriePrüfen überprüft werden.

Anwendung

  1. Select one or more objects.
  2. Select the Part → Compound → Set Tolerance option from the menu.
  3. A Tolerance object is created and the original object hidden.

Eigenschaften

Tolerance Set

  • DatenObjects: Objects to have tolerance adjusted.
  • DatenRefine: If true, refinement of the resulting shape is enabled.
  • Datenmax Tolerance: Sets the maximum tolerance for the shape and all subshapes (if any).
  • Datenmin Tolerance: Sets the minimum tolerance for the shape and all subshapes (if any).
