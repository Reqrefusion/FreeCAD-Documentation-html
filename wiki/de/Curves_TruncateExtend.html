Beschreibung

Das Werkzeug Curves TruncateExtend erstellt einen Schnitt durch eine Form und verlängert bzw. kürzt sie um einen gegebenen Abstand.

Anwendung

  1. Select a shape and a plane or only a plane in a PartDesign body.
  2. There are several ways to invoke the tool:
    • Press the Truncate Extend button.
    • Select the Surfaces → Truncate Extend option from the menu.
  3. A Truncated_Extend object is created.

Eigenschaften

Siehe auch: Eigenschafteneditor.

Ein Truncated_Extend-Objekt wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem besitzt es die folgenden zusätzlichen Eigenschaften:

Daten

Input Objects

  • Daten-EigenschaftCutter (Link): Das ebene Objekt, das das Quellobjekt schneidet
  • Daten-EigenschaftSource (Link): Das Objekt, das gekürzt oder erweitert wird

Settings

  • Daten-EigenschaftDistance (Distance): Die zu verkürzende (bei negativem Wert) oder zu verlängernde (bei positivem Wert) Distanz
  • Daten-EigenschaftRefine (Bool): Form verfeinern (überflüssige Kanten bereinigen)
  • Daten-EigenschaftReverse (Bool): Umgekehrte Normalschneideebene
