Beschreibung

Der Befehl KnotenBefestigen befestigt einen bestehenden Knoten des Leitungsverlaufs an einem externen Knoten oder Objekt. Die Befestigungen von Knoten werden in speziellen Eigenschaften des Leitungsverlaufs festgehalten. Für Details siehe Leitungsverlauf-Eigenschaften.

It can be used interchangeably with the "Attach point" command from Cables Edit, see the WireFlex Description.

Sobald der Leitungs-Knoten angehängt ist, behält er dieselbe Position im 3D-Raum wie der externe Knoten, an den er angehängt ist. Wenn der externe Knoten bewegt wird, bewegt sich der angehängte Leitungs-Knoten mit. Wenn die Leitung in Draft Bearbeiten bearbeitet wird, kann der angehängte Knoten nicht verschoben werden. Er springt auf seine angehängte Position zurück.

Anwendung

  1. Wähle einen Knoten des bestehenden Leitungsverlauf-Objekts in der 3D-Ansicht aus.
  2. Wähle einen externen Knoten in der [[3D_

view/de|3D-Ansicht]] mit gedrückter STRG-Taste aus.

  1. Erzeuge einen neuen Leitungsverlauf-Knoten befestigen mit einer dieser Methoden:
    • Drücke den Knopf Knoten hinzufügen.
    • Wähle die Option Leitungsverlauf → Knoten hinzufügen aus dem Menü.
    • Rechts-Klick in der Baumansicht oder der 3D-Ansicht und wähle den Befehl Leitungsverlauf → Knoten hinzufügen aus dem Kontextmenü.

Hinweis: Die Auswahl von externen Elementen, die keine Knoten sind, wird wie eine Auswahl des gesamten Objekts behandelt. Das Anhängen an ein Objekt wird als Anhängen an seine Placement.Base verstanden.

Hinweise

  • Ab Version 0.2.0 kann dieser Befehl auch durch Verwendung des Knoten-Kontextmenüs im Befehl Cables Bearbeiten ausgeführt werden.
  • Siehe Beschreibung und Anwendung für Details zur Verwendung von Leitungsverläufen
