The required parameters are: 

{{Macro/de
|Name=SomeMacro
|Description=Dieses Makro verbessert die Volumenkörper in jeder Hinsicht.
|Author=WikiUserName
|Download=Mögliche Dateien zum Download, zB das Icon toolBar ...
|Date=2018-11-20
|Version=2.0
|FCVersion= 0.16 to 0.19
}}

Andere Parameter können weggelassen werden oder ihr Wert bleibt leer.

Erforderliche Parameter

Parameter werden mit ihrem Wert nach dem Gleichheitszeichen angegeben. parameter=value.

Jeder Parameter in der Liste ist durch ein Pipe-Symbol getrennt |. 

|... |Name=SomeMacro|Author=WikiUserName| ...

Parameter können in derselben Zeile oder in einer separaten Zeile platziert werden. 

|...
|Name=SomeMacro
|Author=WikiUserName
| ...

Die folgenden Parameter sind erforderlich.

Name
Der richtig großgeschriebene Name des zu dokumentierenden Makros.
Der Name wird verwendet, um die Symbolgrafik Name.svg oder Name.png aus dem Wiki-Repository abzurufen. Die Vorlage funktioniert ordnungsgemäß, wenn das Symbol nicht vorhanden ist. In diesem Fall wird ein generisches Symbol angezeigt.
Description
Eine Beschreibung für das Makro. Diese Beschreibung wird im Addon-Manager angezeigt.
Author
Der Autor des Makros. Dieser Name ist mit der Benutzerseite im Wiki verknüpft, dh User:Author. Da Makros nicht Teil des offiziellen FreeCAD-Codes sind, sollten Benutzer ihre Fragen an ihre jeweiligen Entwickler richten. Der Autor eines Makros sollte im forum unter demselben Benutzernamen wie im Wiki erreichbar sein.
Download
Mögliche Dateien zum Download, zum Beispiel das Icon toolBar und eine andere Datei. Geben Sie den vollständigen Link des Objekts ab : Download=[https://www.freecadweb.org/wiki/images/2/2c/Text-x-python.png Text-x-python.png]
Date
Alle Makros müssen ein Datum haben. Dies ist hilfreich, um anzugeben, wann das Makro zuletzt geändert wurde, und den Benutzer über das Alter des Codes zu warnen. Sehr alte Makros sind möglicherweise veraltet und funktionieren möglicherweise nicht mit einer aktuellen Version von FreeCAD.
FCVersion
FreeCAD-Version (en) zum Ausführen des Makrobeispiels: 0.16 to 0.19

Optionale Parameter

Die folgenden Parameter sind optional. Wenn sie nicht in der Vorlage enthalten sind, wird ein Standardwert angezeigt.

Translate
Der in Ihre Sprache übersetzte Name, der im Fenstertitel angezeigt wird. Zum Beispiel Translate=Some_translated_name
Links
Anderer Hilfelink für das Makrobeispiel Video, Seite Tutorials, die sich auf das Makro bezieht.
Version
Die Version des Makros Wenn kein Wert angegeben wird, wird standardmäßig 1.0 verwendet. Dies ist hilfreich für Autoren, damit sie bestimmte Versionen ihrer eigenen Makros dokumentieren können.
Shortcut
Einige Makros können Verknüpfungen zum Starten bestimmter Tools enthalten. Geben Sie die Tastenkombinationen an, um diesen Wert zu drücken. Wenn kein Wert angegeben wird, wird standardmäßig None verwendet.
SeeAlso
Optionale Links zu anderen verwandten Befehlen, zum Beispiel SeeAlso=[[Some Tool|Tool]].
Icon
Eine Bilddatei, die sich von Name.svg oder Name.png unterscheidet und als Symbol verwendet wird. Dies kann zum Einschließen eines bestimmten Symbols verwendet werden, das beispielsweise nicht nach dem Befehl benannt ist, Icon=Some_other_name.svg
