SheetMetal ExtendBySketch

Menüeintrag
SheetMetal → Extend by Sketch

SheetMetal

T
Eingeführt in Version
0.7.57

SheetMetal Extrude

Beschreibung

Der Befehl SheetMetal MitSkizzeErweitern erweitert eine Blechplatte an einer ausgewählten Kantenfläche.

In Verbindung mit einer Konturskizze kann er verzahnte Geometrie erstellen, um ein Profil zu schließen:

Drei Profile mit hinzuzufügenden Konturskizzen → Drei Ergebnisse

Anwendung

(Prepare a sketch for interlocking tabs)
  1. Select one or more edge faces to be extended.
  2. Press Ctrl and select the sketch in the Tree View or the 3D View to add it to the selection.
  3. There are several ways to invoke the command:
    • Press the Extend by Sketch button.
    • Select the Sheet Metal → Extend by Sketch option from the menu.
    • Right-click in the Tree view or the 3D view and select the Sheet Metal → Extend by Sketch option from the context menu.
    • Use the keyboard shortcut: T.
  4. An Extend object will be created and the Extend properties Task Panel opens.
    • Optionally press the Select button and add or remove faces.
    • Press the Preview button to finish the selection and display the changes.
    • Optionally press the Extend Sketch... button and add or remove sketches.
    • Optionally check the Intersect Clearance check box and adjust the value to make room for interlocking tabs.
  5. Press the OK button to finish the command and close the Task Panel.

Aufgaben-Fenster

Ein vorhandenes Extend-Objekt in der Baumansicht doppelt anklicken, um das Aufgaben-Fenster erneut zu öffnen und die Parameter zu bearbeiten.

  • Select: Ändert die Anzahl der Flächen in der Eigenschaft base Object.
  • Extend Sketch…: Ändert die Anzahl der Skizzen in der Eigenschaft Sketch.
  • CheckBox Intersect clearance: Schaltet die Eigenschaft Use Subtraction um.
  • Intersect clearance-Wert: Eingabe für die Eigenschaft Offset.
  • Refine: Schaltet die Eigenschaft Refine um.

Hinweise

  • Dieser Befehl erstellt das gleiche Extend-Objekt wie der Befehl Fläche erweitern. Ob die Daten-EigenschaftSketch leer ist oder nicht, bestimmt welches Aufgaben-Fenster angezeigt wird, wenn das Extend-Objekt bearbeitet wird.
  • Siehe Fläche erweitern für weitere Hinweise.

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

Ein SheetMetal-Extend-Objekt wird von einem Part-Formelement abgeleitet oder, wenn es sich in einem PartDesign-Körper befindet, von einem PartDesign Formelement und erbt alle seine Eigenschaften. Außerdem hat es die folgenden zusätzlichen Eigenschaften:

Daten

Parameters

  • Daten-Eigenschaftbase Object (LinkSub): "Base object". Verweis zum Objekt und den ebenen Flächen, die erweitert werden sollen.
  • Daten-Eigenschaftgap1 (Distance): "Gap from the left side". Abstand von der linken Seite, Standardwert: 0,00 mm.
  • Daten-Eigenschaftgap2 (Distance): "Gap from the right side". Abstand von der rechten Seite, Standardwert: 0,00 mm.
  • Daten-Eigenschaftlength (Length): "Length of Wall". Kantenlänge, Standardwert: 10,00 mm.
    • Länge der Erweiterung, wenn die Daten-Eigenschaftreversed = false ist.
    • Tiefe des Ausschnitts, wenn die Daten-Eigenschaftreversed = true ist.
  • Daten-Eigenschaftreversed (Bool): "Reverse extend direction (cut)". Standardwert: false. Schaltet zwischen Erweiterung und Ausschnitt um.

Parameters Ext.

  • DatenOffset (Distance): "Offset for subtraction". Default: 0,20 mm. Clearance for interlocking geometry
  • DatenRefine (Bool): "Use Refine". Default: true.
  • DatenSketch (Link): "Wall Sketch". Link to the sketches defining the interlocking geometry. Lists the selected sketches by default.
  • DatenUse Subtraction (Bool): "Use Subtraction". Default: false Toggles if intersected walls of the same object are cut out for tabs defined by the linked Sketches.
