SheetMetal ExtendBySketch

Menu location
SheetMetal → Extend by Sketch

Beschreibung

The SheetMetal ExtendBySketch command extends a sheet metal plate at a selected edge face.

In connection with an outline sketch it can create interlocking geometry to close a profile:

Three profiles with outline sketches to add → three results

Anwendung

(Prepare a sketch for interlocking tabs)
  1. Select one or more edge faces to be extended.
  2. Press Ctrl and select the sketch in the Tree View or the 3D View to add it to the selection.
  3. There are several ways to invoke the command:
    • Press the Extend by Sketch button.
    • Select the Sheet Metal → Extend by Sketch option from the menu.
    • Right-click in the Tree view or the 3D view and select the Sheet Metal → Extend by Sketch option from the context menu.
    • Use the keyboard shortcut: T.
  4. An Extend object will be created and the Extend properties Task Panel opens.
    • Optionally press the Select button and add or remove faces.
    • Press the Preview button to finish the selection and display the changes.
    • Optionally press the Extend Sketch... button and add or remove sketches.
    • Optionally check the Intersect Clearance check box and adjust the value to make room for interlocking tabs.
  5. Press the OK button to finish the command and close the Task Panel.

Aufgaben-Fenster

Double-click an existing Extend object in the Tree View to re-open the Task Panel and edit the parameters.

  • Select: Changes the amount of faces in the base Object property.
  • Extend Sketch…: Changes the amount of sketches in the Sketch property.
  • Intersect clearance check box: Toggles the Use Subtraction property.
  • Intersect clearance value: Sets the Offset property.
  • Refine: Toggles the Refine property.

Hinweise

  • This command creates the same Extend object as the Extend Face command. Whether or not the DatenSketch property is empty controls which Task Panel is displayed when the Extend object is edited.
  • See Extend Face for additional notes.

Eigenschaften

See also: Property View.

A SheetMetal Extend object is derived from a Part Feature object or, if it is inside a PartDesign Body, from a PartDesign Feature object, and inherits all its properties. It also has the following additional properties:

Data

Parameters

  • Datenbase Object (LinkSub): "Base object". Link to the object and the planar faces to be extended.
  • Datengap1 (Distance): "Gap from the left side". Default: 0,00 mm.
  • Datengap2 (Distance): "Gap from the right side". Default: 0,00 mm.
  • Datenlength (Length): "Length of Wall". Default: 10,00 mm.
    • extension length if Datenreversed = false.
    • cut out length if Datenreversed = true.
  • Datenreversed (Bool): "Reverse extend direction (cut)". Default: false. Toggles between extension and cut out.

Parameters Ext.

  • DatenOffset (Distance): "Offset for subtraction". Default: 0,20 mm. Clearance for interlocking geometry
  • DatenRefine (Bool): "Use Refine". Default: true.
  • DatenSketch (Link): "Wall Sketch". Link to the sketches defining the interlocking geometry. Lists the selected sketches by default.
  • DatenUse Subtraction (Bool): "Use Subtraction". Default: false Toggles if intersected walls of the same object are cut out for tabs defined by the linked Sketches.
