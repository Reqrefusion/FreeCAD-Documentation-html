Beschreibung

Das Werkzeug Curves Rohrschale erstellt ein Sweep-Objekt PipeShell.

Anwendung

  1. Die Kante in der 3D-Ansicht auswählen, die als Leitkurve (Sweep-Path) verwendet wird.
  2. Ein oder mehrere benötigte Profile (Profile-Objekte) in der Baumansicht auswählen.
  3. Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
    • Die Schaltfläche Pipeshell drücken.
    • Den Menüeintrag Oberflächen → Pipeshell auswählen..
  4. Ein PipeShell-Objekt wird erstellt.
  5. Wahlweise die Werte im Eigenschafteneditor anpassen, um einen anderen Sweeping-Algorithm oder eine andere Ausgabeform auszuwählen und die zugehörigen Eigenschaften einzustellen (siehe Eigenschaften).

Eigenschaften

Siehe auch: Eigenschafteneditor.

Ein PipeShell-Objekt wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem besitzt es die folgenden zusätzlichen Eigenschaften:

Daten

Main

  • DatenMode (Enumeration): PipeShell mode used to select the sweeping algorithm:
    • Frene (default): ...
    • DiscreteTrihedron: ...
    • FixedTrihetron: ...
    • Binormal: ...
    • ShapeSupport: ...
    • AuxiliarySpine: ...
  • DatenOutput (Enumeration): Determines the shape of the object:
    • Sections (default): ...
    • Lofted Sections: ...
    • Surface: ...
  • DatenProfiles (LinkList):: Profiles that are swept along spine.
  • DatenSpine (LinkSubList): Sweep path.

Mode

  • Daten (Hidden)Auxiliary (Link): Auxiliary spine.
  • Daten (Hidden)Contact (Enumeration): Type of contact to auxiliary spine.
    • NoContact (default): ...
    • Contact: ...
    • ContactOnBorder: ...
  • Daten (Hidden)Corrected (Bool): Corrected Frenet.
  • Daten (Hidden)Direction (Vector): Direction of the Binormal and FixedTrihedron modes.
  • Daten (Hidden)EquiCurvi (Bool): Curvilinear equivalence.
  • Daten (Hidden)Location (Vector): Location of the FixedTrihedron mode.
  • Daten (Hidden)Support (Link): Shape of the ShapeSupport mode.

Settings

  • DatenMaxDegree (Integer): Maximum degree of the generated surface.
  • DatenMaxSegments (Integer): Maximum number of segments of the generated surface.
  • DatenSamples (Integer): Number of samples for preview.
  • DatenSolid (Bool): Make solid object.
  • DatenTol3d (Float): Tolerance 3D.
  • DatenTolAng (Float): Tolerance angular.
  • DatenTolBound (Float): Tolerance boundary.

Hinweise

  • Pipeshell erfordert ein Linienzugobjekt als Leitkurve (Sweep-Path), und mindestens ein Rohrschalenprofil (Pipeshell-Profile).
  • Die beiden Werkzeuge Rohrschale und Rohrschalenprofil arbeiten zusammen, wie ein "erweitertes" Austragungswerkzeug (Sweep-Tool).
