Beschreibung

The Cable Materials creates a series of Arch Materials and Arch Multimaterials ready to use with Cable. Two multimaterials are created by default: cableMultiMat3w and cableMultimat5w. They have proper naming of their composition which corresponds to default names in Sub Colors property of a Cable.

The chosen multimaterial can be attached to the Material property of a Cable.

Example of the default multimaterials:
There are J, L1, N, PE, CU names visible

Anwendung

  1. Create default set of multimaterials by one of these methods:
    • Press the Cable Materials button.
    • Select the Cables → Cable Materials option from the menu.
    • Right-click in the Tree view or the 3D view and select the Cables → Cable Materials option from the context menu.

Hinweise

Die Standard-MehrfachMaterialien können als Vorlagen für die Erstellung anderer benutzerdefinierter Listen von MehrfachMaterial dienen.
