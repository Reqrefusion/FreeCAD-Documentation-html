Macro Multi Export

Description
Ausgewählte Körper gleichzeitig in mehrere Dateiformate (STL, STEP, 3MF, OBJ, IGES, BREP, PLY, AMF) exportieren. Mit einem benutzerfreundlichen Dialogfeld für die Formatauswahl und die Konfiguration der Ausgabe.

Beschreibung

Dieses Makro bietet eine bequeme Möglichkeit, ausgewählte FreeCAD-Objekte gleichzeitig in mehrere Dateiformate zu exportieren. Anstatt jedes Format einzeln zu exportieren, können alle gewünschten Formate in einem einzigen Dialogfeld ausgewählt und gleichzeitig exportiert werden.

Unterstützte Exportformate:

  • STL - Stereolithographie (üblich für den 3D-Druck)
  • STEP - Standard for Exchange of Product Data (CAD-Austausch)
  • 3MF - 3D Manufacturing Format (modernes 3D-Druckformat)
  • OBJ - Wavefront OBJ (3D-Grafiken und Spiel-Engines)
  • IGES - Initial Graphics Exchange Specification (veraltetes CAD-Format)
  • BREP - OpenCASCADE native format (bewahrt die exakte Geometrie)
  • PLY - Polygon File Format (3D-Scannen und -Drucken)
  • AMF - Additive Manufacturing Format (XML-basierter 3D-Druck)

Funktionen:

  • Exportiert in einem einzigen Vorgang in mehrere Formate.
  • Benutzerfreundlicher Dialog zur Formatauswahl.
  • Konfigurierbares Ausgabeverzeichnis und Basisdateiname.
  • Einstellungen für die Maschen-Toleranz für STL/OBJ/PLY/3MF/AMF-Exporte.
  • Vorschau der zu erstellenden Dateien.
  • Schnellauswahl-Schaltflächen: „Alle auswählen“, „Keine auswählen“, „Standardeinstellungen zurücksetzen“.
  • Standardformate: STL, STEP und 3MF sind vorgewählt.

Anwendung

  1. Select one or more objects in the 3D view that you want to export.
  2. Run the macro from Macro → Macros → MultiExport → Execute.
  3. In the dialog:
    • Review the selected objects in the "Objects to Export" section.
    • Check the formats you want to export to (STL, STEP, 3MF are selected by default).
    • Choose an output directory using the "Browse..." button.
    • Enter a base filename (the format extension will be appended automatically).
    • Optionally adjust mesh tolerance settings for mesh-based formats.
  4. Click "Export" to create the files.

Voraussetzungen

  • FreeCAD 0.21 oder höher.
  • Objekte mit Festkörpern (Part oder PartDesign-Körper).

Installation

  1. Download the macro file: CutObjectForMagnets.FCMacro.
  2. Copy the file to your FreeCAD macro directory:
    • macOS: ~/Library/Application Support/FreeCAD/Macro/
    • Linux: ~/.local/share/FreeCAD/Macro/
    • Windows: %APPDATA%/FreeCAD/Macro/
  3. Optionally, download the toolbar icon and place it in the same directory.
