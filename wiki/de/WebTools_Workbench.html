Übersicht

WebWerkzeuge ist ein externer Arbeitsbereich, der eine Reihe von Werkzeugen zur Kommunikation mit Web Diensten aus FreeCAD heraus enthält.

Installation

Dieser Arbeitsbereich kann mit dem Addon Manager installiert werden. Für eine manuelle Installation siehe Weitere Arbeitsbereiche installieren.

Werkzeuge

  • BIM Server: Öffnet ein BIMServer Fenster.
  • Git: Verwaltet Dateien mit Git.
  • Sketchfab: Exportiert und lädt Objekte auf Dein SketchFab Konto hoch.

Verweise

  • Quellcode bereitgestellt auf GitHub: https://github.com/yorikvanhavre/WebTools
