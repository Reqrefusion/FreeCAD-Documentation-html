Beschreibung

Der Befehl SheetMetal VonFestkörper wandelt einen Festkörper oder ein Schalenobjekt in ein Blechobjekt um (eingeführt in Version 0.8.0).

Anwendung

  1. Select a base object (a solid or a shell).
  2. There are several ways to invoke the command:
    • Press the Convert to Sheet Metal button.
    • Select the Sheet Metal → Convert to Sheet Metal option from the menu.
    • Right-click in the Tree View or the 3D View and select the Sheet Metal → Convert to Sheet Metal option from the context menu.
    • Use the keyboard shortcut: C then S.
  3. A SolidToSheet object is created, and the Convert to Sheet Metal Task Panel opens:
    The Faces to remove and seam edges list is empty if the base object was selected in the Tree View, or it contains the subelements that were used to select the base object in the 3D View.
    • Adjust the parameters in the Task Panel.
    • Optionally press the Select button to edit the Faces to remove and seam edges list.
      (A Preview button indicates the list edit mode)
      • Optionally select faces/edges in the 3D View to add them to the list, or remove them respectively.
      • Optionally press the Clear All to remove all items from the Faces to remove and seam edges list.
      • Optionally check list items and press the Clear Selected to remove only the selected items from the Faces to remove and seam edges list.
      • Press the Preview button to confirm and finish editing.
  4. Press the OK button to close the Task Panel and finish the command.
  5. Optionally adjust the profile parameters in the Property View.

Aufgaben-Fenster

Ein vorhandenes SolidToSheet-Objekt in der Baumansicht doppelt anklicken, um das Aufgaben-Fenster erneut zu öffnen und die Parameter zu bearbeiten.

File:SheetMetal FromSolid-Task.png

  • Thickness: Eingabe für die Eigenschaft Thickness.
  • Bend Radius: Eingabe für die Eigenschaft Radius.
  • Invert: Schaltet die Eigenschaft Invert um; .
  • Select: Aktiviert den Bearbeitungsmodus, der ermöglicht die Liste der zu entfernenden Flächen bzw. Stoßkanten innerhalb der Eigenschaft base Object hinzuzufügen. Wenn gedrückt:
    • Preview: Zeigt den Bearbeitungsmodus an. Drücken, um den Bearbeitungsmodus zu beenden (ausgeblendet bis Select gedrückt wurde).
    • Clear All: Drücken, um die Liste zu leeren (ausgeblendet bis Select gedrückt wurde). Wandelt sich zu Clear Selected, wenn wenigstens ein Listenelement aktiviert wurde; entfernt nur die ausgewählten Elemente.

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

Ein SheetMetal-SolidToSheet-Objekt wird von einem Part-Formelement abgeleitet oder, wenn es sich in einem PartDesign-Körper befindet, von einem PartDesign Formelement und erbt alle seine Eigenschaften. Außerdem hat es die folgenden zusätzlichen Eigenschaften:

Daten

Parameters

  • DatenInvert (Bool): "Biegerichtung umkehren". Steuert, ob die Wandstärke auf der Innen- oder der Außenseite des Basisobjekts aufgebaut wird.
  • DatenRadius (Length): Innenradius der automatisch hinzugefügten Bögen.
  • DatenThickness (Length): Wandstärke der Blechform.
  • Datenbase Object (LinkSub): "Basisobjekt". Verknüpfung zu dem Festkörper bzw. dem Schalenobjekt und seinen ausgewählten Unterelementen.
