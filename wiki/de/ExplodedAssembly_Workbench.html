Einleitung

Der Arbeitsbereich ExplodedAssembly ist ein externer Arbeitsbereich zum erstellen von Explosionsansichten und Animationen von Baugruppen.

Funktionsumfang

  • Graphische Erstellung der Explosionsansichten von Baugruppen (ganz ohne Programmieren!)
  • Erstellung einzelner Explosionsansichten von Unterbaugruppen
  • Drehung von Schrauben und Muttern für realistische Montage bzw. Demontage
  • Mit den mitgelieferten HIlfs-Zusammenbauwekzeuge lassen sich Einzelteile zueinander Positionieren
  • Funktion in Arbeit: Erstellung eines Bewegungsablaufs aus Linienzügen und Skizzen

Installation

Automatische Installation

Dieser Arbeitsbereich kann mit dem Addon-Manager installiert werden.

Von GitHub

Verwendung von Git unter Ubuntu und Mint:

  • Die Eingabeaufforderung (Terminal) mit den Tasten Strg+Alt+t öffnen
  • Git installieren: sudo apt-get install git
  • Das Repository klonen: git clone https://github.com/JMG1/ExplodedAssembly ~/.FreeCAD/Mod/ExplodedAssembly

Das war's schon. Wenn man FreeCAD das nächste Mal startet, sollte der Arbeitsbereich verfügbar sein.

Um manuell zu installieren, lädt man dieses Repository als ZIP herunter und:

  • Für Ubuntu, Mint und ähnliche Betriebssysteme extrahiert man es in: /home/username/.local/share/FreeCAD/Mod (0.20 und neuer) oder /home/username/.FreeCAD/Mod (0.19 und davor)
  • Für Windows extrahieren man es in: C:\Users\your_user_name\AppData\Roaming\FreeCAD\Mod
