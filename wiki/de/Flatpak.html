Installieren

Stabil

Wenn man die Installation auf Benutzerebene durchführen möchte oder man keine sudo-Rechte hat, fügt man den folgenden Befehlen das Flag --user hinzu. 

# add flathub repo just to be sure as it might not be enabled if it is your first time using flatpak
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
flatpak install flathub org.freecad.FreeCAD

Entwicklungsversionen

Wenn man die Installation auf Benutzerebene durchführen möchte oder man keine sudo-Rechte hat, fügt man den folgenden Befehlen das Flag --user hinzu. 

# flathub-beta repo is not enabled by default
flatpak remote-add --if-not-exists flathub-beta https://flathub.org/beta-repo/flathub-beta.flatpakrepo
flatpak install flathub-beta org.freecad.FreeCAD

Ausführen

Man kann Flatpak über die Desktop-Datei oder mit dem folgenden Befehl starten: 

flatpak run org.freecad.FreeCAD

Die verschiedenen Zweige können parallel installiert werden. Um auszuwählen, welcher ausgeführt werden soll, verwendet man das Flag --branch: 

flatpak run --branch=beta org.freecad.FreeCAD

Um eine bestimmte ausführbare Datei (z. B. FreeCADCmd, um ohne GUI auszuführen) aus Flatpak heraus auszuführen, verwendet man das Flag --command: 

flatpak run --command=FreeCADCmd org.freecad.FreeCAD
