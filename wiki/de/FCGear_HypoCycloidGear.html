Beschreibung

Der Befehl FCGear Hypozykloidgetriebe erstellt zwei splineförmige Nockenscheiben und einen Satz Rollen, die während der Bewegung mit den Außenflächen der Scheiben in Kontakt bleiben.

Links: Hypozykloidgetriebe. Rechts: Getriebe und transparent dargestellt, ebenfalls mit Rückwärtsgang und Rollensatz

Bitte beschreiben Sie kurz, was mit einem solchen Getriebe erreicht werden kann.

Anwendung

  1. Zum Arbeitsbereich Gear wechseln.
  2. Es gibt mehrere Möglichkeiten den Befehl aufzurufen:
    • Die Schaltfläche HypoCycloid Gear drücken.
    • Den MenüeintragGear → HypoCycloid Gear auswählen.
  3. Die Einstellungen den geforderten Randbedingungen entsprechend ändern (siehe Eigenschaften).

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

Ein FCGear-HypoCycloidGear-Objekt, wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem hat es die folgenden zusätzlichen Eigenschaften:

Daten

Disks

  • Daten-Eigenschaftdisk-height (Length): Der Standardwert ist 10 mm. Höhe.
  • Daten-Eigenschaftshow_disk0 (Bool): Standardwert ist true. Hauptkamera-Scheibe anzeigen.
  • Daten-Eigenschaftshow_disk1 (Bool): Standardwert ist true. Zeige eine weitere umgekehrte Nockenscheibe oben an.

Pins

  • Daten-Eigenschaftcenter_pins (Bool): Standardwert ist true. Zentrieren Sie die Z-Achse des Stifts auf die erzeugten Scheiben.
  • Daten-Eigenschaftpin-height (Length): Der Standardwert ist 20 mm. Höhe.
  • Daten-Eigenschaftshow_pins (Bool): Standardwert ist true. Stifte an Ort und Stelle erstellen.

gear_parameter

  • Dateneccentricity (Float): Default is 1.5. Eccentricity.
  • Datenhole_radius (Length): Default is 30 mm. Center hole's radius.
  • Datenpin_circle_radius (Length): Default is 66 mm. Pin ball circle radius (overrides Tooth Pitch).
  • Datenpressure_angle_lim (Angle): Default is 50°. Pressure angle limit.
  • Datenpressure_angle_offset (Float): Default is 0.01. Offest in pressure angle.
  • Datenroller_diameter (Float): Default is 3.0. Roller diameter.
  • Datensegment_count (Integer): Default is 42. Number of points used for spline interpolation.
  • Datenteeth_number (Integer): Default is 42.. Number of teeth in cam.

version

  • Datenversion (String): Default is 1.3.0. freecad.gears-version

Hinweise

Die Standardausrüstung(en) wird/werden wie folgt angezeigt:

Um die Nockenscheiben und den Satz Rollen (Stifte) in verschiedenen Farben darzustellen, benötigen wir drei identische HypocycloidGear-Objekte. Ihre Sichtbarkeit kann umgeschaltet werden:

  • Daten-Eigenschaftshow_disk0 (Bool) für die Hauptnockenscheibe.
  • Daten-Eigenschaftshow_disk1 (Bool) für eine umgekehrte Nockenscheibe oben.
  • Daten-Eigenschaftshow_pins (Bool) für den Satz Stifte.
