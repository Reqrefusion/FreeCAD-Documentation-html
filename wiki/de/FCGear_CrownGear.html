Beschreibung

Der Befehl FCGear Kronenrad erstellt ein Kronenrad, ein CrownGear-Objekt, das auch die Größe des passenden Ritzels berücksichtigt.

Das Kronenrad ähnelt einer ringförmig gebogenen Zahnstange. Der Eingriffswinkel verringert sich kontinuierlich vom äußeren bis zum inneren Durchmesser. Damit wird die variable Umfangsgeschwindigkeit am Kronenrad zur konstanten Umfangsgeschwindigkeit des Ritzels ausgeglichen. Die spitzen Zahnaußenseiten und die steilen Zahnflanken am Innendurchmesser begrenzen die nutzbare Zahnbreite. Kronenräder sind ähnlich leistungsfähig wie Stirnräder. Ein Kronenrad kann mehrere Ritzel antreiben.

Bekannte Anwendungsgebiete von Kronenrädern:

  • Hinterachsantriebe für Autos und Motorräder
  • Schwenkmechanismus für Operationstische
  • Winkel-Fräsköpfe
  • Elektrowerkzeugsysteme mit mehreren Ritzeln und einem Zahnkranz

Oben: Kronenrad

Anwendung

  1. Zum Arbeitsbereich Gear wechseln.
  2. Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
    • Die Schaltfläche Kronenrad drücken.
    • Den Menüeintrag Gear → Kronenrad auswählen.
  3. Das Kronenrad wird standardmäßig ohne Zähne dargestellt. (eingeführt in 0.21)
  4. Die Einstellungen den geforderten Randbedingungen entsprechend ändern (siehe Eigenschaften).
  5. Die Daten-Eigenschaftpreview_mode auf false setzen, um die Zähne darzustellen(see Hinweise).

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

Ein FCGear-Kronenrad, ein CrownGear-Objekt, wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem hat es die folgenden zusätzlichen Eigenschaften:

Daten

accuracy

  • Daten-Eigenschaftnum_profiles (Integer): Der Standardwert ist 4. Anzahl der für die Ausformung verwendeten Profile.
  • Daten-Eigenschaftpreview_mode (Bool): Standardwert ist true.

base

  • Daten-Eigenschaftheight (Length): Der Standardwert ist 2 mm. Wert für die Zahnbreite.
  • Daten-Eigenschaftmodule (Length): Der Standardwert ist 1 mm. Der Modul ist das Verhältnis des Teilkreisdurchmessers des Zahnrads zur Zähnezahl (siehe Hinweise).
  • Daten-Eigenschaftnum_teeth (Integer): Der Standardwert ist 15. Zähnezahl.
  • Daten-Eigenschaftother_teeth (Integer): Der Standardwert ist 15. Zähnezahl des Konstruktionszahnrads (Ritzel) (siehe Hinweise).
  • Daten-Eigenschaftthickness (Length): Der Standardwert ist 5 mm. Höhe von der Zahnspitze bis zur Unterseite des Kronenrads.

involute

  • Daten-Eigenschaftpressure_angle (Angle): Standardwert 20°. Eingriffwinkel (siehe Hinweise).

version

  • Daten-Eigenschaftversion (String):

Hinweise

  • Die Eigenschaft Daten-Eigenschaftpreview_mode ist standardmäßig auf true gesetzt. Wenn das Zahnrad erstellt wird, wird in der Berichtsansicht folgende Meldung angezeigt:
    Zahnradmodul: Kronenzahnrad erstellt, preview_mode = true für verbesserte Leistung. Setzen man die Eigenschaft preview_mode auf false, wenn man bereit ist, die Zähne zu schneiden.
  • module: Gemäß den Richtlinien der ISO (Internationale Organisation für Normung) wird die Modulgröße als Einheit für die Zahnradgröße angegeben. Modul (m): m = 1 (p = 3,1416), m = 2 (p = 6,2832), m = 4 (p = 12,566). Wenn man das Modul mit Pi multipliziert, erhält man die Teilung (p). Die Teilung ist der Abstand zwischen entsprechenden Punkten auf benachbarten Zähnen.
  • other_teeth: Es können nur mehrere Ritzel mit derselben Zähnezahl auf einem Zahnkranz verwendet werden.
  • pressure_parameter: Ändern Sie den Parameter nur, wenn Sie über ausreichende Kenntnisse der Zahnradgeometrie verfügen.
  • Die Geometrie des Zahnkranzes wird in erster Linie durch die Geometrie des Stirnradritzels (other_teeth) bestimmt.
  • Erstellen Sie ein Stirnrad mit Evolventenzahnrad. Die Anzahl der Zähne muss mit dem Parameter other_teeth des Kronenrads identisch sein.
  • Mit den Parametern des Evolventenzahnrads können Anpassungen für optimale Laufeigenschaften vorgenommen werden.

Kronenrad-Stirnrad-Paarung im Überblick

  • (1) Stirnrad
  • (2) Kronenrad
  • (3) Zahnbreite
  • (4) Innendurchmesser
  • (5) Außendurchmesser

Nützliche Formeln

  • Innendurchmesser (4)
    • Innendurchmesser = Modul (Stirnrad) * Zähne (Kronenrad) * cos Druckparameter (Ritzel) : cos Druckparameter (Kronenrad)

Inner diameter (4)

  • inner diameter = module (spur gear) * teeth (crown gear) * cos pressure_paramter (pinion) : cos pressure_parameter (crown gear)

Skripten

Die Macht von Python nutzen, um das Erstellen von Zahnrädern zu automatisieren: 

import FreeCAD as App
import FreeCADGui as Gui
import freecad.gears.commands
gear = freecad.gears.commands.CreateCrownGear.create()
gear.num_teeth = 20
gear.other_teeth = 15
gear.height = 3
gear.preview_mode = False
App.ActiveDocument.recompute()
Gui.SendMsgToActiveView("ViewFit")
