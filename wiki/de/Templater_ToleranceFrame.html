Beschreibung

Das Werkzeug Templater Toleranzrahmen erstellt einen Toleranzrahmen oder einen Bezugsrahmen als SVG-Datei und fügt ihn in die ausgewählte Ansicht ein.

Toleranzrahmen und Bezugsrahmen mit editierbaren Einträgen

Anwendung

  1. Eine Ansicht oder beliebige Geometrie innerhalb der Ansicht auswählen.
  2. Es gibt mehrere Möglichkeiten das Werkzeug aufzurufen:
    • Die Schaltfläche Toleranzrahmen drücken.
    • Den Menüeintrag Templater → Toleranzrahmen auswählen.
  3. Das Aufgaben-Fenster Rahmeneinstellungen wird geöffnet.
  4. Eine Toleranzart für einen Toleranzrahmen auswählen oder einen einzelnen (großen) Buchstaben für einen Bezugsrahmen eingeben.
  5. Wenn eine Toleranzart ausgewählt wurde, die folgenden Werte anpassen:
    (sie werden für einen Bezugsrahmen ignoriert)
    • Wahlweise den Toleranzwert bearbeiten.
    • Wahlweise jedes der Bezugsfelder bearbeiten. Leer lassen, wenn es nicht benötigt wird; die nachfolgenden Felder werden ignoriert .
  6. Ok anklicken, um das Aufgaben-Fenster zu schließen und das Werkzeug zu beenden.
  7. Eine Datei NewSymbol.svg wird im Ordner .../Mod/Templater/Resources gespeichert und der ausgewählten Ansicht als ein neues FeatureFrame-Objekt hinzugefügt.
  8. Das FeatureFrame-Objekt auf seine endgültige Position ziehen.
  9. Wahlweise seine Daten-EigenschaftEditable Texts in der Eigenschaften-Ansicht anpassen.

Aufgaben-Fenster

  • Tolerance type: Sets the tolerance symbol in the first cell of a tolerance frame, or the single letter in the only cell of a datum frame.
  • Tolerance value: Sets the tolerance value in the second cell (and offers some predefined entries to have the diameter symbol at hand).
  • Datum1: Sets the first datum marker in the third cell. Values in following cells will be ignored if empty.
  • Datum2: Sets the second datum marker in the fourth cell. A value in following cell will be ignored if empty.
  • Datum3: Sets the third datum marker in the fifth cell.

Hinweise

  • Die Weite der Zellen werden beim Erstellen berechnet und können nachträglich nicht verändert werden.
  • Alle Texte in den Zellen können editiert werden; im Gegensatz dazu sind die Toleranzsymbole fest programmierte SVG-Elemente und können weder bearbeitet noch ausgetauscht werde.

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

Ein FeatureFrame-Objekt wird von einem TechDraw Symbol, formell ein TechDraw::DrawViewSymbol-Objekt abgeleitet und erbt alle seine Eigenschaften.
