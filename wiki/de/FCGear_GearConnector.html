Beschreibung

Der Befehl FCGear Zahnradpaarung kombiniert zwei Zahnrad-Elemente …

Anwendung

  1. Zum Arbeitsbereich Gear wechseln.
  2. Zwei Elemente mit passender Verzahnung auswählen. (Siehe Hinweise):
    • Das erste muss ein Stirnrad oder ein Hohlrad sein.
    • Das zweite kann ein Stirnrad oder eine Zahnstange sein.
  3. Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
    • Die Schaltfläche Zahnradpaarung drücken.
    • Den Menüeintrag Gear → Zahnradpaarung auswählen.
  4. Die Einstellungen den geforderten Randbedingungen entsprechend ändern (siehe Eigenschaften).

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

An FCGear GearConnector object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Daten

gear

  • Datenangle1 (Angle): Default is 0°. Angle at which second gear is placed.
  • Datenangle2 (Angle): Default is 0°. Angle at which second gear is placed.
  • Datenmaster_gear (Link): Master gear.
  • Datenmaster_gear_stationary (Bool): Default is True. Master gear position is fixed (does not orbit).
  • Datenslave_gear (Link): Slave gear.
  • Datenslave_gear_stationary (Bool): Default is True. Slave gear position is fixed (does not orbit).

version

  • Datenversion (String): The freecad.gears-version.

Hinweise

Die Reihenfolge der Auswahl bestimmt, ob das zweite Element auf dem ersten abrollen kann, wenn die Daten-Eigenschaftangle1 geändert wird.
