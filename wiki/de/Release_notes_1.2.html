Arbeitsbereich FEM

Further FEM improvements

  • The Electric Charge Density load now has a Concentrated checkbox in the Total Source mode to use concentrate instead of distributed load (making it also applicable to edges and vertices) with CalculiX. Pull request #25237
  • The FEM results objects now support the animation of half cycles in addition to reversed full cycles. Pull request #24129
  • The Section Print feature now supports 2D models and electric flux in electrostatic analyses. Pull request #25081
