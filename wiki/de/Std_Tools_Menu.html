Einleitung

Das Std Menü Werkzeuge ist eines der 7 Untermenüs im Standardmenü:

Datei     Bearbeiten     Ansicht     Werkzeuge     Makro     Fenster     Hilfe

Das Menü Werkzeuge stellt Werkzeuge bereit zur Fehlersuche von Modellen, zum Anpassen des Verhaltens von FreeCAD sowie Hilfswerkzeuge.

Werkzeuge

Die folgenden Werkzeuge sind in diesem Menü verfügbar:

The order of the tools matches the version 1.1 menu.

  • Addon-Manager: Öffnet den Addon-Manager, mit dem externe Arbeitsbereiche, Voreinstellungspakete und Makros installiert werden können.
  • Messen: MIsst Objekte eingeführt in 1.0
  • Einheiten-Umrechner...: Öffnet einen Rechner für die Umrechnung von Maßeinheiten.
  • Bild laden...: Lädt ein Bild (zur Ansicht). eingeführt in 0.21
  • Bild speichern...: Speichert ein Bildschirmfoto der aktiven 3D-Ansicht.
  • : Textdokument hinzufügen: Erstellt ein Objekt, das Text enthalten kann.
  • Drehscheibe...: Dreht die Kamera in der aktiven 3D-Ansicht kontinuierlich.
  • Szenengraph untersuchen...: Zeigt eine Übersicht der Szenengraph-Knoten der aktiven 3D-Ansicht an.
  • Abhängigkeitsdiagramm...: Zeigt das Abhängigkeitsdiagramm von Objekten an.
  • Abhängigkeitsdiagramm exportieren... eingeführt in 0.21
  • Dokument-Dienstprogramm...: Hilft bei der Reparatur beschädigter Projektdateien.
  • Parameter bearbeiten...: Öffnet ein Dialogfeld mit Zugriff auf die Parameter, die das Programm steuern.
  • Anpassen...: Öffnet ein Dialogfeld mit Anpassungsoptionen.
