Beschreibung

Das Werkzeug Curves Fläche abwickeln erstellt ebene Abwicklungen von konischen oder zylindrischen Flächen.

Ein Kegelstumpf und zwei Abwicklungen (Flatten-Objekte), eine "in place" (grün), und eine auf der XY-Ebene (rot)

Anwendung

  1. Eine oder mehrere konische oder zylindrische Flächen auswählen.
  2. Es gibt mehrere Möglichkeiten, das Werkzeug aufzurufen:
    • Die Schaltfläche Fläche abwickeln drücken.
    • Die Option Surfaces → Fläche abwickeln aus dem Menü wählen.
  3. Ein Flatten-Objekt wird für jede ausgewählte Fläche erstellt.

Eigenschaften

Siehe auch: Eigenschafteneditor.

Ein Flatten-Objekt wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem besitzt es die folgenden zusätzlichen Eigenschaften:

Daten

Settings

  • Daten-EigenschaftInPlace (Bool): Wickelt die Fläche an Ort und Stelle ab.
    • Template:Wahr: Die Ausgangsfläche wird so platziert, dass sie tangential zur Quellfläche an der Nahtlinie verläuft.
    • Template:Falsch: Die Ausgabebene befindet sich in der Standard-XY-Ebene.
  • Daten-Eigenschaft (versteckt)Size (Float): Größe der darunterliegenden Fläche

Source

  • Daten-EigenschaftSource (LinkSub): Die konische Fläche zum Abflachen
