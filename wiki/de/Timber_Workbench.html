Einleitung

Eine ergänzender Arbeitsbereich für FreeCAD, dessen Hauptziel es ist, Werkzeuge zur Erleichterung der Modellierung von Holzkonstruktionen (Rahmen, Holzrahmen usw.) bereitzustellen.

Referenzen

  • Autor: Jonathan Wiedemann wood-galaxy
  • Homepage: https://github.com/wood-galaxy
  • Quellcode auf GitHub: https://github.com/wood-galaxy/FreeCAD-Timber

Werkzeuge

Detaillierte Beschreibung: https://forum.freecad.org/viewtopic.php?t=12559

Werkzeugleiste

Timber-Werkzeuge

  • Aufteilung: Achse oder strukturelles System entlang einer Linie erstellen
  • Markierung: Ausgewählten Objekten eine Markierung hinzufügen

Timber-Auflistung

  • Auflistung: Objekte auflisten

Timber-Träger

  • Timber Träger: Erstellt ein Strukturobjekt von Grund auf oder aus einem ausgewählten Objekt (Skizze, Linienzug, Fläche oder Volumenkörper).
  • Bearbeitungsschnitt: Erstellt eine Schnittbearbeitung an einem Holzbalkenobjekt, relativ zu einer Ebene (Fläche).
