Beschreibung

Der Befehl Web BrowserURLBearbeiten ändert die URL des aktiven Web-Reiters.

Anwendung

  1. Ein Web-Reiter sollte aktiv sein; falls erforderlich, verwendet man Web OpenWebseiteÖffnen, um einen zu erstellen.
  2. Die Schaltfläche URL festlegen drücken.
  3. Eine Eingabezeile mit der aktuellen URL wird geöffnet.
  4. Die URL bearbeiten und zum Fertigstellen Enter drücken.
  5. Die neue URL wird geladen.
