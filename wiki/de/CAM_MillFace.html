Beschreibung

Der Befehl CAM Fläche erstellt einen Pfad zur Durchführung einer Planfräsbearbeitung auf einer horizontalen Fläche. Dieser Vorgang wird im Allgemeinen verwendet, um

  • eine raue Werkstückoberfläche zu glätten,
  • ausgewählte Flächen auf die gewünschte Tiefe zu fräsen, um nachfolgende Räumvorgänge innerhalb der von diesem Vorgang betroffenen Bereiche vorzubereiten,
  • oder um eine Endbearbeitungsoberfläche auf die ausgewählten Flächen aufzubringen.

Dieser Vorgang enthält eine Daten-EigenschaftBoundaryShape-Eigenschaft, die eine modifizierte Auswahlfläche basierend auf den ausgewählten Flächen ermöglicht.

Anwendung

  1. Select one or more faces to be surfaced. Note: If selected faces exist at different heights, then all will be milled to the Final Depth setting.
  2. There are several ways to invoke the command:
    • Press the Face button.
    • Select the CAM → Face option from the menu.
  3. Select the correct DatenBoundaryShape setting to modify the milling area based on the face(s) selected as Base Geometry.
  4. Adjust the other properties as needed. They are listed further below.

Vorbehalte

  • Die Verwendung auf einer schrägen Ebene kann zu unerwarteten Ergebnissen führen: Es wird dennoch eine Bahn zum Schneiden einer horizontalen Fläche erzeugt. Der Umfang des Schnitts entspricht der vertikalen Projektion der schrägen Ebene, ausgeführt in einer Höhe, die dem tiefsten Punkt dieser Ebene entspricht.
  • Da die CAM-Werkzeuge nur auf die Geometrie der ausgewählten Kanten/Flächen wirken und keinen Bezug zum Rest des 3D-Modells haben, gehen die Pfade nicht über die Grenzen der ausgewählten Ebene hinaus, selbst wenn diese von ungenutztem Material oder Luft umgeben ist. Dadurch bleiben unbearbeitete Ecken zurück. Diese können manchmal mit einem der Bearbeitungswerkzeuge aus dem Menü CAM entfernt werden.

Vertikalflächenfräsen

  • Dieses Werkzeug funktioniert nicht auf einer vertikalen Ebene oder einer vertikalen, nicht ebenen Oberfläche. Vertikale Bearbeitungen können mit dem Werkzeug Flächen-Profil oder Kanten-Profil durchgeführt werden. Dazu muss eine Fläche oder eine geschlossene Kantenlinie ausgewählt werden, die die Ober- oder Unterkante der gewünschten vertikalen Fläche einschließt. Die Ausdehnung des Pfades kann dann mit dem Werkzeug Aufbereitung Begrenzungspfad aus dem Menü CAM reduziert werden. Mit dem Werkzeug Dress-up die Option Create Box wählen und die Größe reduzieren, um den Umfang des Profilpfads zu begrenzen. Mit diesen Einstellungen kann jedoch der Ursprung des Begrenzungsrahmens nicht verschoben werden. Dies muss durch Anpassen der Positionierungseinstellungen in der Baumansicht erfolgen.
  • Dies funktioniert auf zusammengesetzten Oberflächen wie mehreren vertikalen Ebenen oder zylindrischen Oberflächen, die miteinander verbunden sind, solange sie eine durchgehende Oberfläche bilden.

Eigenschaften

Hinweis : Die Namen einiger Eigenschaften in dieser Liste unterscheiden sich geringfügig von den gleichen Einstellungen, die im Aufgabenfenster-Editor verwendet werden.

Daten

Base

Hinweis: Es wird empfohlen, die Eigenschaft Placement von Pfadbearbeitungen nicht zu bearbeiten. Stattdessen verschiebt oder dreht man das CAM-Auftragsmodell nach Bedarf.

  • DatenPlacement: Overall placement [position and rotation] of the object - with respect to the origin (or origin of parent object container)
    • DatenAngle: Angle in degrees applied to rotation of the object around Axis property value
    • DatenAxis: Axis (one or multiple) around which to rotate the object, set in sub-properties: X, Y, Z
      • DatenX: X-axis value
      • DatenY: Y-axis value
      • DatenZ: Z axis value
    • DatenPosition: Position of the object, set in sub-properties: X, Y, Z - with respect to the origin (or origin of parent object container)
      • DatenX: X-distance value
      • DatenY: Y-distance value
      • DatenZ: Z-distance value
  • DatenLabel: User-provided name of the object (UTF-8)

Depth

  • DatenClearance Height: The height needed to clear clamps and obstructions
  • DatenFinal Depth: Final Depth of Tool- lowest value in Z
  • DatenFinish Depth: Maximum material removed on final pass.
  • DatenSafe Height: The above which Rapid motions are allowed.
  • DatenStart Depth: Starting Depth of Tool- first cut depth in Z
  • DatenStep Down: Incremental Step Down of Tool

Face

  • Daten-EigenschaftBoundaryShape: Form zur Berechnung der Grenze
  • Daten-EigenschaftClearEdges: Kanten der Oberfläche bereinigen (nur für die Begrenzungsbox anwendbar)
  • Daten-EigenschaftExcludeRaisedAreas: Erhabene Bereiche innerhalb der Fläche ausfräsen
  • Daten-EigenschaftOffset Pattern: Zu verwendendes Bereinigungs-Muster. (Auswählen, auf welche Weise die horizontalen Bewegungen ausgeführt werden sollen.)

Path

  • Daten-EigenschaftActive: Falsch setzen, um zu verhindern, dass der Vorgang Code generiert
  • Daten-EigenschaftBase: Die Basisgeometrie für diese Bearbeitung
  • Daten-EigenschaftComment: Ein optionaler Kommentar zu dieser Bearbeitung
  • Daten-EigenschaftCoolant Mode: Der Kühlmittelmodus für diese Bearbeitung
  • Daten-EigenschaftCycle Time: Die Zykluszeitabschätzung für diese Bearbeitung
  • Daten-EigenschaftTool Controller: Definiert die in der Bearbeitung verwendete Werkzeugsteuerung
  • Daten-EigenschaftUser Label: Vom Benutzer zugewiesene Bezeichnung

Pocket

  • DatenCut Mode: The direction that the toolpath should go around the part ClockWise (CW) or CounterClockWise (CCW)
  • DatenExtra Offset: Extra offset to apply to the operation. Direction is operation dependent.
  • DatenStartAt: Start pocketing at center or boundary
  • DatenStep Over: Percent of cutter diameter to step over on each pass
  • DatenZig Zag Angle: Angle of the zigzag pattern
  • DatenOffset Pattern: Clearing pattern to use
  • DatenMin Travel: Use 3D Sorting of Path
  • DatenKeep Tool Down: Attempts to avoid unnecessary retractions.

Rotation

  • DatenAttempt Inverse Angle: Automatically attempt Inverse Angle if initial rotation is incorrect.
  • DatenEnable Rotation: Enable rotation to gain access to pockets or areas not normal to Z axis.
  • DatenInverse Angle: Inverse the angle of the rotation. Example: change a rotation from -22.5 to 22.5 degrees.
  • DatenLimit Depth To Face: Enforce the Z-depth of the selected face as the lowest value for final depth. Higher user values for final depth will be observed.
  • DatenReverse Direction: Reverse orientation of Operation by 180 degrees.

Start Point

  • Daten-EigenschaftStart Point: Der benutzerdefinierte Startpunkt für den Pfad dieses Vorgangs.
    • Daten-EigenschaftX: X-Abstandswert
    • Daten-EigenschaftY: Y-Abstandswert
    • Daten-EigenschaftZ: Z-Abstandswert
  • Daten-EigenschaftUse Start Point: Diese Option auf Wahr setzen, wenn man einen Startpunkt manuell festlegen möchte. Den Startpunkt im Feld Startpunkt der Eigenschaftsdaten festlegen.
