Beschreibung

The standard plot module already provides a basic tool to control the plot axes . But that tool is clearly insufficient when multi-axes plots have to be handled, as is the case in the Multi-axes plot tutorial. To overcome that limitation you can install the Plot Workbench using the Add-on manager, so a more complete tool to edit the plot axes will be available.

Anwendung

Die Plot-Registerkarte auswählen, die man bearbeiten möchte, und dann dieses Werkzeug ausführen.

Es wird dringend empfohlen, mit der Skalierung des gesamten Darstellung zu beginnen, damit man sicher sein kann, dass es in den verfügbaren Platz passt. Dazu die Option Auf alle Achsen anwenden aktivieren.

Auswahl auf alle Achsen anwenden

Anschließend kann man die gesamte Grafik mit den vier Schiebereglern so skalieren, dass sie in den sichtbaren Bereich passt.

Schieberegler zum Skalieren der Darstellung

Wenn man mit der allgemeinen Größe des Plots zufrieden ist, kann man die Option Auf alle Achsen anwenden deaktivieren und jede Achsengruppe separat feinabstimmen. Dazu wählt man einfach oben die Achsengruppe aus, die man bearbeiten möchte.

Auswahl für den zu bearbeitenden Achsensatz

Man kann erneut die Schieberegler verwenden, um den von diesem Teilplot abgedeckten Bereich zu steuern. Man kann auch steuern, wo die Markierungen und Titel für die X-Achse und die Y-Achse platziert werden.

Editor für Positionen der ausgewählten Achsen

Genauer gesagt man kann festlegen, ob die X-Achse unterhalb oder oberhalb des Diagramms angezeigt wird und ob die Y-Achse links oder rechts vom Diagramm angezeigt wird. Man kann sogar den Versatz beider Achsen in Bezug auf die Grundlinie des Diagramms festlegen.

Schließlich kann man die für jeden Achsensatz berücksichtigten Minimal- und Maximalwerte festlegen, den sogenannten Zoom.

Editor für minimale und maximale Betrachtungswerte
