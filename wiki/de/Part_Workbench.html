Einleitung

Der Arbeitsbereich Part ermöglicht einen herkömmlichen Arbeitsablauf entsprechend der Methodik konstruktive Festkörpergeometrie (engl.: Constructive Solid Geometry, kurz CSG). In diesem Arbeitsablauf ist jedes Objekt ein unabhängiger Festkörper. Der Arbeitsbereich Part kann diese aus parametrisch festgelegten Skizzen und dem Einsatz von Werkzeugen wie Extrudieren, Drehen, Ausformung usw. erstellen. Außerdem können auch Grundkörper wie Quader, Zylinder usw. erstellt werden. Diese Objekte können durch boolesche Verknüpfungen kombiniert werden, um komplexere Festkörper zu erstellen.

Der Arbeitsbereich Part kann auch Objekte erstellen, die keine Festkörper sind wie Flächen, Hüllen oder Objekte, die nur aus Kanten und Punkten bestehen. Er bietet außerdem eine Vielzahl von Allzweckwerkzeugen für die Geometriebearbeitung, Geometrievalidierung und das Erstellen von Kopien.

Der Arbeitsbereich PartDesign verwendet einen alternativen Arbeitsablauf zum Erstellen von Festkörpern. Eine detaillierte Diskussion der Arbeitsbereiche Part und PartDesign im Vergleich befindet sich unter Part und PartDesign.

Symbolleiste Festkörper

  • Quader: Erstellt einen Quader.
  • Zylinder: Erstellt einen Zylinder.
  • Kugel: Erstellt eine Kugel.
  • Kegel: Erstellt einen Kegel.
  • Torus: Erstellt einen Torus.
  • Hohlzylinder erstellen: Erstellt einen Hohlzylinder.
  • Grundkörper: Ein Werkzeug zum Erstellen eines der folgenden Grundelemente:
  • Ebene: Erstellt eine Ebene.
  • Quader: Erstellt einen Quader. Dieses Objekt kann auch mit dem Werkzeug Quader erstellt werden.
  • Zylinder: Erstellt einen Zylinder. Dieses Objekt kann auch mit dem Werkzeug Zylinder erstellt werden.
  • Kegel: Erstellt einen Kegel. Dieses Objekt kann auch mit dem Werkzeug Kegel erstellt werden.
  • Kugel: Erstellt eine Kugel. Dieses Objekt kann auch mit dem Werkzeug Kugel erstellt werden.
  • Ellipsoid: Erstellt einen Ellipsoid.
  • Torus: Erstellt einen Torus. Dieses Objekt kann auch mit dem Werkzeug Torus erstellt werden.
  • Prisma: Erstellt ein Prisma.
  • Keil: Erstellt einen Keil.
  • Helix: Erstellt eine Wendel (Helix).
  • Spirale: Erstellt einen Spirale.
  • Kreis: Erstellt einen kreisförmigen Bogen.
  • Ellipse: Erstellt einen elliptischen Bogen.
  • Punkt: Erstellt einen Punkt (Knoten).
  • Linie: Erstellt eine Linie.
  • Regelmäßiges Polygon: Erstellt ein regelmäßiges Vieleck (Polygon).
  • Formgenerator: Erstellt Formen aus verschiedenen Grundelementen.

Symbolleiste Part-Werkzeuge

  • Neue Skizze: Erstellt eine neue Skizze und öffnet den Sketcher-Dialog, um sie zu bearbeiten.
  • Extrudieren: Extrudiert ebene Flächen.
  • Drehen: Erstellt einen Festkörper, indem ein Objekt (das kein Festkörper ist) um eine Achse gedreht wird.
  • Spiegeln: Spiegelt das ausgewählte Objekt über eine Spiegelebene.
  • Skalieren: Skaliert eine oder mehrere Formen. eingeführt in 1.0
  • Verrundung: Verrundet die Kanten eines Objekts.
  • Fase: Erstellt Fasen an den Kanten eines Objekts.
  • Fläche aus Kantenzügen: Erstellt eine Fläche aus einer Reihe von Kantenzügen (Konturen).
  • Regelfläche: Erstellt eine Regelfläche.
  • Ausformung: Erstellt eine Ausformung von einem Profil zu einem anderen.
  • Austragung: Erstellt eine Austragung eines oder mehrerer Profile entlang eines Pfades.
  • Schnittkurve: Erstellt Schnittkurven durch Überschneiden eines Objekts mit einem anderen Objekt.
  • Querschnitte: Erzeugt einen oder mehrere Querschnitte durch ein Objekt.
  • Versatz:
  • 3D-Versatz: Erstellt eine parallele Form in einem bestimmten Abstand zum Original.
  • 2D-Versatz: Erstellt einen parallelen Kantenzug in einem bestimmten Abstand zum Original oder vergrößert/verkleinert eine ebene Fläche.
  • Dicke: Höhlt einen Festkörper aus.
  • Auf Oberfläche projizieren: Projiziert ein Logo, einen Text oder eine beliebige Fläche, Kantenzug, Kante auf eine Oberfläche.
  • Aussehen flächenweise festlegen: Weist einzelnen Flächen von Objekten Farben zu.

Symbolleiste Boolesche Verknüpfung

  • Verbund:
  • Verbund: Erstellt einen Verbund aus den ausgewählten Objekten.
  • Verbundobjekt sprengen: Teilt Verbundobjekte auf.
  • Verbundfilter: Entnimmt die einzelnen Stücke aus Verbundobjekten.
  • Boolesche Verknüpfungen: Führt boolesche Verknüpfungen mit zwei Objekten durch.
  • Differenz: Beschneidet ein Objekt mit einem anderen.
  • Vereinigung: Vereinigt zwei oder mehr Objekte.
  • Schnitt: Entnimmt den gemeinsamen Teil (Schnittmenge) von zwei Objekten.
  • Objekte verbinden:
  • Formen verbinden: Vereinigt die umschlossenen Volumen von Hohlobjekten.
  • Formen einbetten: Bettet ein Hohlobjekt in ein anderes Hohlobjekt ein.
  • Form ausschneiden: Erzeugt einen Ausschnitt in einer Wand eines Objekts für ein anderes Hohlobjekt.
  • Aufteilung:
  • BoolescheFragmente: erzeugt alle Teile, die durch boolesche Operationen erhalten werden.
  • Wegschneiden: Zerschneidet und teilt ein Objekt, indem es mit anderen Objekten geschnitten wird.
  • Schneiden zu Verbund: Zerschneidet ein Objekt, indem es mit anderen Objekten geschnitten wird.
  • Boolesches exklusives Oder (XOR): Entfernt Bereiche, die zu einer geraden Anzahl von Objekten gehören.
  • Geometrie prüfen: Prüft die Geometrie von ausgewählten Objekte auf Fehler.
  • Defeaturing: Entfernt Formelemente aus einem Objekt.

Andere Werkzeuge

  • Rechteckauswahl: Wählt Flächen mit einem Auswahlrechteck aus.
  • Form aus Netz: Erstellt Formen aus Netzobjekten.
  • Punkte aus Form: Erstellt Punktobjekte aus geometrischen Objekten.
  • In Festkörper konvertieren: Erstellt Festkörper aus Formobjekten.
  • Formen umkehren: Erstellt parametrische Kopien mit umgekehrten Flächennormalen aus ausgewählten Objekten.
  • Kopie:
  • Einfache Kopie: Erstellt nichtparametrische Kopien ausgewählter Objekte.
  • Transformierte Kopie erstellen: Erstellt nichtparametrische Kopien von Objekten. Dies ist gedacht für Objekte, die in Behältern eingebettet sind.
  • Formelement-Kopie erstellen: Erstellt nichtparametrische Kopien von Unterelementen: Knoten, Kanten und Flächen.
  • Form aufbereiten: Erstellt aus ausgewählten Objekten parametrische Kopien mit aufbereiteten Formen. Dies entfernt überflüssige Kanten von ebenen und zylindrischen Flächen.
  • Toleranzangaben: Erstellt eine parametrische Kopie ausgewählter Objekte, wobei alle enthaltenen Toleranzen auf mindestens einen bestimmten Mindestwert gesetzt werden. eingeführt in 1.1
  • Dauerhafte Schnittdarstellung: Erzeugt eine dauerhafte Schnittdarstellung von Objekten und Baugruppen.
  • Befestigung: Befestigt ein Objekt an ein oder mehrere andere Objekte.

Veraltete Werkzeuge

  • Aus CAD-Datei importieren...: Importiert aus *.IGES-, *.STEP-, oder *.BREP-Dateien. Steht in 1.1 und neuer nicht mehr zur Verfügung.
  • In CAD-Datei exportieren...: Exportiert in *.IGES-, *.STEP-, oder *.BREP- Dateien. Steht in 1.1 und neuer nicht mehr zur Verfügung.

Messwerkzeuge

Das Werkzeug Std Measure ersetzt die folgend gelisteten Werkzeuge. eingeführt in 1.0

  • Linear messen Erzeugt eine lineare Messung. Steht in 1.0 und neuer nicht mehr zur Verfügung.
  • Winkel messen Erzeugt eine Winkelmessung. Steht in 1.0 und neuer nicht mehr zur Verfügung.
  • Messung aktualisieren Aktualisiert alle Messungen. Steht in 1.0 und neuer nicht mehr zur Verfügung.
  • Alles löschen und Messung löschen Löscht alle Messungen. Steht in 1.0 und neuer nicht mehr zur Verfügung.
  • Messung AlleUmschalten und Ansicht Messen alle löschen: Zeigt alle Messungen an oder blendet sie aus. Steht in 1.0 und neuer nicht mehr zur Verfügung.
  • 3D umschalten Zeigt 3D-Messungen an oder blendet sie aus. Steht in 1.0 und neuer nicht mehr zur Verfügung.
  • Delta umschalten Zeigt oder blendet Delta-Messungen aus. Steht in 1.0 und neuer nicht mehr zur Verfügung.
