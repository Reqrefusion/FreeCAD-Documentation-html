Vektoren werden überall in FreeCAD benutzt.

Beispiel: 

v=FreeCAD.Vector()
v=FreeCAD.Vector(1,0,0)
v=FreeCAD.Base.Vector()
v2 = FreeCAD.Vector(3,2,-5)
v3 = v.add(v2)
print v3.Length
Length

Returns: Gibt die Länge des Vektors zurück.

add(Vector)

Description: Addiert einen weiteren Vektor zu diesem Vektor.

Returns: Vektor

cross(Vector)

Description: Das Kreuzprodukt aus diesem Vektor und einem weiteren Vektor.

Returns: Vektor

distanceToLine(Vector1,Vector2)

Description: Der Abstand zwischen dem Vektor und einer Linie durch Vector1 in Richtung Vector2.

Returns: Gleitkommazahl

distanceToPoint(Vector)

Description: Der Abstand zwischen diesem Vektor und einem weiteren Vektor.

Returns: Gleitkommazahl

distanceToLineSegment(Vector1,Vector2)

Description: Ein Vektor zum Punkt mit dem geringsten Abstand auf einem Linienabschnitt von Vector1 zu Vector2.

Returns: Vektor

distanceToPlane(Vector1,Vector2)

Description: Der Abstand zwischen dem Vektor und einer Ebene, die durch einen Punkt und eine Normale festgelegt wird.

Returns: Gleitkommazahl

dot(Vector)

Description: Das Skalarprodukt aus 2 Vektoren.

Returns: Gleitkommazahl

getAngle(Vector)

Description: Der Winkel in Bogenmaß (rad) zwischen diesem Vektor und edinem weiteren Vektor.

Returns: Gleitkommazahl

isEqual(Vector2,tolerance)

Description: Prüft, ob der Abstand zwischen den Punkten, die durch diesen Vektor und Vector2 bestimmt werden, kleiner als oder gleich groß wie die angegebene Toleranz ist.

Returns: True/False

isNormal(Vector2,tolerance)

Description: Prüft, ob dieser Vektor innerhalb der angegebenen Toleranz normal zu Vector2 ist.

Returns: True/False

isOnLineSegment(Vector1,Vector2)

Description: Prüft, ob sich dieser Vektor auf dem Linienabschnitt befindet, der durch Vector1 und Vector2 gebildet wird.

Returns: Vektor

isParallel(Vector2,tolerance)

Description: Prüft, ob dieser Vektor innerhalb der angegebenen Toleranz parallel zu Vector2 verläuft.

Returns: True/False

multiply(Float)

Description: Multipliziert einen Vektor (gleichförmig) mit dem angegebenen Faktor.

Returns: nothing

negative(Vector)

Description: Gibt den Gegenvektor zu diesem Vektor zurück (negiert diesen Vektor).

Returns: Vektor

normalize( )

Description: Normalisiert einen Vektor (setzt seine Länge auf 1,0).

Returns: nichts

projectToLine(Vector1,Vector2)

Description: Projiziert den Vektor auf eine Linie durch Vector1 in Richtung Vector2.

Returns: nichts

projectToPlane(Vector1,Vector2)

Description: Projiziert den Vektor auf eine Ebene, die durch einen Punkt (Vector1) und einer Normalen (Vector2) gebildet wird.

Returns: nichts

scale(Float,Float,Float)

Description: Gleicht dem Multiplizieren, lässt aber unterschiedliche Werte für die X-, Y- und Z-Richtung zu (nicht gleichförmiges Skalieren)

Returns: nichts

sub(Vector)

Description: Zieht einen anderen Vektor von diesem Vektor ab.

Returns: Vektor

x

Returns: Die X-Koordinate eines Vektors.

y

Returns: Die Y-Koordinate eines Vektors.

z

Returns: Die Z-Koordinate eines Vektors.
