Beschreibung

Der Befehl CAM Taschenform erzeugt eine Taschenbearbeitung von ausgewählten Bodenflächen oder Wänden einer oder mehrerer Taschen des Basisobjekts Auftrags.

Das CAM-Pocket Shape-Objekt ist als Teil eines CAM Auftrags konzipiert.

Anwendung

  1. Select the bottom or the wall(s) of a pocket. While it is usually easier to select the bottom, the walls have to be selected when a pocket goes through all.
  2. There are several ways to invoke the command:
    • Press the Pocket Shape button.
    • Select the CAM → Pocket Shape option from the menu.
  3. Adjust the desired properties.

Eigenschaften

Hinweis : Nicht alle diese Eigenschaften sind im Aufgaben-Fenster-Editor verfügbar. Einige sind nur auf der Registerkarte Daten des Eigenschaftenfensters für diese Bearbeitung zugänglich.

Base

Hinweis: Es wird empfohlen, die Eigenschaft Placement von Pfadbearbeitungen nicht zu bearbeiten. Stattdessen verschiebt oder dreht man das CAM-Auftragsmodell nach Bedarf.

  • DatenPlacement: Overall placement[position and rotation] of the object - with respect to the origin (or origin of parent object container)
    • DatenAngle: Angle in degrees applied to rotation of the object around Axis property value
    • DatenAxis: Axis (one or multiple) around which to rotate the object, set in sub-properties: X, Y, Z
      • DatenX: X-axis value
      • DatenY: Y-axis value
      • DatenZ: Z axis value
    • DatenPosition: Position of the object, set in sub-properties: X, Y, Z - with respect to the origin (or origin of parent object container)
      • DatenX: X-distance value
      • DatenY: Y-distance value
      • DatenZ: Z-distance value
  • DatenLabel: User-provided name of the object (UTF-8)

Depth

  • DatenClearance Height: The height needed to clear clamps and obstructions
  • DatenFinal Depth: Final Depth of Tool- lowest value in Z
  • DatenFinish Depth: Maximum material removed on final pass. The height (thickness) of the last cutting level - set for a better finish.
  • DatenSafe Height: The height above which Rapid motions are allowed. (Rapid safety height between locations)
  • DatenStart Depth: Starting depth of Tool - first cut depth in Z
  • DatenStep Down: Incremental step down of Tool during operation

Visuelle Referenz für Tiefeneigenschaften (Einstellungen)

Extension

  • Daten-EigenschaftExtension Corners: Wenn aktiviert, werden verbundene Verlängerungskanten zu Linienzügen zusammengefasst.
  • Daten-EigenschaftExtension Length Default: Standardlänge der Verlängerungen.

Face

  • Daten-EigenschaftOffset Pattern: Zu verwendendes Räumen-Muster. (Man wählt aus, auf welche Weise die horizontalen Bewegungen ausgeführt werden sollen.)

Path

  • Daten-EigenschaftActive: Falsch machen, um zu verhindern, dass der Vorgang Code generiert
  • Daten-EigenschaftComment: Ein optionaler Kommentar für diese Bearbeitung
  • Daten-EigenschaftUser Label: Benutzerdefinierte Beschriftung
  • Daten-EigenschaftTool Controller: Definiert die in der Bearbeitung verwendete Werkzeugsteuerung

Pocket

  • DatenCut Mode: Specifies a CW or CCW move for the cut
  • DatenExtra Offset: Extra offset to apply to the operation. Direction is operation dependent. (Extra value to stay away from final profile good for roughing toolpath)
  • DatenKeep Tool Down: Attempts to avoid unnecessary retractions.
  • DatenMin Travel: Use 3D Sorting of Path (when multiple base geometries used).
  • DatenStart At: Start pocketing at center or boundary.
  • DatenStep Over: Select the horizontal step over (Percent of tool diameter: 100% = tool diameter).
  • DatenUse Outline: Uses the outline of the base geometry.
  • DatenZig Zag Angle: Angle of the zigzag pattern. (Select the path angle relative to X-axis.)

Rotation

  • DatenAttempt Inverse Angle: Automatically attempt Inverse Angle if initial rotation is incorrect.
  • DatenB_Axis Error Override: Visually orient B(y) rotations to model (error in FreeCAD rendering).
  • DatenEnable Rotation: Enable rotation to gain access to pockets or areas not normal to Z axis.
  • DatenInverse Angle: Inverse the angle of the rotation. Example: change a rotation from -22.5 to 22.5 degrees.
  • DatenReverse Direction: Reverse orientation of Operation by 180 degrees.

Start Point

  • Daten-EigenschaftStart Point: Der Startpunkt dieses Pfades.
  • Daten-EigenschaftUse Start Point: True setzen, wenn ein Startpunkt manuell festgelegt werden soll, und dann die Startpunkte in das Feld Start Points der Eigenschaftsdaten eingeben.

Layout des Aufgabenfenster-Editors

Beschreibungen zu den Einstellungen findet man in der Eigenschaften-Liste oben.

Dieser Abschnitt ist lediglich eine Übersichtskarte der Einstellungen im Fenster-Editor für die Bearbeitungen.

Basisgeometrie

  • Hinzufügen: Fügt ausgewählte Elemente hinzu, die als Basis für die Pfade dienen sollen
  • Entfernen: Entfernt die ausgewählten Elemente aus der Liste Basisgeometrie
  • Löschen: Alle Elemente in der Liste Basisgeometrie löschen

Erweiterungen

  • Daten-EigenschaftShow All: Wenn diese Option ausgewählt ist, werden alle potenziellen Erweiterungen angezeigt. Aktivierte Erweiterungen werden in Lila, deaktivierte Erweiterungen in Gelb dargestellt.
  • Daten-EigenschaftExtension Corners
  • Daten-EigenschaftExtension Length Default
  • Aktivieren
  • Deaktivieren
  • Löschen

Tiefen

  • Daten-EigenschaftStart Depth
  • Daten-EigenschaftFinal Depth
  • Daten-EigenschaftStep Down

Höhen

  • Daten-EigenschaftSafe Height
  • Daten-EigenschaftClearance Height

Bearbeitungen

  • DatenTool Controller
  • DatenCut Mode
  • DatenPattern (Offset Pattern)
  • DatenAngle (Zig Zag Angle)
  • DatenStep Over Percent (Step Over)
  • DatenPass Extension: The distance the facing operation will extend beyond the boundary shape (base geometry)
