Einleitung

Der Arbeitsbereich PartDesign enthält Werkzeuge zum Modellieren von Festkörperkomponenten. Sein Fokus liegt hauptsächlich auf der Erstellung mechanischer Komponenten, die hergestellt und zu einem Endprodukt montiert werden können. Dennoch können die erzeugten Körper generell für jeden anderen Zweck verwendet werden, wie z. B. BIM-Modellierung (BIM), Finite-Elemente-Analyse, oder CNC-Bearbeitung und 3D-Druck.

Der Arbeitsbereich PartDesign verwendet eine auf Formelementen basierende Methodik. Eine Komponente wird durch einen Körper-Objektcontainer dargestellt (Body object container). Der Körper legt ein lokales Koordinatensystem fest und enthält die gesamten Formelemente, die die Komponente aufbauen. Die meisten Formelemente basieren auf parametrischen Skizzen sind hinzufügend oder abziehend. Beispielsweise fügt das Werkzeug Block die extrudierte Skizze zum sich entwickelnden Festkörper hinzu, während das Werkzeug Vertiefung die extrudierte Skizze abzieht. Jedes Formelement trägt zum Gesamtergebnis bei und baut dabei auf dem Ergebnis der vorherigen Formelemente auf. Es können auch Grundkörper (Zylinder, Kugeln usw.) sowie Festkörper, die außerhalb des Körpers erstellt wurden, als Formelemente verwendet werden.

Siehe die Seite Formelemente bearbeiten für eine umfassendere Erklärung dieses Prozesses und anschließend Erstellen eines einfachen Bauteils mit PartDesign, um mit dem Erstellen von Festkörpern loszulegn.

Der Arbeitsbereich Part verwendet zum Erstellen von Festkörpern die alternative Methodik der konstruktiven Festkörpergeometrie (engl.: Constructive Solid Geometry, kurz CSG). Eine detaillierte Diskussion der Arbeitsbereiche Part und Part Design im Vergleich befindet sich unter Part und Part Design.

Werkzeuge

Die Werkzeuge für PartDesign befinden sich alle im Menü Part Design und in den PartDesign-Symbolleisten, die beim Laden des Arbeitsbereichs angezeigt werden.

Symbolleiste Part Design Hilfsfunktionen

  • Neuer Körper: Erstellt einen Körper im aktiven Dokument und aktiviert ihn.
  • Neue Skizze:
  • Neue Skizze: Erstellt eine neue Skizze auf einer ausgewählten Fläche oder Ebene. Wenn vor der Ausführung dieses Werkzeugs keine Fläche ausgewählt wurde, wird der Anwender aufgefordert, eine Ebene im Aufgaben-Fenster auszuwählen. Die Oberfläche wechselt dann zum Arbeitsbereich Sketcher in den Bearbeitungsmodus.
  • Skizze befestigen: Befestigt eine Skizze an eine Geometrie, die am aktiven Körper ausgewählt wurde.
  • Skizze bearbeiten: öffnet die ausgewählte Skizze zur Bearbeitung.
  • Skizze überprüfen: Überprüft die Toleranz verschiedener Punkte und passt sie an.
  • Geometrie überprüfen: Prüft die Geometrie von ausgewählten Objekten auf Fehler.
  • Teilformbinder: Erstellt einen Teilformbinder, der Geometrie von einem oder mehreren übergeordneten Objekten referenziert.
  • Klon: Erstellt einen Klon des ausgewählten Körpers.

Symbolleiste Part Design Modellierungsfunktionen

Hinzufügende Werkzeuge

Dies sind Werkzeuge zum Erstellen von Basisformelementen oder zum Hinzufügen von Material zu einem bestehenden Körper.

  • Block: Extrudiert einen Festkörper aus einer ausgewählten Skizze.
  • Drehteil: Erstellt ein Drehteil, durch Drehen einer Skizze um eine Achse. Die Skizze muss ein geschlossenes Profil ergeben.
  • Ausformung: Erstellt einen Festkörper, indem ein Übergang zwischen zwei oder mehr Skizzen erstellt wird.
  • Rohr: Erstellt einen Festkörper, indem eine oder mehrere Skizzen entlang einer offenen oder geschlossenen Leitkurve (Spine) ausgetragen werden.
  • Wendel: Erstellt einen Festkörper durch Austragen einer Skizze entlang einer Wendel.
  • Hinzufügende Grundkörper:
  • Quader: Erstellt einen hinzufügenden Quader.
  • Zylinder: Erstellt einen hinzufügenden Zylinder.
  • Kugel: Erstellt eine hinzufügende Kugel.
  • Kegel: Erstellt einen hinzufügenden Kegel.
  • Ellipsoid: Erstellt ein hinzufügendes Ellipsoid.
  • Torus: Erstellt einen hinzufügenden Torus.
  • Prisma: Erstellt ein hinzufügendes Prisma.
  • Keil: Erstellt einen hinzufügenden Keil.

Abziehende Werkzeuge

Dies sind Werkzeuge zum Abziehen (Entfernen) von Material von einem bestehenden Körper.

  • Vertiefung: Erstellt eine Vertiefung aus einer ausgewählten Skizze.
  • Bohrung: Erstellt ein Bohrungsformelement aus einer ausgewählten Skizze. Die Skizze muss einen oder mehrere Kreise enthalten.
  • Nut: Erstellt eine Nut (Einstich), durch Drehen einer Skizze um eine Achse.
  • Ausformung: Erstellt eine Festkörperform, indem ein Übergang zwischen zwei oder mehreren Skizzen erstellt wird, und zieht diese von dem aktiven Körper ab.
  • Rohr: Erstellt eine Festkörperform, indem eine oder mehrere Skizzen entlang eines offenen oder geschlossenen Pfades ausgetragen werden, und zieht diese von dem aktiven Körper ab.
  • Wendel: Erstellt eine Festkörperform, indem eine Skizze entlang einer Wendel ausgetragen wird, und zieht diese von dem aktiven Körper ab.
  • Abziehende Grundkörper:
  • Quader: Fügt dem aktiven Körper einen abziehenden Quader hinzu.
  • Zylinder: Fügt dem aktiven Körper einen abziehenden Zylinder hinzu.
  • Kugel: Fügt dem aktiven Körper eine abziehende Kugel hinzu.
  • Kegel: Fügt dem aktiven Körper einen abziehenden Kegel hinzu.
  • Ellipsoid: Fügt dem aktiven Körper ein abziehendes Ellipsoid hinzu.
  • Torus: Fügt dem aktiven Körper einen abziehenden Ring hinzu.
  • Prisma: Fügt dem aktiven Körper ein abziehendes Prisma hinzu.
  • ‎Keil: Fügt dem aktiven Körper einen abziehenden Keil hinzu.

Boolesche Verknüpfungen

  • Boolesche Verknüpfung: Importiert einen oder mehrere Körper oder PartDesign-Klone in den aktiven Körper und führt eine boolesche Verknüpfung aus.

Symbolleiste Part Design Modifikationswerkzeuge

Diese Werkzeuge modifizieren Kanten und Flächen.

  • Verrundung: Rundet Kanten des aktiven Körpers ab.
  • Fase: Fast Kanten des aktiven Körpers an.
  • Formschräge: Versieht ausgewählte Flächen des aktiven Körpers mit einer Formschräge.
  • Dicke: Erstellt ein Schalenobjekt (mit konstanter Wandstärke) aus dem aktiven Körper und öffnet ausgewählte Flächen.

Symbolleiste Part Design Transformationswerkzeuge

Dies sind Werkzeuge zur Transformation bestehender Formelemente.

  • Spiegeln: Spiegelt ein oder mehrere Formelemente.
  • Lineares Muster: Erstellt ein lineares Muster aus einem oder mehreren Formelementen.
  • Polares Muster: Erstellt ein kreisförmiges Muster aus einem oder mehreren Formelementen.
  • Mehrfachtransformation: Erstellt ein Muster durch eine beliebige Kombination der oben gelisteten Transformationen und der Transformation Skalieren.
    • Skalieren: Skaliert ein oder mehrere Formelemente. Dieses steht nicht als eigenständiges Transformationswerkzeug zur Verfügung.

Zusätzliche Werkzeuge

Einige zusätzliche Werkzeuge befinden sich im Menü Part Design:

  • Formbinder: Erstellt einen Formbinder, der Geometrie eines einzelnen übergeordneten Objekts referenziert.
  • Evolventenrad: Erstellt den Umriss eines Evolventenzahnrades, der extrudiert werden kann.
  • Kettenrad: Erstellt den Umriss eines Kettenrades, das extrudiert werden kann.
  • Entwurfsassistent für Wellen: Erstellt eine Welle aus einer Wertetabelle und ermöglicht die Analyse von Kräften und Momenten. Die Welle wird mit einer umlaufenden Skizze erstellt, die bearbeitet werden kann.

Kontextmenü

  • Unterdrückt: Checkbox zum Deaktivieren eines bestimmten Formelements ohne es zu löschen. eingeführt in 1.0
  • Arbeitsposition festlegen: Legt die Arbeitsposition neu fest, d.h. das Formelement, das den Körper nach außen darstellt.
  • Objekt in anderen Körper verschieben: verschiebt die ausgewählte Skizze, Bezugsgeometrie oder das Formelement in einen anderen Körper.
  • Objekt nach anderem Objekt verschieben: ermöglicht eine Neuordnung des Körperbaums, indem die ausgewählte Skizze, Bezugsgeometrie oder das Formelement an eine andere Position in der Formelementliste verschoben wird.

Veraltete Werkzeuge

  • Bezugsebene erstellen: Erstellt eine Bezugsebene im aktiven Körper. Dieses Werkzeug ist in 1.1 und neuer nicht verfügbar.
  • Bezugslinie erstellen: Erstellt eine Bezugslinie im aktiven Körper. Dieses Werkzeug ist in 1.1 und neuer nicht verfügbar.
  • Bezugspunkt erstellen: Erstellt einen Bezugspunkt im aktiven Körper. Dieses Werkzeug ist in 1.1 und neuer nicht verfügbar.
  • Lokales Koordinatensystem erstellen: Erstellt ein lokales Koordinatensystem, das an der Bezugsgeometrie im aktiven Körper angeheftet ist. Dieses Werkzeug ist in 1.1 und neuer nicht verfügbar.
  • Migrieren: migriert Dateien von FreeCAD Versionen unter 0.17 zu Version 0.17. Dieses Werkzeug ist nicht verfügbar in 1.0 und neuer.
