Beschreibung

Der Befehl Fasteners ParameterÄndern kann verwendet werden, um die Parameter von Verbindungselementen zu ändern.

Anwendung

  1. Ein oder mehrere Verbindungselemente auswählen. Wenn mehrere Verbindungselemente ausgewählt werden, darf die Auswahl entweder nur Schrauben, nur Scheiben oder nur Muttern enthalten.
  2. Es gibt mehrere Möglichkeiten den Befehl aufzurufen:
    • Die Schaltfläche Parameter ändern drücken.
    • Den Menüeintrag Fasteners → Parameter ändern auswählen.
  3. Das Aufgaben-Fenster Parameter ändern wird geöffnet.
  4. Wahlweise die art des Verbindungselements (Fastener type) ändern.
  5. Wahlweise den Durchmesser ändern mit einer der folgenden Möglichkeiten:
    • Checkbox Durchmesser automatisch anpassen aktivieren, um den Durchmesser der Verbindungselemente an die Kante anzupassen, an die sie befestigt werden.
    • Checkbox Durchmesser automatisch anpassen deaktivieren und einen Durchmesser (Diameter) aus der Ausklappliste auswählen.
  6. Das Kontrollkästchen Länge festlegen deaktiviert lassen. Diese Option funktioniert noch nicht.
