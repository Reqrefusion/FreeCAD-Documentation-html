Beschreibung

The Draft PathTwistedLinkArray command creates a Link array from a selected object by placing copies along a path that are also rotated around the path.. Use the Draft PathTwistedArray command to create a less efficient regular array instead. Except for the type of array that is created, Link array or regular array, this command is identical to the Draft PathTwistedArray command. See there for more information.
