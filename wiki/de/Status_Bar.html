Einleitung

Die Statusleiste ist ein Band, das am unteren Rand der FreeCAD-Oberfläche angezeigt wird. Sie kann verschiedene Informationen und Widgets anzeigen. Die Arbeitsbereiche BIM und Draft fügen der Statusleiste ihre eigenen Widgets hinzu. Es ist auch möglich, eine Symbolleiste in die Statusleiste zu ziehen.

Information und Fenster

Informationen zur Verwendung

Wenn sich der Mauszeiger über einer Schaltfläche oder einem Menüpunkt befindet, werden die Nutzungsinformationen zu diesem Werkzeug in der Statusleiste angezeigt. Diese Informationen entsprechen in der Regel den Hinweistext-Informationen des Werkzeugs.

Vorausgewählte Information

Wenn sich der Mauszeiger über einem Objekt in der 3D-Ansicht befindet, zeigt die Statusleiste das vorgewählte Element des Objekts sowie die aktuellen 3D-Koordinaten an. Diese Informationen werden aktualisiert, wenn der Mauszeiger über das Objekt bewegt wird. Die Aktualisierung wird beendet, wenn sich der Mauszeiger über einem leeren Bereich in der 3D-Ansicht befindet.

Eingabehinweise

eingeführt in 1.1

Für die meisten Sketcher Werkzeuge und Draft Zeichnen-Werkzeuge, werden dynamische Verwendungstipps in der Statusleiste angezeigt.

Schnellmessung

eingeführt in 1.0

Die Schnellmessfunktion nutzt die rechte Seite der Statusleiste, um Messungen basierend auf der Auswahl in der 3D-Ansicht anzuzeigen.

Basierend auf der Auswahl werden die folgenden Messungen angezeigt:

  • Kantenlänge
  • Flächeninhalt
  • Abstand zwischen Punkten, Kanten oder Flächen
  • Winkel zwischen Kanten
  • Radius von kreisförmigen Kanten oder zylindrischen Flächen
  • eingeführt in 1.1: Durchmesser von Kreisen und zylindrischen Flächen
  • eingeführt in 1.1: Achsabstand zwischen kreisförmigen Elementen und Flächen, Kanten oder anderen kreisförmigen Elementen

Benachrichtigungsbereich

In Arbeit.

Ein Rechtsklick auf diesen Bereich öffnet ein Kontextmenü mit den folgenden Optionen:

  • Benutzerbenachrichtigungen löschen: In Arbeit.
  • Alle löschen: In Arbeit.

Navigations-Anzeige

The Navigation indicator shows the current Mouse navigation mode. Clicking the indicator opens a menu with the available modes and a Settings submenu with the following options:

  • Orbit style: See Preferences Editor.
  • Compact: If this option is checked only a mouse icon is shown in the widget.
  • Tooltip: If this option is checked hovering the widget will show a tooltip with detailed usage information.

Zoom-Anzeige

Die Statusleiste liefert Informationen zum Zoomfaktor, indem sie die Größe der aktuellen 3D-Ansicht anzeigt. Durch Klicken auf die Anzeige öffnet sich ein Menü, in dem das Einheitensystem des aktuellen Dokuments oder, falls kein Dokument aktiv ist, das Standard-Einheitensystem geänder werden kann.

Kontextmenü

Ein Rechtsklick auf die Statusleiste öffnet ein Kontextmenü mit den folgenden Optionen:

  • eingeführt in 1.1: Eingabehinweise: Kontrollkästchen zum Aktivieren dieser Funktion.
  • eingeführt in 1.1: Schnellmessung: Ebenso.
  • Benachrichtigungsbereich: Ebenso.
  • Navigationsanzeige: Ebenso.

If the pointer is over the Preselected information field or the Quick measure field the following options are added:

  • Copy: Copies the information selected with the next option.
  • Select All: Selects the current information.

Der Benachrichtigungsbereich hat ein eigenes Kontextmenü.
