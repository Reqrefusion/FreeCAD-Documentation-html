Beschreibung

Der Befehl FreeGrid LagerBox erzeugt eine Lagerbox.

Dieses Werkzeug ist Teil des Arbeitsbereich FreeGrid, dieser ist ein externer Arbeitsbereich, der mit dem Addon-Manager installiert werden kann.

Anwendung

  1. There are several ways to invoke the tool:
    • Press the Storage box button.
    • Select the FreeGrid → Storage box option from the menu.
  2. A storage box is created with properties based on the current preferences.
  3. Optionally select the object and change its properties in the Property editor.

Eigenschaften

Ein FreeGrid-StorageBox-Objekt wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem hat es die folgenden zusätzlichen Eigenschaften:

Daten

Box features

  • DatenBox Grip (Bool): Default is true. Add grip/label area at the rear of the box.
  • DatenBox Grip Depth (Length): Default is 15 mm. Depth of the grip.
  • DatenBox Open Front (Bool): Default is false. Leave the front of the box open.
  • DatenBox Ramp (Bool): Default is true. Add a scoop inside the front of the box.
  • DatenFloor Support (Bool): Default is true. Add integral floor support.

Internal divisions

  • DatenDivision Height (Percent): Default is 100. Height of internal divisions relative to the height of the box.
  • DatenDivisions X (IntegerConstraint): Default is 1. Number of divisions along the X axis.
  • DatenDivisions Y (IntegerConstraint): Default is 1. Number of divisions along the Y axis.

Magnet mount

  • DatenMagnet Diameter (Length): Default is 6 mm. Diameter of the magnets.
  • DatenMagnet Height (Length): Default is 2 mm. Height of the magnets.
  • DatenMagnet Option (Enumeration): Default is allIntersections. Positions of the magnets. The other options are: cornersOny and noMagnets.

Position on grid

  • DatenPosition X (IntegerConstraint): Default is 0. Object position on the grid along the X axis. Starts at zero.
  • DatenPosition Y (IntegerConstraint): Default is 0. Object position on the grid along the Y axis. Starts at zero.

Size

  • DatenDepth (IntegerConstraint): Default is 1. Number of 50 mm units in the Y direction of the object.
  • DatenHeight (Length): Default is 50 mm. Height of the object.
  • DatenWidth (IntegerConstraint): Default is 1. Number of 50 mm units in the X direction of the object.
