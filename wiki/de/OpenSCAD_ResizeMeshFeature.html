Beschreibung

Erstellt ein neues NetzObjekt mit angepasster Größe, wobei jede Achse unabhängig voneinander skaliert werden kann.

Anwendung

  1. Select the mesh object to be resized.
  2. Select the OpenSCAD → Resize Mesh Feature option from the menu.
  3. Enter the resizing value.
  4. A new mesh object is created and resized, the original object is rendered hidden.

Einschränkungen

  • Das neue Netzobjekt ist nicht parametrisch zum ursprünglichen Netzobjekt, was bedeutet, dass Änderungen am ursprünglichen Objekt nicht im neuen gespiegelten Objekt übernommen werden.

Hinweise

  • Die Funktion ändert nicht das vorhandene Netz, sondern erstellt ein neues Netz.

Skripten

Auf diese Funktion kann über Python zugegriffen werden: 

import OpenSCADUtils
import Mesh
#this assumes an existing object in the document named "Mesh" that you wish to mirror
original_mesh = App.ActiveDocument.Mesh
resized_mesh = OpenSCADUtils.resizemesh(original_mesh.Mesh, FreeCAD.Base.Vector(100,50,40))
#New mesh would be 100 mm on the x axis, 50 mm on the y axis, and 40 mm on the z axis.
Mesh.show(resized_mesh)
