Einleitung

Der Arbeitsbereich Curves (Kurven) ist ein externer Arbeitsbereich, der auf Python basiert und eine Sammlung von Werkzeugen für NURBS-Kurven und -Oberflächen enthält. Dieser Arbeitsbereich wurde mit FreeCAD-Master und OCC 7.4 entwickelt.

Hinweis: Einige Werkzeuge funktionieren möglicherweise nicht mit früheren Versionen.

Installation

Der Arbeitsbereich Curves kann mit dem Addon-Manager heruntergeladen werden: Menüeintrag Werkzeuge → Addon-Manager auswählen.

Werkzeuge

Mit dem letzen update wurde die Reihenfolge der Werkzeuge mit den Menüeinträgen der Version 0.6.59 abgeglichen und einige fehlende Werkzeuge wurden hinzugefügt.

Kurvenwerkzeuge

  • Parametrische Linie: Erstellt eine parametrische Linie zwischen zwei Knoten.
  • Freihand-B-Spline: Erstellt eine Freihand-B-Spline-Kurve.
  • Kombi-Kurve: Erstellt eine 3D-Kurve durch Verschneiden (Kombinieren) zweier projizierter Kurven.
  • Kurve verlängern: Verlängert die ausgewählte Kante.
  • Kurven verbinden: Verbindet die ausgewählten Kanten zu B-Spline-Kurven.
  • Kurve teilen: Teilt die ausgewählte Kante.
  • Diskretisieren: Erstellt eine Abfolge von Punkten entlang einer Kante oder eines Linienzuges.
  • Approximieren: Approximiert Punkte zu einer NURBS-Kurve oder -Oberfläche.
  • Interpolieren: Interpoliert Punkte zu einer B-Spline-Kurve.
  • Übergangskurve: Erstellt eine parametrische Übergangskurve zwischen zwei Kanten.
  • Kammdiagramm: Erstellt ein parametrisches Kammdiagramm an ausgewählten Kanten.
  • Kurve auf Oberfläche: Projiziert eine Kurve auf ein (Ober-) Flächenobjekt.

Oberflächenwerkzeuge

  • Zebrawerkzeug: Zebratextur für die Oberflächenprüfung.
  • Fläche beschneiden: Beschneidet eine Fläche mit einer projizierten Kurve.
  • Iso-Kurve: Legt ein UV-ausgerichtetes Gitter auf ausgewählte Oberflächen.
  • Skizze auf Oberfläche: Bildet eine Skizze auf einer Oberfläche ab.
  • Sweep2Rails: Erstellt ein Austragungsobjekt (Sweep_2_rails-Objekt), entweder aus einer Regelfläche und einer Liste mit Profilkanten oder mit zwei Führungskurven (Rails) und mindestens zwei Profilen
  • Profil-Trägerfläche: Erstellt eine Trägerfläche für Skizzen.
  • Pipeshell-Profil: Erstellt ein Profil bzw. einen Querschnitt (Profile-Objekt) für ein PipeShell-Objekt.
  • Pipeshell: Erstellt ein Pipeshell-Sweep-Objekt.
  • Gordon-Fläche: Erstellt eine Oberfläche, die ein Netzwerk aus Kurven überspannt.
  • Fläche aufteilen: Teilt eine Fläche entlang ihrer Iso-Kurven auf.
  • Druckfeder: Erstellt eine Druckfeder.
  • Reflexionslinien: Erstellt Reflexionslinien auf einer Form entsprechend der Ausrichtung einer Ansicht.
  • Mehrfach-Loft: Erstellt eine Form aus Profile-Objekten, die aus mehreren parallele Flächen bestehen.
  • Übergangsfläche: Erstellt eine Fläche zwischen zwei Kanten mit Übergängen zu ihren Trägerflächen, deren Stetigkeit eingestellt werden kann.
  • Übergangskörper: Erstellt einen Übergangskörper zwischen zwei Kanten mit Übergängen zu den Trägerformen, deren Stetigkeit eingestellt werden kann.
  • Fläche abwickeln: Erstellt ebene Abwicklungen von konischen oder zylindrischen Flächen.
  • Rotation Sweep: Erstellt ein RotationSweep-Objekt durch Austragen eines Profils (Profile-Objekt) entlang eines Spines (Rückgratkurve - Sweep-Path) oder um einen Punkt herum.
  • Oberflächenanalyse: Erstellt visuelle Abbildungen für die Analyse auf ausgewählten Formen. Dieses Werkzeug zeigt in der GUI ein falsches Symbol und einen falschen Namen.
  • Schrägungsanalysis: Erstellt eine farbige Überlagerung auf einem Objekt, um die Schrägungswinkel darzustellen.
  • Truncated Extend: Erstellt einen Schnitt durch eine Form und verlängert bzw. kürzt sie um einen gegebenen Abstand.
  • Wasserlinie: Erstellt Wasserlinien-Kurven auf ausgewählten Flächen.

Verschiedene Werkzeuge

  • GeomInfo: Schaltet die Anzeige von Informationen zu den ausgewählten Formen ein bzw. aus.
  • Teilform extrahieren: Erstellt nicht parametrische Kopien von ausgewählten Teilformen.
  • Parametrischer Körper: Erstellt einen parametrischen Festkörper aus ausgewählten Flächen.
  • SVG einfügen: Fügt den SVG-Inhalt der Zwischenablage ein.
  • Objekte zur Konsole: Bewegt Objekte zur Konsole.
  • Angrenzende Flächen auswählen: Wählt die angrenzenden Flächen der ausgewählten Teilform aus.
  • B-Spline zur Konsole: Erstellt ein Python-Skript zum Aufbau der ausgewählten B-spline- oder Bézier-Geometrien.
