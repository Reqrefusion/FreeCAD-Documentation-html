Der Arbeitsbereich Curves (Kurven) ist ein externer Arbeitsbereich, der auf Python basiert und eine Sammlung von Werkzeugen für NURBS-Kurven und -Oberflächen enthält. Dieser Arbeitsbereich wurde mit FreeCAD-Master und OCC 7.4 entwickelt.

Hinweis: Einige Werkzeuge funktionieren möglicherweise nicht mit früheren Versionen.

Einrichtung

Der Arbeitsbereich Curves kann mit dem Addon-Manager heruntergeladen werden: Menüeintrag Werkzeuge → Addon-Manager auswählen.

Werkzeuge

  • Curves Linie: Erzeugt eine parametrische Linie zwischen zwei Knoten.
  • Curves EditierbarerSpline: Erstellt eine Freihand-B-Spline-Kurve.
  • Curves KurveVerlängern: Verlängert die ausgewählte Kante.
  • Curves KurveVerbinden: Verbindet die gewählten Kanten zu B-Spline-Kurven.
  • Curves KurveTeilen: Teilt die ausgewählte Kante.
  • Curves Punktefolge: Verteilt eine Punktefolge entlang einer Kante oder eines Kantenzugs.
  • Curves Interpolieren: Interpoliert eine B-Spline-Kurve aus einer Punktefolge.
  • Curves Parameterkamm: Erstellt einen Parameterkamm entlang ausgewählter Kanten.
  • Curves Zebrawerkzeug: Erstellt und projiziert eine Zebratextur zur Oberflächenprüfung.
  • Curves ParametrischerFestkörper: Erstellt einen parametrischen Festkörper aus ausgewählten Flächen.
  • Curves IsoKurve: Erstellt ein Netz aus isoparametrischen Kurven (kurz Iso-Kurven) auf einer Fläche.
  • Curves SkizzeAufFläche: Bildet eine Skizze auf einer Oberfläche ab.
  • Curves Profilebene: Erstellt eine Trägerebene für Skizzen.
  • Curves SVGEinfügen: Fügt den SVG-Inhalt der Zwischenablage ein.
  • Curves Rohrschalenprofil: Erstellt eine Profilkurve für eine Rohrschale.
  • Curves Rohrschale: Trägt ein Rohrschalenobjekt aus.
  • Curves GordonFläche: Spannt eine Oberfläche über Netzwerk von Kurven auf.
  • Curves FlächeAufteilen: Teilt eine Oberfläche entlang isoparametrischer Kurven (Iso-Kurven) auf.
  • Curves ZurKonsole: Bewegt Objekte zur Konsole.
  • Curves Druckfeder: Erstellt eine Druckfeder.
  • Curves Reflexionslinien: Erstellt die Reflexionslinien auf einer Form, unter Berücksichtigung der Ausrichtung einer Ansicht.
  • Curves MultiLoft: Trägt Flächen über Profile mit mehreren Konturen aus.
