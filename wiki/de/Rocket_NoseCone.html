Beschreibung

Nose cones come in a variety of shapes and sizes, most of which are difficult to model without programming. For many rocket builders, this makes the process unworkable. This command allows for nose cone creation using simple properties combined with a specialized task dialog.

Anwendung

  1. There are several ways to invoke the command:
    • Press the Nose Cone button.
    • Select the Rocket → Nose Cone option from the menu.
    • Double click on a Nose Cone object in the Tree view.
  2. Set options and press OK.

Nasenkegel-Stile

Nose cones can be drawn in one of 3 styles:

  • Solid: The cone is constructed as a solid piece, such as out of balsa wood.

  • Hollow: The cone is hollow on the inside having a specified thickness. The end is not sealed.

  • Capped: Similar to hollow, except the end is sealed.

Properties

Nasenkegel

  • DatenBlunted Diameter: The diameter of the spherical portion at the tip of the nose
  • DatenCoefficient: Combined with the Nose Type, this defines the shape of the nose cone, see Options
  • DatenDiameter: The diameter of the base of the nose cone
  • DatenLength: The length of the nose cone without the shoulder. For all blunted shapes, the length is the actual length of the nose and not the virtual length described in the Nose cone design reference.
  • DatenNose Style: Defines the style of the nose cone, see Options
  • DatenNose Type: Defines the shape of the nose cone using the coefficient when required, see Options
  • DatenOgive Diameter: The diameter of the Ogive circle. This is only required for the Secant Ogive nose cone type
  • DatenResolution: Used internally, this parameter defines the number of data points to use when drawing the outline of the nose cone
  • DatenShoulder: True when the Nose Cone includes a shoulder
  • DatenShoulder Diameter: The diameter of the shoulder. This must be less than the radius of the Nose Cone
  • DatenShoulder Length: The length of the shoulder
  • DatenShoulder Thickness: When the Nose Style is hollow or capped, this will determine the wall thickness of the shoulder
  • DatenThickness: When the Nose Style is hollow or capped, this will determine the wall thickness of the nose cone

Rocket Komponenten

Diese Parameter dienen nur zu Informationszwecken und haben keinen Einfluss auf die Konstruktion der Komponente.

  • DatenDescription: Description of the component
  • DatenManufacturer: Manufacturer when known
  • DatenMaterial: Material when known
  • DatenPart Number: Manufacturer part number
