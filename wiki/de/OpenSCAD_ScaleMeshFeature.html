OpenSCAD NetzelementSkalieren

Mesh Skalieren

Beschreibung

Erstellt ein neues skaliertes Netzobjekt mit unabhängiger Skalierung für jede Achse.

Anwendung

  1. Wähle das zu skalierende Netz-Objekt aus.
  2. Wähle den Befehl OpenSCAD → Netzelement Skalieren aus dem Menü.
  3. Skalierungswert eingeben.
  4. Ein neues Mesh-Objekt wird erstellt und skaliert, das ursprüngliche Objekt wird ausgeblendet.

Einschränkungen

  • Das neue Netzobjekt ist nicht parametrisch zum ursprünglichen Netzobjekt, was bedeutet, dass Änderungen am ursprünglichen Objekt nicht auf das neue skalierte Objekt übertragen werden.

Hinweise

  • Die Funktion ändert nicht das vorhandene Netz, sondern erstellt ein neues Netz.

Skripten

Auf diese Funktion kann über Python zugegriffen werden: 

import OpenSCADUtils
import Mesh
#this assumes an existing object in the document named "Mesh" that you wish to scale
original_mesh = App.ActiveDocument.Mesh
scaled_mesh = OpenSCADUtils.scalemesh(original_mesh.Mesh, FreeCAD.Base.Vector(1,0,0))
Mesh.show(scaled_mesh)
