Beschreibung

Das Werkzeug Cables Leitungsführung erstellt ein Leitungsführungs-Objekt, basierend auf dem ausgewählten Cables Leitungsverlauf oder Cables Verbundverlauf und einem optionalen Profil. Die Leitungsführung ist vollständig parametrisch, von Arch Rohr abgeleitet und erbt alle ihre Eigenschaften.

Dieses Element dient zur Modellierung zusätzlicher Schichten, die zum Aufbau von Kabelkanälen, Kabelbäumen oder Kabelkonfektionen verwendet werden. Es kann andere Kabelkanäle oder Kabel als untergeordnete Elemente unter der Baumansicht gruppieren und eine Form annehmen, die sich aus seiner eigenen parametrischen Form und den Formen seiner untergeordneten Elemente zusammensetzt.


Zwei Sätze Leitungsführungen (MC1, MC2):
Die Leitungsführung MC1 umfasst zwei weitere Leitungsführungen C1, C2 und das Kabel CB1.
Die Leitungsführung MC2 umfasst den Leitungsführung C3 und das Kabel CB2.

Anwendung

  1. Select at least one existing wire object (see Note 1) in the 3D View or Tree View.
  2. Create CableConduit by one of these methods:
    • Press the CableConduit button.
    • Select the Cables → CableConduit option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cables → CableConduit option from the context menu.

Hinweise

  • Note 1: The following wire objects can be safely used for selection as base for CableConduit: WireFlex, CompoundPath, Draft Wire . Other objects containing Points property (e.g. Draft BSpline, Draft BezCurve) can also be used, but results can be different from the expected outcome. See CompoundPath Notes.
  • Note 2: If more than one object was selected when creating a CableConduit, a new CompoundPath object is created first and then the CableConduit is created based on it.
  • Note 3: Optional profile can be added later after CableConduit creation. The following paremetric profile types are possible without external profile: Circle, Square, Rectangle. For details see Arch Pipe Properties.

Eigenschaften

Daten

Component

  • Daten-EigenschaftBase (Link): Der grundlegende Verlauf (Linienzug) einer Leitung.

Für die anderen Eigenschaften in dieser Gruppe siehe Arch Komponente.

Conduit

  • DatenAuto Label Base (Bool): If set to true the name of a base wire will be renamed automatically. To refresh name the property should be changed to false and to true again.
  • DatenMerge Sub Conduits (Bool): If set to true all objects from Sub Conduits property will be merged into final shape. The Part Compound object containing Sub Conduits is created first, then it is added to final shape together with CableConduit own shape.
  • DatenShow Sub Conduits (Bool): Shows/hides all subconduits. This works only if Merge Sub Conduits is set to true.
  • DatenSub Conduits (Link List): List of conduits or cables belonging to the bundle.

Conduit Dimensions

  • DatenGauge (Float): Outer gauge of the conduit. Unit: [mm^2]. It is correlated with dimension properties: Diameter, Width and Height depending on Profile Type property.
  • DatenInsulation Thickness (Length): Thickness of insulation if profile not used. It keeps the same value as Wall thickness.
