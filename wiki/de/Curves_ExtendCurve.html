Beschreibung

Das Werkzeug Curves KurveVerlängern verlängert ausgewählte Kanten an beiden Enden um eine vorgegebene Länge.

Anwendung

  1. Zum Arbeitsbereich Curves wechseln (muss mit dem Addon-Manager installiert werden, wenn noch nicht geschehen)
  2. Es gibt mehrere Möglichkeiten den Befehl aufzurufen:
    • Die Schaltfläche drücken.
    • Den Menüeintrag Curves → Curve extend auswählen

Eigenschaften

See also: Property editor.

A ExtendCurve object is derived from an App FeaturePython object and inherits all its properties. It also has the following additional properties:

Data

Base

  • DatenEdge (LinkSub): Input edge to exend.
  • DatenOutput (Enumeration): Output shape.
    • SingleEdge (default): A single continuous curve.
    • Wire: Separate curves.

Beginning

  • DatenLength Start (Float): Start Extension Length.
  • DatenType Start (Enumeration): Start Extension Type.
    • Straight (default): The extension is a straight line.
    • G2 curve: The extension is another curve that adopts the local curvature of the input edge at their common vertex (G2 continuity).

End

  • DatenLength End (Float): End Extension Length.
  • DatenType End (Enumeration): End Extension Type.
    • Straight (default): The extension is a straight line.
    • G2 curve: The extension is another curve that adopts the local curvature of the input edge at their common vertex (G2 continuity).
