Beschreibung

Der Befehl FCGear Innenverzahnung erstellt ein Evolventenhohlrad, ein InternalInvoluteGear-Objekt, standardmäßig mit einer Geradverzahnung, die auch in eine Schrägverzahnung oder eine Pfeilverzahnung umgewandelt werden kann.

Von links nach rechts: Hohlräder mit Geradverzahnung, Schrägverzahnung, Pfeilverzahnung

Anwendung

  1. Zum Arbeitsbereich Gear wechseln.
  2. Es gibt mehrere Möglichkeiten den Befehl aufzurufen:
    • Die Schaltfläche Innenverzahntes Evolventenzahnrad drücken.
    • Den Menüeintrag Gear → Innenverzahntes Evolventenzahnrad auswählen.
  3. Die Einstellungen den geforderten Randbedingungen entsprechend ändern (siehe Eigenschaften).

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

Ein FCGear-Evolventenhohlrad, ein InternalInvoluteGear-Objekt, wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem hat es die folgenden zusätzlichen Eigenschaften:

Daten

accuracy

  • Daten-Eigenschaftnumpoints (Integer): Standardwert ist 6. Ändert das Evolventenprofil. Das Ändern des Wertes kann zu unerwarteten Ergebnissen führen.

base

  • Daten-Eigenschaftgear (PythonObject): Python gear-Objekt.
  • Daten-Eigenschaftheight (Length): Der Standardwert ist 5 mm. Wert der Zahnradbreite.
  • Daten-Eigenschaftmodule (Length): Der Standardwert ist 1 mm. Der Modul ist das Verhältnis des Referenzdurchmessers des Zahnrads geteilt durch die Anzahl der Zähne (siehe Hinweise).
  • Daten-Eigenschaftnum_teeth (Integer): Der Standardwert ist 15. Anzahl der Zähne (siehe Hinweise).
  • Daten-Eigenschaftthickness (Length): Der Standardwert ist 5 mm. Dicke des unbeschnittenen äußeren Teils des Zahnrads.

computed

  • Daten-Eigenschaftangular_backlash (Angle): (nur lesbar)
  • Daten-Eigenschaftda (Length): (nur lesbar) Innendurchmesser, gemessen am Zahnkopf (Kopfkreisdurchmesser).
  • Daten-Eigenschaftdf (Length): (nur lesbar) Fußkreisdurchmesser, gemessen am Zahnlückengrund.
  • Daten-Eigenschaftdw (Length): (nur lesbar) Arbeits-Teilkreisdurchmesser.
  • Daten-Eigenschaftoutside_diameter (Length): (nur lesbar) Aussendurchmesser.
  • Daten-Eigenschafttransverse_pitch (Length): (nur lesbar) Neigung in der Drehebene.

fillets

  • Daten-Eigenschafthead_fillet (Float): Der Standardwert ist 0 mm.
  • Daten-Eigenschaftroot_fillet (Float): Standardwert ist 0 mm.

helical

  • Daten-Eigenschaftbeta (Angle): Der Standardwert ist 0°. Mit dem Schrägungswinkel β entsteht ein Hohlrad mit Schrägverzahnung – positiver Wert → rechtssteigend, negativer Wert → linkssteigend (siehe Hinweise).
  • Daten-Eigenschaftdouble_helix (Bool): Die Standardeinstellung ist false, true erstellt ein Hohlrad mit Pfeilverzahnung (siehe Hinweise).
  • Daten-Eigenschaftproperties_from_tool (Bool):Der Standardwert ist false. Wenn dieser true und die Daten-Eigenschaftbeta ungleich Null ist, werden intern die Parameter für die gedrehte Verzahnung berechnet.

involute

  • Daten-Eigenschaftpressure_angle (Angle): Der Standardwert ist 20° (siehe Hinweise).
  • Daten-Eigenschaftshift (Float): Der Standardwert ist 0. Erzeugt eine positive und negative Profilverschiebung (siehe Hinweise).

precision

  • Datensimple (Bool): Standardwert ist false. true erzeugt eine vereinfachte Ansicht (ohne Zähne und mit nur einem Zylinder mit Teilkreisdurchmesser).

tolerance

  • Daten-Eigenschaftbacklash (Length): Der Standardwert ist 0 mm. Das Flankenspiel, auch Umfangsspiel bezeichnet, ist der Abstand zwischen den Zähnen eines Zahnradpaares.
  • Daten-Eigenschaftclearance (Float): Der Standardwert ist 0,25. Kopfspiel (siehe Hinweise).
  • Daten-Eigenschafthead (Float): Der Standardwert ist -0,4 mm. Dieser Wert wird verwendet, um die Zahnhöhe zu ändern.
  • Daten-Eigenschaftreversed_backlash (Bool): Für true ergibt die Eigenschaft backlash eine Flankenüberdeckung oder für false (Standard) ein Flankenspiel, (siehe Hinweise).

version

  • Datenversion (String):

Skripten

Die Macht von Python nutzen, um das Erstellen von Zahnrädern zu automatisieren: 

import FreeCAD as App
import FreeCADGui as Gui
import freecad.gears.commands
gear = freecad.gears.commands.CreateInternalInvoluteGear.create()
gear.num_teeth = 20
gear.helix_angle = 20
gear.height = 10
gear.double_helix = True
App.ActiveDocument.recompute()
Gui.SendMsgToActiveView("ViewFit")
