Beschreibung

Das Werkzeug Curves Oberflächenanalyse erstellt visuelle Analysetexturen auf ausgewählten Formen.

Unterschiedliche Analysearten auf einer Kugel abgebildet. Von links nach rechts: ohne, Zebra, Regenbogen, Isopote

Anwendung

  1. Select one or more sub shapes.
  2. There are several ways to invoke the tool:
    • Press the Surface Analysis button.
    • Select the Surfaces → Surface Analysis option from the menu. Note that a wrong name, Title, is used in the menu.
  3. A SurfaceAnalysis object is created and a zebra texture is mapped on the selected objects.
  4. Optionally edit the values in the Property editor to choose a different mode and adjust the related properties (see Properties below).

Hinweise

  • The naming and the icon in the GUI seem to be preliminary...
  • The object in the Tree view is always set to Hidden, and space bar can only switch visibility on but not off. The DatenVisibility property in the Property editor still works properly for this task.


Eigenschaften

Siehe auch: Eigenschafteneditor.

Ein SurfaceAnalysis-Objekt wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem besitzt es die folgenden zusätzlichen Eigenschaften:

Ansicht

AnalysisMode

  • AnsichtDirection (Vector): Analysis Direction.
  • AnsichtFixed (Bool): Fix analysis direction to global coordinate system.
  • AnsichtMode (Enumeration): Analysis Mode.
    • Zebra (default): Applies a black and white pattern.
    • Rainbow: Applies a rainbow pattern.
    • Isophote: Applies a rainbow pattern but only shows stripes according to AnsichtIsoAngles and AnsichtIsoTolerance.
  • AnsichtShading (FloatConstraint): Amount of shading on the analysis overlay.

AnalysisOptions

  • AnsichtColor1 (Color): First color (Zebra, Rainbow, Isophote).
  • AnsichtColor2 (Color): Second color (Zebra, Rainbow, Isophote).
  • AnsichtIsoAngles (FloatList): Angles of isophote curves.
  • AnsichtIsoTolerance (Float): Angular tolerance of isophote curves.
  • AnsichtRainbowAngle1 (FloatConstraint): Start angle of the rainbow.
  • AnsichtRainbowAngle2 (FloatConstraint): End angle of the rainbow.
  • AnsichtStripesNumber (Integer): Number of stripes (Zebra, Rainbow).
  • AnsichtStripesRatio (FloatConstraint): Relative width of stripes (Zebra).

Daten

Group

  • Daten-Eigenschaft (versteckt)Sources (LinkList): Eingabe Unter-Formen.
