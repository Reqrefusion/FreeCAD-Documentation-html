Beschreibung

Der Befehl FreeGrid LagerGitter erzeugt ein Lagergitter.

Dieses Werkzeug ist Teil des Arbeitsbereich FreeGrid, dieser ist ein externer Arbeitsbereich, der mit dem Addon-Manager installiert werden kann.

Anwendung

  1. Es gibt verschiedene Wege das Werkzeug aufzurufen:
    • Drücke den Knopf Storage grid.
    • Wähle den Befehl FreeGrid → Lagergitter aus dem Menü.
  2. Es wird ein Lagergitter mit den Eigenschaften, basierend auf den aktuellen Voreinstellungen erzeugt.
  3. Wähle optional das Objekt und ändere seine Eigenschaften im Eigenschaften-Editor.

Eigenschaften

Ein FreeGrid-StorageBox-Objekt wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem hat es die folgenden zusätzlichen Eigenschaften:

Daten

Grid features

  • DatenCorner Connectors (Bool): Default is true. Add cavities for corner connectors.
  • DatenExtra Bottom Material (Length): Default is 16 mm. Extra thickness under the grid.
  • DatenIs Subtractive (Bool): Default is false. Create a grid suitable for subtractive manufacturing.

Magnet mount

  • Daten-EigenschaftInclude Magnets (Bool): Standard ist true. Magnetaufnahmen einbeziehen.
  • Daten-EigenschaftMagnet Diameter (Length): Standard ist 6 mm. Durchmesser des Magnets.
  • Daten-EigenschaftMagnet Height (Length): Standard ist 2 mm. Höhe des Magnets.

Size

  • Daten-EigenschaftDepth (IntegerConstraint): Standard ist 2. Anzahl der 50 mm-Einheiten in der Y-Richtung des Objekts.
  • Daten-EigenschaftWidth (IntegerConstraint): Standard ist 3. Anzahl der 50 mm-Einheiten in der X-Richtung des Objekts.
