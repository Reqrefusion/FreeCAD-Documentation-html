Beschreibung

Diese Gleichung beschreibt die mechanischen Eigenschaften von Festkörpern.

Informationen zur Mathematik der Gleichung findet man im Elmer-Modellhandbuch im Abschnitt Lineare Elastizität.

Anwendung

  1. Einen Löser Elmer hinzufügen.
  2. Diesen in der Baumansicht auswählen.
  3. Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
    • Die Schaltfläche Elastizitätsgleichung drücken.
    • Aus dem Menü die Option Lösen → Elastizitätsgleichung auswählen.
  4. Die Gleichungslöser-Einstellungen oder die allgemeinen Löser-Einstellungen bei Bedarf ändern.

Hinweis: Für Analysen der nichtlinearen Elastizität muss die Verformungsgleichung benutzt werden. (eingeführt in 0.21) Die Elastizitätsgleichung gilt nur für lineare Elastizität.

Hinweis: Wenn man mehr als einen CPU-Kern für den Löser verwendet (eingeführt in 0.21), kann man die Standardeinstellungen des Lösers nicht verwenden. Die Verwendung nur einer CPU und der Standard-Löser-Einstellungen ist jedoch in vielen Fällen schneller als die Verwendung mehrerer CPUs, da der Elastizitäts-Löser nur dann schnell ist, wenn Daten-EigenschaftLinear Solver Type auf Direct (Standard, beschrieben hier) eingestellt ist. Für die Lösung mit mehreren CPUs kann nur die Daten-EigenschaftLinear Direct Method MUMPS verwendet werden. MUMPS ist jedoch nicht als direkter Download frei verfügbar.

Löser-Einstellungen

Die allgemeinen Löser-Einstellungen findet man unter Elmer-Löser-Einstellungen.

The elasticity equation provides these special settings:

  • DatenCalculate Pangle: If the principal angles should be calculated.
  • DatenCalculate Principal: If all stresses should be calculated.
  • DatenCalculate Strains: If strains will be calculated. This will also calculate the stresses, even if DatenCalculate Principal or DatenCalculate Stresses is false.
  • DatenCalculate Stresses: If stresses should be calculated. Compared to DatenCalculate Principal the Tresca yield criterion and the principal stress will not be calculated.
  • DatenConstant Bulk System: See the Elmer manual for more info.
  • DatenDisplace Mesh: If mesh can be deformed. This is by default true and must be set to false for eigenfrequency analyses.
  • DatenFix Displacement: If displacements or forces are set. thereby DatenModel Lumping is automatically used.
  • DatenGeometric Stiffness: Considers the geometric stiffness of the body.
  • DatenIncompressible: Computation of incompressible material in connection with viscoelastic Maxwell material and a custom DatenVariable.
  • DatenMaxwell Material: Compute the viscoelastic material model.
  • DatenModel Lumping: Uses model lumping.
  • DatenModel Lumping Filename: File to save the results from the model lumping.
  • DatenStability Analysis: If true DatenEigen Analysis becomes a stability analysis (buckling analysis). Otherwise a modal analysis is performed.
  • DatenUpdate Transient System: See the Elmer manual for more info.
  • DatenVariable: The variable for the elasticity equation. Only change this if DatenIncompressible is set to true in accordance to the Elmer manual.

Eigenvalues:

  • DatenEigen Analysis: If an eigen analysis should be performed (calculation of eigenmodes and eigenfrequencies).
  • DatenEigen System Complex: Should be true if the eigen system is complex. it must be false for a damped eigen value analyses.
  • DatenEigen System Compute Residuals: Computes residuals of the eigen value system.
  • DatenEigen System Damped: Set a damped eigen analysis. Can only be used if DatenLinear Solver Type is Iterative.
  • DatenEigen System Select: Selection of which eigenvalues are computed. Note that the selection of Largest* cause an infinite run for recent Elmer solver (as of August 2022).
  • DatenEigen System Tolerance: Convergence tolerance for iterative eigensystem solve. The default is 100 times the DatenLinear Tolerance.
  • DatenEigen System Values: The number of the highest eigenmode that should be calculated.

Gleichung:

  • Daten-EigenschaftPlane Stress: Berechnet die Lösung entsprechend der ebenen Spannungslage. Gilt nur für 2D-Geometrien.

Informationen zum Analyse-Merkmal

Die Elastizitätsgleichung berücksichtigt die folgenden Analysefunktionen, sofern sie eingestellt sind:

  • Fixed boundary condition
  • Displacement boundary condition
  • Force load
  • Initial temperature condition
  • Pressure load
  • Gravity load
  • Spring

Hinweis

  • Mit Ausnahme von Berechnungen in 2D ist es für alle oben genannten Analysefunktionen wichtig, dass sie auf eine Fläche wirken. Funktionen in 3D, die auf Linien oder Eckpunkte eingestellt sind, werden vom Elmer-Löser nicht erkannt.

Eigenmodus-Analyse

Um eine Eigenmodenanalyse (Berechnung der Eigenmoden und Eigenfrequenzen) durchzuführen, muss man

  1. Daten-EigenschaftEigen Analysis auf true setzen.
  2. Daten-EigenschaftDisplace Mesh auf false setzen.
  3. Daten-EigenschaftEigen System Values auf die höchste Anzahl der Eigenmoden setzen, die einen interessieren. Je kleiner diese Zahl ist, desto kürzer ist die Laufzeit des Lösers, da höhere Moden bei der Berechnung weggelassen werden können.
  4. Eine feste Randbedingung hinzufügen und mindestens eine Fläche des Körpers als fixiert festlegen.
  5. Den Löser ausführen.

Es wird dringend empfohlen Daten-EigenschaftLinear Solver Type auf Direkt (Standardeinstellung) einzustellen, da dies wesentlich schneller ist und die Ergebnisse genauer sind.

Knick-Analyse

Um eine Knickanalyse durchzuführen, muss genauso vorgegangen werden wie bei einer Eigenmode-Analyse, und zusätzlich:

  • Setze DatenStability Analysis auf true

Ergebnisse

Die verfügbaren Ergebnisse hängen von der Löser-Einstellung ab. Wenn keine der Daten-EigenschaftCalculate *-Einstellungen auf true gesetzt wurde, wird nur die Verdrängung berechnet. Andernfalls stehen auch die entsprechenden Ergebnisse zur Verfügung. Wenn die Daten-EigenschaftEigen Analysis auf true gesetzt wurde, werden alle Ergebnisse für jeden berechneten Eigenmode verfügbar sein.

Wenn Daten'Eigenanalyse auf true gesetzt wurde, werden die Eigenfrequenzen am Ende des Löser-Protokolls im Löser-Dialogfeld und auch im Dokument LöserElmerAusgabe’ ausgegeben, die nach Abschluss des Lösers in der Baumansicht erstellt wird.

Hinweis: Der Eigenmodenverschiebung d→-Vektor hat einen beliebigen Wert, da das Ergebnis

d→=c⋅u→

wobei u→ ist der Eigenvektor und c ist eine komplexe Zahl.
