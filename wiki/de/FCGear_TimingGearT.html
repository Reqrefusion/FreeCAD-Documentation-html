Beschreibung

Der Befehl FCGear ZahnriemenscheibeT erstellt eine Zahnriemenscheibe Form T, ein TimingGearT-Objekt, …

Oben: Zahnriemenscheibe Form T

Anwendung

  1. Zum Arbeitsbereich Gear wechseln.
  2. Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
    • Die Schaltfläche ZahnriemenscheibeT drücken.
    • Den Menüeintrag Gear → ZahnriemenscheibeT auswählen.
  3. Die Einstellungen den geforderten Randbedingungen entsprechend ändern (siehe Eigenschaften).

Eigenschaften

Siehe auch: Eigenschaften-Ansicht.

Eine FCGear-ZahnriemenscheibeT, ein TimingGearT-Objekt, wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem hat es die folgenden zusätzlichen Eigenschaften:

Daten

base

  • Daten-Eigenschaftalpha (Length): Der Standardwert ist 40°. Winkel der Zahnflanken.
  • Daten-Eigenschaftheight (Length): Der Standardwert ist 5 mm. Extrusionshöhe.
  • Daten-Eigenschaftnum_teeth (Integer): Der Standardwert ist 15. Zähnezahl.
  • Daten-Eigenschaftpitch (Length): Der Standardwert ist 5 mm. Teilung des Zahnrads.
  • Daten-Eigenschafttooth_height (Length): Der Standardwert ist 5 mm. Radiale Höhe des Zahns.
  • Daten-Eigenschaftu (Length): Der Standardwert ist 5 mm. Radialer Abstand vom Zahnkopf zum Teilkreis.

fillets

  • Daten-Eigenschafthead_fillet (FloatConstraint): Standardwert ist 0.4. Eine Verrundung am Zahnkopf, radius = head_fillet x module.
  • Daten-Eigenschaftroot_fillet (FloatConstraint): Standardwert ist 0.4. Eine Verrundung am Zahnfuß, radius = root_fillet x module.

tolerance

  • Daten-Eigenschaftbacklash (Length): Standardwert ist 0 mm. Die Bogenlänge auf dem Teilkreis, um den die Zahndicke reduziert wird.

Skripten

Die Macht von Python nutzen, um das Erstellen von Zahnrädern zu automatisieren: 

import FreeCAD as App
import FreeCADGui as Gui
import freecad.gears.commands
gear = freecad.gears.commands.CreateTimingGearT.create()
gear.num_teeth = 12
gear.height = 5
App.ActiveDocument.recompute()
Gui.SendMsgToActiveView("ViewFit")
