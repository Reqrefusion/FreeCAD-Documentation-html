Beschreibung

Der Befehl FCGear ZahnriemenscheibeT erstellt eine Zahnriemenscheibe Form T, ein TimingGearT-Objekt, …

Oben: Zahnriemenscheibe Form T

Anwendung

  1. Zum Arbeitsbereich Gear wechseln.
  2. Es gibt mehrere Möglichkeiten, den Befehl aufzurufen:
    • Die Schaltfläche ZahnriemenscheibeT drücken.
    • Den Menüeintrag Gear → ZahnriemenscheibeT auswählen.
  3. Die Einstellungen den geforderten Randbedingungen entsprechend ändern (siehe Eigenschaften).

Eigenschaften

Ein FCGear-TimingGearT-Objekt wird von einem Part-Formelement abgeleitet und erbt alle seine Eigenschaften. Außerdem hat es die folgenden zusätzlichen Eigenschaften:

Daten

base

  • Datenalpha (Length): Default is 40°. Angle of tooth flanks.
  • Datenheight (Length): Default is 5 mm. Extrusion height.
  • Datennum_teeth (Integer): Default is 15. Number of teeth.
  • Datenpitch (Length): Default is 5 mm. Pitch of gear.
  • Datentooth_height (Length): Default is 5 mm. Radial height of tooth.
  • Datenu (Length): Default is 5 mm. Radial distance from tooth-head to pitch circle.

fillets

  • Datenhead_fillet (FloatConstraint): Default is 0.4. A fillet for the tooth-head, radius = head_fillet x module.
  • Datenroot_fillet (FloatConstraint): Default is 0.4. A fillet for the tooth-root, radius = root_fillet x module.

tolerance

  • Datenbacklash (Length): Default is 0 mm. The arc length on the pitch circle by which the tooth thickness is reduced.
