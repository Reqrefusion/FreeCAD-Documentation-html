Beschreibung

The Cables Edit command is an extended version of the Draft Edit command which is compatible with the WireFlex objects. It preserves all properties of WireFlex (especially points attachments) while adding or removing vertexes. Editing can be performed with non zero Fillet Radius property and with any value of Path Type property. While in Cables Edit mode, all WireFlex editing commands can be executed using node or edge context menu: Add Vertex, Delete Vertex, Attach Vertex, Remove Vertex Attachment.

Der Befehl Cables Bearbeiten war im Arbeitsbereich Cables Version 0.1.4 und früheren Versionen nicht verfügbar.

Anwendung

  1. Optionally select one or more WireFlex objects. Note that although multiple objects can be in Cables Edit mode, objects can only be edited one at a time.
  2. There are several ways to invoke the command:
    • If you have not yet selected an object: double-click an object in the Tree view. This only works for WireFlex objects.
    • Press the Edit button.
    • Select the Cable Wires → Edit command from the menu.
    • Use the keyboard shortcut: D then E.
    • For a single object: right-click in the Tree view or the 3D view and select the Cable Wires → Edit option from the context menu.

Die nächsten Schritte und Optionen sind dieselben wie in Draft Bearbeiten.

Hinweise

  • Den Befehl Draft Bearbeiten nicht benutzen, um die Leitungsverlaufs’’’-Objekte zu bearbeiten. Es kann die korrekte Reihenfolge der Punktbefestigungen zerstören und zu Fehlern und unerwartetem Verhalten dieser Objekte führen.

Optionen

Die grundlegenden Optionen sind dieselben wie in Draft Bearbeiten. Das Kontextmenü für Knoten oder Kanten wird auf dieselbe Weise aufgerufen wie in Draft Bearbeiten.

Hier sind die Knoten- oder Kantenkontextmenüs, die nur für Leitungsverlauf gelten.

  1. Node context menu: Attach point - AttachVertex command is executed. One external vertex has to be selected first and then keeping the CTRL pressed use the Attach point context menu (e.g. by holding down ALT key and clicking or double clicking the node). This option is visible only for not attached point.
  2. Node context menu: Remove point attachment - RemoveVertexAttachment command is executed. This option is visible only for attached point.
  3. Edge context menu: Make edge horizontal - Makes the edge horizontal by moving the vertex which is closer to the mouse cursor.
  4. Edge context menu: Make edge vertical - Makes the edge vertical by moving the vertex which is closer to the mouse cursor.
  5. Edge context menu: Make edge coaxial - Makes the edge coaxial to adjacent edge by moving the vertex which is closer to the mouse cursor (introduced in version 0.3.0).
