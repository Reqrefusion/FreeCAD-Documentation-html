Die Arbeitsbereich Drawing wurde in FreeCAD 0.17 durch die Arbeitsbereich TechDraw ersetzt. Beide Arbeitsbereiche sind zwar in Version 0.17 enthalten, der Arbeitsbereich Drawing wird jedoch nicht mehr weiterentwickelt und könnte in zukünftigen Versionen von FreeCAD entfernt werden. Benutzern wird empfohlen, auf den Arbeitsbereich TechDraw umzusteigen, der auch Bemaßungs- und Anmerkungswerkzeuge enthält.

Einleitung

Der Arbeitsbereich Zeichnungsbemaßung fügt mächtige Werkzeuge zur Bemaßungs- und Anmerkungserstellung zum Arbeitsbereich Zeichnen (siehe Forumsthema).

Arbeitsbereich Drawing dimensioning für FreeCAD v0.15 oder höher. Man beachte, dass dieser Arbeitsbereich experimentell ist und noch Fehler enthält.

Beabsichtigter Arbeitsablauf

  • Eine Zeichnungsseite und eine Zeichnung des Teils mit dem Arbeitsbereich Drawing Dimensioning erstellen.
  • Zum Arbeitsbereich Drawing Dimensioning wechseln, um dieser Zeichnung Maße hinzuzufügen.

Funktionen

  • Lineare Bemaßung
  • Kreisförmige und radiale Bemaßung
  • Winkelbemaßung
  • Mittellinien
  • Hinzufügen, Bearbeiten und Verschieben von Bemaßungstext
  • Löschen von Bemaßungen
  • Mit "Anfasspunkt hinzufügen" eine Bemaßung oder ein Symbol an einer beliebigen Stelle in der Zeichnung hinzufügen

Begrenzungen

  • Keine parametrische Aktualisierung, wenn die Zeichnung aktualisiert wird, müssen die Bemaßungen neu erstellt werden
  • funktioniert nur mit FreeCAD Version 0.15+

Installation

Automatische Installation

Dieser Arbeitsbereich kann über den Addon-Manager installiert werden. Für die manuelle Installation siehe Weitere Arbeitsbereiche installieren.

Linux-Installationsanweisungen (von GitHub)

Um diesen Arbeitsbereich zu verwenden, klont man dieses Git-Verzeichnis in das FreeCAD-Mod-Verzeichnis und installiert die Python-Bibliotheken pyside und numpy. Auf einem Linux-Debian-basierten System wie Ubuntu kann die Installation wie folgt über BASH durchgeführt werden $ sudo apt-get install git python-numpy python-pyside $ mkdir ~/.FreeCAD/Mod $ cd ~/.FreeCAD/Mod $ git clone https://github.com/hamish2014/FreeCAD_drawing_dimensioning.git

Nach der Installation kann man mit git ganz einfach auf die neueste Version aktualisieren: $ cd ~/.FreeCAD/Mod/FreeCAD_drawing_dimensioning $ git pull $ rm *.pyc

Windows-Installationsanweisungen (von GitHub)

Getestet mit 015.4415 Development Snapshot auf einem Windows 7 64-Bit-System (danke an BPLRFE).

  • Das Git-Repository als ZIP herunterladen
  • vorausgesetzt, FreeCAD ist installiert in "C:\PortableApps\FreeCAD 0_15", im Windows-Explorer zu "C:\PortableApps\FreeCAD 0_15\Mod" gehen
  • Neues Verzeichnis mit dem Namen "DrawingDimensioning". anlegen
  • Das heruntergeladene Repository in "C:\PortableApps\FreeCAD 0_15\Mod\DrawingDimensioning" entpacken

FreeCAD verfügt nun über einen neuen Arbeitsbereich-Eintrag namens "DrawingDimensioning".

Pyside und Numpy sind in den FreeCAD 0.15 dev-Snapshots integriert, sodass diese Python-Pakete nicht einzeln installiert werden müssen.

Um auf die neueste Version zu aktualisieren, den Ordner "DrawingDimensioning" löschen und das Git-Repository erneut herunterladen.

Mac-Installationsanweisungen (von GitHub)

Den Ordner "Zeichnungsbemaßung" in das Verzeichnis FreeCAD.app/Contents/Mod kopieren oder entpacken.

wobei FreeCAD.app der Ordner ist, in dem FreeCAD installiert ist. (Danke, PLChris)

Einstellungen für die Bemaßung festlegen

Die Einheitenpräferenzen werden aus den allgemeinen Einheitenpräferenzen übernommen (mit Ausnahme der Anzahl der Dezimalstellen!). Um die Einheitenpräferenzen festzulegen, geht man zu Bearbeiten → Einstellungen → Allgemein → Einheiten

So richtet man den gewünschten Bemaßungsstil ein:

  1. FreeCAD öffnen
  2. Zum Arbeitsbereich Drawing dimensioning wechseln
  3. Bearbeiten → Einstellungen → Drawing dimensioning

Symbole

Lineares Maße

Lineare Maße / Lineare Maßstapel

Wenn man auf „Lineare Maße“ oder „Lineare Maßstapel“ klickt, wird ein Menü mit verschiedenen Optionen angezeigt. Wiederholen: Wenn diese Option aktiviert ist, wird das Menü nach der Definition eines Maß nicht geschlossen. Rasteroptionen: Ermöglicht das Ausrichten der Bemaßungen an einem Raster.

Raster ein schaltet die Sichtbarkeit des Rasters um.

Abstand: Legt den Abstand des Hauptrasters fest (Standard: 1 mm)

Anzeigedauer: Legt fest, wie oft die Zeilen angezeigt werden (Standard: 20)

Farbe: Linienfarbe der Gitterlinien

Linienbreite: Linienbreite der Gitterlinien (Standard: 0,15) Einheitenoptionen: Legt die Einheiten der Abmessungen fest

Bearbeiten → Einstellungen → Einheit: Gleiche Einheiten wie in den FreeCAD-Einstellungen verwenden

mm: Einheiten müssen in Millimeter angegeben werden

Zoll: Einheiten müssen in Zoll angegeben werden

m: Einheiten müssen in Meter angegeben werden

Benutzerdefiniert: Benutzerdefinierte Skalierung der Einheiten verwenden (Standard: 1/mm) Einstellungen: Einstellungen für die Darstellung der Maße festlegen

Kompakt:

Formatmaske: Ändert das Format der Dimension (Standardwert „%(value)3.3f“). Überschreiben mit manuellem Text möglich

Pfeile: Ändert den Pfeilstil (Auto/Ein/Aus/Aus)

Auto-Platzierung von Text: Maßangaben manuell platzieren

Komma: Komma anstatt Punkt benutzen

Lücke: (Standard: 2)

Überschreitung: (Standard: 1)

arrowL1: erste Länge des Pfeils (Standard: 3)

arrowL2: zweite Länge des Pfeils (Standard: 1)

arrowW: Breite des Pfeils (Standard: 2)

Strichstärke: Die Strichstärke der Maßlinien ändern (Standard: 0,3)

lineColor: Farbe der Maßlinien ändern

Texteigenschaften: Schriftart, Schriftgröße und Farbe (Standard: Verdana, 3,6, rot) wählen

Auto-Platzierungsversatz:

Als Standard festlegen: Einstellungen als Standard akzeptieren

Hinweise:

  • Die Platzierung der Dimensionen kann nachträglich geändert werden, indem man die Dimension in der Baumansicht auswählt und die Werte im Abschnitt Platzierungsklicks auf der Registerkarte Daten ändert
  • Die Änderung der Bemaßung auf eine halbe lineare Bemaßung kann vorgenommen werden, indem die Bemaßung in der Baumansicht ausgewählt und der Wert für „half Dimension_linear” auf „true” gesetzt wird. Die Bemaßung verwendet dann die Hälfte bis zum zweiten Klickpunkt.
