Einleitung

Templater ist ein Externe Arbeitsbereich und gehört nicht zur Standardinstallation von FreeCAD. Sein Ziel ist es, Werkzeuge zum einfachen Erstellen von TechDraw-Vorlagen im gewünschten Format sowie weitere Werkzeuge für die Zeichnungserstellung bereitzustellen.

Installation

Dieser Arbeitsbereich kann mit dem Addon-Manager installiert werden. Für die manuelle Installation siehe Weitere Arbeitsbereiche installieren; eine Zip-Datei kann von hier heruntergeladen werden.

Werkzeuge

  • Hilfsansicht: Fügt eine Hilfsansicht zum aktiven Zeichnungsblatt hinzu, die auf zwei Knotenpunkten basiert, die in einer vorhandenen Ansicht ausgewählt werden.
  • Toleranzrahmen: Fügt einen Toleranzrahmen oder einen Bezugsrahmen zur ausgewählten Ansicht hinzu.
  • NeueVorlageMulti: Erstellt eine Vorlage und fügt sie als neues Zeichnungsblatt in das aktive Dokument ein. Dieses Werkzeug ermöglicht das Erstellen einer Vorlage in vielen unterschiedlichen Formaten und mit einigen optionalen Elementen, wie einem Rahmen, einem Schriftfeld, einer Stückliste usw.
  • NeueVorlageWiki: Erstellt eine kleine Beispielvorlage und fügt sie als neues Zeichnungsblatt in das aktive Dokument ein. Siehe die Anleitung TechDraw TemplateGenerator, um zu erfahren, wie eine fast identische Vorlage per Skript erstellt wird.
