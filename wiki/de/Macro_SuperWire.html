Beschreibung

Dieses Makro erstellt einen Linienzug aus ausgewählten Objekten (Linien und Bögen), in denen normale Linienzug-Erstellungsverfahren fehlschlagen.
Achtung, es wird eine aktuelle Version von FreeCAD benötigt, damit dies funktioniert.

Skript

Werkzeugleisten-Symbol

Macro_SuperWire.FCMacro 

import FreeCAD,FreeCADGui,Part
try:
    import DraftGeomUtils as fcgeo
except:
    from draftlibs import fcgeo

sel = FreeCADGui.Selection.getSelection()
if not sel:
   FreeCAD.Console.PrintWarning("Select something first!")
else:
   elist = []
   for obj in sel:
       if hasattr(obj,"Shape"):
           elist.append(obj.Shape.Edges[0])
   wire = fcgeo.superWire(elist)
   if wire:
       Part.show(wire)
   else:
       FreeCAD.Console.PrintError("SuperWire operation failed!")
