Description

The Draft Snap Widget is an alternative for the Draft Snap toolbar. The widget is only available in the Draft Workbench. It is an optional GUI element that is located in the Status Bar.

The Draft Snap Widget

Usage

See Draft Snap.

The menu of the widget

Preferences

See also: Preferences Editor and Draft Preferences.

  • The Draft Snap Widget is optional: Edit → Preferences → Draft → Interface → Show the Draft Snap Widget in the Draft Workbench.
