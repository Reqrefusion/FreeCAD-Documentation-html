Opis

The FreeGrid Sketch tool generates an inner box profile sketch.

This tool is part of the FreeGrid Workbench, an external workbench that can be installed with the Addon Manager.

Usage

  1. Optionally select an existing storage box.
  2. There are several ways to invoke the tool:
    • Press the Sketch button.
    • Select the FreeGrid → Sketch option from the menu.
  3. If no box has been selected:
    1. The Sketch size task panel opens.
    2. Specify the Width and Depth in terms of 50mm units.
    3. Optionally check the Include open front checkbox to create a sketch for an open box.
    4. Press the OK button.
  4. A sketch is created.
