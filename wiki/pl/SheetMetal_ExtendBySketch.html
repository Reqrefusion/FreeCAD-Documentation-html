Wydłuż ściankę

Opis

Polecenie Wydłuż przez szkic wydłuża płytę blachy w miejscu wskazanej krawędzi ściany.

W połączeniu ze szkicem konturu może utworzyć geometrię zazębiającą się, aby zamknąć profil:

Trzy profile ze szkicami obrysu do dodania → trzy wyniki

Użycie

(Przygotuj szkic dla zazębiających się zakładek)
  1. Wybierz jedną lub więcej ścian krawędziowych do wydłużenia.
  2. Naciśnij Ctrl i wybierz szkic w Widoku drzewa lub Widoku 3D, aby dodać go do zaznaczenia.
  3. Istnieje kilka sposobów uruchomienia polecenia:
    • Naciśnij przycisk Wydłuż przez szkic.
    • Wybierz opcję Sheet Metal → Wydłuż przez szkic z menu.
    • Kliknij prawym przyciskiem myszy w Widoku drzewa lub Widoku 3D i wybierz opcję Sheet Metal → Wydłuż według szkicu z menu kontekstowego.
    • Użyj skrótu klawiaturowego: T.
  4. Zostanie utworzony obiekt Extend i otworzy się Panel zadań Extend properties.
    • Opcjonalnie naciśnij przycisk Wybierz i dodaj lub usuń ściany.
    • Naciśnij przycisk Podgląd, aby zakończyć wybór i wyświetlić zmiany.
    • Opcjonalnie naciśnij przycisk Wydłuż szkic..., aby wybrać inny szkic.
    • Opcjonalnie zaznacz pole wyboru Przetnij prześwit i dostosuj wartość, aby zrobić miejsce dla zazębiających się zakładek.
  5. Naciśnij przycisk OK, aby zakończyć polecenie i zamknąć Panel zadań.

Panel zadań

Kliknij dwukrotnie istniejący obiekt Extend w Widoku drzewa, aby ponownie otworzyć Panel zadań i edytować parametry.

  • Wybiezz: Zmienia liczbę ścian we właściwości Obiekt bazowy.
  • Wydłuż szkic…: Zastępuje szkic we właściwości Szkic.
  • Pole wyboru Przetnij prześwit: Przełącza właściwość Użyj odjęcia.
  • Wartość Przetnij prześwit: Ustawia właściwość Odsunięcie.
  • Udoskonal: Przełącza właściwość Udoskonal.

Uwagi

  • To polecenie tworzy ten sam obiekt Extend co polecenie Wydłuż ścianę. To, czy właściwość DANESzkic jest pusta, czy nie, decyduje o tym, który Panel zadań jest wyświetlany podczas edycji obiektu Extend.
  • Zobacz stronę Wydłuż ścianę, aby uzyskać więcej informacji.

Właściwości

See also: Property View.

A SheetMetal Extend object is derived from a Part Feature object or, if it is inside a PartDesign Body, from a PartDesign Feature object, and inherits all its properties. It also has the following additional properties:

Data

Parameters

  • DANEbase Object (LinkSub): "Base object". Link to the object and the planar faces to be extended.
  • DANEgap1 (Distance): "Gap from the left side". Default: 0,00 mm.
  • DANEgap2 (Distance): "Gap from the right side". Default: 0,00 mm.
  • DANElength (Length): "Length of Wall". Default: 10,00 mm.
    • extension length if DANEreversed = false.
    • cut out length if DANEreversed = true.
  • DANEreversed (Bool): "Reverse extend direction (cut)". Default: false. Toggles between extension and cut out.

Parameters Ext

  • DANEOffset (Distance): "Offset for subtraction". Default: 0,20 mm. Clearance for interlocking geometry
  • DANERefine (Bool): "Use Refine". Default: true.
  • DANESketch (Link): "Wall Sketch". Link to the sketch defining the interlocking geometry. Contains the selected sketch by default.
  • DANEUse Subtraction (Bool): "Use Subtraction". Default: false Toggles if intersected walls of the same object are cut out for tabs defined by the linked Sketch.
