Opis

Polecenie Dodaj brzeg tworzy brzegi na wybranych krawędziach. Zmieniając właściwość DANEkąt kołnierza, można go przekształcić w brzeg.

Brzeg zazwyczaj składa się z gięcia cylindrycznego o kącie około 180° oraz płaskiego pasa (ścianki).

Dwie wybrane krawędzie → dwa zagięte brzegi

Użycie

  1. Wybierz jedną lub więcej krawędzi bryły.
  2. Istnieje kilka sposobów na uruchomienie tego polecenia:
    • Naciśnij przycisk Utwórz brzeg.
    • Wybierz opcję Sheet Metal → Utwórz brzeg z menu.
    • Kliknij prawym przyciskiem myszy w Widoku drzewa lub Widoku 3D i wybierz opcję Sheet Metal → Utwórz brzeg z menu kontekstowego.
    • Użyj skrótu klawiaturowego: Z.
  3. Tworzony jest obiekt Hem, który składa się z jednego nowego brzegu na każdej wybranej krawędzi, a Panel zadań Hem properties zostanie otwarty:
    • Z wybraną kartą Ogólne:
      • Opcjonalnie naciśnij przycisk Wybierz, aby dodać więcej krawędzi.
        • Opcjonalnie naciśnij przycisk Wyczyść zaznaczenie, aby wyczyścić listę wybranych krawędzi.
        • Naciśnij przycisk Podgląd, aby zakończyć wybór i wyświetlić zmiany.
      • Opcjonalnie dostosuj parametry gięcia na tej karcie. Zobacz sekcję Ogólne w opisie Panelu zadań poniżej.
    • Opcjonalnie przełącz się na inną kartę, aby dostosować więcej parametrów. Zobacz sekcję Odsunięcia lub Fazowanie w opisie Panelu zadań poniżej.
  4. Naciśnij przycisk OK, aby zakończyć polecenie i zamknąć Panel zadań.
  5. Opcjonalnie dostosuj parametry w Widoku właściwości.

Panel zadań

Kliknij dwukrotnie istniejący obiekt Hem w Widoku drzewa, aby ponownie otworzyć Panel zadań i edytować parametry.

Ogólne


  • Wybierz: Zmienia liczbę krawędzi we właściwości Obiekt bazowy.
  • Typ: Ustawia właściwość Typ brzegu.
    • Płaski aktywuje następujące podopcje:
      • Szerokość: Ustawia właściwość Szerokość.
      • Odwróć ściankę: Przełącza właściwość Odwróć.
      • Szerokość z gięciem: Przełącza właściwość Szerokość z gięciem.
    • Otwarty aktywuje następujące podopcje:
      • Szerokość: Ustawia właściwość Szerokość.
      • Odwróć ściankę: Przełącza właściwość Odwróć.
      • Szerokość z gięciem: Przełącza właściwość Szerokość z gięciem.
      • Otwór: Ustawia właściwość Otwór.
    • Łezka aktywuje następujące podopcje:
      • Szerokość: Ustawia właściwość Szerokość.
      • Odwróć ściankę: Przełącza właściwość Odwróć.
      • Szerokość z gięciem: Przełącza właściwość Szerokość z gięciem.
      • Promień gięcia: Ustawia właściwość Promień gięcia.
      • Otwarta: Przełącza właściwość Otwarta.
        • Otwór: Ustawia właściwość Otwór.
    • Zwinięty aktywuje następujące podopcje:
      • Promień gięcia: Ustawia właściwość Promień gięcia.
      • Otwarta: Przełącza właściwość Otwarta.
        • Kąt zwijania: Ustawia właściwość Kąt zwijania.
  • Pozycja: Ustawia właściwość Typ gięcia.
  • Rozwiń: Przełącza właściwość Rozwiń.

Odsunięcia

  • Gap A: Sets the gap1 property.
  • Gap B: Sets the gap2 property.
  • Rectangle and Round radio buttons: Toggle the relief Type property.
  • Width: Sets the reliefw property.
  • Depth: Sets the reliefd property.

Miter

(Not available for rolled hems)

  • Auto Miter: Toggles the Auto Miter property.
    If checked:
    • Minimum Gap: Sets the minGap property.
    • Max Extend Distance: Sets the max Extend Dist property.
    If unchecked:
    • Angle 1: sets the miterangle1 property.
    • Angle 2: sets the miterangle2 property.


Notes

  • See Create a sheet metal object for a basic workflow.

Properties


See also: Property View.

A SheetMetal Hem object is derived from a Part Feature object or, if it is inside a PartDesign Body, from a PartDesign Feature object, and inherits all its properties. It also has the following additional properties:

Data

Parameters

  • DANEBend Type (Enumeration): "Bend Type". Material Outside (default), Material Inside, Thickness Outside.
  • DANEHem Type (Enumeration): "Type of Hem". Flat (default), Open, Teardrop, Rolled.
  • DANEInclude Bend (Bool): "Width includes bend geometry". Default: false.
  • DANERoll Angle (Angle): "Roll Angle". Default angle: 90,00°.
  • DANEbase Object (LinkSub): "Base Object". Link to the planar face to receive a hem.
  • DANEgap1 (Distance): "Gap from Left side". Default: 0,00 mm.
  • DANEgap2 (Distance): "Gap from Right side". Default: 0,00 mm.
  • DANEinvert (Bool): "Invert Bend Direction". Default: false.
  • DANEopened (Bool): "Opened Hem". Default: false.
  • DANEopening (Length): "Openening. of Hem". Default: 1,00 mm
  • DANEradius (Length): "Bend Radius", the default value depends on the radius property of the parent feature:
  • DANEwidth (Length): "Width of Wall". Default: 10,00 mm.

Parameters Ex

  • DANEAuto Miter (Bool): "Enable Auto Miter". Default: true.
  • DANEkfactor (FloatConstraint): "Location of Neutral Line. Caution: Using ANSI standards, not DIN.".
    Default: 0,50. K factor (also known as neutral factor) for the bend. Used to calculate bend allowance when unfolding.
  • DANEmax Extend Dist (Length): "Auto Miter maximum Extend Distance". Default: 5,00 mm.
  • DANEmin Gap (Length): "Auto Miter Minimum Gap". Default: 0,20 mm.
  • DANEmin Relief Gap (Length): "Minimum Gap to Relief Cut". Default: 1,00 mm.
  • DANEunfold (Bool): "Shows Unfold View of Current Bend". Default: false
    true unfolds the bend.

Parameters Miterangle

  • DANEmiterangle1 (Angle): "Bend Miter Angle from Left Side". Default angle: 0,00°.
  • DANEmiterangle2 (Angle): "Bend Miter Angle from Right Side". Default angle: 0,00°.

Parameters Relief

  • DANERelief Factor (Float): "Relief Factor". Default: 0,70.
  • DANEUse Relief Factor (Bool): "Use Relief Factor". Default: false.
  • DANErelief Type (Enumeration): "Relief Type". Rectangle (default), Round. Enabled only when a gap value is set.
  • DANEreliefd (Length): "Relief Depth". Default: 1,00 mm. Enabled only when a gap value is set.
  • DANEreliefw (Length): "Relief Width". Default: 0,80 mm. Enabled only when a gap value is set.
