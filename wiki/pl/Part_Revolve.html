Opis

Narzędzie Obrót obraca wybrany obiekt tworząc wyciągnięcie wokół wybranej osi. Dozwolone są następujące typy kształtów, które prowadzą do uzyskania kształtów wyjściowych:

Kształt wejściowy Forma wyjściowa
Wierzchołek Krawędź
Krawędź Powierzchnia
Linia łamana Powłoka
Powierzchnia Bryła
Powłoka Bryła złozona (Compsolid)

Można też użyć Szkicu. Bryły lub bryły złożone nie są dozwolone jako kształty wejściowe. Również standardowe kształty złożone nie są obecnie dozwolone.

Przykład obrotu

Użycie

  1. Opcjonalnie wybierz jeden lub więcej kształtów w Widoku 3D lub w Widoku drzewa.
  2. Istnieje kilka sposobów na wywołanie tego polecenia:
    • Wciśnij przycisk Obrót.
    • Wybierz opcję Część → Obrót z menu.
  3. Otwarty zostanie Panel zadań Obrót.
    • Opcjonalnie kliknij na obiekcie na liście Kształt aby (ponownie) wskazać kształt.
    • Opcjonalnie trzymaj klawisz Shift wciśnięty i kliknij na obiekcie na liście Kształt aby dodać kształt do wskazania lub usunąć go ze wskazania.
    • Ustaw oś obrotu i kąt oraz opcjonalnie inne parametry (zobacz sekcję Właściwości aby uzyskać więcej informacji).
  4. Wciśnij przycisk OK aby zamknąć panel zadań Obrót.
  5. Dla każdego wskazanego kształtu utworzony zostanie jeden obiekt Revolve.

Każdy kształt wejściowy jest umieszczany pod swoim obiektem Revolve.

Panel zadań

  • Przycisk OK tworzy obrót i zamyka panel.
  • Przycisk Cancel zamyka panel bez wykonywania jakiejkolwiek operacji.
  • Lista kształtów: tutaj wybierasz kształty do obrotu. Jeśli zaznaczono wiele obiektów, zostanie utworzonych wiele obiektów Obrót.
  • Sekcja Oś obrotu: określa oś obrotu.
    • Pola Środek X/Y/Z: ustawiają punkt bazowy osi obrotu.
    • Pola Kierunek X/Y/Z: ustawiają kierunek osi obrotu. Kliknij przycisk, aby ustawić kierunek na odpowiadającą oś.
    • Przycisk Wybierz odniesienie: kliknij go, a następnie wybierz krawędź w Widoku 3D. Ta krawędź pojawi się w polu tekstowym pod przyciskiem w formacie "NazwaObiektu:EdgeN". Możesz także wpisać odnośnik ręcznie lub go usunąć. Wartości Kierunek X/Y/Z zostaną wypełnione zgodnie z kierunkiem krawędzi.
  • Pole Kąt: ustawia kąt obrotu w stopniach.
  • Kąt symetryczny: jeśli zaznaczone, obrót będzie wykonywany do przodu i do tyłu o połowę zadanego kąta.
  • Pole wyboru Utwórz bryłę: jeśli zaznaczone, obrót zamkniętej polilinii lub krawędzi utworzy bryłę. Jest zaznaczone domyślnie, jeśli przed wywołaniem narzędzia Obrót była wstępnie zaznaczona zamknięta polilinia.

Uwagi

  • Argument Kąt określa jak daleko obiekt ma być obrócony. Współrzędne przesuwają punkt początkowy osi obrotu względem punktu początkowego układu współrzędnych.
  • Jeśli wybierasz własną oś, liczby definiują kierunek osi obrotu względem układu współrzędnych: jeśli współrzędna Z jest 0 a współrzędne X i Y są niezerowe, oś będzie leżała na płaszczyźnie X-Y. Jej kąt jest taki, że jej styczna jest stosunkiem podanych współrzędnych X i Y.
  • Obiekty typu łącze powiązane z odpowiednimi typami obiektów mogą być również używane jako kształty i do określania osi.
  • Jeśli obiekt do obrócenia przecina oś obrotu, operacja nie powiedzie się w większości przypadków.

Właściwości

Zapoznaj się również z informacjami na stronie: Widok właściwości.

A Part Revolve object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

Extrude

  • DANESource (Link): The shape to revolve.
  • DANEBase (Vector): The base point of the revolution axis.
  • DANEAxis (Vector): The direction of the revolution axis.
  • DANEAxis Link (LinkSub): Link to the edge to use as the revolution axis. Overrides DANEBase and DANEAxis if set.
  • DANEAngle (Angle): The angle span of the revolution. If zero and an arc is used for the DANEAxis Lik, the arc's span will be used.
  • DANESymmetric (Bool): If true, the revolution is extended symmetrically from the profile.
  • DANESolid (Bool): If true, revolving a closed edge or a closed wire will yield a solid. If False, a shell will result.
  • DANEFace Maker Class (String): If DANESolid is true, the facemaker class to use when converting wires to faces, otherwise ignored.
