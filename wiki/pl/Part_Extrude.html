Opis

Narzędzie Wyciągnięcie powoduje wydłużenie kształtu o określoną odległość i w określonym kierunku. Typ kształtu wyjściowego zależy od typu kształtu wejściowego i wybranych opcji.

W najczęstszych scenariuszach poniższa lista zawiera oczekiwany typ kształtu wyjściowego z danego typu kształtu wejściowego,

  • Wyciągnięcie wierzchołka (punktu) spowoduje utworzenie prostej krawędzi (linii).
  • Wyciągnięcie otwartej krawędzi (np. linii, łuku) spowoduje utworzenie otwartej ściany (np. płaszczyzny)
  • Wyciągnięcie zamkniętej krawędzi (np. okręgu) opcjonalnie utworzy zamkniętą ścianę (np. otwarty walec) lub, jeśli parametr bryła ma wartość Prawda, utworzy bryłę (np. zamknięty, pełny walec).
  • Wyciągnięcie otwartej polilinii (np. polilinia środowiska Rysunek Roboczy) spowoduje utworzenie otwartej powłoki (kilka połączonych ścian).
  • Wyciągnięcie zamkniętej polilinii (np. polilinia środowiska Rysunek Roboczy) opcjonalnie utworzy powłokę (kilka połączonych ścian) lub, jeśli parametr bryła ma wartość Prawda, utworzy bryłę
  • Wyciągnięcie ściany (np. płaszczyzny) spowoduje utworzenie bryły (np. prostopadłościanu).
  • Wyciągnięcie Kształtu z tekstu utworzy złożenie brył (ciąg jest złożeniem liter, z których każda jest bryłą)
  • Wyciągnięcie powłoki ze ścian Prawda utworzy bryłę złożoną.

Przykłady wyciągnięcia

Użycie

  1. Opcjonalnie wybierz jeden lub więcej kształtów w Widoku 3D lub w Widoku drzewa.
  2. Istnieje kilka sposobów na wywołanie tego polecenia:
    • Wciśnij przycisk Wyciągnięcie.
    • Wybierz opcję Część → Wyciągnięcie z menu.
  3. Otwarty zostanie Panel zadań Wyciągnięcie.
    • Opcjonalnie kliknij na obiekcie na liście Kształt aby (ponownie) wskazać kształt.
    • Opcjonalnie trzymaj klawisz Shift wciśnięty i kliknij na obiekcie na liście Kształt aby dodać kształt do wskazania lub usunąć go ze wskazania.
    • Ustaw kierunek i długość wyciągnięcia oraz opcjonalnie inne parametry (zobacz sekcję Właściwości aby uzyskać więcej informacji).
  4. Wciśnij przycisk OK aby zamknąć panel zadań.
  5. Dla każdego wskazanego kształtu utworzony zostanie jeden obiekt Extrude.

Każdy kształt wejściowy jest umieszczony pod swoim obiektem Extrude.

Panel zadań

  • Przycisk OK tworzy wyciągnięcie i zamyka panel.
  • Przycisk Zamknij zamyka panel, nie robiąc nic.
  • Przycisk Zastosuj tworzy wyciągnięcie, ale nie zamyka panelu. Następnie możesz wybrać kolejny kształt z listy na dole i utworzyć więcej wyciągnięć.
  • Przyciski wyboru Kierunek: ustawia sposób obliczania kierunku wyciągnięcia.
  • Pole Odwrócone: odwraca kierunek wyciągnięcia.
  • Przycisk Wybierz: kliknij go, a następnie wybierz krawędź w oknie Widoku 3D. Krawędź ta pojawi się w polu tekstowym obok przycisku, w formacie "ObjectName:EdgeN". Możesz również wpisać łącze ręcznie lub je wymazać. Wartości X,Y,Z zostaną wypełnione zgodnie z kierunkiem krawędzi.
  • Przyciski X, Y, Z: Kliknij przycisk, aby ustawić kierunek wyciągnięcia na oś dodatnią. Kliknij go ponownie, aby ustawić go na oś ujemną.
  • Pola wejściowe X, Y, Z: ustawiają lub wyświetlają wektor kierunku wyciągnięcia. Jeśli obie długości wynoszą zero, długość tego wektora ustawia długość wyciągnięcia, a wartości są zawsze w mm, niezależnie od preferencji dotyczących jednostek.
  • Pola długości: ustawiają długość wyciągnięcia. Te pola wejściowe obsługują jednostki.
  • Symetrycznie: rozkłada wyciągnięcie w obu kierunkach, dzięki czemu profil pozostaje ustawiony środkowo.
  • Kąt ścięcia wzdłuż: kąt pochylenia dla kierunku wzdłuż. Dodatni kąt oznacza, że profil jest rozszerzany na drugim końcu wyciągnięcia.
  • Kąt ścięcia przeciwnie: jak wyżej dla kierunku przeciwnego.
  • Pole wyboru Utwórz bryłę: jeśli pole jest zaznaczone, wyciągnięcie zamkniętej krawędzi lub krzywej spowoduje utworzenie bryły. Jest domyślnie zaznaczone, jeśli zamknięta krawędź została wstępnie wybrana przed wywołaniem funkcji Wyciągnięcia.
  • Lista kształtów: tutaj można wybrać kształty do wyciągnięcia. W przypadku wybrania wielu obiektów tworzonych jest wiele obiektów wyciągnięcia.

Uwagi

  • Obiekty typu odnośnik powiązane z odpowiednimi typami obiektów i kontenery App: Część z odpowiednimi widocznymi obiektami wewnątrz mogą być również używane jako profile i do określania kierunku.
  • Panel zadań nie oferuje jeszcze podglądu. Przycisk Zastosuj utworzy obiekt wyciągnięcia za każdym razem, gdy go klikniesz, co może być przydatne jako podgląd. Jednak pozostaną one i kolejne zostaną utworzone po kliknięciu przycisku OK. Cofnij może być przydatne do wyczyszczenia ich przed kliknięciem OK.

Porównanie z wyciągnięciem Projekt Części

Wyciągnięcie środowiska Projekt Części jest również funkcją wyciągnięcia, ale istnieją między nimi istotne różnice:

  • Wyciągnięcie środowiska Część zawsze tworzy samodzielny kształt. Wyciągnięcie środowiska Projekt Części łączy wynik wyciągnięcia z resztą Zawartości.
  • Wyciągnięcie środowiska Część może być umieszczone gdziekolwiek w drzewie modelu. Wyciągnięcie środowiska Projekt Części można umieścić tylko wewnątrz Zawartości środowiska Projekt Części.
  • Wyciągnięcie części może wyciągać dowolny obiekt, który ma geometrię części (kształt OpenCASCADE), z wyjątkiem brył i brył złożonych.
  • Wyciągnięcie środowiska Część może wyciągać pojedyncze ściany innych obiektów. Wyciągnięcie środowiska Projekt Części będzie akceptować jako profil tylko szkic lub ściany obiektów środowiska Projekt Części.

Właściwości

Zapoznaj się również z informacjami na stronie: Widok właściwości.

A Part Extude object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

Extrude

  • DANEBase (Link): The shape to extrude.
  • DANEDir (Vector): The extrusion direction. If DANEDir Mode is Custom you can edit this property, otherwise it is read-only.
  • DANEDir Mode (Enumeration): Sets how DANEDir is controlled. Custom means DANEDir is editable. Edge means DANEDir is obtained from an edge (line) linked by DANEDir Link. Normal means DANEDir is perpendicular to the plane of the input shape.
  • DANEDir Link (LinkSub): Parametric link to an edge (line) that sets the extrusion direction.
  • DANELength Fwd (Distance): Length of extrusion along direction. If both DANELength Fwd and DANELength Rev are zero, the length of the DANEDir vector is used.
  • DANELength Rev (Distance): Length of additional extrusion, against direction.
  • DANESolid (Bool): If true, extruding a closed edge or a closed wire will yield a solid. If False, a shell will result.
  • DANEReversed (Bool): Reverses the extrusion direction.
  • DANESymmetric (Bool): If true, extrusion is centered at the input shape, and the total length is DANELength Fwd. DANELength Rev is ignored.
  • DANETaper Angle (Angle): Applies an angle to the extrusion, so that sides of the extrusion are drafted by the specified angle. Positive angle means the cross-section expands. Inner structures receive the opposite taper angle. This is done to facilitate the design of molds and molded parts.
  • DANETaper Angle Rev (Angle): Sets the taper for the reversed part of the extrusion (the part from DANELength Rev).
  • DANE (Hidden)Face Maker Class (String): Superseded by DANEFace Make Mode (read-only).
  • DANEFace Make Mode (Enumeration): If DANESolid is true, the facemaker class to use when converting wires to faces, otherwise ignored. The options are Simple, Cheese, Extrusion and Bullseye (default).
