Description

The Robot CreateRobot command places a robot in the model.

Usage

  1. Press the Place Robot button.
  2. Select a VRML file (*.vrlm or *.wrl).
  3. Select a kinematic CSV file (*.csv).

Notes

  • Only robots with up to six axes with rotational movement are supported.
  • Robots with translational movements are not supported.
  • The robot must be defined via code and a VRML file, for more info refer to Robot 6-Axis.
  • Predefined robots are no longer available in FreeCAD v1.1 and above. Robot definition files must be downloaded from the website of the manufacturer.
