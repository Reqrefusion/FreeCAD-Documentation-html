Rysunek Roboczy: Połącz

Lokalizacja w menu
Modyfikacja → Połącz

Rysunek Roboczy, BIM
Domyślny skrót
J O
Wprowadzono w wersji
0.18

Rozdziel

Opis

Polecenie Połącz środowiska pracy Rysunek Roboczy łączy linie i polilinie w jedną linię. Polecenie to jest odpowiednikiem polecenia Rozdziel.

Użycie

  1. Punkty końcowe linii i / lub polilinii, które mają zostać połączone, muszą się dokładnie pokrywać. W razie potrzeby najpierw dostosuj położenie punktów, aby upewnić się, że tak jest.
  2. Wybierz dwie lub więcej linii i / lub polilinii.
  3. Istnieje kilka sposobów na wywołanie polecenia:
    • Naciśnij przycisk Połącz.
    • Środowisko pracy Rysunek Roboczy: Wybierz z menu opcję Modyfikacja → Połącz.
    • Środowisko pracy BIM: Wybierz opcję Modyfikacja → Połącz z menu.
    • Użyj skrótu klawiaturowego: J, a następnie O.

Uwagi

  • Liniie i polilinie mogą być również łączone za pomocą narzędzia polilinia lub Ulepsz kształt.
  • Aby połączyć obiekty, które nie są liniami lub poliliniami, możesz spróbować użyć Ulepsz kształt i/lub Rozbij kształt na nich jeden lub więcej razy.

Tworzenie skryptów

Zobacz również stronę: Dokumentacja API generowana automatycznie oraz Podstawy pisania skryptów dla FreeCAD.

Do Połączenia linii należy użyć metody join_wires (dostępne w wersji 0.19) modułu Rysunek Roboczy. Metoda ta zastępuje przestarzałą metodę joinWires. Metoda ta zwraca Brak. 

join_wires(wires)
  • wires jest listą obiektów linii, które mają zostać połączone.

Przykład: 

import FreeCAD as App
import Draft

doc = App.newDocument()

p1 = App.Vector(0, 0, 0)
p2 = App.Vector(500, 0, 0)
p3 = App.Vector(500, 500, 0)
p4 = App.Vector(0, 500, 0)

wire1 = Draft.make_wire([p1, p2])
wire2 = Draft.make_wire([p2, p3])
wire3 = Draft.make_wire([p3, p4])
wire4 = Draft.make_wire([p4, p1])

Draft.join_wires([wire1, wire3, wire2, wire4])
doc.recompute()
