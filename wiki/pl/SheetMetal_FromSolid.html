Opis

Polecenie Z brył konwertuje bryłę lub powłokę na obiekt blaszany (wprowadzone w wersji 0.8.0).

Prostopadłościan z wybraną górną ścianą i krawędziami → Wynikowy obiekt SolidToSheet

Użycie

  1. Wybierz obiekt bazowy (bryłę lub powłokę).
  2. Istnieje kilka sposobów na uruchomienie tego polecenia:
    • Naciśnij przycisk Konwertuj na blachę.
    • Wybierz opcję Sheet Metal → Konwertuj na blachę z menu.
    • Kliknij prawym przyciskiem myszy w Widoku drzewa lub Widoku 3D i wybierz opcję Sheet Metal → Konwertuj na blachę z menu kontekstowego.
    • Użyj skrótu klawiaturowego: C następnie S.
  3. Utworzony zostanie obiekt SolidToSheet, a Panel zadań Konwertuj na blachę zostanie otwarty:
    Lista Ściany do usunięcia i krawędzie szwów jest pusta, jeśli obiekt bazowy został wybrany w Widoku drzewa, lub zawiera podobiekty użyte do wybrania obiektu bazowego w Widoku 3D.
    • Dostosuj parametry w Panelu zadań.
    • Opcjonalnie naciśnij przycisk Select, aby edytować listę Ściany do usunięcia i krawędzie szwów.
      (Przycisk Preview wskazuje tryb edycji listy)
      • Opcjonalnie wybierz ściany/krawędzie w Widoku 3D, aby dodać je do listy lub odpowiednio z niej usunąć.
      • Opcjonalnie naciśnij przycisk Wyczyść wszystko, aby usunąć wszystkie elementy z listy Ściany do usunięcia i krawędzie szwów.
      • Opcjonalnie zaznacz elementy listy i naciśnij przycisk Wyczyść wybrane, aby usunąć tylko zaznaczone elementy z listy Ściany do usunięcia i krawędzie szwów.
      • Naciśnij przycisk Podgląd, aby potwierdzić i zakończyć edycję.
  4. Naciśnij przycisk OK, aby zamknąć Panel zadań i zakończyć polecenie.
  5. Opcjonalnie dostosuj parametry profilu w Widoku właściwości.

Panel zadań

Double-click an existing SolidToSheet object in the Tree View to re-open the task panel and edit the parameters.

  • Thickness: Sets the Thickness property.
  • Bend Radius: Sets the Radius property.
  • Invert: Toggles the Invert property; .
  • Select: Activates the edit mode that allows to add the list of faces to remove and/or seam edges within the base Object property. If pressed:
    • Preview: Indicates the edit mode. Press to finish the edit mode (hidden until Select is pressed).
    • Clear All: Press to clear the list (hidden until Select is pressed). Changes to Clear Selected if at least one list item is checked; removes only the selected items.

Properties

See also: Property View.

A SheetMetal SolidToSheet object is derived from a Part Feature object or, if it is inside a PartDesign Body, from a PartDesign Feature object, and inherits all its properties. It also has the following additional properties:

Data

Parameters

  • DANEInvert (Bool): "Invert bend direction". Controls whether the wall thickness builds up inside or outside the base object.
  • DANERadius (Length): Inner radius of automatically added bends.
  • DANEThickness (Length): Wall thickness of the sheet metal shape.
  • DANEbase Object (LinkSub): "Base object". Link to the solid/shell and its selected sub-elements.
