Opis

Narzędzie Curves TruncateExtend tnie kształt i skraca go lub wydłuża o podaną odległość.

Użycie

  1. Wybierz kształt i płaszczyznę lub tylko płaszczyznę w Zawartości środowiska Projekt Części.
  2. Istnieje kilka sposobów na wywołanie tego polecenia:
    • Wciśnij przycisk Skróć-wydłuż.
    • Wybierz opcję Surfaces → Truncate Extend z menu.
  3. Utworzony zostanie obiekt Truncated_Extend.

Properties

Data

Input Objects

  • DANECutter (Link): The planar object that cuts the Source object
  • DANESource (Link): The object that will be truncated or extended

Settings

  • DANEDistance (Distance): The distance to truncate (if negative) or extend (if positive)
  • DANERefine (Bool): Refine shape (clean up redundant edges)
  • DANEReverse (Bool): Reverse cutter plane normal
