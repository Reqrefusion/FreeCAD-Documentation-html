Description

The Robot Simulate command simulates robot movement along a selected trajectory.

Usage

  1. Select a robot and a trajectory in the Tree View.
  2. There are several ways to invoke the command:
    • Press the Simulate Trajectory button.
    • Select the Robot → Simulate Trajectory option from the menu.
  3. The task panel opens.
  4. In the Trajectory section the following buttons are available (from left to right):
    • Move to the first waypoint.
    • Move to previous waypoint.
    • Stop movement.
    • Start movement.
    • Move to next waypoint.
    • Move to last waypoint.
  5. Press the Close button when done.
