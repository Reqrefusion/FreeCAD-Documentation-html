Opis

The FCGear GearConnector command combines two gear items …

Użycie

  1. Przełącz się do środowiska pracy Gear.
  2. Wybierz dwa elementy o pasujących zębach (patrz Uwagi):
    • Pierwszy musi być kołem zębatym czołowym lub wewnętrznym.
    • Drugi może być kołem zębatym czołowym lub listwą zębatą
  3. Polecenie można wywołać na kilka sposobów:
    • Naciśnij przycisk Połączenie zębatek na pasku narzędzi.
    • Wybierz opcję z menu Gear → Połączenie zębatek.
  4. Zmodyfikuj parametr koła zębatego na wymagane jeśli jest to niezbędne (patrz Właściwości).

Właściwości

See also: Property View.

An FCGear GearConnector object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

gear

  • DANEangle1 (Angle): Default is 0°. Angle at which second gear is placed.
  • DANEangle2 (Angle): Default is 0°. Angle at which second gear is placed.
  • DANEmaster_gear (Link): Master gear.
  • DANEmaster_gear_stationary (Bool): Default is True. Master gear position is fixed (does not orbit).
  • DANEslave_gear (Link): Slave gear.
  • DANEslave_gear_stationary (Bool): Default is True. Slave gear position is fixed (does not orbit).

version

  • DANEversion (String): The freecad.gears-version.

Uwagi

Kolejność wyboru określa, czy drugi element może obracać się wokół pierwszego elementu, gdy zmieniana jest właściwość DANEangle1.
