Opis

Narzędzie Dodaj wierzchołek dodaje nowy wierzchołek do obiektu Przewód giętki. Zachowuje prawidłowe przypisanie istniejących powiązań wierzchołków przewodu. Może być używane zamiennie z poleceniem "Dodaj punkt" z Edycja, patrz opis Przewodu giętkiego.

Użycie

  1. Wybierz jedną krawędź istniejącego obiektu Przewód giętki w Widoku 3D.
  2. Utwórz nowy wierzchołek jedną z poniższych metod:
    • Naciśnij przycisk Dodaj wierzchołek.
    • Wybierz z menu opcję Przewody kabli → Dodaj wierzchołek.
    • Kliknij prawym przyciskiem myszy w Widoku drzewa lub Widoku 3D i wybierz z menu kontekstowego opcję Przewody kabli → Dodaj wierzchołek.

The new vertex is added in the middle of selected edge. The vertex position can be edited later with Draft Edit command or attached to other objects with Attach Vertex.

Notes

  • From version 0.2.0 this command can also be executed using node context menu in Cables Edit command
  • See the WireFlex Description and WireFlex Usage for the details how to use WireFlex
