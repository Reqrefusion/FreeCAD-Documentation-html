Siatka: Skaluj

Opis

Tworzy nowy obiekt skalowanej siatki z niezależnym skalowaniem dla każdej osi.

Użycie

  1. Wybierz obiekt siatki, który ma zostać przeskalowany.
  2. Z menu wybierz opcję OpenSCAD → Skaluj cechę siatki.
  3. Wprowadź wartość skalowania.
  4. Tworzony jest nowy obiekt siatki, który zostaje skalowany, a oryginalny obiekt zostaje ukryty.

Ograniczenia

  • Nowy obiekt siatki nie jest parametryczny w stosunku do oryginalnego obiektu siatki, co oznacza, że wszelkie zmiany w oryginalnym obiekcie nie będą odzwierciedlone w nowym przeskalowanym obiekcie.

Uwagi

  • Ta funkcja nie modyfikuje istniejącej siatki, tylko zwraca nową siatkę.

Tworzenie skryptów

Ta funkcja jest dostępna z poziomu Pythona: 

import OpenSCADUtils
import Mesh
#this assumes an existing object in the document named "Mesh" that you wish to scale
original_mesh = App.ActiveDocument.Mesh
scaled_mesh = OpenSCADUtils.scalemesh(original_mesh.Mesh, FreeCAD.Base.Vector(1,0,0))
Mesh.show(scaled_mesh)
