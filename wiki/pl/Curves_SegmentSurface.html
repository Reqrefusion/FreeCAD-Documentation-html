Opis

Polecenie Podziel powierzchnię pozwala na segmentację powierzchni na izokrzywych. Narzędzie to jest częścią zewnętrznego środowiska pracy o nazwie Krzywe.

Użycie

  1. Przełącz się do środowiska pracy Krzywe (instalacja za pomocą Menadżera dodatków jest konieczna, jeśli nie zainstalowano go wcześniej).
  2. Aby wywołać polecenie, wykonaj jedną z następujących czynności:
    • Naciśnij przycisk na pasku narzędzi Curves.
    • Użyj pozycji Surfaces → Segment surface w menu.

Właściwości

See also: Property editor.

A Segment_Surface object is derived from an App FeaturePython object and inherits all its properties. It also has the following additional properties:

Data

Base

  • DANEOption (Enumeration): Option list.
    • Custom (default): ...
    • Auto: ...
  • DANESource (LinkSub): Initial Face.

Option Auto

  • DANEDirection (Enumeration): Segmenting direction.
    • U (default): ...
    • V: ...
    • Both: ...

UDirection

  • DANEKnots U (FloatList): Splitting parameters in U direction.
  • DANEKnots UProvider (Link): Object generating normalized parameters in U direction.
  • DANENumber U (Integer): Split the U parameter range in the given number of segments.

VDirection

  • DANEKnots V (FloatList): Splitting parameters in V direction.
  • DANEKnots VProvider (Link): Object generating normalized parameters in V direction.
  • DANENumber V (Integer): Split the V parameter range in the given number of segments.
