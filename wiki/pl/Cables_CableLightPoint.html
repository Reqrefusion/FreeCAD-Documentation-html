Description

New concept of Cable Light Point Preset and SuppLines child object: introduced in version 0.3.0

The Cable Light Point creates a parametric light point object. The light point is a simple ring symbolizing the point where a cable exits from e.g. the wall to connect a light source (e.g. a lamp). It has default size upon creation, but some parameters, like Diameter, Height or Thickness can be changed later. The Cable Light Point belongs to the Arch Component class and inherits all of its properties. The Cable Light Point has SuppLines child object with two crosses. Each SuppLines cross can be used as attachment supports for cable passing through the ring. SuppLines object is not a part of Cable Light Point shape, it is a separate child object also visible in the Tree View.

Cable Light Point object can be colored by attaching desired Material to the Material property.

Usage

  1. Select the vertex, point on surface or object in the 3D View or select nothing.
  2. Create Cable Light Point by one of these methods:
    • Press the Cable Light Point button.
    • Select the Cables → Cable Light Point option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cables → Cable Light Point option from the context menu.

Once created the Cable Light Point is placed at the position of selected vertex, point or object in the 3D space (or at the point (0,0,0) if nothing was selected).

Notes

  • The SuppLines objects have reversed relation with their Cable Light Point parent to avoid cyclic dependencies. They are shown as a children in the Tree View only for visual organization of objects. Undesirable consequence: when the Cable Light Point is copied, all its child SuppLines objects must always be manually copied with it.
  • SuppLines child object is created automatically, its shape and placement is controlled by Cable Light Point.

Properties

A Cable Light Point object shares the common properties and behaviors of all Arch Components.

Data

Cable Light Point

  • DANEClaim Children (Bool): If set to true it will claim the linked SuppLines as children in the Tree View (introduced in version 0.3.3).
  • DANEDiameter (Length): Specifies the outer diameter of the ring. The default diameter is 20 mm.
  • DANEHeight (Length): Specifies the height of the ring. The default height is 5 mm.
  • DANEPreset (Enumeration): Specifies the predefined set of parameters for Cable Light Point. The default preset is Customized. This property is currently not used.
  • DANEThickness (Length): Specifies the wall thickness of the ring. The default thickness is 2 mm.
