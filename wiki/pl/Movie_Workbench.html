Wprowadzenie

Movie is an external workbench to animate cameras and objects, and to record and play videos.

Paski narzędzi

Cameras and objects tools

  • MovieCamera
  • Enable a MovieCamera
  • Set position A
  • Set position B
  • MovieObjects
  • Enable a MovieObjects
  • Set an axis
  • Exclude a MovieObjects

Animation tools

  • Return to beginning
  • Take a step back
  • Play backward the animation
  • Pause the animation
  • Play the animation
  • Move one step forward
  • Move to the end

Record and play tools

  • Clapperboard
  • Enables a Clapperboard
  • "R1" - Record 3DViews
  • "R2" - Record Renders
  • Stop Recording
  • Create video
  • Play video

Instalacja

The Movie Workbench is available through the Addon Manager. Any Movie 3rd party dependencies should be managed automatically through the Addon Manager.
