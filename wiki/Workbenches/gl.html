FreeCAD, como moitas aplicacións de deseño modernas como Revit ou CATIA, baséase no concepto de Workbench. Un banco de traballo pódese considerar como un conxunto de ferramentas especialmente agrupadas para unha determinada tarefa. Nun taller de mobles tradicional, terías unha mesa de traballo para a persoa que traballa con madeira, outra para a que traballa con pezas de metal e quizais unha terceira para a persoa que monta todas as pezas.

En FreeCAD, aplícase o mesmo concepto. As ferramentas agrúpanse en bancos de traballo segundo as tarefas coas que están relacionadas.

Ao cambiar dun banco de traballo a outro, as ferramentas dispoñibles na interface cambian. As barras de ferramentas, as barras de comandos e posiblemente outras partes da interface cambian ao novo banco de traballo, pero o contido da escena non cambia. Poderías, por exemplo, comezar a debuxar formas 2D co banco de traballo Draft e despois traballar máis nelas co banco de traballo Part.

Ten en conta que ás veces un Workbench denomínase "módulo". Non obstante, os Workbenches e os Módulos son entidades diferentes. Un Módulo é calquera extensión de FreeCAD, mentres que un Workbench é un tipo especial de Módulo cunha configuración GUI (barras de ferramentas e menús).

Bancos de traballo preinstalados

Actual

Os bancos de traballo que se listan a continuación inclúense con cada instalación de FreeCAD.

Introducido en 1.0: Algúns bancos de traballo que se usan con pouca frecuencia están desactivados por defecto. Están marcados con D. Pódese activar un banco de traballo desactivado en Preferencias. Os bancos de traballo desactivados aínda se poden activar a través de do menú de TabBar Workbench selector.

  • Std Base. Este non é realmente un banco de traballo, senón unha categoría de comandos e ferramentas "estándar" que se poden usar en todos os bancos de traballo.
  • O Banco de traballo de montaxe para construír e resolver ensamblaxes mecánicas. Introducido en 1.0
  • O Banco de Traballo BIM (Building Information Modelling) para traballar con elementos arquitectónicos e crear modelos BIM. Combina o Banco de traballo en arco e o Banco de Traballo BIM anteriormente externo dispoñible na versión 0.21 e anteriores.
  • O Banco de Traballo CAM (Computer Aided Manufacturing) é usado para producir instruccións de G-Code. Este Banco de Traballo era chamado "Path Workbench" na versión 0.21 e anteriores.
  • O Banco de traballo Borrador (Draft) contén ferramentas en 2D e operacións básicas de CAD 2D e 3D.
  • O Banco de Traballo FEM (Finite Element Analysis) proporciona un fluxo de traballo de análise de elementos finitos (FEA).
  • O Banco de Traballo Inspección (Inspection) está feito para proporcionarche ferramentas específicas para o exame de formas. Aínda está nas primeiras etapas de desenvolvemento. D
  • O Banco de Traballo Material xestiona o sistema de materiais FreeCAD. Introducido en 1.0
  • O Banco de Traballo Malla para traballar con mallas trianguladas.
  • O Banco de Traballo OpenSCAD para a interoperabilidade con OpenSCAD e para a reparación do historial do modelo de xeometría sólida construtiva (CSG). D
  • O Banco de Traballo Parte (Part) para traballar con primitivas xeométricas e operacións booleanas.
  • The Part Design Workbench for building Part shapes from sketches.
  • The Points Workbench for working with point clouds.
  • The Reverse Engineering Workbench is intended to provide specific tools to convert shapes/solids/meshes into parametric FreeCAD-compatible features.
  • The Robot Workbench for studying robot movements. Currently unmaintained. D
  • The Sketcher Workbench for working with geometry-constrained sketches.
  • The Spreadsheet Workbench for creating and manipulating spreadsheet data.
  • The Surface Workbench provides tools to create and modify surfaces. It is similar to the Part Builder Face from edges option.
  • The TechDraw Workbench for producing technical drawings from 3D models. It is the successor of the Drawing Workbench.
  • The Test Framework Workbench is for debugging FreeCAD. D

Obsolete

The following workbenches are no longer included after version 0.21:

  • The Start Workbench allows you to quickly jump to one of the most common workbenches. This workbench has been replaced by the Start page that can be accessed via the Help menu.
  • The Web Workbench provides you with a browser window instead of the 3D view within FreeCAD.

The following workbenches are no longer included after version 0.20:

  • The Drawing Workbench was used for producing technical drawings. The TechDraw Workbench is its more advanced replacement.
  • The Image Workbench was used for working with bitmap images. Its functionality has been integrated in Std Base. See Std Import and Std ViewLoadImage.
  • The Raytracing Workbench was used for ray-tracing (rendering). The external Render Workbench should be used instead.

External workbenches

FreeCAD workbenches are easy to program in Python, there are therefore many people developing additional workbenches outside of the FreeCAD main development area.

The Addons page lists all that are known to this community. They are easily installable from within FreeCAD, using the Addon Manager, found under menu Tools → Addon Manager.
