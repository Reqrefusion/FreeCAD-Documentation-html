==v0.13: re: 2nd Method The button described as "None" will probably not show up as "None". The button label is set to the default working plane from Draft Preferences. The button is located beside the "set shape colour, set line colour, ..." buttons.

The button does not have a tool tip ("Set Working Plane" perhaps?)
