Description

The Sketcher ViewSection tool toggles a temporary section plane that hides any objects and parts of objects in front of the sketch plane while the sketch is in edit mode. This is a per-sketch setting stored in the ViewSection View property of sketches.

Partially hidden sketch and fully visible sketch

Usage

See also: Sketcher Preferences.

  1. This tool is only available if a sketch is in edit mode.
  2. There are several ways to invoke the tool:
    • Press the Toggle Section View button.
    • Select the Sketch → Toggle Section View option from the menu.
    • Right-click in the 3D View and select the Toggle Section View option from the context menu.
    • Use the keyboard shortcut: Q then S.
