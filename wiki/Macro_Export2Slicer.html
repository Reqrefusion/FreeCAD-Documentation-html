Exports selected object as mesh and opens it in slicer app. If no objects selected exports all of them.

One may need to change slicer path. To do so open Tools > Edit parameters > BaseApp/Preferences/Macros/Export2Slicer'

One can also change other parameters:

- export format (set 'stl' or 'amf')

- AngularDeflection (default = 0.07)

- Output folder

- Slicer command to run
