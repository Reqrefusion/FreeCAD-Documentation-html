Vue d'ensemble

WebTools est un atelier externe contenant une série d'outils pour communiquer via des services Web depuis FreeCAD.

Installation

Cet atelier peut être installé à partir du Gestionnaire des extensions. Pour une installation manuelle, voir Installer des ateliers supplémentaires.

Outils

  • BIM server: Ouvre une fenêtre BimServer
  • Git: Gérer des fichiers avec GIT
  • Sketchfab exporter: Exporte et télécharge des objets vers votre compte SketchFab.

Liens

  • Code source hébergé sur GitHub: https://github.com/yorikvanhavre/WebTools
