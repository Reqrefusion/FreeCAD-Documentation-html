Macro Multi Export

Description
Exportez simultanément les corps sélectionnés vers plusieurs formats de fichiers (STL, STEP, 3MF, OBJ, IGES, BREP, PLY, AMF) à l'aide d'une boîte de dialogue conviviale permettant de sélectionner le format et de configurer la sortie.

Description

Cette macro offre un moyen pratique d'exporter simultanément les objets FreeCAD sélectionnés vers plusieurs formats de fichiers. Au lieu d'exporter vers chaque format individuellement, vous pouvez sélectionner tous les formats souhaités dans une seule boîte de dialogue et les exporter simultanément.

Formats d'exportation pris en charge :

  • STL - Stéréolithographie (couramment utilisé pour l'impression 3D)
  • STEP - Norme pour l'échange de données sur les produits (échange CAO)
  • 3MF - Format de fabrication 3D (format d'impression 3D moderne)
  • OBJ - Wavefront OBJ (graphiques 3D et moteurs de jeux)
  • IGES - Initial Graphics Exchange Specification (format CAO hérité)
  • BREP - Format natif OpenCASCADE (conserve la géométrie exacte)
  • PLY - Format de fichier polygonal (numérisation et impression 3D)
  • AMF - Format de fabrication additive (impression 3D basée sur XML)

Fonctions :

  • Exportation vers plusieurs formats en une seule opération.
  • Boîte de dialogue conviviale pour la sélection du format.
  • Répertoire de sortie et nom de fichier de base configurables.
  • Paramètres de tolérance du maillage pour les exportations STL/OBJ/PLY/3MF/AMF.
  • Aperçu des fichiers à créer.
  • Boutons de sélection rapide : « Select All », « Select None », « Reset Defaults ».
  • Formats par défaut : STL, STEP et 3MF sont présélectionnés.

Utilisation

  1. Select one or more objects in the 3D view that you want to export.
  2. Run the macro from Macro → Macros → MultiExport → Execute.
  3. In the dialog:
    • Review the selected objects in the "Objects to Export" section.
    • Check the formats you want to export to (STL, STEP, 3MF are selected by default).
    • Choose an output directory using the "Browse..." button.
    • Enter a base filename (the format extension will be appended automatically).
    • Optionally adjust mesh tolerance settings for mesh-based formats.
  4. Click "Export" to create the files.

Requirements

  • FreeCAD 0.21 or later.
  • Objects with solid shapes (Part or PartDesign bodies).

Installation

  1. Download the macro file: MultiExport.FCMacro.
  2. Copy the file to your FreeCAD macro directory:
    • macOS: ~/Library/Application Support/FreeCAD/Macro/
    • Linux: ~/.local/share/FreeCAD/Macro/
    • Windows: %APPDATA%/FreeCAD/Macro/
  3. Optionally, download the toolbar icon and place it in the same directory.
