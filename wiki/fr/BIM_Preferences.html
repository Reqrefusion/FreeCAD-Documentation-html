Introduction

Les préférences pour l' atelier BIM peuvent être trouvées dans l'éditeur de préférences. Dans le menu, sélectionnez Édition → Préférences... puis BIM. Ce groupe n'est disponible que si l'atelier BIM a été chargé dans la session FreeCAD en cours.

Il y a trois pages : Paramètres généraux, Valeurs par défaut et IFC natifs.

Certaines préférences avancées ne peuvent être modifiées que dans l'éditeur de paramètres. Voir Réglage fin.

Cette page a été mise à jour pour la version 1.0.
