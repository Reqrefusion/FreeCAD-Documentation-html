Description

L'outil Curves Interpoler interpole les points avec une courbe B-spline.

Utilisation

  1. Sélectionnez plusieurs sommets ou un objet contenant au moins 2 sommets.
  2. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Interpolate.
    • Sélectionnez l'option Curves → Interpolate du menu.
  3. Un objet Interpolation_Curve est créé.

Propriétés

Voir aussi : Éditeur de propriétés

Un objet Interpolation_Curve est dérivé d'un Part Feature et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

General

  • DonnéesCustomTangents (Bool) : tangentes spécifiées par l'utilisateur.
  • Données (Hidden)DetectAligned (Bool) : interpole 3 points alignés sur une ligne.
  • DonnéesPeriodic (Bool) : définit la courbe fermée.
  • Données (Hidden)PointList (LinkSubList) : liste de points à interpoler.
  • DonnéesPolygonal (Bool) : interpole avec une courbe polygonale de degré 1.
  • DonnéesSource (Link) : objet source qui fournit les points à interpoler.
  • DonnéesStartOffset (Integer) : décale l'indice de départ de la liste de points.
  • DonnéesTangentFlags (BoolList) : indique l'activation des tangentes.
  • DonnéesTangents (VectorList) : applique une tangence aux points interpolés.
  • DonnéesTolerance (Float) : tolérance de l'interpolation.

Parameters

  • Données (Hidden)Parameters (FloatList) : paramètres des points interpolés.
  • DonnéesParametrization (Enumeration) : type de paramétrisation.
    • ChordLength (default) : ...
    • Centripetal : ...
    • Uniform : ...
    • Custom : ...
  • DonnéesWireOutput (Bool) : produit une polyligne ou une arête unique.

Spiral

  • DonnéesFaceSupport (LinkSub) : support de la face de la spirale.
  • DonnéesUTurns (Integer) : nombre de tours entre 2 points, dans la direction U.
  • DonnéesVTurns (Integer) : nombre de tours entre 2 points, dans le sens V.
