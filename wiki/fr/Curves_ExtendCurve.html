Description

L'outil Curves Prolonger une courbe prolonge les arêtes sélectionnées aux deux extrémités d'une longueur donnée.

Utilisation

  1. Sélectionnez une ou plusieurs arêtes dans la vue 3D.
  2. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Prolonger une courbe.
    • Sélectionnez l'option Curves → Prolonger une courbe du menu.
  3. Un objet ExtendedCurve est créé pour chaque arête sélectionnée.
  4. Vous pouvez modifier les valeurs dans l'éditeur de propriétés pour ajuster les longueurs et les types d'extension ainsi que la forme résultante (voir Propriétés ci-dessous).

Propriétés

Voir aussi : Éditeur de propriétés

Un objet ExtendCurve est dérivé d'un Part Feature et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Base

  • DonnéesEdge (LinkSub) : arrête initiale à prolonger.
  • DonnéesOutput (Enumeration) : forme résultante.
    • SingleEdge (valeur par défaut) : une seule courbe continue.
    • Wire : courbes séparées.

Beginning

  • DonnéesLength Start (Float) : longueur du prolongement au point de départ.
  • DonnéesType Start (Enumeration) : type de prolongement au point de départ.
    • Straight (par défaut) : le prolongement est une ligne droite.
    • G2 curve : le prolongement est une autre courbe qui adopte la courbure locale de l'arête initiale au sommet commun (continuité G2).

End

  • DonnéesLength End (Float) : longueur du prolongement au dernier point.
  • DonnéesType End (Enumeration) : type de prolongement au dernier point.
    • Straight (par défaut) : le prolongement est une ligne droite.
    • G2 curve : le prolongement est une autre courbe qui adopte la courbure locale de l'arête initiale au sommet commun (continuité G2).
