Description

La commande Différence crée un nouvel objet maillé non paramétrique, un Mesh Feature, qui est la différence de deux objets maillé : un objet maillé est coupé de l'autre.

OpenSCAD doit être installé pour utiliser cette commande et le chemin d'accès à son exécutable doit être défini dans les préférences d'OpenSCAD.

A gauche deux objets maillés, à droite l'objet maillé qui est la différence de ces objets si le cube est sélectionné comme objet principal


Utilisation

  1. Sélectionnez l'objet maillé principal.
  2. Ajoutez l'objet maillé que vous souhaitez couper de l'objet principal à la sélection. Les objets maillés doivent se chevaucher.
  3. Il existe plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Créer une différence
    • Sélectionnez l'option Maillages → Opération booléenne → Créer une différence du menu.
