Description

Définit une charge de flux thermique sur une surface. Les types de flux thermique suivants sont pris en charge :

  • charge de flux thermique de surface régulière q
  • charge de flux thermique convectif sur une surface à une température T avec un coefficient de transfert thermique h et avec une température ambiante (puits/ambiant) T0. Le flux thermique convectif q satisfera : q=h(T−T0).
  • introduit dans la version 1.0 : flux thermique par rayonnement sur une surface. Il satisfait : q=ϵσ(T4−T04) où ϵ est l'émissivité de la surface et σ est la constante de Stefan-Boltzmann.

Radiateur avec flux thermique convectif sur les surfaces exposées à l'air

Utilisation

  1. Il existe plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Charge de flux thermique.
    • Sélectionnez l'option Modèle → Conditions limites et charges thermiques → Charge de flux thermique du menu.
  2. Cliquez sur le bouton Ajouter. Dans la vue 3D, sélectionnez la ou les faces sur lesquelles la charge de flux thermique doit être appliquée. Si vous le souhaitez, cliquez sur le bouton Supprimer pour supprimer les faces sélectionnées de la liste de sélection.
  3. Choisissez le type de flux thermique et spécifiez ses paramètres :
    • Flux thermique de surface : flux thermique général, entrez le flux thermique de surface en Watts par unité de surface (W/m^2).
    • Convection (par défaut) : flux thermique convectif, entrez le coefficient de transfert thermique et la température ambiante souhaités.
    • Rayonnement : flux thermique par rayonnement, entrez l'émissivité de la surface et la température ambiante. introduit dans la version 1.0

Propriétés

  • DonnéesAmbient Temp : température ambiante (pour les modes « Convection » et « Rayonnement »).
  • DonnéesFilm Coef : coefficient de transfert thermique (pour le mode « Convection »).
  • DonnéesEmissivity : émissivité (pour le mode « Rayonnement »).
  • DonnéesDFlux : flux thermique distribué (pour le mode « Flux thermique de surface »).
  • DonnéesConstraint Type : type de charge de flux thermique : « Flux thermique de surface », « Convection » ou « Rayonnement ».
  • DonnéesCavity Radiation : active le rayonnement de cavité (surface à surface) (pour le mode « Rayonnement »). introduit dans la version 1.1
  • DonnéesCavity Name : nom de la définition de cavité à laquelle appartient la contrainte de flux thermique (pour le mode « Rayonnement »). introduit dans la version 1.1
  • DonnéesEnable Amplitude : permet d'utiliser l'amplitude de CalculiX pour la variation temporelle de la charge de flux thermique. À un moment donné, la magnitude de la charge est multipliée par la valeur de l'amplitude correspondante à ce moment. L'interpolation linéaire est utilisée entre les valeurs spécifiées. Des valeurs constantes sont utilisées en dehors de la plage spécifiée. introduit dans la version 1.1
  • DonnéesAmplitude Values : tableau d'amplitude au format suivant (par défaut, augmentation linéaire de 0 % à t=0 à 100 % à t=1) introduit dans la version 1.1 :
time_1, amplitude_value_1
...
La valeur par défaut est une augmentation linéaire de 0 % à t=0 à 100 % à t=1 :
0, 0
1, 1

Remarques

  • La charge de flux thermique utilise les jeux de paramètres de CalculiX suivants en fonction du mode sélectionné :
    • *FILM pour Convection
    • *RADIATE pour Rayonnement, introduit dans la version 1.0
    • *DFLUX pour Flux thermique de surface
