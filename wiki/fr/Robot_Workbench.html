La raison pour laquelle cet atelier se trouve toujours dans le code source principal est que cet atelier est programmé en C++. Si cet atelier pouvait être programmé en Python, alors il pourrait être transformé en atelier externe et il pourrait être déplacé vers un référentiel distinct.

Introduction

L' atelier Robot permet de simuler les robots industriels à 6 axes, comme par exemple Kuka.

Vous pouvez accomplir les tâches suivantes :

  • mettre en place un environnement de simulation avec un robot et des pièces de travail
  • créer et remplir des trajectoires
  • décomposer les fonctions d'une pièce CAO en une trajectoire
  • simuler le mouvement et l'accessibilité d'un robot
  • exporter la trajectoire vers un fichier programme de robot

Pour commencer, essayez le tutoriel Robot et voyez l'interface de programmation dans le fichier d'exemple RobotExample.py .

Outils

Barres d'outils de Robot

  • Création de robot : place un robot dans le modèle.
  • Trajectoire : crée une trajectoire vide.
  • Point de passage : insère une position d'outil robotique dans une trajectoire.
  • Position présélectionnée : insère une position présélectionnée dans une trajectoire.
  • Trajectoire à partir d'arêtes : génère une trajectoire à partir des arêtes sélectionnées.
  • Modifier la trajectoire : modifie certaines valeurs d'une trajectoire.
  • Réunir des trajectoires : regroupe et relie plusieurs trajectoires en une seule.
  • Position d'origine : définit la position d'origine d'un robot.
  • Retour origine: déplace un robot vers sa position d'origine.
  • Simulation : simule le mouvement d'un robot suivant une trajectoire sélectionnée.

Autres outils

  • Insérer un robot :
  • Outil : ajoute un outil à un robot.
  • Orientation par défaut : définit l'orientation par défaut pour la création ultérieure de points de trajectoire.
  • Valeurs par défaut : définit les valeurs par défaut pour la vitesse, l'accélération et la continuité pour la création ultérieure de points de trajectoire.
  • Exporter une trajectoire :
  • Exporter Kuka compact : exporte une trajectoire sous forme de sous-routine KRL compact.
  • Exporter Kuka complet : exporte une trajectoire sous forme de sous-routine KRL complète.
