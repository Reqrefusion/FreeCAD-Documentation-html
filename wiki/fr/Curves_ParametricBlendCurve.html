Description

The Curves ParametricBlendCurve tool creates a Blend curve between two edges.

Usage

  1. Select two curves.
  2. There are several ways to invoke the tool:
    • Press the Blend curve button.
    • Select the Curves → Blend curve option from the menu.
  3. A Blend_Curve object is created.
  4. Optionally adjust values in the Property editor:
    • Set the DonnéesParameter1 and DonnéesParameter2 properties to 0.0 mm to use the selected cuves' start vertices to place the blend curve's start and end vertices.
    • Set the DonnéesReverse1 and DonnéesReverse2 properties to true to switch the selected curves' start and end vertices.

Notes

To reverse the orientation of the blend curve's start and end tangents prefix - to the value of its DonnéesScale1 or DonnéesScale2 property.

Properties

See also: Property editor.

A Blend_Curve object is derived from an App FeaturePython object and inherits all its properties. It also has the following additional properties:

Data

Blend Curve

  • DonnéesAuto Scale (Bool): Compute scales to get minimal curvature along curve.
  • DonnéesCurvePts (VectorList): Poles of the Bezier curve.
  • DonnéesOutput (Enumeration): Output type.
    • Single (default): ...
    • Joined: ...
    • Wire: ...

{{TitleProperty|Edge1}

  • DonnéesContinuity1 (Enumeration): Output type.
  • DonnéesEdge1 (LinkSub): Edge 1.
  • DonnéesParameter1 (Distance): Location on first edge.
  • DonnéesReverse1 (Bool): Reverse Edge.
  • DonnéesScale1 (Bool): Scale of blend curve, negate the value to reverse the tangent orientation.

{{TitleProperty|Edge2}

  • DonnéesContinuity2 (Enumeration): Output type.
  • DonnéesEdge2 (LinkSub): Edge 2.
  • DonnéesParameter2 (Distance): Location on second edge.
  • DonnéesReverse2 (Bool): Reverse Edge.
  • DonnéesScale2 (Bool): Scale of blend curve, negate the value to reverse the tangent orientation.
