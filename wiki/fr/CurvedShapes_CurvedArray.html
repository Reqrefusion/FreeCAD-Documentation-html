Description

Crée un réseau et redimensionne les éléments dans les limites d'une ou plusieurs courbes génératrices. Dans cet exemple, la forme de base orange est redimensionnée dans les limites des courbes génératrices rouges et violettes. Il n'est pas nécessaire de relier les courbes entre elles. Les courbes génératrices doivent se trouver sur le plan XY- XZ- ou YZ- ou être parallèles à celui-ci.

Image du GitHub de chbergmann

Utilisation

  1. Sélectionnez les courbes d'entrée.
  2. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Curved Array.
    • Sélectionnez l'option Curved Shapes → Curved Array du menu.

Remarques

  • La première courbe que vous sélectionnez pour la création de CurvedArray sera l'élément balayé et redimensionné dans les limites des autres courbes sélectionnées.

Propriétés

Base

  • DonnéesBase : objet pour créer un réseau à partir de
  • DonnéesHullcurves : liste d'une ou plusieurs courbes limites
  • DonnéesAxis : axe de direction de la forme de base
  • DonnéesItems : nombre d'éléments du réseau
  • DonnéesOffsetStart : décalage de la première partie dans la direction de l'axe
  • DonnéesOffsetEnd : décalage de la dernière partie de la fin dans la direction opposée de l'axe
  • DonnéesTwist : applique une rotation autour de l'axe aux éléments du réseau.
  • DonnéesSurface : crée une surface sur les éléments du réseau
  • DonnéesSolid : crée un solide si la base est une forme fermée
