Description

L'outil Templater Modèle crée un petit modèle d'exemple et l'insère en tant que nouvelle page de dessin dans le document actif. Consultez le tutoriel TechDraw Création de modèles pour apprendre à créer un modèle presque identique à l'aide d'un script.

Image empruntée à la page du tutoriel

Utilisation

  1. Il existe plusieurs façons de lancer l'outil.
    • Appuyez sur le bouton Nouveau modèle Wiki.
    • Sélectionnez l'option Templater → Nouveau modèle Wiki du menu.
  2. Le panneau de tâches Template settings s'ouvre.
    • Vous pouvez cocher l'option Ink-blue pour colorer les textes modifiables.
  3. Cliquez sur Ok pour fermer le panneau de tâches et quitter l'outil.
  4. Un nouveau modèle sera enregistré dans le dossier .../Mod/Templater/Resources.
  5. Le modèle créé s'affichera dans une nouvelle vue TechDraw.
