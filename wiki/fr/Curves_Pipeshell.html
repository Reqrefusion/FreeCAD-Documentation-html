Description

L'outil Enveloppe de tube crée un balayage pour former une enveloppe de tube.

Utilisation

  1. Sélectionnez l'arête qui construit la trajectoire de balayage dans la vue 3D.
  2. Sélectionnez un ou plusieurs profils requis dans la vue en arborescence.
  3. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Pipeshell.
    • Sélectionnez l'option Surfaces → Pipeshell du menu.
  4. Un objet PipeShell est créé.
  5. Vous pouvez modifier les valeurs dans l'éditeur de propriétés pour choisir un algorithme de balayage différent ou une forme de sortie différente et ajustez les propriétés correspondantes (voir Propriétés ci-dessous).

Propriétés

Voir aussi : Éditeur de propriétés

Un objet PipeShell est dérivé d'un Part Feature et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Main

  • DonnéesMode (Enumeration) : mode du PipeShell utilisé pour sélectionner l'algorithme de balayage :
    • Frene (par défaut) : ...
    • DiscreteTrihedron : ...
    • FixedTrihetron : ...
    • Binormal : ...
    • ShapeSupport : ...
    • AuxiliarySpine : ...
  • DonnéesOutput (Enumeration) : détermine la forme de l'objet :
    • Sections (par défaut) : ...
    • Lofted Sections : ...
    • Surface : ...
  • DonnéesProfils (LinkList): profils qui sont balayés le long de la colonne vertébrale.
  • DonnéesSpine (LinkSubList) : trajectoire de balayage.

Mode

  • Données (Hidden)Auxiliary (Link) : trajectoire auxiliaire.
  • Données (Hidden)Contact (Enumeration) : type de contact avec la trajectoire auxiliaire.
    • NoContact (par défaut) : ...
    • Contact : ...
    • ContactOnBorder: ...
  • Données (Hidden)Corrected (Bool) : Frenet corrigé.
  • Données (Hidden)Direction (Vector) : direction des modes Binormal et FixedTrihedron.
  • Données (Hidden)EquiCurvi (Bool) : équivalence curviligne.
  • Données (Hidden)Location (Vector) : emplacement du mode FixedTrihedron.
  • Données (Hidden)Support (Link) : forme du mode ShapeSupport.

Settings

  • DonnéesMaxDegree (Integer) : degré maximum de la surface générée.
  • DonnéesMaxSegments (Integer) : nombre maximal de segments de la surface générée.
  • DonnéesSamples (Integer) : nombre d'échantillons pour la prévisualisation.
  • DonnéesSolid (Bool) : crée un objet solide.
  • DonnéesTol3d (Float) : tolérance 3D.
  • DonnéesTolAng (Float) : tolérance angulaire.
  • DonnéesTolBound (Float) : limite de tolérance.

Remarques

  • Pipeshell a besoin d'une polyligne (comme trajectoire de balayage) et d'au moins un Pipeshell Profile.
  • Les deux outils Pipeshell et Pipeshell Profile fonctionnent ensemble comme un outil de balayage "avancé".
