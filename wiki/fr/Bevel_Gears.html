Atelier Gear

Introduction

Le tutoriel Engrenages coniques décrit comment rassembler les données nécessaires pour créer une paire d'engrenages coniques à l'aide de l'outil Engrenage conique de l'atelier Gear dans un contexte de PartDesign.

Deux engrenages coniques en prise

Un engrenage conique peut être construit à partir du cercle primitif ou à partir du sommet, selon sa propriété reset_origin. Dans les étapes suivantes de ce tutoriel, nous devons mettre à l'échelle l'engrenage conique sans déplacer le sommet. Nous allons donc désactiver cette propriété.

Conditions

L'outil Engrenage conique crée un engrenage conique pour des axes perpendiculaires et un rapport de 1:1 par défaut. Pour d'autres angles d'axe et différents rapports, nous devons déterminer l'angle de cône de pas approprié pour les deux engrenages coniques appariés. De plus, nous devons ajuster les valeurs de base et calculer les diamètres primaires.

Valeurs de base

En supposant que nous nous sommes creusé la tête pour réfléchir à l'espace disponible, aux charges, au rapport et à l'angle entre les axes, nous avons trouvé les valeurs de base suivantes :

Module : m = 2 mm
Rapport : 5 : 3
Angle entre les axes : 60°

Nombre de dents

Afin de ne pas surcharger nos processeurs, nous allons définir le nombre de dents (z) du petit engrenage à seulement 9 et donc le nombre de dents du grand engrenage à 15 (15 : 9 équivaut à 5 : 3).

z1 = 15 (pour l'engrenage conique 1)
z2 = 9 (pour l'engrenage conique 2)

Diamètre primitif

Nous pouvons maintenant calculer les diamètres primaires (d) :

d1 = z1 * m = 15 * 2 mm = 30 mm
d2 = z2 * m = 9 * 2 mm = 18 mm

Angles des cônes

C'est maintenant au tour de FreeCAD de trouver les angles du cône de pas (δ) :

  1. Créez un nouveau document.
  2. Créez un corps.
  3. Créer une esquisse.

Nous utilisons l'esquisse pour dessiner deux triangles connectés dont les axes de symétrie sont perpendiculaires aux côtés représentant les diamètres primaires. Outre plusieurs contraintes cachées de coïncidence et de symétrie, nous utilisons deux contraintes de longueur pour les diamètres primaires et une contrainte d'angle pour l'angle de l'axe afin de contraindre entièrement l'esquisse. Nous pouvons ensuite utiliser deux contraintes de référence pour obtenir les deux angles du cône primitif (et quitter l'esquisse) :

En rouge : angle d'axe contraint et deux diamètres primitifs.
En bleu : angles des cônes primitifs résultants

Nous avons maintenant les angles des cônes primitifs :

δ1 = 38,21°
δ2 = 21,79°

Engrenages bruts

Nous sommes prêts à créer notre paire d'engrenages coniques.

Engrenage conique 1

  1. Create a new PartDesign Body (Body001).
  2. Invoke the BevelGear command.
  3. A bevel gear (BevelGear) is created immediately with its pitch diameter side touching the origin, and awaiting adjustment.
  4. In the Property View adjust these properties of the bevel gear:
    1. Set the Donnéesmodule property to 2 mm.
    2. Set the Donnéesnum_teeth property to 15.
    3. Set the Donnéespitch_angle property to 38,21° (two decimal places should be more than enough).
    4. Uncheck the Donnéesreset_origin property to place the bevel gear's virtual tip on the origin (this will turn the bevel gear upside down).
  5. Use Transform to swivel the bevel gear around the origin so that its face on the pitch circle side is parallel to the horizontal line of the sketch representing the same pitch circle of the larger gear (if we filled in the values properly the line should be coplanar with the face).

Bevel gear 1 properly positioned

Bevel Gear 2

  1. Create another new PartDesign Body (Body002).
  2. Invoke the BevelGear command.
  3. A bevel gear (BevelGear002) is created.
  4. In the Property View adjust these properties of the bevel gear:
    1. Set the Donnéesmodule property to 2 mm.
    2. Set the Donnéesnum_teeth property to 9.
    3. Set the Donnéespitch_angle property to 21,79°.
    4. Uncheck the Donnéesreset_origin property to place the bevel gear's virtual tip on the origin (this will turn the bevel gear upside down).
  5. Use Transform to swivel the bevel gear around the origin so that its face on the pitch circle side is parallel to the oblique line of the sketch representing the same pitch circle of the smaller gear (if we filled in the values properly the line should again be coplanar with the face).

Bevel gear 2 properly positioned

At this point both bevel gears should mesh with each other where their pitch cones meet in the initial sketch, but their shapes don't look like real bevel gears.

Meshing raw bevel gears

Intersection Objects

We have to define the desired cross-sections from their axes outwards, and then revolve them to create objects that will be intersected with the results of the steps above to result in a proper pair of bevel gears.

Revolution Object 1

  1. Activate the first body (Body).
  2. Create another sketch, and in the sketch:
    1. Project the lines representing the pitch diameters of our first sketch into our new sketch.
    2. Draw a construction line from their intersection point to the origin (his is where the pitch cones meet).
    3. Draw a closed wire of six straight lines:
      • One line collinear with the vertical sketch axis,
      • two lines parallel to the horizontal projection line to define position and thickness of the hub area,
      • two lines perpendicular to the construction line, one running through the intersection point, that define the width of the teeth,
      • one line closing the wire that is only required to run outside the raw bevel gear since it already has a correctly modeled teeth hight.
  3. Leave the sketch.
  4. Create another new PartDesign Body (Body003).
  5. Use a Sub-Shape Binder to copy the sketch into the body.
  6. Create a Revolve object (Revolution) from the sub-shape binder.

Left: Sketch of cross-section. Right: Resulting Revolution object 1

Revolution Object 2

  1. Activate the first body (Body) again.
  2. Create another sketch, and in the sketch:
    1. Project the lines representing the pitch diameters, and also the line representing the axis of our smaller bevel gear from our first sketch into our new sketch.
    2. Draw a construction line from their intersection point of the pitch diameters to the origin.
    3. Draw a closed wire of six straight lines:
      • One line collinear with the projected axis,
      • two lines parallel to the oblique projected pitch diameter to define position and thickness of the hub area,
      • two lines perpendicular to the construction line, one running through the intersection point, that define the width of the teeth,
      • one line closing the wire that is only required to run outside the raw bevel gear since it already has a correctly modeled teeth hight.
  3. Leave the sketch.
  4. Create another new PartDesign Body (Body004)..
  5. Use a Sub-Shape Binder to copy the sketch into the body.
  6. Create a Revolve object (Revolution001) from the sub-shape binder.

Left: Sketch of cross-section. Right: Resulting Revolution object 2

Finished Bevel Gears

Now we will intersect the raw bevel gears with the revolution objects to find their common shapes. Having hidden all objects but our raw bevel gears and revolution objects, we will notice that the gears do not protrude enough from the revolution object. Since we built the bevel gears from the tip we can (mis-) use the module value to "scale" the gear and adjust its height before intersecting.

Bevel Gear 1

  1. Activate bevel gear 1 (Body001).
  2. In the Property View adjust these properties of the bevel gear:
    1. Set the Donnéesmodule property to 2.5 mm.
    2. Set the Donnéesheight property to 15 mm.
  3. Press the Recompute button to see the changes.
  4. invoke PartDesign Boolean Operation.
  5. The Boolean Parameters dialog opens in the Task Panel:
    • Press the Add Body button
    • Select the revolution object 1 (Body003).
    • Press the Add Body button again to finish the selection.
    • Select Common from the options below the list.
  6. Press the OK button to shut the dialog and finish the operation.
  7. The shape of bevel gear 1 is finished.

Left: Enlarged raw bevel gear 1 and revolution object 1. Right: Resulting finished bevel gear 1

Bevel Gear 2

  1. Activate bevel gear 2 (Body002).
  2. In the Property View adjust these properties of the bevel gear:
    1. Set the Donnéesmodule property to 2.5 mm.
    2. Set the Donnéesheight property to 15 mm.
  3. Press the Recompute button to see the changes.
  4. Invoke PartDesign Boolean Operation.
  5. The Boolean Parameters dialog opens in the Task Panel:
    • Press the Add Body button
    • Select the revolution object 2 (Body004).
    • Press the Add Body button again to finish the selection.
    • Select Common from the options below the list.
  6. Press the OK button to shut the dialog and finish the operation.
  7. The shape of bevel gear 2 is finished.

Left: Enlarged raw bevel gear 2 and revolution object 2. Right: Resulting finished bevel gear 2

What is Left to do?

Now we have two decent bevel gears but they cannot be mounted to axles or shafts, They have to be extended with some shape elements, either cylindric ones for bearings to be able to rotate around an axle, or prismatic ones to be able to transmit torque into a shaft.

Such additions are quite simple to model so that they are not covered by this tutorial.
