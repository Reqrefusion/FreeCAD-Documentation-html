Description

L'outil BIM Dalle crée un objet Slab, un élément de structure horizontal, à partir d'une forme plane. Les dalles sont des objets Arch Structure spécialisés dont DonnéesIfc Type est prédéfini sur « Slab ».

Bien qu'il se comporte comme un Arch Structure standard, l'outil BIM initialise spécifiquement la dalle avec une hauteur (épaisseur) par défaut de 200 mm et une direction d'extrusion pointant vers le bas.

Utilisation

  1. Vous pouvez sélectionner un objet plan.
  2. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Dalle.
    • Sélectionnez l'option 3D/BIM → Dalle du menu.
    • Utilisez le raccourci clavier : S puis B.
  3. Si vous n'avez pas encore sélectionné d'objet plan : sélectionnez-en un (comme demandé dans la vue rapport).
  4. Un nouvel objet Dalle est créé et l'outil est fermé.
  5. Vous pouvez modifier certaines propriétés dans l'éditeur de propriétés.

Remarques

  • Wrapping behavior : l'outil « enveloppe » l'objet sélectionné. Si vous sélectionnez un sous-élément (comme une face dans la vue 3D), l'outil identifie l'objet parent et l'utilise comme DonnéesBase.
  • Thickness : pour modifier l'épaisseur de la dalle, modifiez la propriété DonnéesHeight.
  • Orientation : pour faire avoir la dalle vers le haut, modifiez la propriété DonnéesNormal en (0, 0, 1).

Propriétés

Un objet Slab partage les propriétés et les comportements communs à toutes les Arch Structures. DonnéesIfcType est prédéfini à Slab.

Script

The BIM Slab is created using the same Python function as Arch Structures: 

import FreeCAD
import Arch
import Draft

# Create a rectangle profile
rect = Draft.make_rectangle(length=2000, width=3000)
FreeCAD.ActiveDocument.recompute()

# Create a structure and define it as a slab
slab = Arch.makeStructure(rect)
slab.IfcType = "Slab"
slab.Height = 150  # mm
slab.Normal = (0, 0, -1)  # Slab is extruded downwards

FreeCAD.ActiveDocument.recompute()
