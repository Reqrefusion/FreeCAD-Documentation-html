Description

Déconnecter un câble d'une terminaison détache les sommets connectés de wireFlex à des terminaisons de câbles. Les connexions des sommets sont conservées dans les propriétés spéciales de WireFlex. Pour plus de détails, voir WireFlex Propriétés.

La déconnection ne modifie pas les positions existantes des terminaisons ou des câbles. La connexion électrique entre la terminaison et le câble n'existe plus. Les propriétés des terminaisons du groupe Net sont mises à jour en conséquence.

Lorsque la terminaison de câble est déplacée par la suite, les sommets déconnectés du câble restent en place. Lorsque le câble est modifié par Cables Edit, les sommets déconnectés peuvent être déplacés librement.


Utilisation

  1. Sélectionnez l'objet WireFlex dans la vue 3D ou dans la vue arborescente.
  2. Sélectionnez une terminaison de câble dans la vue 3D ou dans la vue arborescente en maintenant la touche CTRL enfoncée.
  3. Exécutez la commande Detach Wire From Terminal à l'aide de l'une des méthodes suivantes :
    • Appuyez sur le bouton Detach Wire From Terminal.
    • Sélectionnez l'option Cable Wires → Detach Wire From Terminal du menu.
    • Cliquez avec le bouton droit de la souris dans la vue arborescente ou la vue 3D et sélectionnez l'option Cable Wires → Detach Wire From Terminal dans le menu contextuel.
    • Utilisez le raccourci clavier : T puis R.

Remarques

  • Une fois détaché, le terminaison dispose d'un espace libre pour connecter un autre câble.
  • Voir les sections WireFlex Description et WireFlex Utilisation pour plus de détails sur l'utilisation de WireFlex.
