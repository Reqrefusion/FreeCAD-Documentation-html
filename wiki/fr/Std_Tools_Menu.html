Introduction

Le Std Menu Outils est l'un des 7 sous-menus du menu standard :

Fichier     Édition     Affichage     Outils     Macro     Fenêtre     Aide

Le menu Outils fournit des outils pour le débogage des modèles, la personnalisation du comportement de FreeCAD, ainsi que des outils auxiliaires.

Outils

Les outils suivants sont disponibles dans ce menu :

L'ordre des outils correspond au menu de la version 1.1.

  • Gestionnaire des extensions : ouvre le gestionnaire des extensions avec lequel les ateliers externes, les kits de préférences et les macros peuvent être installés.
  • Mesurer : mesure les objets. introduit dans la version 1.0
  • Clarify Selection: Lists all objects of different types located near the cursor to facilitate their selection. introduced in 1.1
  • Convertisseur d'unités : ouvre une calculatrice pour la conversion des unités.
  • Charger une image : charge une image. introduit dans la version 0.21
  • Capture d'écran : enregistre une capture d'écran de la vue 3D active.
  • Ajouter un document texte : crée un objet capable de contenir du texte.
  • Vue en rotation : fait tourner continuellement la caméra dans la vue 3D active.
  • Inspecteur du graphe de scène : affiche une vue d'ensemble des nœuds du graphe de scène de la vue 3D active.
  • Graphe des dépendances : affiche le graphe des dépendances des objets.
  • Exporter le graphique des dépendances introduit dans la version 0.21
  • Utilitaire de projet : aide à réparer les fichiers de projet endommagés.
  • Éditeur de paramètres : ouvre une fenêtre de dialogue donnant accès aux paramètres qui contrôlent le programme.
  • Personnaliser : ouvre une fenêtre de dialogue avec des options de personnalisation.
