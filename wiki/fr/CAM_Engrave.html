Description

La commande Gravure sert principalement à graver une Draft Forme à partir d'un texte sur une pièce. Elle peut également être utilisée avec d'autres géométries 2D planes, par exemple des formes importées au format SVG.

Utilisation

  1. Il existe plusieurs façons de lancer la commande :
    • Cliquez sur le bouton Graver.
    • Sélectionnez l'option CAM → Graver du menu.
  2. Dans la section Géométrie de base, sélectionnez la géométrie à graver.
    • Une Draft Forme à partir d'un texte peut être utilisée.
    • D'autres géométries 2D planes, telles que des formes importées au format SVG, peuvent également être utilisées.
    • Dans la mesure du possible, sélectionnez la forme plane entière plutôt que de sélectionner chaque arête dans la vue 3D.
  3. Dans la section Profondeurs, vérifiez et ajustez les paramètres de profondeur selon vos besoins.
  4. Confirmez l'opération.

Depth

  • DonnéesClearance Height : hauteur nécessaire pour éviter les pinces et les obstructions.
  • DonnéesFinal Depth : profondeur finale de l'outil - valeur la plus basse de Z.
  • DonnéesSafe Height : seuil supérieur duquel les mouvements rapides sont autorisés.
  • DonnéesStart Depth : profondeur initiale de l'outil - première profondeur de coupe en Z.
  • DonnéesStep Down : pas de descente incrémentale de l'outil.

Path

  • DonnéesActive : mettre à False pour empêcher l'opération de générer du code

  • DonnéesStart Vertex : index du sommet pour commencer le parcours à partir de
  • DonnéesTool Controller : contrôleur d'outil qui sera utilisé pour calculer le parcours
  • DonnéesUser Label : étiquette attribuée par l'utilisateur
