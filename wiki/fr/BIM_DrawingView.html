Description

L'outil BIM Dessin 2D crée un objet conteneur qui est un type spécialisé de Arch Partie de bâtiment destiné à contenir des objets de projection 2D.

Utilisé dans son flux de travail habituel avec un Arch Plan de coupe, cet outil automatise l'ensemble du processus d'extraction 2D. Il crée automatiquement les lignes de projection d'arrière-plan (via BIM Vue 2D) et les coupes de section solide (via BIM Coupe 2D), les aplatit sur le plan XY et les regroupe dans un seul dossier « Drawing ».

Ce conteneur de dessins groupés peut ensuite être facilement exporté sous forme de fichier DXF ou SVG 2D unique, ou inséré sous forme de vue unique dans une page de TechDraw.

Utilisation

Flux de travail automatique

  1. Sélectionnez un seul Arch Plan de coupe dans l'arborescence ou la vue 3D.
  2. Il existe plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Dessin 2D.
    • Sélectionnez l'option Annotation → Créer des vues 2D → Dessin 2D du menu.
    • Utilisez le raccourci clavier : V puis D.
  3. Un nouvel objet « Drawing » est créé dans l'arborescence. Sous celui-ci, vous trouverez deux nouveaux objets 2D : « Viewed lines » et « Cut lines » (représentant respectivement la projection et les remplissages des coupes).

Flux de travail manuel

  1. Ensure nothing is selected.
  2. Invoke the tool using any of the methods described above.
  3. A new, empty "Drawing" container is created in the Tree view.
  4. You can manually drag and drop existing 2D shapes, Draft objects, or Shape2DViews into this container to group them for export or drawing sheets.

Notes

  • The recommended workflow: this tool is the most efficient way to generate architectural drawings. By selecting a Section Plane and running this command, FreeCAD automatically performs the manual steps of creating both a Section View (for background lines) and a Section Cut (for solid fills).
  • Vertical sections and elevations: when a vertical Section Plane is projected, the building's horizontal axis is mapped to the global X-axis, and the building's vertical axis (height) is mapped to the global Y-axis. It effectively projects vertical elevations on the XY-plane automatically.
  • A virtual drafting table: this command forces DonnéesInPlace to false for all its children. This creates a "virtual drafting table" at the global origin (Z=0) where all project drawings are collected flat, regardless of their 3D orientation. This ensures that TechDraw can handle them as standard 2D vector groups and that DXF exports are perfectly flat.

Properties

The BIM DrawingView is essentially an Arch BuildingPart with its DonnéesIfcType set to Annotation and its DonnéesObjectType set to DRAWING.

To keep the property panel clean for 2D drafting, several 3D-specific properties inherited from the BuildingPart (such as DonnéesHeight, DonnéesArea, and DonnéesLevelOffset) are hidden from the user interface.
