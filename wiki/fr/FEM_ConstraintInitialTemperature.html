Description

Définit une température initiale pour une analyse thermomécanique.

introduit dans la version 1.0 : peut également être utilisé pour prescrire un champ de température dans une étape d'analyse statique (afin de simuler la dilatation thermique sans avoir recours à une analyse thermomécanique avec condition limite de température).

Deux cubes ayant des températures initiales différentes échangeant de la chaleur

Utilisation

  1. Il existe plusieurs façons de lancer la commande :
    • Appuyer sur le bouton Température initiale.
    • Sélectionner l'option Modèle → Conditions limites et charges thermiques → Température initiale du menu.
  2. Entrer une valeur de température initiale pour l'analyse.

Limitations

version 1.0 et précédentes : cet outil applique la température initiale à tous les nœuds du modèle FEA. Il n'est pas possible de sélectionner chaque région individuellement.

Propriétés

  • Donnéesinitial Temperature : magnitude de la température initiale.
  • DonnéesReferences : régions auxquelles la température initiale est appliquée. introduit dans la version 1.1
  • DonnéesEnable Final Temperature : active le champ de température prescrit dans une étape d'analyse statique. introduit dans la version 1.1
  • DonnéesFinal Temperature : valeur d'un champ de température prescrit dans une étape d'analyse statique. introduit dans la version 1.1
  • DonnéesEnable Amplitude : active l'utilisation de l'amplitude de CalculiX pour la variation temporelle du champ de température prescrit. À un moment donné, l'amplitude du champ est multipliée par la valeur d'amplitude correspondant à ce moment. Une interpolation linéaire est utilisée entre les valeurs spécifiées. Des valeurs constantes sont utilisées en dehors de la plage spécifiée. introduit dans la version 1.1
  • DonnéesAmplitude Values : tableau d'amplitudes au format suivant introduit dans la version 1.1 :
time_1, amplitude_value_1
…

Remarques

  • Cet outil utilise le jeu de paramètres *INITIAL CONDITIONS de CalculiX.
  • Option de température finale utilise le jeu de paramètres *TEMPERATURE de CalculiX. introduit dans la version 1.1
  • La température initiale doit être définie pour toutes les analyses thermomécaniques effectuées avec CalculiX, même celles en régime permanent.
