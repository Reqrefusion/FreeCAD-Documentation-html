Description

L'outil Courbe iso applique un réseau orienté UV sur une surface sélectionnée. Cet outil fait partie des ateliers externes appelé Curves.

Ci-dessus: montre les surfaces avant (gauche) et après (droite) une fois appliqué Courbe iso

Utilisation

  1. Passer à l'atelier Curves (à installer à partir du gestionnaire des extensions si ce n'est pas déjà fait)
  2. Sélectionnez une surface unique dans la vue 3D
  3. Pour appeler la commande, effectuez l'une des opérations suivantes:
    • Appuyez sur le bouton
    • Utilisez l'entrée Surfaces → IsoCurve dans le menu Curves
  4. Un treillis orienté UV sur la surface est créé.
  5. Modifiez NumberU ou/et NumberV pour obtenir plus de courbes.

Remarques

  • Les courbes peuvent être extraites en tant que sous-élément (par exemple: avec Curves Joindre courbe) pour d'autres utilisations ou simplement pour aider à visualiser la forme.
