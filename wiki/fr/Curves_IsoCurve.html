Description

L'outil Courbes iso applique un réseau orienté UV sur des surfaces sélectionnées.

Ci-dessus : la surface avant (à gauche) et après (à droite) l'application de l'outil.

Utilisation

  1. Sélectionnez une ou plusieurs faces dans la vue 3D.
  2. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Créer des courbes iso.
    • Sélectionnez l'option Surfaces → Créer des courbes iso du menu.
  3. Un objet IsoCurve est créé pour chaque face sélectionnée, appliquant un réseau orienté UV sur chaque face.
  4. Vous pouvez modifier les valeurs dans l'éditeur de propriétés pour changer le nombre de courbes iso dans les directions U et V et ajuster les propriétés correspondantes (voir Propriétés ci-dessous).

Remarques

  • Les courbes peuvent être extraites en tant que sous-élément (par exemple: avec Curves Relier des courbes) pour d'autres utilisations ou simplement pour aider à visualiser la forme.

Propriétés

Voir aussi : Éditeur de propriétés

Un objet IsoCurve est dérivé d'un Part Feature et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Iso Curve

  • DonnéesFace (LinkSub) : face de référence.
  • DonnéesMode (Enumeration) : nombre de courbes iso.
    • Multi (par défaut) : ...
    • Single : ...
  • DonnéesNumber U (Integer) : nombre de courbes iso dans la direction U.
  • DonnéesNumber V (Integer) : nombre de courbes iso dans la direction V.
  • DonnéesOrientation (Enumeration) : orientation des courbes.
    • U (par défaut) : ...
    • V : ...
  • DonnéesParameter (Float) : paramètre des courbes iso.
