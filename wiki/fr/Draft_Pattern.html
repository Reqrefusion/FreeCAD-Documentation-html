Description

Les objets Draft ayant une propriété DonnéesMake Face peuvent afficher un motif SVG au lieu d'une couleur de surface unie.

Une ellipse et un polygone avec un motif SVG

Utilisation

  1. Assurez-vous que les objets sont fermés et planaires, et qu'ils ne s'auto-intersectent pas :
  2. Pour fermer une Draft Polyligne, une Draft B-spline, une Draft Courbe de Bézier cubique ou une Draft Courbe de Bézier, définissez leur propriété DonnéesClosed à true.
  3. Pour fermer un Draft Cercle ou une Draft Ellipse, attribuez la même valeur aux propriétés DonnéesFirst Angle et DonnéesLast Angle.
  4. Sélectionnez les objets.
  5. Passez à l'onglet Visualisation de l'éditeur de propriétés.
  6. La propriété VueDisplay Mode doit être définie à Flat Lines.
  7. Sélectionnez un VuePattern.
  8. Changez éventuellement la VuePattern Size. Notez qu'une valeur plus élevée donne un motif plus dense.
  9. Le motif n'est pas affiché lorsque les objets sont sélectionnés. Désélectionnez-les pour vérifier le résultat.
  10. Vous pouvez éventuellement resélectionner les objets pour modifier les propriétés du motif.

Remarques

  • Les modèles SVG sont stockés dans des fichiers .SVG. Il est possible d'utiliser vos propres motifs personnalisés. Voir Préférences.
  • Les motifs eux-mêmes ne sont pas enregistrés dans le document FreeCAD. Les objets dont le motif VuePattern ne peut être trouvé sont affichés avec une couleur de surface solide à la place.

Préférences

Voir aussi : Réglage des préférences et Draft Préférences.

  • Pour changer VuePattern Size utilisée pour les nouveaux objets : Édition → Préférences → Draft → Paramètres visuels → Taille des motifs SVG.
  • Pour spécifier un répertoire contenant des motifs SVG supplémentaires : Édition → Préférences → Draft → Paramètres visuels → Emplacement des motifs SVG supplémentaires.
