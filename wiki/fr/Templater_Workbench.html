This workbench is launched with tools that are still in proof of concept state and need a bit feedback to turn them into proper tools.

Introduction

Templater est un atelier externe qui ne fait pas partie de l'installation standard de FreeCAD. Son objectif est de fournir des outils permettant de créer facilement des modèles TechDraw dans le format souhaité, ainsi que d'autres outils liés au dessin technique.

Installation

Cet atelier peut être installé à l'aide du gestionnaire d'extensions. Pour une installation manuelle, consultez Installation d'autres ateliers et téléchargez un fichier zip à partir de : ici.

Outils

  • Templater Vue auxiliaire : ajoute une vue auxiliaire à la page de dessin active en fonction de deux sommets sélectionnés dans une vue existante.
  • Templater Cadre de tolérance : ajoute un cadre de tolérance ou un cadre de référence à la vue sélectionnée.
  • Templater Modèle multi format : crée un modèle et l'insère comme nouvelle page de dessin dans le document actif. Cet outil permet de créer un modèle dans de nombreux formats différents et avec plusieurs éléments optionnels tels qu'un cadre, un cartouche, une nomenclature, etc.
  • Templater Modèle : crée un petit modèle et l'insère comme nouvelle page de dessin dans le document actif. Consultez le tutoriel TechDraw Création de modèles pour apprendre à créer un modèle presque identique à l'aide d'un script.
