Description

PartDesign Clone crée une copie liée d'un objet sélectionné, qui suivra toutes les modifications ultérieures apportées à l'objet d'origine (sauf le placement). Par exemple, est celui où vous voulez faire une opération PartDesign Booléenne sur un objet créé dans un autre atelier. La plupart des types d'objets sont acceptés, à condition qu'il s'agisse de solides. Si vous avez besoin de cloner plusieurs objets (corps) ou un Part, vous pouvez utiliser l'outil Draft Cloner. Un inconvénient est que le clone de l'atelier PartDesign définit l'emplacement en cours du clone à zéro (translation cartésienne et orientation spatiale). Tandis que le clone de l'atelier Draft calcule et définit les valeurs numériques de la position et de l'orientation en cours des objets clonés par rapport au conteneur de l'objet cloné.

Clone de la tige tout en étant translatée dans l'espace 3D comme objet indépendant

Utilisation

  1. Dans l'arborescence du modèle, sélectionnez l'objet à cloner.
  2. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Créer un clone.
    • Sélectionnez l'option PartDesign → Créer un clone du menu.

Propriétés

  • DonnéesBase Feature : définit l'objet d'origine sur lequel le clone est basé. Pour le remplacer, appuyez sur le bouton ... pour obtenir la liste des objets disponibles.
  • DonnéesPlacement : définit l'orientation et la position du clone dans l'espace 3D. Voir Placement.
  • DonnéesLabel : étiquette donnée à l'objet Clone. Changez en fonction de vos besoins.
