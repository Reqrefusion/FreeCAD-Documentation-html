Introduction

Std Base n'est pas vraiment un atelier mais plutôt une catégorie de commandes et d'outils "standards" qui peuvent être utilisés dans tous les ateliers.

Outils

La plupart des outils Std Base sont accessibles depuis le menu standard. Ceux qui ne sont disponibles que par une barre d'outils ou un menu contextuel sont répertoriés sous la barre d'outils Structure et les outils supplémentaires.

Menu standard

Le menu standard est composé de 7 sous-menus. Chaque sous-menu a une page dédiée. Cliquez simplement sur l'un des noms ci-dessous.

Barre d'outils Structure

  • Part : crée un nouveau part et le rend actif.
  • Créer des références introduit dans la version 1.1 :
  • Système de coordonnées : crée un objet de système de coordonnées qui peut être attaché à d'autres objets. introduit dans la version 1.1
  • Plan de référence : crée un objet plan de référence qui peut être attaché à d'autres objets. introduit dans la version 1.1
  • Ligne de référence : crée un objet ligne de référence qui peut être attaché à d'autres objets. introduit dans la version 1.1
  • Point de référence : crée un objet point de référence qui peut être attaché à d'autres objets. introduit dans la version 1.1
  • Groupe : crée un nouveau groupe.
  • Outils de liens :
  • Créer un lien : crée un lien.
  • Créer un sous-lien : crée un lien vers un sous-objet ou un sous-élément.
  • Remplacer par un lien : remplace le(s) objet(s) par un(des) nouveau(x) lien(s).
  • Délier : remplace le(s) lien(s) par leur(s) objet(s) lié(s).
  • Importer des liens : importe un (des) lien(s) externe(s) sélectionné(s).
  • Importer tous les liens : importe tous les liens externes.
  • Jeu de variables : crée un ensemble de propriétés qui peuvent être utilisées comme variables dans les expressions. introduit dans la version 1.0

Outils supplémentaires

  • Créer un groupe de liens : crée un groupe de liens.
  • Actions sur les expressions :
  • Copier la sélection : copie les données des expressions d'objets sélectionnés dans le presse-papiers.
  • Copier le document actif : copie les données des expressions du document actif dans le presse-papiers.
  • Copier tous les documents : copie les données des expressions de tous les documents dans le presse-papiers.
  • Coller : colle les données des expressions à partir du presse-papiers.
  • Filtre : introduit dans la version 1.0
  • Sélection de sommets : permet uniquement de sélectionner des sommets.
  • Sélection d'arêtes : permet uniquement la sélection des arêtes.
  • Sélection de faces : permet uniquement la sélection des faces.
  • Supprimer tous les filtres : permet de sélectionner tous les sous-éléments.
  • Sélectionner toutes les instances : sélectionne toutes les instances d'un objet dans la vue en arborescence.
  • Basculer le figeage : active/désactive le figeage des objets. Un objet figé n'est pas recalculé lorsque ses parents changent. introduit dans la version 1.0

Outils obsolètes

  • FAQ de FreeCAD : ouvre la page wiki FAQ de FreeCAD. Non disponible dans version 1.1 et suivantes.
  • Documentation pour programmer en Python : ouvre la page wiki de documentation destinée aux utilisateurs avancés de FreeCAD. Non disponible dans version 1.1 et suivantes.
  • Aide : ouvre la documentation d'aide. Non disponible dans version 1.1 et suivantes.
  • Mesurer une distance : crée un objet pour mesurer et afficher une distance. Non disponible dans version 1.0 et suivantes. Utilisez plutôt Std Mesure.
  • Supprimer les mesures : efface les mesures de Part. Non disponible dans version 1.0 et suivantes. Utilisez plutôt Std Mesure.
  • Basculer les mesures : permet de basculer l'affichage des mesures de Part. Non disponible dans version 1.0 et suivantes. Utilisez plutôt Std Mesure.
