Description

La commande SheetMetal Étendre par une esquisse étend une tôle au niveau d'une face d'une arête sélectionnée.

En relation avec une esquisse, elle peut créer une géométrie imbriquée pour fermer un profil :

Trois profils avec des esquisses à ajouter → trois résultats

Utilisation

(Préparez un esquisse pour les languettes d'interverrouillage)
  1. Sélectionnez une ou plusieurs faces à prolonger.
  2. Appuyez sur Ctrl et sélectionnez l'esquisse dans l'arborescence ou la vue 3D pour l'ajouter à la sélection.
  3. Il existe plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Étendre par une esquisse.
    • Sélectionnez l'option Tôle → Étendre par une esquisse du menu.
    • Cliquez avec le bouton droit de la souris dans l'arborescence ou la vue 3D et sélectionnez l'option Sheet Metal → Étendre par une esquisse dans le menu contextuel.
    • Utilisez le raccourci clavier : T.
  4. Un objet Extend sera créé et le panneau des tâches Extend properties s'ouvrira.
    • Vous pouvez appuyer sur le bouton Select et ajouter ou supprimer des faces.
    • Appuyez sur le bouton Preview pour terminer la sélection et afficher les modifications.
    • Vous pouvez appuyer sur le bouton Extend Sketch... pour sélectionner un autre esquisse.
    • Vous pouvez cocher la case Intersect Clearance et ajuster la valeur pour laisser de la place aux languettes d'interverrouillage.
  5. Appuyez sur le bouton OK pour quitter la commande et fermer le panneau des tâches.

Panneau des tâches

Double-cliquez sur un objet Extend existant dans la vue en arborescence pour ouvrir de nouveau le panneau des tâches et modifier les paramètres.

  • Sélectionner : modifie le nombre de faces dans la propriété base Object.
  • Étendre une esquisse… : remplace l'esquisse dans la propriété Sketch.
  • Case à cocher Jeu d'intersection : active ou désactive la propriété Use Subtraction.
  • Valeur Dégagement d'intersection : définit la propriété Offset.
  • Affiner : active ou désactive la propriété Refine.

Remarques

  • Cette commande crée le même objet Extend que la commande Extend Face. Le fait que la propriété DonnéesSketch soit vide ou non détermine quel panneau de tâches s'affiche lorsque l'objet Extend est modifié.
  • Voir Prolonger une face pour des remarques supplémentaires.

Propriétés

Voir aussi : Éditeur de propriétés

Un objet SheetMetal Extend est dérivé d'un objet Part Feature ou, s'il se trouve à l'intérieur d'un PartDesign Corps, à partir d'un objet PartDesign Feature, dont il hérite de toutes les propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Parameters

  • Donnéesbase Object (LinkSub) : objet de base. Lien vers l'objet et les faces planes à étendre.
  • Donnéesgap1 (Distance) : écart par rapport au côté gauche. Par défaut : 0,00 mm.
  • Donnéesgap2 (Distance) : écart par rapport au côté droit. Par défaut : 0,00 mm.
  • Donnéeslength (Length) : longueur de la paroi. Par défaut : 10,00 mm.
    • Longueur d'extension si Donnéesreversed = false.
    • Longueur de découpe si Donnéesreversed = true.
  • Donnéesreversed (Bool) : inverse le sens d'extension (découpe). Par défaut : false. Permet de basculer entre l'extension et la découpe.

Parameters Ext

  • DonnéesOffset (Distance): "Offset for subtraction". Default: 0,20 mm. Clearance for interlocking geometry
  • DonnéesRefine (Bool): "Use Refine". Default: true.
  • DonnéesSketch (Link): "Wall Sketch". Link to the sketch defining the interlocking geometry. Contains the selected sketch by default.
  • DonnéesUse Subtraction (Bool): "Use Subtraction". Default: false Toggles if intersected walls of the same object are cut out for tabs defined by the linked Sketch.
