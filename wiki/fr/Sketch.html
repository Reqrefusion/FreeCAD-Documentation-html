Introduction

Dans FreeCAD, le mot "Sketch" est normalement utilisé pour désigner un objet esquisse (classe Sketcher::Sketcher::SketchObject) définie par l'atelier Sketcher. Il s'agit d'un dessin 2D qui utilise des contraintes mathématiques pour décrire avec précision la géométrie 2D.

Voir Sketcher SketchObject pour plus d'informations sur le type d'objet.

Utilisation

Il existe deux manières courantes de créer une esquisse: directement dans l'atelier Sketcher, ou à partir de l'atelier PartDesign.

Atelier Sketcher

  1. Passez dans l'atelier Sketcher.
  2. Cliquez sur le bouton Créer une esquisse.

Atelier PartDesign

  1. Passez dans l' atelier PartDesign.
  2. Cliquez sur le bouton PartDesign Corps.
  3. Cliquez sur le bouton PartDesign Créer une esquisse.

Une fois l'édition de l'esquisse terminée, fermez-la pour quitter le mode d'édition. Double-cliquez dessus pour entrer à nouveau en mode édition.

Remarques

Une esquisse est couramment utilisée en conjonction avec l' atelier PartDesign pour créer des solides par extrusion, en utilisant l'opération PartDesign Protrusion.

Néanmoins, une esquisse peut toujours être créée à toute autre fin. Elle ne doit pas être liée à un PartDesign Corps. Par exemple, l'outil Arch Fenêtre de l' atelier BIM utilise des esquisses pour définir les formes des fenêtres et des portes. De la même manière, elles peuvent être utilisées pour définir la forme de Arch Murs.
