Description

Définit une condition limite de température ou, en option, une charge de flux thermique concentré.

Plaque 2D soumise à une différence de température

Utilisation

  1. Il existe plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Condition limite de température.
    • Sélectionnez l'option Modèle → Conditions limites et charges thermiques → Condition limite de température
  2. Appuyez sur le bouton Ajouter.
  3. Dans la vue 3D, sélectionnez les objets auxquels la condition limite doit être appliquée, qui peuvent être des sommets, des arêtes ou des faces. Vous pouvez également appuyer sur le bouton Supprimer et cliquez sur les objets que vous souhaitez supprimer de la sélection.
  4. Choisissez le type de contrainte et spécifiez son paramètre :
    • Temperature (par défaut) : condition limite de température, entrez la Température (K),
    • CFlux : charge de flux de chaleur concentré, entrez le Flux de chaleur concentré (mW). Cette valeur sera divisée par le nombre de nœuds de l'entité géométrique sous-jacente pour obtenir un flux total d'une magnitude donnée sur cette entité.

Propriétés

  • DonnéesEnable Amplitude : permet d'utiliser l'amplitude de CalculiX pour la variation temporelle de la condition limite de température. À un moment donné, la magnitude de la condition limite est multipliée par la valeur de l'amplitude correspondante à ce moment. L'interpolation linéaire est utilisée entre les valeurs spécifiées. Des valeurs constantes sont utilisées en dehors de la plage spécifiée. introduit dans la version 1.1
  • DonnéesAmplitude Values : tableau d'amplitude au format suivant introduit dans la version 1.1 :
time_1, amplitude_value_1
…
La valeur par défaut est une augmentation linéaire de 0 % à t=0 à 100 % à t=1 :
0, 0
1, 1

Remarques

  • La condition limite de température utilise le le jeu de paramètres *BOUNDARY dans CalculiX.
  • L'option flux de chaleur concentré utilise le le jeu de paramètres *CFLUX dans CalculiX.
