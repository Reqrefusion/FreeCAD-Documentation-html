Description

L'outil Cables Gaine crée un objet CableConduit basé sur une sélection de Cables WireFlex ou Cables Composites et éventuellement un profil. Le gaine est entièrement paramétrique et est dérivé de Arch Conduite et hérite de toutes ses propriétés.

Cet élément est conçu pour modéliser les couches supplémentaires utilisées pour construire des conduits de câbles, des harnais ou des assemblages de câbles. Il peut regrouper d'autres conduits ou câbles comme ses enfants sous l'arborescence et prendre une forme qui est un composite de sa propre forme paramétrique et des formes de ses enfants.


Deux ensembles de gaines de câbles (MC1, MC2) :
La gaine MC1 comprend deux autres gaines C1, C2 et le câble CB1.
La gaine MC2 comprend la gaine C3 et le câble CB2.

Utilisation

  1. Sélectionnez au moins un objet fil existant (voir Note 1) dans la vue 3D ou la vue en arborescence.
  2. Créez la gaine par l'une des méthodes suivantes :
    • Appuyez sur le bouton CableConduit.
    • Sélectionnez l'option Cables → CableConduit du menu.
    • Cliquez avec le bouton droit de la souris dans la vue 3D ou la vue en arborescence et sélectionnez l'option Cables → CableConduit dans le menu contextuel.

Remarques

  • Remarque 1 : les objets polylignes suivants peuvent être utilisés en toute sécurité comme base de sélection pour une gaine : WireFlex, Composite, Draft Polyligne. D'autres objets contenant la propriété Points (par exemple Draft B-spline, Draft Courbe de Bézier) peuvent également être utilisés, mais les résultats peuvent être différents du résultat escompté. Voir Remarques sur le composite.
  • Remarque 2 : si plus d'un objet a été sélectionné lors de la création d'une gaine, un nouvel objet CompoundPath est d'abord créé, puis la gaine est créée sur la base de cet objet.
  • Remarque 3 : un profil optionnel peut être ajouté plus tard après la création de la gaine. Les types de profils parémétriques suivants sont possibles sans profil externe : cercle, carré, rectangle. Pour plus de détails, voir Propriétés de Arch Conduite.

Propriétés

Données

Component

  • DonnéesBase (Link) : la polyligne de base d'un câble.

Pour les autres propriétés du groupe, voir Arch Composant.

Conduit

  • DonnéesAuto Label Base (Bool) : si la propriété est définie sur TRUE, le nom d'une polyligne de base sera renommé automatiquement. Pour rafraîchir le nom, la propriété doit être changée en FALSE puis à nouveau en TRUE.
  • DonnéesMerge Sub Conduits (Bool) : si la valeur est TRUE, tous les objets de la propriété Sub Conduits seront fusionnés dans la forme finale. L'objet Part Composite contenant les sous-gaines est créé en premier, puis il est ajouté à la forme finale avec la propre forme de la gaine.
  • DonnéesShow Sub Conduits (Bool) : affiche/masque tous les sous-gaines. Cela ne fonctionne que si Merge Sub Conduits est réglé sur TRUE.
  • DonnéesSub Conduits (Link List) : liste des gaines ou des câbles appartenant au faisceau.

Conduit Dimensions

  • DonnéesGauge (Float) : jauge extérieure due la gaine. Unité : [mm^2]. Il est corrélé avec les propriétés des dimensions : Diameter, Width et Height en fonction de la propriété Profile Type.
  • DonnéesInsulation Thickness (Length) : épaisseur de l'isolation si le profil n'est pas utilisé. Elle conserve la même valeur que Wall thickness.
