Description

La commande Part Point de référence crée un objet datum point qui peut être ancré à d'autres objets. Un point de référence est un des objets de référence. Un objet de référence est généralement utilisé pour ancrer plusieurs autres objets. Si la position ou l'orientation d'un objet de référence change, tous les objets qui lui sont ancrés suivront.

Utilisation

Voir Part Système de coordonnées.

Remarques

Voir Part Système de coordonnées.

Propriétés

Voir aussi : Éditeur de propriétés

Un objet Part::DatumPoint est dérivé d'un objet App::Point. Toutes ses propriétés sont héritées, à l'exception de celles énumérées sous Attachment.
