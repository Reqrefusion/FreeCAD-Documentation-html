Description

SuppLines est un objet dépendant et ne peut pas être créé directement.

It can be created as a child object during creation of: CableBox, CableConnector, ElectricalDevice, CableLightPoit.

Il sert de support aux objets WireFlex qui constituent la base des câbles. L'objet SuppLines lui-même est attaché aux autres objets mentionnés ci-dessus et conserve un emplacement prédéfini par rapport à ceux-ci. SuppLines a la forme d'une croix à cinq sommets qui peut être utilisée comme support de fixation pour jusqu'à cinq câbles entrant dans un seul trou dans une boîte à câbles. Il peut également servir de point de référence pour les câbles entrant dans le connecteur ou l'appareil électrique. C'est à l'utilisateur de décider si SuppLines lui est utile ou non.

Important: the real relation between SuppLines object and its parent is reversed to avoid cyclic dependencies and to speed up recalculation. This has some consequences (see Notes).

Example of a default SuppLines shape.

Usage

The SuppLines object cannot be created directly. It is made during the creation of: CableBox, CableConnector, ElectricalDevice, CableLightPoint depending on Number Of Supp Lines property of these objects.

Notes

  • The SuppLines object has reversed relation with its parent to avoid cyclic dependencies. It is shown as a child in the Tree View only for visual organization of objects. Undesirable consequence: when a parent is copied, all its child SuppLines objects must always be manually copied with it.
  • Depending of parent the single SuppLines object can have one or more crosses.

Properties

The SuppLines object has not any visible properties to edit. Its shape can be created and modified by its parent object or take the form of a cross with a default size.
