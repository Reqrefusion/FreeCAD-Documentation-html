Description

La commande Fasteners Rechercher affiche une boîte de dialogue permettant de rechercher et d'insérer des fixations par leur nom ou leur description.

Le panneau des tâches de la commande

Utilisation

For general usage instructions see Fasteners Workbench.

  1. There are several ways to invoke the command:
    • Press the Search Fasteners button.
    • Select the Fasteners → Search Fasteners option from the menu.
  2. The Search Fasteners dialog box opens.
  3. Enter a search term in the input box at the top. At least 2 characters are required.
  4. The list of search results updates as more characters are entered.
  5. Select a fastener from the list.
  6. Press the Insert Fastener button.
  7. The dialog is modeless, meaning it can stay open while you continue working in FreeCAD.
  8. Optionally press the Close button to close the dialog box.
