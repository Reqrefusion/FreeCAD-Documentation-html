Introduction

Dans FreeCAD, le mot "Drawing" est normalement utilisé pour faire référence à une projection 2D d'un modèle 3D. Ceci est généralement créé avec l'atelier TechDraw.

Utilisation

  1. Commencez avec un modèle 3D déjà créé avec par exemple l'atelier PartDesign. En fait, tout objet qui a une Shape y compris les objets 2D, fonctionnera.
  2. Basculez vers l'atelier TechDraw.
  3. Appuyez sur Feuille standard ou Page selon modèle pour créer une page.
  4. Sélectionnez le modèle existant, puis appuyez sur Vue ou sur Groupe de projections.
  5. Une projection 2D sur la page sera créée. Vous pouvez maintenant ajuster ses propriétés comme DonnéesScale, DonnéesRotation et DonnéesDirection.
  6. Lorsque le dessin est prêt, vous pouvez appuyer sur Exporter en SVG, Exporter en DXF ou utilisez Std Exporter pour exporter la page aux formats SVG, DXF ou PDF pour une utilisation ultérieure dans un autre logiciel ou pour l'impression.

Remarques

Dans un usage informel, un "Drawing" peut être n'importe quelle figure géométrique visible dans la vue 3D, et donc son concept peut être confondu avec celui de "Corps", "Part" ou "modèle".

Cependant, lorsque plus de précision est requise, la distinction doit être faite.

  • Un "Corps" (Body) (PartDesign Corps) est un objet dérivé d'une Part Feature classe (Part::Feature) créé avec l'atelier PartDesign.
  • Un "Part" (App Part) est utilisé pour regrouper plusieurs "Bodies" pour former un assemblage.
  • Un "Drawing" est une projection 2D d'un "Body" ou d'une "Part".
