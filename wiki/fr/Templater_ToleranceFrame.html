Description

L'outil Templater Cadre de tolérance crée un cadre de tolérance ou un cadre de référence sous forme de fichier SVG et l'ajoute à la vue sélectionnée.

Utilisation

  1. Sélectionnez une vue ou n'importe quelle géométrie à l'intérieur de la vue.
  2. Il existe plusieurs façons de lancer l'outil.
    • Appuyez sur le bouton Tolerance Frame.
    • Sélectionnez l'option Templater → Tolerance Frame du menu.
  3. Le panneau de tâches Feature frame settings s'ouvre.
  4. Sélectionnez un Tolerance Type pour un cadre de tolérance ou saisissez une seule lettre (en majuscule) pour un cadre de référence.
  5. Si un type de tolérance a été sélectionné, ajustez les valeurs suivantes :
    (elles seront ignorées pour un cadre de référence)
    • Vous pouvez modifier la valeur de la tolérance.
    • Vous pouvez modifier chaque champ de référence. Laissez-les vides si vous n'en avez pas besoin, les champs suivants seront ignorés.
  6. Cliquez sur Ok pour fermer le panneau de tâches et terminer l'utilisation de l'outil.
  7. Un fichier NewSymbol.svg sera enregistré dans le dossier .../Mod/Templater/Resources et ajouté à la vue sélectionnée en tant que nouvel objet FeatureFrame.
  8. Faites glisser l'objet FeatureFrame vers sa position finale.
  9. Vous pouvez ajuster ses valeurs en modifiant la propriété DonnéesEditable Texts dans l'éditeur de propriétés

Propriétés

Voir aussi : Éditeur de propriétés

Un FeatureFrame est dérivé d'un TechDraw Symbole, formellement un objet TechDraw::DrawViewSymbol et hérite de toutes ses propriétés.
