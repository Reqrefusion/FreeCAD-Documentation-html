Description

Convertit les arêtes en faces. Utile pour préparer une géométrie DXF importée pour pouvoir l'extruder.

Utilisation

  1. Sélectionner les bords à convertir dans la vue 3D.
  2. Choisir OpenSCAD → Convertir les arêtes en faces du menu.

Limitations

Remarques

Voir aussi Draft Agréger pour une solution alternative pour obtenir des faces.
