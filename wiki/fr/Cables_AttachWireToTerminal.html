Description

Connecter un câble à une terminaison relie deux sommets initiaux ou deux sommets finaux d'un wireFlex à des terminaisons de câbles existantes. Les connexions des sommets sont conservées dans les propriétés spéciales de WireFlex. Pour plus de détails, voir WireFlex Propriétés.

The electrical connection between terminal and wire is established. The terminal properties from the Net group are updated accordingly.

Once attached the connected wire vertices keep the same position in 3D space as the CableTerminal to which they are attached to. When the CableTerminal is moved, the attached wire vertices move together with it. When the wire is edited in the Cables Edit the attached vertices can not be moved. They jump back to their attached position.

This tool prevents more than one wire from being connected to a single CableTerminal segment. To enable the connection of more wires to a single terminal, the number of its segments must be increased first by changing the DonnéesNumber Of Connections property.

Usage

There are several selection methods to perform Attach Wire To Terminal command. Each selection method can give different results after running the command:

  1. Use one of the WireFlex selection methods:
    • Select first or last vertex of WireFlex object in the 3D View.
    • Select an edge close to first or last vertex of WireFlex object in the 3D View.
  2. Use one of the CableTerminal selection methods holding the CTRL:
    • Select one CableTerminal vertex in the 3D View (selected vertex will be the entry point of the wire).
    • Select one CableTerminal edge (segment) in the 3D View (first free segment and its vertex with a lower index will be the entry point of the wire).
  3. Run an Attach Wire To Terminal command by one of these methods:
    • Press the Attach Wire To Terminal button.
    • Select the Cable Wires → Attach Wire To Terminal option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cable Wires → Attach Wire To Terminal option from the context menu.
    • Use the keyboard shortcut: T then A.

Notes

  • If the WireFlex object has less then 4 vertices, the missing ones will be added automatically.
  • See the WireFlex Description and WireFlex Usage for the details how to use WireFlex.
  • If a Cable is built upon connected WireFlex some of their properties may need to be adjusted manually for the connection to display correctly (see CableTerminal Notes).
