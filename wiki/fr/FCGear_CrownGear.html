Description

La commande FCGear Couronne dentée crée une couronne dentée qui tient également compte de la taille du pignon correspondant.

La couronne dentée ressemble à une crémaillère incurvée en forme d'anneau. L'angle de pression décroît continuellement du diamètre extérieur vers le diamètre intérieur. Ainsi, la vitesse périphérique variable au niveau de la couronne dentée est compensée par la vitesse périphérique constante du pignon. Les dents externes pointues et les flancs de dents raides sur le diamètre intérieur limitent la largeur de dent utilisable. Les couronnes dentées atteignent des rendements similaires à ceux des roues dentées droites. Une couronne dentée peut entraîner plusieurs pignons.

Champ d'application connu des couronnes dentées :

  • Entraînement d'essieu arrière pour voitures et motos
  • Mécanisme pivotant pour tables d'opération
  • Têtes de fraisage angulaires
  • Systèmes d'outils motorisés avec plusieurs pignons et couronne dentée

Au-dessus : couronne dentée

Utilisation

  1. Passez à l' atelier FCGear.
  2. Il existe plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Couronne dentée dans la barre d'outils.
    • Sélectionnez l'option Gear → Couronne dentée du menu.
  3. La couronne dentée est affichée sans dents par défaut. (introduit dans la version 0.21)
  4. Modifiez les paramètres de la couronne dentée en fonction des conditions requises (voir Propriétés).
  5. Définissez la propriété Donnéespreview_mode à false pour afficher les dents (voir Remarques).

Propriétés

Voir aussi : Éditeur de propriétés

Un objet FCGear CrownGear est dérivé d'un Part Feature et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

accuracy

  • Donnéesnum_profiles (Integer) : valeur par défaut à 4. Nombre de profils utilisés pour le lissage.
  • Donnéespreview_mode (Bool) : valeur par défaut à true.

base

  • Donnéesheight (Length) : valeur par défaut à 2 mm. Valeur pour la largeur de la dent.
  • Donnéesmodule (Length) : valeur par défaut à 1 mm. Module est le rapport du diamètre de référence de la couronne dentée divisé par le nombre de dents (voir Notes).
  • Donnéesnum_teeth (Integer) : valeur par défaut à 15. Nombre de dents.
  • Donnéesother_teeth (Integer) : valeur par défaut à 15. Nombre de dents de la couronne dentée de construction (pignon) (voir Remarques).
  • Donnéesthickness (Length) : valeur par défaut à 5 mm. Hauteur de la pointe de la dent à la face inférieure de la couronne dentée.

involute

  • Donnéespressure_angle (Angle) : valeur par défaut à 20° (voir Remarques).

version

  • Donnéesversion (String) :

Remarques

  • La propriété Donnéespreview_mode est définie à true par défaut et lorsque la couronne dentée est créée, vous trouverez ce message dans la vue du rapport :
    Gear module: Crown gear created, preview_mode = true for improved performance. Set preview_mode property to false when ready to cut teeth.
    (Module d'engrenage : création d'une couronne dentée, preview_mode = true pour une meilleure performance. Définissez la propriété preview_mode à false lorsque vous êtes prêt à couper les dents.)
  • module : en utilisant les directives ISO (Organisation internationale de normalisation), la taille du module est désignée comme l'unité représentant la taille des dents des couronnes dentées. Module (m): m = 1 (p = 3.1416), m = 2 (p = 6.2832), m = 4 (p = 12.566). Si vous multipliez le module par Pi, vous pouvez obtenir le pas (p). Le pas est la distance entre les points correspondants sur les dents adjacentes.
  • other_teeth : plusieurs pignons avec le même nombre de dents ne peuvent être utilisés que sur une seule couronne.
  • pressure_parameter : ne modifiez le paramètre que si une connaissance suffisante de la géométrie de la couronne dentée est disponible.
  • La géométrie de la couronne dentée est principalement déterminée par la géométrie du pignon droit (other_teeth).
  • Créez une roue dentée avec l'Roue dentée à développante. Le nombre de dents doit être identique au paramètre other_teeth de la couronne dentée.
  • Des ajustements pour des caractéristiques de fonctionnement optimales peuvent être effectués avec les paramètres de la roue dentée à développante.

  • Diamètre intérieur (4)
    • inner diamter = module (engrenage droit) * teeth (couronne dentée) * cos pressure_paramter (pignon) : cos pressure_parameter (couronne dentée)

Script

Utilisez la puissance de Python pour automatiser la modélisation de votre roue dentée : 

import FreeCAD as App
import FreeCADGui as Gui
import freecad.gears.commands
gear = freecad.gears.commands.CreateCrownGear.create()
gear.num_teeth = 20
gear.other_teeth = 15
gear.height = 3
gear.preview_mode = False
App.ActiveDocument.recompute()
Gui.SendMsgToActiveView("ViewFit")
