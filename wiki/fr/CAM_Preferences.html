Atelier CAM

Présentation

Les préférences de l'atelier CAM se trouvent dans l'éditeur de préférences Édition → Préférences → CAM.

Il existe cinq pages : Interface graphique, Équipements, Préférences des tâches, Finitions et Avancé.

Interface graphique

Équipements

A définir

introduit dans la version 1.2 : les équipements machine peuvent être gérées depuis les préférences de CAM. La bibliothèque de machines et l'éditeur de machines permettent de créer et de modifier les fichiers de configuration de machine *.fcm utilisés par le post-traitement basé sur la machine.
