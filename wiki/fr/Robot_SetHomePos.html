Description

Description

Définit la position/orientation actuelle du robot sélectionné comme position de départ.

Usage

Utilisation

  1. Sélectionnez le robot dans la Vue en arborescence ou dans la Vue 3D.
  2. Cliquez sur ou choisissez Robot. → Définir la position de départ dans le menu supérieur.
