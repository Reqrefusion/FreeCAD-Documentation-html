Description

La commande FCGear Poulie synchrone type T crée une poulie de synchronisation de type T.

Au-dessus : poulie synchrone type T

Utilisation

  1. Passez à l' atelier Gear.
  2. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Poulie synchrone type T dans la barre d'outils.
    • Sélectionnez l'option Gear → Poulie synchrone type T du menu.
  3. Modifiez le paramètre de la poulie en fonction des conditions requises (voir Propriétés).

Propriétés

Voir aussi : Éditeur de propriétés

Un objet FCGear TimingGearT est dérivé d'un Part Feature et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

base

  • Donnéesalpha (Length) : valeur par défaut 40°. Angle des flancs des dents.
  • Donnéesheight (Length) : valeur par défaut 5 mm. Hauteur d'extrusion.
  • Donnéesnum_teeth (Integer) : valeur par défaut 15. Nombre de dents.
  • Donnéespitch (Length) : valeur par défaut 5 mm. Pas de la poulie.
  • Donnéestooth_height (Length) : valeur par défaut 5 mm. Hauteur radiale de la dent.
  • Donnéesu (Length) : valeur par défaut 5 mm. Distance radiale entre la tête de la dent et le cercle primitif.

fillets

  • Donnéeshead_fillet (FloatConstraint) : valeur par défaut 0.4. Un congé pour la tête de dent, radius = head_fillet x module.
  • Donnéesroot_fillet (FloatConstraint) : valeur par défaut 0.4. Un congé pour la racine de la dent, radius = root_fillet x module.

tolerance

  • Donnéesbacklash (Length) : valeur par défaut 0 mm. Longueur de l'arc sur le cercle primitif par lequel l'épaisseur de la dent est réduite.

Script

Utilisez la puissance de Python pour automatiser la modélisation de votre roue dentée : 

import FreeCAD as App
import FreeCADGui as Gui
import freecad.gears.commands
gear = freecad.gears.commands.CreateTimingGearT.create()
gear.num_teeth = 12
gear.height = 5
App.ActiveDocument.recompute()
Gui.SendMsgToActiveView("ViewFit")
