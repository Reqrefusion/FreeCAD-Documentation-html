Description

Les calques étendus constituent un ensemble prédéfini de calques, chacun étant une version étendue d'un Draft Calque. Outre les propriétés standard d'un calque de Draft, le calque étendu permet de définir la couleur et la taille des points des éléments appartenant au calque. Chaque calque étendu dispose également d'une propriété Auto Member Type conçue pour regrouper automatiquement les éléments du calque.

Les calques étendus ont été créés pour faciliter l'organisation des lignes des WireFlex, des lignes, des points, des SnapLines et des CableTerminals. Ces calques vous permettent d'affecter rapidement et facilement des objets non regroupés d'un type spécifique au calque approprié, puis de les masquer ou de les afficher, ainsi que d'appliquer des modifications groupées à leurs propriétés d'affichage.

Il suffit de recalculer le calque sélectionné (par exemple, en cliquant dessus avec le bouton droit de la souris et en sélectionnant Recalculer l'objet) pour que tous les éléments non regroupés du type sélectionné lui soient attribués.

Utilisation

  1. Créez un ensemble par défaut de couches étendues à l'aide de l'une des méthodes suivantes :
    • Cliquez sur le bouton Layers Extended.
    • Sélectionnez l'option Cable Support → Layers Extended du menu.
    • Cliquez avec le bouton droit de la souris dans l'arborescence ou la vue 3D et sélectionnez l'option Cable Support → Layers Extended dans le menu contextuel.

Remarques

  • Le calque par défaut dont la propriété Auto Member Type est définie sur None peut servir de modèle pour la création d'autres calques étendus.
  • Il est possible d'utiliser plusieurs calques avec la même propriété Auto Member Type en la modifiant manuellement.
  • Si vous exécutez à nouveau la commande LayersExtended, celle-ci générera des couches pour tous les types Auto Member Type possibles qui n'ont pas encore été créés, ainsi qu'un calque supplémentaire avec la propriété Auto Member Type définie sur None.

Propriétés

Voir aussi : Éditeur de propriétés

Un objet Layer Extended est dérivé d'un Draft Calque et en hérite toutes les propriétés. Layer Extended possède également les propriétés supplémentaires suivantes :

Données

Layer

  • DonnéesAuto Member Type (Enumeration) : spécifie le type de membre automatique de ce calque. Lorsque le calque est recalculé, il inclut automatiquement tous les objets du type sélectionné qui n'ont été affectés à aucun autre calque. Valeurs possibles : None, WireFlex, SupportLines, ExtSnapLines, CableTerminal. Lorsque None est sélectionné, le type de membre automatique devient inactif.

Vue

Layer

  • DonnéesPoint Color (Color) : définit la couleur des points du calque.
  • DonnéesPoint Size (Float) : définit la taille des points du calque.
