Description

Les calques étendus constituent un ensemble prédéfini de calques, chacun étant une version étendue d'un Draft Calque. Outre les propriétés standard d'un calque de Draft, le calque étendu permet de définir la couleur et la taille des points des éléments appartenant au calque. Chaque calque étendu dispose également d'une propriété Auto Member Type conçue pour regrouper automatiquement les éléments du calque.

Les calques étendus ont été créés pour faciliter l'organisation des lignes des WireFlex, des lignes, des points, des SnapLines et des CableTerminals. Ces calques vous permettent d'affecter rapidement et facilement des objets non regroupés d'un type spécifique au calque approprié, puis de les masquer ou de les afficher, ainsi que d'appliquer des modifications groupées à leurs propriétés d'affichage.

Simply recompute the selected layer (e.g. by right clicking it and selecting Recompute Object), and all ungrouped elements of the selected type will be assigned to it.

Usage

  1. Create default set of extended layers by one of these methods:
    • Press the Layers Extended button.
    • Select the Cable Support → Layers Extended option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cable Support → Layers Extended option from the context menu.

Notes

  • The default layer with Auto Member Type property set to None can serve as template for creation other extended layers.
  • It is possible to use more than one layer with the same Auto Member Type property by changing it manually.
  • If you run the LayersExtended command again, it will generate layers for all possible Auto Member Type types that have not yet been created, plus one additional layer with None Auto Member Type.

Properties

See also: Property editor.

A Layer Extended object is derived from a Draft Layer and inherits all its properties. Layer Extended has also the following additional properties:

Data

Layer

  • DonnéesAuto Member Type (Enumeration): specifies the type of automatic member of this layer. When layer is recomputed it automatically includes all objects of selected type which were not assigned to any other layer. Possible values: None, WireFlex, SupportLines, ExtSnapLines, CableTerminal. When None is selected the Auto Member Type becomes inactive.

View

Layer

  • DonnéesPoint Color (Color): specifies the point color of the layer.
  • DonnéesPoint Size (Float): specifies the point size of the layer.
