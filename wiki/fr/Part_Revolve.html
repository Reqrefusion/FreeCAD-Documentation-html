Description

La commande Part Révolution fait pivoter l'objet sélectionné autour d'un axe donné. Les formes initiales suivantes sont autorisées et amènent aux formes de sortie énumérées :

Objet initial Objet résultant
Sommet Arête
Arête Face
Polyligne Enveloppe
Face Solide
Coque Solide composé (Compsolid)

Une esquisse peut également être utilisée. Les solides ou les solides composés ne sont pas autorisés comme formes de départ. Les composés normaux ne sont actuellement pas autorisés non plus.

Exemples de révolution

Utilisation

  1. Vous pouvez sélectionner une ou plusieurs formes dans la vue 3D ou dans l'arborescence.
  2. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Créer une révolution.
    • Sélectionnez l'option Part → Créer une révolution du menu.
  3. Le panneau des tâches Paramètres de la révolution s'ouvre.
    • Vous pouvez cliquez sur un élément de la liste Formes pour (re)sélectionner une forme.
    • Vous pouvez également maintenir la touche Ctrl enfoncée et cliquer sur un élément de la liste Formes pour ajouter la forme à la sélection ou la supprimer de la sélection.
    • Définissez l'axe et l'angle de révolution et éventuellement d'autres paramètres (voir la section Propriétés pour plus de détails).
  4. Appuyez sur OK pour fermer le panneau des tâches.
  5. Un objet Revolve sera créé pour chaque forme sélectionnée.

Chaque forme initiale est placée sous son objet Revolve.

Panneau des tâches

  • Le bouton OK crée la révolution et ferme le panneau.
  • Le bouton Annuler ferme le panneau sans effectuer aucune action.
  • Liste des formes : vous sélectionnez ici les formes à faire pivoter. Si plusieurs objets sont sélectionnés, plusieurs objets Revolve sont créés.
  • La section Axe de révolution spécifie l'axe de révolution.
    • Champs des centres X/Y/Z : définit le point de base de l'axe de révolution.
    • Champs des directions X/Y/Z : définit la direction de l'axe de révolution. Cliquez sur un bouton pour définir la direction de l'axe correspondant.
    • Bouton Sélectionner une référence : cliquez dessus, puis sélectionnez une arête dans la vue 3D. Cette arête apparaîtra dans le champ de texte sous le bouton, au format « ObjectName:EdgeN ». Vous pouvez également saisir le lien manuellement ou l'effacer. Les valeurs Direction X/Y/Z seront renseignées en fonction de la direction de l'arête.
  • Champ Angle : définissez l'angle de révolution en degrés.
  • Angle symétrique : si cette case est cochée, la révolution s'étendra vers l'avant et vers l'arrière de la moitié de l'angle spécifié.
  • Case à cocher Créer un solide : si cette case est cochée, la révolution d'un fil ou d'une arête fermé(e) produira un solide. Elle est cochée par défaut si une polyligne fermée a été présélectionnée avant de lancer Part Révolution.

Remarques

  • L'argument Angle spécifie la distance à laquelle l'objet doit être pivoté. Les coordonnées déplacent l'origine de l'axe de rotation par rapport à l'origine du système de coordonnées.
  • Si vous sélectionnez un axe défini par vous, les nombres définissent la direction de l'axe de rotation par rapport au système de coordonnées : si la coordonnée Z est égale à 0 et que les coordonnées Y et X ne sont pas nulles, l'axe se situera dans le plan X-Y. Son angle est tel que sa tangente est le rapport des coordonnées X et Y données.
  • Les objets App Link liés aux types d'objets appropriés peuvent également être utilisés comme formes et pour spécifier l'axe. introduit dans la version 0.20
  • Si l'objet à faire pivoter coupe l'axe de rotation, l'opération échouera dans la plupart des cas.

Properties

See also: Property View.

A Part Revolve object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

Extrude

  • DonnéesSource (Link): The shape to revolve.
  • DonnéesBase (Vector): The base point of the revolution axis.
  • DonnéesAxis (Vector): The direction of the revolution axis.
  • DonnéesAxis Link (LinkSub): Link to the edge to use as the revolution axis. Overrides DonnéesBase and DonnéesAxis if set.
  • DonnéesAngle (Angle): The angle span of the revolution. If zero and an arc is used for the DonnéesAxis Lik, the arc's span will be used.
  • DonnéesSymmetric (Bool): If true, the revolution is extended symmetrically from the profile.
  • DonnéesSolid (Bool): If true, revolving a closed edge or a closed wire will yield a solid. If False, a shell will result.
  • DonnéesFace Maker Class (String): If DonnéesSolid is true, the facemaker class to use when converting wires to faces, otherwise ignored.
