Curves Enveloppe de tube

Description

L'outil Curves Profils d'un tube crée un objet Profile pour une enveloppe de tube.

Utilisation

  1. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Créer les profils d'un tube.
    • Sélectionnez l'option Surfaces → Créer les profils d'un tube du menu.
  2. Un objet Profile est créé.

Remarques

Propriétés

Voir aussi : Éditeur de propriétés

Un objet Profile est dérivé d'un Part Feature et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Profile

  • DonnéesContact (Bool) : translate le profil au contact de la trajectoire.
  • DonnéesCorrection (Oool) : fait pivoter le profil pour qu'il soit orthogonal à la trajectoire.
  • DonnéesLocation (LinkSub) : emplacement du sommet sur la trajectoire.
  • DonnéesProfil (LinkSubList) : sous-formes du profil.
