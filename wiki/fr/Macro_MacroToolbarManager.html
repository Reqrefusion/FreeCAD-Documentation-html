Description

MacroToolbarManager facilite l'édition de barres d'outils de macros personnalisées.

La documentation complète est disponible sur GitHub : MacroToolbarManager. Remarque : la dernière version nécessite une version récente de FreeCAD. Si cela ne fonctionne pas, essayez la version du 27 octobre 2023 disponible via le lien GitHub ci-dessus.
