Description

Curves Segmenter une surface permet de segmenter une surface en isocurves. Cet outil fait partie des ateliers externes appelé Curves.

Utilisation

  1. Passez à l'atelier Curves (à installer à partir du gestionnaire des extensions si ce n'est pas déjà fait)
  2. Pour lancer la commande, effectuez l'une des opérations suivantes :
    • Appuyez sur le bouton dans la barre d'outils de Curves.
    • Utilisez la commande Surfaces → Segment surface.

Propriétés

See also: Property editor.

A Segment_Surface object is derived from an App FeaturePython object and inherits all its properties. It also has the following additional properties:

Data

Base

  • DonnéesOption (Enumeration): Option list.
    • Custom (default): ...
    • Auto: ...
  • DonnéesSource (LinkSub): Initial Face.

Option Auto

  • DonnéesDirection (Enumeration): Segmenting direction.
    • U (default): ...
    • V: ...
    • Both: ...

UDirection

  • DonnéesKnots U (FloatList): Splitting parameters in U direction.
  • DonnéesKnots UProvider (Link): Object generating normalized parameters in U direction.
  • DonnéesNumber U (Integer): Split the U parameter range in the given number of segments.

VDirection

  • DonnéesKnots V (FloatList): Splitting parameters in V direction.
  • DonnéesKnots VProvider (Link): Object generating normalized parameters in V direction.
  • DonnéesNumber V (Integer): Split the V parameter range in the given number of segments.
