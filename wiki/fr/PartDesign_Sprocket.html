Description

Cet outil vous permet de créer un profil 2D d'un pignon (ou roue à chaîne). Il peut être extrudé avec la fonction PartDesign Protrusion.

Utilisation

  1. Activez le bon corps.
  2. Sélectionnez l'option PartDesign → Créer un pignon... du menu.
  3. Définissez DonnéesNumber Of Teeth et DonnéesSprocket Reference.
  4. Cliquez sur OK.
  5. S'il n'y avait pas de corps actif : faites un glisser/déposer du pignon dans un corps pour appliquer d'autres fonctions comme par exemple une extrusion.

Propriétés

  • DonnéesNumber Of Teeth : nombre de dents.
  • DonnéesSprocket Reference : type de pignon. Une liste de définitions de pignons. La liste comprend les normes ANSI et ISO ainsi que certaines définitions de pignons pour bicyclettes et motos.
  • DonnéesPitch : distance entre deux dents.
  • DonnéesRoller Diameter : diamètre de la chaîne à rouleaux pour laquelle le pignon est conçu.
  • DonnéesThickness : épaisseur principale du pignon. Remarque : le pignon ne peut pas simplement être extrudé avec cette épaisseur car les dents ont des chanfreins latéraux. Il faut donc regarder la définition du pignon pour modéliser un pignon 3D valide.
