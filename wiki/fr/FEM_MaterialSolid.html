Description

Crée un matériau pour le solide.

Le panneau des tâches des matériaux FEM

Utilisation

  1. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Matériau pour solide.
    • Sélectionnez l'option Modèle → Matériaux → Matériau pour solide du menu.
  2. Le panneau des tâches de matériau FEM s'ouvre.
  3. Sélectionnez un matériau dans la liste déroulante. Par exemple, pour l'analyse mécanique, CalculiX-Steel est une option courante. Vous pouvez également appuyer sur le bouton Lancer l'éditeur pour accéder à l'éditeur de matériaux.
  4. Vous pouvez également cocher la case Utiliser ce panneau de tâches pour modifier les propriétés du matériau telles que la densité, le module de Young, le coefficient de Poisson, etc.
  5. Si vous appliquez le matériau à l'ensemble de l'objet, ne sélectionnez aucune entité géométrique (laissez la liste de référence vide). Le matériau sera appliqué à l'ensemble du modèle. Sinon, attribuez manuellement le matériau aux régions sélectionnées en sélectionnant certaines d'entre elles pour chaque matériau inséré, mais ne laissez aucune partie du modèle sans matériau attribué.
  6. Cliquez sur le bouton Fermer pour fermer le panneau des tâches.

Remarques

  • Pour modifier un objet MaterialSolid existant : double-cliquez dessus dans la vue en arborescence.
  • Le matériau mécanique utilise le jeu de paramètres *MATERIAL de CalculiX.
