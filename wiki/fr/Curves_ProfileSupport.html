Description

L'outil Curves Plan de support crée un plan de support pour des esquisses.

Utilisation

  1. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Créer un plan de support.
    • Sélectionnez l'option Surfaces → Créer un plan de support du menu.
  2. Un objet Profile est créé.

Remarques

  1. Pour voir comment cet outil peut être utilisé pour construire une section incurvée d'un corps de guitare, voir cette réponse du forum de Chris_G (auteur de l'atelier Curves).

Propriétés

Voir aussi : Éditeur de propriétés

Un objet Profile est dérivé d'un Part Feature et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Profile

  • Données (Dynamic)Edge1 (LinkSub) : première arête de support.
  • Données (Dynamic)Edge2 (LinkSub) : deuxième arête de support.
  • Données (Dynamic)MainAxis (Vector) : axe principal de l'esquisse.
  • Données (Dynamic)Parameter1 (Float) : paramètre sur la première arête.
  • Données (Dynamic)Parameter2 (Float) : paramètre sur la deuxième arête.
