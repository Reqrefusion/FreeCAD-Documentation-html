Description

Épaisseur d'un élément 2D est utilisé pour définir l'épaisseur d'éléments FEM 2D (coque et introduit dans la version 1.0 : contrainte/déformation plane), tous ou situés sur la surface choisie. Elle doit également être définie pour les modèles axisymétriques, même si la valeur de l'épaisseur n'est pas pertinente dans ce cas.

Maillage 2D (à gauche) et son expansion en éléments solides réalisée en interne par CalculiX sur la base de l'épaisseur spécifiée (à droite)

Utilisation

  1. Il existe plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Épaisseur d'un élément 2D.
    • Sélectionnez l'option Modèle → Géométrie de l'élément → Épaisseur d'un élément 2D du menu.
  2. Spécifiez l'épaisseur de la coque.
  3. Si vous le souhaitez, appuyez sur le bouton Ajouter dans le panneau des tâches, puis cliquez sur la face à laquelle vous souhaitez attribuer une épaisseur prescrite. Si la sélection des faces est vide, toutes les faces restantes (dont l'épaisseur n'est pas définie par d'autres éléments de géométrie 2D) seront automatiquement assignées. introduit dans la version 1.1 : pour supprimer des faces de la sélection, appuyez sur le bouton Supprimer.

Limitations

  • Actuellement, les analyses combinant des éléments coque avec des éléments solides ou des éléments arrêtes (poutre) ne sont pas prises en charge.

Propriétés

  • DonnéesThickness : spécifie l'épaisseur des éléments 2D.
  • DonnéesOffset : spécifie le décalage de la surface médiane de la coque par rapport à la géométrie maillée. L'unité est l'épaisseur de la coque. Les valeurs positives indiquent un décalage dans la direction normale positive, les valeurs négatives indiquent un décalage dans la direction opposée. Zéro signifie qu'il n'y a pas de décalage (la surface médiane est située sur la surface géométrique maillée). Par exemple, 0,5 signifie que la surface géométrique maillée sera la surface supérieure de la coque expansée, tandis que -0,5 signifie qu'elle sera la surface inférieure. introduit dans la version 1.1

Remarques

  • Pour afficher les résultats du solveur CalculiX sur le maillage développé à l'épaisseur prescrite, la propriété Beam Shell Result Output 3D du solveur CalculiX doit être mise à True. Pour afficher uniquement les épaisseurs définies avant d'effectuer les calculs réels, on peut effectuer une analyse de type vérification avec cette propriété réglée à True.
  • Cette fonction utilise le jeu de paramètres **SHELL SECTION de CalculiX pour les coques et le jeu de paramètres *SOLID SECTION pour les éléments de contrainte/déformation planes et axisymétriques.
