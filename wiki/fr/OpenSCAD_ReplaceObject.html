Description

Remplace un objet dans la vue en arborescence.

Utilisation

  1. Sélectionnez dans l'ordre correct, l'ancien, le nouveau et l'objet parent dans la vue en arborescence.
  2. Cliquez sur ou choisissez OpenSCAD → Remplacer un objet dans le menu principal.

Remarques

Vous pouvez utiliser cette fonction par exemple pour remplacer un élément dans un objet fusion/coupe/commun/composé.
