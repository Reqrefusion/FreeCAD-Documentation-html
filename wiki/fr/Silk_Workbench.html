L'icône de l'atelier externe Silk

Description

Contents

  • 1 Description
  • 2 Installation
  • 3 Références

L'atelier Silk est un atelier externe qui crée des surfaces NURBS dans FreeCAD.

Installation

L'installation recommandée se fait par le gestionnaire des extensions via Outils → Gestionnaire des extensions.
