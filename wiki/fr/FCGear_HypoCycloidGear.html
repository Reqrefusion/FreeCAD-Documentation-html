Description

La commande FCGear Engrenage hypocycloïde crée deux disques à cames cannelés et un ensemble de galets qui restent en contact avec la surface extérieure des disques pendant le mouvement.

A gauche : engrenage hypocycloïde. A droite : engrenage et transparence avec un engrenage inversé et un jeu de galets

Veuillez fournir une brève description de ce qui peut être réalisé en utilisant un tel train d'engrenages.

Utilisation

  1. Passez à l' atelier Gear.
  2. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le le bouton HypoCycloid Gear dans la barre d'outils.
    • Sélectionnez Gear → HypoCycloid Gear du menu.
  3. Modifiez le paramètre de l'engrenage en fonction des conditions requises (voir Propriétés).

Propriétés

Voir aussi : Éditeur de propriétés

Un objet FCGear HypoCycloidGear est dérivé d'un objet Part Feature et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Disks

  • Donnéesdisk-height (Length) : valeur par défaut 10 mm. Hauteur.
  • Donnéesshow_disk0 (Bool) : valeur par défaut true. Affiche le disque à came principal.
  • Donnéesshow_disk1 (Bool) : valeur par défaut true. Affiche un autre disque à came inversé par dessus.

Pins

  • Donnéescenter_pins (Bool) : valeur par défaut true. Centre l'axe Z des broches sur les disques générés.
  • Donnéespin-height (Length) : valeur par défaut 20 mm. Hauteur.
  • Donnéesshow_pins (Bool) : valeur par défaut true. Crée des broches à cet endroit.

gear_parameter

  • Donnéeseccentricity (Float) : valeur par défaut 1.5. Excentricité.
  • Donnéeshole_radius (Length) : valeur par défaut 30 mm. Rayon du trou central.
  • Donnéespin_circle_radius (Length) : valeur par défaut 66 mm. Rayon du cercle des billes (remplace le pas des dents).
  • Donnéespressure_angle_lim (Angle) : valeur par défaut 50°. Limite de l'angle de pression.
  • Donnéespressure_angle_offset (Float) : valeur par défaut 0.01. Décalage de l'angle de pression.
  • Donnéesroller_diameter (Float) : valeur par défaut 3,0. Diamètre du rouleau.
  • Donnéessegment_count (Integer) : valeur par défaut 42. Nombre de points utilisés pour l'interpolation spline.
  • Donnéesteeth_number (Integer) : valeur par défaut 42. Nombre de dents dans la came.

version

  • Donnéesversion (String) : valeur par défaut 1.3.0. Version de freecad.gears

Remarques

Le ou les engrenages par défaut s'affichent comme suit :

Pour afficher les disques à cames et le jeu de galets (broches) dans des couleurs différentes, nous avons besoin de trois objets HypocycloidGear identiques. Leur visibilité peut être modifiée :

  • Donnéesshow_disk0 (Bool) pour le disque à cames principal.
  • Donnéesshow_disk1 (Bool) pour le disque à cames inversé situé au-dessus.
  • Donnéesshow_pins (Bool) pour le jeu de broches.

A gauche : objets HypocycloidGear tels que créés. A droite : objets repositionnés pour obtenir une meilleure vue sur chaque objet
