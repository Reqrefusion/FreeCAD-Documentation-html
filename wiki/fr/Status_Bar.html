Introduction

La barre d'état est un ruban qui apparaît au bas de l'interface de FreeCAD. Elle peut afficher plusieurs informations et plusieurs widgets. Les ateliers BIM et Draft ajoutent leurs propres widgets à la barre d'état. Il est également possible de faire glisser une barre d'outils dans la barre d'état.

Informations et widgets

Informations d'utilisation

Lorsque le pointeur se trouve sur un bouton ou un élément de menu, les informations d'utilisation de cet outil s'affichent dans la barre d'état. Ces informations sont généralement identiques à celles de l'info-bulle de l'outil.

Informations présélectionnées

Lorsque le pointeur se trouve sur un objet dans la vue 3D, la barre d'état affiche l'élément présélectionné de l'objet ainsi que les coordonnées 3D actuelles. Ces informations sont mises à jour lorsque le pointeur se déplace sur l'objet. La mise à jour s'arrête lorsque le pointeur se trouve sur un espace vide dans la vue 3D.

Suggestions de saisie

introduit dans la version 1.1

Pour la plupart des outils de Sketcher et outils de création de Draft, des suggestions d'entrée dynamiques s'affichent dans la barre d'état.

Mesure rapide

introduit dans la version 1.0

La fonction Mesure rapide utilise la partie droite de la barre d'état pour afficher les mesures basées sur la sélection dans la vue 3D.

En fonction de la sélection, les mesures suivantes s'affichent :

  • Longueur de l'arête
  • Surface de la face
  • Distance entre les points, les arêtes ou les faces
  • Angle entre les arêtes
  • Rayon des arêtes circulaires ou des faces cylindriques
  • introduit dans la version 1.1 : diamètre des cercles et des faces cylindriques
  • introduit dans la version 1.1 : distance entre les axes des éléments circulaires et les faces, les arêtes ou d'autres éléments circulaires

Zone de notification

A définir

Un clic droit sur cette zone ouvre un menu contextuel proposant les options suivantes :

  • Supprimer les notifications de l'utilisateur : à définir.
  • Tout supprimer : à définir.

Indicateur de navigation

L'indicateur de navigation affiche le mode en cours de navigation par la souris. Cliquez sur l'indicateur ouvre un menu avec les modes disponibles et un sous-menu Paramètres avec les options suivantes :

  • Style d'orbite : voir Éditeur de préférences.
  • Compact : si cette option est cochée, seule une icône de souris s'affiche dans le widget.
  • Infobulle : si cette option est cochée, le survol du widget affiche une info-bulle contenant des informations détaillées sur son utilisation.

Indicateur de zoom

La barre d'état fournit des informations sur le niveau de zoom en affichant la taille de la vue 3D en cours. Cliquer sur l'indicateur ouvre un menu permettant de modifier le système d'unités du document en coursou le système d'unités par défaut s'il n'y a pas de document actif.

Menu contextuel

Un clic droit sur la barre d'état ouvre un menu contextuel avec les options suivantes :

  • introduit dans la version 1.1 : Suggestions de saisie : case à cocher pour activer ou désactiver cette fonction.
  • introduit dans la version 1.1 : Mesure rapide : Idem.
  • Zone de notification : Idem.
  • Indicateur de navigation : Idem.

Si le pointeur se trouve sur le champ Informations présélectionnées ou le champ Mesure rapide, les options suivantes sont ajoutées :

  • Copier : copie les informations sélectionnées avec l'option suivante.
  • Tout sélectionner : sélectionne les informations en cours.

La zone de notification dispose de son propre menu contextuel.
