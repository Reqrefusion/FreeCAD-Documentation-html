Introduction

La barre d'état est un ruban qui apparaît au bas de l'interface de FreeCAD. Elle peut afficher plusieurs widgets. Les ateliers BIM et Draft ajoutent leurs propres widgets à la barre d'état. Il est également possible de faire glisser une barre d'outils dans la barre d'état.

Informations et widgets

Informations d'utilisation

Lorsque le pointeur se trouve sur un bouton ou un élément de menu, les informations d'utilisation de cet outil s'affichent dans la barre d'état. Ces informations sont généralement identiques à celles de l'info-bulle de l'outil.

Présélection

Lorsque le pointeur se trouve sur un objet dans la vue 3D, la barre d'état affiche l'élément présélectionné de l'objet ainsi que les coordonnées 3D actuelles. Ces informations sont mises à jour lorsque le pointeur se déplace sur l'objet. La mise à jour s'arrête lorsque le pointeur se trouve sur un espace vide dans la vue 3D.

Suggestions de saisie

introduit dans la version 1.1

Pour la plupart des outils de Sketcher, Draft (introduit dans la version 26.3) et BIM (introduit dans la version 26.3), des suggestions de saisie dynamiques s'affichent dans la barre d'état.

Mesure rapide

introduit dans la version 1.0

La fonction Mesure rapide utilise la partie droite de la barre d'état pour afficher les mesures basées sur la sélection dans la vue 3D.

En fonction de la sélection, les mesures suivantes s'affichent :

  • Longueur de l'arête
  • Surface de la face
  • Distance entre les points, les arêtes ou les faces
  • Angle entre les arêtes
  • Rayon des arêtes circulaires ou des faces cylindriques
  • Diamètre des cercles et des faces cylindriques introduit dans la version 1.1
  • Diamètre des faces circulaires introduit dans la version 26.3
  • Distance entre les axes des éléments circulaires et les faces, les arêtes ou d'autres éléments circulaires introduit dans la version 1.1

Notifications

A définir

Un clic droit sur ce widget ouvre un menu contextuel proposant les options suivantes :

  • Supprimer les notifications de l'utilisateur : à définir.
  • Tout supprimer : à définir.

Styles de navigation

Le widget « Styles de navigation » affiche le style en cours de navigation par la souris. Un clic sur ce widget ouvre un menu présentant les styles disponibles, ainsi qu'un sous-menu Paramètres proposant les options suivantes :

  • Style d'orbite : voir Éditeur de préférences.
  • Compact : si cette option est cochée, seule une icône de souris s'affiche dans le widget.
  • Infobulle : si cette option est cochée, le survol du widget affiche une info-bulle contenant des informations détaillées sur son utilisation.

Système d'unités

Ce widget fournit des informations sur le niveau de zoom en affichant la taille de la vue 3D en cours. Un clic sur le widget ouvre un menu permettant de modifier le système d'unités du document en coursou le système d'unités par défaut s'il n'y a pas de document actif.

Menu contextuel

Un clic droit sur la barre d'état ouvre un menu contextuel comportant des cases à cocher permettant d'activer ou de désactiver les widgets suivants :

  • introduit dans la version 26.3 : Présélection
  • introduit dans la version 26.3 : Indicateur de progression
  • introduit dans la version 1.1 : Conseils de saisie
  • introduit dans la version 1.1 : Mesure rapide
  • introduit dans la version 26.3 : Activer/désactiver les panneaux inférieurs
  • introduit dans la version 26.3 : Notifications
  • Styles de navigation
  • introduit dans la version 26.3 : Système d'unités

Si le curseur se trouve sur le widget Présélection ou sur le widget Mesure rapide, les options suivantes s'affichent :

  • Copier : copie les informations sélectionnées avec l'option suivante.
  • Tout sélectionner : sélectionne les informations en cours.

Le widget Notifications dispose de son propre menu contextuel.
