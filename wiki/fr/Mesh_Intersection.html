Description

La commande Intersection crée un nouvel objet maillé non paramétrique, un Mesh Feature, qui est l'intersection (partie commune) de deux objets maillés.

OpenSCAD doit être installé pour utiliser cette commande et le chemin d'accès à son exécutable doit être défini dans les Préférences d'OpenSCAD.

A gauche deux objets maillés, à droite l'objet maillé qui est l'intersection de ces objets

Utilisation

  1. Sélectionnez deux objets maillés qui se chevauchent partiellement.
  2. Il existe plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Créer une intersection
    • Sélectionnez l'option Maillages → Opération booléenne → Créer une intersection du menu.
