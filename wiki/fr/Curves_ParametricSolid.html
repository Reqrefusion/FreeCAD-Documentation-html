Description

L'outil Solide paramétrique crée un solide paramétrique à partir de faces sélectionnées.

Une forme en aile composée de 4 faces avant (à gauche) et d'un seul solide après l'application de cet outil (à droite)

Utilisation

  1. Sélectionnez quelques faces dans la vue 3D ou quelques objets dans la vue en arborescence.
  2. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Créer un solide paramétrique.
    • Sélectionnez l'option Divers → Créer un solide paramétrique du menu.
  3. Un objet Solid est créé, son statut et l'icône correspondante décrivent le résultat :
    • Solide : les éléments sélectionnés forment une coque fermée et sont reconnus comme un objet solide.
    • Shell : les éléments sélectionnés forment une coque ouverte.
    • Compound : ...
    • none : ...
  4. Il est possible de modifier les valeurs dans l'éditeur de propriétés pour ajouter d'autres éléments de surface ou pour afficher uniquement les arêtes ouvertes des formes surfaciques (voir Propriétés ci-dessous).

Remarques

  • Cet outil tente de construire un solide à partir de faces sélectionnées. Si ce n'est pas possible, il se rabat sur une coquille, puis sur un composite.
  • Si la forme créée est un solide, elle peut être utilisée comme n'importe quel autre solide dans d'autres ateliers (c'est-à-dire les ateliers Part ou PartDesign). Elle a un volume et un poids.

Propriétés

Données

Debug

  • DonnéesShowOpenEdges (Bool) : si la forme résultante n'est pas un solide, les arêtes ouvertes seront affichées.

Solid

  • DonnéesFaces (LinkSubList) : liste des faces pour générer le solide.
  • Données (ReadOnly)ShapeStatus (String) : état de la forme créée.
