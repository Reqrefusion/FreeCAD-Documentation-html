Description

Solide paramétrique génère un solide paramétrique à partir de faces sélectionnées. Cet outil fait partie des ateliers externes appelé Curves.

Ci-dessus: montre l'application avant (gauche) et après (droite) de l'outil Solide paramétrique

Utilisation

  1. Passer à l'atelier Curves (à installer à partir du gestionnaire des extensions si ce n'est pas déjà fait)
  2. Sélectionnez les faces pour créer un solide (dans la vue en arborescence ou dans la vue 3D).
    • Utilisez la touche Ctrl pour sélectionner plusieurs faces. Les faces doivent être fermées ensemble. L'image ci-dessus montre 4 faces qui constituaient la seule aile solide.
  3. Pour appeler la commande, effectuez l'une des opérations suivantes:
    • Appuyez sur le bouton
    • Utilisez l'entrée Misc. → Parametric solid dans le menu Curves
  4. La forme créée peut être utilisée comme solide dans un autre atelier ( atelier Part ou atelier PartDesign). Il a du volume et du poids.

Remarques

  • This tool will try to build a solid from selected faces. If that is not possible, it falls back to a shell, then to a compound.
  • If the created shape is a solid it can be used as any other solid in other workbenches (i.e. Part or PartDesign workbenches). It has volume and weight.

Propriétés

Data

Debug

  • DonnéesShowOpenEdges (Bool): If the output shape is not a solid, this will output the open edges.

Solid

  • DonnéesFaces (LinkSubList): List of faces to build the solid.
  • Données (ReadOnly)ShapeStatus (String): Status of the created shape.
