Description

L'outil Assembly Liaison pivot crée une liaison pivot (également connu sous le nom de liaison articulé, ou charnière en abrégé) permettant la rotation autour d'un seul axe entre deux pièces sélectionnées.

Utilisation

  1. Vous pouvez sélectionner deux entités géométriques de deux formes différentes. Les autres sélections seront rejetées.
  2. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Créer une liaison pivot
    • Sélectionnez l'option Assemblage → Créer une liaison pivot du menu.
    • Utilisez le raccourci clavier : R.
  3. La fenêtre de dialogue Liaison s'ouvre dans le panneau des tâches avec la liste des entités présélectionnées.
  4. Pour les étapes suivantes, voir Assemblage Liaison fixe.

Propriétés

Voir aussi : Éditeur de propriétés

Un objet Revolute est dérivé d'un objet App FeaturePython et hérite de toutes ses propriétés. Voir Assembly Liaison fixe pour des propriétés supplémentaires.
