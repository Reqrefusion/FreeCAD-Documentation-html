Description

La commande /fr|Cables Éditer est une version étendue de la commande Draft Éditer qui est compatible avec les objets WireFlex. Elle préserve toutes les propriétés des WireFlex (en particulier les points d'attache) lors de l'ajout ou de la suppression de sommets. L'édition peut être effectuée avec la propriété Fillet Radius non nulle et avec n'importe quelle valeur de la propriété Path Type. En mode Éditer, toutes les commandes d'édition de WireFlex peuvent être exécutées à l'aide du menu contextuel du nœud ou de l'arête : Ajoueter un sommet, Supprimer un sommet, Attacher un sommet, Supprimer la connexion d'un sommet.

La commande Éditer n'était pas disponible dans les versions 0.1.4 et antérieures de l'atelier Cables.

Utilisation

  1. Vous pouvez sélectionner un ou plusieurs objets WireFlex. Remarquez que même si plusieurs objets peuvent être en mode édition de câbles, les objets ne peuvent être édités qu'un à la fois.
  2. Il y a plusieurs façons de lancer la commande :
    • Si vous n'avez pas encore sélectionné d'objet : double-cliquez sur un objet dans la vue en arborescence. Cela ne fonctionne que pour les objets WireFlex.
    • Appuyez sur le bouton Edit.
    • Sélectionnez l'option Cable Wires → Edit du menu.
    • Utiliser le raccourci clavier : D puis E.
    • Pour un seul objet : cliquez avec le bouton droit de la souris dans la vue en arborescence ou la vue 3D et sélectionnez l'option Cable Wires → Edit dans le menu contextuel.

Les étapes et options suivantes sont les mêmes que dans Draft Éditer.

Remarques

  • N'utilisez pas la commande Draft Éditer pour éditer les objets WireFlex. Elle peut détruire le bon ordre des attachements des points et provoquer des erreurs et un comportement inattendu de ces objets.

Options

Les options de base sont les mêmes que dans Draft Éditer. Le menu contextuel pour le nœud ou l'arête est invoqué en utilisant la même méthode que dans Draft Éditer.

Voici le menu contextuel du nœud ou de l'arête spécifique à un WireFlex.

  1. Menu contextuel du nœud : Attach point. La commande Attacher un sommet est exécutée. Il faut d'abord sélectionner un sommet externe puis, en maintenant la touche CTRL enfoncée, utiliser le menu contextuel Attach point (par exemple, en maintenant la touche ALT enfoncée et en cliquant ou en double-cliquant sur le nœud). Cette option n'est visible que pour les points non attachés.
  2. Menu contextuel du nœud : Remove point attachment. La commande Supprimer la connexion d'un sommet est exécutée. Cette option n'est visible que pour les points attachés.
  3. Menu contextuel de l'arête : Make edge horizontal. Rend l'arête horizontale en déplaçant le sommet le plus proche du curseur de la souris.
  4. Menu contextuel de l'arête : Make edge vertical. Rend l'arête verticale en déplaçant le sommet le plus proche du curseur de la souris.
