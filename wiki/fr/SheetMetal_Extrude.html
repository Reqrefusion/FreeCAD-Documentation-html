SheetMetal Prolonger une face

Emplacement du menu
SheetMetal → Prolonger une face

Description

La commande SheetMetal Prolonger une face prolonge une tôle à partir d'une face de bord sélectionnée.

Elle crée une extension simple le long de la normale du bord sélectionné :

Si une esquisse de contour est ajoutée, elle crée une géométrie de connexion pour fermer un profil :

Trois profils avec des esquisses à ajouter → trois résultats

Utilisation

Extension simple

  1. Sélectionnez une ou plusieurs faces d'arêtes à étendre.
  2. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Prolonger une face.
    • Sélectionnez l'option SheetMetal → Prolonger une face du menu.
    • Cliquez avec le bouton droit de la souris dans l'arborescence ou la vue 3D et sélectionnez l'option SheetMetal → Prolonger une face dans le menu contextuel.
    • Utilisez le raccourci clavier : E.
  3. Un objet Extend sera créé avec une nouvelle extension de face extrudée à partir de chaque face sélectionnée et le panneau des tâches Propriétés de l'extension s'ouvrira (introduit dans la version 0.5.00).
    • Vous pouvez appuyer sur le bouton Sélectionner et ajouter ou supprimer des faces.
      • Appuyez sur le bouton Aperçu pour terminer la sélection et afficher les modifications.
    • Vous pouvez ajuster la Longueur, le Décalage A et le Décalage ».
    • Vous pouvez cocher la case Inverser pour créer une découpe au lieu d'une extension (introduit dans la version 0.7.58).
  4. Appuyez sur le bouton OK pour quitter la commande et fermer le panneau des tâches.
  5. Vous pouvez ajuster les paramètres dans l'éditeur de propriétés.

Panneau des tâches

Un panneau des tâches a été introduit dans la version 0.5.00

Double-cliquez sur un objet Extend existant dans l'arborescence pour ouvrir de nouveau le panneau des tâches et modifier les paramètres.

  • Sélectionner : modifie la quantité de faces dans la propriété Base Object.
  • Longueur : définit la propriété length.
  • Décalage A : définit la propriété gap 1.
  • Décalage B : définit la propriété gap 2.
  • Inverser : active/désactive la propriété reversed. introduit dans la version 0.7.58
  • Amélioration : active/désactive la propriété Refine.

Remarque : depuis la version 0.7.57, un panneau des tâches différent s'affiche si la propriété DonnéesSketch n'est pas vide. Voir SheetMetal Étendre par une esquisse pour plus d'informations.

Extension de connexion

Même si cette méthode est toujours possible, il est désormais recommandé d'utiliser la commande Étendre par une esquisse pour cette tâche (version version 0.7.57 et suivantes).

(Préparez une esquisse pour l'emboîtement des profils)
  1. Sélectionnez la face de l'arête à prolonger.
  2. Lancez la commande comme décrit ci-dessus.
  3. Appuyez sur le bouton OK pour terminer la commande et fermer le panneau des tâches.
  4. Dans l'éditeur de propriétés, appuyez sur le bouton … de la propriété DonnéesSketch.
  5. La fenêtre de dialogue Lien s'ouvre.
    • Sélectionnez l'esquisse préparée dans la liste
    • Appuyez sur le bouton OK pour fermer la boîte de dialogue.
  6. Définissez la propriété DonnéesUse Subtraction sur true pour créer des découpes afin de faire de la place pour les extensions.
  7. Définissez la propriété DonnéesOffset pour ajuster le dégagement autour de l'extension.

Trois profils → position des esquisses → résultats sans découpes → résultats finaux

Remarques

  • Une esquisse peut contenir plus d'un contour.
Après avoir inséré une esquisse, l'un de ses contours au moins doit toucher une face opposée, sinon l'outil ne parviendra pas à créer une extension ou une découpe.
Un seul contour touchant une face opposée suffit à créer une géométrie d'extension à partir de tous les contours de l'esquisse.
  • Chaque découpe aura une forme cuboïde, quelle que soit la forme de l'esquisse du contour correspondant.
  • Les formes autres que les rectangles peuvent se comporter de manière un peu étrange et même si l'objet peut être déplié, le résultat ne sera pas celui escompté.

Trois esquisses et leurs extensions résultantes : plaque triangulaire séparée avec une découpe rectangulaire, cercle sans dégagement → le solide non plié est divisé à une position inattendue

  • Dans une opération d'extension, il est recommandé de laisser la propriété DonnéesRefine définie sur true (par défaut).
  • L'opération d'extension avec une esquisse liée peut échouer en raison de problèmes de coplanarité si la face du côté de l'esquisse et la face du côté opposé sont coplanaires, mais avec des orientations opposées. Un petit décalage peut aider dans un tel cas.

Propriétés

Voir aussi : Éditeur de propriétés

Un objet SheetMetal Extend est dérivé d'un objet Part Feature ou, s'il se trouve à l'intérieur d'un PartDesign Corps, à partir d'un objet PartDesign Feature, dont il hérite de toutes les propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Parameters

  • Donnéesbase Object (LinkSub) : objet de base. Lien vers la face plane à étendre.
  • Donnéesgap1 (Distance) : espace par rapport au côté gauche. Valeur par défaut : 0,00 mm.
  • Donnéesgap2 (Distance) : espace par rapport au côté droit. Valeur par défaut : 0,00 mm.
  • Donnéeslength (Length) : longueur de la paroi. Valeur par défaut : 10,00 mm.

Parameters Ext.

  • DonnéesOffset (Distance) : décaler la soustraction. Valeur par défaut : 20,00 µm.
  • DonnéesRefine (Bool) : utiliser l'amélioration. Valeur par défaut : true.
  • DonnéesSketch (Link) : esquisse de la paroi.
  • DonnéesUse Subtraction (Bool) : utiliser la soustraction. Valeur par défaut : false.
