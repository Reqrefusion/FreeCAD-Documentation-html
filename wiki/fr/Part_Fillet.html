Description

La commande Part Congé crée des congés (arrondis) sur les arêtes sélectionnées d'une forme. Une boîte de dialogue permet de choisir la forme et les arêtes sur lesquelles travailler.

Plaque aux coins recevant des congés

Utilisation

  1. Vous pouvez sélectionner une ou plusieurs arêtes dans la vue 3D.
  2. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Créer un congé.
    • Sélectionnez l'option Part → Créer un congé du menu.
  3. Le panneau des tâches Paramètres des congés s'ouvre.
    • Vous pouvez (re-)sélectionner un élément dans la liste Forme sélectionnée dans la section Forme ou dans la vue 3D.
    • Modifiez les paramètres du congé.
  4. Appuyez sur OK pour fermer le panneau des tâches et appliquer le congé.

Congé de PartDesign versus Part

Il existe une autre commande de congé dans l' atelier PartDesign. Notez que leur fonctionnement est assez différent. Consultez la page de référence PartDesign Congé pour plus de détails sur leurs différences.

Remarques pour l'application de Part Congé

Part Congé peut ne rien faire si le résultat touche ou traverse l'arrête adjacente suivante. Par conséquent, si vous n'obtenez pas le résultat escompté, essayez avec une valeur de « Rayon » plus petite. Il en est de même pour Part Chanfrein.

La commande Congé échoue parfois lorsqu'elle tente de faire des congés sur des formes complexes. Une cause fréquente de cet échec est que la forme recevant le congé n'est géométriquement pas correcte. Cela peut être le résultat de lignes/plans, etc. qui n'ont pas été supprimés après les précédentes opérations utilisées pour construire la forme (par exemple, coupe/intersection/union). Un certain nombre d'étapes peuvent être utilisées pour minimiser les problèmes :

  • Dans la mesure du possible, laissez le congé d'une pièce jusqu'à ce que la pièce soit complètement générée. Cela minimisera l'interaction des congés avec les opérations booléennes suivantes ;
  • Utilisez Part → Vérifier la géométrie pour vérifier les erreurs éventuelles dans la géométrie de la forme et les corriger ;
  • Utilisez Part → Copier → Affiner la forme pour supprimer tout artefact introduit par les opérations booléennes précédentes avant le congé (et dans certains cas entre les opérations de congé en séquence) ;
  • Envisagez d'utiliser Édition → Préférences → Part/PartDesign pour activer la vérification et l'affinage automatiques du modèle après les opérations booléennes et basées sur des esquisses (les performances peuvent être affectées si ces options restent activées).
