Description

La commande Fasteners Restaurer permet de restaurer les objets de fixation enregistrés sans l'atelier Fasteners.

Tree View showing fasteners that need to be restored

Usage

  1. Select the Fasteners → Restore fasteners option from the menu.
  2. All fasteners in the document are checked and restored if required.

Notes

  • The fasteners should still have all their properties. Fasteners that have been simplified with Simplify shape cannot be restored.
