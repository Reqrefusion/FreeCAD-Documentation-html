Description

Supprimer la connexion d'un sommet supprime une connexion de sommet existante d'un WireFlex d'un sommet ou d'un objet externe. Les connexions des sommets sont conservées dans des propriétés spéciales de WireFlex, pour plus de détails voir les propriétés de WireFlex.

It can be used interchangeably with the "Remove point attachment" command from Cables Edit, see the WireFlex Description.

Une fois la connexion supprimée, le sommet de la polyligne peut être déplacé librement dans l'espace 3D à l'aide de Draft Éditer.

Utilisation

  1. Sélectionnez un sommet d'un objet WireFlex existant dans la vue 3D.
  2. Supprimer la connexion du sommet sélectionné par l'une des méthodes suivantes :
    • Appuyez sur le bouton Remove Vertex Attachment.
    • Sélectionnez l'option Cable Wires → Remove Vertex Attachment du menu.
    • Cliquez avec le bouton droit de la souris dans la vue en arborescence ou la vue 3D et sélectionnez l'option Cable Wires → Remove Vertex attachment dans le menu contextuel.

Remarques

  • A partir de la version 0.2.0, cette commande peut également être exécutée en utilisant le menu contextuel du nœud dans la commande Cables Editer.
  • Voir la description du WireFlex et l'utilisation du WireFlex pour plus de détails sur l'utilisation de WireFlex.
