Description

Crée une condition de pression initiale pour une analyse d'écoulement de fluide.

Utilisation

  1. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Condition de pression initiale.
    • Sélectionnez l'option Modèle → Conditions limites de fluide → Condition de pression initiale du menu.
  2. Saisissez une valeur de pression initiale.
  3. Appuyez sur le bouton Ajouter. Pour une analyse 3D, sélectionnez un "solide" (corps) de votre modèle. Pour une analyse 2D, sélectionnez une face. introduit dans la version 1.1 : pour supprimer des objets de la sélection, appuyez sur le bouton Supprimer.

Remarques

Pour des analyses simples, il n'est pas nécessaire de spécifier la pression initiale, mais dans ces cas également, il est recommandé de le faire en tant que meilleure pratique.
