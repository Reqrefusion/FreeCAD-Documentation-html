Description

Relier des courbes relie des arêtes sélectionnées en courbes B-spline. Cet outil fait partie des ateliers externes appelé Curves.

Utilisation

  1. Passer à l'atelier Curves (à installer à partir du gestionnaire des extensions si ce n'est pas déjà fait)
  2. Sélectionnez une ou plusieurs arêtes dans la vue 3D. (utilisez la touche CTRL pour plusieurs sélections)
  3. Les bords doivent être ajoutés dans un ordre consécutif. Et être sélectionné à partir de l'esquisse, du bord de n'importe quelle forme.
  4. Pour appeler la commande, effectuez l'une des opérations suivantes:
    • Appuyez sur le bouton
    • Utilisez la Curves → Join Curves
  5. Tous les bords sont fusionnés en un seul Bspline.

Remarques

  • Edges of a shape, or edges selected from a sketch are accepted and must be added in a consecutive order.
  • The result is a single continuous B-spline.

Propriétés

Data

InputSources

  • DonnéesBase (Link): Join all the edges of this base object.
  • DonnéesEdges (LinkSubList): List of edges to join.

Join

  • DonnéesCornerBreak (Bool): Break on sharp corners.
  • DonnéesForceClosed (Bool): Force closed curve.
  • DonnéesForceContact (Bool): Force connection of edges.
  • DonnéesReverse (Bool): Reverse the output curve.
  • DonnéesStartOffset (Integer): Set the start point of closed curve.
  • DonnéesTolerance (Float): Tolerance.

ShapeApproximation

  • DonnéesActive (Bool): Use approximation.
  • Données (Hidden)ApproxTolerance (Float): Approximation tolerance.
  • Données (Hidden)Continuity (Enumeration): Desired continuity of the curve. (C0, C1, G1, C2, G2, C3 (default), CN)
  • Données (Hidden)DegreeMax (Integer): Maximum degree of the curve.
  • Données (Hidden)DegreeMin (Integer): Minimum degree of the curve.
  • DonnéesExtensionProxy (PythonObject): Proxy object of the approximation extension.
  • Données (Hidden)Parametrization (Enumeration): Parametrization type.
    • ChordLength: ...
    • Centripetal: ...
    • Uniform: ...
  • Données (Hidden)Samples (Integer): Number of samples.
