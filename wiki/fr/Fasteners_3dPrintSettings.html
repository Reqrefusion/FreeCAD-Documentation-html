Description

La commande Fasteners Paramètres impression 3D permet de modifier les paramètres d'impression 3D. Ces paramètres servent à compenser le retrait et d'autres problèmes pouvant affecter les fixations imprimées en 3D.

Les paramètres se trouvent également dans les préférences : dans le menu, sélectionnez Édition → Preferences puis Fasteners.

Utilisation

  1. Il existe plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Modifier les paramètres d'impression 3D.
    • Sélectionnez l'option Fasteners → Modifier les paramètres d'impression 3D du menu.
  2. Le panneau de tâches Paramètres d'impression 3D s'ouvre.
  3. Vous pouvez éventuellement modifier le paramètre Génération de filetage et le régler sur Compatible avec l'imprimante 3D.
  4. Vous pouvez également modifier l'Échelle des vis.
  5. Vous pouvez également modifier l'Échelle des écrous.
  6. Vous pouvez également appuyer sur le bouton Enregistrer et mettre à jour pour enregistrer les paramètres et mettre à jour les fixations filetées dans le document en cours.
  7. Appuyez sur le bouton OK pour enregistrer les paramètres et fermer le panneau de tâches.
