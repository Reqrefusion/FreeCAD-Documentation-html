Description

La commande Draft Groupe nommé crée un Std Groupe nommé et ajoute des objets sélectionnés vers ce groupe.

Utilisation

  1. Vous pouvez sélectionner un ou plusieurs objets.
  2. Il existe plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Créer un nouveau groupe nommé.
    • Sélectionnez l'option Utilitaires → Créer un nouveau groupe nommé du menu, ou dans le menu contextuel de la vue en arborescence ou de la vue 3D.
  3. Le fenêtre de dialogue Créer un nouveau groupe s'ouvre. introduit dans la version 1.1
  4. Saisissez Nom du groupe.
  5. Appuyez sur le bouton OK.

Remarques

  • Pour plus d'informations sur l'organisation de votre modèle, voir Structure du document et Arch Tutoriel.
