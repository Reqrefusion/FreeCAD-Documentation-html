Description

Exporte le projet de tracé de rayons sélectionné dans un fichier.

Utilisation

  1. Sélectionnez un projet de lancer de rayons dans la Vue en arborescence.
  2. Cliquez sur ou choisir Raytracing → Exporter le projet... dans le menu supérieur.
