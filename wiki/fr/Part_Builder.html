Description

La commande Part Générateur de formes crée des formes plus complexes à partir de diverses primitives géométriques.

Utilisation

  1. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Générer une forme.
    • Sélectionnez l'option Partie → Générer une forme du menu.
  2. Le panneau des tâches Créer une forme s'ouvre.
  3. Sélectionnez le mode de création de forme et appuyez éventuellement sur le bouton Sélectionner par une boîte... pour utiliser la sélection par boîte au lieu de sélectionner les entités géométriques une par une :
    • Arête à partir de sommets
      1. Sélectionnez deux sommets dans la vue 3D.
    • Polyligne à partir d'arêtes
      1. Sélectionnez un ensemble d'arêtes reliées dans la vue 3D.
      2. Vous pouvez cocher Plan pour une polyligne plane.
    • Face à partir de sommets
      1. Sélectionnez les sommets délimitant la face dans la vue 3D.
      2. Vous pouvez cocher Plan pour une face plane.
    • Face à partir d'arêtes
      1. Sélectionnez les arêtes entourant la face dans la vue 3D.
      2. Vous pouvez cocher Plan pour une face plane.
    • Coque à partir de faces
      1. Sélectionnez les faces dans la vue 3D.
      2. Vous pouvez cocher Affiner la forme pour affiner la forme.
      3. Vous pouvez cocher Toutes les faces pour inclure toutes les faces incluses dans la coque.
    • Solide à partir d'une coque
      1. Sélectionnez une coque dans la vue 3D.
      2. Vous pouvez cocher Affiner la forme pour affiner la forme.
  4. Appuyez sur le bouton Créer.
  5. L'objet est créé.
  6. Vous pouvez créer des objets supplémentaires.
  7. Appuyez sur le bouton Fermer pour fermer le panneau des tâches et terminer la commande.

Remarques

Un flux de travail possible pour créer un solide à partir d'une image filaire pourrait être le suivant :

  • Dessinez un modèle filaire de votre forme à l'aide des outils de l' atelier Draft (par exemple uune ligne et une polyligne).
  • Créez toutes les faces avec "face à partir d'arêtes".
  • Créez une "coque à partir de faces".
  • Créez un "solide à partir d'une coque".
