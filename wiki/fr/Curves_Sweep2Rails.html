Description

L'outil Curves Deux courbes par balayage ...

Utilisation

  1. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Sweep2Rails.
    • Sélectionnez l'option Surfaces → Sweep2Rails du menu.
  2. Un objet Sweep_2_rails est créé.

Properties

See also: Property editor.

A Sweep_2_rails object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

Base

  • DonnéesBirail (Link): Birail object.
  • DonnéesBlending (Enumeration): Blending method.
    • Average: ...
    • Blend (default): ...
    • Rail1: ...
    • Rail2: ...
  • DonnéesExtend (Bool): Extend to rail limits.
  • DonnéesParametrization (Float): Parametrization of interpolating curves.
  • DonnéesPoints (VectorList): Points.
  • DonnéesProfileSamples (Integer): Profile Samples.
  • DonnéesProfiles (LinkList): List of profiles.
  • DonnéesRailSamples (Integer): Profile Samples.
