Utilisation

Choix

  • Compound I Type I [1 + 1 = 1] : créez un composite unique de tous les objets sélectionnés sans historique.
  • Compound II Type II [1 + 1 = A (1 + 1)] : créez un composite de tous les objets sélectionnés avec l'historique de tous les objets. Idem "Menu → Part → Créer un composite".

<span id="_Option_color">

Couleur de l'option

Si est vérifiée, la couleur de l'objet à travailler sont colorées (bord, sommet)

  • Color  : Donne une couleur à l'objet. (Rouge par défaut 255, 0, 0)

Outils

  • LineEdit : affiche (Index de police/nombre de police) le chemin et le nom de la police.
  • Convert Text (A) : convertir le texte en une chaîne de forme (La hauteur du texte converti est respectée mais le résultat visuel peut ne pas être respecté, voir la propriété Vue combinée pour confirmer). (A) est la hauteur de la valeur automatique du texte.
    • 0,00 Auto  : Si spinbox est égal à 0.0, la hauteur de la VALEUR du texte est respectée. Si différent de 0.0, le Convert Text (A) se change en mode manuel Convert Text (M).
  • Convert Wire (A) : Cette commande convertit le fil en une ligne avec des coordonnées. (ex: un composite dégradé n'a pas de coordonnées, cette fonction crée une ligne avec les coordonnées comme ligne de dépouille et reproduit la polyligne DXF dans un objet dépouille sont détectées: ligne, arc, cercle, ellipse, BSplineCurve.
    • 0,00 Auto  : Donne une épaisseur de la polyligne. Si spinbox est égal à 0.0, la hauteur de la VALEUR du texte est respectée, si différent de 0.0, le Convert Wire (A) se change en mode manuel Convert Wire (M).
  • BezierCurve : Par défaut, le BezierCurve détecté est , s'il est coché, le BezierCurve est cubique et le bouton change Cubic BezierCurve
  • FlatLines : les objets créés sont des FlatLines.
  • Wireframe : Les objets créés sont Wireframe.
  • Points : Les objets créés sont des Points.
  • UpGrade : UpGrade
  • DownGrade : DownGrade

Pour Compound I et Convert Edge

Cette section fonctionne uniquement avec les outils Compound I, Convert Text (A) et Convert Wire (A)

  • None : Tous les objets d'origine restent tels quels.
  • Hidden original objest(s) : masque les objets originaux.
  • Delete original objest(s) : supprime les objets originaux.

Forcer sur une forme : Ligne

Si l'objet ligne, arc ou cercle n'est pas reconnu (comme forme dans un objet 3D de volume), cette section force la création d'une ligne, d'un arc ou d'un cercle.

aucune vérification n'est effectuée, cette section plateau permet de créer une forme 2D demandée (Draft) en fonction des données fournies

  • Lines : pour créer une ligne.
  • Arc : pour créer un arc.
  • Circle : pour créer un cercle
  • Force on : Line : Bouton Force

Commandes

  • Barre de progression
  • Reset : réinitialiser la macro
  • Quit : quitter la macro, bye
  • Help : afficher la page wiki dans le navigateur FreeCAD
