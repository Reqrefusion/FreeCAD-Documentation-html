FreeCAD 26.3 est en cours de développement, il n'y a pas encore de date de sortie prévue.

Il manque des fonctions ? Signalez-les dans le fil de discussion du forum Release notes for v26.3.
Consultez l'aide de FreeCAD pour savoir comment contribuer à FreeCAD.

Toutes les images de cette page doivent utiliser le suffixe _relnotes_26.3

FreeCAD 26.3 a été publié le JJ MM AA, téléchargez la depuis la page Téléchargement. Cette page liste toutes les nouvelles fonctions et les changements.

Les notes de versions plus anciennes de FreeCAD sont disponibles dans la liste des notes de versions.

Général

  • FreeCAD a adopté un nouveau calendrier de publication prévoyant trois versions stables par an et est passé à un système de numérotation des versions de type CalVer (AA.N), à compter de la version 26.3. FEP-0003
  • Il est désormais possible de modifier plusieurs documents simultanément, chacun avec ses propres tâches et ses propres historiques d'annulation/rétablissement, par exemple, en ouvrant deux esquisses en même temps (projet GSoC). Pull request #21978
  • Les fichiers de sauvegarde .FCBak peuvent désormais être ouverts directement depuis la boîte de dialogue Fichier → Ouvrir sans avoir à les renommer au préalable en .FCStd. L'enregistrement d'un tel fichier déclenche Enregistrer sous… pour éviter d'écraser accidentellement la sauvegarde. Pull request #28454
  • Le paramètre de format de date de sauvegarde .FCBak vérifie désormais la validité des caractères utilisés dans les noms de fichiers et remplace les caractères non valides dans l'horodatage lors de la création des sauvegardes. Pull request #25985
  • L'option « Utilisation des fonctions Annuler/Rétablir dans les documents » a été supprimée car elle était redondante. Pull request #30616

Noyau et API

Noyau

Une nouvelle commande permettant d'ajouter une étiquette d'annotation au dernier point sélectionné dans la vue 3D a été ajoutée.

Pull request #23673

Une nouvelle commande Propriétés physiques a été ajoutée pour calculer le volume, la masse, la surface, le centre de gravité, le centre de volume et l'inertie des pièces et des assemblages, avec prise en charge des repères de référence personnalisés et des calculs tenant compte des matériaux.

Pull request #27347

Les référentiels de base utilisent désormais la propriété Show Placement, ce qui rend les décalages des ancrages (sans rotation) dans les coordonnées locales plus intuitifs. Pull request #29176

Cliquez sur l'image si l'animation ne démarre pas. 		L'outil Transformer prend désormais en charge un mode système de coordonnées personnalisé : les valeurs de translation et de rotation sont exprimées par rapport à n'importe quel objet présent dans le document (corps, formes, systèmes de coordonnées locaux, conteneurs de Part, etc.), tandis que la position et l'orientation du manipulateur restent inchangées. Les rotations sont appliquées autour de l’origine en cours du manipulateur (origine de l’objet ou position personnalisée du curseur) en utilisant les axes du système de coordonnées personnalisé. En option, les axes du manipulateur peuvent être alignés visuellement sur le système de coordonnées personnalisé afin de faire glisser l’objet de manière interactive dans ces directions. Cela permet de déplacer ou de faire pivoter un objet au niveau ou autour du manipulateur en cours par rapport au système de coordonnées d’un autre objet.

Pull request #30227
