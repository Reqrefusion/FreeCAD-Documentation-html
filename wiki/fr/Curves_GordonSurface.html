Description

L'outil Curves Surface de Gordon permet d'étendre une surface sur une grille de courbes guides et de courbes de profil.

Pour une surface Gordon, il faut au moins un cadre composé de 2 courbes de guidage et de 2 courbes de profil. Des courbes supplémentaires entre les deux permettent un contrôle plus détaillé de la courbure.

Une grille de 3 guides (jaune) et 4 profils (violet) → Une surface Gordon

Utilisation

  1. Sélectionnez les lignes qui définissent la grille de la surface dans l'ordre approprié :
    • Sélectionnez les courbes de guidage (première direction) d'une arête à l'autre en passant par les courbes intermédiaires.
    • Sélectionnez les courbes de profil (deuxième direction) à partir d'une arête via les courbes intermédiaires jusqu'à l'autre arête.
    • Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Gordon surface.
    • Sélectionnez l'option Surfaces → Gordon surface du menu.
  2. Un objet Gordon est créé.
  3. Vous pouvez modifier les valeurs dans l'éditeur de propriétés pour choisir un type différent de sortie et ajuster les propriétés correspondantes (voir Propriétés ci-dessous).

Remarques

  • Changez la propriété DonnéesOutput en Wireframe pour créer une grille de Gordon. Elle utilise les mêmes courbes de trame mais peut avoir un nombre différent de courbes entre les deux.
    Les propriétés DonnéesSamples U et DonnéesSamples V déterminent la densité de la grille dans les directions U et V respectivement.
    Les échantillons sont égaux au nombre de segments entre les courbes du cadre, 3 et 5 dans cet exemple :

Surface de Gordon → Grille de Gordon, le même objet avec la propriété Output réglée sur Wireframe

  • Les courbes d'entrée de chaque groupe (guides et profils) doivent toucher toutes les courbes de l'autre groupe. En d'autres termes, elles doivent former une grille ou un filet, comme illustré ici :

  • In general, the surface normal of the resulting Gordon surface will be determined by the direction of the profiles.
    In this example, the surface on the left, the profiles were drawn from +Y to -Y and the resulting surface normal is +Z
    and on the right, the profiles are drawn from -Y to +Y, resulting in the surface normal oriented -Z.

  • Part Extrusion peut être utilisé pour créer un solide à partir de la surface.
  • PartDesign Protrusion peut également créer un solide à partir de la surface. Faire glisser la surface dans un corps crée une fonction de base qui peut ensuite être protusée.

Propriétés

See also: Property editor.

A Gordon object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

Base

  • DonnéesOutput (Enumeration): Defines the output type:
    • Surface (default): The result is a single Surface shape.
    • Wireframe: The result is a grid of guide and profile curves.

Gordon

  • DonnéesMax Ctrl Pts (Integer): Max Number of control points
  • DonnéesSources (LinkList): Curve network. User selected lines that are used to create the Gordon surface.
  • Données (Hidden)Tol2D (Float): Parametric tolerance
  • DonnéesTol3D (Float): 3D tolerance

Surface

  • DonnéesFlip Normal (Bool): Flip surface normal

Wireframe

  • DonnéesPlacement : peut être utilisé pour ajuster le placement de la surface Gordon résultante. Voir : Placement
    • Remarque : les propriétés Placement n'ajuste pas le placement des courbes/lignes utilisées pour créer la surface, uniquement la surface.
  • DonnéesLabel : étiquette (nom) spécifiée par l'utilisateur pour la surface, par défaut : Gordon.
  • DonnéesOutput : définit le type de sortie, par défaut : Surface, options : Surface, Wireframe.
  • DonnéesGordon>Max Ctrl Pts : par défaut : 80
  • DonnéesGordon>Sources : lignes sélectionnées par l'utilisateur qui sont utilisées pour créer la surface de Gordon.
  • DonnéesGordon>Tol3D : tolérance 3D, par défaut : 0.01.
  • DonnéesWireframe>Samples U : nombre d'échantillons dans la direction U, par défaut : 16.
    • Cette valeur est utilisée pour déterminer la densité du maillage lorsque la propriété Output est définie sur Wireframe.
  • DonnéesWireframe>Samples V : nombre d'échantillons dans la direction V, par défaut : 16.
    • Cette valeur est utilisée pour déterminer la densité du maillage lorsque la propriété Output est définie sur Wireframe.
    • Surface de Gordon en mode fil de fer, U=16, V=16
