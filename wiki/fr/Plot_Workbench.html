Introduction

FreeCAD est capable d'effectuer des tracés en utilisant la bibliothèque matplotlib de Python. Un module est fourni à cette fin, en tant qu'extension externe dans la version 0.19 et en tant que composant du core à partir de la version 0.20. Les anciennes versions de FreeCAD ne sont pas couvertes par cette documentation.

Les graphes produits offrent les outils standard de matplotlib pour l'édition et la sauvegarde. En outre, l' atelier Plot est fourni en tant qu'extension offrant des outils plus complets pour modifier un graphique et le sauvegarder. L'extension peut être installée avec le gestionnaire des extensions.

Module

Le module peut être lancé dans une console Python ou dans une macro. La première chose que vous devez faire est d'importer le module. Dans FreeCAD 0.19 vous devez d'abord installer l' atelier Plot à l'aide du gestionnaire d'extensions et ensuite vous pouvez importer Plot en tapant : 

from freecad.plot import Plot

Depuis FreeCAD 0.20, le module Plot est déjà inclus dans le programme, vous n'avez donc pas besoin d'installer d'extension mais simplement de taper : 

from FreeCAD.Plot import Plot

Après cela, vous pouvez tracer une ligne droite de (0,0) à (1,2) en tapant tout simplement : 

Plot.plot([0, 1], [0, 2])

Vous pouvez trouver des exemples plus complexes dans le Tutoriel de base et le Tutoriel graphique à plusieurs axes.

Outils de l'atelier

Si vous décidez d'installer l' atelier Plot en utilisant le gestionnaire des extensions, vous disposerez des outils suivants pour gérer les tracés créés avec l'atelier :

  • Enregistrer : enregistrer sous plusieurs formats. Vous pouvez sélectionner la taille de sortie et la résolution.
  • Axes : ajouter, effacer ou éditer les axes du graphique.
  • Séries : éditer les séries titres et style.
  • Grille : afficher/Cacher la grille.
  • Légende : afficher/cacher les légendes.
  • Étiquettes : éditer les étiquettes.
  • Positions : définir les positions d'éléments.

Script

Puisque l'atelier Plot est une couche sur matplotlib, vous êtes libre d'utiliser toutes les commandes de la librairie matplotlib. Voir exemples de scripts pour des exemples.
