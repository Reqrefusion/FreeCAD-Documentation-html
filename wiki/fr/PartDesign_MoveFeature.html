Description

Déplacer l'objet, comme l'indique le nom de cette commande dans le menu contextuel, permet de déplacer des objets d'un corps vers un autre. L'objet déplacé est placé juste en dessous de la fonction résultante de l'autre corps. Les esquisses, la géométrie de référence et les fonctions sont prises en charge.

Utilisation

  1. Dans la vue en arborescence du modèle, cliquez avec le bouton droit sur l'objet à déplacer.
  2. Sélectionnez Déplacer l'objet vers... de la liste du menu contextuel.
  3. Dans la fenêtre de dialogue Sélectionner un corps, sélectionnez le corps pour déplacer l'objet dans le menu déroulant.
  4. Appuyez sur OK.
