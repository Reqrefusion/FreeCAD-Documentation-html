Description

The Std DockOverlayToggleBottom command provides a shortcut to quickly show or hide the entire group of panels that are currently in the overlay system on the bottom side of the screen.

This command does not move panels into or out of the overlay system; it only toggles the visibility of the bottom overlay container itself. This is a convenience feature for users who want to temporarily hide all panels on one side of the view to maximize their workspace without having to exit the overlay mode completely.

Usage

  1. The overlay mode must be active, and there must be at least one panel in the bottom overlay area.
  2. There are several ways to invoke the command:
    • Select the View → Dock panel overlay → Toggle bottom option from the menu.
    • Use the keyboard shortcut: Ctrl+Down.
    • Use the overlay toggle bottom button () from a docked window's title bar.
