Description

La fonction FEM Tableau crée un tableau basé sur les données d'un pipeline ou d'un filtre sélectionné.

Exemples de tableau

Utilisation

  1. Il y a plusieurs façons de lancer la commande :
    • Appuyez sur le bouton Créer un tableau.
    • Sélectionnez l'option Résultats → Visualisations de données → Créer un tableau du menu.
  2. Le panneau des tâches s'ouvre.
  3. Appuyez sur le bouton Ajouter des données de, sélectionnez un pipeline ou un filtre comme source de données et l'un des deux types de tableaux dans la liste déroulante :
    • ajoute un champ de données : liste les valeurs d'une variable de sortie sélectionnée pour tous les nœuds. Si la case Un champ pour chaque trame est activée, il y a une colonne séparée pour chaque trame.
    • ajoute un index sur les trames : liste les valeurs d'une variable de sortie sélectionnée pour un nœud avec un index (numéro) spécifié dans chaque trame.
  4. Configurez le tableau :
    • Appuyez sur le bouton du milieu qui affiche par défaut Index pour définir la source de données de sortie et l'index.
    • Appuyez sur le 3ème bouton en partant de la gauche qui affiche par défaut le texte par défaut pour définir le nom du tableau.
    • Appuyez sur le bouton le plus à droite avec le symbole X pour supprimer l'objet Table.
  5. Appuyez sur le bouton Afficher le tableau pour afficher le tableau.
  6. Appuyez sur le bouton OK pour terminer et fermer le panneau des tâches.
