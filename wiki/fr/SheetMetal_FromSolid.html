Description

La commande SheetMetal Convertir en tôle permet de convertir un solide ou une coque en un objet en tôle (introduite dans la version 0.8.0).

Cuboid with top face and edges selected → Resulting SolidToSheet object

Utilisation

  1. Sélectionnez un objet de base (un solide ou une coque).
  2. Il existe plusieurs façons de lancer la commande :
    • Cliquez sur le bouton Convertir en tôle.
    • Sélectionnez l'option Sheet Metal → Convertir en tôle du menu.
    • Cliquez avec le bouton droit de la souris dans l'arborescence ou la vue 3D et sélectionnez l'option Sheet Metal → Convertir en tôle dans le menu contextuel.
    • Utilisez le raccourci clavier : C puis S.
  3. Un objet SolidToSheet est créé et le panneau des tâches Convertir en tôle s'ouvre :
    La liste Faces à supprimer et arêtes de jonction est vide si l'objet de base a été sélectionné dans l'arborescence, ou elle contient les sous-éléments qui ont été utilisés pour sélectionner l'objet de base dans la vue 3D.
    • Ajustez les paramètres dans le panneau de tâches.
    • Vous pouvez appuyer sur le bouton Sélectionner pour modifier la liste Faces à supprimer et arêtes de jonction.
      (Un bouton Aperçu indique que la liste est en mode édition)
      • Vous pouvez sélectionner des faces/arêtes dans la vue 3D pour les ajouter à la liste ou les supprimer respectivement.
      • Vous pouvez également cliquer sur le bouton Tout effacer pour supprimer tous les éléments de la liste Faces à supprimer et arêtes de jonction.
      • Vous pouvez cocher des éléments de la liste et cliquer sur le bouton Effacer la sélection pour supprimer uniquement les éléments sélectionnés de la liste Faces à supprimer et arêtes de jonction.
      • Cliquez sur le bouton Aperçu pour confirmer et terminer la modification.
  4. Cliquez sur le bouton OK pour fermer le panneau des tâches et terminer la commande.
  5. Vous pouvez également ajuster les paramètres du profil dans l'éditeur de propriétés.

Panneau des tâches

Double-cliquez sur un objet SolidToSheet existant dans l'arborescence pour ouvrir de nouveau le panneau des tâches et modifier les paramètres.

  • Épaisseur : définit la propriété Thickness.
  • Rayon de courbure : définit la propriété Radius.
  • Inverser : active/désactive la propriété Invert.
  • Sélectionner : active le mode d'édition permettant d'ajouter la liste des faces à supprimer et/ou des arêtes de jonction dans la propriété base Object. Si on appuie dessus :
    • Aperçu : indique le mode d'édition. Appuyez pour quitter le mode d'édition (masqué jusqu'à ce que l'on appuie sur Sélectionner).
    • Tout effacer : appuyez pour effacer la liste (masqué jusqu'à ce que Sélectionner soit activé). Devient Effacer la sélection si au moins un élément de la liste est coché. Cela ne supprime que les éléments sélectionnés.

Propriétés

Voir aussi : Éditeur de propriétés

Un objet SheetMetal SolidToSheet est dérivé d'un objet Part Feature ou, s'il se trouve à l'intérieur d'un PartDesign Corps, à partir d'un objet PartDesign Feature, dont il hérite de toutes les propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Parameters

  • DonnéesInvert (Bool) : inverse le sens du pliage. Détermine si l'épaisseur des parois s'accroît vers l'intérieur ou vers l'extérieur de l'objet de base.
  • DonnéesRadius (Length) : rayon intérieur des plis ajoutés automatiquement.
  • DonnéesThickness (Length) : épaisseur des parois de la forme en tôle.
  • Donnéesbase Object (LinkSub) : objet de base. Lien vers le solide/la coque et ses sous-éléments sélectionnés.
