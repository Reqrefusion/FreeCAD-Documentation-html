Description

La fonction Supprimer un sommet supprime un sommet existant de l'objet WireFlex. Elle préserve l'affectation correcte des connexions existantes des sommets des câbles. Elle doit être utilisée à la place de la commande « Supprimer un point » de Draft Éditer, voir WireFlex Description.

Usage

  1. Select one mid vertex of an existing WireFlex object in the 3D view.
  2. Delete selected vertex of WireFlex by one of these methods:
    • Press the Delete Vertex button.
    • Select the Cable Wires → Delete Vertex option from the menu.
    • Right-click in the Tree view or the 3D view and select the Cable Wires → Delete Vertex option from the context menu.

Only mid vertexes can be deleted with this tool. First and last vertex (Vrtx_start and Vrtx_end) cannot be deleted. The position of remaining vertexes can be edited later with Draft Edit command or attached to other objects with Attach Vertex.
