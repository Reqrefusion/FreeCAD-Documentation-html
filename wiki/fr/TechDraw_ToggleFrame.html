Description

L'outil TechDraw Bascule des cadres active ou désactive l'affichage des cadres, étiquettes et sommets d'une vue.

Vue de la projection solide avec les cadres activés et désactivés

Utilisation

  1. S'il y a plusieurs feuilles de dessin dans le document : activez la feuille souhaitée en la sélectionnant dans la vue en arborescence.
  2. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Activer/désactiver les éléments d'édition d'une vue.
    • Sélectionnez l'option TechDraw → Vues de Techdraw → Activer/désactiver les éléments d'édition d'une vue du menu.
    • Si une feuille est affichée dans la zone de vue principale : cliquez avec le bouton droit de la souris sur la fenêtre de la feuille et sélectionnez l'option Activer/désactiver les éléments d'édition d'une vue dans le menu contextuel.
  3. S'il y a plusieurs feuilles de dessin dans le document et que vous n'avez pas encore activé une feuille, la fenêtre de dialogue Sélecteur de feuilles s'ouvre :
    1. Sélectionnez la feuille désirée.
    2. Appuyez sur le bouton OK.
  4. Les cadres d'affichage visibles disparaissent. Les cadres de vue invisibles apparaissent.
  5. Il est possible que des vues différentes soient dans des états d'affichage différents. Si cela se produit, lancez cet outil une ou deux fois pour resynchroniser les vues.

Contexte

Le cadre de vue en pointillé et les points de sommet sont juste donnés pour référence, ils ne font pas réellement partie du dessin donc, vous ne les verrez pas une fois que vous exportez la feuille au format PDF ou SVG.

La procédure suggérée est d'utiliser Activer/désactiver les éléments d'édition d'une vue pour désactiver le cadre entourant la vue ainsi que les points supplémentaires. Avec les points, utilisez les outils de mesure pour sélectionner les bonnes arêtes à mesurer puis désactivez le cadre (et les points) pour voir le résultat final. Pas satisfait? Activez à nouveau le cadre (et les points), sélectionnez d'autres sommets et créez de nouvelles mesures, puis désactivez à nouveau le cadre.

Vous pouvez ajuster la taille des points de sommet dans l'onglet TechDraw Préférences d'échelle. Ne réglez pas leur taille sur zéro, mais sur une taille suffisamment petite ou grande pour que vous puissiez les prendre sans problème.
