Description

Désactiver l'ancrage désactive l'ancrage d'un objet. La propriété Map Mode d'un objet est définie sur Deactivated, les autres propriétés de l'ancrage restent inchangées.

Cette fonction est utile lorsque l'ancrage n'est temporairement pas nécessaire. Le mode Map Mode désactivé permet un déplacement précis des objets, par exemple avec l'outil Transformer. Après la correction du placement, l'outil Ancrer en place peut être utilisé pour appliquer la propriété Attachment Offset corrigée.

Utilisation

  1. Select one or more objects in the 3D View or Tree View holding the CTRL key.
  2. Deactivate Attachment of all selected objects by one of these methods:
    • Press the DeactivateAttachment button.
    • Select the Cable Support → Deactivate Attachment option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cable Support → Deactivate Attachment option from the context menu.
    • Use the keyboard shortcut: X then D.

Notes

Selection of elements in 3D View is treated as a selection of entire objects.
