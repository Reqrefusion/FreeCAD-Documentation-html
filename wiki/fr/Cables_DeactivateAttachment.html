Cables Ancrer en place

Description

Désactiver l'ancrage désactive l'ancrage d'un objet. La propriété Map Mode d'un objet est définie sur Deactivated, les autres propriétés de l'ancrage restent inchangées.

Cette fonction est utile lorsque l'ancrage n'est temporairement pas nécessaire. Le mode Map Mode désactivé permet un déplacement précis des objets, par exemple avec l'outil Transformer. Après la correction du placement, l'outil Ancrer en place peut être utilisé pour appliquer la propriété Attachment Offset corrigée.

Utilisation

  1. Sélectionnez un ou plusieurs objets dans la vue 3D ou l'arborescence en maintenant la touche CTRL enfoncée.
  2. Désactiver l'ancrage de tous les objets sélectionnés à l'aide de l'une des méthodes suivantes :
    • Appuyez sur le bouton Deactivate Attachment.
    • Sélectionnez l'option Cable Support → Deactivate Attachment du menu.
    • Cliquez avec le bouton droit de la souris dans l'arborescence ou la vue 3D et sélectionnez l'option Cable Support → Deactivate Attachment dans le menu contextuel.
    • Utilisez le raccourci clavier : X puis D.

Remarques

La sélection d'éléments dans la vue 3D est traitée comme une sélection d'objets entiers.
