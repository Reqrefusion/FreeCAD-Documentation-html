Description

L'outil Curves Esquisse sur une surface permet d'appliquer une esquisse sur une face courbe arbitraire, comme une étiquette sur une bouteille. L'esquisse doit être attachée à une face (voir Sketch.Support). Le mode Map de l'esquisse n'a aucun effet sur le résultat.

Ci-dessus : l'objet Sketch_On_Surface appliqué à la face du cylindre (à gauche) et l'esquisse source en mode édition (à droite)

Utilisation

Il existe deux méthodes pour utiliser l'outil Esquisse sur une surface :

Créer une esquisse sur une face sélectionnée

  1. Select the target face in the 3D view.
  2. There are several ways to invoke the tool:
    • Press the Sketch on Surface button.
    • Select the Surfaces → Sketch on Surface option from the menu.
  3. A Sketch_On_Surface object is created including a Mapped_Sketch object.
  4. Expand this object in the Tree view to make the Mapped_Sketch appear below.
  5. Edit the sketch and add geometries inside the blue construction bounds.
  6. Optionally edit the values in the Property editor to create extruded faces, or fill faces and equidistant faces from closed sketch wires, which in combination result in a solid shape, and adjust the related properties (see Properties below).

Attaching an existing sketch to a selected face

  1. Attach an existing sketch to the target face:
    • Right-click on a sketch in the Tree view and select Attachment editor from the context menu.
    • The Attachment dialog opens.
      1. Select the target face.
      2. Set the Attachment mode to Deactivated while keeping the face as the first reference.
      3. Press the OK button to close the Attachment dialog.
    • In the Property editor set Placement.Position.z = 0.
  2. Edit the sketch and add a Construction (blue) rectangle around the geometries.
    This rectangle will be the parametric bounds of the face.
  3. Select the sketch.
  4. Invoke the tool as described above.
  5. A Sketch_On_Surface object is created including the selected sketch.
  6. Optionally edit the values in the Property editor as described above.

Remarques

On suppose que toute la géométrie de l'esquisse est incluse dans le cadre de construction bleu. Cela inclut toute autre géométrie de construction, ainsi que la géométrie interne visible des courbes complexes (Bézier, ellipse). Si ce n'est pas le cas, la boîte de délimitation de l'esquisse sera plus grande que le cadre de construction et le mappage final sera réduit en conséquence. Si nécessaire, cachez la géométrie interne qui n'est pas entièrement à l'intérieur du cadre de construction.

Propriétés

See also: Property editor.

A Sketch_On_Surface object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

Settings

  • DonnéesFill Extrusion (Bool): Add extrusion faces
  • DonnéesFill Faces (Bool): Make faces from closed wires
  • DonnéesOffset (Float): Offset distance of mapped sketch
  • DonnéesThickness (Float): Extrusion thickness

SketchOnSurface

  • DonnéesExtra Objects (LinkList): Additional objects that will be mapped on surface
  • DonnéesSketch (Link): Input Sketch

Touchup

  • DonnéesConstruction Bounds (Bool): include construction geometry in sketch bounds
  • DonnéesReverse U (Bool): Reverse U direction
  • DonnéesReverse V (Bool): Reverse V direction
  • DonnéesSwap UV (Bool): Swap U and V directions
