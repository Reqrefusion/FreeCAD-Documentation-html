Description

L'outil Templater Modèle multi format crée un petit modèle d'exemple et l'insère comme nouvelle page de dessin dans le document actif.

Page avec un modèle généré, ISO A3 + nomenclature
(Image empruntée à la page Macro TemplateHelper)

Utilisation

  1. Il existe plusieurs façons de lancer l'outil.
    • Appuyez sur le bouton New Template Multi.
    • Sélectionnez l'option Templater → New Template Multi du menu.
  2. Le panneau de tâches Template settings s'ouvre.
    • Vous pouvez modifier le format de la page si vous le souhaitez.
      Si aucune autre option n'est cochée, le modèle donnera une page vierge.
    • Vous pouvez cocher l'option Draw a frame pour ajouter un cadre au modèle.
      Les options suivantes peuvent être masquées en fonction des autres options cochées.
    • Vous pouvez cocher l'option Place zone indices pour ajouter des indices de zone au cadre.
    • Vous pouvez cocher l'option Tilt indices 90° pour incliner les indices de zone supérieurs et droits afin qu'ils soient lisibles depuis la droite.
    • Vous pouvez cocher l'option Add a title block pour ajouter un cartouche scripté au modèle.
      • Vous pouvez cocher l'option Change to ink-blue pour colorer les textes modifiables.
      • Vous pouvez saisir un nombre de lignes de nomenclature pour ajouter une nomenclature au-dessus du cartouche.
    • Vous pouvez cocher l'option Insert Template pour ajouter le modèle au document, sinon le modèle sera uniquement enregistré mais pas inséré.
    • Vous pouvez cocher l'option Insert title block symbol et sélectionner l'un des blocs de titre désormais disponibles. Cela ajoutera un symbole de bloc de titre au modèle déjà inséré sans bloc de titre scripté.
  3. Cliquez sur Ok pour fermer le panneau de tâches et quitter l'outil.
  4. Un nouveau modèle sera enregistré dans le dossier .../Mod/Templater/Resources.
  5. Si l'option Insert template est cochée, le modèle créé s'affichera dans une nouvelle vue TechDraw.
