Description

L'outil TechDraw Déplacer en haut déplace les vues au sommet de l'ordre de l'empilement. L'ordre de l'empilement contrôle la profondeur apparente des vues sur une feuille.

Utilisation

  1. Sélectionnez une ou plusieurs vues sur une feuille ou dans l'arborescence. Pour cet outil, et TechDraw Empiler en bas, l'ordre de sélection est important.
  2. Il existe plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Déplacer en haut.
    • Sélectionnez l'option TechDraw → Position dans la pile → Déplacer en haut du menu.
  3. La propriété VueStack Order des vues est modifiée.

Script

Voir aussi : Autogenerated API documentation et FreeCAD Débuter avec les scripts.

L'ordre de l'empilement peut être modifié dans les scripts en changeant la propriété StackOrder de ViewObject d'une vue.
