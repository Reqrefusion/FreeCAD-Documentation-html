Description

L'outil Curves Discrétiser discrétise une arête ou une polyligne et crée donc un ensemble de points le long de l'arête/polyligne selon une méthode de discrétisation choisie.

Utilisation

  1. Sélectionnez une ou plusieurs arêtes dans la vue 3D.
  2. Il y a plusieurs façons de lancer l'outil :
    • Appuyez sur le bouton Discretize.
    • Sélectionnez l'option Curves → Discretize du menu.
  3. Un objet Discretized_Edge est créé pour chaque arête sélectionnée.
  4. Vous pouvez modifier les valeurs dans l'éditeur de propriétés pour choisir une méthode de discrétisation différente et ajuster les propriétés correspondantes (voir Propriétés ci-dessous).

Propriétés

Voir aussi : Éditeur de propriétés

Un objet Discretized_Edge est dérivé d'un App FeaturePython et hérite de toutes ses propriétés. Il possède également les propriétés supplémentaires suivantes :

Données

Discretized

  • DonnéesEdge (LinkSub): Input edge.
  • DonnéesPoint (VectorList): Lists the output points.
  • DonnéesTarget (Enumeration): Tool target.
    • SingleEdge (default): Discretizes the selected edge only.
    • Wire: Discretizes the whole wire the edge belongs to.

Method

  • DonnéesAlgorithm (Enumeration): Discretization method.
    • Number (default): A number of points is evenly distributed along the selected edge/wire.
    • QuasiNumber: ...
    • Distance: All point are placed at the same distance to their neighbors except the last pair.
    • Deflection: ...
    • QuasiDeflection: ...
    • Angular-Curvature: ...
  • DonnéesAngular (Float): Angular value for Angular-Curvature Algorithm.
  • DonnéesCurvature (Float): Curvature value for Angular-Curvature Algorithm.
  • DonnéesDeflection (Float): Distance for deflection Algorithm.
  • DonnéesDistance (Float): Distance between edge points.
  • DonnéesMinimum (Integer): Minimum Number of points.
  • DonnéesNumber (Integer): Number of edge points.

Output

  • DonnéesNormalized Parameters (FloatList): Normalized parameters list.

Parameters

  • DonnéesParameter First (Float): Start parameter.
  • DonnéesParameter Last (Float): End parameter.
