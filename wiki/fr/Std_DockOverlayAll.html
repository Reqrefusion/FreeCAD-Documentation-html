Std Basculer la transparence pour tout, Std Basculer une superposition

Description

La commande Std Basculer la superposition pour tout active ou désactive simultanément le mode superposition pour tous les panneaux ancrés.

Le mode superposition permet d'optimiser l'espace disponible pour les vue 3D en transformant les panneaux ancrés (tels que la vue combinée et la vue rapport) en widgets flottants transparents. Lorsqu'ils sont activés, ces panneaux flottent au-dessus de la fenêtre principale, permettant à la vue 3D de s'étendre dans l'espace qu'ils occupaient auparavant. Les panneaux deviennent entièrement opaques et utilisables lorsque le curseur de la souris passe dessus, et redeviennent transparents lorsque le curseur s'éloigne. Cela permet d'interagir avec le modèle 3D « à travers » le panneau transparent.

Utilisation

  1. Il existe plusieurs façons d'appeler la commande :
    • Sélectionnez l'option Affichage → Superposition des panneaux ancrables → Activer/désactiver la superposition pour tout du menu.
    • Utilisez le raccourci clavier : F4.

Préférences

L'apparence des panneaux superposés est contrôlée par des feuilles de style. Cela peut être configuré de deux manières :

  • Détection automatique : le système sélectionne automatiquement un thème de superposition en fonction du thème de l'application de l'utilisateur. Si votre thème principal (défini dans Édition → Préférences... → Affichage → Interface utilisateur → Feuille de style (avancée)) est détecté comme « clair », une superposition de style clair est utilisée. Sinon, une superposition de style sombre est utilisée. Cela fonctionne pour la plupart des utilisateurs sans aucune configuration.
  • Remplacement manuel :
    • Une feuille de style intégrée peut être sélectionnée via Édition → Préférences... → Affichage → Interface utilisateur → Feuille de style de la superposition.
    • Pour un contrôle avancé, la feuille de style peut être spécifiée à l'aide de l'éditeur de paramètres.
      1. Sélectionnez l'option Outils → Éditer les paramètres... du menu.
      2. Accédez au sous-groupe suivant : BaseApp → Preferences → MainWindow.
      3. Recherchez le paramètre OverlayActiveStyleSheet ou créez-le si nécessaire.
      4. Remplacez sa valeur par le nom d'une autre feuille de style intégrée (par exemple, Light Theme + Light Background.qss) ou par le chemin d'accès complet d'un fichier .qss personnalisé.
