Introduction

L'atelier Stemfie est un atelier externe visant à mettre en œuvre un ensemble paramétrique de pièces pour le projet Stemfie.

Example of a Stemfie Workbench set of parts

Installation

L'atelier Stemfie n'est pas inclus dans le package FreeCAD par défaut mais peut être facilement installé via le gestionnaire des extensions. Lancez-le à partir de Outils → Gestionnaire des extensions. Le code de l'atelier Stemfie est hébergé et développé sur github et peut également être installé manuellement en le copiant dans le répertoire MOD de FreeCAD.
