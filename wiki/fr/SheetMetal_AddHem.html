Description

La commande SheetMetal Ourletpermet de créer des ourlets sur les bords des surfaces sélectionnés. En modifiant la propriété Donnéesangle, un bord peut être transformé en ourlet.

Un ourlet se compose généralement d'un coude cylindrique d'environ 180° et d'une bande plane (tôle).

Deux bords sélectionnés → deux ourlets en onglet

Utilisation

  1. Sélectionnez une ou plusieurs arêtes d'une forme solide.
  2. Il existe plusieurs façons de lancer la commande :
    • Cliquez sur le bouton Créer un ourlet.
    • Sélectionnez l'option SheetMetal → Créer un ourlet dans le menu.
    • Cliquez avec le bouton droit de la souris dans l'arborescence ou la vue 3D et sélectionnez l'option SheetMetal → Créer un ourlet dans le menu contextuel.
    • Utilisez le raccourci clavier : Z.
  3. Un objet Hem est créé, composé d'un nouvel ourlet sur chaque bord sélectionné/ Le panneau des tâches Propriétés de l'ourlet s'ouvre :
    • Avec l'onglet Général sélectionné :
      • Vous pouvez appuyer sur le bouton Sélectionner pour ajouter d'autres bords.
        • Vous pouvez également appuyer sur le bouton Effacer la sélection pour effacer la liste des bords sélectionnés.
        • Appuyez sur le bouton Aperçu pour finaliser la sélection et afficher les modifications.
      • Vous pouvez également ajuster les paramètres de courbure dans cet onglet. Voir la section Général de la description du panneau de tâches ci-dessous
    • Vous pouvez passer à un autre onglet pour régler d'autres paramètres. Voir les sections Décalages ou Chanfrein de la description du panneau de tâches ci-dessous.
  4. Appuyez sur le bouton OK pour terminer la commande et fermer le panneau de tâches.
  5. Vous pouvez régler les paramètres dans l'éditeur de propriétés.

Panneau des tâches

Double-cliquez sur un objet Hem existant dans l'arborescence pour ouvrir de nouveau le panneau des tâches et modifier les paramètres.

Général


  • Select: Changes the amount of edges in the base Object property.
  • Type: Sets the HemType property.
    • Flat activates these sub-options:
      • Width: Sets the width property.
      • Reverse the wall: Toggles the invert property.
      • Width includes bend: Toggles the Include Bend property.
    • Open: activates these sub-options:
      • Width: Sets the width property.
      • Reverse the wall: Toggles the invert property.
      • Width includes bend: Toggles the Include Bend property.
      • Opening: Sets the opening property.
    • Tear Drop: activates these sub-options:
      • Width: Sets the width property.
      • Reverse the wall: Toggles the invert property.
      • Width includes bend: Toggles the Include Bend property.
      • Bend radius: Sets the radius property.
      • Open: Toggles the opened property.
        • Opening: Sets the opening property.
    • Rolled: activates these sub-options:
      • Bend radius: Sets the radius property.
      • Open: Toggles the opened property.
        • Roll angle: Sets the Roll Angle property.
  • Position: Sets the Bend Type property.
  • Unfold: Toggles the unfold property.


Décalages

  • Gap A: Sets the gap1 property.
  • Gap B: Sets the gap2 property.
  • Rectangle and Round radio buttons: Toggle the relief Type property.
  • Width: Sets the reliefw property.
  • Depth: Sets the reliefd property.

Onglet

(Not available for rolled hems)

  • Auto Miter: Toggles the Auto Miter property.
    If checked:
    • Minimum Gap: Sets the minGap property.
    • Max Extend Distance: Sets the max Extend Dist property.
    If unchecked:
    • Angle 1: sets the miterangle1 property.
    • Angle 2: sets the miterangle2 property.


Notes

  • See Create a sheet metal object for a basic workflow.

Properties


See also: Property View.

A SheetMetal Hem object is derived from a Part Feature object or, if it is inside a PartDesign Body, from a PartDesign Feature object, and inherits all its properties. It also has the following additional properties:

Data

Parameters

  • DonnéesBend Type (Enumeration): "Bend Type". Material Outside (default), Material Inside, Thickness Outside.
  • DonnéesHem Type (Enumeration): "Type of Hem". Flat (default), Open, Teardrop, Rolled.
  • DonnéesInclude Bend (Bool): "Width includes bend geometry". Default: false.
  • DonnéesRoll Angle (Angle): "Roll Angle". Default angle: 90,00°.
  • Donnéesbase Object (LinkSub): "Base Object". Link to the planar face to receive a hem.
  • Donnéesgap1 (Distance): "Gap from Left side". Default: 0,00 mm.
  • Donnéesgap2 (Distance): "Gap from Right side". Default: 0,00 mm.
  • Donnéesinvert (Bool): "Invert Bend Direction". Default: false.
  • Donnéesopened (Bool): "Opened Hem". Default: false.
  • Donnéesopening (Length): "Openening. of Hem". Default: 1,00 mm
  • Donnéesradius (Length): "Bend Radius", the default value depends on the radius property of the parent feature:
  • Donnéeswidth (Length): "Width of Wall". Default: 10,00 mm.

Parameters Ex

  • DonnéesAuto Miter (Bool): "Enable Auto Miter". Default: true.
  • Donnéeskfactor (FloatConstraint): "Location of Neutral Line. Caution: Using ANSI standards, not DIN.".
    Default: 0,50. K factor (also known as neutral factor) for the bend. Used to calculate bend allowance when unfolding.
  • Donnéesmax Extend Dist (Length): "Auto Miter maximum Extend Distance". Default: 5,00 mm.
  • Donnéesmin Gap (Length): "Auto Miter Minimum Gap". Default: 0,20 mm.
  • Donnéesmin Relief Gap (Length): "Minimum Gap to Relief Cut". Default: 1,00 mm.
  • Donnéesunfold (Bool): "Shows Unfold View of Current Bend". Default: false
    true unfolds the bend.

Parameters Miterangle

  • Donnéesmiterangle1 (Angle): "Bend Miter Angle from Left Side". Default angle: 0,00°.
  • Donnéesmiterangle2 (Angle): "Bend Miter Angle from Right Side". Default angle: 0,00°.

Parameters Relief

  • DonnéesRelief Factor (Float): "Relief Factor". Default: 0,70.
  • DonnéesUse Relief Factor (Bool): "Use Relief Factor". Default: false.
  • Donnéesrelief Type (Enumeration): "Relief Type". Rectangle (default), Round. Enabled only when a gap value is set.
  • Donnéesreliefd (Length): "Relief Depth". Default: 1,00 mm. Enabled only when a gap value is set.
  • Donnéesreliefw (Length): "Relief Width". Default: 0,80 mm. Enabled only when a gap value is set.
