Description

Matériau crée une série de Arch Matériau et de Arch Multimatériaux prêts à être utilisés avec Câble ou Boîte. Les multimatériaux par défaut suivants conviennent pour Cable : cableMultiMat3w, cableMultiMat5w, cableMultiMat8w. Ils ont une dénomination appropriée de leur composition qui correspond aux noms par défaut dans la propriété Sub Colors d'un câble. Le multimatériau boxMultiMat2m (introduit dans la version 0.3.0) peut être utilisé par CableBox équipé de rails DIN.

Lors de la création de multimatériaux pour les câbles, le champ Name de la couche unique dans la liste Composition du multimatériau doit correspondre à l'entrée appropriée dans la propriété Sub Colors d'un câble.

Isolation du câble bicolore (fonction introduite dans la version 0.3.0) : si deux couches de la liste Composition dans le multimatériau ont le même champ Name mais des champs Material différents, elles seront utilisées comme deux couleurs pour l'isolation d'un seul sous-câble portant le même nom, placé dans la propriété Sub Colors d'un câble.

Utilisation

  1. Créez un ensemble de multi-matériaux par défaut à l'aide de l'une des méthodes suivantes :
    • Appuyez sur le bouton Cable Materials.
    • Sélectionnez l'option Cables → Cable Materials du menu.
    • Cliquez avec le bouton droit de la souris dans la vue en arborescence ou la vue 3D et sélectionnez l'option Cables → Cable Materials dans le menu contextuel.

Remarques

Les multi-matériaux par défaut peuvent servir de modèles pour la création d'autres ensembles de multi-matériaux personnalisés.
