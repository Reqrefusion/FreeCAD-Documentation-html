Introduction

Le Std Menu Édition est l'un des 7 sous-menus du menu standard :

Fichier     Édition     Affichage     Outils     Macro     Fenêtre     Aide

Le menu Édition fournit des outils pour modifier le modèle.

Outils

Les outils suivants sont disponibles dans ce menu :

  • Annuler : annule la dernière commande.
  • Rétablir : rétablit une action précédemment annulée.
  • Couper : copie les cellules de la feuille de calcul dans le presse-papiers et les efface.
  • Copier : copie des objets dans le presse-papiers.
  • Coller : colle des objets du presse-papiers dans le document.
  • Dupliquer la sélection : duplique les objets dans le document. Si l'atelier PartDesign est actif, l'outil PartDesign Dupliquer la sélection (introduit dans la version 1.0) apparaît ici à la place.
  • Recalculer : recalcule le document.
  • Sélection d'objets par boîte : permet la sélection de tout objet dans une zone rectangulaire.
  • Sélection de faces par boîte : permet de sélectionner des faces depuis une zone rectangulaire.
  • Tout sélectionner : sélectionne tous les élémnets.
  • Effacer : efface les objets sélectionnés.
  • Vers la console Python : crée des variables dans la console Python référençant un objet.
  • Positionner... : ouvre le panneau des tâches Placement d'un objet.
  • Transformer : transforme un objet dans la vue 3D.
  • Aligner... : aligne un objet par rapport à un objet de référence fixe.
  • Basculer le mode édition : active ou désactive le mode d'édition d'un objet.
  • Propriétés : affiche le panneau de l'éditeur de propriétés. introduit dans la version 1.0
  • Mode d'édition : définit le mode d'édition à utiliser lorsqu'un objet est double-cliqué dans la vue en arborescence. introduit dans la version 0.20
  • Préférences : ouvre l'éditeur de préférences. -- sous MacOS, cette option de menu peut se situer dans le menu FreeCAD au lieu du menu Édition.
