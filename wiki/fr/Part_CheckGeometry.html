Description

La commande Part Vérifier la géométrie exécute une vérification et indique si la géométrie est un solide valide. L'outil vérifie si la Représentation par les Bords (BRep ou B-rep) du modèle est valide.

Utilisation

  1. Sélectionnez une pièce (attention à sélectionner la pièce entière et pas seulement une face pour vérifier la validité du solide).
  2. Il y a plusieurs façons de lancer la commande :
    • Pressez le bouton Vérifier la géométrie.
    • Sélectionnez l'option Part → Vérifier la géométrie du menu.
  3. Le panneau des tâches Paramètres s'ouvre sauf si Ne pas exécuter cette page de paramètres est activé. Voir Options pour plus d'informations. Cliquez sur Lancer la vérification.

Les résultats seront présentés dans le panneau des tâches. Si la vérification a produit des erreurs : cliquez dans le rapport sur un message d'erreur spécifique et l'objet géométrique correspondant (arête, face, etc.) sera mis en surbrillance dans la vue 3D.

Options

Sauter cette page des paramètres

Si cette case est cochée, les prochaines utilisations de l'outil n'afficheront pas la fenêtre de dialogue des Réglages.

Exécuter la vérification des opérations booléennes

Si cette case est cochée, une vérification supplémentaire des opérations booléennes (BOP) est effectuée.

Utiliser un seul thread

Vérifier la géométrie avec un seul thread, plus lent mais plus stable.

Journal des erreurs

Si cette case est cochée, toutes les erreurs trouvées seront également enregistrées dans la vue rapport.

Développer le contenu des formes

En plus de détecter les erreurs potentielles de géométrie, cet outil affiche une série de propriétés concernant l'objet sélectionné :

  • Objet vérifié
  • Type de forme
  • Nombre d'entités géométriques : sommets, arêtes, fils, faces, coques, solides, compsolides, composés, formes totales...
  • Propriétés géométriques et de masse :
    • Aire
    • Volume
    • Masse
    • Longueur
    • Centre de masse
    • Orientation
    • Axe de symétrie
    • Point de symétrie
    • Moments
    • Premier axe d'inertie
    • Deuxième axe d'inertie
    • Troisième axe d'inertie
    • Rayon de giration
    • Placement global

Afficher le contenu avancé des formes

Afficher le contenu avancé des formes. Les modifications prendront effet la prochaine fois que vous utiliserez l'outil de vérification de géométrie.

Vérifier les différentes opérations booléennes

L'utilisateur peut choisir le type d'opération booléenne à vérifier :

  • Mauvais type
  • Auto-intersection
  • Arête trop petite
  • Face non récupérable
  • Continuité
  • Incompatibilité de face
  • Incompatibilité de sommet
  • Incompatibilité d'arête
  • Courbe non valide sur la surface

Remarques

  • Les objets App Link liés aux types d'objets appropriés et les conteneurs App Part contenant les objets visibles appropriés peuvent également être vérifiés à l'aide de cet outil. Pour les App Links, la forme de l'objet lié est vérifiée. Pour les conteneurs App Part, les objets visibles qu'ils contiennent sont vérifiés en tant que composés. introduit dans la version 0.20
  • FreeCAD ne dispose pas de méthodes pour réparer automatiquement la géométrie. Si des défauts sont détectés, les étapes de la création du modèle doivent être examinées et réparées manuellement.
