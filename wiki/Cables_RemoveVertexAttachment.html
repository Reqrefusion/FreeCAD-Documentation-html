Description

The Remove Vertex Attachment removes an existing vertex attachment of WireFlex from external vertex or object. Attachments of vertexes are kept in special properties of WireFlex, for details see WireFlex Properties.

Once attachment is removed the wire vertex can be moved freely in 3D space using Draft Edit.

Usage

  1. Select one vertex of an existing WireFlex object in the 3D view.
  2. Remove selected vertex attachment by one of this methods:
    • Press the Remove Vertex Attachment button.
    • Select the Cable Wires → Remove Vertex Attachment option from the menu.
    • Right-click in the Tree view or the 3D view and select the Cable Wires → Remove Vertex attachment option from the context menu.

Notes

  • See the WireFlex Description and WireFlex Usage for the details how to use WireFlex.
