Introduction

The Std Tools Menu is one of the 7 sub-menus in the standard menu:

File     Edit     View     Tools     Macro     Windows     Help

The Tools menu provides tools for debugging models, customizing FreeCAD's behavior, as well as auxiliary tools.

Tools

The following tools are available in this menu:

The order of the tools matches the version 1.1 menu.

  • Addon manager: Opens the Addon manager with which external workbenches, preference packs and macros can be installed.
  • Measure: Measures objects. introduced in 1.0
  • Units converter...: Opens a calculator for the conversion of units.
  • Load image...: Loads an image. introduced in 0.21
  • Save image...: Saves a screenshot of the active 3D view.
  • Add text document: Creates an object capable of holding text.
  • View turntable...: Continuously rotates the camera in the active 3D view.
  • Scene inspector...: Displays an overview of scenegraph nodes of the active 3D view.
  • Dependency graph...: Shows the dependency graph of objects.
  • Export dependency graph... introduced in 0.21
  • Document utility...: Assists in repairing damaged project files.
  • Edit parameters...: Opens a dialog box giving access to the parameters that control the program.
  • Customize...: Opens a dialog box with customization options.
