기본적인 설치

FreeCAD의 최신 안정화 버전을 설치 프로그램으로 간편하게 설치할 수 있습니다. 내려받기 페이지를 참고하세요.

불안정할 수 있는 개발자 버전을 설치하고 싶다면 주간 빌드를 참고하세요.

설치 프로그램을 내려받고, 두 번 눌러 설치를 진행하세요.

다음은 일부 기술 옵션에 대한 추가 정보입니다. 대부분 사용자는 설치 프로그램만 있으면 충분합니다. 설치가 완료되면 시작하기로 이동하세요.

= Windows 시스템에서 모든 사용자를 위해 설치하는 방법

기본적으로 FreeCAD는 설치 프로그램을 실행하는 사용자만 사용할 수 있게 설치됩니다. 이 사용자에게 사용자 권한만 있는 경우 기본 설치 경로는 다음과 같습니다:

C:\Users\<username>\AppData\Local\Programs\FreeCAD X.YY

설치 프로그램을 관리자 계정으로 실행하거나 관리자 권한으로 실행하는 경우, FreeCAD를 시스템의 모든 사용자가 사용할 수 있게 설치할지, 본인만 사용할 수 있게 설치할지 선택할 수 있습니다. 기본값에서는 모든 사용자가 사용할 수 있도록 설치됩니다.

모든 사용자용으로 설치하는 경우 기본 설치 경로는 다음과 같습니다.

C:\Program Files\FreeCAD X.YY

명령창에서 자동 설치

FreeCAD를 자동으로 설치하려면 명령줄에서 설치 프로그램을 실행하면 됩니다. 

FreeCAD-~.exe /S

모든 옵션에는 기본 설정이 사용됩니다. 사용자 지정 설치 경로는 다음과 같은 방법으로 지정할 수 있습니다: 

FreeCAD-~.exe /S /D=A path to FreeCAD with spaces

기본적으로 자동 설치 시에도 설치 프로그램의 손상 여부를 검사할 때 짧은 팝업 창이 표시됩니다. 이 순환 중복 검사는 최대 몇 초밖에 걸리지 않습니다. 이 검사를 비활성화하려면 다음 단계를 따르세요: 

FreeCAD-~.exe /S /NCRC

이 검사는 설치 프로그램을 완전히 받아졌는지 확인하기 위한 것이기 때문에 /NCRC 플래그는 권장되지 않습니다.

Chocolatey

소프트웨어를 최신 상태로 유지하려면 Chocolatey와 같은 패키지 관리자를 사용하는 것이 좋습니다. [1]의 지침에 따라 Chocolatey를 설치한 후 PowerShell 터미널을 관리자 권한으로 열고 다음을 실행하세요: 

choco install freecad

다음 명령어를 사용하여 소프트웨어를 업데이트할 수 있습니다: 

choco upgrade freecad

Chocolatey 저장소에서 최신 버전을 다운로드할 수 있습니다. Chocolatey 패키지에 문제가 있는 경우 이곳에서 관리자에게 문의하실 수 있습니다.

삭제하기

FreeCAD를 제거하려면 Windows 소프트웨어 제거 도구를 사용하는 것이 좋습니다. 또는 제거 프로그램을 직접 실행할 수도 있습니다. 파일명은 다음과 같습니다:

Uninstall-FreeCAD.exe

FreeCAD가 설치된 폴더에서 찾을 수 있습니다.

명령줄을 사용하여 자동으로 설치 제거 프로그램을 실행할 수도 있습니다. 

Uninstall-FreeCAD.exe /S

FreeCAD가 실행 중인 경우, 사용 중인 FreeCAD 버전이 제거하려는 버전이 아니더라도 (자동) 제거는 실패합니다.
