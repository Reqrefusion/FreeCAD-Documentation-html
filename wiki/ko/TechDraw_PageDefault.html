기술도면 페이지 템플릿, 기술도면 템플릿

설명

기술도면 기본 페이지 도구는 기술도면 기본 설정에 지정된 템플릿 파일을 사용하여 새 페이지 개체를 만듭니다.

기술도면에 함께 제공되는 기본 템플릿: A4_LandscapeTD.svg

용법

  1. 활성 문서가 있어야 합니다.
  2. 도구를 호출하는 방법은 여러 가지가 있습니다.
    • 기본 페이지 삽입 버튼을 누르세요.
    • 메뉴에서 기술도면 → 페이지 → 기본 페이지 삽입을 선택하세요.

보충 설명

  • 페이지가 데이터Keep Updated 속성을 통해 "업데이트 유지 안 함"으로 표시되었거나, 창 컨텍스트 메뉴에서 Toggle Keep Updated 옵션을 선택하거나 환경 설정에서 설정을 변경하면 3D 모형의 변경 사항이 페이지에 반영되지 않고 무시됩니다. 이 때 페이지에 이상 현상(기하학적 요소 누락, 치수 값 누락 등)이 나타날 수도 있는데 페이지 다시 그리기 도구를 사용하여 페이지를 업데이트하면 이러한 문제가 저절로 해결됩니다. 페이지 업데이트가 중단된 동안에는 나무보기에 라는 아이콘이 페이지에 표시됩니다. 이 설정은 시작 프로세스에도 영향을 미칩니다. 페이지가 "업데이트 유지 안 함"으로 설정된 경우, 프로그램 시작 시 해당 페이지가 표시되지 않습니다.
  • 사용자 구성 파일 user.cfg에 기본 템플릿이 지정되지 않은 경우 도구는 다음을 시도합니다.
$INSTALL_DIR/Mod/TechDraw/Templates/A4_LandscapeTD.svg

여기서 $INSTALL_DIR은 프리캐드가 설치된 디렉토리입니다. 예: 

/usr/share/freecad/Mod/TechDraw/Templates/A4_LandscapeTD.svg

속성

자료

Base

  • 데이터Projection Type: 이 페이지의 기본 투상법(제1각 또는 제3각법).

Page

  • 데이터Keep Updated: false인 경우, 3D 모형 변경 사항이 페이지에 자동으로 갱신되지 않습니다. 복잡한 모형인 경우 이 값이 true로 설정되어 있으면 모형을 변경할 때마다 매번 자동으로 페이지를 갱신하는데 시간이 걸립니다. 이 때 false로 설정하면 유용합니다. 보충 설명을 참조하세요.
  • 데이터Template: 이 페이지의 도면 템플릿 대상체에 대한 링크입니다.
  • 데이터Views: 이 페이지의 보기들(정면도, 평면도, 등)에 대한 링크 목록입니다.
  • 데이터Scale: 이 페이지의 보기에 대한 기본 척도입니다.
  • 데이터Next Balloon Index: 풍선 주석에 대한 자동 번호 매기기.

보기

Grid

  • 보기Show Grid: 이 페이지에 격자를 표시합니다.
  • 보기Grid Spacing: 격자 선 사이의 거리.
