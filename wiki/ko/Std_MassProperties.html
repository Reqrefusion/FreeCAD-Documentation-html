Std MassProperties

Menu location
Tools → Mass Properties
Workbenches
All
Default shortcut
None
Introduced in version
1.2
See also
Std Measure

설명

Std MassProperties 명령어는 질량 속성 측정 기능을 시작합니다. 이 명령어는 다음과 같은 정보를 제공합니다:

  • 물리적 속성
    • 부피
    • 질량
    • 밀도
    • 표면적
  • 무게 중심
  • 부피 중심
  • 관성
    • 관성 행렬
    • 주 관성 모멘트
    • 축을 중심으로 한 관성 - 기준선을 관성 계산의 기준으로 사용하는 경우 앞서 언급한 관성 측정값을 대체합니다.

명령 작업판

용법

  1. 물리량을 확인하고 싶은 고체를 선택할 수 있습니다.
  2. 명령을 실행하는 방법은 여러 가지가 있습니다.
    • 질량 속성 버튼을 누르세요.
  3. 작업 패널이 열립니다. 자세한 내용은 선택사항을 참조하세요.
  4. 미리 선택된 고체가 없는 경우 3D보기 또는 나무보기에서 고체를 선택합니다.
  5. 측정값을 초기화하려면 초기화 버튼을 누르고, 저장하려면 저장 버튼을 누르세요. 저장된 측정값은 나무 보기의 측정값 모둠에 저장됩니다.
  6. 명령을 완료하려면 Esc 키 또는 닫기 버튼을 누르십시오.

선택사항

  • 참조:
    • 무게중 - 무게중심을 기준으로 질량관성을 계산합니다.
    • Custom - 선택… 버튼을 누르고 참조를 선택하세요.기준선 또는 지역 좌표계.
  • 단위:
    • mm, kg, kg*mm^2
    • m, kg, kg*m^2
    • in, lb, lb*in^2
    • ft, lb, lb*ft^2
  • 무게중심 또는 부피중심 섹션에 있는 기준점 생성 버튼을 누르면 해당 위치에 기준점이 생성됩니다.
  • 관성 섹션의 주축 LCS 생성버튼을 클릭하면 해당 로컬 좌표계가 생성됩니다.
