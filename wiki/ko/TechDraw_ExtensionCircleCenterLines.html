설명

기술도면 원의 중심선 도구는 원과 호에 중심선을 추가합니다.

오른쪽에는 중심선이 있는 원

용법

  1. Optionally specify the line attributes with the TechDraw ExtensionSelectLineAttributes tool.
  2. Select one or more circles or arcs.
  3. There are several ways to invoke the tool:
    • Press the Add Circle Centerlines button.
    • Select the TechDraw → Extensions: Centerlines/Threading → Add Circle Centerlines option from the menu.
  4. Centerlines are added to each circle and arc.
