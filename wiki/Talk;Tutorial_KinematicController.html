Use of QDoubleSpinBox instead of QLineEdit and Layouts

Latest comment: just now1 comment1 person in discussion

Have you considered using QDoubleSpinBox instead of QLineEdit? The main benefits would be: getting rid of many string conversions (QDoubleSpinBox ingests floats), automatic validation, respecting localisation and e.g. the use of the mouse wheel for changing values.

Also, using Layouts instead of absolute positions and dimensions would probably be a lot more straight forward, adaptable and in line with other Qt stuff.
