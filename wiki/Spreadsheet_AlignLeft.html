Description

The Spreadsheet AlignLeft tool aligns the contents of selected cells to the left.

Usage

  1. Make sure there is an active spreadsheet.
  2. Select one or more cells.
  3. There are several ways to invoke the tool:
    • Press the Align Left button.
    • Select the Spreadsheet → Alignment → Align Left option from the menu.
