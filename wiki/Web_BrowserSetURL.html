Description

Use the Web BrowserSetURL command to change the URL of the active Web tab.

Usage

  1. Make sure a Web tab is active, use the Web OpenWebsite command to create one if required.
  2. Press the Set URL button.
  3. A prompt line opens with the current URL.
  4. Edit the URL and press Enter to finish.
  5. The new URL is loaded.
