Description

Creați un obiect parametric Pană/Wedge. Valoarea implicită pentru această reținere este o bază pătratică mai mare și un vârf pătrat mai mic.

Usage

See Part Primitives.

Example

A Part Wedge object created with the scripting example below is shown here.

Notes

  • The values of the coordinates of the wedge must be such that a valid solid can be create. This means that the front and rear base shapes can both be single edges, but not if they are parallel. And if one of the base shapes is a vertex the other shape must be a rectangular face.

Properties

See also: Property View.

A Part Wedge object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

Attachment

The object has the same attachment properties as a Part Part2DObject.

Wedge

  • DateXmin (Distance): The lowest X-coordinate of the front face of the wedge. The default is 0mm.
  • DateYmin (Distance): The Y-coordinate of the front face of the wedge. The default is 0mm.
  • DateZmin (Distance): The lowest Z-coordinate of the front face of the wedge. The default is 0mm.
  • DateX2min (Distance): The lowest X-coordinate of the rear face of the wedge. The default is 2mm.
  • DateZ2min (Distance): The lowest Z-coordinate of the rear face of the wedge. The default is 2mm.
  • DateXmax (Distance): The highest X-coordinate of the front face of the wedge. The default is 10mm.
  • DateYmax (Distance): The Y-coordinate of the rear face of the wedge. The default is 10mm.
  • DateZmax (Distance): The highest Z-coordinate of the front face of the wedge. The default is 10mm.
  • DateX2max (Distance): The highest X-coordinate of the rear face of the wedge. The default is 8mm.
  • DateZ2max (Distance): The highest Z-coordinate of the rear face of the wedge. The default is 8mm.

Scripting

See also: Autogenerated API documentation, Part scripting and FreeCAD Scripting Basics.

A Part Wedge can be created with the addObject() method of the document: 

wedge = FreeCAD.ActiveDocument.addObject("Part::Wedge", "myWedge")
  • Where "myWedge" is the name for the object.
  • The function returns the newly created object.

Example: 

import FreeCAD as App

doc = App.activeDocument()

wedge = doc.addObject("Part::Wedge", "myWedge")
wedge.Xmin = 1
wedge.Ymin = 2
wedge.Zmin = 3
wedge.X2min = 4
wedge.Z2min = 6
wedge.Xmax = 15
wedge.Ymax = 20
wedge.Zmax = 55
wedge.X2max = 10
wedge.Z2max = 12
wedge.Placement = App.Placement(App.Vector(1, 2, 3), App.Rotation(75, 60, 30))

doc.recompute()
