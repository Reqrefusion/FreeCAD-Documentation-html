Descriere

The Sketcher SelectElementsWithDoFs tool selects the under-constrained elements in the sketch.

If such elements exist in a sketch the Sketch Edit section of the Sketcher Dialog displays this message:

  • Under-constrained: n Degrees of Freedom

Where n is the remaining number of degrees of freedom. Clicking the underlined text will select the under-constrained elements.

Please note that a sketch can also have under-constrained elements if one of the other messages is displayed.

Usage

  1. There are several ways to invoke the tool:
    • Click the underlined text in the Sketcher Dialog as described above.
    • Select the Sketch → Visual Helpers → Select Under-Constrained Elements option from the menu.
    • Use the keyboard shortcut: Z then F.
  2. The under-constrained sketch elements are selected.
  3. Optionally click in an empty area in the 3D View to clear the selection.
