Introducere

Acest instrument vă permite să a dăugați un fișier *.IGES, *.STEP, *.BREP la documentul curent.

Utilizare

  1. Click pe Part → ImportCAD din meniul principal.
  2. Selectați fișierul de importat

Notes

  • The Std Import command supports many more file formats.
