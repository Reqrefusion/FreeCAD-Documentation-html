Introduction

The Std Tools Menu is one of the 7 sub-menus in the standard menu:

Tools

The Tools menu provides tools for debugging models, customizing FreeCAD's behavior, as well as auxiliary tools.

Tools

The following tools are available in this menu:

The order of the tools matches the version 1.1 menu.

  • Addon Manager: Opens the Addon Manager with which external workbenches, preference packs and macros can be installed.
  • Measure: Measures objects. introduced in 1.0
  • Clarify Selection: Lists all objects of different types located near the cursor to facilitate their selection. introduced in 1.1
  • Quick Measure: Checkbox to disable quick measure. introduced in 1.1

Units Calculator oferă conversii rapide între sistemele de măsurare de la ex mm la inci.

  • Load Image…: Loads an image. introduced in 0.21

Save picture ia un instantaneu/captură a(l) ferestrei 3D actuale.

  • Text Document: Creates an object capable of holding text.

View turntable afișează o vedere rotativă a modelului în fereastra 3D.

Scene inspector afișează detalii despre reprezentarea Open Inventor a ferestrei 3D.

Dependency graph arată cum sunt legate diferitele caracteristici ale unui model.

  • Export Dependency Graph introduced in 0.21

Project utility asistă la repararea fișierelor de proiect deteriorate.

Edit parameters permite multor setări care controlează FreeCAD să fie schimbate. Nu stabilește parametri pentru un model.

Customize permite meniurile, tastele de comenzi rapide și barele de instrumente să fie modificate.
