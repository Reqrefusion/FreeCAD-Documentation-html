Description

The Robot ExportKukaCompact command exports the trajectory as a compact KRL subroutine.

Usage

  1. Select a robot and trajectory in the Tree View.
  2. Select the Robot → Export Trajectory → Kuka Compact Subroutine option from the menu.
  3. A KRL file (*.src) is created.
