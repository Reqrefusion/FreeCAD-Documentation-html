Description

Cuts through a surface and makes the crossection available. Normal vector and position can be freely chosen to get the desired cut.

This command is similar to Part CrossSections, but it is fully parametric and has an option to reduce the complexity of the output curve. It tries to remove overlapping edges.

Image of a simple FreeCAD project with two sketches using the SurfaceCut on a surface

Usage

  1. Select the surface to cut through
  2. There are several ways to invoke the command:
    • Press the Surface Cut button
    • Select the Curved Shapes → Surface Cut option from the menu

Properties

Interpolated Middle

  • DataFace: Set to true if the cut should be a face instead of a cross section
  • DataNormal: The normal vector of the resulting face
  • DataPosition: Position of the cut origin
  • DataSimplify: Parameter to simplify cut edges
  • DataSurfaces: The surface(s) to cut through
