Описание

The FCGear WormGear command creates a worm gear. By default this is a single thread worm with one start that can also be transformed into a double or multiple thread worm with two or several starts respectively.

Червяк можно рассматривать как частный случай косозубой передачи. Представьте, что на цилиндрической шестерне есть только один зуб. Теперь увеличьте угол наклона спирали настолько, чтобы зуб несколько раз оборачивался вокруг шестерни, прежде чем появится на противоположной стороне. В результате получится однозаходный червяк.

Для однозаходного червяка каждый полный оборот (360 градусов) червяка продвигает шестерню на один зуб. Таким образом, шестерня с 24 зубьями обеспечивает передаточное число 24:1. Для многозаходного червяка передаточное число равно количеству зубьев на шестерне, делённому на количество заходов червяка.

Червяк можно использовать только с червячным колесом. Это называется червячной передачей. Как и другие зубчатые передачи, червячный привод может снижать скорость вращения или передавать больший крутящий момент. Одно из главных преимуществ червячных редукторов - возможность передачи движения на 90 градусов. Червячная передача также является самоблокирующейся.

Применение

  1. Переключись на Верстак Шестерня.
  2. Есть несколько способов вызвать команду:
    • Нажми кнопку Червяк червячной пары на панели инструментов.
    • Выбери в меню Шестерня → Червяк червячной пары.
  3. Измени параметры шестерни в соответствии с требуемыми значениями (смотри Свойства).

Свойства

See also: Property View.

Объект Червяк червячной пары (FCGear WormGear) является производным от объекта Конструктивный элемент и наследует все его свойства. Он также имеет следующие дополнительные свойства:

Данные

base - базовые

  • Данныеdiameter - диаметр (Длина): По умолчанию 5 мм. Диаметр шага.
  • Данныеheight - высота (Длина): По умолчанию 5 мм. Значение длины червяка.
  • Данныеmodule - модуль (Длина): По умолчанию 1 мм. Модуль - это отношение делительного диаметра шестерни к числу зубьев (смотри Примечания).
  • Данныеreverse_pitch - обратный_шаг (Булево): По умолчанию false, true меняет направление вращения с правого на левое.
  • Данныеnum_teeth - число_зубьев (Целое число): По умолчанию 3. Число зубьев (заходов) (смотри Примечание).

computed - вычисленные

  • Данныеbeta - бета (Угол): (только для чтения) Ведущий угол (смотри также информацию в Примечаниях и Полезные формулы).

involute - эвольвента

  • Данныеpressure_angle - угол_давления (Угол): По умолчанию 20 ° (смотри Примечания).

tolerance - допуск

  • Данныеclearance - зазор (С плавающей точкой): По умолчанию 0.25 (смотри Примечания).
  • Данныеhead - головка (С плавающей точкой): По умолчанию 0 мм. Это значение используется для изменения высоты зуба.

version - версия

Примечания

  • beta - бета: Если ведущий угол меньше 5°, то это самоблокирующийся механизм. Типичный пример - колки для настройки на гитаре или укулеле.
  • clearance - зазор: В червячной передаче зазор - это расстояние между вершиной зуба червяка и корнем зуба червячного колеса.
  • module - модуль: В соответствии с рекомендациями ISO (Международная организация по стандартизации), размер модуля обозначается как единица измерения размеров зубьев шестерни. Модуль (m): m = 1 (p = 3,1416), m = 2 (p = 6,2832), m = 4 (p = 12,566). Если умножить модуль на Pi, можно получить Pitch - Шаг (p). Шаг - это расстояние между соответствующими точками на соседних зубцах. При изменении модуля меняется и ведущий угол (beta).
  • num_teeth - число_зубьев: Количество зубьев у червяка называется числом нитей. Соответственно, говорят об одно-, двух- и многониточных червяках. Как правило, производятся в основном однониточные черви, но в особых случаях число нитей может достигать четырёх (иногда и больше). При изменении числа зубьев меняется и beta - бета.
  • pressure_parameter - угол_давления: Изменяйте этот параметр только при наличии достаточных знаний о геометрии шестерни.

Полезные формулы

  • бета (ведущий_угол) = arctan (модуль * число_зубьев : диаметр_шага (диаметр))
  • осевой_шаг = pi * модуль * число_зубьев
  • бета (ведущий_угол) = arctan (осевой_шаг : (диаметр_шага (диаметр) * pi))
  • длина_червяка = 4,5 * модуль * pi

Червячное колесо

Червячное колесо должно быть спроектировано вручную. Для этого можно использовать Эвольвентную шестерню для простой конструкции. В любом случае требуется глубокое знание о типах передач.

Scripting

Use the power of Python to automate your gear modeling: 

import FreeCAD as App
import FreeCADGui as Gui
import freecad.gears.commands
gear = freecad.gears.commands.CreateWormGear.create()
gear.num_teeth = 2
gear.diameter = 10
gear.height = 20
App.ActiveDocument.recompute()
Gui.SendMsgToActiveView("ViewFit")
