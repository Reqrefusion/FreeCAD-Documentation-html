Введение

Дополнительный верстак для FreeCAD, основной целью которого является предоставление инструментов для облегчения моделирования деревянных конструкций (каркасов, деревянных рам и т.д.). Это вилка от FreeCAD-Timber (Jonathan Wiedemann wood-galaxy), которая больше не поддерживается.

Справочные данные

  • Author: JeromeL63.
  • Главная страница и исходный код.

Функции

  • Создание балок с расширенным позиционированием.
  • Создание панелей с расширенным позиционированием.
  • Атрибуты, используемые для удобного выбора и фильтрации объектов (имя, группы, подгруппы, материал, тип ....).
  • Выбор и показ по атрибутам.
  • Списки для счетов-фактур и производственных ведомостей.

Инструменты

Wood Frame включает в себя следующие инструменты:

  • Добавление слоя в документ: объекты, добавленные в слой, могут иметь одинаковые визуальные свойства.
  • Добавление деревянной балки. Для перемещения точки вставки используйте Numpad.
  • Добавление деревянной панели. Этот инструмент работает так же, как и "Добавить балку", но имеет ограничение на размер панели. Чтобы переместить точку вставки, используйте Numpad.
  • Вырезать объект с помощью плоскости.
  • Вырезать объект линией с нормальной рабочей плоскостью.
  • Копировать объект: дублирует элемент.
  • Инструмент «Растянуть»: выделение конца балки и добавление или уменьшение её длины.
  • Редактирование атрибутов существующих объектов (имя, группы, подгруппы, материал, тип...).
  • Выделение или скрытие элементов в 3D-виде по их определённым атрибутам.
  • Создание списка деревянных балок: Создать электронную таблицу с перечнем различных материалов, выбранных в 3D виде.
  • Редактирование атрибутов.
  • Выбор по атрибутам.
  • Выровнять вид с камеры по текущей рабочей плоскости: Выровнять вид по текущей рабочей плоскости.
  • Создать контейнер экспорта: Создание раздела, содержащего объекты для экспорта, и настройки представлений.
  • Добавить вид в контейнер для экспорта.
  • Экспорт видов в DXF: экспорт видов из контейнера в файл DXF.

Установка

Чтобы установить этот внешний верстак с GitHub, перейдите в Менеджер дополнений и нажмите на кнопку Настроить, затем добавьте https://github.com/JeromeL63/Wood-Frame в качестве пользовательского репозитория. Обновите список дополнений и добавьте Wood Frame.

Wood Frame совместим с Freecad 0.19.
