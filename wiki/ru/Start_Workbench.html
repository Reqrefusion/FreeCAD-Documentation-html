Оглавление
The Start Workbench is no longer included after version 0.21.
It has been replaced by the Start page that can be accessed via the Help menu.

Верстак Start на самом деле не является верстаком в полном смысле. Это просто страница "приветствия", которая отображается при открытии FreeCAD без загруженного документа.

В этом интерфейсе вы можете просмотреть полезную информацию и перейти к недавно открытым файлам, примерам, документации и последним новостям из мира FreeCAD. Вы можете изменить поведение Верстака Start или изменить верстак по умолчанию в Редакторе Настроек в меню Правка → Настройки → Start.
