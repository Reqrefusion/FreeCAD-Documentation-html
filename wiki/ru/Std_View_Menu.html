Введение

Меню "Вид" это одно из 7-ми меню стандартного меню:

Файл     Правка     Вид     Инструменты     Макросы     Окна     Справка

Меню Вид предоставляет инструменты для настройки 3D вида, свойств отображения объектов, а также инструменты, связанные с отображением компонентов интерфейса.

Инструменты

В этом меню доступны следующие инструменты:

  • Создать новый вид: Создает новый вид.
  • Ортогональная проекция: Переключает вид в режим ортогональной проекции.
  • Перспективная проекция: Переключает вид в режим перспективной проекции.
  • На весь экран: Переключает главное окно в полноэкранный режим.
  • Стандартные виды
    • Уместить всё: Уместить всё содержимое на экране.
    • Уместить выделенное: Уместить выбранные объекты на экране.
    • Axonometric
      • Изометрическая: Установить изометрический вид.
      • Диметрическая: Установить диметрический вид.
      • Триметрическая: Установить триметрический вид.
    • Домой: Установить вид по умолчанию.
    • Спереди: Установить вид спереди.
    • Сверху: Установить вид сверху.
    • Справа: Установить вид справа.
    • Сзади: Установить вид сзади.
    • Снизу: Установить вид снизу.
    • Слева: Установить вид слева.
    • Повернуть влево: Повернуть вид на 90 градусов против часовой стрелки.
    • Повернуть вправо: Повернуть вид на 90 градусов по часовой стрелке.
    • Сохранить рабочий вид: Сохраняет настройки камеры активного 3D-вида. представлено в версии 0.21
    • Восстановить рабочий вид: Восстанавливает сохраненные настройки камеры активного 3D-вида. представлено в версии 0.21
  • Положения камеры
    • Сохранить виды в файл...: Сохраняет виды в файл.
    • Загрузить виды из файла...: Загружает виды из файла.
    • Запомнить вид: Запоминает текущий вид в сохраненом виде.
    • Забыть все виды: Удаляет все сохраненные виды.
  • Стиль представления
    • Как есть: Переключает стиль представления в режим: Как есть.
    • Точки: Переключает стиль представления в режим: Точки.
    • Каркас: Переключает стиль представления в режим: Каркас.
    • Скрытые линии: Переключает стиль представления в режим: Скрытые линии (wireframe).
    • Без затенения: Переключает стиль представления в режим: Без затенения.
    • Только грани: Переключает стиль представления в режим: Только грани.
    • Плоские линии: Переключает стиль представления в режим: Плоские линии.
  • Bounding Box: Toggles the bounding box highlighting mode.
  • Стерео
    • Стерео красный/голубой: Переключить режим стерео в красного/голубой режим.
    • Четверная буферизация стерео: Переключить стерео в режим quad буффера.
    • Стерео с чередованием строк: Переключить стерео в режим чередования строк.
    • Стерео с чередованием столбцов: Переключить стерео в режим чередования столбцов.
    • Выключить стерео: Выключить стерео режим.
    • Выводить положение камеры: вывести настройки камеры в отчёт и в Python консоль.
  • Масштаб
    • Увеличить: Увеличивает масштаб изображения вида.
    • Уменьшить:Уменьшает масштаб изображения вида.
    • Увеличить область: Масштабирует вид под размер прямоугольной области.
  • Окно документа
    • Закрепленное: Встраивает 3D вид открепленный от основного окна приложения, обратно в приложение.
    • Откреплённое: Отсоединяет 3D вид от основного окна приложения в виде отдельного окна.
    • На весь экран: Переключает 3D вид в полноэкранный режим.
  • Показать/скрыть оси координат: Переключает видимость осей координат.
  • Плоскость сечения: Позволяет просмотреть объект в разрезе.
  • Текстурирование...: Временно накладывает текстуру поверх всех объектов.
  • Видимость
    • Скрыть/показать выделенное: Переключает видимость выбранных объектов.
    • Показать выделенное: Показывает выбранные объекты.
    • Скрыть выделенное: Скрывает выбранные объекты.
    • Выбрать видимые объекты: Выбрать все видимые объекты.
    • Инвертировать все видимости: Изменить видимость всех объектов на противоположную.
    • Показать все объекты: Делает все объекты видимыми.
    • Скрыть все объекты: Скрывает все объекты.
    • Откл/вкл выделяемость: Откл/вкл возможность выбирать объекты.
    • Скрыть/показать всё измерения: Откл/вкл отображение измерений деталей.
    • Удалить все измерения: Удаляет все измерения детали.
  • Навигация/редактирование
  • Material: Sets the material of selected objects. -- Available if the Material Workbench has been loaded directly or indirectly (via for example the Part Workbench or the PartDesign Workbench). introduced in 1.0
  • Внешний вид...
  • Случайный цвет: Устанавливает цвет фигуры случайным образом.
  • Appearance per Face: Sets the display properties of selected faces. -- Available if the Part Workbench or the PartDesign Workbench is active.
  • Toggle Transparency: Toggles the transparency of selected objects. introduced in 1.0
  • Верстак -- Выбрать верстак из подменю.
  • Панели инструментов -- Каждая панель инструментов может быть включена или выключена в подменю. Некоторые верстаки добавляют свои собственные панели инструментов.
    • Файл
    • Workbench
    • Макрос
    • Вид
    • Structure
  • Панели -- Каждую панель можно включить или выключить в этом подменю.
    • Отчёт
    • Дерево документа
    • Просмотр свойств
    • Просмотр выделения
    • Комбо панель
    • Консоль Python
    • Просмотр DAG
  • Overlay Docked Panel introduced in 1.0
    • Toggle Overlay for All Panels: Toggles overlay mode for all docked windows.
    • Toggle Transparent Panels: Toggles transparent mode for all docked overlay windows. This makes the docked windows stay transparent at all times.
    • Toggle Overlay: Toggles overlay mode for the docked window under the cursor.
    • Toggle Transparent Mode: Toggles transparent mode for the docked window under cursor. This makes the docked window stay transparent at all times.
    • Bypass Mouse Events in Overlay Panels: Bypasses all mouse events in docked overlay windows.
    • Toggle Left: Shows/hides the left overlay panel.
    • Toggle Right: Shows/hides the right overlay panel.
    • Toggle Top: Shows/hides the top overlay panel.
    • Toggle Bottom: Shows/hides the bottom overlay panel.
  • Link Navigation
    • Go to Linked Object: Selects the linked object and switches to its document.
    • Go to Deepest Linked Object: Selects the deepest linked object and switches to its document.
    • Select All Links: Selects all links to an object.
  • Tree View Actions
    • Sync View: Toggles the Tree View SyncView mode.
    • Sync Selection: Toggles the Tree View SyncSelection mode.
    • Sync Placement: Toggles the Tree View SyncPlacement mode.
    • Preselection: Toggles the Tree View PreSelection mode.
    • Record Selection: Toggles the Tree View RecordSelection mode.
    • Single Document: Switches the Tree View to SingleDocument mode.
    • Multi Document: Switches the Tree View to MultiDocument mode.
    • Collapse/Expand: Switches the Tree View to CollapseDocument mode.
    • Initiate Dragging: Initiates a drag operation for selected objects in the Tree View.
    • Go to Selection: Scrolls the Tree View to the first created object in a 3D View selection.
    • Selection Back: Restores the previous Tree View selection.
    • Selection Forward: Restores the next Tree View selection.

Строка состояния: Переключает видимость строки состояния.
