  • Сообщество

FreeCAD основан на Qt framework и отличается простым и понятным интерфейсом. Более опытные пользователи САПР смогут найти сходство с другими программами, в то время как новичкам будет легко ориентироваться и открывать для себя различные возможности FreeCAD. Вот внешний вид FreeCAD по умолчанию:

Стартовая страница служит в качестве приветственного экрана, предназначенного для быстрого и удобного доступа к основным областям FreeCAD, которые пользователь может захотеть изучить. С её помощью пользователи могут легко создавать новые детали, открывать недавние файлы и начинать 2D-черчение. Кроме того, он содержит ярлыки на полезные ресурсы, такие как учебники и пользовательские форумы, которые неоценимы как для новичков, так и для опытных пользователей, желающих получить рекомендации или советы. Пользователи могут легко настроить внешний вид стартовой страницы в соответствии со своими предпочтениями.

По мере освоения FreeCAD вы можете изменять настройки в разделе предпочтений. Это позволит настроить FreeCAD так, чтобы при запуске он открывался непосредственно в одном из рабочих столов с готовым новым документом. В качестве альтернативы можно просто закрыть вкладку Стартовая страница и вручную создать новый документ.

Верстаки

В FreeCAD используется система «верстаков», похожая на структуру концепций, используемых в таких программах для проектирования, как Revit или CATIA. Идея верстака аналогична специализированным станциям в научной лаборатории, где разные рабочие места оборудованы для разных типов экспериментов. В лаборатории одна зона может быть посвящена химии, другая - биологии, третья - физике, каждая из которых оснащена специальными инструментами, необходимыми для этих дисциплин.

В контексте FreeCAD каждый Верстак предназначен для выполнения определённого типа задач, организуя все необходимые инструменты для этой деятельности в одном интерфейсе. При переключении между Верстаками набор инструментов и элементов управления, отображаемых в пользовательском интерфейсе, изменяется в соответствии с требованиями выбранной задачи, хотя фактическое содержимое проекта или «сцена», над которой вы работаете, не меняется. Это позволяет плавно переходить от одного рабочего процесса к другому, например, начинать проектирование с базовых 2D-фигур в верстаке Draft, а затем дорабатывать их с помощью продвинутых инструментов моделирования в верстаке Part.

Термины «Верстак» и «Модуль» иногда используются как взаимозаменяемые, но в FreeCAD они имеют разные значения. Модуль - это любое расширение, добавляющее функциональность FreeCAD, а верстак - это особый вид модуля, оснащённый собственными компонентами пользовательского интерфейса, такими как панели инструментов и меню, предназначенными для выполнения определённых типов задач. Таким образом, каждый Верстак является Модулем, но не каждый Модуль может считаться Верстаком.

Важнейший элемент управления в FreeCAD это переключатель Верстаков, для перехода от одного верстака к другому.

  • Верстак Assembly для создания и решения механических сборок. представлено в версии 1.0
  • Верстак BIM (Строительные Конструкции) для работы с архитектурными элементами и создания BIM моделей. Он объединяет Верстак Arch и ранее внешний Верстак BIM, представленный 0.21 и ниже.
  • Верстак CAM (ЧПУ) используется для создания инструкций G-кода. А 0.21 и ниже этот верстак назывался "Path".
  • Верстак Draft (Набросок) содержит 2D-инструменты и основные операции 2D и 3D САПР (CAD).
  • Верстак FEM (МКЭ) обеспечивает рабочий процесс анализа Методом Конечных Элементов (МКЭ).
  • Верстак Inspection (Инспектирование) создан для того, чтобы предоставить вам специальные инструменты для изучения фигур. Все ещё находится на ранних стадиях разработки.
  • Верстак Material (Материал) работает с системой материалов FreeCAD. представлено в версии 1.0
  • Верстак Mesh (Сетки) для работы с триангулярными (разбитыми на треугольники) сетками.
  • Верстак OpenSCAD обеспечивает совместимость с OpenSCAD и восстановление истории построения модели конструктивной твердотельной геометрии (CSG).
  • Верстак Part (Деталь) для работы с геометрическими примитивами и булевыми операциями.
  • Верстак Part Design (Проектная Деталь) для построения форм Детали из эскизов.
  • Верстак Points (Точки) для работы с облаками точек.
  • Верстак Reverse Engineering (Обратное Проектирование) предназначен для предоставления специальных инструментов для преобразования фигур/твёрдых тел/схем в параметрические компоненты, совместимые с FreeCAD.
  • Верстак Robot (Робот) для изучения движений робота. В настоящее время не обслуживается.
  • Верстак Sketcher (Скетчер) для работы с геометрически-ограниченными эскизами.
  • Верстак Spreadsheet (Электронная таблица) для создания и работы с данными электронных таблиц.
  • Верстак Surface (Поверхность) предоставляет инструменты для создания и модификации поверхностей. Он похож на опцию Грань из рёбер Построителя Форм верстака Part.
  • Верстак TechDraw (Технический Чертёж) для создания технических чертежей из 3D-моделей. Это преемник Верстака Drawing (Чертёж).
  • Верстак Test Framework (Система тестирования) предназначен для отладки FreeCAD.

Верстаки часто сбивают с толку новых пользователей, поскольку не всегда легко понять, в каком верстаке искать тот или иной инструмент. Но их довольно быстро освоить, и через некоторое время они будут казаться естественными. Новые пользователи быстро понимают, что Верстаки - это удобный способ организации множества инструментов, которые может предложить FreeCAD. Кроме того, верстаки можно полностью настраивать.

Интерфейс

Давайте приглядимся к различным частям интерфейса:

Главное окно приложения можно условно разделить на 11 разделов:

  1. The Main view area, which can contain different tabbed windows.
  2. The 3D view, normally embedded in the main view area. The 3D view is the central element of the interface, displaying and allowing manipulation of the objects you are working on. It is possible to have multiple views of the same document (or objects) or to have several documents open simultaneously. Additionally, each view can be detached from the main window separately.
  3. The Model and the Tasks tab.
    1. The Model tab shows you the contents and structure of your document.
    2. The Tasks tab is where FreeCAD will prompt you for values specific to the workbench and tool you are currently using (for example dimensions of an object).
  4. The Property editor which appears when the Model tab is active in the interface. It allows managing the publicly exposed properties of the objects in the document. It consists of the Data and View sections, showing the visualization properties and the parametric properties of the objects respectively.
  5. The Selection view which indicates the objects or sub-elements of objects (vertices, edges, faces) that are selected.
  6. The Report view where messages, warnings and errors are shown.
  7. The Python console. The Python console, where all the commands executed are printed, and where you can enter Python code.
  8. The Status bar where some messages and tooltips appear.
  9. The toolbar area, where the toolbars are docked.
  10. The workbench selector, where you select the active workbench.
  11. The standard menu, which holds basic operations of the program.

Most of the above panels can be hidden or revealed using the View → Panels menu

Настройка интерфейса

Интерфейс FreeCAD глубоко настраиваем. Все панели и инструменты могут перемещаться в разные места или накладывать друг на друга. Они так же могут быть закрыты и открыты при необходимости через меню Вид или правым кликом на пустом месте интерфейса. Однако доступны много других опций, вроде создания пользовательских панелей инструментов с инструментами из других Верстаков, или назначение и изменение клавиатурных сокращений.
