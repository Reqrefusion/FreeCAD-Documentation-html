Введение

В FreeCAD слово "Эскиз" обычно используется для обозначения Sketcher SketchObject (Sketcher::SketchObject класс), который определяется Верстаком Sketcher. Это 2D-чертеж, который использует математические ограничения для точного описания 2D-геометрии.

Дополнительные сведения об этом типе объектов см. в разделе Sketcher SketchObject.

Применение

Существует два наиболее распространенных способа создания эскиза: непосредственно с помощью верстака Sketcher или через верстак PartDesign.

Верстак Sketcher

  1. Переключитесь на верстак Sketcher.
  2. Нажмите на Создать новый эскиз.

Верстак PartDesign

  1. Переключитесь на Верстак PartDesign.
  2. Нажмите на Создать новое тело.
  3. Нажмите на Создать эскиз.

После завершения редактирования эскиза закройте его, чтобы выйти из режима редактирования. Дважды щёлкните по нему, чтобы снова войти в режим редактирования.

Примечания

Эскиз обычно используется вместе с верстаком PartDesign для создания твёрдых тел выдавливанием, используя операцию Выдавить выбранный эскиз.

Тем не менее, Эскиз(Sketch) всегда может быть создан сам по себе для любой другой цели; он не обязан быть привязан к Созданию нового тела верстака ПроектнаяДеталь (PartDesign). Например, инструмент Архитектура Окно (Arch Window) в Верстаке Архитектура (Arch) использует эскизы для определения формы окон и дверей; точно так же эскиз может быть использован для определения формы Архитектуры Стен (Arch Walls).
