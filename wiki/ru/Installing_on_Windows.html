Стандартная установка

Самый простой способ установить последнюю стабильную версию FreeCAD - использовать программу установки:

Если вы хотите загрузить версию для разработчиков (которая может быть нестабильной), смотрите страницу Еженедельных сборок.

После загрузки программы установки, дважды щёлкните по ней, чтобы запустить процесс установки.

Ниже приведена дополнительная информация о некоторых технических вариантах. Но большинству пользователей не нужно больше, чем эта программа установки. Перейдите в Начало работы после завершения установки.

Установка для всех пользователей Системы Windows

По умолчания FreeCAD устанавливается только для пользователя от которого происходит запуск программы установщика. Если пользователь не имеет административных прав, то путь установки по умолчанию будет следующий:

C:\Users\<username>\AppData\Local\Programs\FreeCAD X.YY

Если программа установщик запущена от имени администратора, или текущий пользователь является администратором, от в процессе установки FreeCAD будет доступен выбор установить только для текущего пользователя или для всех пользователей в системе.

При установке для всех пользователей в системе, путь установки по умолчанию будет следующий:

C:\Program Files\FreeCAD X.YY

Тихая (автоматическая) установка

Для установки FreeCAD в тихом режиме можно запустить установщик из командной строки: 

FreeCAD-~.exe /S

Для всех параметров будут использоваться настройки по умолчанию. Пользовательский путь установки можно указать следующим образом: 

FreeCAD-~.exe /S /D=A path to FreeCAD with spaces

По умолчанию, даже при скрытой установке, при проверке установщика на наличие повреждений появляется короткое всплывающее окно. Эта так называемая циклическая проверка избыточности занимает не более нескольких секунд. Чтобы отключить проверку на наличие повреждений: 

FreeCAD-~.exe /S /NCRC

Обратите внимание, что использование флага /NCRC не рекомендуется, поскольку проверка на повреждение гарантирует, что установщик был, например, полностью загружен.

Chocolatey

Настоятельно рекомендуется использовать менеджер пакетов, например Chocolatey, для обновления программного обеспечения. Вы можете установить Chocolatey, следуя этим инструкциям, а затем открыть терминал PowerShell от имени администратора и выполнить следующую команду: 

choco install freecad

Время от времени вы можете обновлять программное обеспечение с помощью следующих команд: 

choco upgrade freecad

Это позволит получить последнюю доступную версию из репозитория Chocolatey. Если у вас возникнут какие-либо проблемы с пакетом Chocolatey, вы можете связаться с разработчиками на этой странице.

Удаление

To uninstall FreeCAD it is preferable to use the Windows tools for uninstalling software. Alternatively you can execute the uninstaller directly. This is the file:

Uninstall-FreeCAD.exe

You can find it in the folder where FreeCAD is installed.

The uninstaller can also be executed silently using the command line: 

Uninstall-FreeCAD.exe /S

Note that (silent) uninstallation will fail if there is a running instance of FreeCAD, even if that instance is not the version being uninstalled.
