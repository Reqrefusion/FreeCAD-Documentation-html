Оглавление

FreeCAD, подобно многим современным приложениям для проектирования, таким как Revit или CATIA, базируется на концепции верстаков. Верстаки можно рассматривать как набор инструментов, специально сгруппированных под определенные задачи. В традиционных мебельных мастерских, вы имели бы рабочий стол для человека работающего с деревом, другой для работающего с металлическими частями, и, возможно, третий для того, который монтирует все это вместе.

Тоже самое относится и к FreeCAD. Инструменты сгруппированы в верстаки в соответствии с задачами, к которым они относятся.

Когда вы переключаетесь с одного верстака на другой, доступные в интерфейсе инструменты меняются. Панели инструментов, командные панели и, возможно, другие части интерфейса, переключаются на новый инструментарий, но содержание вашей сцены не меняется. Вы могли бы, например, начать рисовать 2D форму в верстаке Draft (Набросок), а дальше работать над ней в верстаке Part (Деталь).

Следует отметить, что иногда верстаки называют модулями. Однако, стоит отметить, что верстаки и модули это разные понятия. Модуль - это любое расширение FreeCAD, в то время как Верстак - это специальный тип модуля со своей собственной конфигурацией графического интерфейса (панели инструментов и меню).

Встроенные верстаки

Текущие

Следующие верстаки идут в комплекте с каждой установкой FreeCAD:

  • Стандартные инструменты. На самом деле это не совсем верстак, а скорее категория 'стандартных' команд и инструментов, которые можно использовать на всех верстаках.
  • Верстак Assembly (Сборка) для создания и решения механических сборок. представлено в версии 1.0
  • Верстак BIM (Строительные Конструкции) для работы с архитектурными элементами и создания BIM моделей. Он объединяет Верстак Arch и ранее внешний Верстак BIM, представленный 0.21 и ниже.
  • Верстак CAM (ЧПУ) используется для создания инструкций G-кода. А 0.21 и ниже этот верстак назывался "Path".
  • Верстак Draft (Набросок) содержит 2D-инструменты и основные операции 2D и 3D САПР (CAD).
  • Верстак FEM (МКЭ) обеспечивает рабочий процесс анализа Методом Конечных Элементов (МКЭ).
  • Верстак Inspection (Инспектирование) создан для того, чтобы предоставить вам специальные инструменты для изучения фигур. Все ещё находится на ранних стадиях разработки.
  • Верстак Material (Материал) работает с системой материалов FreeCAD. представлено в версии 1.0
  • Верстак Mesh (Сетки) для работы с триангулярными (разбитыми на треугольники) сетками.
  • Верстак OpenSCAD обеспечивает совместимость с OpenSCAD и восстановление истории построения модели конструктивной твердотельной геометрии (CSG).
  • Верстак Part (Деталь) для работы с геометрическими примитивами и булевыми операциями.
  • Верстак Part Design (Проектная Деталь) для построения форм Детали из эскизов.
  • Верстак Points (Точки) для работы с облаками точек.
  • Верстак Reverse Engineering (Обратное Проектирование) предназначен для предоставления специальных инструментов для преобразования фигур/твёрдых тел/схем в параметрические компоненты, совместимые с FreeCAD.
  • Верстак Robot (Робот) для изучения движений робота. В настоящее время не обслуживается.
  • Верстак Sketcher (Скетчер) для работы с геометрически-ограниченными эскизами.
  • Верстак Spreadsheet (Электронная таблица) для создания и работы с данными электронных таблиц.
  • Верстак Surface (Поверхность) предоставляет инструменты для создания и модификации поверхностей. Он похож на опцию Грань из рёбер Построителя Форм верстака Part.
  • Верстак TechDraw (Технический Чертёж) для создания технических чертежей из 3D-моделей. Это преемник Верстака Drawing (Чертёж).
  • Верстак Test Framework (Система тестирования) предназначен для отладки FreeCAD.

Устаревшие

После версии 0.21 следующие верстаки больше не входят в комплект программы:

  • Верстак Start позволяет быстро перейти к одному из самых популярных верстаков. На смену этому верстаку пришла стартовая страница, доступ к которой можно получить через Меню Помощь.
  • Верстак Web предоставляет окно браузера вместо 3D вида FreeCAD.

После версии 0.20 следующие верстаки больше не входят в комплект программы:

  • Верстак Drawing использовался для создания технических чертежей. Верстак TechDraw является его более продвинутой заменой.
  • Верстак Image использовался для работы с растровыми изображениями. Его функциональность была интегрирована в Стандартные инструменты. Смотри Стандартный Импорт и Стандартный ViewLoadImage (Вид Загрузить Изображение).
  • Для трассировки лучей (рендеринга) использовался Верстак Raytracing. Вместо него следует использовать внешний Верстак Render.

Внешние верстаки

Верстаки FreeCAD легко программируются на Python, поэтому многие люди разрабатывают дополнительные верстаки за пределами основной области разработки FreeCAD.

На странице внешние верстаки перечислены все известные сообществу. Большинство из них легко устанавливаются из FreeCAD с помощью Менеджера дополнений, который находится в меню Инструменты→ Менеджер дополнений.
