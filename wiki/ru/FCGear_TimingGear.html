Описание

Распределительные Шестерни предназначены для того, чтобы распредвал и коленчатый вал вращали цепь ГРМ. Коленчатый вал вращается, чтобы перемещать поршни вверх и вниз внутри цилиндров. Распределительный вал вращается, чтобы впускные и выпускные клапаны в цилиндрах открывались и закрывались. Эти компоненты важны для правильной работы двигателя.

Распределительные шестерни соединены с зубчатым ремнём или цепью механизма.

Выше: Распределительное колесо

Применение

  1. Переключись на Верстак Шестерня.
  2. Есть несколько способов вызвать команду:
    • Нажми кнопку Распределительная Шестерня на панели инструментов.
    • Выбери в меню Шестерня → Распределительная Шестерня.
  3. Измени параметры шестерни в соответствии с требуемыми значениями (смотри Свойства).

Свойства

Объект Распределительная Шестерня (FCGear TimingGear) является производным от объекта Конструктивный элемент и наследует все его свойства. Он также имеет следующие дополнительные свойства:

Данные

base - базовые

  • Данныеheight - высота (Длина): По умолчанию 5 мм. Значение ширины шестерни.
  • Данныеteeth - число_зубьев (Целое число): По умолчанию 15. Количество зубьев.
  • Данныеtype (Перечисление): По умолчанию gt2. Тип распределительного колеса - шаг профиля для зубчатых ремней (смотри Примечания).

computed - вычисленные

  • Данныеh (Длина): (только для чтения) Радиальная высота зубьев.
  • Данныеoffset - смещение (Длина): (только для чтения) Смещение по оси X средней точки второй дуги.
  • Данныеpitch - шаг (Длина): (только для чтения) Шаг шестерни.
  • Данныеr0 (Длина): (только для чтения) Радиус первой дуги.
  • Данныеr1 (Длина): (только для чтения) Радиус второй дуги.
  • Данныеrs (Длина): (только для чтения) Радиус третьей дуги.
  • Данныеu (Длина): (только для чтения) Разность радиальных значений между шагом ... диаметром и головкой зубчатого колеса.

version - версия

Примечания

  • type - тип: Шаг зубчатых ремней (расстояние от центра до центра последовательных зубьев) указывается в типах. У GT2 шаг составляет 2 мм, у GT3 - 3 мм, у GT5 - 5 мм и т. д.

Полезные формулы

  • addendum diameter = pitch diameter + 2 * module
  • belt length = 2 * axle base + belt tooth pitch : 2 * (teeth 1 + 2) + belt tooth pitch² : 4 * pi² * axle base * (teeth 1 - 2)²
  • axle base = belt length : 4 - belt tooth pitch : 8 * (teeth 1+2) + ¼ * sqrt [belt length - ½ * belt tooth pitch * (teeth 1+2)² - 2 * belt tooth pitch² * (teeth 1+2)² : pi²]
