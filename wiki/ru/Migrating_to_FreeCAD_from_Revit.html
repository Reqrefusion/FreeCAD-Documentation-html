Эта страница поможет вам быстро начать работу с FreeCAD, если вы переходите с Autodesk Revit.

Налаживание отношений

FreeCAD — это многофункциональное приложение для 3D-моделирования. Хотя оно не является исключительно BIM-приложением, в нем есть специальный BIM Workbench, который входит в комплект при установке FreeCAD и содержит набор инструментов для работы с BIM.

Ключевые понятия

Несмотря на то, что вы сможете выполнять те же задачи и получать те же результаты, к которым привыкли в Revit, эти программы отличаются друг от друга, и вам следует изучить эти различия.

  • Формат файлов Revit (.RVT) является проприетарным, не имеет документации и не поддерживается FreeCAD (его поддержка потребовала бы огромных усилий, которые поглотили бы все ресурсы FreeCAD, но результат был бы весьма неполным). Вместо этого следует использовать открытые форматы, такие как IFC, которые довольно хорошо поддерживаются обоими приложениями. Семейства также можно экспортировать в ACIS/SAT, что поддерживается дополнением InventorLoader, которое можно установить через менеджер дополнений.
  • BIM-объекты, такие как окна, оборудование или мебель, обычно легко найти на популярных сайтах, например bimobject.com в формате IFC. Но этот формат не всегда гарантирует качество геометрии. В библиотеке grabcad представлен хороший выбор загружаемых объектов в формате STEP с высоким качеством геометрии.
  • В FreeCAD нет понятия семейства. Но есть много способов получить тот же результат, то есть несколько объектов с одинаковыми свойствами, связанных между собой (при изменении одного объекта изменяются все). Самый простой способ — клонирование. Клон — это копия объекта, но при изменении исходного объекта обновляются и его клоны. Но есть и другие, более тонкие способы сделать так, чтобы объекты имели общие свойства. Например, вы можете нарисовать 2D-профиль и создать несколько объектов на его основе.
  • Несмотря на удобство, вам не обязательно использовать инструменты BIM для создания BIM-объектов. Любые объекты FreeCAD или даже объекты, созданные в другой программе, могут стать BIM-объектами. Для этого нужно выбрать один объект и воспользоваться инструментом Component. Это также позволяет использовать другие инструменты, например из PartDesign Workbench, для создания сложных объектов, которые обычно трудно создать с помощью стандартных инструментов BIM.
  • Objects created with one BIM tool can have all supported BIM types. You only need to change their IFC Type property. So, for example, you can create a beam using the wall tool, just by changing its IFC type to "Beam" afterwards.
  • BIM tools follow the Draft Workbench philosophy: They can be created anywhere and anyhow in the 3D space. You just need to set your working plane correctly, then next BIM objects will be created on that plane.
  • There is no mandatory building structure (floors in Revit) in FreeCAD. You can group your objects using groups, or Building parts (which are commonly used to make levels, but can actually be used for any kind of grouping) to work the same way as in Revit, but you are the one who chooses the most appropriate way for your project.

Suggested adoption workflow

  • Don't try to migrate at once. Learning a new software is a complex task. FreeCAD is free, start with installing and exploring it, then when you can, use it to model a small part of a project, then as you gain knowledge and mastery, do more and more in FreeCAD and less in Revit.
  • Make sure you can always fall back to Revit if things go wrong: Export to IFC early and often, open the file in Revit, make sure everything is there.
  • Creating views and sheets works like in Revit, if you are using Draft views (the recommended way). Create shape views of your objects or levels, move them out of your model, annotate them, place everything in a group or Building part, then create TechDraw pages and add your Draft views to it.
