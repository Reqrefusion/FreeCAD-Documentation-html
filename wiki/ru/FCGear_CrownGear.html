Описание

Коронное колесо напоминает кольцеобразную изогнутую рейку. Угол давления плавно уменьшается от внешнего к внутреннему диаметру. Таким образом, переменная периферийная скорость коронного колеса компенсируется постоянной периферийной скоростью шестерни. Остроконечные внешние зубья и крутые боковые поверхности зубьев на внутреннем диаметре ограничивают полезную ширину зубьев. Коронные шестерни имеют такой же КПД, как и прямозубые. Одна коронная шестерня может приводить в движение несколько шестерён.

Известная область применения коронных зубчатых колес:

  • Приводы задней оси для автомобилей и мотоциклов
  • Поворотные механизмы для операционных столов
  • Угловые фрезерные головки
  • Системы приводных инструментов с несколькими осями и коронной шестернёй

Применение

  1. Переключись на Верстак Шестерня.
  2. Есть несколько способов вызвать команду:
    • Нажми кнопку Коронная Шестерня на панели инструментов.
    • Выбери в меню Шестерня → Коронная Шестерня.
  3. По умолчанию коронная шестерня отображается без зубьев. (представлено в версии 0.21)
  4. Измени параметры шестерни в соответствии с требуемыми значениями (смотри Свойства).
  5. Установи свойство Данныеpreview_mode - пред_просмотр в false, чтобы отобразить зубцы (смотри Примечания).

Свойства

Объект Коронная Шестерня (FCGear CrownGear) является производным от объекта Конструктивный элемент и наследует все его свойства. Он также имеет следующие дополнительные свойства:

Данные

accuracy - детализация

  • Данныеnum_profiles (Integer): Default is 4. Number of profiles used for loft.
  • Данныеpreview_mode (Bool): Default is true.

base

  • Данныеheight (Length): Default is 2 mm. Value for the tooth width.
  • Данныеmodule (Length): Default is 1 mm. Module is the ratio of the reference diameter of the gear divided by the number of teeth (see Notes).
  • Данныеother_teeth (Integer): Default is 15. Number of teeth of the construction gear (pinion) (see Notes).
  • Данныеteeth (Integer): Default is 15. Number of teeth.
  • Данныеthickness (Length): Default is 5 mm. Height from the tip of tooth to the lower side of the crown wheel.

involute

  • Данныеpressure_angle (Angle): Default is 20 ° (see Notes).

version

  • Данныеversion (String):

Notes

  • The Данныеpreview_mode property is set to true by default and when the gear is created you'll find this message in the report view:
    Gear module: Crown gear created, preview_mode = true for improved performance. Set preview_mode property to false when ready to cut teeth.
  • module: Using ISO (International Organization for Standardization) guidelines, Module size is designated as the unit representing gear tooth-sizes. Module (m): m = 1 (p = 3.1416), m = 2 (p = 6.2832), m = 4 (p = 12.566). If you multiply Module by Pi, you can obtain Pitch (p). Pitch is the distance between corresponding points on adjacent teeth.
  • other_teeth: Several pinions with the same number of teeth only can be used on one crown wheel.
  • pressure_parameter: Only change the parameter, if sufficient knowledge of the gear geometry is available.
  • The geometry of the crown gear is primarily determined by the spur pinion geometry (other_teeth).
  • Create spur gear with Involute gear. The number of teeth must be identical to the parameter other_teeth of the crown gear.
  • Adjustments for optimal running characteristics can be made with the parameters of involute gear.

Useful formulas

  • Inner diameter (4)
    • inner diamter = module (spur gear) * teeth (crown gear) * cos pressure_paramter (pinion) : cos pressure_parameter (crown gear)
  • Outer diameter (5)
    • outer diamter = inner diameter + 2x height (tooth width crown gear)
