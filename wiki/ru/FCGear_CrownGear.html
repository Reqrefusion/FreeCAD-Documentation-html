Описание

Коронное колесо напоминает кольцеобразную изогнутую рейку. Угол давления плавно уменьшается от внешнего к внутреннему диаметру. Таким образом, переменная периферийная скорость коронного колеса компенсируется постоянной периферийной скоростью шестерни. Остроконечные внешние зубья и крутые боковые поверхности зубьев на внутреннем диаметре ограничивают полезную ширину зубьев. Коронные шестерни имеют такой же КПД, как и прямозубые. Одна коронная шестерня может приводить в движение несколько шестерён.

Известная область применения коронных зубчатых колес:

  • Приводы задней оси для автомобилей и мотоциклов
  • Поворотные механизмы для операционных столов
  • Угловые фрезерные головки
  • Системы приводных инструментов с несколькими осями и коронной шестернёй

Выше: Коронное зубчатое колесо

Применение

  1. Переключись на Верстак Шестерня.
  2. Есть несколько способов вызвать команду:
    • Нажми кнопку Коронная Шестерня на панели инструментов.
    • Выбери в меню Шестерня → Коронная Шестерня.
  3. По умолчанию коронная шестерня отображается без зубьев. (представлено в версии 0.21)
  4. Измени параметры шестерни в соответствии с требуемыми значениями (смотри Свойства).
  5. Установи свойство Данныеpreview_mode - пред_просмотр в false, чтобы отобразить зубцы (смотри Примечания).

Свойства

Объект Коронная Шестерня (FCGear CrownGear) является производным от объекта Конструктивный элемент и наследует все его свойства. Он также имеет следующие дополнительные свойства:

Данные

accuracy - детализация

  • Данныеnumpoints - число_точек (Целое число): По умолчанию 4. Количество профилей, используемых для выдавливания.
  • Данныеpreview_mode - пред_просмотр (Булево): По умолчанию true.

base - базовые

  • Данныеheight - высота (Длина): По умолчанию2 мм. Значение ширины зубьев.
  • Данныеmodule - модуль (Длина): По умолчанию 1 мм. Модуль - это отношение базового диаметра шестерни к числу зубьев (смотри Примечания).
  • Данныеother_teeth - ответные_зубья (Целое число): По умолчанию 15. Число зубьев ведущей шестерни (ответной) (смотри Примечания).
  • Данныеnum_teeth - число_зубьев (Целое число): По умолчанию 15. Число зубьев.
  • Данныеthickness - толщина (Длина): По умолчанию 5 мм. Высота от верхушки зуба до нижней стороны коронного колеса.

involute - эвольвента

  • Данныеpressure_angle - угол_давления (Угол): По умолчанию 20 ° (смотри Примечания).

version - версия

Примечания

  • Свойство Данныеpreview_mode - пред_просмотр по умолчанию установлено в true, и при создании шестерни вы увидите это сообщение в окне просмотра отчёта:
    Модуль шестерни: Создана коронная шестерня, режим предварительного просмотра = true для улучшения производительности. Установите свойство preview_mode (пред_просмотр) на false, когда подготовитесь к нарезке зубьев.
  • module - модуль: В соответствии с рекомендациями ISO (Международная организация по стандартизации), размер модуля обозначается как единица измерения размеров зубьев шестерни. Модуль (m): m = 1 (p = 3,1416), m = 2 (p = 6,2832), m = 4 (p = 12,566). Если умножить модуль на Pi, можно получить Pitch - Шаг (p). Шаг - это расстояние между соответствующими точками на соседних зубьях.
  • other_teeth - ответные_зубья: На одном коронном колесе можно использовать несколько зубчатых колёс с одинаковым количеством зубьев.
  • pressure_parameter - угол_давления: Изменяйте этот параметр только при наличии достаточных знаний о геометрии шестерни.
  • Геометрия коронной шестерни в первую очередь определяется геометрией цилиндрической шестерни (other_teeth - ответные_зубья).
  • Создайте прямозубую шестерню с помощью команды Эвольвентная Шестерня. Количество зубьев должно совпадать с параметром other_teeth - ответные_зубья коронной шестерни.
  • С помощью параметров эвольвентной передачи можно настроить оптимальные ходовые характеристики.

Обзор комплекта коронной и цилиндрической шестерни

  • (1) Цилиндрическая шестерня
  • (2) Коронная шестерня
  • (3) Ширина зуба
  • (4) Внутренний диаметр
  • (5) Наружный диаметр

Полезные формулы

  • Внутренний диаметр (4)
    • внутр_диаметр = модуль (цилиндр_шестерни) * число_зубьев (коронной_шестерни) * cos угла_давления (цилиндр_шестерни) : cos угла_давления (коронной_шестерни)
  • Наружный диаметр (5)
    • наружный_диаметр = внутренний_диаметр + 2x высота (ширина зубьев коронной_шестерни)
