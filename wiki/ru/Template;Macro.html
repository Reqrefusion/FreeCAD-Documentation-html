Использование

Этот шаблон является основным инструментом для документирования макросов. Он должен быть вставлен в качестве первой строки на страницах документации, после <languages/> а также <translate> tags. Смотрите образец справа.

SomeMacro

Обязательные параметры: 

{{Macro/ru
|Name=SomeMacro
|Description=Этот макрос улучшает твердые формы во всех отношениях.
|Author=WikiUserName
|Download=Возможные файлы для загрузки, например значок панели инструментов ...
|Date=2018-11-20
|Version=2.0
|FCVersion= 0.16 to 0.19
}}

Другие параметры могут быть опущены или их значение может быть пустым.

Обязательные параметры

Параметры указываются с его значением после знака равенства, parameter=value.

Каждый параметр в списке разделен символом канала |. 

|... |Name=SomeMacro|Author=WikiUserName| ...

Параметры могут быть размещены в одной строке или каждый в отдельной строке. 

|...
|Name=SomeMacro
|Author=WikiUserName
| ...

Следующие параметры обязательны.

Name
Правильно прописное имя макроса для документирования.
Имя используется для извлечения графического значка Name.svg или Name.png из хранилища вики. Шаблон работает правильно, если значок не существует; в этом случае отображается общий значок .
Description
Описание макроса, это описание будет отображаться в менеджере дополнений.
Author

Автор макроса. Это имя связано со страницей пользователя в вики, то есть User:Author. Поскольку макросы не являются частью официального кода FreeCAD, пользователи должны направлять свои вопросы о них своим конкретным создателям. Автор макроса должен быть доступен на forum с тем же именем пользователя, что и в вики.

Download
Возможные файлы для загрузки, например значок панели инструментов и другие файлы. Дайте полную ссылку на объект ex: Download=[https://www.freecadweb.org/wiki/images/2/2c/Text-x-python.png Text-x-python.png]
Date
Все макросы должны иметь дату. Это полезно для указания времени последнего изменения макроса и предупреждения пользователя о возрасте кода. Очень старые макросы могут быть устаревшими и могут не работать с последней версией FreeCAD.
FCVersion
FreeCAD версия (ы) для запуска примера макроса: FCVersion=0.16 to 0.19

Необязательные параметры

Следующие параметры являются необязательными. Если они не включены в шаблон, они будут отображать значение по умолчанию.

Translate
Имя переведено на ваш язык и будет отображаться в заголовке окна. Например, Translate=Some_translated_name
Links
Другая справочная ссылка для примера макроса Видео, страница учебников, связанная с макросом.
Version
Версия макроса. Если значение не указано, по умолчанию используется значение 1.0. Это полезно для авторов, так как они могут задокументировать конкретные версии своих макросов.
Shortcut
Некоторые макросы могут содержать ярлыки для запуска определенных инструментов. Укажите кнопки быстрого доступа для нажатия с этим значением. Если значение не указано, по умолчанию None.
SeeAlso
Необязательные ссылки на другие связанные команды, например, SeeAlso=[[Some Tool|Tool]].
Icon
Файл изображения, отличный от Name.svg или Name.png для использования в качестве значка. Это может быть использовано для включения определенного значка, который не назван в честь команды, например, Icon=Some_other_name.svg
