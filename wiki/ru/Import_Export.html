Введение

Эта страница содержит описание различных форматов файлов, которые FreeCAD может импортировать и экспортировать. Для полноты картины в список включен и собственный формат FreeCAD. Для описания некоторых форматов есть отдельные соответствующие вики-страницы, на которые можно перейти, нажав на расширение в первой колонке.

Сопутствующая информация

Дополнительную информацию см. на следующих страницах:

  • Настройки Импорта и Экспорта
  • FreeCAD how to Import Export
