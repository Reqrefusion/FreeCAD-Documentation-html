Настройка интерфейса
Настройки Импорта Экспорта
Оглавление

Введение

Для запуска редактора настроек выберите опцию Правка → Настройки... в меню или (представлено в версии 1.1) воспользуйтесь сочетанием клавиш Ctrl+,. На macOS опция Настройки... появляется в меню FreeCAD (это место по умолчанию для настроек на Mac).

FreeCAD использует концепцию верстака, где каждый верстак отвечает за определённые задачи и функции. У верстака могут быть свои собственные настройки. Они появляются в редакторе Настроек, обычно в специальной группе, только если верстак был загружен в текущем сеансе FreeCAD. Некоторые верстаки добавляют поддержку определённых форматов импорта и экспорта. Если есть соответствующие настройки, они появятся на одной или нескольких дополнительных страницах в группе Импорт-Экспорт редактора настроек.

Чтобы сэкономить ресурсы, FreeCAD не загружает автоматически все доступные верстаки. Дополнительную информацию смотрите в разделе Верстаки. Список настроек, связанных с верстаком, смотри в Настройки, связанные с Верстаком.

Если ни один верстак не загружен, в редакторе настроек появляются семь групп: Основные, Отображение, Верстаки, Python, Менеджер дополнений, Импорт-Экспорт и Measure - Измерения. Под ними будут отображаться все загруженные рабочие столы с собственными настройками.

Нажатие кнопки Сброс в левом нижнем углу диалога Настройки открывает меню (представлено в версии 1.0) с опциями для сброса настроек к значениям по умолчанию. Вы можете сбросить текущую страницу, текущую группу или все настройки. В 0.21 и ниже кнопка сбрасывает все настройки.

Некоторые дополнительные настройки можно изменить только в Редакторе параметров. На странице Точных настроек перечислены некоторые из них.

Эта страница обновлена для версии 1.0.

Основные

Эта группа настроек состоит из семи страниц: Основные, Документ, Выделение, Кэш, Область уведомлений, Просмотр отчёта и Справка.

В 0.21 и ниже седьмая страница, Справка, доступна только в том случае, если было установлено дополнение Справка.

Основные

На этой вкладке вы можете настроить следующие параметры:

Параметр Описание
Язык интерфейса Язык интерфейса пользовательского приложения.
Система единиц измерения по умолчанию Система единиц для всех компонентов приложения. представлено в версии 1.0: Используется по умолчанию для системы единиц документа в новых документах. Система единиц активного документа переопределяет систему единиц по умолчанию, если не отмечен следующий параметр.
Игнорировать систему единиц проекта и использовать значение по умолчанию представлено в версии 1.0 Если флажок установлен, системы единиц измерения в проекте игнорируются и не отменяют систему единиц измерения по умолчанию.
Количество десятичных знаков Количество десятичных знаков, которые должны быть указаны для чисел и размеров.
Минимальная доля дюйма Минимальная дробь в дюймах, которая должна быть указана для чисел и размеров. Доступно только для систем единиц с дробными дюймами.
Формат чисел Определяет формат числа. Возможны следующие варианты:
  • Операционная система: Используется десятичный разделитель, определяемый операционной системой.
  • Выбранный язык: Используется десятичный разделитель выбранного языка интерфейса FreeCAD.
  • C/POSIX: Точка используется в качестве десятичного разделителя.
Замена десятичного разделителя Если флажок установлен, десятичный разделитель цифровой клавиатуры будет заменен разделителем, определенным настройкой Формат чисел. представлено в версии 0.21: Обратите внимание, что если этот параметр включен, то при использовании десятичного разделителя цифровой клавиатуры в консоли Python и редакторе макросов всегда будет печататься точка/период.
Тема Указывает тему для настройки внешнего вида пользовательского интерфейса.
Размер значков на панелях инструментов Определяет размер значков панели инструментов. Возможны следующие варианты:
  • Малый (16px)
  • Средний (24px)
  • Крупный (32px)
  • Очень крупный (48px)
Режим вида дерева и свойств Определяет, как будет отображаться дерево проекта в пользовательском интерфейсе (требуется перезапуск). Варианты:
  • Комбинированный: объединяет Дерево проекта и Редактор параметров в одну панель.
  • Независимый: Разделяет Дерево проекта и Редактор свойств на отдельные панели.
Размер списка последних файлов Указывает, сколько файлов должно быть перечислено в списке последних файлов.

По желанию из этого списка можно исключить импортированные файлы и включить экспортированные. См. раздел Тонкая настройка.

Включить фон из повторяющегося узора Если флажок установлен, то фон главного окна FreeCAD по умолчанию будет состоять из плиток данного изображения:

Этот параметр действует только в том случае, если Таблица стилей не выделенна.

Изображение можно изменить, добавив папки Gui/Images в папку:

%APPDATA%/FreeCAD (в Windows),

$HOME/.FreeCAD (в Linux) или

$HOME/Library/Preferences/FreeCAD (в macOS).

Поместите файл с именем background.png в папку Images и снимите/отметьте этот флажок, чтобы увидеть изменённый файл.

Включение экрана заставки при запуске Если флажок установлен, при запуске FreeCAD будет показана заставка.

Изображение заставки можно изменить, поместив файл с именем splash_image.png в папку Images, уже упомянутую в разделе Включить фон из повторяющегося узора.

Включить мигание курсора Если флажок установлен, текстовый курсор в консоли Python и редакторе Макросов будет мигать.
Активировать обработку наложения представлено в версии 1.0 Если флажок установлен, пристыкованные окна можно обрабатывать как прозрачные элементы.
Наборы настроек Перечисляет доступные пакеты настроек, а также позволяет импортировать, сохранять, управлять и изменять их.

Документ

На этой вкладке вы можете настроить следующие параметры:

Параметр Описание
Создавать новый документ при запуске Если отмечено, FreeCAD создаст новый документ при запуске.
Уровень сжатия сохраняемого документа Указывает уровень сжатия для файлов FCStd. Файлы FCStd являются файлами с ZIP-сжатием. Поэтому вы можете переименовать их суффикс .FCStd в .zip и открыть их с помощью программы архивации ZIP.
Отмены/Повторы Если пункт выбран, все изменения в документах сохраняются, чтобы их можно было отменить/повторить.
Максимум шагов отмен/повторов Указывает, сколько шагов отмены/повтора должно быть записано.
Разрешить отмену перерасчёта Если флажок установлен, повторные вычисления можно прервать, нажав Esc. Это может немного увеличить время повторных вычислений.
Запуск автоматического восстановления при запуске Если файл восстановления доступен, FreeCAD автоматически запустит восстановление файла при запуске. Таким образом, файлы могут быть восстановлены, если произошёл сбой.
Автосохранение каждые Определяет частоту записи файла восстановления.
Сохранить миниатюру в файл проекта при сохранении документа Если флажок установлен, то при сохранении документа будет сохранена миниатюра. Миниатюра будет отображаться, например, в списке последних файлов на стартовой странице. Для миниатюры можно выбрать Размер в диапазоне от 128×128 до 512×512 пикселей. Обычные размеры кратные двум: 128, 256, 512.
Добавить логотип программы в генерируемую миниатюру Если флажок установлен, к миниатюре будет добавлен логотип программы FreeCAD .
Максимальное количество файлов резервных копий, при пересохранении документа Если флажок установлен, при сохранении документа будут сохраняться резервные файлы. Можно указать количество сохраняемых резервных файлов. Резервные файлы - это ранее сохранённые версии документа.
Использовать дату и расширение FCBak Если флажок установлен, файлы резервных копий получат расширение .FCbak, а их имена получат суффикс даты в соответствии с указанным Форматом даты. Описание формата даты смотри в Шпаргалка по форматам даты и времени.

По умолчанию файл резервной копии будет иметь, например, такое имя TD-Cube.20200315-215654.FCBak. (оригинальное имя файла - TD-Cube.FCStd).

Разрешить идентичные метки (Label) в одном документе Если флажок установлен, объекты могут иметь одинаковые метки.
Отключите частичную загрузку внешних связанных объектов Если включена частичная загрузка внешних связанных объектов, то при автоматическом открытии связанного документа вместе с основным документом будут загружены только связанные объекты и их зависимости. Такой частично загруженный документ нельзя редактировать. Чтобы загрузить его полностью, дважды щёлкните значок документа в дереве проекта. Более подробное объяснение этой функции можно найти на странице документации Assembly3 (Сборка3).
Имя автора Все созданные документы получат указанное имя автора. Если вы не хотите включать эту информацию, оставьте поле автора пустым. Если отмечена опция Устанавливать при сохранении, то при сохранении файла в поле Автор последних изменений будет установлен указанный автор. Это поле можно просмотреть с помощью пункта меню Файл → Информация о документе.
Компания Все документы, которые будут созданы, получат указанное название компании.
Лицензия по умолчанию Указывает лицензию для новых документов. Для предопределенных лицензий автоматически устанавливается URL -адрес лицензии. Выберите Другое для пользовательской лицензии.
URL адрес лицензии Указывает URL-адрес, описывающий лицензию, выбранную в Лицензии по умолчанию.

Выделение

На этой вкладке вы можете настроить следующие параметры:

Параметр Описание
Включить предвыборку Если флажок установлен ,то включается предварительное выделение в 3D виде и для него будет использоваться указанный цвет. Предварительный выбор означает, что, например, края объектов будут подсвечиваться при наведении на них мыши, чтобы указать, что они могут быть выбраны.
Включить выбор Если флажок установлен, включается выделение в 3D-виде и для него используется указанный цвет. Если флажок не установлен, объекты могут быть выбраны только в Дереве проекта.

Обратите внимание, что у объектов, созданных при не отмеченной опции, свойство ВидSelectable - Выбираемый будет установлено в false. При таком значении они не могут быть выбраны в 3D-виде, даже если эта опция будет отмечена снова.

Укажите радиус (пикс.) Устанавливает область для выделения элементов в 3D виде. Большее значение (в пикселях) облегчает выбор элементов, но может сделать невозможным выбор некоторых мелких элементов.
Автопереключение в 3D-вид, содержащий выбранный элемент Включает режим Дерево проекта Синхронизировать вид.
Автоматически разворачивать элемент дерева при выделении соответствующего объекта в 3D-виде Включает режим Дерево проекта Синхронизировать выделение.
Выделить объект в 3D просмотре при наведении курсора на элемент дерева Включает Режим предварительного выбора в дереве проекта.
Записывать выделение в дереве для перехода назад/вперед с помощью кнопки навигации Включает режим Дерево проекта Записать выделение.
Добавить флажки для выделения в дереве документов Каждый элемент Дерева проекта будет иметь флажок. Это удобно, например, для выбора нескольких элементов на сенсорном экране.

Кэш

Эти настройки связаны с каталогом кэша, в котором FreeCAD хранит временные файлы.

На этой вкладке вы можете настроить следующие параметры:

Параметр Описание
Местоположение (только для чтения) Показывает путь к директории кэша. Используйте кнопку для просмотра каталога.
Проверять периодически при запуске программы Управляет частотой проверки размера каталога. Возможны следующие варианты:
  • Всегда
  • Ежедневно
  • Еженедельно
  • Ежемесячно
  • Ежегодно
  • Никогда
Ограничение размера кэша Указывает максимальный размер каталога. Ты получишь уведомление, если будет выполнена проверка и размер превысит это значение. Возможны следующие варианты:
  • 100 MB
  • 300 MB
  • 500 MB
  • 1 GB
  • 2 GB
  • 3 GB
Текущий размер кэша Показывает текущий размер каталога, если он доступен. Нажмите кнопку Проверить сейчас..., чтобы обновить значение.

Область уведомлений

Эти настройки управляют Областью Уведомлений и её сообщениями.

На этой странице (представлено в версии 0.21) вы можете определить следующие параметры:

Параметр Описание
Включить область уведомлений Если флажок установлен, область уведомлений будет отображаться в строке состояния.
Включить ненавязчивые уведомления Если флажок установлен, то ненавязчивые всплывающие уведомления будут появляться над строкой состояния.
Ошибки отладки Если флажок установлен, ошибки, предназначенные для разработчиков, будут отображаться в Области Уведомлений.
Отладочные предупреждения Если флажок установлен, предупреждения, предназначенные для разработчиков, будут отображаться в Области Уведомлений.
Максимальная длительность Максимальная продолжительность отображения уведомлений (если не нажимать на кнопки мыши).
Минимальная длительность Минимальная продолжительность (тоже).
Максимальное количество одновременных уведомлений Максимальное количество уведомлений, которые будут одновременно присутствовать в контейнере всплывающих уведомлений.
Ширина уведомлений Ширина всплывающего окна уведомления в пикселях.
Скрывать, когда другое окно активировано Если флажок установлен, открытые уведомления будут исчезать при активации другого окна.
Не показывать, когда неактивен Если флажок установлен, уведомления не будут появляться, если окно FreeCAD не является активным окном.
Максимальное количество уведомлений Ограничивает количество уведомлений, которые будут храниться в списке. Если 0, то лимит отсутствует..
Автоудаление уведомлений пользователя Если флажок установлен, уведомления будут удаляться из списка сообщений по истечении Максимальной длительности.

Просмотр отчёта

Эти настройки управляют поведением Просмотр отчёта. Эту панель можно открыть с помощью пункта меню Вид → Панели → Просмотр отчёта.

В Просмотре отчёта используются те же настройки шрифта, что и в Редакторе макросов.

На этой вкладке вы можете настроить следующие параметры:

Параметр Описание
Записывать обычные сообщения (normal) Если флажок установлен, обычные сообщения будут записываться. Они будут отображаться в Отчёте с цветом, установленным для Обычных сообщений.
Записывать сообщения журнала (log) Если флажок установлен, сообщения журнала будут записываться. Они будут отображаться в Отчёте с цветом, установленным для Сообщений журнала.
Записывать предупреждения (warnings) Если флажок установлен, предупреждения будут записываться. Они будут отображаться в Отчёте с цветом, установленным для Предупреждений.
Записывать сообщения об ошибках (error) Если флажок установлен, сообщения об ошибках будут записываться. Они будут отображаться в Отчёте с цветом, установленным для Ошибок.
Показывать отчёт при возникновении ошибки Если флажок установлен, то при возникновении ошибки будет автоматически отображаться Отчёт.
Показать отчёт при появлении предупреждения Если флажок установлен, то при появлении предупреждения будет автоматически отображаться Отчёт.
Показать отчёт при получении обычного сообщения Если флажок установлен, то Отчёт будет автоматически отображаться при регистрации обычного сообщения.
Показать отчёт при получении сообщений журнала Если флажок установлен, то Отчёт будет автоматически отображаться при регистрации сообщений журнала.
Включить код времени для каждой записи Если флажок установлен, каждое сообщение и предупреждение получит метку времени.
Обычные сообщения (normal) Устанавливает цвет шрифта для обычных сообщений.
Сообщения журнала (log) Устанавливает цвет шрифта для сообщений журнала.
Предупреждения (warnings) Устанавливает цвет шрифта для предупреждающих сообщений.
Сообщения об Ошибках (errors) Устанавливает цвет шрифта для сообщений об ошибках.
Перенаправлять внутренний вывод Python в отчёт Если флажок установлен, то внутренний вывод Python будет перенаправляться из консоли Python в отчёт.
Перенаправлять внутренние сообщения об ошибках Python в отчёт Если флажок установлен, то внутренние сообщения об ошибках Python будут перенаправляться из консоли Python в отчёт.

Использовать програмное обеспечение OpenGL Если этот флажок установлен, OpenGL будет использовать ЦП вместо графического процессора. Эта опция полезна для устранения неполадок с видеокартой и драйверами. Изменение этой опции требует перезапуска приложения.
Использовать OpenGL VBO (объекты буфера вершин) Если флажок установлен, то будут использоваться объекты буфера вершин (VBO). VBO - это функция OpenGL, которая предоставляет методы для загрузки данных вершин (положение, вектор нормали, цвет и т.д.) на видеокарту. VBO обеспечивает существенный выигрыш в производительности, потому что данные находятся в графической памяти, а не в системной памяти, и поэтому они могут отображаться непосредственно графическим процессором. Для получения дополнительной информации см. эту веб-страницу.
Кэш рендера "Кэш рендеринга" или "Ускорение рендеринга" более подробно описаны в FreeCAD assembly3 render-caching. Доступны следующие опции:
  • Авто: Coin3D самостоятельно определяет, где размещать кэш (по умолчанию).
  • Распределённый: вручную включается кэш для всех корневых узлов поставщиков представлений.
  • Централизованный: вручную отключается кэш во всех узлах всех поставщиков представлений, и кэш размещается только в корневом узле графа сцены. Это обеспечивает максимальную скорость рендеринга, но замедляет реакцию на любые изменения сцены.
Сглаживание Указывает, используется ли и какой тип многодискретного сглаживания используется.
Прозрачные объек Указывает тип рендеринга прозрачных объектов. Доступны следующие варианты::
  • Один проход: Рендеринг выполняется за один проход (по умолчанию). Это может привести к появлению треугольных артефактов. Если они появляются, для их устранения можно использовать тип Проход фона.
  • Проход фона: Рендеринг выполняется за два прохода. Полигоны задней грани отрисовываются в первом проходе, а полигоны передней грани — во втором.
Размер метки Указывает размер вершин (точек) в Верстаке Скетчер (Sketcher). Область, на которую можно щёлкнуть мышью, можно дополнительно увеличить, увеличив Радиус выбора.
Расстояние между глаз для стерео режима Указывает расстояние между глазами, используемое для стереопроекций. Указанное значение является коэффициентом, который будет умножен на размер ограничительной рамки 3D-объекта, который в данный момент отображается.
Цвет подсветки Если флажок установлен, то подсветка включена с заданным цветом. Подсветка используется для отображения задних сторон граней (обычно их не видно в твёрдых телах, если только вы не разрезаете их плоскостью отсечения или если грани не ориентированы правильно). Она активна только для объектов, у которых свойство Lighting - Освещение (на вкладке Вид) установлено в значение "One side - Одна сторона". Если эта опция отключена, задние стороны граней объектов в режиме освещения "Одна сторона" будут чёрными.

Связанный параметр Интенсивность определяет интенсивность подсветки.

Тип камеры Указывает тип проекции камеры. Доступны следующие варианты:
  • Прорисовка перспективной проекции: объекты будут отображаться в перспективной проекции.

Эта группа настроек (представлено в версии 0.21) состоит из трёх вкладок: Макрокоманда, Главный и Редактор.

Макрокоманда

На этой вкладке вы можете настроить следующие параметры:

Параметр Описание
Запуск макросов в локальном окружении Если этот флажок установлен, переменные, определённые макросами, создаются как локальные переменные, в противном случае — как глобальные переменные Python.
Куда записывать макросы Указывает путь к файлам макросов.
Записать команды интерфейса Если этот флажок установлен, записанные макросы будут также содержать команды пользовательского интерфейса.
Записывать как комментарий Если этот флажок установлен, записанные макросы будут также содержать команды пользовательского интерфейса, но в виде комментариев. Это полезно, если вы не хотите выполнять эти команды при запуске макроса, но хотите видеть, что было сделано во время записи.
Показывать команды скриптов в консоли Python Если этот флажок установлен, команды, выполняемые макроскриптами, отображаются в консоли Python. Эту консоль можно открыть с помощью пункта меню Вид → Панели → Консоль Python.
Размер списка недавних макросов Управляет количеством недавних макросов, отображаемых в меню.
Количество горячих клавиш Управляет количеством недавних макросов, которым динамически назначаются горячие клавиши.
Комбинация клавиш Управляет тем, какие комбинации клавиш используются для быстрого доступа, например, Ctrl+Shift+ создаёт комбинации быстрого доступа в виде Ctrl+Shift+1, Ctrl+Shift+2, и так далее.

Главный

Эти настройки контролируют поведение консоли Python. Эту консоль можно открыть с помощью пункта меню Вид → Панели → Консоль Python.

Обратите внимание, что настройки цвета и шрифта для консоли определяются на странице Редактора кода.

На этой вкладке вы можете настроить следующие параметры:

Параметр Описание
Включить перенос слов Если этот флажок установлен, слова будут переноситься, если они превышают доступное горизонтальное пространство в консоли.
Включить большой курсор в виде блока Если этот флажок установлен, курсор будет иметь форму блока.
Сохранить историю Если этот флажок установлен, история Python сохраняется между сессиями.
Интервал профилировщика Python (миллисекунды) представлено в версии 1.0 Интервал, с которым профилировщик запускается при выполнении кода Python (для обеспечения отзывчивости графического интерфейса). Установите значение 0, чтобы отключить.
Путь к внешнему исполняемому файлу Python (необязательно) Используется для установки пакетов с помощью pip и отладки с помощью debugpy. Автоматически определяется при необходимости, если не указано иное.

Редактор кода

Эти настройки контролируют поведение Редактора макросов. Этот редактор можно открыть с помощью пункта меню Макросы → Макрос... → Редактировать или опцией меню Создать.

Настройки цвета и шрифта также используются консолью Python. Настройки шрифта также используются режимом Просмотра Отчётов.

На этой вкладке вы можете настроить следующие параметры:

Параметр Описание
Показывать номера строк Если этот флажок установлен, строки кода будут пронумерованы.
Включить большой курсор в виде блока Если этот флажок установлен, курсор будет иметь форму блока.
Шаг табуляции Указывает растр табуляции (количество пробелов). Если, например, установлено значение 6, нажатие Tab приведёт к перемещению на 7, 13, 19 и т. д. символ, в зависимости от текущего положения курсора. Этот параметр используется только в том случае, если выбрана опция Символы табуляции.
Размер отступа Указывает, сколько пробелов будет вставлено при нажатии Tab. Этот параметр используется только в том случае, если выбрана опция Заменять на пробелы.
Символы табуляции Если этот параметр выбран, нажатие клавиши Tab вставит табулятор с размером, заданным параметром Шаг табуляции.
Заменять на пробелы Если этот параметр выбран, нажатие клавиши Tab вставит количество пробелов, заданное в Размер отступа.
Показать пункты Указывает тип кода, к которому будут применены настройки цвета и шрифта. Результат можно проверить в поле Предпросмотр.
Шрифт Указывает семейство шрифтов, которое должно использоваться для выбранного типа кода.
Размер Указывает размер шрифта, который должен использоваться для выбранного типа кода.
Цвет Указывает цвет, который должен использоваться для выбранного типа кода.

Настройки Менеджера дополнений

На этой вкладке вы можете настроить следующие параметры:

Параметр Описание
Автоматически проверять наличие обновления при запуске (требуется git) Если этот флажок установлен, менеджер дополнений будет проверять наличие обновлений при запуске. Для этого необходимо установить Git.
Скачать метаданные макроса (примерно 10 МБ) Если этот флажок установлен, метаданные макросов загружаются для отображения в основном списке дополнений диспетчера дополнений. Эти данные кэшируются локально.
Частота обновления кэша Управляет частотой обновления информации о доступности и метаданных дополнений, хранящихся в локальном кэше. Доступны следующие варианты:
  • Вручную (без автоматического обновления)
  • Ежедневно
  • Еженедельно
Скрыть аддоны без лицензии представлено в версии 1.0 Если этот флажок установлен, дополнения без лицензии не включаются в список.
Скрыть дополнения с лицензией Free/Libre, отличной от FSF представлено в версии 1.0 Если этот флажок установлен, дополнения с лицензией Free/Libre, не опубликованные Free Software Foundation, не включаются в список.
Скрыть дополнения с лицензией, не одобренной OSI представлено в версии 1.0 Если этот флажок установлен, дополнения с лицензией, не одобренной Open Source Initiative, не включаются в список.
Скрыть дополнения, помеченные как "Только Python 2" Если этот флажок установлен, дополнения, помеченные как "Только для Python 2", не включаются в список. Эти дополнения вряд ли будут работать в текущей версии FreeCAD.
Скрыть дополнения, помеченные "Устаревшие" Если этот флажок установлен, дополнения, помеченные как "Устаревшие", не включаются в список.
Скрыть дополнения, требующие более новой версии FreeCAD Если этот флажок установлен, дополнения, требующие более новой версии FreeCAD, не включаются в список.
Пользовательские репозитории Здесь можно указать пользовательские репозитории.
