Макрос Screen Wiki (Снимок для Вики)

Описание
Специальный макрос для пользователя Wiki. Этот макрос позволяет сохранить 3D-вид в нужном формате. 3D-вид или полное 3D-окно FreeCAD принимает нужные размеры. Поворот выбранного объекта или 3D-вида возможен, чтобы задать угол поворота, количество изображений рассчитывается автоматически, можно задать угол вылета и угол прилета. Для сборки изображений и создания анимированного файла необходимо использовать другую программу-например Gimp.

Макрос Screen Wiki Окно поворота

Применение

Параметры изображения

Размеры

  1. 400x200
  2. 600x400 (По умолчанию)
  3. 1024x768
  4. 320x240 (QVGA)
  5. 320x480 (HVGA)
  6. 400x300
  7. 480x360
  8. 640x480 (VGA)
  9. 768x576 (PAL)
  10. 800x600 (SVGA)
  11. 960x720
  12. 1024x768 (XGA)

Формат изображения

  1. 600 px Ширина (Length) (По умолчанию: 600 px)
  2. 400 px Высота (Height) (По умолчанию: 400 px)

Окно

  1. Window FC : Полное окно FreeCAD
  2. Screen 3D : 3D-вид FreeCAD

Цвет Фона

  1. Current (Текущий) (По умолчанию)
  2. Color (Цвет)
  3. Transparent (Прозрачный)
  4. Restore (Восстановить)

Команды

  1. Set Screen: Закрепленное окно
  2. Tile Screen: Летающее окно
  3. Save Image: Сохранить изображение, например: imageBox_000.png (число _000 увеличивается с каждым новым изображением)
  4. Follow: После сохранения первого изображения нажмите эту кнопку, чтобы сохранить следующее изображение с тем же именем. Количество сохраненных изображений увеличивается, например: imageBox_001.png, imageBox_002.png, imageBox_003.png, и т.д...
  5. New Image: Сохранить одно новое изображение без изменения счётчика
  6. Rotation: Доступ к меню поворота (название раздела "Image options" измените на "Rotation options")
  7. Quit: ___Screen_Wiki конец__________
  8. ToolBar: Уменьшить окно изображения в панели инструментов, при этом опция Rotation в этом режиме недоступна
    2. Кнопка Переключение Y/N мини-панели инструментов

Настройки поворота

Вращение вкл.

  1. 3D View: Весь вид поворачивается
  2. Object: Выбранный объект поворачивается

Оси

X: Вращение вокруг оси X
Y: Вращение вокруг оси Y
Z: Вращение вокруг оси Z
D: Вращение по направлению.
  • Чтобы воспользоваться этой опцией, сначала выберите объект, а затем - направляющую. Если D установлен, и если ни одно направление не выбрано, то направление будет Vector(0, 0, 0)

Поворот BoundBox в точке

  1. Object: Вращение относительно центра BoundBox выбранного объекта
  2. Sub Object: Вращение относительно центра BoundBox выбранного суб-объекта

Углы

  • Angle Rotation (Угол Поворота)
  1. -: Уменьшить значение на 10 градусов
  2. 0 Degrees  : Значение
  3. +: Увеличить значение на 10 градусов
  • Number images (Количество изображений)
  1. -: Уменьшить значение на 10 изображений
  2. 0 Images (+1)  : Значение
  3. +: Увеличить значение на 10 изображений
  • Angle Begin Rotation (Угол Начала Вращения)
  1. -: Уменьшить значение на 10 градусов
  2. 0 Degrees  : Значение: Угол начального поворота
  3. +: Увеличить значение на 10 градусов
  • Degrees Angle End Rotation (Угол в Градусах Окончания Вращения)
  1. -: Уменьшить значение на 10 градусов
  2. 360 Degrees : Значение: Угол окончания вращения
  3. +: Увеличить значение на 10 градусов

Команды

  • Delay between 2 images (Задержка между 2 изображениями)
  1. 0,00 Delay second  : Если возникает проблема с сохранением изображений из-за скорости, то добавьте задержку в X секунд.
  2. Reverse: Отмеченный, этот параметр отменяет поворот 3D-вида или объекта
  3. Point center: Визуализирует центр вращения, если точка видна, то она включается в изображение (PS: точка может быть скрыта объектом).
  4. Original position: Эта опция восстанавливает исходное положение 3D-вида или вращаемого объекта. Вместо этого 3D-вид или объект останутся в последней позиции поворота.
  5. Test Rot.: Тестирование поворота без сохранения изображений
  6. Save the animation: Сохранить анимацию
