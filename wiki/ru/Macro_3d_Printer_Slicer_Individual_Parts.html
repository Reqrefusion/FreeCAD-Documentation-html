Macro 3d Printer Slicer Individual Parts

Description
Экспорт в слайсинговое ПО для 3D-принтеров на основе подобного макроса от cae2100. Экспортирует один STL-файл для каждой видимой детали в той же папке, что и исходный файл разработки, а затем открывает его в слайсинговом ПО.
Когда этот код запускается, он экспортирует текущую открытую разработку в несколько STL-файлов, названных по меткам деталей, и открывает их в слайсинговом ПО, которое вы используете. Макрос ищет Cura в пути, но вы можете добавить любое другое слайсинговое ПО, изменив строку в макросе.

Описание

Этот код при запуске экспортирует видимые тела верхнего уровня (тела, находящиеся глубже в дереве, будут игнорироваться) текущего открытого проекта в отдельные STL-файлы и откроет их в используемом вами слайсере. По умолчанию макрос будет искать Cura, но вы можете поменять его на любой другой слайсер, изменив переменную SLICERAPP в исходном коде.

Лучше всего использовать его, создав ссылку на макрос на панели инструментов, и когда вы будете готовы нарезать объект, просто нажмите на него, и ваши объекты, как они отображаются на экране в FreeCAD, появятся в интерфейсе вашего ПО для нарезки, готовые к нарезке. Это также создаст несколько файлов STL с тем же именем файла проекта и меткой детали в той же директории, что и файл проекта.

Скрипт

The SLICERAPP variable can be changed to any slicing software of your choosing. If a specific object is not exported you might have to add the respective type to the doexport array.

import FreeCAD
import Mesh
import sys
import math
import os
import subprocess

SLICERAPP= "cura"		 			# Put your Slicer program here

# some fuctions
def getPlacement(quat,vect,obj):
  if quat[3] > -1  and quat[3] < 1:
    delta = math.acos(quat[3])*2.0
    scale = math.sin(delta/2)
    rx = quat[0]/scale
    ry = quat[1]/scale
    rz = quat[2]/scale
  else:
    delta = 0
    rx = 0
    ry = 0
    rz = 1
  info0 = "translation "+str(vect.x)+" "+str(vect.y)+" "+str(vect.z)
  info1 = "rotation "+str(rx)+" "+str(ry)+" "+str(rz)+" "+str(delta)
  return info0+" "+info1
# some definitions
placement = App.Placement(App.Vector(0,0,0),App.Rotation(0,0,0,1))
# user need to set this directory where slicing software is located
OutDir = FreeCAD.ActiveDocument.FileName.replace(".FCStd", "--")
visible_objs = []
# Get Objects in document
doc = App.ActiveDocument
objs = doc.Objects
stlFile = ""
stlFiles = [ SLICERAPP ]
# hide all
for obj in objs:
   print(obj.Label + "//" + obj.TypeId)
   print(len(obj.InList))
   if obj.ViewObject.isVisible() and hasattr(obj, 'Shape') and (len(obj.InList) <= 1):
      visible_objs.append(obj)
for obj in visible_objs:
  stlFile = OutDir+str(obj.Label)+".stl"
  Mesh.export([obj],stlFile)
  stlFiles.append(stlFile)
  print ("Exporting " + stlFile + "\n")
print ("Calling subprocess: " + str(stlFiles)+"\n")
subprocess.Popen(stlFiles)
