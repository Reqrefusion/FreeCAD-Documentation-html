Введение

Меню "Инструменты" это одно из 7-ми меню стандартного меню:

Файл     Правка     Вид     Инструменты     Макросы     Окна     Справка

Меню "Инструменты" предоставляет инструменты для отладки моделей, настройки поведения FreeCAD, а также вспомогательные инструменты.

Инструменты

В этом меню доступны следующие инструменты:

The order of the tools matches the version 1.1 menu.

  • Addon Manager: Opens the Addon Manager with which external workbenches, preference packs and macros can be installed.
  • Measure: Measures objects. introduced in 1.0
  • Clarify Selection: Lists all objects of different types located near the cursor to facilitate their selection. introduced in 1.1
  • Калькулятор единиц обеспечивает быстрое преобразование между системами изменений, например, метрической и дюймовой.
  • Load Image…: Loads an image. introduced in 0.21
  • Сохранить изображение...: Сохраняет снимок экрана текущего 3D вида.
  • Добавить текстовый документ: Создаёт объект, способный содержать текст.
  • Поворотный просмотр Вращает камеру вокруг модели в активном 3D Виде.
  • Инспектор сцены...: Открывает окно обзора узлов графической сцены активного 3D-вида.
  • Граф зависимостей...: Показывает граф зависимостей объектов.
  • Export Dependency Graph introduced in 0.21
  • Утилита распаковки файла проекта...: Помогает в восстановлении поврежденных файлов проекта.
  • Редактор параметров...: Открывает диалоговое окно, позволяющее изменять многие установки, которые управляют программой FreeCAD.
  • Настройка позволяет менять меню, быстрые клавиши и панели инструментов.
