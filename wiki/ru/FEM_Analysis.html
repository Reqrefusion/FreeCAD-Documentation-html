Описание

Команда Анализ FEM создает контейнер «Анализ», в котором размещаются все объекты анализа конечных элементов. Этот контейнер является обязательным. Для анализа механических объектов необходим как минимум один из следующих объектов (помимо сетки):

  • твердотельный материал,
  • ограничение краев объекта или ограничение по смещению объекта или ограничение для твердого тела.

Применение

  1. Существует несколько способов вызова команды:
    • Нажмите кнопку Новый анализ.
    • Выберите пункт меню Model → Новый анализ.
    • Используйте сочетание клавиш: S, затем A.
  2. Созданный контейнер Анализа устанавливается как активный.
  3. Другие объекты можно добавлять или удалять из контейнера Анализа с помощью перетаскивания.
  4. Чтобы добавить новые объекты FEM в документ, анализ должен быть активен. Двойной щелчок по контейнеру Анализа активирует его.

Свойства

  • ДанныеOutpuDir: указывает рабочий каталог анализа

Программирование

Большая часть кода здесь устарела в версии 0.17.

  • новый анализ
MechanicalAnalysis.makeMechanicalAnalysis( name )
  • добавление объекта в анализ
App.ActiveDocument.MechanicalAnalysis.Member = App.ActiveDocument.MechanicalAnalysis.Member + [ (object) ]
  • удаление объекта из анализа
member = App.ActiveDocument.MechanicalAnalysis.Member
member.remove( documentobject )
 App.ActiveDocument.MechanicalAnalysis.Member = member

Примеры: 

import MechanicalAnalysis
analysis = MechanicalAnalysis.makeMechanicalAnalysis("MechanicalAnalysis")
FemGui.setActiveAnalysis(analysis)

addobj = App.ActiveDocument.getObject("MechanicalMaterial")
App.ActiveDocument.MechanicalAnalysis.Member = App.ActiveDocument.MechanicalAnalysis.Member + [addobj]

removeobj = App.ActiveDocument.getObject("MechanicalMaterial")
member = App.ActiveDocument.MechanicalAnalysis.Member
member.remove(removeobj)
App.ActiveDocument.MechanicalAnalysis.Member = member
