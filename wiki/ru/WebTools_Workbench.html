Обзор

WebTools - это внешний верстак, содержащий ряд инструментов для взаимодействия с веб-сервисами непосредственно внутри FreeCAD.

Установка

Этот верстак можно установить из Менеджер дополнений. Для ручной установки смотрите Установка дополнительных верстаков.

Инструменты

  • Сервер BIM: Открывает окно BimServer.
  • Git: Управляет файлами с помощью GIT
  • Sketchfab: Экспортирует и загружает объекты в вашу учетную запись SketchFab.

Ссылки
