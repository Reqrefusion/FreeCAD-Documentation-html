Описание

The FCGear GearConnector command combines two gear items …

Применение

  1. Переключись на Верстак Шестерня.
  2. Выберите два соответствующих зубчатых элемента (Смотри Примечания):
    • Первый должен быть цилиндрическим или внутренним зубчатым колесом.
    • Вторым может быть цилиндрическая шестерня или зубчатая рейка.
  3. Есть несколько способов вызвать команду:
    • Нажми кнопку Объединить две шестерни на панели инструментов.
    • Выбери в меню Шестерня → Объединить две шестерни.
  4. Измени параметр передачи в соответствии с требуемыми условиями (смотри Свойства).

Свойства

See also: Property View.

An FCGear GearConnector object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

gear

  • Данныеangle1 (Angle): Default is 0°. Angle at which second gear is placed.
  • Данныеangle2 (Angle): Default is 0°. Angle at which second gear is placed.
  • Данныеmaster_gear (Link): Master gear.
  • Данныеmaster_gear_stationary (Bool): Default is True. Master gear position is fixed (does not orbit).
  • Данныеslave_gear (Link): Slave gear.
  • Данныеslave_gear_stationary (Bool): Default is True. Slave gear position is fixed (does not orbit).

version

  • Данныеversion (String): The freecad.gears-version.

Примечания

Порядок выбора определяет, сможет ли второй элемент катиться вокруг первого при изменении свойства Данныеangle1 - угол1.
