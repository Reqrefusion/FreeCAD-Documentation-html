Описание

Creates an array and resizes the items in the bounds of one or more hull curves. In this example, the orange base shape is rescaled in the bounds of the red and violet hullcurves. The curves do not have to be connected. The hullcurves should lie on or parallel to the XY- XZ- or YZ- plane.

Image from chbergmann's GitHub

Usage

  1. Select the input curves.
  2. There are several ways to invoke the command:
    • Press the Curved Array button
    • Select the Curved Shapes → Curved Array option from the menu

Notes

  • The first curve that you select for CurvedArray creation will be the item that is swept and resized in the bounds of the other selected curves.

Properties

Base

  • ДанныеBase: The object to make an array from
  • ДанныеHullcurves: List of one or more bounding curves
  • ДанныеAxis: Direction axis of the Base shape
  • ДанныеItems: Nr. of array items
  • ДанныеOffsetStart: Offset of the first part in Axis direction
  • ДанныеOffsetEnd: Offset of the last part from the end in opposite Axis direction
  • ДанныеTwist: Applies a rotation around Axis to the array items.
  • ДанныеSurface: Creates a surface over the array items
  • ДанныеSolid: Creates a solid if Base is a closed shape
