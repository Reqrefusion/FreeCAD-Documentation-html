Описание

Инструмент PartDesign Уклон создает угловой уклон выбранных поверхностей объекта. Он добавляет в документ объект Уклон с его соответствующим представлением в Дереве проекта.

Куб с примененным уклоном

Применение

Добавление уклона

  1. При желании можно выбрать тело, к которому нужно применить инструмент Уклон.
  2. Выберите одну или несколько граней тела.
  3. Существует несколько способов вызова инструмента:
    • Нажмите кнопку Уклон.
    • Выберите пункт меню Part Design → Особенности формы → Уклон.
  4. Если активного тела нет, а в документе два или более тел, откроется диалоговое окно Active Body Required, в котором вам будет предложено активировать одно из них. Если тело одно, оно будет активировано автоматически.
  5. Откроется панель задач с Параметрами Уклона. Дополнительную информацию см. в разделе Параметры.
  6. Нажмите кнопку OK для завершения.


На заметку:
  • Поскольку для данного элемента должна быть как минимум одна грань, последнюю оставшуюся грань в списке удалить нельзя.

Редактирование уклона

  1. Выполните одно из следующих действий:
    • Дважды щелкните по объекту Уклон в Дереве проекта.
    • Щелкните правой кнопкой мыши по объекту Уклон в Дереве проекта и выберите Редактировать Уклон из контекстного меню.
  2. Откроется панель задач с параметрами Уклона. Дополнительную информацию см. в разделе Параметры.
  3. Нажмите кнопку OK для завершения.

Опции

  • Добавить грань: Добавьте грани к выделенной области, нажав кнопку Добавить грань и выбрав несколько граней.
  • Удалить грань: Выберите способ удаления граней из выделенной области:
    • Выберите одну или несколько граней в списке и нажмите клавишу Del или щелкните правой кнопкой мыши список и выберите Удалить из контекстного меню.
    • Нажмите кнопку Удалить грань. Все ранее выбранные грани будут выделены фиолетовым цветом. Выберите каждую грань, которую нужно удалить.
  • Угол уклона: Установите угол уклона, введя значение или щелкнув стрелки вверх/вниз.
  • Нейтральная плоскость: Установите нейтральную плоскость, нажав кнопку Нейтральная плоскость и выбрав плоскость, размеры которой не должны изменяться.
  • Направление уклона: Установите направление уклона, нажав кнопку Направление уклона, затем выберите ребро. "Направление уклона" эффективно только в том случае, если задана нейтральная плоскость. Результаты могут быть непредсказуемыми.
  • Изменить направление уклона: Инвертируйте направление вытягивания, установив флажок Изменить направление уклона. Это позволит переключать угол наклона между положительным и отрицательным углами.

Примечания

  • Инструмент Уклон работает только с гранями, не соединенными касательно с другими гранями. Распространенная ошибка — попытка применить угол наклона к грани, к которой уже применено скругление. Чтобы исправить это, удалите скругление, примените необходимый угол наклона, а затем повторно примените скругление.
  • Как поясняется в OCC documentation, грань, выбранная для применения инструмента Уклон, сужается путем наклона на заданный угол наклона относительно линии пересечения нейтральной плоскости и выбранной грани. Направление уклона указывает сторону нейтральной плоскости, с которой удаляется материал, если угол наклона положительный, или добавляется, если угол наклона отрицательный. Это также подразумевает, что направление уклона должно быть перпендикулярно нейтральной плоскости и играет аналогичную роль, как и флажок Обратное направление уклона.

Свойства

См. также: Редактор свойств.

Объект PartDesign Draft является производным от объекта Part Feature и наследует все его свойства. Он также имеет следующие дополнительные свойства:

Данные

Base

  • ДанныеAngle (Angle): Не может быть отрицательным. По умолчанию: 1.5 °.
  • ДанныеReversed (Bool): По умолчанию: false.
  • ДанныеBase (LinkSub): Вложенная ссылка на список выбранных ребер и граней родительского объекта.
  • ДанныеSupport Transform (Bool): Include the base additive/subtractive shape when used in pattern features. If disabled, only the dressed part of the shape is used for patterning. По умолчанию: false.
  • Данные (hidden)Add Sub Shape (PartShape)
  • Данные (hidden)Base Feature (Link): Ссылка на родительскую функцию.
  • Данные (hidden)_ Body (LinkHidden): Ссылка на родительское тело.

Draft

  • ДанныеNeutral Plane (LinkSub): Дочерняя ссылка на список родительского объекта, содержащий нейтральную плоскость.
  • ДанныеPull Direction (LinkSub)

Part Design

  • ДанныеRefine (Bool): Refine shape (clean up redundant edges) after adding/subtracting. The default value is determined by the Automatically refine model after sketch-based operation preference. See PartDesign Preferences.

Пример

Выберите одну или несколько граней объекта перед запуском инструмента. Здесь выбраны 2 грани.

Отображение параметров Уклона в панели задач.

Добавлены две грани и применен уклон в 10 градусов. Нижняя плоскость осталась размерно стабильной, а уклон уменьшил верхнюю плоскость.

Нейтральная плоскость изменена на верхнюю грань. Теперь верхняя плоскость сохранила свои размерные характеристики, в то время как уклон увеличил нижнюю плоскость.

Направление уклона примененор к нижнему правому ребру, в результате чего уклон направлен ​​влево.

Установка флажка "Обратное направление" привела к возникновению внутреннего, а не внешнего уклона.
