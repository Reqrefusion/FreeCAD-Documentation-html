Описание

The FCGear LanternGear command creates a lantern gear wheel. This can be used as a cog/chain wheel in a roller chain transmission, or as a pinion for a roller rack and pinion gear.

Цевочная зубчатая передача - это особая форма циклоидальной зубчатой передачи, в которой окружность качения и делительная окружность имеют одинаковый размер. Кроме того, зубья большого колеса в коробке передач заменяются цилиндрами. Малое колесо получает циклоидное зацепление. В результате получается одностороннее точечное зацепление. Цевочные передачи могут быть только прямозубыми.

Поскольку их конструкция очень проста, они являются одними из самых древних видов зубчатых передач. Цевочные передачи используются, когда требуются большие передаточные отношения, например, в поворотных механизмах мельниц или кранов для перевалки древесины.

Цевочное зубчатое колесо с роликовыми цепями - это экономичная и надёжная альтернатива реечным приводам. Направляя натянутую цепь звёздочки по касательной к ней, линейное движение цепи преобразуется во вращательное движение звёздочки. И наоборот, линейное движение цепи может быть реализовано за счет вращательного движения звёздочки (мотоцикл/велосипед).

Применение

  1. Переключись на Верстак Шестерня.
  2. Есть несколько способов вызвать команду:
    • Нажми кнопку Цевочная Шестерня - Звёздочка на панели инструментов.
    • Выбери в меню Шестерня → Цевочная Шестерня - Звёздочка.
  3. Измени параметры шестерни в соответствии с требуемыми значениями (смотри Свойства).

Свойства

See also: Property View.

Объект Цевочная Шестерня - Звёздочка (FCGear LanternGear) является производным от объекта Конструктивный элемент и наследует все его свойства. Он также имеет следующие дополнительные свойства:

Данные

accuracy - детализация

  • Данныеnum_profiles - число_профилей (Целое число): По умолчанию 10. Обычно это значение не требует изменения.

base - базовые

  • Данныеbolt_radius - радиус_втулки (Длина): По умолчанию 1 мм. Диаметр цилиндра на вращающемся колесе, который выполняет функцию второго "зубчатого колеса".
  • Данныеheight - высота (Длина): По умолчанию 5 мм. Значение толщины зубчатого колеса.
  • Данныеmodule - модуль (Длина): По умолчанию 1 мм. Модуль - это отношение делительного диаметра шестерни к числу зубьев (смотри Примечания).

gear_parameter - параметры_звёздочки

  • Данныеteeth - число_зубьев (Целое число): По умолчанию 15. Число зубьев.

tolerance - допуск

  • Данныеhead - головка (С плавающей точкой): По умолчанию 0.

version - версия

  • Данныеversion - версия (Строка):

Примечания

  • module - модуль: В соответствии с рекомендациями ISO (Международная организация по стандартизации), размер модуля обозначается как единица измерения размеров зубьев шестерни. Модуль (m): m = 1 (p = 3,1416), m = 2 (p = 6,2832), m = 4 (p = 12,566). Если умножить модуль на Pi, можно получить Шаг (p). Шаг - это расстояние между соответствующими точками на соседних зубьях.

Полезные формулы

  • внешний_диаметр = модуль * (число_зубьев +2)
  • диаметр_шага = модуль * число_зубьев
  • межосевое_расстояние = диаметр_шага (колеса_1 + 2) : 2

Scripting

Use the power of Python to automate your gear modeling: 

import FreeCAD as App
import FreeCADGui as Gui
import freecad.gears.commands
gear = freecad.gears.commands.CreateLanternGear.create()
gear.num_teeth = 20
gear.height = 4
App.ActiveDocument.recompute()
Gui.SendMsgToActiveView("ViewFit")
