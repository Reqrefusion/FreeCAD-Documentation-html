Описание

Цевочная зубчатая передача - это особая форма циклоидальной зубчатой передачи, в которой окружность качения и делительная окружность имеют одинаковый размер. Кроме того, зубья большого колеса в коробке передач заменяются цилиндрами. Малое колесо получает циклоидное зацепление. В результате получается одностороннее точечное зацепление. Цевочные передачи могут быть только прямозубыми.

Поскольку их конструкция очень проста, они являются одними из самых древних видов зубчатых передач. Цевочные передачи используются, когда требуются большие передаточные отношения, например, в поворотных механизмах мельниц или кранов для перевалки древесины.

Цевочное зубчатое колесо с роликовыми цепями - это экономичная и надёжная альтернатива реечным приводам. Направляя натянутую цепь звёздочки по касательной к ней, линейное движение цепи преобразуется во вращательное движение звёздочки. И наоборот, линейное движение цепи может быть реализовано за счет вращательного движения звёздочки (мотоцикл/велосипед).

Применение

  1. Switch to the FCGear Workbench.
  2. There are several ways to invoke the command:
    • Press the Lantern Gear button in the toolbar.
    • Select the Gear → Lantern Gear option from the menu.
  3. Change the gear parameter to the required conditions (see Properties).

Properties

Data

An FCGear LanternGear object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

accuracy

  • Данныеnum_profiles (Integer): Default is 10. The value normally does not need to be changed.

base

  • Данныеbolt_radius (Length): Default is 1 mm. Diameter of the cylinder on the rotating disc which functions as a second "gear wheel".
  • Данныеheight (Length): Default is 5 mm. Value of the gear width.
  • Данныеmodule (Length): Default is 1 mm. Module is the ratio of the reference diameter of the gear divided by the number of teeth (see Notes).

gear_parameter

  • Данныеteeth (Integer): Default is 15. Number of teeth.

tolerance

  • Данныеhead (Float): Default is 0.

version

  • Данныеversion (String):

Notes

  • module: Using ISO (International Organization for Standardization) guidelines, Module size is designated as the unit representing gear tooth-sizes. Module (m): m = 1 (p = 3.1416), m = 2 (p = 6.2832), m = 4 (p = 12.566). If you multiply Module by Pi, you can obtain Pitch (p). Pitch is the distance between corresponding points on adjacent teeth.

Useful formulas

  • addendum diameter = module * (teeth +2)
  • pitch diameter = module * teeth
  • axle base = pitch diameter (lantern gear 1 + 2) : 2
