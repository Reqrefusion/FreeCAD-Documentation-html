3D вид во FreeCAD - это экземпляр Coin3D графа сцены (scenegraph), который формирует самое важное окно в интерфейсе. Coin3D - это библиотека, реализующая стандарт описания сцен OpenInventor 2.1.

Некоторые свойства представления, такие как цвет фона, стиль навигации мышью и шаг масштабирования, могут быть настроены в редакторе настроек.

3D-вид - это компонент интерфейса FreeCAD. По умолчанию он показывает небольшой виджет с координатными осями и навигационный куб также с координатными осями; сетка может быть отображена и настроена при загрузке Верстака Драфт.

Контекстное меню

Параметры контекстного меню 3D-вида зависят от выбранного объекта (объектов) и активного в данный момент верстака. Чтобы отобразить это меню, выберите один или несколько объектов, а затем щёлкните правой кнопкой мыши в 3D-виде.

Подробности

FreeCAD использует библиотеку Quarter для использования Coin3D в среде Qt.

Можно напрямую взаимодействовать с 3D-графом сцены из консоли Python, используя библиотеку Python Pivy.

Дополнительные сведения см. в документации для опытных пользователей:

  • Граф сцены (Scenegraph), описание библиотеки Coin3D.
  • Pivy, использование Coin3D из консоли Python.
  • Сторонние библиотеки используемые FreeCAD.
  • Coin3D C++ документация.
