Введение

FrameForge предназначен для создания рам и балок, а также для выполнения операций (косые срезы, обрезка) с такими профилями.

Нижеприведённое руководство также доступно на GitHub.

Руководство

Создание каркаса

Балки (профиля) прикрепляются к рёбрам (например, из эскиза) или Параметрическим Линиям. Для начала мы создадим простую раму.

  1. В новом файле перейди в верстак FrameForge.
  2. Создай эскиз и укажи ориентацию XY.

  3. Нарисуй в эскизе простой квадрат. Это будет наш каркас.

  4. Закрой режим редактирования Эскиза.

Создание рамы

  1. Запусти инструмент Create Profile (Создать Профиль).

  2. Выбери профиль из списков (Материал / Семейство / Размер). Размер можно изменить прямо под семейством, инструмент имеет множество предустановленных профилей, ты также можешь изменить их параметры.

  3. В 3D-виде выбери рёбра, к которым необходимо применить профиль.

  4. Нажми OK на панели задач. Теперь у тебя есть четыре профиля и твоя первая рама.

Переход к 3D

Мы можем создавать более сложные формы, и сделать это можно несколькими способами.

Дополнительные эскизы (часть 1)

Мы можем добавить в наш проект ещё несколько эскизов:

  1. Создай новый Эскиз.
  2. Выбери ту же ориентацию, что и ранее (XY).
  3. Нарисуй квадрат такого же размера и в том же месте, что и раньше.
  4. Измени положение эскиза, чтобы он находился на 400 мм выше первого.

  5. Теперь ты можешь снова использовать инструмент Create Profile (Создать Профиль), чтобы создать ещё одну квадратную рамку.

Параметрическая Линия

Ты можешь создавать параметрические линии, соединяя две вершины (точки). Эти линии также можно использовать для профилей.

  1. Чтобы увидеть эскизы, временно скрой объекты профиля с помощью клавиши Пробел.

  2. Выбери две вершины.

  3. Создай Parametric Line (Параметрическую Линию).

  4. Проделай то же самое с остальными 3-мя стойками рамы.
  5. Снова используй инструмент Create Profile (Создать Профиль), чтобы прикрепить профили к 4 линиям.
    1. Запустити инструмент Create Profile (Создать Профиль).
    2. Выбери нужный профиль.
    3. Выбери 4 параметрические линии.
    4. Нажми OK.

Дополнительные эскизы (часть 2)

Есть ещё один способ добавления эскизов, который позволяет делать более сложные вещи.

Иногда требуется добавить эскиз в определённое место и связать его с другим эскизом. Таким образом, при изменении первого эскиза второй эскиз будет меняться вместе с ним. Это невозможно сделать с помощью функции Position / Base Placement (Позиция/Размещение Базы), которая является абсолютной позицией. Для этого необходимо "Map (Сопоставить)" второй эскиз с первым.

  1. Создай новый Эскиз и установи его ориентацию на YZ. Для справки я добавил кружок к эскизу, чтобы ты смог увидеть, где он находится.

  2. Щёлкни на свойство Map Mode (Режим Карты):

  3. Ты можешь изменить Map Mode (Режим Карты), выбирая грани, вершины и рёбра. Здесь наш эскиз круга был переориентирован. Есть много вариантов.

  4. Затем ты можешь отредактировать эскиз и создать больше линий и рамок.

Скосы и углы

Как видишь, соединения пока не очень хорошие. Профили центрируются по каркасу и заканчиваются на концах рёбер.

Мы будем делать углы и скосы. Для этого есть два способа.

Через свойство Bevels (Скосы)

Это предпочтительный вариант для простых рамок.

  1. Давай скроем всё, кроме сделанной нами рамки.

  2. Выберите один из профилей и в Редакторе параметров найди Bevel Start/End Cut1/Cut2 (Скос Начало/Конец Срез1/Срез2).

  3. Существует 4 параметра (Start/End Cut1/Cut2). Они позволяют создавать скосы по двум осям, в начале или в конце профиля. Отрицательные углы используются, для компенсации направлений.
  4. Вы можете изменять свойства нескольких профилей одновременно.

С помощью инструмента Miter Ends (Скос Торцев)

  1. Давай рассмотрим другую базовую рамку.

  2. Сначала мы должны добавить смещения к существующим профилям. Смещения увеличивают размеры кромок. Ты можешь изменять профили по одному или все сразу.

  3. Сними выделение со всех объектов, затем выбери два соприкасающихся профиля. Тебе необходимо выбирать грани в 3D виде, а не объекты в Дереве проекта.

  4. Нажми на инструмент Create Miter Ends (Создать Скос Торцев), чтобы создать два подрезанных профиля.

  5. Повтори то же самое для других углов второй рамки.

С помощью инструмента Trim Profile (Обрезка Профиля)

  1. Когда все профили снова сделать видимыми, мы можем увидеть, что вертикальные профили вырезаны не так, как следовало бы.

  2. Запусти инструмент Trim Profile (Обрезка Профиля).

  3. Select the vertical profile first, add it as the Trimmed Object with the button.

  4. Select the Trimming Boundaries you want to cut with. Here I have rotated the view to select two bottom faces.

  5. You can change the cut type: straight or following the other profile.

  6. You also can add faces to trim the other side of the profile.

Organizing Objects

Part Container

I often use Part containers to group profiles, sketches, etc.

You should drag only one profile at a time into the container. I don't know why, but FreeCAD is not happy about a group drag. Sometime parts and profiles then jump out of the Part container.

Fusion

You can fuse profiles together. It allows to group objects.

Using profiles in PartDesign

  1. To use these profiles in PartDesign you need to create a fusion and then a Body.

  2. Drag and drop the fusion onto the Body.

  3. You now have a standard PartDesign Body and you can use PartDesign to do whatever you want. You can for example create holes.
