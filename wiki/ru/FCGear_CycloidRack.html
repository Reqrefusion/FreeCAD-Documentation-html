Описание

Циклоидная рейка слева направо: Прямозубая передача, косозубая передача, двойная косозубая передача

Применение

  1. Переключись на Верстак Шестерня.
  2. Есть несколько способов вызвать команду:
    • Нажми кнопку Циклоидная Рейка на панели инструментов.
    • Выбери в меню Шестерня → Циклоидная Рейка.
  3. Измени параметры шестерни в соответствии с требуемыми значениями (смотри Свойства).

Свойства

Объект Циклоидная Рейка (FCGear CycloidRack) является производным от объекта Конструктивный элемент и наследует все его свойства. Он также имеет следующие дополнительные свойства:

accuracy - детализация

  • Данныеnumpoints (Integer): Default is 15. Number of points for spline.

base

  • Данныеadd_endings (bool): If true (default), then the total length of the rack is teeth * pitch. If false, then the rack starts with a tooth-flank.
  • Данныеheight (Length): Default is 5 mm. Value of the gear width.
  • Данныеteeth (Integer): Default is 15. Number of teeth.
  • Данныеthickness (Length): Default is 5 mm. Thickness of the uncut part of the rack.

computed

  • Данныеtransverse_pitch (Length): (read-only) Pitch in the transverse plane.

cycloid

  • Данныеinner_diameter (Float): Default is 7.5. Diameter of the rolling circle of hypocycloid, normalized by the Данныеmodule (see Notes).
  • Данныеouter_diameter (Float): Default is 7.5. Diameter of the rolling circle of epicycloid, normalized by the Данныеmodule (see Notes).

fillets

  • Данныеhead_fillet (Float): Default is 0.
  • Данныеroot_fillet (Float): Default is 0.

helical

  • Данныеbeta (Angle): Default is 0 °. With the helix angle β a helical gear is created (positive value → rotation direction right, negative value → rotation direction left).
  • Данныеdouble_helix (Bool): Default is false, true creates a double helix gear (see Notes).

involute

  • Данныеmodule (Length): Default is 1 mm. For racks the module equals the pitch and so is the distance between corresponding points on adjacent teeth (see Notes).

precision

  • Данныеsimplified (Bool): Default is false. If true the rack is drawn with a constant number of teeth to avoid topological renaming.

tolerance

  • Данныеclearance (Float): Default is 0.25 (see Notes).
  • Данныеhead (Float): Default is 0. Additional length of the tip of the teeth, normalized by the Данныеmodule.

version

  • Данныеversion (String):
