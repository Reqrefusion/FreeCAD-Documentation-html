Quick Manuals — FreeCAD v1.0.2 (2025-08-05)

Create TechDraw Template — Inkscape v1.4.2 (2025-05-13)

  • Create a new SVG document
  • Delete all layers if present
  • Use EDIT -> XML-Editor (Shift + Control + T)
  • Add an attribute to the root (topmost) item <svg>:
name = xmlns:freecad
value = https://www.freecad.org/wiki/index.php?title=Svg_Namespace
  • Add a standard text that should be editable
  • Use the XML-Editor to add an attribute to each editable text item:
name = freecad:editable
value = SomeFieldIdentifier
  • Save the document as Plain (not Inkscape-) SVG
  • Copy the template.svg to the FreeCAD template folder:
<FreeCAD>\data\Mod\TechDraw\Templates
  • Open the template file in TechDraw
