Description

The SheetMetal UnattendedUnfold command unfolds a sheet metal object.

This command is not available by default, see Notes.

If the parent body of a selected planar face has been subject to unfolding before, this command will skip the menu in the task panel. Otherwise it will show an error message asking for either "set a Manual K-factor" or "use a Material Definition Sheet".

With the first use of the Unfold command the parent body's label received a suffix (such as _material_0.5din) and after that it is ready to be used with this command.

Usage

  1. Select one planar face of a sheet metal part.
  2. There are several ways to invoke the command:
    • Press the Unattended Unfold button.
    • Select the Sheet Metal → Unattended Unfold option from the menu.
    • Right-click in the Tree view or the 3D view and select the Sheet Metal → Unattended Unfold option from the context menu.
    • Use the keyboard shortcut: U.
  3. An Unfold object will be created.
  4. Optionally adjust the parameters in the Property editor.

Notes

To make this command available, first enable Engineering Mode in the preferences. Go to Edit → Preferences → SheetMetal → General Settings, and set Engineering UX Mode to Enabled. Changing this preference requires a restart of FreeCAD.

Properties

See also: Property editor.

A SheetMetal Unfold object is derived from a Part Feature object and inherits all its properties. It has no additional properties.

Limitations

See Unfold for limitations.
