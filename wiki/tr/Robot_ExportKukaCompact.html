Description

Açıklama

Bir robot program dosyasını dışa aktarın.
Dışa aktarma bir KUKA kompakt alt yordamı veya bir KUKA tam alt yordamı olabilir.

Kullanım

  1. Ağaç görünümünde robot ve yörüngeyi seçin.
  2. Üst menüden Robot → Yörünge Dışa aktar → Kuka kompakt alt yordam ...

veya

  1. Ağaç görünümünde robot ve yörüngeyi seçin.
  2. Üst menüden Robot → Yörünge Dışa aktar → Kuka tam alt yordam ....
  1. Select robot and trajectory in the Tree view.
  2. There are several ways to Export trajectory:
    • Choose Robot → Export trajectory → Kuka compact subroutine... from the top menu.
    • Choose Robot → Export trajectory → Kuka full subroutine... from the top menu.

Bir KRL dosyası (* .src) oluşturulacaktır.
