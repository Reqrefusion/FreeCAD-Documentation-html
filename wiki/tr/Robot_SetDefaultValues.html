Description

The Robot SetDefaultValues command sets the default values for speed, acceleration, and continuity for subsequent waypoint creation.

Usage

  1. Select the Robot → Set Default Values option from the menu.
  2. Specify the following values:
    • Speed
    • Continuity
    • Acceleration
