Description

The Curves SplitCurve splits the selected edge. This tool is part of the external workbench called Curves.

Usage

  1. Switch to the Curves workbench (install from Addon Manager is necessary, if not previously installed)
  2. To invoke the command, do one of the following:
    • Press the button
    • Use the Curves → Split curve
