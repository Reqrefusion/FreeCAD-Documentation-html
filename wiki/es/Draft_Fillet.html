Descripción

El comando redondeo crea un empalme, una esquina redondeada, o un chaflán, un borde recto, entre dos bordes seleccionados.

En 0.21 and below el comando solo funciona correctamente si ambos bordes seleccionados son rectos.

En 1.0 and below si los objetos seleccionados tienen múltiples aristas, se utilizará la primera. Esta puede no ser la arista que se seleccionó en la Vista 3D.

Varios redondeos y chaflanes creados entre dos bordes

Uso

  1. Seleccione dos aristas que se unan en un solo punto.
  2. Hay varias maneras de ejecutar el comando:
    • Pulse el botón Redondeo.
    • Entorno Borrador: Seleccione la opción Dibujar → Redondeo del menú.
    • Entorno de Trabajo BIM: Seleccione la opción Dibujar en 2D → Redondeo del menú.
    • Utilice el atajo de teclado: F seguido de I.
  3. Introduzca el radio de redondeo (Radio de Redondeo). Tenga en cuenta que el comando no se ejecutará si el radio es demasiado grande para los objetos de borde seleccionados.
  4. De manera optativa, marque la opción Eliminar Objetos Originales.
  5. Otra alternativa es marcar la opción Crear Chaflán.
  6. Si ha seleccionado una de las dos opciones anteriores: Haga clic en el cuadro de texto Radio de Redondeo.
  7. Pulse Intro.

Opciones

  • Pulse Esc o el botón Cerrar para cancelar el comando.

Notas

  • Un Redondeo no se puede editar ni está vinculado a las aristas que se usaron para crearlo.
  • Un Alambre con al menos tres puntos se puede redondear o chaflanar modificando su DatosRadio de redondeo o DatosTamaño Chaflán, respectivamente. Dado que las Líneas y los Alambres se pueden unir con los comandos correspondientes, como el comando Unión o el comando Actualizar, esto proporciona un método alternativo para crear redondeos y chaflanes.

Propiedades

Véase también: Vista de Propiedades.

Un objeto Draft Fillet se deriva de un Part Part2DObject y hereda todas sus propiedades. También tiene las siguientes propiedades adicionales:

Datos

Borrador

  • DatosFinal (Vector Distancia): (solo lectura) especifica el punto final del redondeo.
  • DatosRadio Redondeo (Longitud) (solo lectura):Radio con el que se creó el redondeo.
  • DatosLongitud (Longitud) (solo lectura): Especifica la longitud total del redondeo.
  • DatosComienzo (Vector Distancia) (solo lectura): Especifica el punto de inicio del redondeo.

Vista

Borrador

  • VistaTamaño Flecha (Longitud): Especifica el tamaño del símbolo que se muestra al final del filete.
  • VistaTipo Flecha (Enumeración): especifica el tipo de símbolo que se muestra al final del filete, que puede ser Punto, Circulo, Flecha, Tick o Tick-2.
  • VistaFinal Flecha (Boolenao): Especifica si se debe mostrar un símbolo al final del filete, para que pueda usarse como línea de anotación.
  • VistaPatrón (Enumeración): no se utiliza.
  • VistaTamaño Patrón (Float): no se utiliza.

Programación de Scripts

Véase también: Documentación de la API autogenerada y Conceptos Básicos de scripting en FreeCAD.

Para crear un redondeo, utilice el método make_filet del módulo Draft: 

fillet = make_fillet([edge1, edge2], radius=100, chamfer=False, delete=False)
  • Crea un objeto Fillet entre los objetos de borde edge1 y edge2, utilizando radius para la curvatura.
  • Si chamfer es True, creará un borde recto en lugar de uno redondeado.
  • Si delete es True, eliminará los objetos edge1 y edge2, dejando solo el nuevo objeto.

Ejemplo: 

import FreeCAD as App
import Draft

doc = App.newDocument()

p1 = App.Vector(0, 0, 0)
p2 = App.Vector(1000, 1000, 0)
p3 = App.Vector(2000, 0, 0)

edge1 = Draft.make_line(p1, p2)
edge2 = Draft.make_line(p2, p3)

doc.recompute()

fillet = Draft.make_fillet([edge1, edge2], radius=500)

doc.recompute()
