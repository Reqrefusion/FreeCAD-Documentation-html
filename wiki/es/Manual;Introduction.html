FreeCAD (https://www.freecad.org) es una herramienta de modelado paramétrico 3D de código abierto y gratuita, para diseño asistido por computadora (CAD), que se utiliza para crear objetos que van desde diseños simples hasta proyectos complejos que consisten en extensos ensamblajes de numerosas piezas. Gracias a su naturaleza de código abierto, FreeCAD se puede descargar, distribuir, modificar y usar libremente tanto para fines personales como comerciales. El código fuente se publica en abierto bajo la licencia [1]. Esto convierte a FreeCAD en una opción atractiva no solo para usuarios con poca experiencia, como aficionados a la impresión 3D, sino también para usuarios experimentados que busquen probar una herramienta paramétrica diferente.

El software se lanzó inicialmente en 2002 cuando estaba en una fase de desarrollo bastante elemental. Desde entonces, gracias a las continuas contribuciones de una dedicada comunidad de desarrolladores y usuarios, FreeCAD ha crecido lenta pero constantemente. El apasionado esfuerzo de numerosos colaboradores, no solo ha mejorado las capacidades de FreeCAD como herramienta CAD, sino que también ha propiciado el desarrollo de funcionalidades adicionales en distintos ámbitos estrechamente vinculados al paquete base de FreeCAD. Algunos ejemplos destacados incluyen el entorno de trabajo FEM, que permite realizar sencillos análisis de elementos finitos, así como el entorno de trabajo BIM, que ofrece capacidades de modelado de información para la construcción (BIM).

Este manual de usuario se está redactando y actualizando para la versión 1.0 de FreeCAD. Su objetivo es servir de guía para aquellas personas que deseen empezar a utilizar FreeCAD en sus proyectos. No se requieren conocimientos previos extensos sobre ningún software CAD, y el contenido de este manual está organizado de manera sencilla y completa, dividido en varias áreas de aplicación. La guía incluye instrucciones sobre la instalación de FreeCAD, el uso de sus bancos de trabajo más destacados y se amplia a temas más complejos, como la programación con secuencias de comandos en Python. Se ha hecho un gran esfuerzo para presentar la información de forma organizada, paso a paso y con imágenes. Esta estructura garantiza que tanto los usuarios novatos de CAD como las personas con más experiencia puedan beneficiarse del estudio de este manual. El contenido de este manual se publica bajo la licencia Creative Commons 4.0 y puede utilizarse, descargarse, copiarse y modificarse libremente. El manual de FreeCAD se mejora continuamente gracias a las experiencias y contribuciones de los usuarios. Hay disponible un script en Python para convertir la última versión de este manual a formatos PDF o EPUB para su lectura sin conexión.

Otro recurso valioso es la wiki de FreeCAD, donde numerosos colaboradores han añadido información útil que cubre todos los aspectos del software. Debido a la gran cantidad de información disponible navegar y utilizar la wiki puede resultar algo complicado y abrumador, especialmente para los nuevos usuarios. Por este motivo, se recomienda estudiar primero el manual de usuario, que ofrece una introducción más sencilla y fluida, antes de profundizar en los detalles más extensos disponibles en la wiki. Por último, los foros de FreeCAD también son muy útiles y proporcionan apoyo e información adicionales.
