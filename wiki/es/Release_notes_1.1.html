FreeCAD 1.1 fue lanzado el 24 de Marzo del 2026, consíguelo desde la página Descarga. Este es un resumen de las nuevas características y cambios.

Las notas de lanzamiento de versiones anteriores de FreeCAD se pueden encontrar en la Lista de características.

Generalidades

  • Soporte mejorado para Wayland. Solicitud de extracción n.° 21917, Solicitud de extracción n.° 23768 y Solicitud de extracción n.° 23946

Interfaz de usuario

Se agregó iluminación de tres puntos para mejorar la representación de modelos 3D.

Solicitud de extracción n.° 19397

Se agregó una barra de búsqueda al Editor de Preferencias.

Pull request #22002

Se implementó un nuevo Editor de Temas y un sistema de tokens de temas para permitir una mejor personalización de hojas de estilo.

Pull request #20668 y Pull request #20528

Un diálogo consolidado y mejorado para Agregar Propiedades que soporta expresiones, enumeraciones y unidades. Un Editor de Expresiones optimizado con mejor autocompletado, comportamiento de redimensionado y entrada de VarSet. Pull request #22719, Pull request #23426, Pull request #22964 yPull request #22944.

Otras mejoras de la interfaz de usuario

  • Se agregó un atajo por defecto para Std DlgPreferences. Pull request #15536
  • Se mejoró la página de preferencias del área de notificaciones. Pull request #15207

  • Se agregó la propiedad de Vista Mostrar Plano para visualizar el plano en el que un objeto 2D está basado. Pull request #18910

  • Se agregó la propiedad de Vista Mostrar Ubicación para visualizar la ubicación del sistema de coordenadas en el origen de un objeto. Pull request #19671
  • Se agregó el estilo de navegación de Solidworks. Pull request #19568
  • Las animaciones del Cubo de Navegación acumulan la rotación dependiendo de la frecuencia con que se clica en los botones planos. Pull request #19719

  • La lista por defecto de los Escritorios de Trabajo ahora es más corta y los escritorios menos usados están desactivados por defecto. Esto incluye los escritorios Inspección, OpenSCAD y Test Framework. Pull request #23034

Solicitud de extracción n.° 18332

384px
Haz clic en la imagen si la animación no se inicia. 		La herramienta 24px Transformación se ha renovado y ahora permite entradas precisas además de arrastrar en la vista 3D. Es posible alinear el arrastrador interactivo con cualquier elemento del documento y transformar el objeto en el sistema de coordenadas local (U, V, W) del arrastrador, o en el sistema de coordenadas global del documento.
