Descripción

La herramienta Superficie de Relleno Geométrico crea una superficie paramétrica a partir de dos, tres o cuatro bordes de contorno, tratando de crear una transición suave entre ellos.

Template:Leyenda

Uso

  1. Existen varias formas de usar la herramienta:
    • Pulse el botón Rellenar Curvas de Contorno.
    • Seleccione la opción Superficie → Rellenar Curvas de Contorno del menú.
  1. Seleccione los bordes en la Vista 3D. Los bordes deben estar conectados y formar un contorno cerrado.
  1. Pulse Aceptar.

Nota: Una vez creada, no es posible aplicar restricciones adicionales a la superficie creada.

Opciones

Tipo de Relleno: Estirada, Coons, or Curvada.

Propiedades

Un objeto Surface GeomFillSurface (clase Surface::GeomFillSurface) se deriva de la Part Feature básica (clase Part::Feature, a través de la subclase Part::Spline), por lo tanto, comparte todas las propiedades de esta última.

Además de las propiedades descritas en Características de Pieza, el objeto Surface GeomFillSurface tiene las siguientes propiedades en la Vista de Propiedades.

Datos

Base

  • DatosTipo de Relleno (Enumeración): El algoritmo de relleno aplicado; Estirado, el estilo con los parches más planos; Coons, un estilo redondeado con menos profundidad que Curvado; Curvado, el estilo con los parches más redondeados.
  • DatosBoundary List (LinkSubList): una lista de aristas que se utilizarán para construir la superficie.
  • Datos ((hidden))Reversed List (BoolList):

Vistas

Base

  • VistaControl Points (Bool): por defecto es false; si se establece en true, mostrará una superposición con los puntos de control de la superficie.

Torsión de la superficie

La forma de la superficie depende de la dirección de los bordes elegidos; si se seleccionan bordes y el resultado es una superficie que se "torsiona" sobre sí misma, es posible que uno de los bordes necesite su lista de vértices en orden inverso. Una superficie que se retuerce sobre sí misma probablemente tendrá autointersecciones y, por lo tanto, será una Forma no válida; esto se puede verificar con Verificar Geometría.

Por ejemplo, si dos curvas tienen los puntos 

curve1 = [a, b, c, d]
curve2 = [e, f, g]

y la superficie resultante después de usar Relleno Geométrico de Superficie o Secciones de Superficie es una superficie retorcida, puede crear una tercera curva que sea igual a una de las dos curvas originales pero con una lista de puntos invertida.

Either 

curve1 = [a, b, c, d]
curve3 = [g, f, e]

or 

curve3 = [d, c, b, a]
curve2 = [e, f, g]

should work to generate a surface that doesn't twist.

In practical terms this means that all edges used to generate a surface should be created preferably in the same clockwise or anti-clockwise direction. Following this simple rule usually guarantees that the surface will follow the smoothest direction and won't twist.

When the surface's VistaLighting property is One side, a face will be painted completely black if its normal direction points into the 3D View (away from the current viewer), indicating a flipped face with respect to the other colored faces.

Left: the boundary edges are oriented in the same direction, and thus the generated surface is smooth. Right: the boundary edges have opposite directions, and thus the generated surface twists on itself, resulting in self-intersections.

Scripting

See also: FreeCAD Scripting Basics.

The Surface GeomFillSurface tool can be used in macros and from the Python Console by adding the Surface::GeomFillSurface object.

  • The edges to be used to define the surface must be assigned as a LinkSubList to the BoundaryList property of the object.
  • The type of algorithm must be assigned like a string to the FillType property.
  • All objects with edges need to be computed before they can be used as input for the properties of the GeomFillSurface object.
import FreeCAD as App
import Draft

doc = App.newDocument()

a = App.Vector(-140, -100, 0)
b = App.Vector(175, -108, 0)
c = App.Vector(200, 101, 0)
d = App.Vector(-135, 107, 70)

points1 = [a, App.Vector(-55, -91, 65), App.Vector(35, -85, -5), b]
obj1 = Draft.make_bspline(points1)

points2 = [b, App.Vector(217, -45, 55), App.Vector(217, 35, -15), c]
obj2 = Draft.make_bspline(points2)

points3 = [c, App.Vector(33, 121, 55), App.Vector(0, 91, 15), App.Vector(-80, 121, -40), d]
obj3 = Draft.make_bspline(points3)

points4 = [d, App.Vector(-140, 0, 45), a]
obj4 = Draft.make_bspline(points4)
doc.recompute()

surf = doc.addObject("Surface::GeomFillSurface", "Surface")
surf.BoundaryList = [(obj1, "Edge1"),
                     (obj2, "Edge1"),
                     (obj3, "Edge1"),
                     (obj4, "Edge1")]
doc.recompute()
