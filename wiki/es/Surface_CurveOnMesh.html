Descripción

La herramienta Superficie sobre Malla crea segmentos de spline aproximados sobre una Malla seleccionada.

Si el objeto no es una Malla, sino una Forma o superficie paramétrica, primero debe convertirse en una malla usando Mesh FromPartShape.

Estos bordes creados sobre la malla pueden usarse posteriormente para recrear la superficie de forma paramétrica mediante el uso de herramientas como Surface GeomFillSurface y Surface Sections.

Arriba a la izquierda: objeto de malla con puntos seleccionados en la superficie. Arriba a la derecha: splines creadas al seleccionar varios puntos en la malla. Abajo a la izquierda: una superficie paramétrica reconstruida a partir de las splines aproximadas, utilizando Secciones de Superficie.

Uso

  1. Seleccione una Malla
  2. Hay varias maneras de usar la herramienta:
    • Pulse el botón Curva sobre Malla
    • Seleccione la opción Superficie → Curva sobre Malla del menú.
  3. Pulse el botón Iniciar.
  4. Con el puntero del ratón, seleccione puntos en la superficie de la malla en la Vista 3D; seleccione tantos puntos como sean necesarios para dibujar una línea de vista previa suave.
  5. Cuando haya añadido suficientes puntos, haga clic con el botón derecho en la Vista 3D para abrir el menú contextual y seleccione Crear. Dependiendo de la suavidad de la malla original, se creará una o varias splines en la Vista de Árbol.
  6. Repita la secuencia Template:Botón → Seleccionar → Crear para añadir más splines aproximadas.
  7. La nueva spline se creará y aparecerá en la Vista de Árbol inmediatamente después de seleccionar Crear; el Panel_de_Tareas permanecerá activo.
  8. Pulse Cerrar para cerrar el Panel de Tareas y finalizar el comando por completo.

Tras pulsar Comenzar, el menú contextual (clic derecho) en la Vista 3D muestra varias opciones junto a Crear.

  • Cerrar Cable: Si se han seleccionado al menos tres puntos, esta opción permitirá unir el último punto con el primero mediante una línea.
  • Limpiar: Borrará los puntos provisionales seleccionados en la malla y permitirá seleccionar otros nuevos.
  • Cancelar: Borrará los puntos provisionales seleccionados y detendrá la selección. Pulse Start de nuevo para seleccionar puntos otra vez.

Opciones

(Editor: esta información debe ser verificada)

Sección Cable:

  • Ajustar Tolerancias a Vértices: por defecto, 10 px; indica la distancia mínima entre dos puntos al seleccionarlos con el puntero.
  • Ajustar Umbral: por defecto, 45º; indica la desviación angular entre dos puntos de la malla necesaria para crear una nueva spline en lugar de extender la anterior.

Aproximación de spline: Si está marcada la casilla checked, se crearán objetos spline; de ​​lo contrario, se crearán objetos de línea recta simples (polilíneas).

  • Tolerancia a la malla: su valor predeterminado es 0.2. Este parámetro tiene en cuenta las imperfecciones de la malla. Cuanto menor sea este número, mayor será la precisión con la que se considerará la malla, especialmente si es muy fina.
  • Continuidad: Su valor predeterminado es C2. Determina la continuidad de la spline; puede ser C0 (tangente), C1 (curvatura) y C3 (curvatura de aceleración).
  • Grado Máximo de Curva: por defecto es 5. Determina el grado máximo de la spline para aproximar la superficie; puede ser un valor de 1 a 8.

Propiedades

Si Aproximación de Spline no está marcada, la herramienta Curva sobre Malla crea una Característica de Pieza básica.

If Spline approximation is checked, the Curve on mesh tool creates a Part Spline object (Part::Spline class) which is derived from the basic Part Feature (Part::Feature class), therefore it shares all the latter's properties.

In addition to the properties described in Part Feature, the Part Spline object has the following properties in the Property View.

View

Base

  • VistaControl Points (Bool): it defaults to false; if set to true, it will show an overlay with the control points of the surface.
