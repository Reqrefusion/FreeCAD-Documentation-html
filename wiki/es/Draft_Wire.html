Descripción

El comando Borrador: Cable Crea una polilínea, una secuencia de varios segmentos de línea conectados. El comando también se puede usar para Unir Líneas y Cables

Las esquinas de un alambre de dibujo se pueden redondear o achaflanar modificando su propiedad DatosRadio de Redondeo o DatosTamaño del Chaflán, respectivamente. También es posible subdividir los bordes de un alambre de dibujo modificando su propiedad DatosSubdivisiones.

Cable definido por múltiples puntos

Crear

Uso

Véase también: Bandeja del Borrador, Ajustar y Restringir.

  1. Existen varias formas de ejecutar el comando:
    • Pulse el botón Polilínea.
    • Entorno de Trabajo Borrador: Seleccione la opción Dibujar → Polilínea del menú.
    • Entorno de Trabajo BIM: Seleccione la opción Dibujar en 2D → Polilínea del menú.
    • Utilice el atajo de teclado: P seguido de L.
  2. Se abrirá el panel de tareas Polilínea. Consulte Opciones para obtener más información.
  3. Seleccione el primer punto en la Vista 3D o introduzca las coordenadas y pulse el botón Introducir Punto.
  4. Seleccione puntos adicionales en la Vista 3D o introduzca las coordenadas y pulse el botón Introducir Punto.
  5. Pulse Esc o el botón Cerrar para finalizar el comando.

Opciones

Los atajos de teclado para caracteres individuales disponibles en el panel de tareas se pueden modificar. Consulte Preferencias del Borrador. Los atajos que se mencionan aquí son los predeterminados.

  • Para introducir manualmente las coordenadas, introduzca las componentes X, Y y Z, y pulse la tecla Intro después de cada uno. También puede pulsar el botón Introducir Punto cuando tenga los valores deseados. Se recomienda mover el puntero fuera de la vista 3D antes de introducir las coordenadas.
  • Pulse la R o haga clic en la casilla Relativo para activar o desactivar el modo relativo. Si el modo relativo está activado, las coordenadas son relativas al último punto, si está disponible; de ​​lo contrario, son relativas al origen del sistema de coordenadas.
  • Pulse la G o haga clic en la casilla Global para activar o desactivar el modo global. Si el modo global está activado, las coordenadas son relativas al sistema de coordenadas global; de lo contrario, son relativas al plano de trabajo.
  • Pulse F o haga clic en la casilla de verificación Redondear1.0 and below o Crear Caraintroduced in 1.1 para activar o desactivar el modo de creación de caras. Si el modo de creación de caras está activado, el cable creado tendrá DatosCrear Cara establecido en true y una cara rellena, siempre que esté cerrado y no se autointerseque. Un cable con autointersección y una cara no se mostrará correctamente; para dicho cable, DatosCrear Cara debe establecerse en false.
  • Pulse N o haga clic en la casilla de verificación Continuar para activar o desactivar el modo de continuación. Si el modo de continuación está activado, el comando se reiniciará después de usar Terminar o Cerrar, o después de crear un cable cerrado ajustando al primer punto del cable, lo que le permitirá seguir creando cables.
  • Pulse / o el botón Deshacer para deshacer el último punto.
  • Pulse A o el botón Terminar para finalizar el comando y dejar el cable abierto.
  • Pulse la tecla O o el botón Cerrar para finalizar el comando y cerrar el cable. También puede crear un cable cerrado ajustándolo al primer punto del cable.
  • Pulse la tecla W o el botón Wipe para eliminar los segmentos ya colocados, pero continúe trabajando desde el último punto.
  • Pulse la tecla U o el botón Ajustar Plano de Trabajo para ajustar el plano de trabajo actual en la orientación del último segmento.
  • Pulse la tecla S para activar o desactivar el ajuste a borrador (Ajuste).
  • Pulse Esc o el botón Cerrar para finalizar el comando.

Unir

Uso

  1. Los puntos finales de las Líneas y/o cables de dibujo que se van a unir deben coincidir exactamente. Si es necesario, ajuste primero los puntos para asegurarse de que así sea.
  2. Seleccione dos o más Líneas y/o cables de dibujo (Draft Wire).
  3. Hay varias maneras de ejecutar el comando:
    • Pulse el botón Draft Wire.
    • Seleccione la opción Dibujar → Polilínea del menú.
    • Utilice el atajo de teclado: P seguido de L.

Notas

  • Un Cable se puede editar con el comando Editar.
  • Un cable se puede convertir en una BSpline con el comando De Cable a BSpline.
  • Las Líneas y los cables de borrador también se pueden unir con el comando Unir o el comando Draft Actualizar.

Propiedades

Véase también: Vista de propiedades.

Un objeto Cable se deriva de un objeto Part Part2DObject1.0 and below o de un objeto Característica1.1 and above y hereda todas sus propiedades. También tiene las siguientes propiedades adicionales:

Datos

Adjunto

El objeto tiene las mismas propiedades de conexión que un Part Part2DObject.

Borrador

  • DatosArea (Área) (Solo lectura): Especifica el área de la cara del cable. El valor será 0.0 si DatosCrear Cara es false o si no se puede crear la cara.
  • DatosBase (Enlace)
  • DatosTamaño de Chaflán (Longitud): Especifica la longitud de los chaflanes en las esquinas del cable.
  • DatosCerrado (Booleano): Especifica si el cable está cerrado o no. Si el cable está inicialmente abierto, este valor es false. Al establecerlo en true, se dibujará un segmento de línea para cerrar el cable. Si el cable está inicialmente cerrado, este valor es true. Al establecerlo en false, se eliminará el último segmento de línea y el cable quedará abierto.
  • DatosFinal (Vector Distancia): Especifica el punto final del cable.
  • DatosRadio de Redondeo (Longitud): Especifica el radio de los redondeos en las esquinas del cable.
  • DatosLongitud (Longitud) (solo lectura): Especifica la longitud total del cable.
  • DatosCrear Cara (Booleano): especifica si el cable forma una cara o no. Si es true, se crea una cara; de lo contrario, solo los bordes se consideran parte del objeto. Esta propiedad solo funciona si DatosCerradp es true y si el cable no se autointerseca.
  • DatosPuntos (Lista de Vectores): Especifica los puntos del cable en su sistema de coordenadas local.
  • DatosComenzar (Vector Distancia): Especifica el punto de inicio del cable.
  • DatosSubdivisiones (Entero): Especifica el número de subdivisiones para cada arista del alambre. Si es 1, cada arista se dividirá en 2 segmentos iguales. Las subdivisiones se aplican antes de los chaflanes y los redondeos.
  • DatosHerramienta (Enlace)

Vista

Borrador

  • VistaTamaño Extremo Flecha (Longitud): Especifica el tamaño del símbolo que se muestra al final del cable. introduced in 1.1
  • VistaTamaño Comienzo Flecha (Longitud): íÍdem al inicio del cable. introduced in 1.1
  • VistaSímbolo Final Flecha (Enumeración): Especifica el tipo de símbolo que se muestra al final del cable, que puede ser Punto, Circulo, Flecha, Tick, Tick-2 o Ninguno. introduced in 1.1
  • VistaSímbolo Comienzo Flecha (Enumeration): Ídem al inicio del cable. introduced in 1.1
  • VistaPatrón (Enumeración): Especifica el Patrón con el que se rellenará la cara del alambre cerrado. Esta propiedad solo funciona si DatosCrear Cara es true y si VistaMostrar Modo es una Línea.
  • VistaTamaño Patrón (Float): especifica el tamaño del Patrón.

Programación de Scripts

Véase también: Documentación de API generada automáticamente and Programación Básica de Scrpits en FreeCAD.

Para crear un Cable, utilice el método make_wire (introduced in 0.19) del módulo Draft. Este método reemplaza al método obsoleto makeWire. 

wire = make_wire(pointslist, closed=False, placement=None, face=None, support=None)
wire = make_wire(Part.Wire, closed=False, placement=None, face=None, support=None)
  • Crea un objeto Wire con la lista de puntos proporcionada, pointslist.
    • Cada punto de la lista se define por su FreeCAD.Vector, con unidades en milímetros.
    • Alternativamente, la entrada puede ser un Part.Wire, del cual se extraen los puntos.
  • Si closed es True, o si el primer y el último punto son idénticos, el alambre se cierra.
  • Si placement es None, la forma se crea en el origen.
  • Si face es True y el alambre está cerrado, el alambre formará una cara, es decir, aparecerá relleno.

Ejemplo: 

import FreeCAD as App
import Draft

doc = App.newDocument()

p1 = App.Vector(0, 0, 0)
p2 = App.Vector(1000, 1000, 0)
p3 = App.Vector(2000, 0, 0)

wire1 = Draft.make_wire([p1, p2, p3], closed=True)
wire2 = Draft.make_wire([p1, 2*p3, 1.3*p2], closed=True)
wire3 = Draft.make_wire([1.3*p3, p1, -1.7*p2], closed=True)

doc.recompute()
