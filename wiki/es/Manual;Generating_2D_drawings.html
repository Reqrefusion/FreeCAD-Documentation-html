Cuando un modelo no se puede imprimir ni fresar directamente con una máquina (por ejemplo, si es demasiado grande, como un edificio) o requiere montaje manual una vez que las piezas estén listas, normalmente tendrás que explicarle a otra persona cómo hacerlo. En campos técnicos (ingeniería, arquitectura, etc.), esto se suele hacer con planos. Los planos se entregan a la persona responsable del montaje del producto final y le explicarán cómo hacerlo.

Ejemplos típicos son las instrucciones de Ikea, los planos arquitectónicos y los planos arquitectónicos. Estos planos suelen contener, no solo el dibujo en sí, sino también numerosas anotaciones, como texto, dimensiones, números y símbolos que ayudan a comprender qué se debe hacer y cómo.

En FreeCAD, el entorno de trabajo responsable de realizar dichos dibujos es el TechDraw Workbench.

TechDraw Workbench permite crear hojas, que pueden estar en blanco o utilizar una plantilla predefinida (template) con elementos predefinidos, como bordes y un título. En estas hojas, puede colocar vistas de los objetos 3D modelados previamente y configurar su apariencia. También puede añadir anotaciones, como cotas, textos y otros símbolos comunes en dibujos técnicos.

Las hojas de dibujo, una vez terminadas, se pueden imprimir o exportar como archivos SVG, PDF o DXF.

En el siguiente ejercicio, veremos cómo crear un dibujo sencillo de un modelo de silla que se encuentra en la biblioteca de FreeCAD [1] (Diseño Industrial → Sillas → Silla tipo Ikea). La biblioteca de FreeCAD se puede añadir fácilmente a su instalación de FreeCAD (consulte el capítulo Instalación de este manual), o bien puede descargar el modelo desde la página web de la biblioteca o mediante el enlace directo que se proporciona al final de este capítulo.

  • Cargue el archivo IkeaLikeChair desde la biblioteca. Puede elegir entre la versión .FCStd, que cargará todo el historial de modelado, o la versión .step, que creará un solo objeto, sin el historial. Como no necesitaremos modelar más ahora, es mejor elegir la versión .step, ya que será más fácil de manipular.

  • Cambie a la TechDraw Workbench
  • Pulse el botón TechDraw PageTemplate.
  • Seleccione la plantilla A4_Portrait_ISO7200TD. Se abrirá una nueva pestaña en la ventana de FreeCAD, mostrando la nueva página.
  • En la vista de árbol (o en la pestaña Modelo), seleccione el modelo de la silla. Probablemente se llame algo como: "Abrir traductor CASCADE STEP".
  • Pulse el botón TechDraw View.
  • Se creará un objeto de vista en nuestra página. Seleccione el objeto de vista en la vista de árbol y, a continuación, asigne a la vista las siguientes propiedades en la pestaña Datos de la vista combinada:
    • En la categoría Base:
      • X: 70 mm
      • Y: 120 mm
      • Rotación: 0
      • Escala: 0.1
    • En la categoría Proyección (pulse la flecha desplegable para modificar individualmente los componentes x, y, z de estas propiedades):
      • Dirección: [0 0 1]
      • Dirección X: [0 -1 0] (Cambie primero el campo y, modificando luego el campo x)
  • Ahora tenemos una vista superior de la silla. Pulse el botón TechDraw ToggleFrame para desactivar los marcos, las etiquetas y los vértices de la vista.

  • Repitamos la operación dos veces para crear dos vistas más. Definiremos sus valores X e Y, que indican la posición de la vista en la página, para mostrarlas separadas de la vista superior, y su orientación, para crear diferentes orientaciones. Asignaremos a cada nueva vista las siguientes propiedades:
    • Vista001 (vista frontal): X: 70, Y: 220, Escala: 0.1, Rotación: 0, Dirección: (-1,0,0), Dirección X: (0,-1,0)
    • Vista002 (vista lateral): X: 150, Y: 220, Escala: 0.1, Rotación: 0, Dirección: (0,-1,0), Dirección X: (1,0,0)
  • Después de esto, obtenemos la siguiente página:

  • Tenga en cuenta que puede haber formas más sencillas de obtener las vistas que desea. Simplemente puede rotar la vista 3D de su modelo y, una vez que tenga la vista que desea, seleccione el modelo en la vista de árbol y haga clic en 16px Nueva vista. Esto insertará automáticamente una vista con las propiedades de rotación y dirección deseadas. También puede usar la herramienta 16px TechDraw ProjectionGroup.
  • Podemos ajustar el aspecto de nuestras vistas si queremos; por ejemplo, podemos cambiar su propiedad Ancho de línea (en la pestaña Vista en la Vista combinada) a 0,5.

Ahora colocaremos cotas e indicaciones en nuestro dibujo. Hay dos maneras de añadir cotas a un modelo: una es colocar las cotas dentro del modelo 3D, usando la herramienta Dimension del Draft Workbench, y luego colocar una vista de estas cotas en nuestra hoja con la herramienta TechDraw DraftView. La otra es hacerlo directamente en la hoja de TechDraw. Usaremos este último método.

  • Pulse el botón para activar los vértices.
  • Use Ctrl + clic izquierdo del ratón para seleccionar los dos vértices entre los que quiere medir la distancia.
  • Pulse el botón TechDraw LengthDimension.

  • Repeat the operation, until all the dimensions you wish to indicate are placed. Use the TechDraw VerticalDimension and TechDraw HorizontalDimension tools as necessary.
  • Take a minute to look at the properties of the Dimension object in the Combo View.
  • Please note that if you are dimensioning an axonometric view (e.g., isometric view) instead of a multiview view (e.g., front view) as we have done here, you will need to use the TechDraw LinkDimension tool to get an accurate dimension.

  • We will now place the two callouts shown in the image above, using the TechDraw Balloon tool.

  1. Looking at the Page in the 3D view window, select the View to which the Balloon will be attached, as shown in the image above.
  2. Press the Balloon button.
  3. The cursor is now displayed as a balloon icon. Click on the page to place the balloon origin at the desired position.
  4. The balloon bubble may be dragged to the desired position.
  5. Change the balloon properties by double-clicking the balloon label or the balloon object in the tree view. This will open the Balloon Task dialog. Set the Value field to the desired text and change the Symbol drop-down menu selection to None
  6. Press OK
  7. Repeat the operation for the second callout.
  • We will now fill in the sheet title block.
    • Make sure that the View frames, labels, and vertices are visible. If not, hit the Toggle button.
    • Edit the text in each section of the sheet title block by clicking on the small green square on the left side of the text.

Our page can now be exported to SVG for further work in graphical applications like Inkscape or to DXF. Select the page in the tree view and then select menu File → Export. The DXF format is importable in almost all existing 2D CAD applications. TechDraw pages can also be directly printed or exported to PDF.
