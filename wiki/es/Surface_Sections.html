Descripción

La herramienta Secciones de Superficie crea una superficie a partir de una serie de bordes seccionales.

Izquierda: Aristas de control (secciones transversales). Derecha: Superficie producida a partir de estas aristas.

Uso

  1. Asegúrese de tener al menos dos aristas o curvas en el espacio. Por ejemplo, puede crearlas con las herramientas del Entorno de Trabajo Borrador o Entorno de Trabajo Croquizador.
  2. Hay varias maneras de usar la herramienta:
    • Pulse el botón Secciones.
    • Seleccione la opción Superficie → Secciones del menú.
  3. Pulse Añadir arista.
  4. Utilice el puntero para seleccionar los bordes deseados en la Vista_3D; se mostrará una vista previa de la forma final tras seleccionar dos bordes válidos.
  5. Pulse Aceptar para completar la operación.

Opciones

  • Add Edge: Pulse una vez para empezar a seleccionar aristas en la Vista 3D. Se pueden elegir líneas individuales como BSplines del Borrador y BSplines del Croquizador, así como cualquier arista de objetos sólidos, como las de Cuerpos y Primitivas.
  • Eliminar Borde: Pulse una vez para empezar a seleccionar aristas en la Vista 3D; Deben ser aristas que se hayan seleccionado previamente con Añadir Arista.
  • Botón Derecho del Ratón: Abra el menú contextual y seleccione Eliminar, o pulse Suprimir en el teclado, para eliminar la arista seleccionada en la lista.
  • Arrastra: Arrastre el elemento seleccionado en la lista para cambiar el orden de procesamiento; la lista se procesa de arriba abajo.
  • Pulse Cancelar o Escape para cancelar la operación.

Propiedades

Un objeto Surface Sections (clase Surface::Sections) se deriva de la Part Feature básica (clase Part::Feature, a través de la subclase Part::Spline), por lo tanto, comparte todas las propiedades de esta última.

Además de las propiedades descritas en Características de Pieza, el objeto Surface Sections tiene las siguientes propiedades en la Vista de Propiedades.

Datos

Secciones

  • DatosNSections (LinkSubList): Una lista de aristas que se utilizarán para construir la superficie.

Vistas

Base

  • VistaControl Points (Bool): Por defecto es false; si se establece en true, mostrará una superposición con los puntos de control de la superficie.

Torsión de la Superficie

La forma de la superficie depende de la dirección de los bordes seleccionados; si se seleccionan bordes y el resultado es una superficie que se retuerce sobre sí misma, es posible que uno de los bordes necesite su lista de vértices en orden inverso. Consulte la información en Superficie de Relleno Geométrico para obtener una explicación más completa.

Programación de Scripts

Ver también: Conceptos Básicos de Programación de Scripts en FreeCAD.

The Surface Sections tool can be used in macros and from the Python Console by adding the Surface::Sections object.

  • The edges to be used to define the surface must be assigned as a LinkSubList to the NSections property of the object.
  • All objects with edges need to be computed before they can be used as input for the properties of the Sections object.
import FreeCAD as App
import Draft

doc = App.newDocument()

pl1 = App.Placement()
obj1 = Draft.make_circle(50, placement=pl1, face=False, startangle=0, endangle=180)

pl2 = App.Placement(App.Vector(0, 0, 25), App.Rotation())
obj2 = Draft.make_circle(30, placement=pl2, face=False, startangle=0, endangle=180)

points3 = [App.Vector(18, -10, 50),
           App.Vector(12, 10, 50),
           App.Vector(-12, 10, 50),
           App.Vector(-18, -10, 50)]
obj3 = Draft.make_bspline(points3)

points4 = [App.Vector(15, -20, 100),
           App.Vector(0, 6, 100),
           App.Vector(-15, -20, 100)]
obj4 = Draft.make_bspline(points4)
doc.recompute()

surf = doc.addObject("Surface::Sections", "Surface")
surf.NSections = [(obj1, "Edge1"),
                  (obj2, "Edge1"),
                  (obj3, "Edge1"),
                  (obj4, "Edge1")]
doc.recompute()
