Description

Extrae la parte común (intersección) entre los objetos piezas seleccionados. Esta operación es completamente paramétrica y los componentes pueden ser modificados y el resultado recalculado.

This tool is an automated form of the Boolean operation.

Usage

  1. Select two shapes
  2. There are several ways to invoke the tool:
    • Press the Intersection button.
    • Select the Part → Boolean → Intersection option from the menu.

Supported inputs

Input objects must be OpenCASCADE shapes. Examples: stuff made with Part, PartDesign, Sketcher workbenches. Not meshes (unless those were converted to shapes) - for meshes, there are specific Boolean tools in MeshDesign workbench.
