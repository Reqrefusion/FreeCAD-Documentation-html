Superficie: Extender Cara

Ubicación en el Menú
Superficie → Extender Cara

Descripción

La herramienta Extender Cara extrapola una cara o superficie existente en sus límites con sus parámetros U y V locales.

Template:Captión

Uso

  1. Asegúrese de tener un objeto con caras. El objeto puede crearse con el 16px Entorno de Trabajo de Superficies, pero también puede ser cualquier otro objeto, por ejemplo, creado con el Entorno de Trabajo Pieza o el 16px Entorno de Diseño de Pieza.
  2. Seleccione la cara que desea extender haciendo clic sobre ella en la Vista 3D.
  3. Hay varias maneras de usar la herramienta:
    • Pulse el botón Extender Cara.
    • Seleccione la opción Superficie → Extender Cara del menú.

Opciones

Este comando no tiene opciones. Funciona con la selección o no funciona.

Propiedades

Un objeto Superficie Extendida (clase Surface::Extend) se deriva de la Característica de Pieza básica (clase Part::Feature, a través de la subclase Part::Spline), por lo tanto, comparte todas las propiedades de esta última.

Además de las propiedades descritas en Características de Pieza, el objeto Superficie Extendida tiene las siguientes propiedades en la Vista de Propiedades.

Datos

Base

  • DatosFace (LinkSub): el subelemento de un objeto que se extenderá; debe ser una cara.
  • DatosTolerance (FloatConstraint): su valor predeterminado es 0.1.
  • DatosExtend UNeg (FloatConstraint): por defecto, 0.05. La proporción del parámetro U local que se extenderá en la dirección negativa.
  • DatosExtend UPos (FloatConstraint): por defecto, 0.05. La proporción del parámetro U local que se extenderá en la dirección positiva.
  • DatosExtend USymetric (Bool): por defecto, true, en cuyo caso DatosExtend UNeg y DatosExtend UPos tendrán el mismo valor.
  • DatosExtend VNeg (FloatConstraint): por defecto, 0.05. La proporción de la V local que se extenderá en la dirección negativa.
  • DatosExtend VPos (FloatConstraint): por defecto, 0.05. La proporción de la dirección V local que se extenderá en la dirección positiva.
  • DatosExtend VSymetric (Bool): por defecto, true, en cuyo caso DatosExtend VNeg y DatosExtend VPos tendrán el mismo valor.
  • DatosSampleU (IntegerConstraint): Por defecto a 32.
  • DatosSampleV (IntegerConstraint): Por defecto a 32.

Vistas

Base

  • VistaControl Points (Bool): it defaults to false; if set to true, it will show an overlay with the control points of the surface.

Scripting

See also: FreeCAD Scripting Basics.

The Surface Extend tool can be used in macros and from the Python Console by adding the Surface::Extend object.

  • The face to extend must be assigned as a LinkSub to the Face property of the object. It must contain only a single face.
import FreeCAD as App
import Draft

doc = App.newDocument()

a = App.Vector(-20, -20, 0)
b = App.Vector(-18, 25, 0)
c = App.Vector(60, 26, 0)
d = App.Vector(33, -20, 0)

points = [a, App.Vector(-20, -8, 0), b, c,
          App.Vector(37, 4, 0), d,
          App.Vector(-2, -18, 0), a]
obj = Draft.make_bspline(points)
doc.recompute()

if App.GuiUp:
    obj.ViewObject.Visibility = False

surf = doc.addObject("Surface::Filling", "Surface")
surf.BoundaryEdges = [(obj, "Edge1")]
doc.recompute()

# ---------------------------------------------------------
points_spl = [App.Vector(-10, 0, 2),
              App.Vector(4, 0, 7),
              App.Vector(18, 0, -5),
              App.Vector(25, 0, 0),
              App.Vector(30, 0, 0)]
aux_edge = Draft.make_bspline(points_spl)
doc.recompute()

surf.UnboundEdges = [(aux_edge, "Edge1")]
doc.recompute()

# ---------------------------------------------------------
surf_extended = doc.addObject("Surface::Extend", "Surface")
surf_extended.Face = [surf, "Face1"]
doc.recompute()
