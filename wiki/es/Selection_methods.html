Vista general

Los métodos de selección de FreeCAD permiten seleccionar objetos en la Interfaz, como la Vista 3D, la Vista de Árbol, la Vista de Selección y otros cuadros de diálogo. Algunos métodos de selección son específicos de cada entorno de trabajo y se documentan en la documentación correspondiente.

Vista 3D

En la Vista 3D hay varias formas de seleccionar objetos.

Selección sencilla

La selección simple con el ratón (por defecto, clic izquierdo) y la preselección (al pasar el ratón por encima) se describen en la página Navegación con el Ratón.

Clics repetidos

El primer clic selecciona un subelemento (vértice, arista o cara) del objeto que se encuentra bajo el cursor. Un segundo clic selecciona el objeto completo.

El tercer clic extiende la selección a su objeto contenedor (Cuerpo del Diseño de Pieza, Pieza Estándar y otros). Los clics posteriores expanden la selección hacia arriba en la cadena de contenedores.

Comandos de selección

  • Para seleccionar todos los objetos: Seleccionar todo.
  • Para seleccionar varios objetos principales mediante un recuadro: Selección estándar.
  • Para seleccionar varias caras mediante un recuadro: Selección estándar de Elemento de Caja o Selección de Elemento de Caja.

Drag box selection

introduced in 26.3

With common navigation styles, objects can be box selected in the 3D View without using the Std BoxSelection command. You can drag a rectangle from left to right or vice-versa. The selection behavior is the same as with that command.

Vista de selección

La Vista de Selección muestra los nombres de los objetos seleccionados, incluyendo su nombre completo dentro de un objeto, por ejemplo, Unnamed#Body.Box001.Face17.

También permite realizar algunas acciones como la Selección de Ajuste de Vista Estándar o enviar el objeto a la Consola Python.

Exportación de objetos

Esto debería estar en la página de Vista de Selección.

Seleccione cualquier objeto complejo, por ejemplo, un Cuerpo o una Pieza, luego en la Vista de Seleción seleccione nuevamente el objeto y luego presione Ctrl + C en el teclado para abrir el cuadro de diálogo Selección de Objeto. Esto permite copiar el objeto seleccionado junto con todos, o solo algunos, de los objetos de dependencia de ese objeto. Por ejemplo, para una Pieza, los objetos posibles para seleccionar incluyen la Pieza en sí, pero también su origen, sus tres ejes base (X,Y ó Z) y sus tres planos base (XY, YZ, XZ).

Tras pulsar OK, los objetos seleccionados se copian en la memoria y, a continuación, se pueden pegar en el documento para duplicar únicamente estos objetos.

Cuadro de diálogo de selección de objetos que se inicia desde la Vista de Selección.

Vista de Árbol

En la vista de árbol, los elementos se pueden seleccionar o desmarcar uno por uno manteniendo presionada la tecla Ctrl y haciendo clic con el mouse sucesivamente.

Se puede seleccionar un rango de elementos haciendo clic en el primer elemento, manteniendo presionada la tecla Shift y haciendo clic en el último elemento.

Al seleccionar un solo elemento, también se mostrarán sus propiedades en el Editor de Propiedades.

Al hacer doble clic, se abrirá cualquier Panel de Tareas asociado que contenga acciones. Asegúrese de cerrar este panel de tareas antes de ejecutar otro comando o cambiar a cualquier otro entorno de trabajo.

Hay más métodos disponibles abriendo el menú contextual (clic derecho), dependiendo del objeto seleccionado o del entorno de trabajo activo; consulte la información en Vista de Árbol.

Programación de Scripts

La selección de objetos es intrínsecamente una tarea gráfica y, por lo tanto, solo está disponible cuando la interfaz gráfica de usuario está cargada.

Estos métodos se pueden usar en Macros o desde la Consola de Python: 

import FreeCADGui as Gui

Gui.Selection.addSelection
Gui.Selection.addSelectionGate
Gui.Selection.Filter

El método addSelectionGate impide que el usuario seleccione objetos que no estén especificados en la cadena de selección. Aparece un símbolo cuando el puntero se sitúa sobre un elemento que no pertenece al grupo especificado. 

Gui.Selection.addSelectionGate("SELECT Part::Feature SUBELEMENT Edge")

#### or
Gui.Selection.addSelectionGate("SELECT Part::Feature SUBELEMENT Face")

#### or
Gui.Selection.addSelectionGate("SELECT Part::Feature SUBELEMENT Vertex")

Para eliminar SelectionGate(): 

Gui.Selection.removeSelectionGate()

Consulte la Documentación Fuente y la Ayuda Estándar de Python para obtener más ayuda sobre el uso de estas herramientas.
