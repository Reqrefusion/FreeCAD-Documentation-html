FreeCAD utiliza el framework Qt y se caracteriza por una interfaz sencilla y directa. Los usuarios de CAD con más experiencia podrán identificar similitudes con otros programas, mientras que los nuevos usuarios encontrarán fácil navegar y descubrir las diversas opciones que ofrece FreeCAD. Este es el aspecto predeterminado de FreeCAD:

La página de inicio sirve como pantalla de bienvenida, diseñada para facilitar un acceso rápido y sencillo a las principales áreas de FreeCAD que el usuario pueda desear explorar. A través de ella, los usuarios pueden crear nuevas piezas sin esfuerzo, abrir archivos recientes e iniciar el dibujo en 2D. Además, cuenta con accesos directos a recursos útiles, como tutoriales y foros de usuarios, que son muy valiosos tanto para principiantes como para usuarios experimentados que buscan orientación o consejos. Los usuarios pueden personalizar fácilmente el aspecto de la página de inicio según sus preferencias.

A medida que adquiera más destreza con FreeCAD, es posible que desee ajustar la configuración en las preferencias. Esto le permitirá configurar FreeCAD para que se abra directamente en uno de los entornos de trabajo con un nuevo documento listo para usar al iniciarlo. También puede simplemente cerrar la pestaña Página de inicio y crear manualmente un nuevo documento.

Bancos de trabajo

FreeCAD emplea un sistema denominado "bancos de trabajo", similar a los marcos conceptuales utilizados en software de diseño avanzado como Revit o CATIA. La idea de un banco de trabajo es análoga a las estaciones especializadas de un laboratorio científico, donde diferentes estaciones de trabajo están equipadas para distintos tipos de experimentos. En un laboratorio, puede haber un área dedicada a la química, otra a la biología y una tercera a la física, cada una equipada con las herramientas específicas necesarias para esas disciplinas.

En el contexto de FreeCAD, cada Banco de Trabajo está adaptado a un tipo de tarea concreto, organizando todas las herramientas necesarias para esa actividad en una sola interfaz. Al cambiar de Banco de Trabajo, el conjunto de herramientas y controles visibles en la interfaz de usuario se ajusta para reflejar las necesidades de la tarea seleccionada, aunque el contenido real del proyecto o la «escena» en la que se está trabajando no cambia. Esto permite transiciones fluidas en el flujo de trabajo, como comenzar un diseño con formas 2D básicas en el Banco de Trabajo Draft y luego elaborar estos diseños con herramientas de modelado avanzadas en el Banco de Trabajo Part.

Los términos "Banco de Trabajo" y "Módulo" se utilizan a veces indistintamente, pero tienen significados distintos dentro de FreeCAD. Un módulo es cualquier extensión que añade funcionalidad a FreeCAD, mientras que un Banco de Trabajo es un tipo específico de módulo equipado con sus propios componentes de interfaz de usuario, como barras de herramientas y menús, diseñado para facilitar tipos específicos de tareas. Por lo tanto, cada Banco de Trabajo es un módulo, pero no todos los módulos pueden considerarse Bancos de Trabajo.

El control más importante de la interfaz de FreeCAD es el selector de Banco de Trabajo, que se utiliza para cambiar de un Banco de Trabajo a otro:

  • El Banco de Trabajo Assembly para construir y resolver ensamblajes mecánicos. introduced in 1.0
  • El Banco de Trabajo BIM para trabajar con elementos arquitectónicos y crear modelos BIM. Combina el Banco de Trabajo Arch y el Banco de Trabajo BIM, anteriormente externo, disponible en 0.21 and below.
  • El Banco de Trabajo CAM es usado para producir instrucciones de código-G. Este banco de trabajo se llamaba "Banco de Trabajo Path" en 0.21 and below.
  • El Banco de Trabajo Borrador contiene herramientas 2D y operaciones básicas de CAD 2D y 3D.
  • El Banco de Trabajo MEF proporciona un flujo de trabajo de análisis de elementos finitos (FEA).
  • El Banco de Trabajo de Inspección está hecho para dar herramientas específicas para el examen de formas. Todavía está en una etapa temprana de desarrollo.
  • El Banco de Trabajo Material maneja el sistema de materiales de FreeCAD. introduced in 1.0
  • El Banco de Trabajo Malla para trabajar con mallas trianguladas.
  • El Banco de Trabajo OpenSCAD para la interoperabilidad con OpenSCAD y la reparación del historial del modelo de geometría sólida constructiva (CSG).
  • El Banco de Trabajo Part para trabajar con primitivas geométricas y operaciones booleanas.
  • El Banco de Trabajo Dieño de Partes para crear formas de piezas a partir de bocetos.
  • El Banco de Trabajo Puntos para trabajar con nubes de puntos.
  • El Banco de Trabajo Ingeniería Inversa pretende proporcionar herramientas específicas para convertir formas/sólidos/mallas en características paramétricas compatibles con FreeCAD.
  • El Banco de Trabajo Robots para estudiar los movimientos de los robots. Actualmente sin mantenimiento.
  • El Banco de Trabajo Bocetista para trabajar con bocetos con restricciones geométricas.
  • El Banco de Trabajo Hoja de Cálculo para crear y manipular datos de hojas de cálculo.
  • El Banco de Trabajo Superficie proporciona herramientas para crear y modificar superficies. Es similar a la opción Crear Cara a Partir de Aristas del Constructor de Partes.
  • El Banco de Trabajo DibujoTécnico para producir dibujos técnicos a partir de modelos 3D. Es el sucesor del Banco de Trabajo de Dibujo.
  • El Bando de Trabajo Entorno de Pueba sirve para depurar FreeCAD.

Los Bancos de Trabajo suelen confundir a los nuevos usuarios, ya que no siempre es fácil saber en qué Banco de Trabajo buscar una herramienta específica. Sin embargo, son fáciles de aprender y, al cabo de poco tiempo, se utilizan con total naturalidad. Los nuevos usuarios se dan cuenta rápidamente de que los Bancos de Trabajo son una forma cómoda de organizar la gran cantidad de herramientas que ofrece FreeCAD. Además, los Bancos de Trabajo también son totalmente personalizables.

La interfaz

Echemos un vistazo más detallado a las diferentes partes de la interfaz:

La ventana principal de la aplicación se puede dividir a grandes rasgos en 11 secciones:

  1. El Área de vista principal, que puede contener diferentes ventanas con pestañas.
  2. La Vista 3D, normalmente integrada en el Área de vista principal. La vista 3D es el elemento central de la interfaz, que muestra y permite la manipulación de los objetos con los que se está trabajando. Es posible tener múltiples vistas del mismo documento (u objetos) o tener varios documentos abiertos simultáneamente. Además, cada vista se puede separar de la ventana principal por separado.
  3. La pestaña Modelo y la pestaña Tareas.
    1. La pestaña Modelo muestra el contenido y la estructura de su documento.
    2. La pestaña Tareas es donde FreeCAD le pedirá valores específicos para el entorno de trabajo y la herramienta que está utilizando actualmente (por ejemplo, las dimensiones de un objeto).
  4. El Editor de propiedades que aparece cuando la pestaña Modelo está activa en la interfaz. Permite gestionar las propiedades expuestas públicamente de los objetos del documento. Consta de las secciones Datos y Vista, que muestran las propiedades de visualización y las propiedades paramétricas de los objetos, respectivamente.
  5. La Vista de selección que indica los objetos o subelementos de los objetos (vértices, aristas, caras) que están seleccionados.
  6. La Vista de informe donde se muestran los mensajes, advertencias y errores.
  7. La Consola Python. La consola Python, donde se imprimen todos los comandos ejecutados y donde se puede introducir código Python.
  8. La Barra de estado, donde aparecen algunos mensajes e información sobre herramientas.
  9. El área de la barra de herramientas, donde se acoplan las barras de herramientas.
  10. El Selector de Banco de Trabajo, donde se selecciona el Banco de Trabajo activo.
  11. El Menú estándar, que contiene las operaciones básicas del programa.

La mayoría de los paneles anteriores se pueden ocultar o mostrar utilizando el menú Ver → Panels menu

Personalización de la interfaz

La interfaz de FreeCAD está diseñada para una amplia personalización. Todas las barras de herramientas y paneles se pueden reubicar, apilar o incluso acoplar en diversas configuraciones según las preferencias del usuario. Además, se pueden cerrar y volver a abrir según sea necesario. Más allá de estas capacidades, los usuarios tienen muchas otras opciones, incluida la posibilidad de crear barras de herramientas personalizadas con herramientas seleccionadas de cualquiera de los entornos de trabajo disponibles, y de asignar o modificar atajos de teclado para optimizar el flujo de trabajo. Esta flexibilidad garantiza que los usuarios puedan adaptar el entorno a sus necesidades y preferencias específicas.
