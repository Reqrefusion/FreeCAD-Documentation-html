Descripción

La herramienta Rellenar crea una superficie a partir de una serie de aristas de contorno conectadas. La curvatura de la superficie se puede controlar adicionalmente mediante aristas y vértices que no forman parte del contorno, así como mediante una superficie de soporte.

La geometría base puede pertenecer a curvas creadas con el Entorno de Trabajo Borrador o el Croquizador, pero también puede pertenecer a objetos sólidos como los creados con los entornos Pieza o Diseño de Pieza.

Dos superficies rellenas delimitadas por cuatro aristas situadas en el plano XY. La superficie de la derecha está controlada adicionalmente por una arista que no es un límite.

Uso

  1. Existen varias formas de invocar la herramienta:
    • Pulse el botón Rellenar.
    • Seleccione la opción Superficie → Rellenar del menú.
  2. Se abrirá el panel de tareas Límites. Consulte Opciones.
  3. Seleccione dos o más aristas en la Vista 3D:
    • No es necesario pulsar el botón Añadir Arista en la sección Límites en este momento.
    • Las aristas deben seleccionarse en orden consecutivo.
    • Las aristas deben estar conectadas, pero no es necesario cerrar completamente el contorno.
    • El contorno completo no debe autointersecarse.
    • Para un contorno circular de 360°, se pueden seleccionar dos aristas semicirculares.
  4. Se mostrará una vista previa de la forma final una vez que se haya seleccionado suficiente geometría válida.
  5. Opcionalmente, seleccione una Superficie de soporte. Consulte el Ejemplo.
  6. Opcionalmente, seleccione una o más Restricciones de Arista.
  7. Opcionalmente, seleccione una o más Restricciones de Vértice.
  8. Pulse el botón Aceptar.

Opciones

  • En la sección Límites se pueden especificar una superficie de soporte y aristas de contorno:
    • Pulse el botón Superfice Soporte y seleccione una cara en la Vista 3D para añadir una superficie de soporte.
      • Haga clic en el icono para eliminar la superficie de soporte.
    • Pulse el botón Añadir Arista una vez para empezar a seleccionar aristas de contorno en la Vista 3D.
    • Existen varias maneras de deseleccionar aristas de contorno:
      • Pulse el botón Eliminar Arista una vez para empezar a deseleccionar aristas en la Vista 3D.
      • Seleccione una arista de la lista y pulse Borrar.
      • Haga clic con el botón derecho en una arista de la lista y seleccione Eliminar en el menú contextual.
  • En la sección Restricciones de Borde se pueden especificar bordes que no sean de límite:
    • Las opciones de selección son similares a las de los bordes de límite.
  • En la sección Restricciones de Vértices se pueden especificar los vértices que no son de contorno:
    • Las opciones de selección son similares a las de las aristas de contorno.
  • Pulse Escape o el botón Cancelar para abortar la operación.

Ejemplo

La superficie de Soporte actúa como una restricción adicional para la superficie. El siguiente ejemplo sencillo le dará una idea de cómo funciona:

  1. En el Entorno de Pieza cree un 16pxCilindro y establezca su DatosÁngulo en 180°.
  2. Cambie al Entorno de Sperficie y pulse el botón Rellenar.
  3. Seleccione los dos bordes semicirculares y los dos bordes rectos que los conectan.
  4. El resultado coincide con los cuatro bordes del contorno, pero la forma interior es bastante diferente de la cara cilíndrica.
  5. Edite el objeto Superficie y, en el menú Superficie de Soporte, seleccione la cara cilíndrica.
  6. La forma modificada se ajusta mucho mejor a la cara cilíndrica.

Propiedades

Un objeto Surface Filling (clase Surface::Filling) se deriva de la Part Feature básica (clase Part::Feature, a través de la subclase Part::Spline), por lo tanto, comparte todas las propiedades de esta última.

In addition to the properties described in Part Feature, the Surface Filling object has the following properties in the Property View.

Data

Filling

  • DatosBoundary Edges (LinkSubList): boundary edges; C0 is required for edges without a corresponding face.
  • DatosBoundary Faces (StringList):
  • DatosBoundary Order (IntegerList): order of constraint on boundary faces; 0, 1, and 2 are possible.
  • DatosUnbound Edges (LinkSubList): unbound constraint edges; C0 is required for edges without a corresponding face.
  • DatosUnbound Faces (StringList):
  • DatosUnbound Order (IntegerList): order of constraint on unbound faces; 0, 1, and 2 are possible.
  • DatosFree Faces (LinkSubList): free constraint on a face.
  • DatosFree Order (IntegerList): order of constraint on free faces.
  • DatosPoints (LinkSubList): constraint points on surface.
  • DatosInitial Face (LinkSub): initial surface to use.
  • DatosDegree (Integer): starting degree, it defaults to 3.
  • DatosPoints On Curve (Integer): number of points on an edge for constraint.
  • DatosIterations (Integer): number of iterations, it defaults to 2.
  • DatosAnisotropy (Bool): it defaults to false.
  • DatosTolerance2d (Float): 2D tolerance, it defaults to 0.0.
  • DatosTolerance3d (Float): 3D tolerance, it defaults to 0.0.
  • DatosTol Angular (Float): G1 tolerance, it defaults to 0.01.
  • DatosTol Curvature (Float): G2 tolerance, it defaults to 0.10.
  • DatosMaximum Degree (Integer): maximum curve degree, it defaults to 8.
  • DatosMaximum Segments (Integer): maximum number of segments, it defaults to 9.

View

Base

  • VistaControl Points (Bool): it defaults to false; if set to true, it will show an overlay with the control points of the surface.

Scripting

See also: FreeCAD Scripting Basics.

The Surface Filling tool can be used in macros and from the Python Console by adding the Surface::Filling object.

  • The edges to be used to define the surface must be assigned as a LinkSubList to the BoundaryEdges property of the object.
  • Auxiliary edges and vertices must be assigned as a LinkSubLists to the UnboundEdges and Points properties of the object.
  • All objects with edges need to be computed before they can be used as input for the properties of the Filling object.
import FreeCAD as App
import Draft

doc = App.newDocument()

a = App.Vector(-20, -20, 0)
b = App.Vector(-18, 25, 0)
c = App.Vector(60, 26, 0)
d = App.Vector(33, -20, 0)

points1 = [a, App.Vector(-20, -8, 0), App.Vector(-17, 7, 0), b]
obj1 = Draft.make_bspline(points1)

points2 = [b, App.Vector(0, 25, 0), c]
obj2 = Draft.make_bspline(points2)

points3 = [c, App.Vector(37, 4, 0), d]
obj3 = Draft.make_bspline(points3)

points4 = [d, App.Vector(-2, -18, 0), a]
obj4 = Draft.make_bspline(points4)
doc.recompute()

surf = doc.addObject("Surface::Filling", "Surface")
surf.BoundaryEdges = [(obj1, "Edge1"),
                      (obj2, "Edge1"),
                      (obj3, "Edge1"),
                      (obj4, "Edge1")]
doc.recompute()

# ---------------------------------------------------------
points_spl = [App.Vector(-10, 0, 2),
              App.Vector(4, 0, 7),
              App.Vector(18, 0, -5),
              App.Vector(25, 0, 0),
              App.Vector(30, 0, 0)]
aux_edge = Draft.make_bspline(points_spl)
doc.recompute()

surf.UnboundEdges = [(aux_edge, "Edge1")]
doc.recompute()

# ---------------------------------------------------------
aux_v1 = Draft.make_line(App.Vector(-13, -12, 5),
                         App.Vector(-13, -12, -5))
aux_v2 = Draft.make_line(App.Vector(-3, 18, 5),
                         App.Vector(-3, 18, -5))
doc.recompute()

surf.Points = [(aux_v1, "Vertex2"),
               (aux_v2, "Vertex1")]
doc.recompute()
