Instalación

Estable

Si desea instalar a nivel de usuario o no tiene privilegios de sudo, agregue la bandera --user a los siguientes comandos. 

# add flathub repo just to be sure as it might not be enabled if it is your first time using flatpak
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo
flatpak install flathub org.freecad.FreeCAD

Versiones de desarrollo

Si desea instalar a nivel de usuario o no tiene privilegios de sudo, agregue la bandera --user a los siguientes comandos. 

# flathub-beta repo is not enabled by default
flatpak remote-add --if-not-exists flathub-beta https://flathub.org/beta-repo/flathub-beta.flatpakrepo
flatpak install flathub-beta org.freecad.FreeCAD

Corriendo

Puede iniciar el archivo flatpak utilizando el archivo desktop o utilizando el siguiente comando: 

flatpak run org.freecad.FreeCAD

Las diferentes ramas se pueden instalar en paralelo. Para elegir cuál ejecutar, utilice la bandera --branch: 

flatpak run --branch=beta org.freecad.FreeCAD

Para ejecutar un ejecutable específico (por ejemplo, FreeCADCmd para ejecutarlo sin interfaz gráfica) desde el flatpak, utilice la bandera --command: 

flatpak run --command=FreeCADCmd org.freecad.FreeCAD
