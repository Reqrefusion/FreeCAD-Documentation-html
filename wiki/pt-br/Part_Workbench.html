Introdução

As capacidades de modelagem sólida do FreeCAD são baseadas em OpenCASCADE Technology(OCCT) kernel, um sistema de CAD de nível profissional que caracteriza a criação e a manipulação avançadas da geometria 3D. A Bancada de trabalho Part é uma camada situada em cima das bibliotecas OCCT, que dá ao usuário acesso às primitivas e funções geométricas da OCCT. Essencialmente todas as funções de desenho 2D e 3D em cada bancada de trabalho ( Draft, Sketcher, PartDesign, etc.), são baseadas nestas funções expostas pela Bancada de trabalho Part. Portanto, a Bancada de trabalho Part é considerada o componente central das capacidades de modelagem do FreeCAD.

The Part Workbench can also create objects that are not solids, such as faces, shells, and objects with only edges or vertices. It also provides a variety of general purpose tools for geometry manipulation, geometry validation, and making copies.

The PartDesign Workbench uses an alternative workflow for creating solids. For a detailed discussion of the Part Workbench versus the Part Design Workbench see Part and Part Design.

Ferramentas

Solids toolbar

  • Cubo: Cria um cubo sólido especificando suas dimensões.
  • Cilindro: Cria um cilindro sólido especificando suas dimensões.
  • Esfera: Cria uma esfera sólida especificando suas dimensões.
  • Cone: Cria um cone sólido especificando suas dimensões.
  • Toro: Cria um toro.
  • Tubo: Cria um tubo.
  • Criar primitivas...: Uma ferramenta para criar uma das seguintes primitivas:
  • Plano: Cria um plano.
  • Caixa: Cria uma caixa. Este objeto também pode ser criado com a ferramenta Cubo.
  • Cilindro: Cria um cilindro. Este objeto também pode ser criado com a ferramenta Cilindro.
  • Cone: Cria um cone. Este objeto também pode ser criado com a ferramenta Cone.
  • Esfera: Cria uma esfera. Este objeto também pode ser criado com a ferramenta Esfera.
  • Elipsóide: Cria um elipsóide.
  • Toro: Cria um toro. Este objeto também pode ser criado com a ferramenta Toro.
  • Prisma: Cria um prisma.
  • Cunha: Cria uma cunha.
  • Helix: Cria uma hélice.
  • Espiral: Cria uma espiral.
  • Círculo: Cria um arco circular.
  • Elipse: Cria um arco elíptico.
  • Ponto: Cria um ponto.
  • Linha: Cria uma linha.
  • Polígono regular: Cria um polígono regular.
  • Construtor de formas...: Cria formas a partir de vários formas primitivas.

Part tools toolbar

  • New Sketch: Creates a new sketch and opens the Sketcher Dialog to edit it.
  • Extrusão: Faz a extrusão de faces planas de um objeto.
  • Revolver: Cria um sólido através da revolução de outro objeto (não sólido) ao redor de um eixo.
  • Espelho: Cria uma peça simétrica de uma selecionada através de um plano de simetria.
  • Scale: Scales one or more shapes. introduced in 1.0
  • Filete: Arredonda as arestas de um objeto.
  • Chanfro: Cria um chanfro nas arestas de um objeto.
  • Fazer face a partir de fios: Faz uma face a partir de um conjunto de fios (contornos).
  • Superfície regrada: Cria uma superfície a partir de duas arestas ou dois arames.
  • Loft: Cria uma superfície (ou sólido) de um perfil a outro.
  • Varredura: Varre um ou mais perfis ao longo de um caminho.
  • Seção: Cria uma seção cruzando um objeto com um plano de seção.
  • Cruzamentos...: Cria uma ou mais seções transversais através de um objeto.
  • Deslocamento:
  • Deslocamento 3D: Constrói uma forma paralela a uma certa distância da forma original.
  • Deslocamento 2D: Constrói um arame paralelo a uma certa distância do arame original ou amplia/encolhe uma face plana.
  • Espessura: Utilitário para gerar uma espessura em um sólido ao selecionar faces.
  • Projeção na superfície: Projeta um logotipo, texto ou qualquer face, fio ou borda em uma superfície.
  • Cores definidas: Atribui cores a faces individuais de objetos.

Booleano

  • Composição:
  • Criar composição: Cria uma composição a partir dos objetos selecionados.
  • Explodir composição: Ferramenta para dividir composição de formas.
  • Filtro de composição: O filtro de composição pode ser utilizado para extrair peças individuais.
  • Operações booleanas: Realiza operações Booleanas sobre os objetos.
  • Recortar: Corta (subtrai) um objeto de outro.
  • União: Faz a união de dois objetos.
  • Interseção: Extrai a parte comum de dois objetos.
  • Juntar:
  • Conectar: Conecta interiores de paredes de objetos.
  • Embutir: Incorpora um objeto murado dentro de outro objeto murado.
  • Corte: Cria um corte em uma parede de um objeto para outro objeto murado.
  • Divisão:
  • Fragmentos booleanos: Cria todas as peças que podem ser obtidas por meio das operações booleanas entre objetos. Divide os objetos onde eles se interceptam.
  • Fatiar uma peça: Ferramenta para dividir formas através da intersecção com outras formas.
  • Fatiar: Divide um objeto em pedaços através da interseção com outro objeto.
  • XOR: Remove o espaço compartilhado por um número par de objetos (versão simétrica da Recortar).
  • Verificar geometria: Verifica a geometria dos objetos selecionados em busca de erros.
  • Desnaturalização: Remove recursos de um objeto.

Outras ferramentas

  • Caixa de seleção: Seleciona faces a partir de uma área retangular.
  • Forma a partir de malha: Cria uma forma a partir de um objeto de malha.
  • Pontos da malha: Cria um objeto feito de pontos a partir de um objeto geométrico.
  • Converter em sólido: Converte uma forma em um sólido.
  • Inverter formas: Inverte as normais de todas as faces do objeto selecionado.
  • Criar uma cópia:
  • Criar uma cópia simples: Cria uma cópia simples de um objeto selecionado.
  • Criar cópia transformada: Cria uma cópia transformada de um objeto selecionado.
  • Criar uma cópia do elemento forma: Cria uma cópia a partir de um elemento (vértice, borda, face) de um objeto selecionado.
  • Refinar a forma: Limpa as faces removendo linhas desnecessárias.
  • Set Tolerance: creates a parametric copy of selected objects with all contained tolerances set to at least a certain minimum value. introduced in 1.1
  • Persistent Section Cut: Creates persistent cuts of objects and assemblies.
  • Anexo: Prende um objeto a outro objeto.

Ferramentas obsoletas

  • Importar: Esta ferramenta te permite adicionar um arquivo *.IGES, *.STEP, *.BREP ao documento.
  • Exportar: Esta ferramenta te permite exportar um objeto como arquivo *.IGES, *.STEP ou *.BREP.

Medida

The Std Measure tool replaces the tools listed below. introduced in 1.0

  • Medida Linear: Cria uma medição linear.
  • Medida Angular: Cria uma medição angular.
  • Medida Atualização: Atualiza todas as medidas.
  • Limpar tudo: Limpa todas as medidas.
  • Alternar tudo: Mostra ou oculta todas as medidas.
  • Alternar 3D: Mostra ou oculta medidas 3D.
  • Alternar o Delta: Mostra ou esconde medidas delta.
