Description

The Robot InsertWaypoint command inserts a robot tool position into a trajectory.

Usage

  1. Select a robot and a trajectory in the Tree View
  2. There are several ways to invoke the command:
    • Press the Insert in Trajectory button.
    • Select the Robot → Insert in Trajectory option from the menu.
    • Use the keyboard shortcut: A.

Notes

To move the tool position of a robot do the following:

  1. Select the robot.
  2. Switch to the task panel.
  3. Drag the sliders.
