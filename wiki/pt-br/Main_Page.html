Introdução

Este é o wiki de documentação do FreeCAD. Há várias maneiras de usar esta documentação: explorando os hubs, seguindo o manual ou usando as entradas do menu Ajuda do FreeCAD. Este é um trabalho em andamento, escrito pela comunidade de usuários e desenvolvedores do FreeCAD. Se você encontrar informações incorretas ou ausentes, por favor, ajude o FreeCAD!

Os Tópicos

Hub de usuários: Esta página contém documentação útil para os usuários do FreeCAD em geral: uma lista de todas as bancadas, instruções detalhadas sobre como instalar e usar o aplicativo FreeCAD, tutoriais e tudo o que você precisa para começar.
Hub de usuários avançados: Esta página reúne documentação para usuários avançados e pessoas interessadas em escrever scripts em Python. Lá você também encontrará um repositório de macros, instruções sobre como instalá-las e usá-las, e mais informações sobre como personalizar o FreeCAD de acordo com suas necessidades específicas.
Hub de desenvolvedores: Esta seção contém material para desenvolvedores: como compilar o FreeCAD você mesmo, como o código fonte do FreeCAD é estruturado, como navegar pelo código fonte, como desenvolver novas bancadas e incorporar o FreeCAD em sua própria aplicação.

Manual

O manual do FreeCAD é outra maneira mais linear de apresentar as informações contidas nesta wiki. Ele foi projetado para ser lido como um livro e irá introduzir suavemente você a muitas outras páginas dos hubs acima. Versões em ebook também estão disponíveis, assim como algumas traduções em formato pdf.

Participe

Como participar

Há muito o que fazer dentro do projeto FreeCAD, caso você esteja interessado em nos ajudar. Claro, há tarefas de programação para programadores em C++ ou Python, mas também há muitas coisas que você pode fazer mesmo que não saiba programar, como:

  • escrever documentação e editar a wiki (pt-br)
  • ajudar iniciantes
  • traduzir o aplicativo e a documentação
  • ajudar com a embalagem da versão mais recente do FreeCAD para o seu sistema operacional favorito
  • ajudar outras pessoas ao seu redor a descobrir o FreeCAD.

A página ajude o FreeCAD descreve tudo com mais detalhes. A partir de 2016, o FreeCAD também participa do Google Summer of Code (GSoC).

Código fonte

O FreeCAD pode ser compilado em todas as plataformas usando CMake. O código-fonte é licenciado sob LGPL - licenciado e hospedado em GitHub e espelhado em GitLab, SourceForge e CodeBerg. Há instruções de compilação para Windows, Linux e MacOS. A documentação do código-fonte está hospedada aqui, gerada pelo Doxygen e documentado no wiki.

Sobre o desenvolvimento

Confira o Development roadmap para notícias sobre o que está sendo planejado, as páginas Changelog e Roadmap no FreeCAD tracker para ver o progresso em direção à próxima versão, ou as Project statistics para obter ainda mais informações sobre a base de código do FreeCAD. Toda a comunicação sobre desenvolvimento acontece no forum, portanto, não deixe de visitá-lo se tiver interesse em participar.
