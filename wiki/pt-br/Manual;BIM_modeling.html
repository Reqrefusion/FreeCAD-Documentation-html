Building Information Modeling (BIM) é um processo usado em arquitetura, engenharia e construção para criar e gerenciar representações digitais de estruturas físicas. Ela integra não apenas geometria 3D, mas também dados críticos como materiais, custos e cronogramas, permitindo análises avançadas e colaboração durante todo o ciclo de vida de um projeto.

No FreeCAD, a funcionalidade BIM evoluiu significativamente, especialmente com o lançamento da versão 1.0, onde as bancadas de trabalho Arch e BIM, antes separadas, foram fundidas em uma única bancada BIM integrada. Essa consolidação otimiza os fluxos de trabalho, permitindo que os usuários modelem, documentem e gerenciem projetos de construção com mais eficiência em um único ambiente.

Um grande avanço introduzido no FreeCAD v1.0 é a adoção do conceito IFC Nativo. Anteriormente, como a maioria dos aplicativos BIM, o FreeCAD traduzia dados entre seu modelo de dados interno e o formato de arquivo IFC (Industry Foundation Classes), o que levava à potencial perda de dados durante os processos de abertura e salvamento. Com o IFC Nativo, os usuários do FreeCAD agora podem abrir, manipular e salvar arquivos IFC diretamente, onde o próprio arquivo IFC serve como estrutura de dados. Essa abordagem elimina a tradução desnecessária de dados e garante que as modificações sejam salvas sem reescrever o arquivo inteiro, tornando-o compatível com sistemas de controle de versão como o Git e proporcionando um fluxo de trabalho mais transparente e preciso para o manuseio de arquivos IFC.

Neste capítulo, veremos como modelar este pequeno edifício:

  • Crie um novo documento e acesse o BIM Workbench.
  • Abra o menu Edit (Editar) → Preferences (Preferências) → Draft (Rascunho) → Gride and Snapping (Grade e Encaixe) e defina:
    • Main lines every (Linhas principais) 10.
    • Grid spacing (Espaçamento da grade) 1000mm para ter uma grade de um metro, o que é conveniente para o tamanho do nosso edifício.
    • Grid size (Tamanho da grade) 100 linhas.
  • Na snapping toolbar (barra de ferramentas de encaixe), certifique-se de que o botão Snap grid (Grade) esteja habilitado para que possamos usar a grade o máximo possível.
  • Se você não conseguir enxergar os eixos, clique no botão Toggle grid (Alternar grade).
  • Defina o Working Plane (Plano de Trabalho) como o plano XY
  • Desenhe quatro retas com a ferramenta Draft Line (Linha de Rascunho). Você pode inserir as coordenadas manualmente ou simplesmente selecionar os pontos na grade com o mouse. Usaremos metros para nossas dimensões:
    • Do ponto (0,0) ao ponto (0,3)
    • Do ponto (0,3) ao ponto (4,3)
    • Do ponto (4,3) ao ponto (4,0)
    • Do ponto (4,0) ao ponto (0,0)

Observe que desenhamos as linhas consistentemente na mesma direção (sentido horário). Embora isso não seja obrigatório, ajuda a garantir que as paredes, que construiremos em seguida, tenham orientações esquerda e direita consistentes. Você pode se perguntar por que não desenhamos apenas um retângulo, o que teria sido mais simples. No entanto, usar quatro linhas separadas nos dá a oportunidade de demonstrar funcionalidades BIM adicionais, como combinar vários objetos em um, o que é uma parte essencial do fluxo de trabalho.

  • Depois de criar as linhas, verifique seus pontos inicial e final e ajuste, se necessário, para que fiquem exatamente corretas.

  • Selecione as quatro linhas e pressione o botão Wall.
  • Defina a Height (Altura) das paredes como 3 m (padrão).
  • Defina a propriedade Aligment (Alinhamento) como left (esquerda). Definir a propriedade Alinhamento para a esquerda garante que as paredes criadas sejam posicionadas à esquerda das linhas desenhadas. No BIM Workbench do FreeCAD, as paredes são normalmente geradas com base em uma linha de referência, e o alinhamento à esquerda ou à direita determina de que lado da linha a parede será posicionada.

Se você não desenhou as linhas na mesma ordem indicada (sentido horário), a orientação de algumas paredes pode ser invertida, o que significa que elas podem ser posicionadas no lado oposto da linha (à direita em vez de à esquerda). Nesse caso, você precisaria ajustar o alinhamento à direita para essas paredes específicas para garantir que todas estejam alinhadas de forma consistente. Uma vez definido corretamente, você terá quatro paredes que se cruzam nos cantos, posicionadas na parte interna da linha de base, formando o layout desejado.

Após criar as paredes, o próximo passo é uni-las para que se interseccionem corretamente. Isso é necessário quando as paredes não se conectam perfeitamente em suas interseções. Para isso, selecione uma parede como "hospedeira" e adicione as outras paredes como "adições", mesclando sua geometria com a hospedeira. Todos os objetos no BIM Workbench podem ter múltiplas adições (que adicionam geometria) ou subtrações (que removem geometria). Essas relações podem ser gerenciadas a qualquer momento clicando duas vezes no objeto na árvore, permitindo ajustes flexíveis para garantir que as paredes e outros elementos arquitetônicos se integrem perfeitamente.

  • Selecione as quatro paredes com a tecla Template:TECLA pressionada, sendo a última a parede que você escolheu para se tornar a hospedeira.
  • Pressione o botão Add (Adicionar). As quatro paredes agora foram transformadas em uma:

No entanto, as paredes individuais ainda são acessíveis expandindo a parede na visualização em árvore.

  • Agora, vamos posicionar uma porta pressionando a ferramenta Door (Porta).
  • Comece selecionando a parede. Embora esta etapa não seja obrigatória, é um hábito útil a ser desenvolvido. Se um objeto for selecionado antes de iniciar uma operação, a operação será aplicada automaticamente a essa entidade por padrão.
  • Defina o Working Plane (Plano de Trabalho) como auto (automático) para que não fiquemos restritos ao plano do solo.
  • Pressione o botão Door (Porta).
  • No painel de criação de portas, selecione a predefinição Glass door (Porta de vidro) e defina sua Width (Largura) como 1 m e sua Height (Altura) como 2,1 m. Você notará que pode escolher entre vários tipos de portas e configurar seus parâmetros como desejar. No FreeCAD, uma porta é derivada por uma operação Window (janela).
  • Certifique-se de que a opção Snap near (Elemento Próximo) esteja ativada para que possamos ajustar as faces.
  • Posicione sua porta aproximadamente no meio da face frontal da parede:

  • Agora podemos definir a localização precisa expandindo os objetos parede e janela na visualização em árvore e alterando a propriedade Placement (Posicionamento) do esboço base da nossa porta. Defina sua posição como x = 0,5 m, y = 0, z = 0. Nossa porta agora está exatamente onde queremos:

  • Vamos posicionar uma janela ao lado da nossa porta. Selecione a parede, pressione a ferramenta Window (Janela), selecione a predefinição Open 2-pane (Abrir 2 painéis) e posicione uma janela de 1m x 1m na mesma face da porta. Defina o posicionamento do "underlying sketch" para x = 0, y = 0, z = 1,1m, de forma que a linha superior da janela fique alinhada com a parte superior da porta.

As janelas são sempre baseadas em esboços (sketches). Você pode criar janelas personalizadas facilmente desenhando primeiro um esboço (sketch) em uma face e, em seguida, transformando esse esboço (sketch) em uma janela, selecionando-a e clicando no botão da janela. Em seguida, você pode definir os parâmetros da janela — como quais partes do esboço (sketch) devem ser extrudadas e em que proporção — clicando duas vezes na janela na visualização em árvore. Agora, vamos criar uma laje:

  • Defina o Working Plane (Plano de Trabalho) como o plano XY
  • Crie um rectangle (retângulo) com length (comprimento) de 5 m, height (altura) de 4 m e posicione-o nas posições x:-0,5 m, y:-0,7 m, z:0.
  • Selecione o retângulo
  • Clique na ferramenta Slab (Laje) para criar uma laje a partir do retângulo
  • Mantenha os valores padrão de 0,2 m para a propriedade height (altura) e defina a direction (direção) normal como (0,0,-1), para que a extrusão seja para baixo. Embora pudéssemos ter movido o objeto 0,2 m para baixo, é uma boa prática manter os objetos extrudados alinhados com seu perfil base para manter a consistência e a precisão no modelo.
  • Defina a propriedade (property) Ifc Type (Tipo Ifc) da laje (slab) como slab (laje). Isso não é necessário no FreeCAD, mas é importante para a exportação IFC, pois garantirá que o objeto seja exportado com o tipo IFC correto.

  • Vamos colocar um roof (teto) sobre nossas cabeças agora. Podemos fazer isso facilmente usando a ferramenta Roof (Telhado).
  • Pressione a opção Snap working plane (Plano de trabalho) para habilitar o desenho em todos os planos.
  • Ao escolher uma das faces superiores da nossa casa, pressione o botão Select plane (Selecionar plano). O plano de trabalho agora está definido para essa face.
  • Crie um rectangle (retângulo), ajustando-o a dois pontos opostos das paredes:
  • Na aba data (dados) do telhado, defina o parâmetro Runs como 1600.
  • Se desejar alterar a cor do telhado, você pode fazê-lo na view tab (aba Visualizar)

With that, our model is now complete. The next step is to organize it properly to ensure it exports correctly to the IFC format. IFC files require all building elements to be grouped within a building object, and optionally, within a specific story. Additionally, all buildings must be located on a site. However, FreeCAD's IFC exporter will automatically generate a default site if one isn't present, so we don't need to add it manually. It's important to properly structure the model to comply with IFC standards, ensuring smooth collaboration and compatibility with other BIM software. Proper organization will also help avoid any data loss or errors during the export process.

  • Select the walls, the slab, and the roof.
  • Press the Floor button
  • Select the floor we just created
  • Press the Building button

Our model is now ready to export:

The IFC format is one of the most precious assets in a free BIM world, because it allows the exchange of data between any application and actor of the construction world, in an open manner (the format is open, free and maintained by an independent consortium). Exporting your BIM models as IFC ensures that anyone can see and analyze them, no matter the application used.

  • Select the top object you want to export, that is, the Building object.
  • Select menu File -> Export -> Industry Foundation Classes and save your file.
  • The resulting IFC file can now be opened in a wide range of applications and viewers (the image below shows the file opened in the free IfcPlusPlus viewer). Checking the exported file in such a viewer application before distributing it to other people is important to check that all the data contained in the file is correct. FreeCAD itself can also be used to re-open the resulting IFC file.

We can use the TechDraw Workbench to create a drawing of our building. The process is similar to what was shown in the previous section, so we won't go into too much detail here. Simply create a new view by using the insert Default Page option, then select the view you want to display in the drawing and add dimensions where necessary. This will allow us to create a professional 2D representation of the 3D model for documentation or presentation purposes.

Our page is now ready, and we can export it to SVG or DXF formats, or print it. The SVG format allows you to open the file using illustration applications such as Inkscape, with which you can quickly enhance technical drawings and turn them into much nicer presentation drawings. It offers many more possibilities than the DXF format.
