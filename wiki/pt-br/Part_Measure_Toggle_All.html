Default shortcut

Introduced in version

See also
Part Measure Toggle 3D, Part Measure Toggle Delta

Description

This command toggles the visibility of all measurements in the 3D view.

Usage

  1. There are several ways to invoke the command:
    • Press the Toggle All button.
    • Select the Measure → Toggle All option from the menu.
