Usage

Utilização

No rotation, default orientation - local axis 1 aligned with the global Y-axis (green)

90 degree rotation

Properties

DadosRotation: specifies the angle of rotation.

Limitations

  • Currently, multiple rotations are not supported (a single rotation is applied to all beams in the model).

Notes

  • To visualize the rotated cross-section it is necessary to set Beam Shell Result Output 3D in the FEM SolverCalculixCcxtools to True and run the analysis.
  • This feature uses the *BEAM SECTION card in CalculiX.
