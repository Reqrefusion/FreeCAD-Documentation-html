O FreeCAD é baseado no Qt framework e se caracteriza por uma interface simples e direta. Usuários de CAD mais experientes poderão identificar semelhanças com outros softwares, enquanto novos usuários acharão fácil navegar e descobrir as diversas opções que o FreeCAD oferece. Aqui está a aparência padrão do FreeCAD:

A Página Inicial serve como uma tela de boas-vindas, projetada para facilitar o acesso rápido e fácil às principais áreas do FreeCAD que um usuário possa desejar explorar. Através dela, os usuários podem criar novas peças, abrir arquivos recentes e iniciar esboços 2D sem esforço. Além disso, ela apresenta atalhos para recursos úteis, como tutoriais e fóruns de usuários, que são inestimáveis ​​tanto para iniciantes quanto para usuários experientes que buscam orientação ou dicas. Os usuários podem personalizar facilmente a aparência da Página Inicial de acordo com suas preferências.

À medida que você se familiariza com o FreeCAD, pode ajustar as configurações nas preferências. Isso pode configurar o FreeCAD para abrir diretamente em uma das bancadas de trabalho com um novo documento pronto para uso ao iniciá-lo. Como alternativa, você pode simplesmente fechar a aba Página Inicial e criar manualmente um novo documento.

"Workbenches" (Bancadas de trabalho)

O FreeCAD utiliza um sistema chamado "Workbenches", semelhante às estruturas conceituais usadas em softwares de design avançado como Revit ou CATIA. A ideia de uma Workbench é análoga às estações especializadas em um laboratório científico, onde diferentes estações de trabalho são equipadas para diferentes tipos de experimentos. Em um laboratório, você pode ter uma área dedicada à química, outra à biologia e uma terceira à física, cada uma equipada com as ferramentas específicas necessárias para cada disciplina.

No contexto do FreeCAD, cada Workbench é adaptado a um tipo específico de tarefa, organizando todas as ferramentas necessárias para essa atividade em uma única interface. Ao alternar entre Workbenchs, o conjunto de ferramentas e controles visíveis na interface do usuário se ajusta para refletir as necessidades da tarefa selecionada, embora o conteúdo real do projeto ou "cena" em que você está trabalhando não seja alterado. Isso permite transições suaves no fluxo de trabalho, como iniciar um projeto com formas 2D básicas no Draft Workbench e, em seguida, elaborar esses projetos com ferramentas de modelagem avançadas no Part Workbench.

Os termos "Workbench" e "Module" às vezes são usados ​​indistintamente, mas têm significados distintos dentro do FreeCAD. Um Module é qualquer extensão que adiciona funcionalidade ao FreeCAD, enquanto um Workbench é um tipo específico de Module equipado com seus próprios componentes de interface de usuário, como barras de ferramentas e menus, projetados para facilitar tipos específicos de tarefas. Portanto, todo Workbench é um Module, mas nem todo Module se qualifica como um Workbench.

O controle mais importante da interface do FreeCAD é o seletor de Workbench selector, que você usa para alternar de uma bancada de trabalho para outra:

  • A Assembly workbench para construir e resolver montagens mecânicas. introduzido na versão 1.0
  • O BIM Workbench para trabalhar com elementos arquitetônicos e criar modelos BIM. Ele combina o Arch Workbench e o antigo BIM Workbench externo, disponível em 0.21 and below.
  • O CAM Workbench é usado para produzir instruções em G-Code. Este workbench era chamado de "Path Workbench" em 0.21 and below.
  • O Draft Workbench contém ferramentas 2D e operações básicas de CAD 2D e 3D.
  • O FEM Workbench fornece fluxo de trabalho de Análise de Elementos Finitos (FEA).
  • O Inspection Workbench foi criado para fornecer ferramentas específicas para o exame de formas. Ainda em estágios iniciais de desenvolvimento.
  • O Material Workbench gerencia o sistema de materiais do FreeCAD. introduced in 1.0
  • O Mesh Workbench para trabalhar com malhas trianguladas.
  • O OpenSCAD Workbench para interoperabilidade com o OpenSCAD e reparo do histórico do modelo de constructive solid geometry (CSG).
  • O Part Workbench para trabalhar com primitivas geométricas e operações booleanas.
  • A Part Design Workbench para criar formas de peças a partir de esboços.
  • O Points Workbench para trabalhar com nuvens de pontos.
  • O Reverse Engineering Workbench tem como objetivo fornecer ferramentas específicas para converter formas/sólidos/malhas em recursos paramétricos compatíveis com o FreeCAD.
  • A Robot Workbench para estudar os movimentos dos robôs. Atualmente sem manutenção.
  • O Sketcher Workbench para trabalhar com esboços com restrições geométricas.
  • O Spreadsheet Workbench para criar e manipular dados de planilhas.
  • O Surface Workbench fornece ferramentas para criar e modificar superfícies. É semelhante à opção Face a partir das arestas do Part Builder.
  • O TechDraw Workbench para produzir desenhos técnicos a partir de modelos 3D. É o sucessor do Drawing Workbench.
  • O Test Framework Workbench é para depuração do FreeCAD.

As bancadas de trabalho costumam confundir os novos usuários, pois nem sempre é fácil saber em qual bancada procurar uma ferramenta específica. Mas elas são fáceis de aprender e, em pouco tempo, se tornarão naturais. Novos usuários rapidamente percebem que as bancadas de trabalho são uma maneira conveniente de organizar a infinidade de ferramentas que o FreeCAD oferece. Além disso, as bancadas de trabalho também são totalmente personalizáveis.

A interface

Vamos dar uma olhada mais detalhada nas diferentes partes da interface:

A janela principal do aplicativo pode ser dividida em 11 seções:

  1. A Main View area, que pode conter diferentes janelas com abas.
  2. A 3D view, normalmente incorporada na main view area. A 3D view é o elemento central da interface, exibindo e permitindo a manipulação dos objetos nos quais você está trabalhando. É possível ter múltiplas visualizações do mesmo documento (ou objetos) ou ter vários documentos abertos simultaneamente. Além disso, cada visualização pode ser separada da janela principal separadamente.
  3. O Model e a Tasks tab.
    1. A Model tab mostra o conteúdo e a estrutura do seu documento.
    2. A Tasks tab é onde o FreeCAD solicitará valores específicos para a bancada e a ferramenta que você está usando no momento (por exemplo, dimensões de um objeto).
  4. O Property editor, que aparece quando a aba Modelo está ativa na interface. Permite gerenciar as propriedades publicamente expostas dos objetos no documento. Consiste nas seções Dados e Visualização, exibindo as propriedades de visualização e as propriedades paramétricas dos objetos, respectivamente.
  5. A Selection view, que indica os objetos ou subelementos de objetos (vértices, arestas, faces) selecionados.
  6. A Report view, onde mensagens, avisos e erros são exibidos.
  7. O Python console. O console Python, onde todos os comandos executados são impressos e onde você pode inserir código Python.
  8. A Status bar, onde algumas mensagens e dicas de ferramentas aparecem.
  9. A área da barra de ferramentas, onde as barras de ferramentas são encaixadas.
  10. O workbench selector, onde você seleciona a bancada ativa.
  11. O standard menu, que contém as operações básicas do programa.

A maioria dos painéis acima podem ser ocultados ou revelados usando o menu View → Panels menu

Personalizando a interface

A interface do FreeCAD foi projetada para ampla personalização. Todas as barras de ferramentas e painéis podem ser realocados, empilhados ou até mesmo encaixados em diversas configurações, de acordo com a preferência do usuário. Além disso, eles podem ser fechados e reabertos conforme necessário. Além desses recursos, os usuários têm inúmeras outras opções, incluindo a capacidade de criar barras de ferramentas personalizadas com ferramentas selecionadas de qualquer uma das bancadas de trabalho disponíveis e de atribuir ou modificar atalhos de teclado para otimizar o fluxo de trabalho. Essa flexibilidade garante que os usuários possam personalizar o ambiente para atender às suas necessidades e preferências específicas.
