Introdução

O Menu de Ferramentas Padrão é um dos 7 submenus no menu padrãomenu padrão:

Arquivo     Editar     Vista     Ferramentas     Macro     Janelas     Ajuda

O menu Ferramentas fornece ferramentas para depuração de modelos, personalização do comportamento do FreeCAD, bem como ferramentas auxiliares.

Ferramentas

As seguintes ferramentas estão disponíveis neste menu:

The order of the tools matches the version 1.1 menu.

  • Gerenciador de addons: abre o gerenciador de complementos com o qual as bancadas de trabalho externas e macros podem ser instaladas.
  • Medida:introduced in 1.0
  • Clarify Selection: Lists all objects of different types located near the cursor to facilitate their selection. introduced in 1.1
  • Calculadora de unidades...: Abre uma calculadora para a conversão de unidades.
  • Carregar imagem...: Carrega uma imagem. introduced in 0.21
  • * Salvar imagem...: Salva uma captura de tela da visualização 3D ativa.
  • Adicionar documento de texto: cria um objeto capaz de conter texto.
  • Ver toca-discos...:: Gira continuamente a câmera na visualização 3D ativa.
  • Inspetor de cena...: Exibe uma visão geral dos nós do cenógrafo da visualização 3D ativa.
  • Gráfico de dependência...: Mostra o gráfico de dependência de objetos.
  • Gráfico de dependência de exportação... introduced in 0.21
  • Utilitário de documento...: Auxilia na reparação de arquivos de projeto danificados.
  • Editar parametros...: Abre uma caixa de diálogo que dá acesso aos parâmetros que controlam o programa.
  • Personalizar...: Abre uma caixa de diálogo com opções de personalização.
