Description

The Spreadsheet AlignVCenter tool aligns the contents of selected cells to the vertical center.

Usage

  1. Make sure there is an active spreadsheet.
  2. Select one or more cells.
  3. There are several ways to invoke the tool:
    • Press the Align Vertical Center button.
    • Select the Spreadsheet → Alignment → Align Vertical Center option from the menu.
