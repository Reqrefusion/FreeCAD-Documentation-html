Description

The Spreadsheet MergeCells tool merges a rectangular cell range.

Usage

  1. Make sure there is an active spreadsheet.
  2. Select two or more cells that form a rectangular cell range. The range may include already merged cells.
  3. There are several ways to invoke the tool:
    • Press the Merge Cells button.
    • Select the Spreadsheet → Merge Cells option from the menu.
    • Right-click one of the selected cells and select the Merge Cells option from the context menu.
