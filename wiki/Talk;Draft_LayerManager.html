Layers are an organizational tool. The layers implementation the BIM workbench offers options to control several properties of your objects:

  • Graphical appearance
  • Layer properties for export to vector formats like DXF & [FreeCAD_and_DWG_Import|DWG]]
  • NativeIFC properties

BIM layers are not activated by default in a new file.

Use the Autogroup button from the Draft_Tray to enable layers.

Use the Layer Manager button from the Manage Toolbar to edit layer properties.

Because usage of BIM_Layers is almost identical to Draft Layers, that documentation is useful for BIM_Workbench users, it is not duplicated here.

Unique to BIM Layers are options for managing NativeIFC properties. This is a work in progress. Users can read Yorik's blog for updates from a lead developer of NativeIFC functionality in FreeCAD. --Duncan.lithgow (talk) 13:16, 4 January 2026 (UTC)Reply
