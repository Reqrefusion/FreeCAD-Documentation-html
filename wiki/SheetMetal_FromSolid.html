Description

The SheetMetal FromSolid converts a solid or shell to a sheet metal object (introduced in version 0.8.0).

Usage

  1. Select a base object (a solid or a shell).
  2. There are several ways to invoke the command:
    • Press the Convert to Sheet Metal button.
    • Select the Sheet Metal → Convert to Sheet Metal option from the menu.
    • Right-click in the Tree View or the 3D View and select the Sheet Metal → Convert to Sheet Metal option from the context menu.
    • Use the keyboard shortcut: C then S.
  3. A SolidToSheet object is created, and the Convert to Sheet Metal Task Panel opens:
    The Faces to remove and seam edges list is empty if the base object was selected in the Tree View, or it contains the subelements that were used to select the base object in the 3D View.
    • Adjust the parameters in the Task Panel.
    • Optionally press the Select button to edit the Faces to remove and seam edges list.
      (A Preview button indicates the list edit mode)
      • Optionally select faces/edges in the 3D View to add them to the list, or remove them respectively.
      • Optionally press the Clear All to remove all items from the Faces to remove and seam edges list.
      • Optionally check list items and press the Clear Selected to remove only the selected items from the Faces to remove and seam edges list.
      • Press the Preview button to confirm and finish editing.
  4. Press the OK button to close the Task Panel and finish the command.
  5. Optionally adjust the profile parameters in the Property View.

Task Panel

Double-click an existing SolidToSheet object in the Tree View to re-open the task panel and edit the parameters.

  • Thickness: Sets the Thickness property.
  • Bend Radius: Sets the Radius property.
  • Invert: Toggles the Invert property; .
  • Select: Activates the edit mode that allows to add the list of faces to remove and/or seam edges within the base Object property. If pressed:
    • Preview: Indicates edit mode. Press to finish the edit mode (hidden until Select is pressed).
    • Clear All: Press to clear the list (hidden until Select is pressed). Changes to Clear Selected if at least one list item is checked; removes only the selected items.

Properties

See also: Property View.

A SheetMetal SolidToSheet object is derived from a Part Feature object or, if it is inside a PartDesign Body, from a PartDesign Feature object, and inherits all its properties. It also has the following additional properties:

Data

Parameters

  • DataInvert (Bool): "Invert bend direction". Controls whether the wall thickness builds up inside or outside the base object.
  • DataRadius (Length): Inner radius of automatically added bends.
  • DataThickness (Length): Wall thickness of the sheet metal shape.
  • Database Object (LinkSub): "Base object". Link to the solid/shell and its selected sub-elements.
