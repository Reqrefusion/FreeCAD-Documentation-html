Descrizione

CSV2Objects è una macro di FreeCAD che genera grandi quantità di oggetti di testo 3D con valori basati su CSV posizionati su linee guida orizzontali dello schizzo.

Mappa le colonne CSV sulle linee guida, crea ShapeString sul piano dello schizzo, le estrude, opzionalmente le fonde con un solido di base e supporta l'esportazione batch in STL, 3MF e STEP. Lo stato originale del documento viene ripristinato dopo l'esportazione finale.

Funzionalità

  • Reads CSV files (configurable encoding and delimiter) with row/column information
  • Maps CSV columns to horizontal sketch guide lines
  • Per-line controls:
    • Alignment (Left / Center / Right / Auto-fit)
    • Text height
    • Extrusion height
  • Live preview with row navigator (preview any CSV row)
  • Optional Adopt preview as object mode to create persistent editable objects without exporting
  • Optional boolean fuse with a target body (per generated row)
  • System font picker with fallback to custom TTF/OTF font files
  • Export formats: STL, 3MF, STEP
  • Output folder is created next to the current ``.FCStd`` file
  • Filenames include the document name and used CSV values (sanitized); existing exports of the same type are replaced
  • Document is reverted to its original state after export

Installazione

È possibile installare la macro in diversi modi:

  • Tramite Addon Manager (scheda Macro) una volta indicizzata questa macro.
  • Manualmente scaricando il file della macro da GitHub:

https://raw.githubusercontent.com/DasLukas/CSV2Objects/main/CSV2Objects.FCMacro

e copiandola nella directory delle macro di FreeCAD.

Percorsi di installazione manuale

  • macOS: ``~/Library/Preferences/FreeCAD/Macro``
  • Windows: ``%APPDATA%\\FreeCAD\\Macro``
  • Linux: ``~/.FreeCAD/Macro`` (or ``~/.local/share/FreeCAD/Macro`` su alcuni sistemi)

Utilizzo

  1. Open a saved document containing a sketch with horizontal guide lines (optionally with a target solid).
  2. Run the macro CSV2Objects.FCMacro from the Macro menu.
  3. In the task panel, choose a CSV file, encoding and delimiter, then click Load CSV.
  4. Pick a sketch (guide lines are detected automatically) and assign CSV columns to each active line.
  5. Choose alignment (or Auto-fit), per-line text height, and per-line extrusion height.
  6. Select a font, extrusion mode and export format (STL / 3MF / STEP).
  7. Use the live preview to adjust your settings (optional).
  8. Press OK to generate and export geometry for all CSV rows.
  9. Press Cancel to clear preview objects and close the dialog.

Note

  • Il documento deve essere salvato prima dell'esportazione per poter determinare la cartella di output.
  • La sovrapposizione "X" nel pannello Attività attiva/disattiva la visibilità del pannello e non chiude la finestra di dialogo: utilizzare invece Annulla.
