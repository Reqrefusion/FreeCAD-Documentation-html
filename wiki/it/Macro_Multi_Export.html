Descrizione

Questa macro offre un modo pratico per esportare gli oggetti FreeCAD selezionati contemporaneamente in più formati di file. Invece di esportare in ogni formato singolarmente, è possibile selezionare tutti i formati desiderati in un'unica finestra di dialogo ed esportarli simultaneamente.

Formati di esportazione supportati:

  • STL - Stereolitografia (comune per la stampa 3D)
  • STEP - Standard per lo scambio di dati di prodotto (interscambio CAD)
  • 3MF - Formato di produzione 3D (formato moderno per la stampa 3D)
  • OBJ - Wavefront OBJ (grafica 3D e motori di gioco)
  • IGES - Initial Graphics Exchange Specification (formato CAD legacy)
  • BREP - Formato nativo OpenCASCADE (preserva la geometria esatta)
  • PLY - Formato file poligonale (scansione e stampa 3D)
  • AMF - Formato di produzione additiva (stampa 3D basata su XML)

Caratteristiche:

  • Esportazione di più formati in un'unica operazione.
  • Finestra di dialogo intuitiva per la selezione del formato.
  • Cartella di output e nome file di base configurabili.
  • Impostazioni di tolleranza della mesh per le esportazioni in formato STL/OBJ/PLY/3MF/AMF.
  • Anteprima dei file da creare.
  • Pulsanti di selezione rapida: "Select All", "Select None", "Reset Defaults".
  • Formati predefiniti: STL, STEP e 3MF sono preselezionati.

Utilizzo

  1. Selezionare uno o più oggetti nella vista 3D che si desidera esportare.
  2. Eseguire la macro da Macro → Macro → MultiExport → Esegui.
  3. Nella finestra di dialogo:
    • Rivedere gli oggetti selezionati nella sezione "Objects to Export".
    • Selezionare i formati di esportazione desiderati (STL, STEP, 3MF sono selezionati per impostazione predefinita).
    • Scegliere una cartella di output utilizzando il pulsante "Browse...".
    • Inserire un nome file di base (l'estensione del formato verrà aggiunta automaticamente).
    • Facoltativamente, sistemare le impostazioni di tolleranza della mesh per i formati basati su mesh.
  4. Fai clic su "Export" per creare i file.

Requisiti

  • FreeCAD 0.21 o versioni successive.
  • Oggetti con forme solide (corpi Part o PartDesign).

Installazione

  • Installare tramite il FreeCAD Addon Manager o manualmente dal repository freecad-macro-3d-print-multi-export su GitHub.
