Descrizione

Lo strumento TechDraw Vista interrotta inserisce una "vista interrotta" basata su una vista esistente o su uno o più oggetti, come Corpi o Parti. La Vista interrotta richiede inoltre uno o più schizzi che definiscano la posizione e la dimensione delle aree da rimuovere. Vista interrotta si comporta in modo simile alle altre visualizzazioni. La direzione di proiezione viene presa dalla vista della parte esistente, dalla direzione della telecamera 3D o dalla normale di una faccia selezionata.

Utilizzo

  1. Facoltativamente ruotare la Vista 3D. La vista 3D determina i valori iniziali delle proprietà DatiDirection e DatiXDirection della vista interrotta.
  2. Selezionare l'oggetto di cui si desidera creare una vista interrotta o selezionare una vista TechDraw esistente che contiene questo oggetto.
  3. Aggiungere uno o più schizzi di interruzione alla selezione selezionandoli nella Vista ad albero. Ogni schizzo dovrebbe contenere solo due linee parallele. E' possibile anche usare altri oggetti con due bordi paralleli.
  4. Esistono diversi modi per richiamare lo strumento:
    • Premere il pulsante Inserisci Vista interrotta.
    • Selezionare l'opzione TechDraw → Viste TechDraw → Inserisci Vista interrotta dal menu.

Proprietà

Vedere anche: Editor delle proprietà.

Una Vista Interrotta, formalmente un oggetto TechDraw::DrawBrokenView, deriva da un oggetto Part View, formalmente un oggetto TechDraw::DrawViewPart ed eredita tutti le sue proprietà. Ha inoltre le seguenti proprietà aggiuntive:

Dati

Broken View

  • DatiBreaks (LinkList): oggetti nella vista 3D che definiscono i punti di inizio/fine e la direzione delle interruzioni in questa vista.
  • DatiGap (Length): la distanza di separazione per le interruzioni in questa vista (lunghezza 3D non scalata).

Note

  • Le interruzioni devono essere verticali od orizzontali. Le interruzioni oblique non sono supportate.
  • Vedere anche TechDraw Vista.

Script

Vedere anche: Autogenerated API documentation e Script di base per FreeCAD.

Vista interrotta può essere creato con macro e dalla console Python utilizzando le seguenti funzioni: 

doc = FreeCAD.ActiveDocument
box = doc.Box
profile = doc.Sketch
page = doc.Page

brokenView = doc.addObject("TechDraw::DrawBrokenView", "BrokenView")
page.addView(brokenView)
brokenView.Source= box
brokenView.Breaks = [doc.Sketch]
