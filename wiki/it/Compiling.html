Docker container

Compilare in Windows Compilare in Linux [Compilare in MacOS] Compilare in Docker

Per compilare su BSD e altri sistemi Unix, è possibile seguire le istruzioni generali di Compilazione in Linux ma utilizzando le dipendenze appropriate per tali sistemi.

Su Windows è anche possibile compilare FreeCAD seguendo le istruzioni di Compilazione in Windows o usando MSYS2/MinGW seguendo Compilare in MinGW.

Per compilare FreeCAD senza la sua interfaccia utente grafica, utilizzare l'immagine FreeCAD Docker CLI mode.

Per compilare la documentazione di programmazione, vedere Documentazione del codice sorgente.

Per imparare a usare il sistema di controllo di revisione e contribuire con il codice, vedere Gestione del codice sorgente.
