Panoramica

WebTools questo è un ambiente di lavoro esterno contenente una serie di strumenti per comunicare con i servizi Web dall'interno FreeCAD.

WebTools is an external workbench containing a series of tools to communicate with Web services from within FreeCAD.

Installazione

Questo ambiente può essere installato da Addon Manager. Per l'installazione manuale vedere Installazione di altri ambienti di lavoro.

Gli strumenti

  • BIM server: Apre una finestra BimServer
  • Git: Gestisce file con GIT
  • Sketchfab: Esporta e carica oggetti su un account SketchFab.

Link

  • Codice sorgente ospitato su GitHub: https://github.com/yorikvanhavre/WebTools
