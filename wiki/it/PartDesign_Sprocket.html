Descrizione

This tool allows you to create a 2D profile of a sprocket (or chainwheel). It can be padded with the PartDesign Pad feature.

Utilizzo

  1. Optionally activate the correct body.
  2. Select the Part Design → Sprocket... option from the menu.
  3. Set the DatiNumber Of Teeth and the DatiSprocket Reference.
  4. Click OK.
  5. If there was no active body: drag and drop the sprocket into a body for the application of further features like padding.

Proprietà

  • DatiNumber Of Teeth: Number of teeth
  • DatiSprocket Reference: The sprocket type. A list of sprocket definitions. introduced in 0.20 The list includes ANSI and ISO-norms as well as some bicycle and motorcycle sprocket definitions.
  • DatiPitch: Distance between two teeth
  • DatiRoller Diameter: Diameter of the chain rollers the sprocket is designed for
  • DatiThickness:The principal thickness of the sprocket. Note: The sprocket cannot just be padded with this thickness because the teeth have side chamfers. Therefore look at the sprocket definition to model a valid 3D sprocket. introduced in 0.20
