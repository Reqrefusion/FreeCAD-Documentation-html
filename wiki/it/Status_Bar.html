Introduzione

La Barra di Stato è una barra multifunzione che appare nella parte inferiore dell'interfaccia di FreeCAD. Può visualizzare diverse informazioni e diversi widget. Gli ambienti di lavoro BIM e Draft aggiungono i propri widget alla Barra di Stato. È anche possibile trascinare una barra degli strumenti nella Barra di Stato.

Informazione e widget

Informazioni sull'utilizzo

Quando il puntatore si trova sopra un pulsante o una voce di menu, le informazioni sull'utilizzo di quello strumento vengono visualizzate nella barra di stato. Queste informazioni sono solitamente le stesse della descrizione comandi dello strumento.

Informazioni preselezionate

Quando il puntatore si trova sopra un oggetto nella Vista 3D, la barra di stato mostra l'elemento preselezionato dell'oggetto e le coordinate 3D correnti. Queste informazioni vengono aggiornate man mano che il puntatore viene spostato sopra l'oggetto. L'aggiornamento si interrompe quando il puntatore si trova sopra uno spazio vuoto nella Vista 3D.

Suggerimenti di input

disponibile dalla versione 1.1

Per la maggior parte degli strumenti di Sketcher e degli strumenti di Draft, i suggerimenti di input dinamici vengono visualizzati nella barra di stato.

Misura rapida

disponibile dalla versione 1.0

La funzione Misurazione rapida utilizza il lato destro della barra di stato per mostrare le misurazioni in base alla selezione nella vista 3D.

A seconda della selezione vengono visualizzate le seguenti misure:

  • Lunghezza del bordo
  • Area della faccia
  • Distanza tra punti, bordi o facce
  • Angolo tra i bordi
  • Raggio di bordi circolari o di facce cilindriche
  • Diametro di cerchi e facce cilindriche, disponibile dalla versione 1.1.
  • Distanza dell'asse tra elementi circolari e facce, bordi o altri elementi circolari, disponibile dalla versione 1.1.

Indicatore di navigazione

L'indicatore di navigazione mostra lo stile corrente di Navigazione con il mouse. Cliccando sull'indicatore si apre un menu con le modalità disponibili e un sottomenu Impostazioni con le seguenti opzioni:

  • Stile Orbita: Vedi Editor delle preferenze.
  • Compatto: Se questa opzione è selezionata, nel widget verrà visualizzata solo l'icona del mouse.
  • Suggerimento: Se questa opzione è selezionata, passando il mouse sopra il widget verrà visualizzata una descrizione del comando con informazioni dettagliate sull'utilizzo.

Indicatore di zoom

La barra di stato fornisce informazioni sul livello di zoom mostrando la dimensione della Vista 3D corrente. Facendo clic sull'indicatore si apre un menu per modificare il sistema di unità del documento corrente o il sistema di unità predefinito se non è presente alcun documento attivo.

Menu contestuale

Right-clicking the Status Bar opens a context menu with the following options:

  • introduced in 1.1: Input hints: Checkbox to toggle this feature.
  • introduced in 1.1: Quick measure: Idem.
  • Notification Area: Idem.
  • Navigation indicator: Idem.

If the pointer is over the Preselected information field or the Quick measure field the following options are added:

  • Copy: Copies the information selected with the next option.
  • Select All: Selects the current information.

The Notification Area has its own context menu.
