Introduzione

Per impostare e manipolare la tabella dei parametri di FreeCAD normalmente si usa l'Editor delle preferenze nel menu Modifica → Preferenze.

Tuttavia, è anche possibile accedere, modificare e creare i parametri manualmente, usando l'Editor dei parametri che si trova nel menu Strumenti → Modifica parametri.

Questa pagina elenca i parametri che non sono accessibili tramite l'editor delle preferenze, ma che è possibile impostare manualmente per ottimizzare l'installazione di FreeCAD o superare i problemi. Tutti i parametri si trovano in BaseApp/Preferences/.

Generale

  • Bitmaps/Theme/Name (string): Specificare il nome del tema delle icone per sovrascrivere il tema delle icone di sistema utilizzato da FreeCAD. Utilizzato solo se Bitmaps/Theme/UseIconTheme è true.
  • Bitmaps/Theme/ThemeSearchPaths (boolean): Impostare su false in modo che FreeCAD utilizzi le icone incluse invece del tema delle icone di sistema su Linux. versione 0.21 e precedenti. Per le versioni successive, utilizzare invece Bitmaps/Theme/UseIconTheme.
  • Bitmaps/Theme/UseIconTheme (boolean): Impostare su true per forzare Qt a utilizzare le icone del tema di sistema. Il valore predefinito è false, quindi FreeCAD utilizzerà le proprie icone. Non influisce sugli altri meccanismi del tema di sistema di Qt, come finestre di dialogo, pulsanti e altri. Questi dovrebbero sempre utilizzare le icone del tema di sistema. disponibile dalla versione 1.0
  • Dialog/DontUseNativeColorDialog (boolean): Impostazione della finestra di dialogo del selettore colore. Impostare su false se si desidera che FreeCAD utilizzi la finestra di dialogo dei colori nativa del sistema anziché la finestra di dialogo dei colori di Qt. Il valore predefinito è true.
  • Dialog/DontUseNativeDialog (boolean): Impostazione della finestra di dialogo File. Impostare su false se si desidera utilizzare la finestra di dialogo File nativa all'apertura dei file o su true per utilizzare la finestra di dialogo Selezione file Qt. L'impostazione predefinita dipende da un'impostazione in fase di compilazione: #define (USE_QT_FILEDIALOG).
  • Dialog/DontUseNativeFontDialog (boolean): Utilizzato dal comando ShapeString di Draft. Impostare su false per utilizzare la finestra di dialogo Font nativa. Il valore predefinito è true.
  • DockWindows/DAGView/Enabled (boolean): Impostare su true per abilitare un widget agganciabile beta DAG view. Dopo aver modificato il valore del parametro, è necessario riavviare FreeCAD affinché il widget sia disponibile nell'elenco Visualizza/Pannelli..
  • DockWindows/PropertyView/Enabled (boolean): Impostare su true per abilitare un widget agganciabile Vista Proprietà indipendente dalla Vista Combinata. Dopo aver modificato il valore del parametro, è necessario riavviare FreeCAD affinché il widget sia disponibile nell'elenco Visualizza/Pannelli.
  • DockWindows/TreeView/Enabled (boolean): Impostare su true per abilitare un widget agganciabile Vista ad albero indipendente dalla Vista combinata. Dopo aver modificato il valore del parametro, è necessario riavviare FreeCAD affinché il widget sia disponibile nell'elenco Visualizza/Pannelli..
  • Document/AutoNameDynamicProperty (boolean): Impostare su true per fare in modo che FreeCAD rinomini automaticamente le proprietà dinamiche con un nome specificato non valido, invece di generare un'eccezione. Nota che il codice Python non avrà accesso al nuovo nome.
  • Document/ChangeViewProviderTouchDocument (boolean): Impostare su false in modo che le modifiche alla visibilità degli elementi non contrassegnino il documento come modificato.
  • Document/SaveThumbnailFix (boolean): Impostare su true per risolvere un problema con Qt5 che impedisce la generazione di miniature di file .FCStd.
  • General/LockToolBars (boolean): Impostare su true per impedire che le barre degli strumenti siano trascinabili e per nascondere le piccole maniglie di trascinamento. Utilizzato principalmente in combinazione con fogli di stile che rendono le barre degli strumenti verticali.
  • General/RecentIncludesExported (boolean): Impostare su true per includere i file esportati nell'elenco File recenti. Il valore predefinito è false.
  • General/RecentIncludesImported (boolean): Impostare su false per escludere i file importati dall'elenco File recenti. Il valore predefinito è true.
  • General/ShowSplasherMessages (boolean): Impostare su false per ignorare la visualizzazione dei messaggi nella schermata iniziale. Questo potrebbe ridurre il tempo di avvio di FreeCAD. Il valore predefinito è true.
  • Macro/DuplicateFrom001 (boolean): Impostare su true per iniziare sempre la ricerca del nome file macro duplicato suggerito con @001 anziché con @NNN, se applicabile. Il valore predefinito è false.
  • Macro/DuplicateIgnoreExtraNote (boolean): Impostare su true per ignorare la nota aggiuntiva quando si suggerisce un nome file macro duplicato. La nota aggiuntiva è il testo nel nome file che segue "@NNN" e prima di ".FCMacro". Esempio: "my_macro@005.my_note.FCMacro". Se impostato su true, il nome file suggerito successivo è "my_macro@006.FCMacro". Se impostato su false, il nome file suggerito successivo è "my_macro@006.my_note.FCMacro". Per essere riconosciuto come nota aggiuntiva, il testo deve iniziare con un punto (".") dopo "@NNN". In caso contrario, ad esempio, "my_macro@006_my_note.FCMacro" riceve "my_macro@006_my_note@001.FCMacro" come nuovo nome file suggerito, il che potrebbe essere opportuno in alcuni casi. Il valore predefinito è false.
  • Macro/ReplaceSpaces (boolean): Impostare su false se non si desidera che gli spazi nei nomi dei file vengano automaticamente convertiti in caratteri di sottolineatura durante la creazione, la ridenominazione o la duplicazione di una macro. Non influisce sui file esistenti, ma solo quando si crea un nuovo file o si rinomina o si duplica un file esistente. Il valore predefinito è true.
  • MainWindow/ClearMenuBar (boolean): Impostare su true per cancellare la barra dei menu al cambio del workbench, utile quando si utilizza un menu globale, in quanto potrebbe non aggiornarsi al cambio del workbench e riempirsi rapidamente con le voci di menu di ciascun workbench. Il valore predefinito è false. Su macOS viene cancellato in entrambi i casi per aggirare un bug di Qt.
  • MainWindow/ToolBarNameAsToolTip (boolean): Impostare su false per non visualizzare il nome della barra degli strumenti come suggerimento. Il valore predefinito è true.
  • PropertyView/AutoTransactionView (boolean): Impostare su true in modo che le modifiche alle proprietà della scheda Visualizza vengano aggiunte allo stack di annullamento (e quindi siano annullabili). Il valore predefinito è false.
  • Selection/AutoShowSelectionView (boolean): Impostare su true per visualizzare automaticamente il riquadro Visualizzazione selezione quando si seleziona qualcosa. Il valore predefinito è false.
  • Selection/singleClickFeatureSelect (boolean): Impostare su false per disabilitare la selezione con un singolo clic di una feature in PartDesign. Il valore predefinito è true.
  • TreeView/HideColumn (boolean): Impostare su true per nascondere la colonna 'Descrizione' nella Vista ad albero. Il valore predefinito è false.
  • TreeView/TreeViewStretchDescription (boolean): Impostare su true per estendere la colonna "Descrizione" nella Vista ad albero fino al bordo destro del riquadro. Il valore predefinito è false.
  • Units/PropertyVectorListCopyPrecision (integer): Numero di cifre della precisione da utilizzare quando si copia una tabella di vettori dal menu contestuale agli appunti di sistema quando si utilizza l'editor delle proprietà per le proprietà di tipo App::PropertyVectorList. Il valore predefinito è 16. disponibile dalla versione 1.1
  • View/AxisLetterColor (unsigned): Colore per le lettere del sistema di coordinate mostrato nell'angolo in basso a destra della vista 3D. Il valore predefinito è 255. Vedere qui per informazioni sul codice del colore.
  • View/AxisXColor (unsigned): Colore per gli elementi dell'asse X del gizmo Trasforma. Il valore predefinito è 3425907456. Vedere qui per informazioni sul codice del colore.
  • View/AxisYColor (unsigned): Idem per gli elementi dell'asse Y. Il valore predefinito è 869020416.
  • View/AxisZColor (unsigned): Idem per gli elementi dell'asse Z. Il valore predefinito è 859032576.
  • View/LocalCoordinateSystemSize (float): Dimensione dei sistemi di coordinate locali. Il valore predefinito è 2.0.
  • View/NavigationDebug (boolean): Abilita l'output di debug degli stili di navigazione (a partire dalla versione v0.19, solo lo stile di navigazione Gesture può segnalare qualcosa).
  • View/SavePicture (string): Impostare su FramebufferObject, PixelBuffer o CoinOffscreenRenderer per metodi diversi nel produrre immagini dalla vista 3D.
  • View/ToggleTransparency (integer): La massima trasparenza in percentuale che l'utente richiede per passare da zero. Il valore predefinito è 70. disponibile dalla versione 1.1

Nome file di esportazione predefinito

  • General/ExportDefaultFilenameMultiple (string): Impostare il nome file predefinito da utilizzare quando si esportano più oggetti. Il valore predefinito è %F.
  • General/ExportDefaultFilenameSingle (string): Impostare il nome file predefinito da utilizzare quando si esporta un singolo oggetto. Il valore predefinito è %F-%P-.

Entrambe queste opzioni supportano l'inserimento automatico di varie informazioni nel nome del file, utilizzando i seguenti caratteri di formato:

  • %F - il nome del file .FCStd (o l'etichetta, se non è ancora stato salvato)
  • %Lx - l'etichetta dell'oggetto/degli oggetti selezionati, separati dal carattere 'x'
  • %Px - l'etichetta dell'oggetto/degli oggetti selezionati e del loro primo genitore, separati dal carattere 'x'
  • %U - la data e l'ora in UTC ISO 8601
  • %D - la data e l'ora nel fuso orario locale ISO 8601

Tutti gli altri caratteri vengono mantenuti alla lettera. Se il nome del file risultante non è valido, verrà modificato al momento del salvataggio, sostituendo i caratteri non validi con il carattere di sottolineatura (_).

Relativamente al mouse

  • General/ComboBoxWheelEventFilter (boolean): Impostare su true in modo che i widget non rilevino l'evento della rotellina del mouse e impediscano lo scorrimento delle aree scorrevoli. Richiede il riavvio di FreeCAD per attivarsi.
  • View/GestureMoveThreshold (integer): la distanza (px) di cui deve spostarsi il cursore del mouse per accedere alle modalità di rotazione o panoramica dello stile di navigazione Gesture. Il valore predefinito è 5.
  • View/GestureRollFwdCommand, View/GestureRollBackCommand (string): comandi da eseguire tramite gesti di rotazione del pulsante del mouse in stile navigazione gestuale.
  • View/GestureTapHoldTimeout (integer): Impostare il tempo di attesa (in millisecondi) per entrare in modalità panoramica nello stile di navigazione Gesture. Può essere utile aumentarlo se il trascinamento della geometria in Sketcher risulta difficoltoso. Il valore predefinito è 700.

  • General/TasksKeyEsc (boolean): Creare e impostare su false per disabilitare il tasto ESC per uscire dal pannello delle azioni in tutti gli ambienti (ovvero se il pannello delle azioni ha il focus).

Ambienti di lavoro specifici

Ambiente BIM

  • Mod/BIM/DefaultPageScale (float): Scala predefinita per le nuove pagine TechDraw create dall'Ambiente BIM, nel caso in cui il modello non contenga alcun campo di testo modificabile "Scale" o "Scaling" (senza distinzione tra maiuscole e minuscole).

Ambiente CAM

  • The CAM Workbench has two switches to enable experimental features documented on the CAM experimental page.

Ambiente Draft

  • Mod/Draft/DefaultAnnoDisplayMode (integer): Set to 1 to create Draft annotations (texts, dimensions and labels) with their VistaDisplay Mode set to Screen. Set to 0 for new annotations with this property set to World. Defaults to 0. introduced in 1.0
  • Mod/Draft/DimAutoFlipText (boolean): Set to false to disable the automatic orientation of the text of newly created dimensions relative to the current working plane. Defaults to true. introduced in 1.1
  • Mod/Draft/GridHideInOtherWorkbenches (boolean): Set to false to keep the Draft grid when switching to workbenches other than BIM or Draft. Defaults to true. introduced in 1.0
  • Mod/Draft/SvgLinesBlack (boolean): Set to true to convert the color white to black in SVG output. Set to false to keep the color white. Affects export to Flattened SVG and TechDraw DraftViews. Defaults to true.
  • Mod/Draft/useSupport (boolean): Set to true to set the DatiSupport property of Draft objects created on a face of an exiting base object to that base object. This was standard behavior before FreeCAD version 0.19. Note that this parameter may not be supported in future versions. Defaults to false.

Ambiente Part

  • Mod/Part/ParametricRefine (boolean): Set to false so Part RefineShape creates an independent copy rather than a linked one. Defaults to true.

Ambiente Part Design

  • Mod/PartDesign/AdditiveHelixPreview (boolean): Set to true to ensure an additive helix that does not intersect the body is visible in the preview. Defaults to false.
  • Mod/PartDesign/DefaultDatumColor (unsigned): Diffuse color and transparency for PartDesign datums, PartDesign ShapeBinders and PartDesign SubShapeBinders. Defaults to 4292280473. See here for information about the color value.
  • Mod/PartDesign/SubtractiveHelixPreview (boolean): Set to true to ensure a subtractive helix that does not intersect the body is visible in the preview. Defaults to true.
  • Mod/PartDesign/SwitchToTask (boolean): Set to false to prevent the PartDesign Workbench from switching to the Task panel when starting. Defaults to true. introduced in 1.1: Available in the Preferences Editor.
  • Mod/PartDesign/SwitchToWB (boolean): Set to false to prevent the PartDesign Workbench to be automatically called when a PartDesign Body is activated. Defaults to true.

Ambiente Sketcher

  • Mod/Sketcher/RadiusDiameterConstraintDisplayAngleRandomness (float): Set an angle randomness on the above value. Value is the range of the random angle, centered on base angle. Defaults to 0.
  • Mod/Sketcher/RadiusDiameterConstraintDisplayBaseAngle (float): Set the angle (from horizontal) used to display radius/diameter constraints in Sketcher at creation time. Defaults to 15.
  • Mod/Sketcher/RoundRectangleSuggConstraints (boolean): Set to false to disable the addition of two extra construction points when creating a rounded rectangle. introduced in 0.21

Colori delle etichette dei vincoli

The label in Sketcher that displays the current status of the constraints (e.g. "Underconstrained," "Fully Constrained," etc.) is styleable on a per-state basis either using the Qt stylesheet, or via user preferences. User preferences take precedence if they have been set (in Mod/Sketcher/General):

  • EmptySketchMessageColor - Defaults to 50% opacity black
  • UnderconstrainedMessageColor - Defaults to black
  • MalformedConstraintMessageColor - Defaults to red
  • ConflictingConstraintMessageColor - Defaults to red
  • RedundantConstraintMessageColor - Defaults to orange red
  • PartiallyRedundantConstraintMessageColor - Defaults to royal blue
  • SolverFailedMessageColor - Defaults to red
  • FullyConstrainedMessageColor - Defaults to green

Ambiente Start

L'Ambiente Start non è più incluso dopo la versione 0.21.

  • Mod/Start/DefaultImportXXX (string): Where XXX is a lowercase file extension. For example DefaultImportifc for .IFC files. Allows to set a default import module to be used when clicking an icon on the start page, when several importers are available. For example, setting DefaultImportifc = ifc_import will use the NativeIFC importer if available. introduced in 0.21
  • Mod/Start/TimeFormat (string): A time format string such as %m/%d/%Y %H:%M:%S used for the date in the tooltip that is shown when an item on the start page is hovered.

Modulo Help

  • Mod/Help/UseWebModule (boolean): Allows to force the use of the Web module to open MDI tabs. This can be useful to work around QWebEngine issues in some versions of Qt5. Defaults to false. introduced in 1.0
