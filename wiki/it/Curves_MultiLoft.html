Descrizione

The Curves MultiLoft tool lofts profile objects made of multiple faces in parallel.

Utilizzo

  1. There are several ways to invoke the tool:
    • Press the MultiLoft button.
    • Select the Surfaces → MultiLoft option from the menu.
  2. A MultiLoft object is created.

Properties

See also: Property View.

A MultiLoft object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

Multiloft

  • DatiClosed (Bool): Closes the loft.
  • DatiMaxDegree (Integer): Max B-spline degree.
  • DatiRuled (Bool): Ruled loft.
  • DatiSources (LinkList): Objects to loft.
