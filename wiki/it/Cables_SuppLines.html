Description

The SuppLines is a dependent object and cannot be created directly.

It can be created as a child object during creation of: CableBox, CableConnector, ElectricalDevice, CableLightPoit.

It serves as a support for WireFlex objects which form a Base for Cable. The SuppLines object itself is attached to other objects mentioned above and keeps predefined placement relative to them. The SuppLines has a shape of a cross with five vertices which can be used as attachment supports for up to five cables entering single hole in a cable box. It can also be a reference point for cable entering the connector or electrical device. It is up to the user to find the SuppLines helpful or not.

Important: the real relation between SuppLines object and its parent is reversed to avoid cyclic dependencies and to speed up recalculation. This has some consequences (see Notes).

Example of a default SuppLines shape.

Usage

L'oggetto SuppLines non può essere creato direttamente. Viene creato durante la creazione di: CableBox, CableConnector, ElectricalDevice, CableLightPoit a seconda della proprietà Number Of Supp Lines di questi oggetti.

Notes

  • The SuppLines object has reversed relation with its parent to avoid cyclic dependencies. It is shown as a child in the Tree View only for visual organization of objects. Undesirable consequence: when a parent is copied, all its child SuppLines objects must always be manually copied with it.
  • Depending of parent the single SuppLines object can have one or more crosses.

Properties

The SuppLines object has not any visible properties to edit. Its shape can be created and modified by its parent object or take the form of a cross with a default size.
