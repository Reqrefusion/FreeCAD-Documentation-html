Material

Introduzione

Le preferenze per l'Ambiente Material si trovano nell'Editor delle preferenze. Nel menu selezionare Modifica → Preferenze... e poi Materiale.

Ci sono due pagine: Generale e Materiale predefinito.

Generale

Risorse della scheda

  • Usa materiali incorporati: Se selezionata, verranno elencati i materiali predefiniti di FreeCAD.
  • Usa materiali da ambienti esterni: Se selezionata, verranno elencati i materiali aggiunti da workbench esterni (aggiuntivi).
  • Usa materiali dalla cartella dei materiali nella cartella delle preferenze di dell'utente di FreeCAD: Se selezionata, verranno elencati i materiali dalla directory delle preferenze di FreeCAD.
  • Usa materiali dalla cartella definita dall'utente: Se selezionata, verranno elencati i materiali dalla directory specificata.
  • Cartella dell'utente: origine per materiali aggiuntivi.

Ordinamento e duplicati schede

  • Elimina duplicati della scheda: se selezionata, i materiali duplicati vengono eliminati dall'elenco.
  • Ordina per risorse: se selezionata, i materiali vengono visualizzati ordinati in base alle risorse (posizioni). Se non selezionata, vengono ordinati in base al nome.

Selettore Materiale

  • Mostra preferiti: se selezionata, i materiali aggiunti ai Preferiti vengono visualizzati nel Selettore Materiali.
  • Mostra recenti: se selezionata, i materiali utilizzati di recente vengono visualizzati nel Selettore Materiali.
  • Mostra librerie vuote: se selezionata, le librerie vuote vengono visualizzate nel Selettore Materiali.
  • Mostra cartelle vuote: se selezionata, le cartelle vuote vengono visualizzate nel Selettore Materiali.
  • Mostra file precedenti: se selezionata, i file precedenti vengono visualizzati nel Selettore Materiali.

Editor dei materiali

  • Show favorites: If checked, materials added to Favorites are shown in the Material Editor.
  • Show recent: If checked, recently used materials are shown in the Material Editor.
  • Show empty libraries: If checked, empty libraries are shown in the Material Editor.
  • Show empty folders: If checked, empty folders are shown in the Material Editor.
  • Show legacy files: If checked, legacy files are shown in the Material Editor.

Materiale predefinito

Materiale predefinito

  • Selettore Materiale: specifica il materiale utilizzato di default in FreeCAD.
  • Avvia editor: apre l'Editor Materiali.
