Introduzione

Dopo aver installato FreeCAD sul tuo sistema operativo (Windows, Linux o Mac), si può prendere in considerazione l'installazione di uno o più dei seguenti componenti aggiuntivi.

File di aiuto

Vedere Installazione del file della Guida.

Ambienti di lavoro esterni

Oltre ai pacchetti degli ambienti di lavoro forniti di default con FreeCAD, vi è un'ampia collezione di utili ambienti complementari realizzati dai membri della comunità.

Software di terze parti

FreeCAD supporta molti pacchetti software di terze parti pronti all'uso. In molti casi l'unica cosa da fare è installare il software e quando FreeCAD viene riavviato lo troverà automaticamente e sarà in grado di usarlo. Questa sezione intende fornire un elenco di tali pacchetti software, insieme ad alcune informazioni su dove vengono utilizzati in FreeCAD e da dove possono essere scaricati.

Supporti

GitPython

GitPython è una libreria python che viene utilizzata per interagire con i repository Git. L'Addon Manager può usare questa libreria. GitPython è incluso nei programmi di installazione di FreeCAD per Windows e Mac.

GraphViz

GraphViz è un software di visualizzazione di grafici open source. È utilizzato dallo strumento Grafico delle dipendenze.

OpenCAMLib

OpenCAMLib è una libreria open source di algoritmi di produzione assistita da computer (CAM). È utilizzata nell'ambiente CAM. Vedere la pagina OpenCamLib per le istruzioni d'installazione.

OpenSCAD

OpenSCAD è un modellatore solido 3D. L'ambiente OpenSCAD dipende da questo software e Mesh Workbench lo utilizza per i suoi strumenti booleani. È richiesto anche per l'importazione di file SCAD con lo strumento Std Import.

Formati file

Tutto il software in questa sezione viene utilizzato dagli strumenti Importa o Esporta.

CADExchanger

CADExchanger è un'applicazione commerciale per lo scambio di vari formati di file CAD. Esiste un ambiente di lavoro esterno per utilizzare questa applicazione in FreeCAD.

DXF Importer

FreeCAD ha un importatore ed esportatore nativo per file DXF, programmato in C++. Attualmente non implementa tutte le funzionalità del formato DXF. Per queste funzionalità sono ancora disponibili l'importatore ed esportatore Python legacy. Questi richiedono la libreria Python Draft-dxf-importer. Vedere la pagina Importare i file DXF in FreeCAD per maggiori informazioni.

Convertitori DWG

FreeCAD non può leggere e scrivere direttamente file DWG. Per convertire file DXF in file DWG e viceversa, FreeCAD si affida a convertitori esterni. È disponibile il supporto integrato per i seguenti convertitori DWG:

  • LibreDWG (open-source, manca il supporto per alcune entità DWG).
  • ODA File Converter (free, ma non open-source).
  • QCAD pro (commerciale). introduced in 0.20

Vedere Preferenze di Importa/Esporta e Importare i file DWG in FreeCAD per maggiori informazioni.

IfcOpenShell

IfcOpenShell è una libreria per lavorare con il formato file Industry Foundation Classes (IFC) utilizzato nella progettazione architettonica. La libreria è utilizzata anche dagli strumenti Arch IfcExplorer (0.18 and below) e BIM IfcExplorer. IfcOpenShell è incluso nei programmi di installazione di FreeCAD per Windows e Mac.

IfcJson

IfcJson è una libreria necessaria per l'esportazione nel formato file IFCJSON. IFCJSON è un nuovo formato IFC che non è ancora supportato da molte applicazioni.

Pycollada

Pycollada, noto anche come python-collada, è una libreria Python per leggere e scrivere file Collada (DAE). Pycollada è incluso nei programmi di installazione di FreeCAD per Windows e Mac.

Rendering

LuxCoreRender

LuxCoreRender è un motore di rendering, reboot del progetto LuxRender. Ufficialmente non è supportato dall'ambiente Raytracing, ma potrebbe valere la pena di provarlo. È ufficialmente supportato dal nuovo ambiente Render, inteso come futuro sostituto dell'ambiente Raytracing. Vedere la pagina LuxCoreRender per ulteriori informazioni e istruzioni di installazione.

LuxRender

LuxRender è uno dei due motori di rendering supportati dall'ambiente Raytracing. Nel 2013 il progetto è stato riavviato diventando LuxCoreRender/it, con un'importante riscrittura del codice e modifiche alla compatibilità. Ufficialmente l'ambiente Raytracing supporta solo LuxRender che è stato dismesso (l'ultima versione è la 1.6 del 28/12/2017), mentre il nuovo ambiente Render (destinato in futuro a sostituire l'ambiente Raytracing) supporta LuxCoreRender ed ha abbandonato il supporto per LuxRender. Comunque, anche se ufficialmente non supportato, LuxCoreRender potrebbe funzionare con l'ambiente Raytracing, potrebbe valere la pena di provarlo. Vedere la pagina LuxRender per ulteriori informazioni e istruzioni di installazione e LuxCoreRender se si desidera provare un software più moderno.

POV-Ray

POV-Ray è un noto ray-tracer che può rendere immagini fotorealistiche. È uno dei due motori di rendering attualmente supportati dall'ambiente Raytracing. Consulta la pagina POV-Ray per ulteriori informazioni e istruzioni di installazione.

Elementi finiti

CalculiX

CalculiX è una suite di due pacchetti di elementi finiti: CalculiX CrunchiX, un risolutore FEM, e CalculiX GraphiX, un frontend GUI. Solo il risolutore è supportato da FreeCAD. Viene utilizzato dallo strumento FEM SolverCalculixCcxtools.

Gmsh

Gmsh è un generatore automatico di mesh agli elementi finiti. è utilizzato dagli strumenti FEM MeshGmshFromShape e Mesh FromPartShape.

Elmer

Elmer è un software di simulazione multifisica, che è stato reso open source nel 2005. In FreeCAD i suoi moduli Grid e Solver sono utilizzati dallo strumento FEM SolverElmer.

FEniCS

FEniCS è una piattaforma di calcolo per risolvere equazioni differenziali alle derivate parziali (PDE), ampiamente utilizzate per la risoluzione di problemi FEM. Viene utilizzato dall'ambiente FEM.

Z88

Z88 è un altro programma FEM, contenente un mesher, un risolutore e convertitori. Viene utilizzato dallo strumento FEM SolverZ88. FreeCAD richiede il pacchetto open source Z88OS.

OpenFOAM

OpenFOAM è una vasta raccolta di librerie per simulazioni di fluidodinamica computazionale (CFD). OpenFOAM è utilizzato dall'ambiente Cfd e CfdOF external_workbench.
