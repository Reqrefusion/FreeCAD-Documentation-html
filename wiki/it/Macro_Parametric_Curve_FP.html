Macro Parametric_Curve_FP

Description
Update of Macro 3D Parametric Curve, but with new features. Creates a Feature Python object, offers spreadsheet and JSON integration, expanded parameters.
It has support for the same a, b, c parameters, but also can have as many d parameters in the form of d1, d2, d3, d4, etc. as you like.
It also supports saving formulas to a text file in JSON format and support for Spreadsheet integration of the current formula.
Full documentation can be found Full Documentation on Github Parametric_Curve_FP on github.

Descrizione

Aggiornamento: quando si aggiorna dalla versione 0.2024.05.08b alla 0.2025.01.28b, è necessario prima disinstallare la vecchia versione e poi installare la nuova. Non eseguire nuovamente il file .FCMacro dopo l'aggiornamento. È necessario eliminare tale file ed eseguire il file .py dalla versione 0.2025.01.28 in poi.

Questa macro è un aggiornamento della Macro 3D Parametric Curve di Gomez Lucio e successivamente modificata da Laurent Despeyroux il 9 febbraio 2015. La macro è stata aggiornata a un oggetto Python Feature parametrico. Supporta gli stessi parametri a, b, c, ma può anche avere tutti i parametri d desiderati, sotto forma di d1, d2, d3, d4, ecc. Se si desidera fare riferimento a un VarSet o a un oggetto DynamicData dd in una formula, è possibile utilizzare il comando fc(expr). Ad esempio, se è presente un valore float in un oggetto dd denominato my_float e si desidera farvi riferimento nella formula per il parametro b, immettere per b: fc(dd.my_float) oppure, se si desidera utilizzarlo in modo più complesso, b: fc(dd.my_float) * a + pi come altro esempio.

Supporta inoltre il salvataggio delle formule in un file di testo in formato JSON e l'integrazione della formula corrente in un foglio di calcolo. La documentazione completa è disponibile su github Parametric_Curve_FP.
