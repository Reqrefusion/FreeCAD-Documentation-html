Descrizione

Lo strumento Editor dei materiali fornisce accesso all'editor dei materiali, parte del sistema materiali di FreeCAD. Il sistema dei materiali può essere utilizzato da tutti gli ambienti di lavoro. Gestisce le proprietà dei materiali e anche le proprietà estetiche degli oggetti.

Utilizzo

  1. Esistono diversi modi per avviare l'editor:
    • Ambiente Material:
      • Premere il pulsante Modifica....
      • Selezionare l'opzione Materiali → Modifica... dal menu.
    • Ambiente FEM:
      • Premere il pulsante Editor dei materiali.
      • Selezionare l'opzione Modello → Materiali → Editor dei materiali dal menu.
    • Tutti gli ambienti:
      1. Selezionare un oggetto.
      2. Eseguire una delle seguenti operazioni:
        • Selezionare l'opzione Visualizza → Materiale... dal menu.
        • Selezionare l'opzione Materiale... dal menu contestuale Vista ad albero o Vista 3D.
      3. Si apre il pannello attività Materiale.
      4. Premere il pulsante Avvia editor.
  2. Selezionare un materiale dall'elenco multilivello nel pannello di sinistra. Sono disponibili le seguenti categorie:
    • Preferiti
    • Recenti
    • System
      • Appearance
      • Fluid
      • Machining
      • Patterns
      • Standard
      • Test
    • User
  3. Utilizzare le tre schede nel pannello destro dell'editor per modificare le proprietà del materiale selezionato:
    • Generale:
      • Nome
      • Autore
      • Licenza
      • Genitore
      • URL di origine
      • Riferimento alla fonte
      • Etichette
      • Descrizione
    • Proprietà fisiche - varie proprietà fisiche
    • Aspetto:
      • Basic Rendering:
        • Colore ambiente
        • Colore diffuso
        • Colore Emissivo
        • Lucentezza
        • Colore speculare
        • Trasparenza
  4. Eventualmente utilizzare i pulsanti nella scheda Generale:
    • Premere il pulsante Nuovo per creare un nuovo materiale con proprietà vuote.
    • Premere il pulsante Eredita nuovo per creare un nuovo materiale con le proprietà copiate dal materiale selezionato.
    • Premere il pulsante * per aggiungere il materiale selezionato ai preferiti.
  5. Se le proprietà del materiale sono state modificate: premere il pulsante Salva per salvarle.
  6. Eseguire una delle seguenti operazioni:
    • Premere il pulsante OK per confermare la selezione del materiale e chiudere l'editor.
    • Premere il pulsante Annulla per annullare la selezione del materiale e chiudere l'editor. Questo non annullerà le modifiche alle proprietà del materiale già salvate.
  7. Se il pannello attività Materiale è aperto: premere il pulsante Chiudi per chiuderlo.
