Descrizione

The Curves ZebraTool creates a Zebra texture for surface inspection. This tool is part of the external workbench called Curves.

Utilizzo

  1. Switch to the Curves workbench (install from Addon Manager is necessary, if not previously installed)
  2. Select the surface to be tested
  3. To invoke the command, do one of the following:
    • Press the Zebra texture for surface inspection button in the tool bar.
    • Use the Surfaces → Zebra Tool
  4. Change the Zebra Tool parameter to the required conditions

Proprietà

  • DatiBlack Stripes Width: Change the black stripes width.
  • DatiScale: Scale the Zebra stripes.
  • DatiRotation: Rotate the Zebra stripes.

Note

  • Click on the Quit button to exit the Zebra Tool.
