Descrizione

Il comando Part Esporta esporta gli oggetti selezionati in un formato file diverso. I formati di file supportati sono: STEP (con o senza colori), IGES (con o senza colori ) e BREP.

Utilizzo

  1. Impostare le Preferenze di importazione esportazione nell'Editor delle preferenze.
  2. Selezionare uno o più oggetti. Vedere Esporta.
  3. Selezionare l'opzione Part → Esporta file CAD... dal menu.
  4. Selezionare il formato file corretto nella finestra di dialogo.
  5. Inserire un nome file.
  6. Premere il pulsante Salva.

Note

  • Il comando Esporta supporta molti più formati di file.
