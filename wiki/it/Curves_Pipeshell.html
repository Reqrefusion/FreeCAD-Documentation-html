Descrizione

The Curves Pipeshell tool creates a Pipeshell sweep object.

Utilizzo

  1. Select the edge that builds the sweep path in the 3D View.
  2. Select one or more required profiles in the Tree View.
  3. There are several ways to invoke the tool:
    • Press the Pipeshell button.
    • Select the Surfaces → Pipeshell option from the menu.
  4. A PipeShell object is created.
  5. Optionally edit the values in the Property View to choose a different sweeping algorithm, or a different output shape and adjust the related properties (see Properties below).

Proprietà

See also: Property View.

A PipeShell object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Data

Main

  • DatiMode (Enumeration): PipeShell mode used to select the sweeping algorithm:
    • Frene (default): ...
    • DiscreteTrihedron: ...
    • FixedTrihetron: ...
    • Binormal: ...
    • ShapeSupport: ...
    • AuxiliarySpine: ...
  • DatiOutput (Enumeration): Determines the shape of the object:
    • Sections (default): ...
    • Lofted Sections: ...
    • Surface: ...
  • DatiProfiles (LinkList):: Profiles that are swept along spine.
  • DatiSpine (LinkSubList): Sweep path.

Mode

  • Dati (Hidden)Auxiliary (Link): Auxiliary spine.
  • Dati (Hidden)Contact (Enumeration): Type of contact to auxiliary spine.
    • NoContact (default): ...
    • Contact: ...
    • ContactOnBorder: ...
  • Dati (Hidden)Corrected (Bool): Corrected Frenet.
  • Dati (Hidden)Direction (Vector): Direction of the Binormal and FixedTrihedron modes.
  • Dati (Hidden)EquiCurvi (Bool): Curvilinear equivalence.
  • Dati (Hidden)Location (Vector): Location of the FixedTrihedron mode.
  • Dati (Hidden)Support (Link): Shape of the ShapeSupport mode.

Settings

  • DatiMaxDegree (Integer): Maximum degree of the generated surface.
  • DatiMaxSegments (Integer): Maximum number of segments of the generated surface.
  • DatiSamples (Integer): Number of samples for preview.
  • DatiSolid (Bool): Make solid object.
  • DatiTol3d (Float): Tolerance 3D.
  • DatiTolAng (Float): Tolerance angular.
  • DatiTolBound (Float): Tolerance boundary.

Notes

  • Pipeshell needs a wire object (as the sweep path), and at least one Pipeshell Profile.
  • The two tools Pipeshell and Pipeshell Profile work together as an "Advanced" Sweep tool.
