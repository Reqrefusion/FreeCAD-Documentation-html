Descrizione

Lo strumento Spessore di PartDesign trasforma un corpo solido in un oggetto cavo con almeno una faccia aperta, conferendo a ciascuna delle facce rimanenti uno spessore uniforme. Aggiunge un oggetto Spessore al documento con la relativa rappresentazione nella vista ad albero.

Base solid (A) → Solid with selected face to be opened (B) → Resulting hollow object (C)

Utilizzo

Add a thickness

  1. Facoltativamente attivare il corpo a cui applicare lo spessore.
  2. Selezionare una o più facce del Corpo.
  3. Esistono diversi modi per richiamare lo strumento:
    • Premere il pulsante Spessore.
    • Selezionare l'opzione Part Design → Applica una funzione di dress-up → Spessore dal menu.
  4. Se non è presente alcun Corpo attivo e nel documento sono presenti due o più Corpi, si aprirà la finestra di dialogo Corpo attivo richiesto che richiederà di attivarne uno. Se è presente un solo Corpo, questo verrà attivato automaticamente.
  5. Si apre il pannello delle attività Parametri di spessore. Vedere Opzioni per maggiori informazioni.
  6. Premere il pulsante OK per terminare.


Remember:
  • Since there must be at least one face for the feature, the last remaining face in the list cannot be removed.

Edit a thickness

  1. Do one of the following:
    • Double-click the Thickness object in the Tree view
    • Right-click the Thickness object in the Tree view and select Edit Thickness from the context menu.
  2. The Thickness parameters task panel opens. See Options for more information.
  3. Press the OK button to finish.

Opzioni

  • Thickness: Spessore della parete dell'oggetto risultante. Impostare il valore desiderato.
  • Mode
    • Skin: Selezionare questa opzione per ottenere un oggetto simile ad un vaso, senza testa ma con il fondo
    • Pipe: Selezionare questa opzione per ottenere un oggetto simile ad un tubo, senza testa e senza fondo. In questo caso può essere conveniente selezionare le facce da eliminare prima di avviare lo strumento. Aiutarsi con i pulsanti delle viste predefinite o utilizzare i tasti numerici.
    • Recto Verso:
  • Join Type
    • Arc: rimuove i bordi esterni e crea un raccordo con un raggio uguale allo spessore definito.
    • Intersection: quando le facce sono spostate verso l'esterno, gli spigoli tra le facce vengono mantenuti.
  • Make thickness inwards: quando è selezionato, le facce sono spostate verso l'interno.

Note

  • Se lo spessore va verso l'interno, il valore deve essere inferiore all'altezza più piccola del corpo.
  • Lo strumento potrebbe non funzionare con forme complesse. Sweep Additivo o Loft additivo potrebbero funzionare meglio per creare forme complesse.
  • Errori noti:
    • BRep_API: comando non eseguito.
    • BRep_Tool: nessun parametro sul bordo.
    • Fallisce silenziosamente.

Proprietà

See also: Property editor.

A PartDesign Thickness object is derived from a Part Feature object and inherits all its properties. It also has the following additional properties:

Dati

Base

  • DatiBase (LinkSub): Sub-link to the parent feature's list of selected edges and faces.
  • DatiSupport Transform (Bool): Include the base additive/subtractive shape when used in pattern features. If disabled, only the dressed part of the shape is used for patterning. Default: false.
  • Dati (hidden)Add Sub Shape (PartShape)
  • Dati (hidden)Base Feature (Link): Link to the parent feature.
  • Dati (hidden)_ Body (LinkHidden): Link to the parent body.

Part Design

  • DatiRefine (Bool): Refine shape (clean up redundant edges) after adding/subtracting. The default value is determined by the Automatically refine model after sketch-based operation preference. See PartDesign Preferences.

Thickness

  • DatiValue (Length): Thickness value. Default: 1 mm.
  • DatiMode (Enumeration): Mode. Skin (default), Pipe or Recto verso. Only Skin is implemented.
  • DatiJoin (Enumeration): Join type. Arc (default) or Intersection.
  • DatiReversed (Bool): Apply the thickness towards the solids interior. Default: false.
  • DatiIntersection (Bool): Enable intersection-handling. Default: false.
