Description

Descrizione

Esporta un file di programmazione del robot.
Si può esportare una subroutine KUKA compatta o una subroutine KUKA completa.

Utilizzo

  1. Selezionare il robot e la traiettoria nella vista ad albero.
  2. Scegliere Robot → Esporta traiettoria → Subroutine Kuka compatta... dal menu in alto.

oppure

  1. Selezionare il robot e la traiettoria nella vista ad albero.
  2. Scegliere Robot → Esporta traiettoria → Subroutine Kuka completa... dal menu in alto.
  1. Select a robot and a trajectory in the Tree View.
  2. Select the Robot → Export Trajectory → Kuka Compact Subroutine option from the menu.
  3. A KRL file (*.src) is created.
