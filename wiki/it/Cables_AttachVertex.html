Description

The Attach Vertex attaches an existing vertex of WireFlex to external vertex or object. Attachments of vertexes are kept in special properties of WireFlex, for details see WireFlex Properties.

It can be used interchangeably with the "Attach point" command from Cables Edit, see the WireFlex Description.

Once attached the wire vertex keeps the same position in 3D space as the external vertex to which it is attached to. When the external vertex is moved, the attached wire vertex moves together with it. When the wire is edited in the Cables Edit the attached vertex can not be moved. It jumps back to its attached position.

Usage

  1. Select one vertex of an existing WireFlex object in the 3D View.
  2. Select one external vertex in the 3D View holding the CTRL key.
  3. Create a new WireFlex vertex attachment by one of these methods:
    • Press the Attach Vertex button.
    • Select the Cable Wires → Attach Vertex option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cable Wires → Attach Vertex option from the context menu.

Nota: la selezione di elementi esterni diversi dai vertici viene trattata come una selezione dell'intero oggetto. L'associazione all'oggetto è intesa come un'associazione al suo posizionamento base.

Notes

  • From version 0.2.0 this command can also be executed using node context menu in Cables Edit command
  • See the WireFlex Description and WireFlex Usage for the details how to use WireFlex.
