Descrizione

I Attach Vertex collegamenti di un vertice esistente di WireFlex al vertice o all'oggetto esterno. Gli allegati dei vertici sono mantenuti nelle proprietà speciali di WireFlex, per i dettagli vedi WireFlex Properties.

Può essere utilizzato in modo intercambiabile con il comando "Punto di collegamento" da Cables Edit,vedi il WireFlex Description.

Una volta collegato, il vertice del filo mantiene la stessa posizione nello spazio 3D del vertice esterno a cui è collegato. Quando il vertice esterno viene spostato, il vertice del filo collegato si sposta insieme ad esso. Quando il filo viene modificato nel Cables Edit il vertice attaccato non può essere spostato. Torna alla sua posizione attaccata.

Utilizzo

  1. Select one vertex of an existing WireFlex object in the 3D View.
  2. Select one external vertex in the 3D View holding the CTRL key.
  3. Create a new WireFlex vertex attachment by one of these methods:
    • Press the Attach Vertex button.
    • Select the Cable Wires → Attach Vertex option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cable Wires → Attach Vertex option from the context menu.

Nota: la selezione di elementi esterni diversi dai vertici viene trattata come una selezione dell'intero oggetto. L'associazione all'oggetto è intesa come un'associazione al suo posizionamento base.

Note

  • Dalla versione 0.2.0 questo comando può essere eseguito anche utilizzando il menu contestuale del nodo ne comando Cables Edit
  • vedi il WireFlex Description e WireFlex Usage per i dettagli su come utilizzare WireFlex.
