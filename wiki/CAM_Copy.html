Description

The tool Copy creates a copy of a selected path.

Usage

  1. Select a CAM object
  2. There are several ways to invoke the command:
    • Press the Copy the operation in the job button in the CAM toolbar
    • Use the P then Y keyboard shortcut
    • Use the CAM → Path Modification → Copy the operation in the job entry in the CAM menu

Properties

  • DataBase: The base path to be copied

Options

The copy stays linked with the original path. If the original changes, so does the copy.
