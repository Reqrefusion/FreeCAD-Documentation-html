Description

The Support Point creates a Draft Point in bright color. It serves as support for creating WireFlex. Its has additional Attachment properties which can be used e.g. for attaching cables to walls or ceilings etc. The Support Point has the same purpose as Support Line and can serve as its complement.

Usage

The same as Support Line.

Properties

A Support Point object has all properties of the Draft Point plus Attachment properties.
