Description

The Std DlgPreferences command opens the Preferences Editor.

Usage

  1. There are several ways to invoke the command:
    • Select the Edit → Preferences... option from the menu.
    • Use the keyboard shortcut: Ctrl+,. introduced in 1.1

Notes

  • For more information see Preferences Editor.
  • On macOS this command may appear in the FreeCAD menu instead of the Edit menu.
