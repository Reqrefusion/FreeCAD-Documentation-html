Description

The Sketcher CreateChamfer tool creates a chamfer between two non-parallel edges. This is the same tool as Sketcher CreateFillet but with a different initial mode.

Usage

See also: Drawing aids.

  1. There are several ways to invoke the tool:
    • Press the Create chamfer button.
    • Select the Sketcher → Sketcher tools → Create chamfer option from the menu.
    • Right-click in the 3D view and select the Create chamfer option from the context menu.
    • Use the keyboard shortcut: G then F, then C.
  2. For further steps see Sketcher CreateFillet.
