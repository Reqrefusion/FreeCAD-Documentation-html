Description

The Std DockOverlayToggle command moves a single docked panel into or out of the overlay system. This provides finer-grained control over the workspace, as apposed to the global Std DockOverlayAll command, which affects all panels at once.

To use this command, the mouse cursor must be positioned over the desired panel. When invoked, the panel will switch its state:

  • If it is a normal, docked panel, it will be moved into the overlay system, becoming a transparent floating widget.
  • If it is already an overlay panel, it will be moved out of the system and returned to its last known normal docked position.

Usage

  1. The mouse cursor must be positioned over the docked panel you wish to affect. The panel can be in either its normal docked state or in overlay mode.
  2. There are several ways to invoke the command:
    • Select the View → Dock panel overlay → Toggle overlay option from the menu.
    • Use the keyboard shortcut: F3.
    • Use the overlay toggle button () from a docked window's title bar.

Preferences

The appearance of the panels in this "transparent active" state is controlled by the same stylesheet as the standard overlay mode.
