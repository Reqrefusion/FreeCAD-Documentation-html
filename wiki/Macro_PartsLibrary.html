Parts Library

Description
Starts the Parts Library browser FreeCAD.
The FreeCAD Parts Library is a community effort to provide a free library of reusable Parts to be inserted into your projects.
You are welcome to participate and submit parts to the library, but keep in mind you must have rights to share them. See the readme file for more details.
