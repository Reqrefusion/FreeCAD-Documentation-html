Description

The Material creates a series of Arch Materials and Arch Multimaterials ready to use with Cable or Cable Box. The following default multimaterials are suitable for Cable: cableMultiMat3w, cableMultiMat5w, cableMultiMat8w. They have proper naming of their composition which corresponds to default names in Sub Colors property of a Cable. The (introduced in version 0.3.0) boxMultiMat2m multimaterial can be used by CableBox equipped with DIN rails.

When creating multimaterials for cables the Name field of the single layer in the Composition list in multimaterial should correspond to proper entry in the Sub Colors property of a cable.

Two-color cable insulation (feature introduced in version 0.3.0): If two layers in the Composition list in multimaterial have the same Name field but different Material fields they will be used as two colors for the single subcable insulation with the same name placed in Sub Colors property of a cable.

Usage

  1. Create default set of multimaterials by one of these methods:
    • Press the Cable Materials button.
    • Select the Cables → Cable Materials option from the menu.
    • Right-click in the Tree view or the 3D view and select the Cables → Cable Materials option from the context menu.

Notes

The default multimaterials can serve as templates for creation other custom multimaterial sets.
