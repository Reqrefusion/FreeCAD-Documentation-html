Toolbars

Cameras and Objects tools

  • File:Movie CreateMovieCamera.svg MovieCamera
  • File:Movie EnableMovieCamera.svg Enable a MovieCamera
  • File:Movie SetMoviePosA.svg Set position A
  • File:Movie SetMoviePosB.svg Set position B
  • File:Movie CreateMovieObjects.svg MovieObjects
  • File:Movie EnableMovieObjects.svg Enable a MovieObjects
  • File:Movie SetMovieObjectsAxis.svg Set an axis
  • File:Movie ExcludeMovieObjects.svg Exclude a MovieObjects

Animation tools

  • File:Movie IniMovieAnimation.svg Return to beginning
  • File:Movie PrevMovieAnimation.svg Take a step back
  • File:Movie PlayBackwardMovieAnimation.svg Play backward the animation
  • File:Movie PauseMovieAnimation.svg Pause the animation
  • File:Movie PlayMovieAnimation.svg Play the animation
  • File:Movie PostMovieAnimation.svg Move one step forward
  • File:Movie EndMovieAnimation.svg Move to the end

Record and play tools

  • File:Movie CreateClapperboard.svg Clapperboard
  • File:Movie EnableMovieClapperboard.svg Enables a Clapperboard
  • File:Movie StartRecord3DView.svg "R1" - Record 3DViews
  • File:Movie StartRecordRender.svg "R2" - Record Renders
  • File:Movie StopRecordCamera.svg Stop Recording
  • File:Movie CreateVideo.svg Create video
  • File:Movie PlayVideo.svg Play video

Installation

The Movie Workbench is available through the Addon Manager. Any Movie 3rd party dependencies should be managed automatically through the Addon Manager.
