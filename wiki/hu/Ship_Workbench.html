Introduction

The Ship Workbench is used to create structures that are common to ships.

The Tools

Ship Design

  • Load an example ship geometry
  • Create a new ship
  • Areas curve
  • Hydrostatics

Weights

  • Create a new ship weight
  • Create a new tank
  • Tank capacity curve
  • Create a new loading condition
  • GZ curve computation

Resistance

  • Resistance Amadeo prediction
  • Resistance Blount and Fox prediction
  • Resistance Holtrop prediction
  • Resistance Savitsky prediction

Tutorials

  • FreeCAD-Ship s60 tutorial
  • FreeCAD-Ship s60 tutorial (II)

History

The Ship Workbench was unmaintained since v0.17 but has been resurrected in v0.19. As of v0.19 it is also designated an external workbench, so it is no longer included in the base system. Instead it's available via the Addon Manager.
