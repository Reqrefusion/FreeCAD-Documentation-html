Description

The Curves Discretize command allows to discretize an edge or a wire. This tool is part of the external workbench called Curves.

Usage

  1. Switch to the Curves workbench (install from Addon Manager is necessary, if not previously installed)
  2. To invoke the command, do one of the following:
    • Press the button in the Curves toolbar.
    • Use the Curves → Discretize entry in the dropdown menu.
