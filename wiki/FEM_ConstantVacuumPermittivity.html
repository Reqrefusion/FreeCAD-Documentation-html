Description

Overwrites vacuum permitivity in Elmer electrostatic and magnetodynamic analyses with a custom value.

Usage

  1. To create a new ConstantVacuumPermittivity object do the following:
    • Select the menu Model → Overwrite Constants → Constant vacuum permittivity.
  2. To edit an existing ConstantVacuumPermittivity object:
    • Select it in the Tree view.
  3. Specify a non-default vacuum permittivity value (in F/m).
