Introdução

Esta é a documentação wiki do FreeCAD. Há várias maneiras de usar esta documentação: explorando os tópicos, seguindo o manual ou utilizando as entradas do menu Ajuda do FreeCAD. Este é um trabalho em desenvolvimento, escrito pela comunidade de utilizadores e desenvolvedores do FreeCAD. Se encontrar informações incorretas ou ausentes, por favor, ajude o FreeCAD!

Os Tópicos

Esta página contém documentação útil para utilizadores do FreeCAD em geral: Uma lista de todas as bancadas de trabalho, instruções detalhadas para instalar e usar o FreeCAD, tutoriais, e tudo que o que precisa para começar.

Para utilizadores avançados: Esta página reúne informação para utilizadores avançados e pessoas interessadas em escrever scripts Python. Irá encontrar também um repositório de macros, instruções sobre como as instalar e utilizar, e mais informação sobre como personalizar o FreeCAD de acordo com as suas necessidades específicas.

Para desenvolvedores: Esta secção contém material para desenvolvedores: como compilar o FreeCAD você mesmo, como o código fonte do FreeCAD está estruturado, como navegar pelo código fonte, como desenvolver novas bancadas de trabalho, e como incorporar o FreeCAD na sua própria aplicação.

Manual

Manual do FreeCAD é outra forma, mais linear, de apresentar a informação contida nesta wiki. Foi feita para ser lida como um livro, e poderá levar-nos de forma suave a muitas páginas dos tópicos acima resumidos. versões ebook estão também disponíveis, assim como algumas traduções em formato pdf.

Participe

Como participar

Há muito que fazer dentro do projeto FreeCAD, se estiver interessado em nos ajudar. Há tarefas de programação, para programadores em C++ ou Python, mas também há muitas coisas que pode fazer mesmo se não souber programar, tais como:

  • escrever documentação e editando a wiki
  • ajudar os novos utilizadores
  • traduzir a aplicação e a documentação
  • ajudar a disponibilizar a última versão do FreeCAD para seu sistema operativo favorito
  • ou simplesmente ajudar outras pessoas a descobrir o FreeCAD.

A página Ajude o FreeCAD descreve tudo com mais detalhe. A partir de 2016, o FreeCAD também participa do Google Summer of Code.

Código fonte

O FreeCAD pode ser compilado em todas as plataformas usando o CMake. O código-fonte é LGPL-licenciado e disponível no GitHub e espelhado no https://gitlab.com/freecad/FreeCAD GitLab], SourceForge e CodeBerg. Existem instruções de compilação para Windows, Unix/Linux e MacOSX. A documentação do código-fonte está disponível aqui, gerada por [[[Doxygen/pt|Doxygen]], e documentada na wiki.

Sobre o desenvolvimento

Visite o Mapa de desenvolvimento para ver novidades sobre o que está a ser planeado, o Histórico de alterações e o Roteiro de alterações para ver o progresso para a próxima versão, ou as Estatísticas do projeto para ter ainda mais informação sobre o código de FreeCAD. Todas as comunicações sobre o desenvolvimento são feitas no forum, então não deixe de o visitar se estiver interessado em participar.
