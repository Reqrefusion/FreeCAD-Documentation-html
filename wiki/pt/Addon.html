Introdução

No FreeCAD e na sua documentação, uma extensão é um componente que não faz parte da instalação base, mas que pose ser adicionado ao sistema de várias formas.

Diferentes tipos

Há três tipos de extensões:

  • Macros: pequenos trechos de código Python que oferece uma nova ferramenta ou uma nova funcionalidade num só ficheiro, que termina em .FCMacro.
  • Bancadas de trabalho: Colecção de ficheiros Python que providenciam comandos Gui (ferramentas) relacionadas com um tópico em particular, por exemplo, ferramentas para desenhar móveis, ou ferramentas para trabalhar em arquitectura, ou ferramentas para desenhar barcos, etc. Estas bancadas de trabalho normalmente definem novas barras de ferramentas onde comandos aparecem como botões.
  • Pacotes de preferências: colecções de preferências do utilizador partilháveis. introduced in 0.20

Instalação

A forma recomendada para instalar extensões é com o Gestor de extensões.

Mas para macros e bancadas de trabalho a instalação manual também é possível:

  • Como instalar macros
  • Instalar mais bancadas de trabalho

Informação para desenvolvedores

Se desenvolveu uma macro ou uma bancada de trabalho e quer que este seja incluído no Gestor de extensões, leia sobre como o fazer nas páginas do repositório: (FreeCAD-addons e FreeCAD-macros). Se adicionar a sua Macro à página Receitas de Macros, não tem mais nada a fazer, porque vai ser automaticamente detectada pelo Gestor de extensões.
