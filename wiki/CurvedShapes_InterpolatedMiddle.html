Description

Interpolates a shape between two 2D curves. The interpolation can be used to form a surface or a solid.

Image of a simple FreeCAD project with two sketches using the InterpolatedMiddle tool

Usage

  1. Select two sketches that shall be used.
  2. There are several ways to invoke the command:
    • Press the Interpolated Middle button
    • Select the Curved Shapes → Interpolated Middle option from the menu

Notes

  • Be aware of the draw order of lines in a sketch as it influences the result greatly.
  • If you need a solid of the interpolation enable the "make Solid" property.
