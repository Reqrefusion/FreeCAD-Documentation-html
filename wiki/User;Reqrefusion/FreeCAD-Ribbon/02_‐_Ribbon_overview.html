The ribbon exists of the following sections:

  • Ribbon tab bar
  • Ribbon panels
  • Application menu
  • Quick access toolbar
  • The right toolbar

The ribbon tab bar

The order of the ribbon tab bar is determined by the order of workbench as they are set in the FreeCAD preferences -> Workbenches.

The ribbon panels

Ribbon panels are based on the FreeCAD toolbars including custom toolbars created for specific workbenches. The panels can have small, medium and large buttons. The panels are divided in rows. The number of rows per button size are:

  • small buttons are divided over 3 rows
  • Medium buttons are divided over 2 rows
  • large buttons are divided over 1 row (whole panel)

The application menu

The application menu contains FreeCAD's menus including workbench specific menus.

The quick access toolbar

The quick access toolbar contains commands that are often used in every workbench. As default these commands are:

  • New document
  • Save document
  • Undo
  • Redo
  • Reload
  • Cut
  • Copy
  • Paste

The right toolbar

The right toolbar has the following buttons:

  • The help button with FreeCAD's help menu and the additions for the Ribbon
  • The overlay button shows the buttons for the Ribbons implementation of the overlay functions
  • The preference button with a menu that contains:
    • The button for FreeCAD's preferences
    • The button for FreeCAD's customize dialog. (For shortcut keys and such)
    • The preference menu for the ribbon.
  • a Pin button to fix the ribbon or enable autohide.
