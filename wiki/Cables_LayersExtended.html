Description

The Layers Extended are a predefined set of layers, each of which is an extended version of a Draft Layer. The extended layer in addition to the standard properties of a Draft Layer, allows to define the color and size of the points of elements belonging to the layer. Each extended layer has also Auto Member Type property which is designed to automatically collect layer elements.

Extended layers were created to make it easier to organize WireFlex lines, SupportLines, SupportPoints, SnapLines, CableTerminals. These layers allow you to quickly and easily assign ungrouped objects of a specific type to the appropriate layer, and then hide or show them, as well as apply bulk changes to their view properties.

Simply recompute the selected layer (e.g. by right clicking it and selecting Recompute Object), and all ungrouped elements of the selected type will be assigned to it.

Usage

  1. Create default set of extended layers by one of these methods:
    • Press the Layers Extended button.
    • Select the Cable Support → Layers Extended option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cable Support → Layers Extended option from the context menu.

Notes

  • The default layer with Auto Member Type property set to None can serve as template for creation other extended layers.
  • It is possible to use more than one layer with the same Auto Member Type property by changing it manually.
  • If you run the LayersExtended command again, it will generate layers for all possible Auto Member Type types that have not yet been created, plus one additional layer with None Auto Member Type.

Properties

See also: Property View.

A Layer Extended object is derived from a Draft Layer and inherits all its properties. Layer Extended has also the following additional properties:

Data

Layer

  • DataAuto Member Type (Enumeration): specifies the type of automatic member of this layer. When layer is recomputed it automatically includes all objects of selected type which were not assigned to any other layer. Possible values: None, WireFlex, SupportLines, ExtSnapLines, CableTerminal. When None is selected the Auto Member Type becomes inactive.

View

Layer

  • DataPoint Color (Color): specifies the point color of the layer.
  • DataPoint Size (Float): specifies the point size of the layer.
