Introduction

Ve FreeCADu se slovo "Tělo" obvykle používá k označení objektu PartDesign Těla (třída PartDesign::Body), který je definován v Pracovním okně PartDesign. Jedná se o kontejnerový objekt, který může obsahovat 2D náčrtky a 3D geometrické prvky pro vytvoření pevného tvaru.

Viz PartDesign Tělo, kde najdete více informací o tomto typu objektu.

Použití

  1. Přepněte do pracovní plochy PartDesign.
  2. Stiskněte tlačítko Vytvořit tělo.
  3. Stiskněte tlačítko Vytvořit náčrt pro vytvoření nového náčrtu.
  4. Vytvořte uzavřený obrys a poté pomocí Deska vytáhněte náčrt a vytvořte počáteční těleso.
  5. Přidejte další náčrtky a desky a pomocí dalších nástrojů pracovní plochy PartDesign upravte a transformujte původní těleso.

Alternativně můžete místo náčrtků přidat základní prvky PartDesign, například Přídavný kvádr. K vytvoření konečného pevného tělesa lze použít libovolný počet přídavných a odečtových prvků.

Poznámky

A Body is required when using the PartDesign Workbench in a feature editing methodology.

A Body is not required when using the Part Workbench, as this workbench uses a constructive solid geometry workflow, which is based on primitive shapes and boolean operations.
