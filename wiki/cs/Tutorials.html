Na této stránce najdete výběr kvalitních psaných návodů. Úplný, netříděný seznam návodů je k dispozici v Category:Tutorials, zatímco úplný a řaditelný seznam naleznete v tabulce níže.

Chcete-li přispět psaním dokumentace a návodů pro wiki, seznamte se nejprve s obecnými pravidly wiki na stránce Stránky wiki a přečtěte si také Pokyny pro tvorbu návodů.

Věnujte prosím pozornost verzi FreeCADu použité v návodu, protože některé návody mohou být vytvořeny pro starší verzi programu. Přestože obecný postup modelování bývá stále použitelný, některé nástroje se mohly změnit.

Viz také videonávody a knihy.

Architktura a BIM

  • Arch_tutorial (v0.14)
    Základní úvod do pracovního prostředí BIM. Návod je rozsáhlý a ukazuje typický pracovní postup – od importu výkresů ve formátu DXF až po vytvoření 3D modelu.
  • Modelování BIM
    Jak vymodelovat malý dům, vytvořit technický výkres v pracovním prostředí TechDraw a exportovat model do formátu IFC.
  • Otevřená okna a dveře (v0.18)
    Jak zobrazit okna a dveře v otevřené poloze, včetně symbolů v pohledu a půdorysu, a vytvořit jednoduchý půdorys pomocí pracovního prostředí TechDraw.
  • Návrh vlastních oken (v0.18)
    Jak navrhnout vlastní dveře a okna pomocí pracovního prostředí Sketcher a upravit jejich normály tak, aby byly správně umístěny do stěn.
  • Návod pro panely Arch (v0.15)
    Modelování střešního panelu malého domu pomocí pracovního prostředí Sketcher, nástroje Okno a nástroje Panel.
  • Modelování WikiHouse
    Opětovné vytvoření projektu WikiHouse pomocí náčrtů a panelů, přičemž výchozím bodem je import modelu polygonové sítě vytvořeného ve SketchUpu.
  • Návod k NativeIFC
    Naučte se pracovat s nativními soubory IFC, objekty a základními koncepty v pracovním prostředí BIM.

Modelování dílů

FreeCAD nabízí dva hlavní pracovní postupy pro modelování dílů:

  • skládání objektů metodou konstruktivní geometrie těles (CSG) pomocí pracovního prostředí Part, a
  • parametrické modelování a úpravy prvků pomocí pracovního prostředí Part Design.

Upozorňujeme, že pracovní postup v pracovním prostředí Part Design prošel od FreeCADu 0.17 významnými změnami. Některé návody proto ještě nebyly aktualizovány a mohou odkazovat na verzi 0.16.

  • Vytvoření jednoduchého dílu v pracovním prostředí Part
    Úvod do FreeCADu a pracovního prostředí Part s využitím základních těles.
  • Vytvoření jednoduchého dílu pomocí pracovních prostředí Draft a Part
    Úvod do modelování těles pomocí pracovního prostředí Draft vytvořením 2D profilu.
  • Vytvoření jednoduchého dílu v pracovním prostředí Part Design (v0.17)
    Úvod do pracovního postupu v Part Design – vytvoření náčrtu, použití funkcí Přidání materiálu (Pad), Kapsa (Pocket) a přesunutí objektu.
  • Základní návod pro Part Design (v0.17)
    Vymodelujte jednoduchý díl pomocí metodiky úpravy prvků: vytvoření náčrtu, použití funkcí Přidání materiálu (Pad), externích referencí, Kapsa (Pocket) a Zrcadlení (Mirror).
  • Základní návod pro Part Design 0.19 (v0.19 a novější)
    Aktualizovaná verze předchozího návodu, která vytváří stejný model pomocí postupů eliminujících problém topologického pojmenování.
  • Modelování stojánku pro hlavice elektrického zubního kartáčku (v0.16 a novější)
    Používá více funkcí: náčrt, vazby vzdálenosti a totožnosti, Přidání materiálu (Pad), externí reference, Zaoblení (Fillet), Zkosení (Chamfer), Lineární pole (Linear Pattern) a Zúkos (Draft).
  • Modelování pro návrh výrobků (v0.16)
    Modelování kostky Lego: náčrty, vazby svislé a vodorovné vzdálenosti, Přidání materiálu (Pad), Kapsa (Pocket), externí reference, Lineární pole (Linear Pattern) a sestavení.
  • Tradiční modelování metodou CSG
    Modelování stolu pomocí jednoduchých těles, jako jsou kvádry a válce, a provádění booleovských operací (sjednocení a odečtení).
  • Návod pro Draft ShapeString (v0.19)
    Vytvoření gravírovaného textu na tělese: vysunutím objektu ShapeString vytvořte těleso a následně jej pomocí booleovské operace Odečtení vyřežte z jiného tělesa.
  • Vytvoření děrované koule (v0.19)
    Pomocí základních těles, jako jsou kvádry a válce, a booleovských operací, například Sjednocení a Odečtení, vytvořte dutou děrovanou kouli.
  • Základní návod k modelování
    Vytvoření úhelníku dvěma způsoby: pomocí základních těles a booleovských operací (CSG) a vysunutím rovinného profilu.
  • Návod na model letadla
    Pochopte práci s umístěním objektů ve FreeCADu vytvořením jednoduchého modelu letadla. Poté se naučte pracovat s rotačními úhly: vybočení (Yaw – osa Z), klopení (Pitch – osa Y) a náklon (Roll – osa X).
  • Návod pro vytváření závitů (v0.19)
    Naučte se vytvářet závity pomocí několika různých postupů využívajících nástroje Šroubovice Part, Přidání materiálu tažením po trajektorii, Tažení po trajektorii, Sjednocení a Odečtení.

Projekt Raspberry Pi vytvořil jednoduché návody, které se snadno sledují, zejména pro uživatele, kteří s CAD systémy teprve začínají:

  • freecad-dice – vymodelujte hrací kostku se šesti stěnami a případně si ji vytiskněte na 3D tiskárně.
  • freecad-headphone-tidy – vymodelujte navíječ pro uspořádání a uložení sluchátek a případně si jej vytiskněte na 3D tiskárně.
  • freecad-chess-set – vymodelujte kompletní šachovou soupravu v modernistickém stylu Bauhaus.
  • Repozitář raspberrypilearning (CC-BY 4.0) obsahuje další ukázkové projekty.

Kreslení a náčrtování

  • Tradiční 2D kreslení
    Nakreslete půdorys pomocí úseček, lomených čar, obdélníků a kruhových oblouků a doplňte šrafy, popisky a kóty. Výsledek exportujte do formátu DXF.
  • Návod pro pracovní prostředí Draft (v0.19)
    Základní úvod do nástrojů pracovního prostředí Draft: pracovní rovina, mřížka, úsečka, oblouk, povýšení geometrie (Upgrade), obdélník, kružnice, mnohoúhelník, pole, kóty, popisky a ShapeString.
  • Přednáška o pracovním prostředí Sketcher (v0.19)
    Více než 80stránkový dokument ve formátu PDF, který slouží jako podrobná příručka k pracovnímu prostředí Sketcher. Vysvětluje základy práce ve Sketcheru a podrobně popisuje vytváření geometrických tvarů i jednotlivé geometrické vazby.
  • Základní návod pro pracovní prostředí Sketcher (v0.19)
    Základní úvod do nástrojů pracovního prostředí Sketcher: konstrukční režim, úsečka, kružnice, oblouk a geometrické vazby (shodnost, svislost, vodorovnost, tečnost, vzdálenost, úhel a poloměr).
  • Procvičování geometrických vazeb ve Sketcheru (v0.19)
    Naučte se efektivně zavazbovat náčrty. Pokud je to možné, upřednostňujte geometrické vazby před rozměrovými.

Technické výkresy

  • Základní návod pro pracovní prostředí TechDraw (v0.17)
    Základní úvod do nástrojů pracovního prostředí TechDraw: stránka, pohled, měřítko, svislé a vodorovné kóty, popisky, skupiny promítání a propojení kót s 3D modelem.
  • Vytvoření nové šablony výkresu (v0.17)
    Návod, jak vytvořit šablonu výkresového listu v programu Inkscape pro použití v pracovním prostředí TechDraw. Naučíte se nastavit velikost listu, nakreslit rámeček výkresu a definovat pevná i editovatelná textová pole.
  • Měření úhlů na otvorech (v0.19)
    Návod pro přidání středových čar a následné kótování úhlů mezi otvory.

MKP (FEM)

  • Příprava geometrie a tvorba sítě pro FEM (v1.0)
    Obecný přehled nejčastějších problémů souvisejících s přípravou geometrie a tvorbou sítě pro analýzy metodou konečných prvků (FEM). Obsahuje řadu praktických doporučení a popisuje současné možnosti pracovního prostředí FEM při práci s geometrií a sítí.
  • Analýza konzolového nosníku v CalculiXu (v0.20)
    Tento příklad je součástí každé instalace FreeCADu a ukazuje základní analýzu pomocí MKP řešiče CalculiX. Odstraňte aktuální výsledky, znovu spusťte výpočet a poté si v grafickém okně zobrazte deformace a napětí na deformované síti.
  • Jednoduchý úvod do FEM (v0.20)
    Krátký úvod do jednotlivých kroků potřebných pro provedení analýzy v pracovním prostředí FEM: vytvoření modelu, vytvoření sítě, přidání okrajových podmínek a sil, přiřazení materiálu, spuštění výpočtu a zobrazení výsledků.
  • FEM analýza smyku kompozitního bloku (v0.17)
    Prozkoumejte deformaci bloku tvořeného tvrdým jádrem obklopeným měkčím materiálem: vytvořte oblasti sítě, přiřaďte materiály, nastavte kluzné okrajové podmínky, přidejte smykové zatížení, spusťte výpočet a výsledky zobrazte pomocí roviny řezu (Clip Plane).
  • Analýza železobetonu pomocí FEM (v0.19)
    Odhadněte potřebnou míru vyztužení betonové konstrukce tak, aby se zabránilo křehkému porušení při tahu nebo smyku.
  • Elektrostatická úloha – kapacita dvou koulí (v0.19)
    Tento příklad ukazuje, jak simulovat elektrickou kapacitu. Seznámíte se s přípravou modelu, jeho jednotlivými částmi, řešením pomocí řešiče Elmer a zobrazením výsledků pomocí filtru Clip.
  • Post-Processing of FEM results with Paraview (v0.19)
    This tutorial explains the basics of transferring data from the FEM Workbench to Paraview and shows some of the options and settings for displaying data.

CNC & 3D Printing

  • CAM Workbench for the impatient
    This is a quick presentation of the workflow for the CAM Workbench: create a job, define the output, define the milling tool, define the path operations, start the simulation, and generate a G-code output file.
  • Manual:Preparing models for 3D printing (v1.0)
    Convert a solid object to a mesh object using the Mesh Workbench, export the mesh to STL format, and use Slic3r to prepare the G-code. Alternatively use the Cura Workbench or the CAM Workbench to generate the G-code.

Rendering

  • Creating renderings
    Quickly produce a rendered image of your bodies with POV-Ray and LuxRender, if they are installed in your system.
  • Raytracing_tutorial (v0.16)
    Describes the basic workflow of the Raytracing Workbench using POV-Ray or LuxRender: set the path to the renderers, create a project, set the camera position, select the model, run the renderer.
  • Intermediate FreeCAD and POV-ray tutorial (v0.18)
    Workflow to produce a better render with POV-Ray: create a project, add objects, set the camera, save the .pov file, manually edit the file to improve the textures, planes, lights, and then run the renderer.
  • Rendering a FreeCAD assembly with Blender (v0.18)
    Export bodies from FreeCAD to Wavefront .obj, import the file into Blender, set up a simple Sun light, assign basic materials with the Principled BSDF shader, and produce a rendered image with EEVEE and Cycles.

Robot workbench

Development of the Robot Workbench is currently abandoned. Please inquire on our forum regarding any news or alternative workbenches.
  • Robot tutorial (v0.17)
    Simulate the movement of an industrial robot: set up a trajectory, set up home position, change the robot position, insert various waypoints, and simulate the robot movement.
