Úvod

Po instalaci FreeCADu pro váš operační systém (Windows, Linux nebo Mac), můžete zvážit instalaci jedné nebo více z následujících doplňkových komponent.

Soubory nápovědy

Viz Instalace souboru nápovědy.

Externí pracovní prostředí

Kromě výchozích pracovních ploch dodávaných s FreeCADem existuje také velká sbírka užitečných externích pracovních ploch vytvořených členy komunity.

Software třetích stran

FreeCAD podporuje několik softwarových balíčků třetích stran ihned po instalaci. V mnoha případech stačí pouze nainstalovat software a po restartu FreeCADu ho automaticky najde a bude moci používat. Tato sekce má za cíl poskytnout seznam takových softwarových balíčků spolu s informacemi o tom, kde se v FreeCADu používají a kde je lze stáhnout.

Podpora

GitPython

GitPython je knihovna pro práci s Git repozitáři. Tuto knihovnu může používat Správce doplňků. GitPython je součástí instalačních balíčků FreeCADu pro Windows a Mac.

GraphViz

GraphViz je open source software pro vizualizaci grafů. Používá se v nástroji Std Graf závislostí.

OpenCAMLib

OpenCAMLib je otevřená knihovna algoritmů pro počítačem podporovanou výrobu (CAM). Používá se v Pracovním prostředí CAM. Pokyny k instalaci najdete na stránce OpenCamLib.

OpenSCAD

OpenSCAD je robustní 3D modelář. Pracovní prostředí OpenSCAD je na tomto softwaru závislé a Pracovní prostředí Mesh ho používá pro své booleovské nástroje. Je také nezbytný pro import souborů SCAD pomocí nástroje Std Import.

Formáty souborů

Veškerý software v této části bude použit nástroji Import nebo Export.

CADExchanger

CADExchanger je komerční aplikace pro výměnu různých formátů CAD souborů. Pro použití této aplikace ve FreeCADu existuje externí pracovní prostředí.

DXF Importér

FreeCAD has a native importer and exporter for DXF files, programmed in C++. Currently they do not implement all features of the DXF format. For those features the legacy Python importer and exporter are still available. These require the Draft-dxf-importer Python library. See the FreeCAD and DXF Import page for more information.

DWG konvertory

FreeCAD nemůže přímo číst a zapisovat soubory DWG. Pro převod souborů DXF do formátu DWG a naopak využívá FreeCAD externí převaděče. Vestavěná podpora je k dispozici pro následující převaděče DWG:

  • LibreDWG (open-source, postrádá podporu pro některé entity DWG).
  • ODA File Converter (zdarma, ale ne open-source).
  • QCAD pro (komerční). introduced in 0.20

Více informací najdete v kapitolách Nastavení importu a exportu a FreeCAD a import DWG.

IfcOpenShell

IfcOpenShell je knihovna pro práci s formátem souborů Industry Foundation Classes (IFC) používaným v architektonickém navrhování. Knihovna je také používána nástroji Arch IfcExplorer (0.18 and below) a BIM IfcExplorer. IfcOpenShell je součástí instalačních balíčků FreeCAD pro Windows a Mac.

IfcJson

IfcJson je knihovna potřebná pro export do formátu souboru IFCJSON. IFCJSON je nový formát IFC, který zatím není podporován mnoha aplikacemi.

Pycollada

Pycollada, také známý jako python-collada, je knihovna Pythonu pro čtení a zápis souborů Collada (DAE). Pycollada je součástí instalačních balíčků FreeCAD pro Windows a Mac.

Vykreslování

LuxCoreRender

LuxCoreRender je renderovací engine, restart projektu LuxRender. Oficiálně není podporován pracovním prostředím Raytracing, ale možná stojí za to ho vyzkoušet. Oficiálně je podporován novým pracovním prostředím Render, který má v budoucnu nahradit pracovní prostředí Raytracing. Více informací a pokyny k instalaci najdete na stránce LuxCoreRender.

LuxRender

je jednou ze dvou renderovacích enginů podporovaných Pracovním prostředím Raytracing. V roce 2013 byl projekt restartován a stal se LuxCoreRender, s významným přepisem kódu a změnami narušujícími kompatibilitu. Oficiálně Raytracing Workbench podporuje pouze opuštěný LuxRender (poslední verze je 1.6, 2017-12-28), zatímco nové pracovní prostředí Render (určené jako budoucí náhrada pracovního prostředí Raytracing) podporuje místo toho LuxCoreRender a upustilo od podpory LuxRender. I když není oficiálně podporováno, LuxCoreRender může s pracovním prostředím Raytracing fungovat, takže stojí za to ho vyzkoušet. Více informací a pokyny k instalaci najdete na stránce LuxRender a pokud chcete vyzkoušet modernější software, podívejte se na stránku LuxCoreRender.

POV-Ray

POV-Ray je dobře známý ray-tracer, který dokáže vykreslovat fotorealistické obrázky. Je to jeden ze dvou renderovacích enginů, které v současné době podporuje pracovní plocha Raytracing. Více informací a pokyny k instalaci najdete na stránce POV-Ray.

Konečný prvek

CalculiX

CalculiX je sada dvou balíčků konečných prvků: CalculiX CrunchiX, řešič FEM, a CalculiX GraphiX, grafické uživatelské rozhraní. FreeCAD podporuje pouze řešič. Používá se nástrojem FEM SolverCalculixCcxtools.

Gmsh

Gmsh je automatický generátor konečných prvků. Používá se v nástrojích FEM MeshGmshFromShape a Mesh FromPartShape.

Elmer

Elmer is a multi-physics simulation software, which was open sourced in 2005. In FreeCAD its Grid and Solver modules are used by the FEM SolverElmer tool.

FEniCS

FEniCS je výpočetní platforma pro řešení parciálních diferenciálních rovnic (PDE), které se široce používají při řešení problémů FEM. Používá se v pracovním prostředí FEM.

Z88

Z88 je další program FEM, který obsahuje síťovací nástroj, řešič a převaděče. Používá se v nástroji FEM SolverZ88. FreeCAD vyžaduje balíček open source Z88OS.

OpenFOAM

OpenFOAM is a large collection of libraries for computational fluid dynamics (CFD) simulations. OpenFOAM is used by the Cfd and CfdOF external workbenches.
