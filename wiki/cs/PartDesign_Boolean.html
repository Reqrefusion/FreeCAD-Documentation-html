Popis

PartDesign Booleovské operace importuje jedno nebo více PartDesign těles nebo PartDesign klonů (označených jako "tělesa nástrojů") do aktivního PartDesign tělesa a aplikuje na ně booleovskou operaci (spojit, rozdělit nebo průnik).

Vlevo: aktivní těleso (A) s nástrojovými tělesy (B) a (C); výsledek vpravo po booleovském odečtení.

Použití

  1. [[PartDesign_Body/cs#Active_status|Aktivujte těleso, které má přijmout booleovskou vlastnost. Poznámka: Je důležité, aby nebylo vybráno ani aktivní těleso, ani žádná z vlastností, které obsahuje!
  2. Existuje několik způsobů, jak tento nástroj spustit:
    • Stiskněte tlačítko Booleovské operace.
    • V menu vyberte možnost Part Design → Booleovská operace.
  3. In Boolovské parametry, klikněte na tlačítko Přidat tělo. Aktivní tělo dočasně zmizí z 3D pohledu, aby se usnadnil výběr nástroje těla.
  4. Ve 3D pohledu vyberte Tělo, které chcete použít v booleovské funkci. Opakujte postup pro přidání dalších Těl.
  5. V rozbalovacím menu vyberte typ booleovské operace (Spojit, Rozdělit nebo Společné).
  6. KLikněte naOK.

Alternatively, objects can be selected prior to pressing the Boolean button. They will be added automatically. When pre-selecting you are not restricted to Bodies. PartDesign ShapeBinder, PartDesign SubShapeBinder and even solids created with other workbenches can then also be selected.

Options

  • Fuse: merges the tool body or bodies to the active body.
  • Cut: subtracts the tool body or bodies from the active body.
  • Common: extracts the intersection from the selected body or bodies with the active body
  • Press the Remove Body button to remove a body, by selecting it in the 3D View.

Properties

  • ÚdajeType: sets the Boolean operation (Fuse, Cut, Common)
  • ÚdajeLabel: name given to the operation, this name can be changed at convenience.
  • ÚdajeGroup: lists the tool bodies.
  • PohledDisplay: sets the display between 2 modes:
    • Result (default): displays the result of the Boolean feature. In this mode, the tool Bodies cannot be displayed in their original state, even when their visibility is toggled on.
    • Tools: displays the tool Bodies in their original state. This mode is useful when tool Bodies need to be edited, or used in later operations.
  • PohledSelectable: true or false. If set to false, the feature cannot be selected in the 3D View.
  • PohledVisibility: true or false. Toggles the feature's visibility in the 3D View.

Limitations

  • For Common to work with more than one tool body, they must all intersect each other along with the active Body.
  • Tool bodies adopt the local origin of the active Body. If the active Body is not located at (0,0,0) in the global coordinate system, the tool bodies' placement need to be relative to the active Body. It may be easier to leave the active Body's placement at the origin before applying the Boolean feature, than to change its placement.
