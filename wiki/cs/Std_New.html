Description

The Std New command creates a new empty document and makes it the active document.

Usage

  1. There are several ways to invoke the command:
    • Press the New Document button.
    • Select the File → New Document option from the menu.
    • Use the keyboard shortcut: Ctrl+N.

Preferences

See also: Preferences Editor.

  • By default FreeCAD start without a new document. Check the Edit → Preferences → General → Document → Create new document at start up option to change this behavior.
  • Some document properties: author name, company name and license information, can be preset: Edit → Preferences → General → Document → Authoring and License.

Properties

See also: Property View.

Most properties can also be changed in the dialog box of the Std ProjectInfo command.

Data

Base

  • ÚdajeComment (String): Any comment that may apply.
  • ÚdajeCompany (String): Company name.
  • ÚdajeCreated By (String): Author name.
  • ÚdajeCreation Date (String): Automatic date stamp (read-only).
  • ÚdajeFile Name (String): The full path of the file. Blank if the document has not been saved (read-only).
  • ÚdajeId (String): Not implemented yet.
  • ÚdajeLabel (String): The name that will appear in the Tree View. Replaced by the name of the document after reopening.
  • ÚdajeLast Modified By (String): Author name.
  • ÚdajeLast Modified Date (String): Automatic date stamp (read-only).
  • ÚdajeLicense (String): License type.
  • ÚdajeLicense URL (String): License URL.
  • Údaje (Hidden)Material (Map): Map with material properties.
  • Údaje (Hidden)Meta (Map): Map with additional meta information.
  • ÚdajeShow Hidden (Bool): If true, items that have been hidden in the Tree View will be displayed anyway. Hiding items in the tree can be useful when working on larger models.
  • ÚdajeTip (Link): Not implemented yet.
  • ÚdajeTip Name (String): Not implemented yet.
  • ÚdajeTransient Dir (String): The transient directory used for recovery data (read-only).
  • Údaje (Hidden)Uid (UUID): UUID of the document (read-only).
  • ÚdajeUnit System (Enumeration): The document unit system. Initial value depends on the Default unit system. introduced in 1.0
  • ÚdajeUse Hasher (Bool): If true, a hasher is used to mitigate the topological naming problem. introduced in 1.0

Scripting

See also: Autogenerated API documentation and FreeCAD Scripting Basics.

To create a new document use the newDocument([name], [hidden=False]) method of the FreeCAD application. The document name must be unique, which is checked automatically. If no name is supplied, the document will be named "Untitled". If hidden=True is used, the new document won't be displayed in the GUI and no tab will appear for it. 

import FreeCAD
from pathlib import Path

# The folder and filename we will use:
fld = "D:/testfiles/"
fnm = fld + "test.FCStd"

# Make sure fld exists:
Path(fld).mkdir(parents=True, exist_ok=True)

doc = FreeCAD.newDocument()
doc.saveAs(fnm)

FreeCAD.closeDocument(doc.Name)

doc = FreeCAD.open(fnm)
doc.save()

FreeCAD.closeDocument(doc.Name)
