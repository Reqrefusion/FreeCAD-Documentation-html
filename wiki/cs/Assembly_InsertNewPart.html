Description

The Assembly InsertNewPart tool inserts a new Part container, optionally embedded in a new document, and adds it to the currently active Assembly.

Usage

  1. There are several ways to invoke the tool:
    • Press the New Part button.
    • Select the Assembly → Insert Component → New Part option from the menu.
