Popis

The Part MakeSolid command creates solids from shape objects.

This command is typically used as one of the steps to create a solid from a mesh. See Part ShapeFromMesh for more information.

Usage

  1. Select one or more shape objects.
  2. Select the Part → Convert to Solid option from the menu.
  3. For each selected object a solid is created as a separate new object.

Notes

  • The selected shape objects are not analyzed or validated.
  • It is recommended to use Part RefineShape before converting to a solid.

Properties

See also: Property View.

The created objects are Part Feature objects with no additional properties.
