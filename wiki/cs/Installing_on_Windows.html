Standardní instalace

Nejjednodušší způsob, jak nainstalovat nejnovější stabilní verzi FreeCADu, je použít instalační program, více informací najdete na stránce Download.

Pokud si chcete stáhnout vývojovou verzi, která může být nestabilní, podívejte se na stránku weekly builds.

Po stažení instalačního programu na něj dvakrát klikněte a spusťte instalační proces.

Níže najdete další informace o některých technických možnostech. Většina uživatelů však nepotřebuje nic víc než instalační program. Po dokončení instalace přejděte na stránku Začínáme.

Instalace pro všechny uživatele systému Windows

Ve výchozím nastavení bude FreeCAD nainstalován pro uživatele, který spustí instalační program. Pokud má tento uživatel pouze uživatelská oprávnění, výchozí instalační cesta je:

C:\Users\<username>\AppData\Local\Programs\FreeCAD X.YY

Pokud instalační program spustí uživatel s právy správce nebo ho spustíte jako správce, můžete si vybrat, zda má být FreeCAD nainstalován pro všechny uživatele systému, nebo pouze pro vás. Výchozí nastavení je pro všechny uživatele.

Pokud je nainstalován pro všechny uživatele, výchozí instalační cesta je:

C:\Program Files\FreeCAD X.YY

Tichá instalace

Chcete-li nainstalovat FreeCAD v tichém režimu, můžete spustit instalační program z příkazového řádku: 

FreeCAD-~.exe /S

Pro všechny možnosti budou použita výchozí nastavení. Vlastní instalační cestu lze zadat tímto způsobem: 

FreeCAD-~.exe /S /D=A path to FreeCAD with spaces

Ve výchozím nastavení se i při tiché instalaci zobrazí krátké vyskakovací okno, když se kontroluje, zda instalační program není poškozen. Tato takzvaná cyklická redundance trvá nanejvýš několik sekund. Chcete-li tuto kontrolu poškození deaktivovat: 

FreeCAD-~.exe /S /NCRC

Upozorňujeme, že tento příznak /NCRC se ‚‘'nedoporučuje'‚‘ používat, protože kontrola poškození například zajišťuje, že instalační program byl stažen kompletně.

Chocolatey

Důrazně doporučujeme používat správce balíčků, jako je Chocolatey, aby byl váš software vždy aktuální. Chocolatey můžete nainstalovat postupem podle těchto pokynů a poté otevřít terminál PowerShell jako správce a spustit: 

choco install freecad

Občas můžete aktualizovat svůj software pomocí: 

choco upgrade freecad

Tím získáte nejnovější verzi dostupnou v úložišti Chocolatey. Pokud máte nějaké problémy s balíčkem Chocolatey, můžete kontaktovat správce na této stránce.

Odinstalace

K odinstalování FreeCADu je vhodné použít nástroje Windows pro odinstalování softwaru. Alternativně můžete spustit přímo odinstalační program. Jedná se o tento soubor:

Uninstall-FreeCAD.exe

Najdete ho ve složce, kde je nainstalován FreeCAD.

Odinstalační program lze také spustit v tichém režimu pomocí příkazového řádku: 

Uninstall-FreeCAD.exe /S

Upozorňujeme, že (tichá) odinstalace selže, pokud běží instance FreeCADu, i když se nejedná o verzi, která se odinstalovává.
