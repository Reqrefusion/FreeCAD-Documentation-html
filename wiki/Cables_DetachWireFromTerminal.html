Description

The Detach Wire From Terminal detaches previously attached vertices of WireFlex from existing CableTerminal. Attachments of vertices are removed from special properties of WireFlex, for details see WireFlex Properties.

Detachment does not change existing terminal or wire positions. The electrical connection between terminal and wire no longer exists. The terminal properties from the Net group are updated accordingly.

When the CableTerminal is moved afterwards, the detached wire vertices stay in place. When the wire is edited in the Cables Edit the detached vertices can be moved freely.


Usage

  1. Select WireFlex object in the 3D View or in the Tree View.
  2. Select CableTerminal in the 3D View or in the Tree View holding the CTRL.
  3. Run an Detach Wire From Terminal command by one of these methods:
    • Press the Detach Wire From Terminal button.
    • Select the Cable Wires → Detach Wire From Terminal option from the menu.
    • Right-click in the Tree View or the 3D View and select the Cable Wires → Detach Wire From Terminal option from the context menu.
    • Use the keyboard shortcut: T then R.

Notes

  • After detachment the terminal has free space to attach another wire.
  • See the WireFlex Description and WireFlex Usage for the details how to use WireFlex.
