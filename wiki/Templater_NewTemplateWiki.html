Description

The Templater NewTemplateWiki tool creates a small example template and inserts it as a new drawing page into the active document. See the TechDraw TemplateGenerator tutorial to learn how an almost identical template is scripted.

(Image borrowed from the tutorial page)

Usage

  1. There are several ways to invoke the tool.
    • Press the New Template Wiki button.
    • Select the Templater → New Template Wiki option from the menu.
  2. The Template settings task panel opens.
    • Optionally check the Ink-blue option to color editable texts.
  3. Click Ok to close the task panel and finish the tool.
  4. A new template will be saved in the .../Mod/Templater/Resources folder.
  5. The created template will be displayed in a new TechDraw view.
